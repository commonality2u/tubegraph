## Updating the Facebook Application

Before proceeding with the dark mode setup, it is crucial to ensure the Facebook application is downloaded and updated to its latest version <a class="yt-timestamp" data-t="00:00:52">[00:00:52]</a>, <a class="yt-timestamp" data-t="00:00:57">[00:00:57]</a>.

To update the application:
1.  Go to the App Store (for iPhone) or Google Play Store (for Android) <a class="yt-timestamp" data-t="00:01:00">[00:01:00]</a>.
2.  Search for "Facebook" <a class="yt-timestamp" data-t="00:01:06">[00:01:06]</a>.
3.  Click on the Facebook application <a class="yt-timestamp" data-t="00:01:09">[00:01:09]</a>.
4.  Tap "Update" if an update is available <a class="yt-timestamp" data-t="00:01:10">[00:01:10]</a>. If it's already updated, tap "Open" <a class="yt-timestamp" data-t="00:01:12">[00:01:12]</a>, <a class="yt-timestamp" data-t="00:01:13">[00:01:13]</a>.

For more details on [[updating_facebook_app_on_iphone_and_android | updating Facebook app on iPhone and Android]], refer to related resources.

## Activating Dark Mode

Once the app is updated and opened:
1.  Tap the three lines located on the bottom right of the menu <a class="yt-timestamp" data-t="00:01:16">[00:01:16]</a>, <a class="yt-timestamp" data-t="00:01:19">[00:01:19]</a>.
2.  Navigate to the "Configuration and privacy" section <a class="yt-timestamp" data-t="00:01:21">[00:01:21]</a>.
3.  Look for an option called "Dark mode," which is represented by a moon icon <a class="yt-timestamp" data-t="00:01:22">[00:01:22]</a>, <a class="yt-timestamp" data-t="00:01:27">[00:01:27]</a>.
4.  Tap on the "Dark mode" option <a class="yt-timestamp" data-t="00:01:28">[00:01:28]</a>.

You will find three options within the dark mode settings <a class="yt-timestamp" data-t="00:01:31">[00:01:31]</a>:

*   **System**: This option is selected by default <a class="yt-timestamp" data-t="00:01:34">[00:01:34]</a>. It adjusts Facebook's display to match your mobile phone's system-wide dark mode setting <a class="yt-timestamp" data-t="00:01:36">[00:01:36]</a>, <a class="yt-timestamp" data-t="00:01:40">[00:01:40]</a>. If your phone is in dark mode, Facebook will be dark; if your phone is in light/white mode, Facebook will be light <a class="yt-timestamp" data-t="00:01:43">[00:01:43]</a>, <a class="yt-timestamp" data-t="00:01:47">[00:01:47]</a>. This ensures consistency across all applications <a class="yt-timestamp" data-t="00:02:13">[00:02:13]</a>, <a class="yt-timestamp" data-t="00:02:16">[00:02:16]</a>. For more information on [[using_system_settings_for_facebook_dark_mode | using system settings for Facebook dark mode]], check relevant guides.

*   **Deactivate**: This option turns off dark mode specifically for Facebook, even if your phone's system is in dark mode. This means Facebook will appear in white/light mode <a class="yt-timestamp" data-t="00:01:53">[00:01:53]</a>, <a class="yt-timestamp" data-t="00:01:56">[00:01:56]</a>, <a class="yt-timestamp" data-t="00:01:57">[00:01:57]</a>.

*   **Activate**: This option forces Facebook into dark mode, regardless of your mobile phone's system settings <a class="yt-timestamp" data-t="00:02:00">[00:02:00]</a>, <a class="yt-timestamp" data-t="00:02:04">[00:02:04]</a>, <a class="yt-timestamp" data-t="00:02:07">[00:02:07]</a>.

For a general guide on [[how_to_activate_dark_mode_on_facebook | how to activate dark mode on Facebook]], see other resources.

### Dark Mode Appearance

The dark mode in Facebook is not entirely black; it uses a shade of gray that is slightly lighter than true black <a class="yt-timestamp" data-t="00:02:20">[00:02:20]</a>, <a class="yt-timestamp" data-t="00:02:23">[00:02:23]</a>, <a class="yt-timestamp" data-t="00:02:27">[00:02:27]</a>.
