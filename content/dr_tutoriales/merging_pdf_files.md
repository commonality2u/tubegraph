This guide explains how to merge multiple PDF documents into a single file, using the free online tool [[using_i_love_pdf_to_join_pdfs | I Love PDF]]. This process does not require any external programs, only a web browser like Google Chrome <a class="yt-timestamp" data-t="00:00:17">[00:00:17]</a>.

## How to Merge PDFs

Follow these steps to combine your PDF files:

1.  **Access the Tool**
    Open Google Chrome <a class="yt-timestamp" data-t="00:00:14">[00:00:14]</a> and navigate to [[using_i_love_pdf_to_join_pdfs | I Love PDF]] <a class="yt-timestamp" data-t="00:00:46">[00:00:46]</a>. Specifically, look for the "Merge PDF" or "Join PDF" section <a class="yt-timestamp" data-t="00:00:49">[00:00:49]</a>, <a class="yt-timestamp" data-t="00:00:52">[00:00:52]</a>.
2.  **[[selecting_and_uploading_pdfs_for_merging | Select and Upload PDFs]]**
    On the [[using_i_love_pdf_to_join_pdfs | I Love PDF]] website, click the button to [[selecting_and_uploading_pdfs_for_merging | select your PDF files]] <a class="yt-timestamp" data-t="00:01:14">[00:01:14]</a>. Browse to the location of your PDFs (e.g., Safari and Multicinema tickets are used as examples) <a class="yt-timestamp" data-t="00:01:22">[00:01:22]</a>, <a class="yt-timestamp" data-t="00:01:29">[00:01:29]</a>. Select all the PDF files you wish to merge <a class="yt-timestamp" data-t="00:01:29">[00:01:29]</a> and click "Open" <a class="yt-timestamp" data-t="00:01:29">[00:01:29]</a>.
3.  **Initiate Merging**
    Once the PDFs have loaded onto the website <a class="yt-timestamp" data-t="00:01:38">[00:01:38]</a>, click the "Merge PDF" or "Join PDF" button <a class="yt-timestamp" data-t="00:01:38">[00:01:38]</a>, <a class="yt-timestamp" data-t="00:01:43">[00:01:43]</a>.
4.  **[[downloading_and_renaming_merged_pdf | Download Merged PDF]]**
    After the merging process is complete, you can [[downloading_and_renaming_merged_pdf | download the newly merged PDF file]] <a class="yt-timestamp" data-t="00:01:43">[00:01:43]</a>. The file will typically be saved in your Downloads section with a combined name (e.g., "Multicinema Safari Merge") <a class="yt-timestamp" data-t="00:01:55">[00:01:55]</a>. You can [[downloading_and_renaming_merged_pdf | rename the file]] as desired <a class="yt-timestamp" data-t="00:01:55">[00:01:55]</a>.
5.  **Verify Merged File**
    Open the downloaded PDF to confirm that all original documents have been successfully combined into a single file <a class="yt-timestamp" data-t="00:01:50">[00:01:50]</a>, <a class="yt-timestamp" data-t="00:02:00">[00:02:00]</a>.
