The Glass extension is a [[using_web_browsers_for_installation | Google Chrome]] extension that utilizes Chat GPT to convert YouTube video content into text transcripts <a class="yt-timestamp" data-t="00:00:11">[00:00:11]</a> <a class="yt-timestamp" data-t="00:00:17">[00:00:17]</a>. This tool simplifies the process of extracting dialogue or information from videos <a class="yt-timestamp" data-t="00:00:17">[00:00:17]</a> <a class="yt-timestamp" data-t="00:00:20">[00:00:20]</a>.

## Availability
While the Glass extension can be installed on [[using_web_browsers_for_installation | Google Chrome]] and Safari, and its parent code can be used with Python <a class="yt-timestamp" data-t="00:00:23">[00:00:23]</a> <a class="yt-timestamp" data-t="00:00:25">[00:00:25]</a>, this guide focuses on its installation and use within [[using_web_browsers_for_installation | Google Chrome]] <a class="yt-timestamp" data-t="00:00:32">[00:00:32]</a>.

## Installation Process
To install the Glass extension on [[using_web_browsers_for_installation | Google Chrome]]:
1.  Open a new tab in your [[using_web_browsers_for_installation | Chrome]] browser <a class="yt-timestamp" data-t="00:00:40">[00:00:40]</a>.
2.  Navigate to the [[using_web_browsers_for_installation | Chrome]] Web Store or search for "Glass extension" <a class="yt-timestamp" data-t="00:00:34">[00:00:34]</a>.
3.  Add the extension to your [[using_web_browsers_for_installation | Chrome]] browser <a class="yt-timestamp" data-t="00:00:34">[00:00:34]</a>.

## Using the Extension
Once the Glass extension is installed, a small window displaying "transcript and summery" (transcript and abstract) will appear on YouTube video pages <a class="yt-timestamp" data-t="00:00:57">[00:00:57]</a> <a class="yt-timestamp" data-t="00:01:00">[00:01:00]</a>.

To use the extension:
1.  Open any YouTube video <a class="yt-timestamp" data-t="00:00:43">[00:00:43]</a>.
2.  Click on the "transcript and summery" window <a class="yt-timestamp" data-t="00:01:02">[00:01:02]</a>.
3.  The full transcript of the video will be displayed, with each paragraph appearing at specific time sequences (e.g., seconds or minutes) <a class="yt-timestamp" data-t="00:01:07">[00:01:07]</a> <a class="yt-timestamp" data-t="00:01:10">[00:01:10]</a> <a class="yt-timestamp" data-t="00:01:13">[00:01:13]</a>. The entire video's dialogue is available, including content up to the final spoken word <a class="yt-timestamp" data-t="00:01:15">[00:01:15]</a> <a class="yt-timestamp" data-t="00:01:19">[00:01:19]</a> <a class="yt-timestamp" data-t="00:01:22">[00:01:22]</a>.

### Copying Transcripts
The transcript content can be easily copied:
*   **Manual Selection**: Select the desired text directly from the window and copy it to a program like Word <a class="yt-timestamp" data-t="00:01:27">[00:01:27]</a> <a class="yt-timestamp" data-t="00:01:29">[00:01:29]</a>.
*   **Direct Copy**: Use the "Copy as text directly" option within the extension to copy the entire transcript <a class="yt-timestamp" data-t="00:01:34">[00:01:34]</a> <a class="yt-timestamp" data-t="00:01:37">[00:01:37]</a>. Once copied, the text can be pasted anywhere <a class="yt-timestamp" data-t="00:01:40">[00:01:40]</a> <a class="yt-timestamp" data-t="00:01:42">[00:01:42]</a>.
