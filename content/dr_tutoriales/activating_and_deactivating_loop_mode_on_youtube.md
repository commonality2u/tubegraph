Loop mode on YouTube allows users to automatically replay a video once it finishes, without needing to manually click replay or restart the video <a class="yt-timestamp" data-t="00:00:05">[00:00:05]</a>. This feature is useful for continuous listening to music or watching videos without interruption <a class="yt-timestamp" data-t="00:00:17">[00:00:17]</a>.

## How to Activate Loop Mode

To activate loop mode for a YouTube video:

1.  Access any YouTube video, for example, by using a browser like Google Chrome <a class="yt-timestamp" data-t="00:00:36">[00:00:36]</a><a class="yt-timestamp" data-t="00:00:42">[00:00:42]</a>.
    *   *Example:* Searching for "how to [[creating_youtube_playlists | create playlists]] in 2021" <a class="yt-timestamp" data-t="00:00:46">[00:00:46]</a>.
2.  While the video is playing, **right-click** anywhere on the video frame <a class="yt-timestamp" data-t="00:01:07">[00:01:07]</a>.
3.  A dropdown menu will appear <a class="yt-timestamp" data-t="00:01:10">[00:01:10]</a>.
4.  From this menu, click on **"Loop"** <a class="yt-timestamp" data-t="00:01:13">[00:01:13]</a>.

Once activated, the video will automatically restart from the beginning when it reaches its end <a class="yt-timestamp" data-t="00:01:16">[00:01:16]</a><a class="yt-timestamp" data-t="00:01:34">[00:01:34]</a>.

## How to Deactivate Loop Mode

To deactivate loop mode:

1.  While the video is playing and in loop mode, **right-click** on the video screen again <a class="yt-timestamp" data-t="00:01:45">[00:01:45]</a>.
2.  The same dropdown menu will appear <a class="yt-timestamp" data-t="00:01:47">[00:01:47]</a>.
3.  Click on **"Loop"** once more <a class="yt-timestamp" data-t="00:01:49">[00:01:49]</a>.

Deactivating loop mode ensures that when the video finishes, it will end and pause playback instead of restarting <a class="yt-timestamp" data-t="00:01:57">[00:01:57]</a><a class="yt-timestamp" data-t="00:02:02">[00:02:02]</a>.
