Google Calendar can be shared with others, either publicly or with specific individuals, allowing for collaborative scheduling and event management <a class="yt-timestamp" data-t="00:00:11">[00:00:11]</a>.

## Accessing Google Calendar
To access Google Calendar, open a new tab in your web browser and click on the "little squares" (Google applications icon) located at the top right of the page <a class="yt-timestamp" data-t="00:00:17">[00:00:17]</a>. Then, select "Calendar" <a class="yt-timestamp" data-t="00:00:26">[00:00:26]</a>.

## [[Creating a new Google Calendar | Creating a New Calendar]]
To [[Creating a new Google Calendar | create a new calendar]] for sharing:
1.  Navigate to "Other calendars" <a class="yt-timestamp" data-t="00:01:06">[00:01:06]</a>.
2.  Click the "+" button <a class="yt-timestamp" data-t="00:01:10">[00:01:10]</a>.
3.  Select "Create a calendar" <a class="yt-timestamp" data-t="00:01:12">[00:01:12]</a>.
4.  Enter a name for the calendar, such as "video YouTube tutorials" <a class="yt-timestamp" data-t="00:01:18">[00:01:18]</a>.
5.  Add an optional description <a class="yt-timestamp" data-t="00:01:21">[00:01:21]</a>.
6.  Confirm the time zone and owner <a class="yt-timestamp" data-t="00:01:32">[00:01:32]</a>.
7.  Click "Create" <a class="yt-timestamp" data-t="00:01:36">[00:01:36]</a>.
The calendar will then be created and synchronized with your Google Calendar <a class="yt-timestamp" data-t="00:01:39">[00:01:39]</a> and will appear on the left side of the main screen <a class="yt-timestamp" data-t="00:01:56">[00:01:56]</a>.

### Customizing Calendar Appearance
Once created, you can customize the calendar's appearance. Click the three dots next to the calendar name on the left to edit it <a class="yt-timestamp" data-t="00:02:06">[00:02:06]</a>. For example, you can change its color, such as to red <a class="yt-timestamp" data-t="00:02:14">[00:02:14]</a>.

## [[Making a Google Calendar public or private | Configuring Sharing Options]]
To configure [[Making a Google Calendar public or private | sharing and privacy settings]] for your calendar:
1.  Click the three dots next to the calendar name.
2.  Select "Configure and share" <a class="yt-timestamp" data-t="00:02:29">[00:02:29]</a>.

### Sharing with Specific People (Private)
If you wish to share the calendar only with selected individuals:
1.  In the "Configure and share" settings, scroll down to "Share with specific people" <a class="yt-timestamp" data-t="00:02:59">[00:02:59]</a>.
2.  Click "Add people" <a class="yt-timestamp" data-t="00:03:00">[00:03:00]</a>.
3.  Enter the email address of the person you wish to invite <a class="yt-timestamp" data-t="00:03:01">[00:03:01]</a>.
4.  Set their permissions:
    *   "See all event details" <a class="yt-timestamp" data-t="00:03:09">[00:03:09]</a>
    *   "Make changes to events" <a class="yt-timestamp" data-t="00:03:12">[00:03:12]</a>
    *   "Make changes and manage sharing" (administrator privileges) <a class="yt-timestamp" data-t="00:03:13">[00:03:13]</a>
5.  Click "Send" to send the invitation <a class="yt-timestamp" data-t="00:03:19">[00:03:19]</a>. The invited person will then have access once they accept <a class="yt-timestamp" data-t="00:03:22">[00:03:22]</a>.

### Sharing Publicly
To make the calendar accessible to everyone:
1.  In the "Configure and share" settings, find the option "Make available to public" <a class="yt-timestamp" data-t="00:03:32">[00:03:32]</a>.
2.  Check the box to enable public sharing <a class="yt-timestamp" data-t="00:03:36">[00:03:36]</a>. This allows anyone to view the calendar, including those with Yahoo or Photomail accounts <a class="yt-timestamp" data-t="00:03:50">[00:03:50]</a>.

### Sharing with Non-Gmail Accounts
If you need to share with someone who does not have a Gmail account, you have two options:
1.  Share the calendar publicly, as described above, allowing anyone to view it <a class="yt-timestamp" data-t="00:03:48">[00:03:48]</a>.
2.  Advise them to [[Creating a new Google Calendar | create a Gmail account]], which is a simple process that takes only a few minutes <a class="yt-timestamp" data-t="00:03:56">[00:03:56]</a>. This allows for private sharing <a class="yt-timestamp" data-t="00:04:01">[00:04:01]</a>.

## [[Configuring notifications and events in Google Calendar | Configuring Notifications for Shared Calendars]]
You can also [[Configuring notifications and events in Google Calendar | configure notifications]] for shared calendars to alert users about events:
1.  In the calendar settings, navigate to "Other notifications" <a class="yt-timestamp" data-t="00:04:24">[00:04:24]</a>.
2.  Add notifications to specify when reminders should be sent (e.g., 10 minutes, 10 hours, or 4 hours in advance) <a class="yt-timestamp" data-t="00:04:11">[00:04:11]</a>.
3.  Configure notifications for various event types, such as events themselves or canceled events <a class="yt-timestamp" data-t="00:04:27">[00:04:27]</a>. These notifications can be sent via email or mobile device, depending on user configuration <a class="yt-timestamp" data-t="00:05:35">[00:05:35]</a>.

## Adding Events to a Shared Calendar
To add an event to a shared calendar:
1.  On the main calendar view, click the desired date (e.g., April 23rd) <a class="yt-timestamp" data-t="00:04:40">[00:04:40]</a>.
2.  Click the "Add event" button <a class="yt-timestamp" data-t="00:04:50">[00:04:50]</a>.
3.  Enter a title for the event, such as "video calendar tutorial" <a class="yt-timestamp" data-t="00:04:52">[00:04:52]</a>.
4.  Specify the date <a class="yt-timestamp" data-t="00:04:59">[00:04:59]</a>.
5.  (Optional) Add a location or conference details <a class="yt-timestamp" data-t="00:05:05">[00:05:05]</a>.
6.  Add a description <a class="yt-timestamp" data-t="00:05:13">[00:05:13]</a>.
7.  Crucially, select the desired shared calendar (e.g., "YouTube video tutorials") from the calendar dropdown <a class="yt-timestamp" data-t="00:05:20">[00:05:20]</a>.
8.  Click "Save" <a class="yt-timestamp" data-t="00:05:24">[00:05:24]</a>.

Once an event is added to a shared calendar, anyone with access to that calendar will see the event and receive notifications based on their settings <a class="yt-timestamp" data-t="00:05:32">[00:05:32]</a>.
