Windows Defender is a built-in antivirus software that helps protect your computer from external threats <a class="yt-timestamp" data-t="00:01:04">[00:01:04]</a>. However, it can sometimes block downloads and updates of certain products or programs <a class="yt-timestamp" data-t="00:00:10">[00:00:10]</a>. Disabling it can be useful for proceeding with specific installations or updates that might otherwise be blocked <a class="yt-timestamp" data-t="00:00:17">[00:00:17]</a>.

## Step-by-Step Guide to Disabling Windows Defender

To disable [[how_to_activate_windows_defender_step_by_step | Windows Defender]] in Windows 10, follow these steps:

1.  **Access Windows Security**
    *   Go to your computer's Windows taskbar <a class="yt-timestamp" data-t="00:00:39">[00:00:39]</a>.
    *   On the left side, from bottom to top, type "defend" <a class="yt-timestamp" data-t="00:00:43">[00:00:43]</a>.
    *   Select the "[[navigating_windows_security_options | Windows security]]" option <a class="yt-timestamp" data-t="00:00:49">[00:00:49]</a>.
    *   Alternatively, you can access [[how_to_activate_windows_defender_step_by_step | Windows Defender]] from the menu on the left side <a class="yt-timestamp" data-t="00:01:31">[00:01:31]</a>.

2.  **Navigate to Antivirus Protection Settings**
    *   Within [[navigating_windows_security_options | Windows Security]], go to "antivirus protection against threats" <a class="yt-timestamp" data-t="00:00:54">[00:00:54]</a>. This is typically the first option listed <a class="yt-timestamp" data-t="00:01:10">[00:01:10]</a>.
    *   If you wish to deactivate other systems like firewall protection, you would go to the option on the right, typically labeled "number three" <a class="yt-timestamp" data-t="00:01:16">[00:01:16]</a>. For [[how_to_activate_windows_defender_step_by_step | Windows Defender]], focus on the left option <a class="yt-timestamp" data-t="00:01:26">[00:01:26]</a>.

3.  **Manage Settings**
    *   Once you've accessed "antivirus protection against threats," navigate to the "managing the configuration" section <a class="yt-timestamp" data-t="00:01:42">[00:01:42]</a>.

4.  **Disable Protection Features**
    *   To disable [[how_to_activate_windows_defender_step_by_step | Windows Defender]], you need to change the blue bar (toggle switch) from "on" to "deactivated" <a class="yt-timestamp" data-t="00:01:51">[00:01:51]</a>.
    *   A confirmation prompt will appear, asking if you want to allow the application to make changes to the device <a class="yt-timestamp" data-t="00:01:57">[00:01:57]</a>. Select "Yes" <a class="yt-timestamp" data-t="00:02:17">[00:02:17]</a>.
    *   Proceed to deactivate all relevant protection features:
        *   "Real-time protection" <a class="yt-timestamp" data-t="00:02:09">[00:02:09]</a>
        *   "Cloud-based protection" <a class="yt-timestamp" data-t="00:02:26">[00:02:26]</a>
        *   "Sending sample takes tico protection" <a class="yt-timestamp" data-t="00:02:27">[00:02:27]</a>
        *   "Alterations" <a class="yt-timestamp" data-t="00:02:32">[00:02:32]</a>

### Re-enabling Windows Defender
The process to [[how_to_activate_windows_defender_step_by_step | activate Windows Defender]] is the exact reverse of these steps <a class="yt-timestamp" data-t="00:02:02">[00:02:02]</a>.

## Verifying Disablement

After completing these steps, [[how_to_activate_windows_defender_step_by_step | Windows Defender]] will be deactivated <a class="yt-timestamp" data-t="00:02:35">[00:02:35]</a>. You can verify this by returning to the start section within [[navigating_windows_security_options | Windows Security]], where you will see a red cross indicating that antivirus protection is totally disabled <a class="yt-timestamp" data-t="00:02:43">[00:02:43]</a>.
