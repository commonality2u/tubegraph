This guide explains how to manage [[windows_defender_configuration_management | Windows Defender]] on your computer, a process that can be useful for [[importance_of_disabling_windows_defender_for_certain_downloads | downloading updates]] and installing products that [[windows_defender_configuration_management | Windows Defender]] might otherwise block <a class="yt-timestamp" data-t="00:00:10">[00:00:10]</a>.

## Accessing [[windows_defender_configuration_management | Windows Security]]

To begin, you need to access the [[windows_defender_configuration_management | Windows Security]] settings:
1.  Navigate to the [[how_to_deactivate_windows_defender_in_windows_10 | Windows 10]] taskbar <a class="yt-timestamp" data-t="00:00:39">[00:00:39]</a>.
2.  In the search bar (bottom-left), type "defend" <a class="yt-timestamp" data-t="00:00:43">[00:00:43]</a>.
3.  Select the "[[windows_defender_configuration_management | Windows Security]]" option <a class="yt-timestamp" data-t="00:00:51">[00:00:51]</a>.
4.  Once in [[windows_defender_configuration_management | Windows Security]], choose "[[antivirus_threat_protection_options | Antivirus protection against threats]]" <a class="yt-timestamp" data-t="00:00:54">[00:00:54]</a>.

Alternatively, you can access [[windows_defender_configuration_management | Windows Defender]] directly from the menu on the left side of the [[windows_defender_configuration_management | Windows Security]] window <a class="yt-timestamp" data-t="00:01:31">[00:01:31]</a>.

## Managing [[windows_defender_configuration_management | Defender]] Settings

Within the "[[antivirus_threat_protection_options | Antivirus protection against threats]]" section, you will find options to manage your protection <a class="yt-timestamp" data-t="00:01:42">[00:01:42]</a>.

### How to Deactivate [[windows_defender_configuration_management | Windows Defender]]

If your goal is to [[importance_of_disabling_windows_defender_for_certain_downloads | deactivate Windows Defender]] (e.g., for specific installations or updates), follow these steps:
1.  Locate "Manage configuration" within the [[antivirus_threat_protection_options | Antivirus protection against threats]] section <a class="yt-timestamp" data-t="00:01:42">[00:01:42]</a>.
2.  Toggle the blue bar from "on" to "deactivated" for "Real-time protection" <a class="yt-timestamp" data-t="00:01:53">[00:01:53]</a>, <a class="yt-timestamp" data-t="00:02:09">[00:02:09]</a>.
3.  A confirmation prompt from [[windows_defender_configuration_management | Windows Security]] will appear, asking if you want to allow changes to your device; confirm by selecting "Yes" <a class="yt-timestamp" data-t="00:01:56">[00:01:56]</a>, <a class="yt-timestamp" data-t="00:02:14">[00:02:14]</a>.
4.  Repeat this process for other [[windows_defender_configuration_management | Windows Defender]] components, such as:
    *   "Cloud-based protection" <a class="yt-timestamp" data-t="00:02:26">[00:02:26]</a>
    *   "Automatic sample submission" <a class="yt-timestamp" data-t="00:02:27">[00:02:27]</a>
    *   "Tamper protection" <a class="yt-timestamp" data-t="00:02:30">[00:02:30]</a>
5.  Once all these options are deactivated, [[windows_defender_configuration_management | Windows Defender]] will show a red cross, indicating it is completely disabled <a class="yt-timestamp" data-t="00:02:47">[00:02:47]</a>.

### How to Activate [[windows_defender_configuration_management | Windows Defender]]

To activate [[windows_defender_configuration_management | Windows Defender]], the process is the exact reverse of deactivation <a class="yt-timestamp" data-t="00:02:01">[00:02:01]</a>. Simply toggle the previously deactivated options back to "on." This will reinstate the full protection features of [[windows_defender_configuration_management | Windows Defender]].
