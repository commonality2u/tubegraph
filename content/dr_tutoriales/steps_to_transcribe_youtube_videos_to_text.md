Transcribing [[how_to_transcribe_audio_to_text | YouTube videos to text]] can be a very easy process <a class="yt-timestamp" data-t="00:00:00">[00:00:00]</a>. One effective method involves using a [[google_chrome_extension_for_transcribing_youtube_videos | Google Chrome extension]] that leverages [[using_artificial_intelligence_for_video_to_text_conversion | artificial intelligence]] <a class="yt-timestamp" data-t="00:00:11">[00:00:11]</a>.

## Method: Using the "Glass" Google Chrome Extension

This guide focuses on performing transcription within [[using_google_chrome_for_online_transcription | Google Chrome]] using a specific extension called "Glass" <a class="yt-timestamp" data-t="00:00:05">[00:00:05]</a>, <a class="yt-timestamp" data-t="00:00:14">[00:00:14]</a>. The "Glass" extension utilizes Chat GPT to convert YouTube video content into text <a class="yt-timestamp" data-t="00:00:17">[00:00:17]</a>.

### Installation

1.  **Install the Extension**: You will need to install the "Glass" extension on your Chrome browser <a class="yt-timestamp" data-t="00:00:34">[00:00:34]</a>.
    *   While the primary focus is on Chrome, the "Glass" extension can also be installed on Safari, or its parent code can be used from Python <a class="yt-timestamp" data-t="00:00:23">[00:00:23]</a>, <a class="yt-timestamp" data-t="00:00:29">[00:00:29]</a>.

### Usage Steps

1.  **Open YouTube**: After installing the extension, open a new tab and navigate to the YouTube platform <a class="yt-timestamp" data-t="00:00:40">[00:00:40]</a>, <a class="yt-timestamp" data-t="00:00:43">[00:00:43]</a>.
2.  **Select a Video**: Choose any YouTube video you wish to transcribe <a class="yt-timestamp" data-t="00:00:46">[00:00:46]</a>.
3.  **Locate Transcript Window**: Once the "Glass" extension is installed, a small window labeled "transcript and summery" (meaning "Transcript and abstract") will appear on the video screen <a class="yt-timestamp" data-t="00:00:54">[00:00:54]</a>, <a class="yt-timestamp" data-t="00:00:57">[00:00:57]</a>, <a class="yt-timestamp" data-t="00:01:00">[00:01:00]</a>.
4.  **View Transcript**: Click on the "transcript and summery" window <a class="yt-timestamp" data-t="00:01:02">[00:01:02]</a>.
    *   You will then see the transcript, presented as paragraphs that correspond to specific time sequences (seconds) of the video, showing what the speaker is saying <a class="yt-timestamp" data-t="00:01:07">[00:01:07]</a>, <a class="yt-timestamp" data-t="00:01:10">[00:01:10]</a>, <a class="yt-timestamp" data-t="00:01:13">[00:01:13]</a>. This transcript covers the entire video <a class="yt-timestamp" data-t="00:01:15">[00:01:15]</a>.
5.  **Copy the Transcript**:
    *   **To Word**: To copy the transcript to a document like Word, simply select the desired text within the transcript window and then copy it <a class="yt-timestamp" data-t="00:01:27">[00:01:27]</a>, <a class="yt-timestamp" data-t="00:01:29">[00:01:29]</a>.
    *   **As Text**: Alternatively, there is a direct "Copy here as text" option available within the window <a class="yt-timestamp" data-t="00:01:34">[00:01:34]</a>, <a class="yt-timestamp" data-t="00:01:37">[00:01:37]</a>. Once copied, the text can be pasted wherever needed <a class="yt-timestamp" data-t="00:01:40">[00:01:40]</a>, <a class="yt-timestamp" data-t="00:01:42">[00:01:42]</a>.
