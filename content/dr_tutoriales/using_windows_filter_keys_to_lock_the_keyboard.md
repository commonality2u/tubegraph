This tutorial outlines how to block and unblock your computer's keyboard using Windows filter keys, a feature that can sometimes be activated accidentally <a class="yt-timestamp" data-t="00:00:05">[00:00:05]</a>. This method applies to Windows 7, 8, and 10 <a class="yt-timestamp" data-t="00:00:15">[00:00:15]</a>.

## How to Lock Your Keyboard

The keyboard can be blocked unintentionally through a specific key combination, which activates "Windows filters" <a class="yt-timestamp" data-t="00:01:20">[00:01:20]</a>.

1.  **Locate the Shift Key**: Identify the Shift key, typically located below the Enter key <a class="yt-timestamp" data-t="00:01:08">[00:01:08]</a>.
2.  **Press and Hold**: Press and hold this Shift key for more than eight seconds <a class="yt-timestamp" data-t="00:01:20">[00:01:20]</a>.
3.  **Filter Keys Prompt**: Your computer will chime and display a prompt asking if you want to turn on [[disabling_filter_keys_to_restore_keyboard_functionality | Filter keys]] <a class="yt-timestamp" data-t="00:01:42">[00:01:42]</a>.
4.  **Confirm Activation**: By default, the "Yes" option appears. Many users unintentionally click "Yes" <a class="yt-timestamp" data-t="00:01:53">[00:01:53]</a>. If you click "Yes", your keyboard will stop functioning <a class="yt-timestamp" data-t="00:01:58">[00:01:58]</a>. This feature is designed to block the keyboard <a class="yt-timestamp" data-t="00:01:27">[00:01:27]</a>.

## How to Unlock Your Keyboard

If your keyboard has been locked by accidentally activating the filter keys feature, you can easily [[how_to_unlock_the_keyboard_in_windows_10 | unlock it]] <a class="yt-timestamp" data-t="00:00:21">[00:00:21]</a>.

1.  **Press and Hold Shift Again**: Press and hold the same Shift key (below the Enter key) for eight seconds once more <a class="yt-timestamp" data-t="00:02:11">[00:02:11]</a>.
2.  **Keyboard Reactivation**: The computer will chime again <a class="yt-timestamp" data-t="00:02:26">[00:02:26]</a>, and your keyboard will be reactivated, allowing you to type freely <a class="yt-timestamp" data-t="00:02:29">[00:02:29]</a>.

> [!NOTE]
> This method of blocking and unblocking the keyboard is specifically related to the Windows Filter Keys feature, which is designed to ignore brief or repeated keystrokes <a class="yt-timestamp" data-t="00:01:20">[00:01:20]</a>.
