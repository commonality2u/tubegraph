## Why Compress Video Files?

[[video compression | Video compression]] is useful for several reasons:
*   **Easier Transfer** Transferring large video files via platforms like WhatsApp or email can be challenging due to size limitations <a class="yt-timestamp" data-t="00:00:22">[00:00:22]</a>, <a class="yt-timestamp" data-t="00:00:25">[00:00:25]</a>.
*   **Saving Storage** Reducing video file size helps to free up valuable memory on your computer <a class="yt-timestamp" data-t="00:00:33">[00:00:33]</a>.

The goal of this method is [[maintaining video quality while compressing | maintaining video quality while compressing]] and retaining audio <a class="yt-timestamp" data-t="00:00:15">[00:00:15]</a>, <a class="yt-timestamp" data-t="00:00:19">[00:00:19]</a>, <a class="yt-timestamp" data-t="00:02:06">[00:02:06]</a>, <a class="yt-timestamp" data-t="00:02:56">[00:02:56]</a>.

## Step-by-Step Video Conversion Process

To demonstrate the process, an example video, a chapter of the "Dark" series, initially 1.93 GB in HD (1980 resolution) with multiple audio tracks, is used <a class="yt-timestamp" data-t="00:00:57">[00:00:57]</a>, <a class="yt-timestamp" data-t="00:01:03">[00:01:03]</a>, <a class="yt-timestamp" data-t="00:01:06">[00:01:06]</a>, <a class="yt-timestamp" data-t="00:01:10">[00:01:10]</a>, <a class="yt-timestamp" data-t="00:01:14">[00:01:14]</a>, <a class="yt-timestamp" data-t="00:01:16">[00:01:16]</a>.

1.  **Open VLC** Open the VLC Media Player <a class="yt-timestamp" data-t="00:01:22">[00:01:22]</a>, <a class="yt-timestamp" data-t="00:01:24">[00:01:24]</a>.
2.  **Access Conversion Menu**
    *   Go to the "Media" menu <a class="yt-timestamp" data-t="00:01:30">[00:01:30]</a>, <a class="yt-timestamp" data-t="00:01:33">[00:01:33]</a>.
    *   Select "Convert / Save" <a class="yt-timestamp" data-t="00:01:36">[00:01:36]</a>.
3.  **Add Video File**

    *   Click "Open" <a class="yt-timestamp" data-t="00:01:48">[00:01:48]</a>.

### Configuring Compression Settings

This step is crucial for [[retaining video quality | retaining video quality]] during compression <a class="yt-timestamp" data-t="00:01:57">[00:01:57]</a>, <a class="yt-timestamp" data-t="00:02:02">[00:02:02]</a>.

1.  **Select Profile** Choose the "H.264 + MP3" profile <a class="yt-timestamp" data-t="00:02:06">[00:02:06]</a>, <a class="yt-timestamp" data-t="00:02:10">[00:02:10]</a>.
2.  **Access Profile Settings** Click the "tools" icon next to the profile dropdown <a class="yt-timestamp" data-t="00:02:14">[00:02:14]</a>.
3.  **Configure Settings Tabs**
    *   **Encapsulation:** Set to "MP4/MV" <a class="yt-timestamp" data-t="00:02:18">[00:02:18]</a>.
    *   **Video Codec:**
        *   Codec: H.264 <a class="yt-timestamp" data-t="00:02:23">[00:02:23]</a>.
        *   Bitrate: Keep as "unused" or "same as the source" <a class="yt-timestamp" data-t="00:02:29">[00:02:29]</a>.
    *   **Resolution:** Set "Scale" to "1" <a class="yt-timestamp" data-t="00:02:33">[00:02:33]</a>, <a class="yt-timestamp" data-t="00:02:39">[00:02:39]</a>. This is very important <a class="yt-timestamp" data-t="00:02:33">[00:02:33]</a>. Filters should not be touched <a class="yt-timestamp" data-t="00:02:43">[00:02:43]</a>.
    *   **Audio:** It is recommended to keep "original audio" to avoid losing the video's audio <a class="yt-timestamp" data-t="00:02:48">[00:02:48]</a>, <a class="yt-timestamp" data-t="00:02:52">[00:02:52]</a>, <a class="yt-timestamp" data-t="00:02:56">[00:02:56]</a>.
4.  **Save Settings** Click "Save" to apply the configurations <a class="yt-timestamp" data-t="00:02:59">[00:02:59]</a>.

### Exporting and Monitoring

1.  **Choose Destination**
    *   Select the destination folder for the compressed file <a class="yt-timestamp" data-t="00:03:02">[00:03:02]</a>, <a class="yt-timestamp" data-t="00:03:07">[00:03:07]</a>.

2.  **Start Conversion** Click "Start" to begin the compression process <a class="yt-timestamp" data-t="00:03:16">[00:03:16]</a>.
