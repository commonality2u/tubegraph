PowerPoint allows users to [[adding_online_videos_to_powerpoint_presentations | add videos to presentations]] from both online sources and local computer files, which offers flexibility depending on playback requirements <a class="yt-timestamp" data-t="00:00:04">[00:00:04]</a>. Integrating videos can make presentations more dynamic and useful <a class="yt-timestamp" data-t="00:00:12">[00:00:12]</a>.

## How to Insert Videos in PowerPoint

To insert a video into a PowerPoint presentation:
1.  Open your PowerPoint presentation <a class="yt-timestamp" data-t="00:00:17">[00:00:17]</a>.
2.  Navigate to the desired slide where you want to add the video, or create a new slide <a class="yt-timestamp" data-t="00:00:30">[00:00:30]</a>.
3.  Go to the "Insert" tab <a class="yt-timestamp" data-t="00:00:39">[00:00:39]</a>.
4.  In the "Insert" tab, locate the "Video" section <a class="yt-timestamp" data-t="00:00:42">[00:00:42]</a>.
5.  PowerPoint offers two primary options: "Online Video" and "Video on My PC" <a class="yt-timestamp" data-t="00:00:47">[00:00:47]</a>.

## Inserting Online Videos

This method is suitable for [[adding_online_videos_to_powerpoint_presentations | adding online videos to PowerPoint presentations]] that are hosted on platforms like YouTube <a class="yt-timestamp" data-t="00:00:58">[00:00:58]</a>.

### Steps
1.  Select the "Online Video" option <a class="yt-timestamp" data-t="00:01:13">[00:01:13]</a>.
2.  Paste the URL (link) of the online video into the provided field <a class="yt-timestamp" data-t="00:01:14">[00:01:14]</a>.
3.  Click "Insert" <a class="yt-timestamp" data-t="00:01:18">[00:01:18]</a>.

### Requirements for Playback
*   An active internet connection is mandatory for the video to play during the presentation <a class="yt-timestamp" data-t="00:01:27">[00:01:27]</a>. If there is no Wi-Fi, the YouTube video will not be visible <a class="yt-timestamp" data-t="00:01:19">[00:01:19]</a>.

### Example
*   A video from YouTube can be inserted using its link <a class="yt-timestamp" data-t="00:01:00">[00:01:00]</a>.

## Inserting Videos from a Computer (Offline)

This method involves [[inserting_videos_from_a_computer_into_powerpoint | embedding a video file directly from your computer]] into the presentation.

### Steps
1.  Select the "Video on My PC" option <a class="yt-timestamp" data-t="00:01:48">[00:01:48]</a>.
2.  Browse your computer files and choose the desired video <a class="yt-timestamp" data-t="00:01:50">[00:01:50]</a>.
3.  Click "Insert" <a class="yt-timestamp" data-t="00:02:00">[00:02:00]</a>.

### Requirements for Playback
*   No internet connection is needed for playback <a class="yt-timestamp" data-t="00:02:13">[00:02:13]</a>. The video is embedded within the PowerPoint file.

## Comparison of Online vs. Offline Video Insertion

| Feature             | Online Video (e.g., YouTube)                               | Video from Computer (Offline)                             |
| :------------------ | :--------------------------------------------------------- | :-------------------------------------------------------- |
| **Source**          | External online platforms                                | Local files on your computer                              |
| **Playback**        | Requires active internet connection <a class="yt-timestamp" data-t="00:01:27">[00:01:27]</a>             | Can be viewed at any time without internet <a class="yt-timestamp" data-t="00:02:13">[00:02:13]</a> |
| **File Size**       | Does not significantly increase presentation file size   | Increases presentation file size (as video is embedded)   |
| **Availability**    | Dependent on online source and internet connectivity     | Always available as long as the file is accessible        |
| **Editing Options** | Limited within PowerPoint                                | More options within PowerPoint for playback settings <a class="yt-timestamp" data-t="00:02:07">[00:02:07]</a> |
