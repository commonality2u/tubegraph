This guide outlines the steps to activate [[managing_antivirus_protection_and_firewall_settings | Windows Defender]] on your computer. Activating [[managing_antivirus_protection_and_firewall_settings | Windows Defender]] is beneficial for downloading updates and installing products, as it prevents [[managing_antivirus_protection_and_firewall_settings | Windows Defender]] from blocking these processes <a class="yt-timestamp" data-t="00:10:00">[00:10:00]</a>.

## Accessing Windows Security Settings

1.  Navigate to your computer's taskbar <a class="yt-timestamp" data-t="00:39:07">[00:39:07]</a>.
2.  In the search bar, type "defend" <a class="yt-timestamp" data-t="00:46:00">[00:46:00]</a>.
3.  Select the "Windows Security" option, specifically the "Antivirus protection against threats" section <a class="yt-timestamp" data-t="00:51:00">[00:51:00]</a>.
    *   Alternatively, you can access [[understanding_windows_defender_settings | Windows Defender]] from the menu on the left <a class="yt-timestamp" data-t="01:31:00">[01:31:00]</a>.
4.  Once in the "Antivirus protection against threats" section, you can go to the "Home" section to view all security options <a class="yt-timestamp" data-t="01:00:00">[01:00:00]</a>.
5.  To manage [[managing_antivirus_protection_and_firewall_settings | Windows Defender]], select the "Antivirus protection against threats" option <a class="yt-timestamp" data-t="01:10:00">[01:10:00]</a>.

## Activating Windows Defender

1.  Within the "Antivirus protection against threats" settings, navigate to "managing the configuration" <a class="yt-timestamp" data-t="01:42:00">[01:42:00]</a>.
2.  To activate [[managing_antivirus_protection_and_firewall_settings | Windows Defender]], change the blue bar from "deactivated" to "on" <a class="yt-timestamp" data-t="01:53:00">[01:53:00]</a>.
3.  When prompted for confirmation by Windows Security, allow the application to make changes to your device <a class="yt-timestamp" data-t="01:57:00">[01:57:00]</a>.
4.  Proceed to activate the following components:
    *   Real-time protection <a class="yt-timestamp" data-t="02:09:00">[02:09:00]</a>
    *   Cloud-based protection <a class="yt-timestamp" data-t="02:26:00">[02:26:00]</a>
    *   Automatic sample submission <a class="yt-timestamp" data-t="02:27:00">[02:27:00]</a>
    *   Tamper protection <a class="yt-timestamp" data-t="02:30:00">[02:30:00]</a>

Once these steps are completed, [[managing_antivirus_protection_and_firewall_settings | Windows Defender]] will be activated <a class="yt-timestamp" data-t="02:35:00">[02:35:00]</a>. You can verify its active status by returning to the start section; the red cross should disappear, indicating that protection is enabled <a class="yt-timestamp" data-t="02:47:00">[02:47:00]</a>.
