## Accessing Search Engine Settings

To change your default search engine in [[google_chrome_advanced_settings_navigation | Google Chrome]]:
1.  Navigate to the top right corner of the browser window <a class="yt-timestamp" data-t="00:01:02">[00:01:02]</a>.
2.  Click on the "settings" or "configuration" option <a class="yt-timestamp" data-t="00:01:08">[00:01:08]</a>, <a class="yt-timestamp" data-t="00:01:10">[00:01:10]</a>. This will open a new tab <a class="yt-timestamp" data-t="00:01:14">[00:01:14]</a>.
3.  Within the settings tab, locate and go to the "Search engine" section <a class="yt-timestamp" data-t="00:01:17">[00:01:17]</a>. This section manages the search engines you use, such as Yahoo or Google <a class="yt-timestamp" data-t="00:01:22">[00:01:22]</a>, <a class="yt-timestamp" data-t="00:01:24">[00:01:24]</a>.

## Managing Search Engines

Under the "Search engine" section, you will find an option to "manage Search Engines" <a class="yt-timestamp" data-t="00:01:44">[00:01:44]</a>, <a class="yt-timestamp" data-t="00:01:47">[00:01:47]</a>. Clicking this will display all currently installed search engines <a class="yt-timestamp" data-t="00:01:47">[00:01:47]</a>. Common options might include Google, Bing, Yahoo, or Ecosia <a class="yt-timestamp" data-t="00:01:50">[00:01:50]</a>, <a class="yt-timestamp" data-t="00:01:54">[00:01:54]</a>, <a class="yt-timestamp" data-t="00:02:00">[00:02:00]</a>.

### Editing or Adding a Search Engine
*   **Editing**: If you wish to edit an existing search engine (e.g., Bing), you can select it to change its name or associated link <a class="yt-timestamp" data-t="00:02:06">[00:02:06]</a>, <a class="yt-timestamp" data-t="00:02:10">[00:02:10]</a>, <a class="yt-timestamp" data-t="00:02:14">[00:02:14]</a>.
*   **Adding Google**: If Google is not listed as an option, you can add it by inputting the following data:
    *   **Name**: Google <a class="yt-timestamp" data-t="00:02:18">[00:02:18]</a>.
    *   **Shortcut**: Google.com <a class="yt-timestamp" data-t="00:02:22">[00:02:22]</a>.
        *   It is crucial to use `.com` for the URL <a class="yt-timestamp" data-t="00:02:22">[00:02:22]</a>. Using country-specific domains like `.es` (for Spain) or `.uk` (for England) might cause [[troubleshooting_search_engine_issues_in_chrome | problems]] if you are not in that specific country <a class="yt-timestamp" data-t="00:02:26">[00:02:26]</a>, <a class="yt-timestamp" data-t="00:02:28">[00:02:28]</a>, <a class="yt-timestamp" data-t="00:02:30">[00:02:30]</a>, <a class="yt-timestamp" data-t="00:02:33">[00:02:33]</a>.
    *   **URL**: The full link to Google <a class="yt-timestamp" data-t="00:02:14">[00:02:14]</a>.
    Once entered, save the changes <a class="yt-timestamp" data-t="00:02:36">[00:02:36]</a>, <a class="yt-timestamp" data-t="00:02:39">[00:02:39]</a>. There is no need to save anything else <a class="yt-timestamp" data-t="00:02:49">[00:02:49]</a>.
