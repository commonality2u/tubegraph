Mediafire is a free cloud platform that allows users to [[sharing_files_using_mediafire | upload files to the cloud]] and share them with others <a class="yt-timestamp" data-t="00:00:11">[00:00:11]</a>. This guide will walk you through the process of uploading files and managing them on Mediafire.

## Accessing Mediafire <a class="yt-timestamp" data-t="00:00:28">[00:00:28]</a>

To begin, open your preferred search engine, such as Google Chrome, which is described as reliable and fast <a class="yt-timestamp" data-t="00:00:31">[00:00:31]</a>. Search for "Mediafire" <a class="yt-timestamp" data-t="00:00:38">[00:00:38]</a> and select the first search result <a class="yt-timestamp" data-t="00:00:44">[00:00:44]</a>.

## [[creating_a_mediafire_account | Creating a Mediafire Account]] <a class="yt-timestamp" data-t="00:00:49">[00:00:49]</a>

If you already have a Mediafire account, you can proceed to the next step to upload files <a class="yt-timestamp" data-t="00:00:56">[00:00:56]</a>. If not, follow these steps to [[creating_a_mediafire_account | create one]]:

1.  Click on "Sign Up" located at the top right of the page <a class="yt-timestamp" data-t="00:01:02">[00:01:02]</a>.
2.  You will be presented with different [[exploring_mediafire_account_plans_and_storage_options | account plans]] <a class="yt-timestamp" data-t="00:01:07">[00:01:07]</a>. For this tutorial, a basic, free account will be created <a class="yt-timestamp" data-t="00:01:12">[00:01:12]</a>.
3.  The basic free account offers a maximum of 10 gigabytes (GB) of cloud storage <a class="yt-timestamp" data-t="00:01:13">[00:01:13]</a>. It is mentioned that a guide might be available in the description to expand storage up to 50 GB <a class="yt-timestamp" data-t="00:01:24">[00:01:24]</a>.
4.  Select "Get Basic" <a class="yt-timestamp" data-t="00:01:31">[00:01:31]</a>.
5.  Fill in your personal information, such as your name, to complete the account creation <a class="yt-timestamp" data-t="00:01:38">[00:01:38]</a>.

## [[how_to_manage_files_and_folders_on_mediafire | Managing Files and Folders]] <a class="yt-timestamp" data-t="00:01:57">[00:01:57]</a>

Once your account is created, you will see your Mediafire platform dashboard <a class="yt-timestamp" data-t="00:01:53">[00:01:53]</a>.

*   The left panel displays the folders you have created <a class="yt-timestamp" data-t="00:01:59">[00:01:59]</a>.
*   The main panel (often labeled "Upload") allows you to upload files, filter them, and perform other actions <a class="yt-timestamp" data-t="00:02:04">[00:02:04]</a>.

### Creating Folders <a class="yt-timestamp" data-t="00:02:16">[00:02:16]</a>

To organize your files, you can create new folders:

1.  Select the option to "create folders" <a class="yt-timestamp" data-t="00:02:17">[00:02:17]</a>.
2.  Provide a name for your new folder, for example, "youtube tutorial" <a class="yt-timestamp" data-t="00:02:19">[00:02:19]</a>.
3.  Once created, the folder will appear with "zero folders inside and zero files inside" <a class="yt-timestamp" data-t="00:02:30">[00:02:30]</a>.

## Uploading Files <a class="yt-timestamp" data-t="00:02:38">[00:02:38]</a>

To upload files:

1.  Enter the folder where you wish to upload the file <a class="yt-timestamp" data-t="00:02:38">[00:02:38]</a>.
2.  Click the "Upload" button <a class="yt-timestamp" data-t="00:02:41">[00:02:41]</a>.
3.  A dialog box will appear to select your file <a class="yt-timestamp" data-t="00:02:52">[00:02:52]</a>. For example, a PNG thumbnail file is chosen <a class="yt-timestamp" data-t="00:02:43">[00:02:43]</a>.
4.  Click "Begin Upload" <a class="yt-timestamp" data-t="00:03:04">[00:03:04]</a>. The file will be queued and then uploaded <a class="yt-timestamp" data-t="00:03:07">[00:03:07]</a>. Small files upload very quickly <a class="yt-timestamp" data-t="00:03:11">[00:03:11]</a>.
5.  Once uploaded, you can close the upload dialog, and the image will appear in your chosen folder <a class="yt-timestamp" data-t="00:03:14">[00:03:14]</a>.

### Uploading Folders <a class="yt-timestamp" data-t="00:03:28">[00:03:28]</a>

The "Upload" option also provides the ability to upload entire folders <a class="yt-timestamp" data-t="00:03:22">[00:03:22]</a>. This is useful for uploading multiple files contained within a single folder simultaneously <a class="yt-timestamp" data-t="00:03:31">[00:03:31]</a>.

### Uploading Websites <a class="yt-timestamp" data-t="00:03:41">[00:03:41]</a>

The option to upload websites is exclusively available for premium users <a class="yt-timestamp" data-t="00:03:46">[00:03:46]</a>. As a basic account holder, this functionality is not accessible <a class="yt-timestamp" data-t="00:03:48">[00:03:48]</a>.

## [[sharing_files_using_mediafire | Sharing Files]] <a class="yt-timestamp" data-t="00:04:10">[00:04:10]</a>

After uploading a file, you can easily share it:

1.  Locate the file you want to share <a class="yt-timestamp" data-t="00:04:02">[00:04:02]</a>.
2.  Click on the three dots next to the file <a class="yt-timestamp" data-t="00:04:12">[00:04:12]</a>.
3.  Select the "Copy link" option <a class="yt-timestamp" data-t="00:04:14">[00:04:14]</a>. The link will be copied to your clipboard <a class="yt-timestamp" data-t="00:04:19">[00:04:19]</a>.
4.  Anyone with this link can download the file from Mediafire <a class="yt-timestamp" data-t="00:04:21">[00:04:21]</a>. This is a very useful way to [[sharing_files_using_mediafire | share files]] with colleagues <a class="yt-timestamp" data-t="00:04:38">[00:04:38]</a>.

### Advanced Sharing Options <a class="yt-timestamp" data-t="00:04:41">[00:04:41]</a>

Alternatively, by selecting "View" from the three-dot menu, you can access a different menu with more options <a class="yt-timestamp" data-t="00:04:41">[00:04:41]</a>. This menu allows you to modify who shares the link and provides direct download options <a class="yt-timestamp" data-t="00:04:47">[00:04:47]</a>.
