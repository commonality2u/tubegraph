Splitting PDF documents allows users to [[extracting_specific_pdf_pages | extract specific pages]] or divide a single PDF into multiple smaller files <a class="yt-timestamp" data-t="00:00:00">[00:00:00]</a>. This article outlines two straightforward methods for achieving this <a class="yt-timestamp" data-t="00:00:05">[00:00:05]</a>.

## Method 1: [[separating_pdf_pages_using_google_chrome | Separating PDF Pages Using Google Chrome]]

This method leverages Google Chrome's built-in PDF viewing and printing capabilities, requiring no external software <a class="yt-timestamp" data-t="00:01:43">[00:01:43]</a>.

1.  **Open PDF with Google Chrome**: Instead of opening the PDF with Adobe Acrobat, right-click the PDF file on your computer and select "Open with" > "Google Chrome" <a class="yt-timestamp" data-t="00:00:33">[00:00:33]</a>. The PDF will open in a Chrome tab, displaying all its pages <a class="yt-timestamp" data-t="00:00:39">[00:00:39]</a>.
2.  **Access Print Dialog**: Locate and click the printer icon button, typically found at the top right of the browser window <a class="yt-timestamp" data-t="00:00:49">[00:00:49]</a>.
3.  **Select Custom Pages**: In the print dialog, under the "Pages" option, change the selection from "All" to "Custom" <a class="yt-timestamp" data-t="00:00:55">[00:00:55]</a>.
4.  **Specify Desired Pages**: Enter the page numbers you wish to extract, separated by commas. For example, to extract pages 3 and 4, you would type "3,4" <a class="yt-timestamp" data-t="00:01:06">[00:01:06]</a>. Only the specified pages will be downloaded <a class="yt-timestamp" data-t="00:01:13">[00:01:13]</a>.
5.  **Print and Save**: Click the "Print" button <a class="yt-timestamp" data-t="00:01:13">[00:01:13]</a>. You will then be prompted to choose a location to save the new PDF file <a class="yt-timestamp" data-t="00:01:15">[00:01:15]</a>. Give it a memorable name, such as "cropped PDF," and click "Save" <a class="yt-timestamp" data-t="00:01:27">[00:01:27]</a>. The newly created PDF will contain only the selected pages <a class="yt-timestamp" data-t="00:01:37">[00:01:37]</a>.

## Method 2: Using I Love PDF Online Tool

[[editing_pdfs_online_for_free | Online PDF tools]] like I Love PDF offer a simple and effective way to split documents.

1.  **Access I Love PDF**: Open your web browser and search for "I love PDF" <a class="yt-timestamp" data-t="00:01:50">[00:01:50]</a>. This is a website that provides various PDF manipulation tools <a class="yt-timestamp" data-t="00:01:53">[00:01:53]</a>.
2.  **Select "Split PDF"**: On the I Love PDF website, choose the "Split PDF" option <a class="yt-timestamp" data-t="00:01:57">[00:01:57]</a>.
3.  **Upload PDF**: Click to select the PDF file you wish to split <a class="yt-timestamp" data-t="00:02:01">[00:02:01]</a>. For example, a PDF with 28 pages was used <a class="yt-timestamp" data-t="00:02:08">[00:02:08]</a>.
4.  **Choose Split Option**: Select "Extract Pages" <a class="yt-timestamp" data-t="00:02:11">[00:02:11]</a>. This will display all the pages, allowing you to select specific ones if desired <a class="yt-timestamp" data-t="00:02:17">[00:02:17]</a>.
5.  **Split and Download**: Click the "Split PDF" button <a class="yt-timestamp" data-t="00:02:22">[00:02:22]</a>. The tool will then divide the PDF into separate pages <a class="yt-timestamp" data-t="00:02:25">[00:02:25]</a>. Finally, click "Download" <a class="yt-timestamp" data-t="00:02:27">[00:02:27]</a>. The separated PDF pages will often be downloaded as a compressed file (e.g., WinRar archive), containing individual PDF files for each page <a class="yt-timestamp" data-t="00:02:30">[00:02:30]</a>. You can then open and view each page separately <a class="yt-timestamp" data-t="00:02:34">[00:02:34]</a>.

These two methods provide efficient ways to split PDF documents, whether you prefer using a browser's built-in features or a dedicated online tool.
