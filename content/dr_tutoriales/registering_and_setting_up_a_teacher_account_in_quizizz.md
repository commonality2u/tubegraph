Quizizz is an online tool that allows users to create and administer didactic tests for students <a class="yt-timestamp" data-t="00:00:07">[00:00:07]</a>.

## Accessing and Registering

To begin using Quizizz, the first step is to access its website <a class="yt-timestamp" data-t="00:00:43">[00:00:43]</a>. The website can be found using a search engine, and a direct link may be provided in the video's first comment <a class="yt-timestamp" data-t="00:00:49">[00:00:49]</a>.

Once on the page, select the "register" option <a class="yt-timestamp" data-t="00:00:54">[00:00:54]</a>. Users can register using a Google email account or other email providers <a class="yt-timestamp" data-t="00:01:01">[00:01:01]</a>, <a class="yt-timestamp" data-t="00:01:04">[00:01:04]</a>.

> [!info] Teacher Registration
> It is crucial to register as a "teacher" because this role is responsible for designing and sharing the tests with students <a class="yt-timestamp" data-t="00:01:16">[00:01:16]</a>, <a class="yt-timestamp" data-t="00:01:18">[00:01:18]</a>, <a class="yt-timestamp" data-t="00:01:27">[00:01:27]</a>. While parents and companies can also participate, the teacher account is for test creation <a class="yt-timestamp" data-t="00:01:21">[00:01:21]</a>, <a class="yt-timestamp" data-t="00:01:24">[00:01:24]</a>.

*   **Profile**: Your user profile is displayed at the top <a class="yt-timestamp" data-t="00:01:40">[00:01:40]</a>.
*   **Create**: This option allows you to [[how_to_use_quizizz_for_creating_online_tests | create a new test]] <a class="yt-timestamp" data-t="00:01:43">[00:01:43]</a>, <a class="yt-timestamp" data-t="00:01:45">[00:01:45]</a>.
*   **Search**: You can search for existing tests created by others based on themes like mathematics, English, or fractions <a class="yt-timestamp" data-t="00:01:45">[00:01:45]</a>, <a class="yt-timestamp" data-t="00:01:47">[00:01:47]</a>, <a class="yt-timestamp" data-t="00:01:50">[00:01:50]</a>.
*   **My Questionnaires**: This section displays tests that you have saved <a class="yt-timestamp" data-t="00:02:10">[00:02:10]</a>, <a class="yt-timestamp" data-t="00:02:12">[00:02:12]</a>.
*   **Reports**: Here, you can view [[student_engagement_and_performance_tracking_on_quizizz | reports on students]] who have completed your quizzes, helping to track grades and identify areas for reinforcement <a class="yt-timestamp" data-t="00:02:15">[00:02:15]</a>, <a class="yt-timestamp" data-t="00:02:18">[00:02:18]</a>, <a class="yt-timestamp" data-t="00:02:21">[00:02:21]</a>, <a class="yt-timestamp" data-t="00:02:28">[00:02:28]</a>.
*   **Classes**: This feature allows you to manage groups of students, who can connect via Google Classrooms or a direct link to receive test notifications <a class="yt-timestamp" data-t="00:02:35">[00:02:35]</a>, <a class="yt-timestamp" data-t="00:02:40">[00:02:40]</a>, <a class="yt-timestamp" data-t="00:02:43">[00:02:43]</a>, <a class="yt-timestamp" data-t="00:02:48">[00:02:48]</a>.
*   **Collections**: This section contains [[creating_and_organizing_quizzes_and_collections_in_quizizz | collections of tests]] you have created or wish to organize <a class="yt-timestamp" data-t="00:03:00">[00:03:00]</a>, <a class="yt-timestamp" data-t="00:03:03">[00:03:03]</a>. You can create new collections (e.g., "Math Fractions") and save existing quizzes into them <a class="yt-timestamp" data-t="00:03:22">[00:03:22]</a>, <a class="yt-timestamp" data-t="00:03:49">[00:03:49]</a>.

## Creating a Questionnaire

To create a new questionnaire:

1.  Click the "create a questionnaire" option <a class="yt-timestamp" data-t="00:04:31">[00:04:31]</a>, <a class="yt-timestamp" data-t="00:04:32">[00:04:32]</a>.
2.  Provide a name for the questionnaire (e.g., "Fractions Tutorial") <a class="yt-timestamp" data-t="00:04:34">[00:04:34]</a>, <a class="yt-timestamp" data-t="00:04:35">[00:04:35]</a>.
3.  Select the relevant theme or subject (e.g., "Mathematics") <a class="yt-timestamp" data-t="00:04:43">[00:04:43]</a>, <a class="yt-timestamp" data-t="00:04:46">[00:04:46]</a>. Multiple themes can be chosen <a class="yt-timestamp" data-t="00:04:47">[00:04:47]</a>, <a class="yt-timestamp" data-t="00:04:48">[00:04:48]</a>.
4.  Click "next" to proceed to the question editor <a class="yt-timestamp" data-t="00:04:57">[00:04:57]</a>, <a class="yt-timestamp" data-t="00:05:00">[00:05:00]</a>.

### Adding Questions

The editor provides several options for adding questions:
*   **New Question**: Create various question types, including multiple choice, multiple box, or fill-in-the-blank <a class="yt-timestamp" data-t="00:05:12">[00:05:12]</a>, <a class="yt-timestamp" data-t="00:05:14">[00:05:14]</a>, <a class="yt-timestamp" data-t="00:05:19">[00:05:19]</a>.
*   **Teleport Question**: Import questions from other existing tests on Quizizz <a class="yt-timestamp" data-t="00:05:10">[00:05:10]</a>, <a class="yt-timestamp" data-t="00:05:11">[00:05:11]</a>, <a class="yt-timestamp" data-t="00:05:27">[00:05:27]</a>. Imported questions can be modified to suit your needs, including changing text, answers, and difficulty <a class="yt-timestamp" data-t="00:05:47">[00:05:47]</a>, <a class="yt-timestamp" data-t="00:05:50">[00:05:50]</a>, <a class="yt-timestamp" data-t="00:06:06">[00:06:06]</a>.
*   **Enhancements**: You can add an image to the quiz for visual appeal <a class="yt-timestamp" data-t="00:06:10">[00:06:10]</a>, <a class="yt-timestamp" data-t="00:06:12">[00:06:12]</a>. Other options exist to improve the questionnaire's quality <a class="yt-timestamp" data-t="00:06:21">[00:06:21]</a>, <a class="yt-timestamp" data-t="00:06:28">[00:06:28]</a>.

2.  Review settings such as the quiz image, language (e.g., Spanish), and grade level (e.g., fifth grade) <a class="yt-timestamp" data-t="00:06:43">[00:06:43]</a>, <a class="yt-timestamp" data-t="00:06:45">[00:06:45]</a>, <a class="yt-timestamp" data-t="00:06:48">[00:06:48]</a>, <a class="yt-timestamp" data-t="00:06:56">[00:06:56]</a>.
3.  Choose whether to publish the quiz publicly for everyone to see or keep it private <a class="yt-timestamp" data-t="00:07:02">[00:07:02]</a>, <a class="yt-timestamp" data-t="00:07:04">[00:07:04]</a>, <a class="yt-timestamp" data-t="00:07:07">[00:07:07]</a>.
4.  The questionnaire will then be created <a class="yt-timestamp" data-t="00:07:08">[00:07:08]</a>, <a class="yt-timestamp" data-t="00:07:10">[00:07:10]</a>.

## Sharing Quizzes with Students

After creation, quizzes can be shared with students in several ways <a class="yt-timestamp" data-t="00:07:16">[00:07:16]</a>, <a class="yt-timestamp" data-t="00:07:18">[00:07:18]</a>:
*   **Assign to Classes**: Quizzes can be assigned to configured classes at a specific date and time (e.g., May 14th at 12 PM) <a class="yt-timestamp" data-t="00:07:20">[00:07:20]</a>, <a class="yt-timestamp" data-t="00:07:23">[00:07:23]</a>, <a class="yt-timestamp" data-t="00:07:31">[00:07:31]</a>, <a class="yt-timestamp" data-t="00:07:45">[00:07:45]</a>. This utilizes the "classes" option mentioned previously <a class="yt-timestamp" data-t="00:07:28">[00:07:28]</a>.
*   **Practice Link**: A practice link can be generated and copied to send to students, allowing them to access and play the quiz whenever they wish <a class="yt-timestamp" data-t="00:07:55">[00:07:55]</a>, <a class="yt-timestamp" data-t="00:07:56">[00:07:56]</a>, <a class="yt-timestamp" data-t="00:08:00">[00:08:00]</a>, <a class="yt-timestamp" data-t="00:08:04">[00:08:04]</a>, <a class="yt-timestamp" data-t="00:08:09">[00:08:09]</a>.
