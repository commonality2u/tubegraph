## The Watermark Problem

When directly exporting a video from [[video_editing_using_filmora_10 | Filmora 10]], the saved video will include a watermark <a class="yt-timestamp" data-t="00:01:16">[00:01:16]</a>. This watermark is what the tutorial aims to help users avoid <a class="yt-timestamp" data-t="00:01:19">[00:01:19]</a>.

## Solution: Using OBS Studio

The recommended method to avoid the watermark is to use OBS Studio, a program designed for screen recording <a class="yt-timestamp" data-t="00:01:26">[00:01:26]</a>.

### How it Works

Instead of exporting the file from [[video_editing_using_filmora_10 | Filmora 10]], the process involves recording the [[video_editing_using_filmora_10 | Filmora 10]] screen using OBS Studio <a class="yt-timestamp" data-t="00:01:36">[00:01:36]</a>. This allows the user to save the edited video as a file without the watermark <a class="yt-timestamp" data-t="00:01:40">[00:01:40]</a>.

### Step-by-Step Process

1.  **Import and Edit Video in [[video_editing_using_filmora_10 | Filmora 10]]**: First, import the video you wish to edit into [[video_editing_using_filmora_10 | Filmora 10]] <a class="yt-timestamp" data-t="00:00:49">[00:00:49]</a>, and complete all necessary edits <a class="yt-timestamp" data-t="00:01:02">[00:01:02]</a>.
2.  **Open OBS Studio**: Download and open OBS Studio <a class="yt-timestamp" data-t="00:01:26">[00:01:26]</a>.
3.  **Prepare Screens**: Arrange your screens so that [[video_editing_using_filmora_10 | Filmora 10]] is open and visible on one screen, and OBS Studio is ready to record <a class="yt-timestamp" data-t="00:02:08">[00:02:08]</a>.
4.  **Select Recording Area**: In OBS Studio, choose the screen displaying the [[video_editing_using_filmora_10 | Filmora 10]] preview as the recording source <a class="yt-timestamp" data-t="00:02:22">[00:02:22]</a>. Maximize the [[video_editing_using_filmora_10 | Filmora 10]] screen to fill the recording area <a class="yt-timestamp" data-t="00:02:30">[00:02:30]</a>.
5.  **Start Recording and Play Video**: Initiate recording in OBS Studio <a class="yt-timestamp" data-t="00:02:45">[00:02:45]</a>, then play the edited video within [[video_editing_using_filmora_10 | Filmora 10]] <a class="yt-timestamp" data-t="00:02:54">[00:02:54]</a>. OBS Studio will record both the video and its sound <a class="yt-timestamp" data-t="00:03:11">[00:03:11]</a>.

By recording the [[video_editing_using_filmora_10 | Filmora 10]] screen with OBS Studio, users can export videos without the [[removing_filmora_10_watermark | Filmora watermark]] after editing <a class="yt-timestamp" data-t="00:03:14">[00:03:14]</a>.
