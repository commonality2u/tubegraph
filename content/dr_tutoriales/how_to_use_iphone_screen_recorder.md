The iPhone screen recorder allows users to record videos, photos, or anything displayed on their screen, making it useful for creating [[recording_video_tutorials_on_iphone | video tutorials]] for friends or colleagues <a class="yt-timestamp" data-t="00:00:04">[00:00:04]</a>.

## Setting up the Screen Recorder

To enable and use the screen recorder on an iPhone:

1.  **Enter Settings** The first step is to enter the iPhone's "Settings" <a class="yt-timestamp" data-t="00:00:16">[00:00:16]</a>.
2.  **Go to Control Center** Navigate to the "Control Center" section, which is represented by a gray icon <a class="yt-timestamp" data-t="00:00:19">[00:00:19]</a>. Do not access it from apps <a class="yt-timestamp" data-t="00:00:25">[00:00:25]</a>.
3.  **Customize Controls** Select "Customize Controls" <a class="yt-timestamp" data-t="00:00:30">[00:00:30]</a>. This section offers various options like the world clock or stopwatch <a class="yt-timestamp" data-t="00:00:32">[00:00:32]</a>.
4.  **Add Screen Recording** Find and add "Screen Recording" to the included controls <a class="yt-timestamp" data-t="00:00:37">[00:00:37]</a>. Once added, it will be listed among the active controls <a class="yt-timestamp" data-t="00:00:46">[00:00:46]</a>. The order of these controls can be customized according to user preference <a class="yt-timestamp" data-t="00:00:47">[00:00:47]</a>.

## Using the Screen Recorder

Once activated in the [[customizing_iphone_control_center_settings | Control Center]] settings:

*   **Access the Recorder** To use the screen recorder, slide down from the top of the screen to access the [[customizing_iphone_control_center_settings | Control Center]] <a class="yt-timestamp" data-t="00:00:59">[00:00:59]</a>. The screen recorder option will always be available there <a class="yt-timestamp" data-t="00:01:00">[00:01:00]</a>.
*   **Record with Sound** To record audio along with the screen video, press and hold the screen recorder icon <a class="yt-timestamp" data-t="00:01:05">[00:01:05]</a>. This action will show an option to ensure the [[activating_microphone_for_screen_recording_on_iphone | microphone is activated]] <a class="yt-timestamp" data-t="00:01:11">[00:01:11]</a>, allowing for videos to be recorded with internal mobile sound <a class="yt-timestamp" data-t="00:01:17">[00:01:17]</a>.
