This article outlines how to set Google as your default search engine in [[using_google_chrome_for_youtube | Google Chrome]] <a class="yt-timestamp" data-t="00:00:00">[00:00:00]</a>. While Google Chrome often has Google set as the default search engine, it's possible for other search engines like Yahoo to appear as the default, especially after downloading extensions or programs <a class="yt-timestamp" data-t="00:00:34">[00:00:34]</a>.

## Accessing Chrome Settings

To begin, ensure you are within [[using_google_chrome_for_youtube | Google Chrome]] <a class="yt-timestamp" data-t="00:00:31">[00:00:31]</a>.
1.  Navigate to the top right corner of your Chrome browser <a class="yt-timestamp" data-t="00:01:01">[00:01:01]</a>.
2.  Click on the "Settings" or "Configuration" option <a class="yt-timestamp" data-t="00:01:08">[00:01:08]</a>. This will open a new tab <a class="yt-timestamp" data-t="00:01:14">[00:01:14]</a>.

## Managing Search Engines

Once in the settings tab:
1.  Locate and select the "Search Engine" section <a class="yt-timestamp" data-t="00:01:17">[00:01:17]</a>.
2.  Within the Search Engine settings, select "Manage Search Engines" <a class="yt-timestamp" data-t="00:01:44">[00:01:44]</a>.
    *   This section will display all search engines currently installed in your browser <a class="yt-timestamp" data-t="00:01:47">[00:01:47]</a>.
    *   You might find other search engines like Bing, Yahoo, or Ecosia set as default <a class="yt-timestamp" data-t="00:01:57">[00:01:57]</a>.

### Setting Google as Default

Ensure that Google is set as your default search engine <a class="yt-timestamp" data-t="00:01:50">[00:01:50]</a>.

### Editing or Adding Google (if missing)

If Google is not present or you wish to [[editing_and_adding_new_search_engines_in_chrome | edit an existing search engine]] <a class="yt-timestamp" data-t="00:02:03">[00:02:03]</a>:
1.  To add Google, click the pencil icon next to an entry (or an add button if available).
2.  Enter "Google" for the name <a class="yt-timestamp" data-t="00:02:18">[00:02:18]</a>.
3.  For the shortcut, input "Google.com" <a class="yt-timestamp" data-t="00:02:22">[00:02:22]</a>.
    *   **Important**: Using "Google.com" is crucial to avoid potential issues in different countries that might use country-specific domains like ".es" or ".uk" <a class="yt-timestamp" data-t="00:02:22">[00:02:22]</a>.
4.  Once finished, click "Save" <a class="yt-timestamp" data-t="00:02:36">[00:02:36]</a>. No further saving actions are required <a class="yt-timestamp" data-t="00:02:49">[00:02:49]</a>.
