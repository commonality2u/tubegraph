This tutorial demonstrates how to correctly apply [[using_superscripts_in_excel | superscripts]] and [[using_subscripts_in_excel | subscripts]] in [[formatting_text_in_excel | Excel]] to display numbers or characters above or below the main text <a class="yt-timestamp" data-t="00:00:12">[00:00:12]</a>. This method is particularly useful for [[formatting_text_in_excel | formatting]] data in graphs or values that require indices <a class="yt-timestamp" data-t="00:00:26">[00:00:26]</a>.

## Traditional (Incorrect) Methods

Traditionally, some users might attempt to indicate indices by using arrows (↑ or ↓) next to the number to suggest it should be a [[using_superscripts_in_excel | superscript]] or [[using_subscripts_in_excel | subscript]] <a class="yt-timestamp" data-t="00:00:44">[00:00:44]</a>. This is not the proper way to display them <a class="yt-timestamp" data-t="00:00:50">[00:00:50]</a>.

## Applying a [[using_superscripts_in_excel | Superscript]]

To apply a [[using_superscripts_in_excel | superscript]]:
1.  Select the specific character(s) within a cell that you want to elevate <a class="yt-timestamp" data-t="00:01:16">[00:01:16]</a>.
2.  Navigate to the "Font" settings menu, typically accessible via a small arrow in the Font group on the Home tab <a class="yt-timestamp" data-t="00:01:19">[00:01:19]</a>.
3.  Check the "Superscript" option <a class="yt-timestamp" data-t="00:01:26">[00:01:26]</a>.

For example, to turn a '3' into a [[using_superscripts_in_excel | superscript]], select the '3', open the Font settings, and check "Superscript" <a class="yt-timestamp" data-t="00:01:33">[00:01:33]</a>. While the number appears correctly above the text in the cell, it may still appear on the same line in the formula bar <a class="yt-timestamp" data-t="00:01:43">[00:01:43]</a>. This method can also be applied to labels in graphs, such as density graphs, to correct the [[formatting_text_in_excel | display]] of indices like a '3' <a class="yt-timestamp" data-t="00:02:09">[00:02:09]</a>.

## Applying a [[using_subscripts_in_excel | Subscript]]

To apply a [[using_subscripts_in_excel | subscript]]:
1.  Select the specific character(s) within a cell that you want to lower <a class="yt-timestamp" data-t="00:02:42">[00:02:42]</a>.
2.  Open the "Font" settings menu <a class="yt-timestamp" data-t="00:02:49">[00:02:49]</a>.
3.  Check the "Subscript" option <a class="yt-timestamp" data-t="00:02:49">[00:02:49]</a>.

After applying, the selected character(s) will appear below the main text in the cell <a class="yt-timestamp" data-t="00:02:53">[00:02:53]</a>.
