The [[how_to_activate_and_install_electronic_dnie | DNIe]] requires a specific code, referred to as a PIN key, to be activated and used <a class="yt-timestamp" data-t="00:00:15">[00:00:15]</a>. This code is typically issued on the same day the [[how_to_activate_and_install_electronic_dnie | DNI]] is obtained <a class="yt-timestamp" data-t="00:00:18">[00:00:18]</a>. It is crucial for validating certificates on the [[how_to_activate_and_install_electronic_dnie | DNIe]] <a class="yt-timestamp" data-t="00:05:43">[00:05:43]</a>.

## Obtaining the DNIe Code (PIN Key)

If you do not have this code or have forgotten it, you can retrieve it by following these steps:

1.  **Visit a Police Station**
    Go to the nearest police station to your home <a class="yt-timestamp" data-t="00:00:25">[00:00:25]</a>.
2.  **Use Automatic Machines**
    Locate the automatic machines available at the police station <a class="yt-timestamp" data-t="00:00:28">[00:00:28]</a>.
3.  **Log in with your ID Card**
    Insert your [[how_to_activate_and_install_electronic_dnie | ID card]] into the machine to log in <a class="yt-timestamp" data-t="00:00:34">[00:00:34]</a>.
4.  **Register and Retrieve Code**
    Follow the on-screen prompts to register, and the machine will provide you with the code <a class="yt-timestamp" data-t="00:00:37">[00:00:37]</a>. This code is a PIN key that will be needed for future processes <a class="yt-timestamp" data-t="00:02:17">[00:02:17]</a>.
