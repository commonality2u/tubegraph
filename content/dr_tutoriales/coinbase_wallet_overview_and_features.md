Coinbase Wallet is a secure application designed for saving cryptocurrencies and tokens <a class="yt-timestamp" data-t="01:03:03">[01:03:03]</a>. It allows users to store a wide range of cryptocurrencies and some NFTs <a class="yt-timestamp" data-t="01:42:15">[01:42:15]</a>. The application is available on Google Play, the iPhone App Store, and as a Google Chrome extension <a class="yt-timestamp" data-t="01:11:11">[01:11:11]</a>. The mobile phone application is very similar for both Android and iOS <a class="yt-timestamp" data-t="01:22:20">[01:22:20]</a>.

## [[how_to_download_and_set_up_coinbase_wallet | Downloading and Setting Up Coinbase Wallet]]

To [[how_to_download_and_set_up_coinbase_wallet | download the application]], Android users can access Google Play, while iPhone users can go to the App Store and search for "Coinbase Wallet" <a class="yt-timestamp" data-t="02:20:00">[02:20:00]</a>.

Upon opening the application, users can choose to enter with an existing Google account or [[how_to_download_and_set_up_coinbase_wallet | create a new account]] <a class="yt-timestamp" data-t="02:42:01">[02:42:01]</a>. Users must confirm the terms of service and privacy policy <a class="yt-timestamp" data-t="02:48:00">[02:48:00]</a>. A username needs to be created <a class="yt-timestamp" data-t="02:53:07">[02:53:07]</a>.

### Privacy Settings

Users can choose their privacy type:
*   **Public**: Allows other Coinbase Wallet users to search for your username and send cryptocurrencies <a class="yt-timestamp" data-t="03:09:00">[03:09:00]</a>. This is generally recommended if you wish for others to send you crypto <a class="yt-timestamp" data-t="03:21:09">[03:21:09]</a>.
*   **Private**: Prevents other wallets from finding your profile <a class="yt-timestamp" data-t="03:16:00">[03:16:00]</a>.

## [[how_to_secure_and_back_up_your_coinbase_wallet | Security and Backup]]

Coinbase Wallet's security features include secure two-factor authentication and a cloud backup option <a class="yt-timestamp" data-t="01:48:00">[01:48:00]</a>. Users can protect their wallet using biometric authentication like Face ID on iPhones or set a password on other mobile devices <a class="yt-timestamp" data-t="03:27:00">[03:27:00]</a>.

### Recovery Phrase

It is crucial to [[how_to_secure_and_back_up_your_coinbase_wallet | back up your Coinbase Wallet]] <a class="yt-timestamp" data-t="03:43:00">[03:43:00]</a>. If the recovery phrase is lost, access to the account will be impossible <a class="yt-timestamp" data-t="03:47:00">[03:47:00]</a>. Users can choose between iCloud backup or manual backup of the 12-word recovery phrase <a class="yt-timestamp" data-t="04:10:00">[04:10:00]</a>. This phrase acts as the final password to access the wallet <a class="yt-timestamp" data-t="04:39:00">[04:39:00]</a>.

## [[managing_cryptocurrencies_and_nfts_in_coinbase_wallet | Managing Cryptocurrencies and NFTs]]

Once inside the Coinbase Wallet, users can view their cryptocurrency and NFT holdings <a class="yt-timestamp" data-t="04:59:00">[04:59:00]</a>. The bottom menu provides access to the wallet, an internet browser, and settings <a class="yt-timestamp" data-t="05:04:00">[05:04:00]</a>.

Within settings, users can configure:
*   **Blockchain Selection**: Choose the blockchain being used, such as Ethereum or Bitcoin, from a list <a class="yt-timestamp" data-t="05:23:00">[05:23:00]</a>.
*   **Active Wallets**: Manage active wallets <a class="yt-timestamp" data-t="05:37:00">[05:37:00]</a>.
*   **Pending Transactions**: Clean up pending transactions <a class="yt-timestamp" data-t="05:39:00">[05:39:00]</a>.

## Sending and Receiving Cryptocurrencies

The wallet's main section offers options to [[sending_and_receiving_cryptocurrencies_with_coinbase_wallet | send, receive, or trade cryptocurrencies]] <a class="yt-timestamp" data-t="05:51:00">[05:51:00]</a>.

### [[sending_and_receiving_cryptocurrencies_with_coinbase_wallet | Receiving Cryptocurrencies]]

To [[sending_and_receiving_cryptocurrencies_with_coinbase_wallet | receive funds]], select "receive" and choose the desired cryptocurrency, such as Litecoin <a class="yt-timestamp" data-t="05:58:00">[05:58:00]</a>. The wallet provides a receiving address and a QR code <a class="yt-timestamp" data-t="06:08:00">[06:08:00]</a>. You can also share your username to accept payments from other Coinbase Wallet users <a class="yt-timestamp" data-t="06:30:00">[06:30:00]</a>.

### [[sending_and_receiving_cryptocurrencies_with_coinbase_wallet | Sending Cryptocurrencies]]

To [[sending_and_receiving_cryptocurrencies_with_coinbase_wallet | send cryptocurrencies]], select the "send" option <a class="yt-timestamp" data-t="06:48:00">[06:48:00]</a>. Funds can be sent from a connected Coinbase account or another wallet <a class="yt-timestamp" data-t="06:56:00">[06:56:00]</a>. Users can scan a QR code or manually enter the recipient's wallet address <a class="yt-timestamp" data-t="08:53:00">[08:53:00]</a>.

## Trading and Converting

Coinbase Wallet allows users to trade (buy/sell) and convert cryptocurrencies.
*   **Buying**: You can buy cryptocurrencies directly from the wallet <a class="yt-timestamp" data-t="07:38:00">[07:38:00]</a>. Payment methods include credit/debit cards or cash held in a linked Coinbase account <a class="yt-timestamp" data-t="08:09:00">[08:09:00]</a>.
*   **Converting**: The "convert" feature enables exchanging one cryptocurrency for another or converting crypto to dollars <a class="yt-timestamp" data-t="09:37:00">[09:37:00]</a>.

## Connecting with Coinbase Platform

Coinbase Wallet can be connected to your main Coinbase account, allowing seamless transfers of cryptocurrencies and cash between both platforms <a class="yt-timestamp" data-t="07:05:00">[07:05:00]</a>. This connection enables users to transfer subscribed coins from the Coinbase platform to the wallet for sending <a class="yt-timestamp" data-t="08:48:00">[08:48:00]</a>.

## Internet Browser and dApps

The wallet includes an internet browser that allows access to decentralized applications (dApps) and online platforms, such as PancakeSwap <a class="yt-timestamp" data-t="10:05:00">[10:05:00]</a>.

## Important Considerations

Coinbase Wallet is an independent wallet separate from the main Coinbase platform <a class="yt-timestamp" data-t="10:29:00">[10:29:00]</a>. Transfers between Coinbase Wallet and other platforms, including Coinbase, incur associated blockchain commissions <a class="yt-timestamp" data-t="10:37:00">[10:37:00]</a>. It's recommended to use networks with lower fees, like Litecoin, for initial transfers, and then convert coins within the wallet if desired <a class="yt-timestamp" data-t="10:43:00">[10:43:00]</a>.
