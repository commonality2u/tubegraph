If you wish to uninstall YouTube Music from your computer <a class="yt-timestamp" data-t="01:32:00">[01:32:00]</a>, the process involves accessing the Control Panel.

## Steps to Uninstall

1.  **Access the Control Panel**:
    *   Open your computer's search function <a class="yt-timestamp" data-t="01:36:00">[01:36:00]</a>.
    *   Type "control panel" into the search bar <a class="yt-timestamp" data-t="01:39:00">[01:39:00]</a>.
    *   Click on the Control Panel application to open it <a class="yt-timestamp" data-t="01:42:00">[01:42:00]</a>.
2.  **Find "Uninstall a program"**:
    *   Within the Control Panel, locate the option labeled "Uninstall a program" <a class="yt-timestamp" data-t="01:44:00">[01:44:00]</a>.
3.  **Locate YouTube Music**:
    *   Scroll through the list of installed programs to find "YouTube Music" <a class="yt-timestamp" data-t="01:47:00">[01:47:00]</a>. It is typically found towards the bottom of the list <a class="yt-timestamp" data-t="01:47:00">[01:47:00]</a>.
4.  **Uninstall**:
    *   Right-click on "YouTube Music" <a class="yt-timestamp" data-t="01:47:00">[01:47:00]</a>.
    *   Select the "Uninstall" option <a class="yt-timestamp" data-t="01:50:00">[01:50:00]</a>.
