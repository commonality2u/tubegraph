## Accessing Signature Settings

To [[how_to_change_your_signature_in_outlook | change your account signature Outlook]] <a class="yt-timestamp" data-t="00:00:00">[00:00:00]</a>:
1.  Navigate to the menu at the top of Outlook <a class="yt-timestamp" data-t="00:00:24">[00:00:24]</a>.
2.  Select "Options" <a class="yt-timestamp" data-t="00:00:31">[00:00:31]</a>.
3.  Go to the "Mail" section, which is labeled as "email" <a class="yt-timestamp" data-t="00:00:35">[00:00:35]</a>.
4.  Find and click on "Signatures" <a class="yt-timestamp" data-t="00:00:40">[00:00:40]</a>. This option allows you to create or modify signatures <a class="yt-timestamp" data-t="00:00:43">[00:00:43]</a>.

## Creating a New Signature

1.  Within the Signatures window, you can create a new signature <a class="yt-timestamp" data-t="00:00:49">[00:00:49]</a>.
2.  Click "New" to create a new signature <a class="yt-timestamp" data-t="00:01:00">[00:01:00]</a>.
3.  Name your new signature (e.g., "Subscribe") <a class="yt-timestamp" data-t="00:01:02">[00:01:02]</a>.
4.  Once created, click "OK" <a class="yt-timestamp" data-t="00:01:09">[00:01:09]</a>.

## Customizing the Signature Content

After naming your signature, you can begin to customize its content <a class="yt-timestamp" data-t="00:01:12">[00:01:12]</a>:
*   You can [[exploring_customization_options_for_outlook_email_signatures | import images]] <a class="yt-timestamp" data-t="00:01:14">[00:01:14]</a>.

*   A signature can include elements like your name, a company logo, email address, and phone number <a class="yt-timestamp" data-t="00:01:28">[00:01:28]</a>.

## Saving and Applying Signatures

1.  After designing your signature, ensure you save it <a class="yt-timestamp" data-t="00:01:36">[00:01:36]</a>.
2.  It is important to choose which signature you want to use by default <a class="yt-timestamp" data-t="00:01:42">[00:01:42]</a>.
3.  You can [[configuring_signatures_for_different_email_accounts_in_outlook | configure signatures for different email accounts]] <a class="yt-timestamp" data-t="00:01:58">[00:01:58]</a>, especially if you [[configuring_signatures_for_different_email_accounts_in_outlook | have several email accounts]] <a class="yt-timestamp" data-t="00:02:00">[00:02:00]</a>.
4.  You can set a default signature for new messages <a class="yt-timestamp" data-t="00:02:02">[00:02:02]</a>.
5.  You can also [[using_signatures_for_replies_and_forwarded_emails_in_outlook | configure if you want to use the signature for replies or forwards]] <a class="yt-timestamp" data-t="00:02:05">[00:02:05]</a>, specifying which signature to use for each <a class="yt-timestamp" data-t="00:02:19">[00:02:19]</a>.
6.  Once all settings are configured, click "OK" <a class="yt-timestamp" data-t="00:02:32">[00:02:32]</a>, then "OK" again on the main options window <a class="yt-timestamp" data-t="00:02:35">[00:02:35]</a>.

## Verifying the Signature

*   When composing a new email, the configured signature will automatically appear <a class="yt-timestamp" data-t="00:02:38">[00:02:38]</a>.
*   When replying to an email, your signature will also appear if configured for replies <a class="yt-timestamp" data-t="00:02:50">[00:02:50]</a>.
