This guide explains how to install the YouTube application directly onto your PC, making it accessible like any other desktop program. It also covers how to locate, pin, and [[uninstalling_the_youtube_app_on_pc | uninstall the app]] if needed.

## Installing the YouTube Application

To download and install YouTube on your PC, follow these steps:

1.  **Open a Web Browser** <a class="yt-timestamp" data-t="00:00:05">[00:00:05]</a>
    You can use any search engine like Google Chrome, Microsoft Edge, Opera, or Firefox <a class="yt-timestamp" data-t="00:00:07">[00:00:07]</a>. In this example, Google Chrome is used <a class="yt-timestamp" data-t="00:00:05">[00:00:05]</a>.
2.  **Access YouTube** <a class="yt-timestamp" data-t="00:00:17">[00:00:17]</a>
    Navigate directly to the YouTube website by typing "YouTube" into the search bar and accessing the main page <a class="yt-timestamp" data-t="00:00:17">[00:00:17]</a>.
3.  **Install the App** <a class="yt-timestamp" data-t="00:00:27">[00:00:27]</a>
    Once on the YouTube page, look for an option to install the application. This is typically indicated by a "Download symbol" or the text "Install YouTube" located towards the top of the page <a class="yt-timestamp" data-t="00:00:29">[00:00:29]</a>. Click this option to begin the installation process <a class="yt-timestamp" data-t="00:00:33">[00:00:33]</a>.
4.  **Confirm Download** <a class="yt-timestamp" data-t="00:00:39">[00:00:39]</a>
    The application will download <a class="yt-timestamp" data-t="00:00:39">[00:00:39]</a> and, once completed, will be ready for use <a class="yt-timestamp" data-t="00:00:41">[00:00:41]</a>.

## Locating and Pinning the YouTube App

After installation, the YouTube app can be found and easily accessed in a couple of ways:

*   **Desktop Shortcut** <a class="yt-timestamp" data-t="00:00:49">[00:00:49]</a>
    A shortcut for the YouTube application will automatically be created on your desktop <a class="yt-timestamp" data-t="00:00:49">[00:00:49]</a>.
*   **Search Engine** <a class="yt-timestamp" data-t="00:00:53">[00:00:53]</a>
    You can also find the application by typing "YouTube" into your computer's search bar <a class="yt-timestamp" data-t="00:00:53">[00:00:53]</a>.
*   **[[pinning_youtube_to_taskbar | Pinning to Taskbar]]** <a class="yt-timestamp" data-t="00:00:56">[00:00:56]</a>
    To quickly access YouTube, you can [[pinning_youtube_to_taskbar | pin it to your taskbar]]. Right-click on the YouTube icon and select the option to "anchor to the taskbar" <a class="yt-timestamp" data-t="00:00:58">[00:00:58]</a>. The icon will then remain on your taskbar <a class="yt-timestamp" data-t="00:01:02">[00:01:02]</a>.

## Using the Installed YouTube App

Once installed and opened, the YouTube app will typically show content associated with your registered Google/Channel account <a class="yt-timestamp" data-t="00:01:10">[00:01:10]</a>. For information on other functionalities, such as [[using_google_chrome_to_loop_youtube_videos | looping videos within YouTube]], refer to related guides <a class="yt-timestamp" data-t="00:01:17">[00:01:17]</a>.

## [[uninstalling_the_youtube_app_on_pc | Uninstalling the YouTube App]]

If you need to [[uninstalling_the_youtube_app_on_pc | remove the YouTube application]] from your PC, follow these steps:

1.  **Access Control Panel** <a class="yt-timestamp" data-t="00:01:41">[00:01:41]</a>
    Open your computer's Control Panel <a class="yt-timestamp" data-t="00:01:41">[00:01:41]</a>. You can usually find it by searching for "Control Panel" in the Windows search bar <a class="yt-timestamp" data-t="00:01:43">[00:01:43]</a>.
2.  **Uninstall a Program** <a class="yt-timestamp" data-t="00:01:49">[00:01:49]</a>
    Within the Control Panel, click on "Uninstall a program" <a class="yt-timestamp" data-t="00:01:49">[00:01:49]</a>.
3.  **Locate and Uninstall YouTube** <a class="yt-timestamp" data-t="00:01:52">[00:01:52]</a>
    Find "YouTube" in the list of installed programs <a class="yt-timestamp" data-t="00:01:52">[00:01:52]</a>. Right-click on it and select "Uninstall" <a class="yt-timestamp" data-t="00:01:57">[00:01:57]</a>.
