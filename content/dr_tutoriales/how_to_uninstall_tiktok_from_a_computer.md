If you decide you no longer wish to have [[accessing_and_using_tiktok_from_a_pc | TikTok]] installed on your computer <a class="yt-timestamp" data-t="01:31:31">[01:31:31]</a>, the process involves using the Control Panel.

## Step-by-Step Uninstallation

1.  Open any search engine (e.g., Google Chrome, Firefox, Opera, Microsoft Edge) <a class="yt-timestamp" data-t="00:14:14">[00:14:14]</a> to begin.
2.  Go to the "Control Panel" <a class="yt-timestamp" data-t="01:37:38">[01:37:38]</a>. You can search for "Control" in your taskbar <a class="yt-timestamp" data-t="01:35:00">[01:35:00]</a> to find it <a class="yt-timestamp" data-t="01:37:38">[01:37:38]</a>.
3.  Within the Control Panel, locate and select "Program" or "Uninstall programs" <a class="yt-timestamp" data-t="01:43:44">[01:43:44]</a>.
4.  Look for "TikTok" in the list of programs, which is organized alphabetically <a class="yt-timestamp" data-t="01:46:47">[01:46:47]</a>.
5.  Once found, you can proceed to uninstall it <a class="yt-timestamp" data-t="01:48:50">[01:48:50]</a>.
