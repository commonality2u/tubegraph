## Accessing the Revolut App

To begin, users need to download the Revolut app, which is typically found in the finance section of a mobile device's app store <a class="yt-timestamp" data-t="00:01:09">[00:01:09]</a>. Once downloaded and opened, the app presents its main menu <a class="yt-timestamp" data-t="00:01:17">[00:01:17]</a>.

## App Interface Overview

The Revolut app is designed with a user-friendly interface, offering a top-to-bottom navigation to explore its various features <a class="yt-timestamp" data-t="00:01:24">[00:01:24]</a>.

### Account Settings and Personal Data
In the top-left corner, users can access their account settings, identified by a profile picture <a class="yt-timestamp" data-t="00:01:29">[00:01:29]</a>. Here, options include:
*   Improving a plan <a class="yt-timestamp" data-t="00:01:32">[00:01:32]</a>
*   Inviting friends <a class="yt-timestamp" data-t="00:01:34">[00:01:34]</a>
*   Viewing personal data <a class="yt-timestamp" data-t="00:01:35">[00:01:35]</a>
*   Checking the pricing plan (e.g., standard account) and boundaries <a class="yt-timestamp" data-t="00:01:37">[00:01:37]</a>
*   Accessing documents attached to the bank account <a class="yt-timestamp" data-t="00:01:42">[00:01:42]</a>
*   Customizing the app's appearance, such as the theme (e.g., black) and application icon, which might depend on the user's plan <a class="yt-timestamp" data-t="00:01:44">[00:01:44]</a>.
*   Managing security settings, including the access key, privacy options, blocking betting, and enabling login with Face ID <a class="yt-timestamp" data-t="00:02:02">[00:02:02]</a>.
*   Finding information about Revolut on platforms like the App Store, Facebook, and Twitter <a class="yt-timestamp" data-t="00:02:08">[00:02:08]</a>.
*   Options to close the account or log out <a class="yt-timestamp" data-t="00:02:13">[00:02:13]</a>.
This section primarily displays the user's Revolut bank account data <a class="yt-timestamp" data-t="00:02:20">[00:02:20]</a>.

### Financial Statistics and Budgeting
On the right side of the main screen, Revolut provides financial statistics, showing expenses categorized by month and spending type (e.g., transfers, supermarkets) <a class="yt-timestamp" data-t="00:02:26">[00:02:26]</a>. This feature allows users to analyze where their money is being spent, even showing the country of expenditure <a class="yt-timestamp" data-t="00:02:40">[00:02:40]</a>.

Users can also set monthly budget goals, which helps in [[managing_finances_with_revolut | tracking and managing finances]] <a class="yt-timestamp" data-t="00:02:58">[00:02:58]</a>. The visual representation of spending can be toggled between bar and line charts <a class="yt-timestamp" data-t="00:03:16">[00:03:16]</a>.

### Multi-Currency Accounts
Revolut allows users to manage multiple currency accounts. For example, a user might have a sterling pound account <a class="yt-timestamp" data-t="00:03:27">[00:03:27]</a>. New accounts can be opened in various currencies such as Euro or Dollar, allowing for easy currency exchange <a class="yt-timestamp" data-t="00:05:03">[00:05:03]</a>.

### Favorites and Alerts
The "favorites" section (marked by a star icon) displays real-time exchange rates for currencies (Euro, Dollar) and assets (Bitcoin, Gold) against the user's primary currency <a class="yt-timestamp" data-t="00:03:33">[00:03:33]</a>. It also includes a converter to show the value of a specific amount in different assets <a class="yt-timestamp" data-t="00:03:45">[00:03:45]</a>.

Users can set up alerts for price movements of assets like Bitcoin, receiving notifications when a desired price is reached, which is useful for investment decisions <a class="yt-timestamp" data-t="00:03:57">[00:03:57]</a>. This section also allows for managing orders <a class="yt-timestamp" data-t="00:04:28">[00:04:28]</a>.

### Notifications
A bell icon typically represents the notifications section, common across most platforms <a class="yt-timestamp" data-t="00:04:36">[00:04:36]</a>.

## Core Features and Sections

The bottom menu or dashboard of the Revolut app provides quick access to several core features: Accounts, Cards, Stocks, and Crypto <a class="yt-timestamp" data-t="00:04:46">[00:04:46]</a>.

### Accounts
This section, as mentioned, allows users to manage their currency accounts <a class="yt-timestamp" data-t="00:04:49">[00:04:49]</a>. Beyond adding new accounts, users can:
*   Add money <a class="yt-timestamp" data-t="00:05:17">[00:05:17]</a>
*   Send money <a class="yt-timestamp" data-t="00:05:18">[00:05:18]</a>
*   Exchange currencies <a class="yt-timestamp" data-t="00:05:20">[00:05:20]</a>
*   Generate bank statements <a class="yt-timestamp" data-t="00:05:22">[00:05:22]</a>
*   View account details <a class="yt-timestamp" data-t="00:05:23">[00:05:23]</a>

#### Sending Money
Revolut simplifies [[international_money_transfers_with_revolut | sending money]] to contacts or other entities <a class="yt-timestamp" data-t="00:05:27">[00:05:27]</a>. Users can:
*   Select from existing contacts <a class="yt-timestamp" data-t="00:05:32">[00:05:32]</a>
*   Create a payment link <a class="yt-timestamp" data-t="00:05:36">[00:05:36]</a>
*   Add a bank consignee by providing account details (for English, Spanish, etc. accounts) <a class="yt-timestamp" data-t="00:05:42">[00:05:42]</a>
*   Invite a friend <a class="yt-timestamp" data-t="00:05:48">[00:05:48]</a>

The process is designed to be very easy and simple, working like a conventional bank but offering immediate transfers, unlike traditional banks that might take longer <a class="yt-timestamp" data-t="00:05:50">[00:05:50]</a>. A history of all transactions is also available <a class="yt-timestamp" data-t="00:06:05">[00:06:05]</a>.

### Analysis
This section provides a deeper look into financial statistics, including net worth and detailed breakdowns of income and expenses <a class="yt-timestamp" data-t="00:06:30">[00:06:30]</a>. It's essentially another direct access point to the financial overview from the main dashboard <a class="yt-timestamp" data-t="00:06:34">[00:06:34]</a>.

### Savings and Shares
Under the analysis part, [[saving_and_investing_features_on_revolut | Revolut offers savings features]], showing how much is saved in different currencies like pounds and USD <a class="yt-timestamp" data-t="00:06:47">[00:06:47]</a>. Higher annual percentage earnings (e.g., 0.20% for pounds, 0.40% for USD) are available with upgraded plans like "Plus" or "Metal" <a class="yt-timestamp" data-t="00:07:02">[00:07:02]</a>. For standard accounts, these savings "trunks" are not available <a class="yt-timestamp" data-t="00:07:15">[00:07:15]</a>.

### Rewards
The rewards section showcases various discounts available to users at different stores and services (e.g., 40% off protein, discounts at Apple, UDM) <a class="yt-timestamp" data-t="00:07:27">[00:07:27]</a>. Premium or Plus account holders typically receive more discounts and benefits <a class="yt-timestamp" data-t="00:07:42">[00:07:42]</a>.

### Cards
This section is dedicated to [[revolut_card_features | managing Revolut cards]], including both virtual and physical cards <a class="yt-timestamp" data-t="00:07:53">[00:07:53]</a>. Key features include:
*   **Virtual Cards**: Including disposable virtual cards, which are highly secure for making online payments <a class="yt-timestamp" data-t="00:08:09">[00:08:09]</a>.
*   **Adding New Cards**: Users can add new debit cards or virtual debit cards <a class="yt-timestamp" data-t="00:08:18">[00:08:18]</a>.
*   **Customization**: Cards can be customized with designs, such as the English flag <a class="yt-timestamp" data-t="00:08:30">[00:08:30]</a>.
*   **Metal Cards**: Users can upgrade their plan to a "Metal" plan to obtain a metal card, which comes with specific characteristics <a class="yt-timestamp" data-t="00:08:35">[00:08:35]</a>.

### Stocks
Revolut allows users to [[investing_and_trading_with_revolut | invest in stocks]] directly from the platform with as little as one dollar <a class="yt-timestamp" data-t="00:09:02">[00:09:02]</a>. The platform offers access to over 800 shares <a class="yt-timestamp" data-t="00:09:14">[00:09:14]</a>, including popular ones like Tesla, Apple, American Airlines, Amazon, and Coinbase <a class="yt-timestamp" data-t="00:09:33">[00:09:33]</a>.

For each stock, users can view:
*   Price charts (e.g., 5-year, 1-year) <a class="yt-timestamp" data-t="00:09:43">[00:09:43]</a>
*   Financial data (e.g., profit margin, annual/quarterly data) <a class="yt-timestamp" data-t="00:09:48">[00:09:48]</a>
*   Relevant news <a class="yt-timestamp" data-t="00:09:57">[00:09:57]</a>
*   Order history (completed orders) <a class="yt-timestamp" data-t="00:10:08">[00:10:08]</a>

To buy or sell shares, users must activate their social security number (or national ID for Spanish users) for tax quantification and government control over gains <a class="yt-timestamp" data-t="00:10:16">[00:10:16]</a>.

### Crypto
Revolut also enables users to [[investing_and_trading_with_revolut | buy cryptocurrencies]] <a class="yt-timestamp" data-t="00:11:09">[00:11:09]</a>. Users can buy various cryptocurrencies, including Bitcoin, Cardano, Aptos, EOS, Ether, and Litecoin <a class="yt-timestamp" data-t="00:12:03">[00:12:03]</a>.

A commission of 2.5% is charged on cryptocurrency transactions for standard plans <a class="yt-timestamp" data-t="00:11:27">[00:11:27]</a>. Upgrading to a different plan, such as "Metal," can reduce these commissions <a class="yt-timestamp" data-t="00:11:32">[00:11:32]</a>.

### Vaults (Savings)
This section allows users to save money and receive an annual reward, such as 0.65% for certain plans (e.g., Metal) <a class="yt-timestamp" data-t="00:12:37">[00:12:37]</a>.

## Bottom Navigation Bar

The bottom navigation bar provides quick access to key functionalities:
*   **"R" icon**: Returns to the main Revolut platform <a class="yt-timestamp" data-t="00:12:56">[00:12:56]</a>.
*   **"Exchange" icon**: Accesses payments, transfers, and allows users to send or request money, and schedule transactions <a class="yt-timestamp" data-t="00:13:01">[00:13:01]</a>.
*   **"Menu" (squares) icon**: Offers an alternative view of the app's features, including cards, ATMs, [[saving_and_investing_features_on_revolut | savings and investments]], stocks, crypto, travel, gifts, invitations, and lounges <a class="yt-timestamp" data-t="00:13:16">[00:13:16]</a>. Some features, like lounge access for travel, depend on the user's Revolut plan <a class="yt-timestamp" data-t="00:13:43">[00:13:43]</a>.
