Quizizz is an online tool that allows teachers to create and administer online didactic tests for students <a class="yt-timestamp" data-t="00:00:05">[00:00:05]</a>. It facilitates real-time connection with students and sends notifications <a class="yt-timestamp" data-t="00:02:56">[00:02:56]</a>.

## Accessing and Registering for Quizizz

To begin, access the Quizizz website <a class="yt-timestamp" data-t="00:00:45">[00:00:45]</a>. If you don't have an account, you will need to register <a class="yt-timestamp" data-t="00:00:54">[00:00:54]</a>. Registration can be done using a Google email or other email addresses <a class="yt-timestamp" data-t="00:01:01">[00:01:01]</a>. It is important to [[how_to_register_and_set_up_a_quizizz_account_for_teachers | register as a teacher]] because teachers design and share the tests <a class="yt-timestamp" data-t="00:01:16">[00:01:16]</a>. Parents and companies can also participate <a class="yt-timestamp" data-t="00:01:21">[00:01:21]</a>.

## Quizizz Interface Overview

Once logged in, pay attention to the left panel <a class="yt-timestamp" data-t="00:01:36">[00:01:36]</a>. This panel displays your profile and various options:
*   **Create**: For designing new tests <a class="yt-timestamp" data-t="00:01:43">[00:01:43]</a>.
*   **Search**: To find tests created by other users <a class="yt-timestamp" data-t="00:01:45">[00:01:45]</a>. You can search for quizzes based on themes like mathematics, English, or specific topics like fractions <a class="yt-timestamp" data-t="00:01:52">[00:01:52]</a>.
*   **My Quizzes**: Displays questionnaires you have saved <a class="yt-timestamp" data-t="00:02:12">[00:02:12]</a>.
*   **Reports**: Shows reports on student performance, helping to track notes and qualifications <a class="yt-timestamp" data-t="00:02:17">[00:02:17]</a>. This feature helps in [[using_quizizz_to_track_student_performance_and_progress | tracking student performance and progress]] to identify areas for reinforcement <a class="yt-timestamp" data-t="00:02:28">[00:02:28]</a>.
*   **Classes**: Where student groups are managed <a class="yt-timestamp" data-t="00:02:35">[00:02:35]</a>. Students connect via [[sharing_quizzes_with_students_via_google_classroom | Google Classroom]] or a link <a class="yt-timestamp" data-t="00:02:41">[00:02:41]</a>.
*   **Collections**: Contains quizzes you have created or want to create <a class="yt-timestamp" data-t="00:03:03">[00:03:03]</a>. Quizzes can be saved into [[organizing_quizzes_and_tests_in_collections | collections]], such as "Math Fractions" <a class="yt-timestamp" data-t="00:03:22">[00:03:22]</a>.

## Creating a New Quiz

To create a new quiz:
1.  Select the "Create" option <a class="yt-timestamp" data-t="01:43">[01:43]</a>.
2.  Choose "Create a questionnaire" <a class="yt-timestamp" data-t="00:04:31">[00:04:31]</a>.
3.  Give your questionnaire a name, such as "Fractions Tutorial" <a class="yt-timestamp" data-t="00:04:34">[00:04:34]</a>.
4.  Select a theme for the quiz, for example, "Mathematics" <a class="yt-timestamp" data-t="00:04:43">[00:04:43]</a>. You can choose from various themes <a class="yt-timestamp" data-t="00:04:46">[00:04:46]</a>.
5.  Click "Next" <a class="yt-timestamp" data-t="00:04:57">[00:04:57]</a>.

### Adding Questions

Within the quiz editor, you have options for adding questions:
*   **New Question**: Allows you to create a question from scratch <a class="yt-timestamp" data-t="00:05:12">[00:05:12]</a>. Available question types include multiple choice, checkbox, and fill-in-the-blank <a class="yt-timestamp" data-t="00:05:18">[00:05:18]</a>.
*   **Teleport Question**: This feature allows you to import questions from other existing quizzes <a class="yt-timestamp" data-t="00:05:25">[00:05:25]</a>. You can search for relevant quizzes (e.g., "fractions") and add specific questions from them to your own quiz <a class="yt-timestamp" data-t="00:05:29">[00:05:29]</a>. Once a question is imported, you can modify it, including its content and solution, to suit your needs and desired difficulty level <a class="yt-timestamp" data-t="00:05:47">[00:05:47]</a>.

You can also enhance your quiz by:
*   Adding an image to make the quiz more attractive <a class="yt-timestamp" data-t="00:06:10">[00:06:10]</a>.
*   Setting grades and importing other tests <a class="yt-timestamp" data-t="00:06:21">[00:06:21]</a>.

### Finalizing and Publishing the Quiz

Once you have finished adding and modifying questions:
1.  Click "Finish" <a class="yt-timestamp" data-t="00:06:40">[00:06:40]</a>.
2.  Set the language (e.g., Spanish) <a class="yt-timestamp" data-t="00:06:47">[00:06:47]</a>.
3.  Choose the appropriate grade rating <a class="yt-timestamp" data-t="00:06:54">[00:06:54]</a>.
4.  Decide whether the quiz should be public (visible to everyone) or private <a class="yt-timestamp" data-t="00:07:03">[00:07:03]</a>.
5.  The questionnaire will then be created <a class="yt-timestamp" data-t="00:07:10">[00:07:10]</a>.

### Sharing Quizzes

You can share your quizzes with students in several ways:
*   **Assign to Classes**: You can assign the quiz to specific classes that you have configured, setting a specific date and time for it to be released <a class="yt-timestamp" data-t="00:07:16">[00:07:16]</a>.
*   **Practice Link**: For immediate use or practice, you can generate a practice link that students can copy and paste to access the quiz whenever they want <a class="yt-timestamp" data-t="00:07:55">[00:07:55]</a>.

### Memes in Quizzes

Quizizz incorporates memes by default within tests to help students relax <a class="yt-timestamp" data-t="00:03:57">[00:03:57]</a>. These memes appear during or at the end of tests to make the experience less serious and more relaxed <a class="yt-timestamp" data-t="00:04:06">[00:04:06]</a>.
