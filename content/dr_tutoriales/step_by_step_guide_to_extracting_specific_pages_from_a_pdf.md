Extracting specific pages from a PDF can be useful for various reasons <a class="yt-timestamp" data-t="00:00:00">[00:00:00]</a>. This guide outlines two methods: one using Google Chrome and another using the online tool [[using_i_love_pdf_to_split_pdf_pages | I Love PDF]].

## Method 1: Using Google Chrome

This method allows you to separate PDF pages without needing external software <a class="yt-timestamp" data-t="00:01:43">[00:01:43]</a>.

1.  **Open PDF with Google Chrome**:
    *   Locate the PDF file on your computer <a class="yt-timestamp" data-t="00:00:17">[00:00:17]</a>.
    *   Right-click the PDF file and select "Open with" <a class="yt-timestamp" data-t="00:00:31">[00:00:31]</a>.
    *   Choose "Google Chrome" from the list of applications <a class="yt-timestamp" data-t="00:00:33">[00:00:33]</a>. The PDF will open in a new tab within Chrome <a class="yt-timestamp" data-t="00:00:36">[00:00:36]</a>, displaying its different pages <a class="yt-timestamp" data-t="00:00:39">[00:00:39]</a>.

2.  **Access the Print Function**:
    *   Click the printer icon located at the top of the Chrome browser window <a class="yt-timestamp" data-t="00:00:49">[00:00:49]</a>.

3.  **Select Custom Pages**:
    *   In the print dialog box, locate the "Pages" setting <a class="yt-timestamp" data-t="00:00:52">[00:00:52]</a>.
    *   Change the option from "All" to "Custom" <a class="yt-timestamp" data-t="00:00:52">[00:00:52]</a>, <a class="yt-timestamp" data-t="00:00:55">[00:00:55]</a>.
    *   Enter the specific page numbers you wish to extract. For example, to extract pages 3 and 4, you would type `3,4` <a class="yt-timestamp" data-t="00:01:06">[00:01:06]</a>, <a class="yt-timestamp" data-t="00:01:09">[00:01:09]</a>. This will only download page three and page four <a class="yt-timestamp" data-t="00:01:09">[00:01:09]</a>, <a class="yt-timestamp" data-t="00:01:13">[00:01:13]</a>.

4.  **Save the New PDF**:
    *   Click the "Print" button <a class="yt-timestamp" data-t="00:01:13">[00:01:13]</a>.
    *   A dialog box will appear, asking you to choose where to save the new PDF file <a class="yt-timestamp" data-t="00:01:15">[00:01:15]</a>.
    *   Provide a name for the new file, such as "cropped PDF" <a class="yt-timestamp" data-t="00:01:22">[00:01:22]</a>.
    *   Click "Save" <a class="yt-timestamp" data-t="00:01:27">[00:01:27]</a>.
    *   The newly created PDF will contain only the selected pages <a class="yt-timestamp" data-t="00:01:34">[00:01:34]</a>, <a class="yt-timestamp" data-t="00:01:37">[00:01:37]</a>.

## Method 2: Using I Love PDF

This method utilizes a free online tool to split your PDF pages <a class="yt-timestamp" data-t="00:01:50">[00:01:50]</a>.

1.  **Access I Love PDF**:
    *   Open Google Chrome and search for "I Love PDF" <a class="yt-timestamp" data-t="00:01:48">[00:01:48]</a>, <a class="yt-timestamp" data-t="00:01:50">[00:01:50]</a>.
    *   Navigate to the I Love PDF website, which allows users to work with PDFs for various tasks <a class="yt-timestamp" data-t="00:01:53">[00:01:53]</a>.

2.  **Select "Split PDF" Tool**:
    *   On the I Love PDF homepage, choose the "Split PDF" option <a class="yt-timestamp" data-t="00:01:57">[00:01:57]</a>.

3.  **Upload Your PDF**:
    *   Click "Select PDF file" and upload the PDF you want to split <a class="yt-timestamp" data-t="00:02:01">[00:02:01]</a>, <a class="yt-timestamp" data-t="00:02:04">[00:02:04]</a>.

4.  **Choose Split Options**:
    *   After uploading, you can choose to extract specific pages <a class="yt-timestamp" data-t="00:02:17">[00:02:17]</a>.
    *   You can select individual pages to extract <a class="yt-timestamp" data-t="00:02:19">[00:02:19]</a>, <a class="yt-timestamp" data-t="00:02:22">[00:02:22]</a> or let the tool divide the entire document into separate pages <a class="yt-timestamp" data-t="00:02:25">[00:02:25]</a>.

5.  **Download Split PDFs**:
    *   Click "Split PDF" <a class="yt-timestamp" data-t="00:02:15">[00:02:15]</a>.
    *   Once processed, click "Download" <a class="yt-timestamp" data-t="00:02:27">[00:02:27]</a>. The separate PDF pages will typically be downloaded as a compressed WinRAR file (or similar ZIP archive) <a class="yt-timestamp" data-t="00:02:30">[00:02:30]</a>.
    *   After unzipping, you will find all the individual pages of the original PDF <a class="yt-timestamp" data-t="00:02:34">[00:02:34]</a>, <a class="yt-timestamp" data-t="00:02:37">[00:02:37]</a>.
