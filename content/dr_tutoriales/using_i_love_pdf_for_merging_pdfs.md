From: [[dr_tutoriales]] <br/> 

Learn how to merge multiple PDF documents into a single file easily and without needing to install any external software <a class="yt-timestamp" data-t="00:00:17">[00:00:17]</a>. This guide uses the [[using_i_love_pdf_tool | I Love PDF]] online tool, which is accessible directly from your web browser <a class="yt-timestamp" data-t="00:00:14">[00:00:14]</a>.

## What You'll Need

*   A computer <a class="yt-timestamp" data-t="00:00:14">[00:00:14]</a>.
*   Google Chrome or any web browser <a class="yt-timestamp" data-t="00:00:14">[00:00:14]</a>.
*   The PDF files you wish to merge <a class="yt-timestamp" data-t="00:00:21">[00:00:21]</a>.

> [!INFO] No External Program Needed
> You do not need to download or install any specific software to merge PDFs using this method <a class="yt-timestamp" data-t="00:00:17">[00:00:17]</a>.

## Step-by-Step Guide

### Accessing the Tool

1.  Open your web browser (e.g., Google Chrome) <a class="yt-timestamp" data-t="00:00:14">[00:00:14]</a>.
2.  Navigate to the [[using_i_love_pdf_tool | I Love PDF]] website <a class="yt-timestamp" data-t="00:00:46">[00:00:46]</a>.
3.  Once on the site, look for and select the "Merge PDF" or "Join PDF" option <a class="yt-timestamp" data-t="00:00:49">[00:00:49]</a>.

### Selecting Files

1.  Click the "Select PDFs" button <a class="yt-timestamp" data-t="00:01:14">[00:01:14]</a>.
2.  Browse your computer to the location of your PDF files <a class="yt-timestamp" data-t="00:01:22">[00:01:22]</a>.
    *   For example, you might select a "Safari ticket" PDF and a "Multicinema ticket" PDF <a class="yt-timestamp" data-t="00:01:22">[00:01:22]</a>.
3.  Select all the PDFs you want to merge. It's crucial to select all of them at this stage <a class="yt-timestamp" data-t="00:01:29">[00:01:29]</a>.
4.  Click "Open" <a class="yt-timestamp" data-t="00:01:29">[00:01:29]</a>. The selected PDFs will then load into the [[using_i_love_pdf_tool | I Love PDF]] interface <a class="yt-timestamp" data-t="00:01:38">[00:01:38]</a>.

### Initiating the Merge

1.  After the PDFs have loaded, click the "Merge PDF" or "Join PDF" button within the tool <a class="yt-timestamp" data-t="00:01:38">[00:01:38]</a>.

### [[downloading_and_managing_merged_pdfs | Downloading and Managing the Merged PDF]]

1.  Once the merging process is complete, you will be able to download the newly merged PDF <a class="yt-timestamp" data-t="00:01:43">[00:01:43]</a>.
2.  The merged PDF will contain all the pages from your original files, combined into one document <a class="yt-timestamp" data-t="00:01:50">[00:01:50]</a>.
3.  You can find the downloaded file in your browser's "Downloads" section, typically named something like "Multicinema Safari Merge" <a class="yt-timestamp" data-t="00:01:55">[00:01:55]</a>.
4.  The name of the merged PDF can be edited at any time <a class="yt-timestamp" data-t="00:01:59">[00:01:59]</a>.
5.  Upon opening the merged PDF, you will see both original documents combined <a class="yt-timestamp" data-t="00:02:00">[00:02:00]</a>.
