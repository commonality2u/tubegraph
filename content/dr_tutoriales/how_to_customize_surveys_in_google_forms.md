[[creating_a_survey_using_google_forms | Creating a survey]] within Google Forms allows for extensive customization to fit various needs, from academic projects to event planning <a class="yt-timestamp" data-t="00:00:00">[00:00:00]</a>.

## Accessing Google Forms
To access Google Forms, navigate to the group of circles, typically found on the right side of the screen, and locate the purple "Forms" icon <a class="yt-timestamp" data-t="00:00:26">[00:00:26]</a>.

## Creating a New Survey
Google Forms offers various [[google_forms_templates | templates]] that can be used as a starting point for surveys, such as:
*   Event attendance confirmation <a class="yt-timestamp" data-t="00:00:59">[00:00:59]</a>
*   Appointment scheduling <a class="yt-timestamp" data-t="00:01:00">[00:01:00]</a>
*   Contact details <a class="yt-timestamp" data-t="00:01:02">[00:01:02]</a>
*   Customer feedback <a class="yt-timestamp" data-t="00:01:15">[00:01:15]</a>
*   Event comments <a class="yt-timestamp" data-t="00:01:15">[00:01:15]</a>
*   Order forms <a class="yt-timestamp" data-t="00:01:16">[00:01:16]</a>

Alternatively, you can [[creating_a_form_with_google_forms | create a new form]] from a blank template <a class="yt-timestamp" data-t="00:01:22">[00:01:22]</a>.

### Setting the Title and Description
Once a new form is started, you can:
*   **Add a Title:** Assign a title to your survey, such as "Survey Number One" or "Survey on Belfast" <a class="yt-timestamp" data-t="00:01:30">[00:01:30]</a>.
*   **Add a Description:** Provide a description, like "Survey for the final degree project of the psychology course" <a class="yt-timestamp" data-t="00:01:47">[00:01:47]</a>.
*   **Customize Text:** The description text can be formatted with bold or italics <a class="yt-timestamp" data-t="00:01:59">[00:01:59]</a>.

### Previewing the Survey
At any point, you can preview how your survey appears by clicking the "preview" option <a class="yt-timestamp" data-t="00:02:05">[00:02:05]</a>.

## Adding and Customizing Questions
Google Forms provides various [[using_different_question_types_in_google_forms | question types]] to customize your survey questions <a class="yt-timestamp" data-t="00:02:29">[00:02:29]</a>:

*   **Multiple Options (Multiple Choice):** Allows respondents to select one option from a list <a class="yt-timestamp" data-t="00:02:52">[00:02:52]</a>. You can add more options as needed <a class="yt-timestamp" data-t="00:02:32">[00:02:32]</a>.
*   **Check Boxes:** Enables respondents to select multiple options from a list <a class="yt-timestamp" data-t="00:02:55">[00:02:55]</a>.
*   **Drop-downs:** Presents options in a drop-down menu, where respondents choose one <a class="yt-timestamp" data-t="00:02:55">[00:02:55]</a>.
*   **Linear Scales:** Useful for rating on a scale (e.g., 1 to 5), allowing you to label the ends of the scale (e.g., "very little" to "much") <a class="yt-timestamp" data-t="00:03:09">[00:03:09]</a>.
*   **Short Answer:** For brief textual responses <a class="yt-timestamp" data-t="00:02:44">[00:02:44]</a>.
*   **Paragraph:** For longer textual responses <a class="yt-timestamp" data-t="00:02:49">[00:02:49]</a>.
*   **Multiple Option Grids / Checkbox Grids:** Allows for creating grid-based questions where respondents can select multiple options across rows and columns <a class="yt-timestamp" data-t="00:04:18">[00:04:18]</a>.
*   **Date and Time:** Questions that allow respondents to input a specific date or time <a class="yt-timestamp" data-t="00:04:29">[00:04:29]</a>.

For surveys, the most commonly used question types are multiple options, check boxes, and drop-downs <a class="yt-timestamp" data-t="00:02:52">[00:02:52]</a>.

## Survey Settings and Configuration
The "Configuration" tab provides [[sharing_and_customization_options_in_google_forms | advanced features in Google Forms for surveys]] <a class="yt-timestamp" data-t="00:05:21">[00:05:21]</a>.

Key customization options for surveys include:
*   **Collecting Email Addresses:** You can choose to automatically collect the email addresses of respondents <a class="yt-timestamp" data-t="00:05:38">[00:05:38]</a>. This can be beneficial for verifying results <a class="yt-timestamp" data-t="00:05:49">[00:05:49]</a>.
*   **Sending Copies of Responses:** Allows respondents to receive a copy of their submitted answers <a class="yt-timestamp" data-t="00:06:00">[00:06:00]</a>. This requires collecting email addresses <a class="yt-timestamp" data-t="00:06:05">[00:06:05]</a>.
*   **Allowing Modification of Answers:** You can enable or disable respondents from modifying their answers after submission <a class="yt-timestamp" data-t="00:06:07">[00:06:07]</a>.
*   **Limiting Responses:** You can restrict a person to fill out the survey only once <a class="yt-timestamp" data-t="00:06:14">[00:06:14]</a>. If this option is not selected, people can fill it multiple times <a class="yt-timestamp" data-t="00:06:16">[00:06:16]</a>.
*   **Converting to a Quiz:** You can convert the form into a questionnaire to assign scores, define correct answers, and provide automatic feedback, which is typically used for tests <a class="yt-timestamp" data-t="00:05:25">[00:05:25]</a>.

## Default Form Settings
Google Forms allows you to save default settings for future forms, which can streamline the creation process for subsequent surveys <a class="yt-timestamp" data-t="00:06:35">[00:06:35]</a>.

*   **Presentation of Results:** Future tutorials may cover how survey results are presented <a class="yt-timestamp" data-t="00:06:28">[00:06:28]</a>.
