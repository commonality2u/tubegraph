From: [[dr_tutoriales]] <br/> 

This tutorial outlines the process of transferring Bitcoin or other cryptocurrencies from a Coinbase account to a Binance account <a class="yt-timestamp" data-t="00:00:06">[00:00:06]</a>. Binance is suggested as a more effective broker for reinvesting cryptocurrencies compared to Coinbase <a class="yt-timestamp" data-t="00:00:14">[00:00:14]</a>.

It is recommended to [[transferring_cryptocurrencies_between_platforms | transfer cryptocurrencies]] from Coinbase to Binance to gain profitability <a class="yt-timestamp" data-t="00:00:25">[00:00:25]</a>. If you are working with Coinbase and do not have a Binance account, you can create one using a provided code for a 10% discount on commissions <a class="yt-timestamp" data-t="00:00:46">[00:00:46]</a>.

## Current Balances <a class="yt-timestamp" data-t="01:04:01">[01:04:01]</a>

The example starts with 109 euros in a Coinbase account, mostly earned from tutorials <a class="yt-timestamp" data-t="01:10:00">[01:10:00]</a>, and an open Binance account <a class="yt-timestamp" data-t="01:16:00">[01:16:00]</a>.

## Understanding the Transfer Process <a class="yt-timestamp" data-t="01:22:00">[01:22:00]</a>

The process involves sending cryptocurrencies from Coinbase to a specific address in Binance <a class="yt-timestamp" data-t="01:28:00">[01:28:00]</a>.

## Obtaining the Binance Deposit Address <a class="yt-timestamp" data-t="01:46:00">[01:46:00]</a>

To find the Binance wallet address for receiving Bitcoin:
1.  Go to "Wallet" <a class="yt-timestamp" data-t="01:50:00">[01:50:00]</a>.
2.  Select "Overview" <a class="yt-timestamp" data-t="01:51:00">[01:51:00]</a>.
3.  Click on the "Deposit" option <a class="yt-timestamp" data-t="01:55:00">[01:55:00]</a>.
4.  Choose "[[how_to_deposit_money_into_binance | Deposit Crypto]]" <a class="yt-timestamp" data-t="02:01:00">[02:01:00]</a>. This option allows you to receive cryptocurrencies from any external wallet using an address or QR code <a class="yt-timestamp" data-t="02:05:00">[02:05:00]</a>.
5.  Select the cryptocurrency you want to deposit; in this case, Bitcoin (BTC) <a class="yt-timestamp" data-t="02:26:00">[02:26:00]</a>.
6.  The system will generate a unique QR code and an address <a class="yt-timestamp" data-t="02:38:00">[02:38:00]</a>. It is crucial to pay close attention to both the QR code and the address <a class="yt-timestamp" data-t="02:40:00">[02:40:00]</a>.

## Sending Bitcoin from Coinbase (Mobile Application) <a class="yt-timestamp" data-t="03:07:00">[03:07:00]</a>

1.  Access the Coinbase application on your mobile phone <a class="yt-timestamp" data-t="03:08:00">[03:08:00]</a>.
2.  Navigate to the specific cryptocurrency you wish to transfer, which is Bitcoin (BTC) <a class="yt-timestamp" data-t="03:14:00">[03:14:00]</a>.
3.  Go to the Bitcoin "Wallet" section <a class="yt-timestamp" data-t="03:20:00">[03:20:00]</a>.
4.  Look for the QR code symbol (usually in the top right corner) <a class="yt-timestamp" data-t="03:41:00">[03:41:00]</a>.
5.  Allow camera access to scan the QR code displayed on the Binance desktop <a class="yt-timestamp" data-t="03:44:00">[03:44:00]</a>. Ensure the correct currency is selected <a class="yt-timestamp" data-t="03:51:00">[03:51:00]</a>.
6.  To send the cryptocurrency, tap the paper plane symbol or arrow <a class="yt-timestamp" data-t="04:10:00">[04:10:00]</a>.
7.  Enter the desired amount to send <a class="yt-timestamp" data-t="04:15:00">[04:15:00]</a> and select "Continue" <a class="yt-timestamp" data-t="04:19:00">[04:19:00]</a>.

## Sending Bitcoin from Coinbase (Desktop) <a class="yt-timestamp" data-t="03:59:00">[03:59:00]</a>

1.  Copy the Bitcoin deposit address from Binance <a class="yt-timestamp" data-t="03:38:00">[03:38:00]</a>.
2.  On Coinbase, click "Send and Receive" <a class="yt-timestamp" data-t="01:28:00">[01:28:00]</a>.
3.  Choose the amount of Bitcoin you want to [[transferring_cryptocurrencies_to_a_bank_account | transfer]] from Coinbase to Binance <a class="yt-timestamp" data-t="04:45:00">[04:45:00]</a>. In the example, "send everything" (all Bitcoin) is chosen <a class="yt-timestamp" data-t="04:50:00">[04:50:00]</a>.
4.  Paste the copied Binance wallet address into the recipient field <a class="yt-timestamp" data-t="04:53:00">[04:53:00]</a>.
5.  Optionally, add a note for the transaction <a class="yt-timestamp" data-t="04:59:00">[04:59:00]</a>.
6.  Press "Continue" <a class="yt-timestamp" data-t="05:05:00">[05:05:00]</a>.
7.  Confirm the address, note, and the amount to send <a class="yt-timestamp" data-t="05:12:00">[05:12:00]</a>.
8.  Observe the commission fee charged. For example, from 110.39 to 109.70 euros, indicating a fee of less than one euro <a class="yt-timestamp" data-t="05:18:00">[05:18:00]</a>.
9.  Click "Send Now" <a class="yt-timestamp" data-t="05:27:00">[05:27:00]</a>.
10. Enter a verification code received on your mobile phone to complete the transaction <a class="yt-timestamp" data-t="05:30:00">[05:30:00]</a>.

## Transaction Confirmation <a class="yt-timestamp" data-t="05:36:00">[05:36:00]</a>

After sending, the balance in Coinbase will disappear <a class="yt-timestamp" data-t="05:38:00">[05:38:00]</a>. The transaction is expected to take about 10 minutes to process <a class="yt-timestamp" data-t="05:42:00">[05:42:00]</a>.

### Verifying Deposit in Binance <a class="yt-timestamp" data-t="06:05:00">[06:05:00]</a>

Once the transfer is complete:
*   You will receive a confirmation message on your mobile phone <a class="yt-timestamp" data-t="06:07:00">[06:07:00]</a>.
*   Go to the transaction history within Binance and refresh to find the completed deposit <a class="yt-timestamp" data-t="06:11:00">[06:11:00]</a>.
*   Check your "Fiat and Spot" wallet in Binance to see the transferred amount <a class="yt-timestamp" data-t="06:25:00">[06:25:00]</a>. The example shows an increase from 2.37 euros to 111.57 euros, confirming the money has been entered into Binance <a class="yt-timestamp" data-t="06:29:00">[06:29:00]</a> and is ready for use <a class="yt-timestamp" data-t="06:39:00">[06:39:00]</a>.
*   For protection and record-keeping, check your email for the transaction confirmation from Coinbase to Binance <a class="yt-timestamp" data-t="06:47:00">[06:47:00]</a>. This email can serve as collateral in case of any issues <a class="yt-timestamp" data-t="06:54:00">[06:54:00]</a>.
