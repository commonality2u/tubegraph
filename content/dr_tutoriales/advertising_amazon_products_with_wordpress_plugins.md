To generate [[generating_commissions_from_amazon_affiliates | commissions]] by promoting products from Amazon, it is possible to use WordPress plugins to advertise Amazon products directly on a website <a class="yt-timestamp" data-t="00:00:19">[00:00:19]</a>.

## WordPress Plugin Features

A specific plugin allows advertising Amazon products without requiring the API code <a class="yt-timestamp" data-t="00:00:19">[00:00:19]</a>. The API code typically becomes available after making three sales <a class="yt-timestamp" data-t="00:00:29">[00:00:29]</a>. A tutorial is available for [[installing_amazon_affiliate_plugins_on_wordpress_without_api | installing Amazon affiliate plugins on WordPress without an API]] <a class="yt-timestamp" data-t="00:00:34">[00:00:34]</a>.

After [[creating_amazon_affiliate_links | obtaining Amazon affiliate links]] through your [[using_amazon_affiliate_account | Amazon affiliate account]], these links can be integrated into your website using such a plugin <a class="yt-timestamp" data-t="00:02:41">[00:02:41]</a>.
