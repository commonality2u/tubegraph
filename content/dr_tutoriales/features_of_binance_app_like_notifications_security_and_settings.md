This article provides a tutorial on [[using_the_binance_app_on_a_mobile_phone | using the Binance app]] from a mobile phone to buy various cryptocurrencies and assets <a class="yt-timestamp" data-t="00:00:05">[00:00:05]</a>. ## Getting Started with the Binance App

The tutorial is applicable to both iPhone and Android devices, as the app offers the same functionalities across platforms <a class="yt-timestamp" data-t="00:01:10">[00:01:10]</a>.

To begin, download the Binance app from Google Play or the App Store <a class="yt-timestamp" data-t="00:01:16">[00:01:16]</a>. Once downloaded, open the app <a class="yt-timestamp" data-t="00:01:38">[00:01:38]</a> and allow notifications <a class="yt-timestamp" data-t="00:01:49">[00:01:49]</a>. Log in using your email and security verification <a class="yt-timestamp" data-t="00:01:55">[00:01:55]</a>.

## App Interface Overview

Upon logging in, the app displays a primary screen with a submenu at the bottom <a class="yt-timestamp" data-t="00:02:33">[00:02:33]</a>. This submenu includes:
*   **Home** <a class="yt-timestamp" data-t="00:02:36">[00:02:36]</a>
*   **Markets** <a class="yt-timestamp" data-t="00:02:36">[00:02:36]</a>
*   **Futures** <a class="yt-timestamp" data-t="00:02:36">[00:02:36]</a>
*   **Wallet** <a class="yt-timestamp" data-t="00:02:39">[00:02:39]</a>

The top menu includes your profile, a bell icon for alerts, and a symbol that allows access to the camera for scanning QR codes <a class="yt-timestamp" data-t="00:02:48">[00:02:48]</a>.

### Profile Menu, Security, and Settings

Accessing your profile reveals various options:
*   **Account Verification**: The account is typically verified from a previous tutorial <a class="yt-timestamp" data-t="00:03:06">[00:03:06]</a>. If not, the app provides the option to verify it <a class="yt-timestamp" data-t="00:03:12">[00:03:12]</a>.
*   **Binance Lite**: An option to switch to Binance Lite is available <a class="yt-timestamp" data-t="00:03:15">[00:03:15]</a>. This format offers fewer dropdowns and menus, simplifying the application's use <a class="yt-timestamp" data-t="00:03:22">[00:03:22]</a>.
*   **Discounts**: Users can find discounts, such as a 25% discount when paying with BNB <a class="yt-timestamp" data-t="00:03:39">[00:03:39]</a>.
*   **Rewards and Programs**: This section includes rewards, [[using_binance_referral_program_on_mobile | referral programs]], gifts, messages, payment, security, and settings <a class="yt-timestamp" data-t="00:03:49">[00:03:49]</a>.
*   **Security Settings**: Users can change security settings and passwords for their account here <a class="yt-timestamp" data-t="00:03:56">[00:03:56]</a>.
*   **Messaging**: Options to send messages to Binance support or other users for cryptocurrency trading are available <a class="yt-timestamp" data-t="00:03:58">[00:03:58]</a>.
*   **Display Settings**: A sun icon allows you to change the display configuration, such as viewing it in a different mode (e.g., dark mode) <a class="yt-timestamp" data-t="00:04:07">[00:04:07]</a>.

### Notifications

The bell icon in the top right corner displays notifications <a class="yt-timestamp" data-t="00:04:20">[00:04:20]</a>. From here, you can:
*   View all notifications received on your mobile phone <a class="yt-timestamp" data-t="00:04:26">[00:04:26]</a>.
*   Erase, mark as read, or clear notifications <a class="yt-timestamp" data-t="00:04:30">[00:04:30]</a>.

## Core App Features

### Home Screen
The home screen displays various cryptocurrencies and options <a class="yt-timestamp" data-t="00:02:40">[00:02:40]</a>. It includes options to transfer, convert, and buy different assets <a class="yt-timestamp" data-t="00:04:53">[00:04:53]</a>. Payment methods include bank transfer, Visa card purchases, and other platform-offered methods <a class="yt-timestamp" data-t="00:05:06">[00:05:06]</a>. Tapping on a currency, like BNB, directly takes you to its buying/selling menu <a class="yt-timestamp" data-t="00:05:20">[00:05:20]</a>. The home screen also shows a classification of cryptocurrency pairs <a class="yt-timestamp" data-t="00:05:39">[00:05:39]</a>.

### Markets Menu
The markets menu provides a more detailed view of all available markets <a class="yt-timestamp" data-t="00:05:53">[00:05:53]</a>. It is categorized into favorites, spot, and futures <a class="yt-timestamp" data-t="00:05:57">[00:05:57]</a>. Users can view all cryptocurrencies existing in the market and contained within the platform <a class="yt-timestamp" data-t="00:06:01">[00:06:01]</a>.
*   **Charts**: You can view charts for any cryptocurrency, adjusting the timeframe (days, hours) <a class="yt-timestamp" data-t="00:06:13">[00:06:13]</a>.
*   **Indicators**: Indicators like MACD can be inserted into the charts <a class="yt-timestamp" data-t="00:06:26">[00:06:26]</a>.
*   **Buy/Sell**: Options to buy and sell are available directly from the chart view <a class="yt-timestamp" data-t="00:06:32">[00:06:32]</a>.
*   **Favorites**: Cryptocurrencies can be saved as favorites using the star icon <a class="yt-timestamp" data-t="00:06:41">[00:06:41]</a>.

### Trade Menu
The trade menu allows direct and fast trading of different assets <a class="yt-timestamp" data-t="00:07:01">[00:07:01]</a>. For a given asset pair (e.g., Bitcoin against USDT):
*   **Order Types**: Users can place limited orders, market orders, stop-loss orders, and OCO (One Cancels the Other) orders <a class="yt-timestamp" data-t="00:07:25">[00:07:25]</a>.
*   **Amount**: The amount of the asset to be traded can be specified, and the corresponding cost in USD is displayed <a class="yt-timestamp" data-t="00:07:29">[00:07:29]</a>.
*   **Asset Selection**: Assets can be easily changed by clicking on the current asset displayed <a class="yt-timestamp" data-t="00:07:47">[00:07:47]</a>.

### Futures Menu
Similar to the spot market, the futures menu allows trading in cryptocurrency futures <a class="yt-timestamp" data-t="00:08:10">[00:08:10]</a>. For more information, refer to [[how_to_use_binance_futures_on_a_mobile_device | How to Use Binance Futures on a Mobile Device]].

### Wallet Menu
The wallet menu provides an [[understanding_binance_wallet_types | overview of your assets]] <a class="yt-timestamp" data-t="00:08:22">[00:08:22]</a>.
*   **Portfolio View**: Shows your portfolio, though it may be censored for privacy <a class="yt-timestamp" data-t="00:08:26">[00:08:26]</a>.
*   **Actions**: Options to deposit, withdraw, and transfer Bitcoins, cryptocurrencies, or funds are available <a class="yt-timestamp" data-t="00:08:31">[00:08:31]</a>.
*   **Wallet Sections**: The wallet is divided into summaries, spot, margin, and futures <a class="yt-timestamp" data-t="00:08:41">[00:08:41]</a>.
    *   **Spot**: Displays profitability, assets, and balances based on purchases <a class="yt-timestamp" data-t="00:08:47">[00:08:47]</a>. It shows the amount bought and daily profitability <a class="yt-timestamp" data-t="00:09:02">[00:09:02]</a>.
    *   **Margin**: For leveraged trading, it shows features related to operations, risk, and correctness of actions <a class="yt-timestamp" data-t="00:09:11">[00:09:11]</a>.
    *   **P2P**: Also accessible from the wallet menu <a class="yt-timestamp" data-t="00:09:22">[00:09:22]</a>.

These functionalities enable users to [[using_binance_mobile_app_for_trading | operate and trade]] easily from their mobile devices <a class="yt-timestamp" data-t="00:08:01">[00:08:01]</a>. For a broader understanding of navigating the app, see [[navigating_the_binance_app_interface | Navigating the Binance App Interface]].
