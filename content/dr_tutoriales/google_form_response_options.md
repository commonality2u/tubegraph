This article provides a guide on the various [[customizing_survey_questions_and_responses | options for responses]] within [[how_to_create_a_google_form | Google Forms]], covering question types, collection [[google_forms_configuration_settings | settings]], and submission processes.

## Accessing Google Forms <a class="yt-timestamp" data-t="00:00:00">[00:00:00]</a>

To begin, ensure you are logged into your Google account <a class="yt-timestamp" data-t="00:00:19">[00:00:19]</a>. You can access [[how_to_create_a_google_form | Google Forms]] by navigating to the Google applications menu on the right side of your screen and locating the purple "Forms" logo <a class="yt-timestamp" data-t="00:00:24">[00:00:24]</a>. Upon access, you can choose to create a new blank [[how_to_create_a_google_form | form]] or select from available templates like contact data or party invitations <a class="yt-timestamp" data-t="00:00:36">[00:00:36]</a>.

## [[customizing_google_form_questions | Customizing Questions]] and [[viewing_responses_in_google_forms | Response]] Types <a class="yt-timestamp" data-t="00:01:07">[00:01:07]</a>

When [[how_to_create_a_google_form | creating a Google Form]], you can define various [[customizing_google_form_questions | question]] types to solicit different kinds of [[viewing_responses_in_google_forms | responses]]:

*   **Short Answer/Paragraph**: Used for brief text [[viewing_responses_in_google_forms | responses]] (e.g., "name" or "Gmail") <a class="yt-timestamp" data-t="00:01:09">[00:01:09]</a> or longer explanations (e.g., "explain your doubts") <a class="yt-timestamp" data-t="00:01:15">[00:01:15]</a>. The [[how_to_create_a_google_form | form]] can automatically detect if a long [[viewing_responses_in_google_forms | response]] is needed based on the [[customizing_google_form_questions | question]]'s wording <a class="yt-timestamp" data-t="00:01:48">[00:01:48]</a>.
*   **Multiple Options (Multiple Choice)**: Allows respondents to select one option from a predefined list <a class="yt-timestamp" data-t="00:01:17">[00:01:17]</a>, such as asking if they have subscribed to a channel or liked a video <a class="yt-timestamp" data-t="00:02:10">[00:02:10]</a>.
*   **Checkboxes**: Enables respondents to select multiple options from a list <a class="yt-timestamp" data-t="00:02:42">[00:02:42]</a>.
*   **Dropdown**: Provides a compact list of options where respondents select one, similar to multiple choice but displayed differently <a class="yt-timestamp" data-t="00:02:46">[00:02:46]</a>.
*   **File Upload**: Allows respondents to upload files, such as screenshots, directly to Google Drive. You can specify whether it's mandatory, limit file types, set the maximum number of files, and define the maximum file size (e.g., 100MB) <a class="yt-timestamp" data-t="00:02:56">[00:02:56]</a>. If this option is enabled, embedding the [[how_to_create_a_google_form | form]] with HTML code is not possible <a class="yt-timestamp" data-t="00:03:32">[00:03:32]</a>.
*   **Date and Time**: Useful for scheduling or asking for specific dates and times, such as for meetings <a class="yt-timestamp" data-t="00:03:54">[00:03:54]</a>.
*   **Linear Scale**: Enables respondents to rate something on a numerical scale (e.g., 1 to 5), with custom labels for the minimum and maximum values (e.g., "very dissatisfied" to "very satisfied" for tutorial satisfaction) <a class="yt-timestamp" data-t="00:04:18">[00:04:18]</a>.
*   **Mandatory Questions**: Any [[customizing_google_form_questions | question]] can be marked as mandatory, requiring a [[viewing_responses_in_google_forms | response]] before the [[how_to_create_a_google_form | form]] can be submitted <a class="yt-timestamp" data-t="00:01:23">[00:01:23]</a>. Mandatory [[customizing_google_form_questions | questions]] are indicated by an asterisk in the preview <a class="yt-timestamp" data-t="00:07:37">[00:07:37]</a>.

## [[google_forms_configuration_settings | Configuration Settings]] for [[viewing_responses_in_google_forms | Responses]] <a class="yt-timestamp" data-t="00:05:40">[00:05:40]</a>

The [[google_forms_configuration_settings | configuration settings]] for [[viewing_responses_in_google_forms | responses]] offer control over how submissions are handled:

*   **Collecting Email Addresses**: You can set the [[how_to_create_a_google_form | form]] to automatically collect respondents' email addresses <a class="yt-timestamp" data-t="00:05:49">[00:05:49]</a>. This is distinct from asking for email via a [[customizing_google_form_questions | question]] <a class="yt-timestamp" data-t="00:05:58">[00:05:58]</a>.
*   **Allowing [[viewing_responses_in_google_forms | Response]] Edits**: You can allow respondents to edit their [[viewing_responses_in_google_forms | answers]] after submission <a class="yt-timestamp" data-t="00:06:12">[00:06:12]</a>.
*   **Limiting [[viewing_responses_in_google_forms | Responses]]**: It's possible to limit a [[how_to_create_a_google_form | form]] to one [[viewing_responses_in_google_forms | response]] per user, though this might not be suitable for all [[how_to_create_a_google_form | forms]] <a class="yt-timestamp" data-t="00:06:15">[00:06:15]</a>.
*   **Total File Size Limit**: For [[how_to_create_a_google_form | forms]] with file uploads, a total size limit for all uploaded files (e.g., 1 GB) can be set <a class="yt-timestamp" data-t="00:06:28">[00:06:28]</a>.

## [[visualizing_and_analyzing_google_form_responses | Submission]] and Confirmation <a class="yt-timestamp" data-t="00:06:35">[00:06:35]</a>

Manage how the [[how_to_create_a_google_form | form]] is submitted and how respondents are informed:

*   **Display Progress Bar**: Shows respondents a progress bar as they fill out the [[how_to_create_a_google_form | form]] <a class="yt-timestamp" data-t="00:06:40">[00:06:40]</a>.
*   **Confirmation Message**: Customize the message displayed after a successful submission (e.g., "Reply sent Correct. Subscribe") <a class="yt-timestamp" data-t="00:06:48">[00:06:48]</a>. The default message is "Your [[viewing_responses_in_google_forms | response]] is recorded" <a class="yt-timestamp" data-t="00:06:51">[00:06:51]</a>.
*   **Link to Submit Another [[viewing_responses_in_google_forms | Response]]**: Provide an option for respondents to submit the [[how_to_create_a_google_form | form]] again <a class="yt-timestamp" data-t="00:07:01">[00:07:01]</a>.

## Sharing and Embedding [[viewing_responses_in_google_forms | Response]] Forms <a class="yt-timestamp" data-t="00:07:28">[00:07:28]</a>

After [[how_to_create_a_google_form | creating]] your [[how_to_create_a_google_form | form]], you can share it:

*   **Via Email**: Invite specific people by email <a class="yt-timestamp" data-t="00:08:08">[00:08:08]</a>. Be aware that your email will appear as the sender <a class="yt-timestamp" data-t="00:08:50">[00:08:50]</a>.
*   **Via Link**: Generate a shareable link, which can be shortened <a class="yt-timestamp" data-t="00:08:10">[00:08:10]</a>. This is recommended for anonymous [[creating_surveys_in_google_forms | surveys]] <a class="yt-timestamp" data-t="00:08:54">[00:08:54]</a>.
*   **Embed HTML**: Embed the [[how_to_create_a_google_form | form]] directly into a webpage using HTML code <a class="yt-timestamp" data-t="00:08:25">[00:08:25]</a>. Note that this option is unavailable if the [[how_to_create_a_google_form | form]] includes a file upload [[customizing_google_form_questions | question]] <a class="yt-timestamp" data-t="00:03:32">[00:03:32]</a>.

You can [[customizing_survey_questions_and_responses | customize]] the visual theme of your [[how_to_create_a_google_form | form]], including changing the header, fonts, and colors to enhance its appeal <a class="yt-timestamp" data-t="00:07:51">[00:07:51]</a>.
