This article explains how to manage and configure your default search engine within [[setting_google_as_default_search_engine_in_google_chrome | Google Chrome]] <a class="yt-timestamp" data-t="00:00:31">[00:00:31]</a>. While Google is often the default <a class="yt-timestamp" data-t="00:00:34">[00:00:34]</a>, your browser might use a different search engine like Yahoo <a class="yt-timestamp" data-t="00:00:49">[00:00:49]</a>, Bing, or Ecosia <a class="yt-timestamp" data-t="00:02:00">[00:02:00]</a>. This often occurs inadvertently when installing browser extensions or other software <a class="yt-timestamp" data-t="00:01:31">[00:01:31]</a>, <a class="yt-timestamp" data-t="00:01:33">[00:01:33]</a>.

## Accessing Search Engine Settings

To adjust your search engine settings:
1.  Open [[setting_google_as_default_search_engine_in_google_chrome | Google Chrome]] <a class="yt-timestamp" data-t="00:00:31">[00:00:31]</a>.
2.  Click the menu icon located in the top-right corner of the browser window <a class="yt-timestamp" data-t="00:01:02">[00:01:02]</a>.
3.  Navigate to "Settings" (or "Configuraci√≥n" in Spanish) <a class="yt-timestamp" data-t="00:01:08">[00:01:08]</a>, <a class="yt-timestamp" data-t="00:01:10">[00:01:10]</a>.
4.  Within the settings tab, locate and select the "Search engine" section <a class="yt-timestamp" data-t="00:01:14">[00:01:14]</a>, <a class="yt-timestamp" data-t="00:01:17">[00:01:17]</a>.

## Managing Search Engines

Under the "Search engine" section, you will find options to manage your search engines <a class="yt-timestamp" data-t="00:01:44">[00:01:44]</a>.

### Default Search Engine
The first option allows you to select your preferred default search engine <a class="yt-timestamp" data-t="00:01:50">[00:01:50]</a>. You can choose from the available options, such as Google, Yahoo, Bing, or others <a class="yt-timestamp" data-t="00:01:24">[00:01:24]</a>, <a class="yt-timestamp" data-t="00:02:00">[00:02:00]</a>.

### Editing and Adding New Search Engines
To manage the list of available search engines, click on "Manage Search Engines" <a class="yt-timestamp" data-t="00:01:44">[00:01:44]</a>. Here, you will see all currently installed search engines <a class="yt-timestamp" data-t="00:01:47">[00:01:47]</a>.

*   **Editing an existing search engine:** You can edit the name or URL of an existing search engine by clicking the pencil icon next to it <a class="yt-timestamp" data-t="00:02:05">[00:02:05]</a>, <a class="yt-timestamp" data-t="00:02:06">[00:02:06]</a>.
*   **Adding Google as a search engine:** If Google is not listed, you can manually add it:
    1.  Click the "Add" button (represented by a pencil icon) <a class="yt-timestamp" data-t="00:02:18">[00:02:18]</a>.
    2.  Enter "Google" for the name <a class="yt-timestamp" data-t="00:02:18">[00:02:18]</a>.
    3.  For the shortcut, enter "Google.com" <a class="yt-timestamp" data-t="00:02:22">[00:02:22]</a>.
    4.  It is crucial to use `.com` for the domain (e.g., `Google.com`) rather than country-specific domains like `.es` or `.uk`, as these might cause issues if you are not in that specific country <a class="yt-timestamp" data-t="00:02:26">[00:02:26]</a>, <a class="yt-timestamp" data-t="00:02:28">[00:02:28]</a>, <a class="yt-timestamp" data-t="00:02:30">[00:02:30]</a>, <a class="yt-timestamp" data-t="00:02:33">[00:02:33]</a>.
    5.  Once entered, click "Save" <a class="yt-timestamp" data-t="00:02:36">[00:02:36]</a>, <a class="yt-timestamp" data-t="00:02:38">[00:02:38]</a>.

After making changes, there's no need to save anything else within the settings window; the changes are applied automatically <a class="yt-timestamp" data-t="00:02:49">[00:02:49]</a>.
