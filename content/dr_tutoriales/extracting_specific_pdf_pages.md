## [[separating_pdf_pages_using_google_chrome | Separating PDF Pages Using Google Chrome]]

This method is straightforward and does not require external programs <a class="yt-timestamp" data-t="00:01:40">[00:01:40]</a>.

### Steps
1.  **Open the PDF with Google Chrome**: Navigate to the PDF file on your computer and right-click to choose "Open with" > "Google Chrome" <a class="yt-timestamp" data-t="00:00:28">[00:00:28]</a>. The PDF will open in a Chrome tab, displaying its pages <a class="yt-timestamp" data-t="00:00:36">[00:00:36]</a>.
2.  **Access the Print Function**: Click on the printer icon, typically located at the top of the Chrome window <a class="yt-timestamp" data-t="00:00:49">[00:00:49]</a>.
3.  **Select Custom Pages**: In the print dialog, change the "Pages" setting from "All" to "Custom" <a class="yt-timestamp" data-t="00:00:52">[00:00:52]</a>.
4.  **Specify Desired Pages**: Enter the page numbers you wish to extract. For example, to extract pages 3 and 4, you would type `3,4` <a class="yt-timestamp" data-t="00:00:58">[00:00:58]</a>.
5.  **[[saving_and_downloading_edited_pdfs | Save the New PDF]]**: Click "Print" <a class="yt-timestamp" data-t="00:01:13">[00:01:13]</a>. You will then be prompted to choose a location to [[saving_and_downloading_edited_pdfs | save]] the new PDF containing only the selected pages <a class="yt-timestamp" data-t="00:01:15">[00:01:15]</a>. You can rename the file, for instance, "cropped PDF" <a class="yt-timestamp" data-t="00:01:17">[00:01:17]</a>.
6.  **Verify Extraction**: Open the newly saved PDF to confirm that it contains only the specified pages <a class="yt-timestamp" data-t="00:01:34">[00:01:34]</a>.

## [[using_pdf_tools_to_split_pdf_documents | Using I Love PDF]]

This method utilizes an online tool to [[using_pdf_tools_to_split_pdf_documents | split PDF documents]] <a class="yt-timestamp" data-t="00:01:45">[00:01:45]</a>.

### Steps
1.  **Access I Love PDF**: Open Google Chrome (or another web browser) and search for "I Love PDF" <a class="yt-timestamp" data-t="00:01:48">[00:01:48]</a>. This is an online platform for working with PDFs <a class="yt-timestamp" data-t="00:01:53">[00:01:53]</a>.
2.  **Choose "Split PDF"**: On the I Love PDF website, select the "Split PDF" option <a class="yt-timestamp" data-t="00:01:57">[00:01:57]</a>.
3.  **Upload Your PDF**: Click to choose the PDF file you wish to split <a class="yt-timestamp" data-t="00:02:01">[00:02:01]</a>. After selecting, click "Open" <a class="yt-timestamp" data-t="00:02:08">[00:02:08]</a>.
4.  **Extract Pages**: Once uploaded, you'll see options to [[using_pdf_tools_to_split_pdf_documents | extract pages]] <a class="yt-timestamp" data-t="00:02:11">[00:02:11]</a>. You can select specific pages <a class="yt-timestamp" data-t="00:02:19">[00:02:19]</a> or choose to split all pages individually <a class="yt-timestamp" data-t="00:02:22">[00:02:22]</a>.
5.  **[[saving_and_downloading_edited_pdfs | Download the Separated PDFs]]**: After processing, click "Download" <a class="yt-timestamp" data-t="00:02:27">[00:02:27]</a>. The separated PDF pages will be downloaded as a compressed WinRAR file <a class="yt-timestamp" data-t="00:02:30">[00:02:30]</a>.
6.  **[[saving_and_managing_extracted_pdf_pages | Manage Extracted Pages]]**: Unzip the downloaded file to access all the individual PDF pages <a class="yt-timestamp" data-t="00:02:34">[00:02:34]</a>. Each page will be a separate PDF file <a class="yt-timestamp" data-t="00:02:37">[00:02:37]</a>. For example, a 28-page PDF would yield 28 individual PDF files <a class="yt-timestamp" data-t="00:02:39">[00:02:39]</a>. You can open any of these files to view the corresponding page <a class="yt-timestamp" data-t="00:02:44">[00:02:44]</a>.
