[[how_to_use_trust_wallet_on_android_and_ios | Trust Wallet]] is a cryptocurrency application available on both Android and iOS devices, offering identical functionalities across platforms <a class="yt-timestamp" data-t="00:00:09">[00:00:09]</a>.

## Getting Started

The first step is to download the [[how_to_use_trust_wallet_on_android_and_ios | Trust Wallet]] application <a class="yt-timestamp" data-t="00:00:54">[00:00:54]</a>.

## Creating a New Wallet

When you first open the [[how_to_use_trust_wallet_on_android_and_ios | Trust Wallet]] application, you will be presented with the option to either import an existing wallet or create a new one <a class="yt-timestamp" data-t="00:01:00">[00:01:00]</a>. To create a new wallet:

1.  Select the option to "create a new wallet" <a class="yt-timestamp" data-t="00:01:03">[00:01:03]</a>.
2.  Accept the privacy policy <a class="yt-timestamp" data-t="00:01:17">[00:01:17]</a>.
3.  Enter an access code (password) twice <a class="yt-timestamp" data-t="00:01:21">[00:01:21]</a>.
4.  The application will prompt you to make a copy of your wallet's security phrase <a class="yt-timestamp" data-t="00:01:29">[00:01:29]</a>. This is typically a 12-word secret phrase <a class="yt-timestamp" data-t="00:01:33">[00:01:33]</a>.
5.  Accept the conditions, which state that if you lose this phrase, you will lose access to your funds, and if you share it, your funds can be stolen <a class="yt-timestamp" data-t="00:01:35">[00:01:35]</a>. [[how_to_use_trust_wallet_on_android_and_ios | Trust Wallet]] is not responsible for lost funds in these scenarios <a class="yt-timestamp" data-t="00:01:43">[00:01:43]</a>.
6.  Copy the secret phrase down in a secure location <a class="yt-timestamp" data-t="00:01:56">[00:01:56]</a>. It is highly recommended to copy it onto physical paper, as a PC is vulnerable to hacking <a class="yt-timestamp" data-t="00:01:57">[00:01:57]</a>. This process is crucial for [[backing_up_and_securing_your_wallet | backing up and securing your wallet]] <a class="yt-timestamp" data-t="00:01:29">[00:01:29]</a>.
7.  After copying, select "continue" <a class="yt-timestamp" data-t="00:02:10">[00:02:10]</a>.
8.  You will then need to put the words in the correct order as a verification process <a class="yt-timestamp" data-t="00:02:13">[00:02:13]</a>.
9.  Once verified, your wallet will be successfully created <a class="yt-timestamp" data-t="00:02:21">[00:02:21]</a>.

## Importing an Existing Wallet

If you already possess a wallet, you can easily import it into [[how_to_use_trust_wallet_on_android_and_ios | Trust Wallet]]:

1.  Choose the "import a wallet" option <a class="yt-timestamp" data-t="00:01:01">[00:01:01]</a>.
2.  You will need to input your existing wallet's secret phrase (the words) in the correct order <a class="yt-timestamp" data-t="00:01:06">[00:01:06]</a>.
3.  After entering the phrase, you will be prompted to set a password for the wallet <a class="yt-timestamp" data-t="00:01:11">[00:01:11]</a>.

### Managing Multiple Wallets

[[how_to_use_trust_wallet_on_android_and_ios | Trust Wallet]] allows you to [[managing_cryptocurrency_wallets | manage multiple wallets]] from a single mobile device <a class="yt-timestamp" data-t="00:06:02">[00:06:02]</a>. You can add new wallets through the "Wallets" section in the settings <a class="yt-timestamp" data-t="00:05:57">[00:05:57]</a>.

For more details on security and other preferences, refer to [[trust_wallet_settings_and_security_preferences | Trust Wallet settings and security preferences]] <a class="yt-timestamp" data-t="00:05:42">[00:05:42]</a>.
