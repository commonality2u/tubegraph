Excel allows for advanced text formatting, including the use of superscripts and subscripts, which are crucial for displaying scientific notation, chemical formulas, or specific mathematical expressions correctly <a class="yt-timestamp" data-t="00:00:12">[00:00:12]</a>. While traditional methods of indicating these, like using arrows, exist <a class="yt-timestamp" data-t="00:00:44">[00:00:44]</a>, Excel provides dedicated formatting options for accurate representation.

## Applying Superscripts

To apply a [[using_superscripts_in_excel | superscript]] to a number or character within a cell:
1.  Select the specific number or character you wish to elevate <a class="yt-timestamp" data-t="00:01:10">[00:01:10]</a>.
2.  Navigate to the "Format Cells" dialog box, often accessible via a small arrow icon in the Font group on the Home tab <a class="yt-timestamp" data-t="00:01:16">[00:01:16]</a>.
3.  Within this dialog, check the "Superscript" option <a class="yt-timestamp" data-t="00:01:26">[00:01:26]</a>.
4.  After applying, the selected text will appear above the baseline in the cell, though it might look normal in the input bar <a class="yt-timestamp" data-t="00:01:43">[00:01:43]</a>, <a class="yt-timestamp" data-t="00:02:00">[00:02:00]</a>. Remember to return to normal formatting for subsequent text if needed <a class="yt-timestamp" data-t="00:01:47">[00:01:47]</a>.

### Example in Graphs
This formatting can also be applied to values displayed within charts and graphs in Excel to improve readability <a class="yt-timestamp" data-t="00:02:09">[00:02:09]</a>. For instance, a density value like "kg/m3" can have the "3" displayed as a [[using_superscripts_in_excel | superscript]] within the graph's axis label or data series <a class="yt-timestamp" data-t="00:02:22">[00:02:22]</a>.

## Applying Subscripts

Similar to [[using_superscripts_in_excel | superscripts]], [[using_subscripts_in_excel | subscripts]] can be applied to position text below the baseline:
1.  Select the character or number that needs to be a [[using_subscripts_in_excel | subscript]] <a class="yt-timestamp" data-t="00:02:49">[00:02:49]</a>.
2.  Open the "Format Cells" dialog.
3.  Select the "Subscript" option <a class="yt-timestamp" data-t="00:02:42">[00:02:42]</a>.
4.  The selected text will then appear below the baseline in the cell <a class="yt-timestamp" data-t="00:02:53">[00:02:53]</a>, <a class="yt-timestamp" data-t="00:02:59">[00:02:59]</a>.
