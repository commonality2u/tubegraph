This guide outlines the process of merging multiple PDF files into a single document, using an online tool without requiring any software installation <a class="yt-timestamp" data-t="00:00:17">[00:00:17]</a>.

## Prerequisites

To begin, you will need:
*   A computer <a class="yt-timestamp" data-t="00:00:14">[00:00:14]</a>
*   Google Chrome browser <a class="yt-timestamp" data-t="00:00:14">[00:00:14]</a>
*   The PDF files you wish to merge <a class="yt-timestamp" data-t="00:00:19">[00:00:19]</a>

## Step-by-Step Guide to Merging PDFs

### 1. Open Google Chrome
Start by opening the Google Chrome browser on your computer <a class="yt-timestamp" data-t="00:00:14">[00:00:14]</a>.

### 2. Access I Love PDF
Navigate to the [[using_i_love_pdf_for_merging_pdfs | I Love PDF]] website <a class="yt-timestamp" data-t="00:00:46">[00:00:46]</a>. This tool is accessible via a direct link often found in the first comment of related tutorials <a class="yt-timestamp" data-t="00:00:43">[00:00:43]</a>.

### 3. Select "Merge PDF" Option
Once on the [[using_i_love_pdf_for_merging_pdfs | I Love PDF]] homepage, choose the "Merge PDF" or "Join PDF" option <a class="yt-timestamp" data-t="00:00:49">[00:00:49]</a>.

### 4. Upload Your PDF Files
Click the "Select PDF files" button and browse to the location of your PDF documents <a class="yt-timestamp" data-t="01:14">[01:14]</a>. For example, you might select a "Safari" ticket and a "Multicinema" ticket <a class="yt-timestamp" data-t="01:22">[01:22]</a>. It is crucial to select all the PDF files you intend to merge <a class="yt-timestamp" data-t="01:29">[01:29]</a>. After selecting, click "Open" <a class="yt-timestamp" data-t="01:29">[01:29]</a>. The selected PDFs will then load into the browser <a class="yt-timestamp" data-t="01:38">[01:38]</a>.

### 5. Initiate the Merge Process
With your PDFs loaded, click the "Merge PDF" or "Join PDF" button <a class="yt-timestamp" data-t="01:38">[01:38]</a>.

### 6. [[downloading_and_managing_merged_pdfs | Download the Merged PDF]]
After the merging process is complete, you will be able to [[downloading_and_managing_merged_pdfs | download the merged PDF]] file <a class="yt-timestamp" data-t="01:43">[01:43]</a>. The merged file will combine all the original PDFs into a single document <a class="yt-timestamp" data-t="01:50">[01:50]</a>.

### 7. Manage the Downloaded File
The newly merged PDF will typically be found in your browser's "Downloads" section <a class="yt-timestamp" data-t="01:55">[01:55]</a>. It will often have a descriptive name, such as "Multicinema Safari Merge" <a class="yt-timestamp" data-t="01:55">[01:55]</a>. You can easily [[downloading_and_managing_merged_pdfs | edit the name]] of the file whenever you wish <a class="yt-timestamp" data-t="01:57">[01:57]</a>. Opening the file will show all the original documents combined <a class="yt-timestamp" data-t="02:00">[02:00]</a>.
