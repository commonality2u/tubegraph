Outlook allows users to easily create and manage email signatures for various purposes <a class="yt-timestamp" data-t="00:00:00">[00:00:00]</a>. These signatures can include text, images, and custom formatting <a class="yt-timestamp" data-t="00:01:14">[00:01:14]</a>.

## Accessing Signature Settings

To change or create a signature in Outlook:
1.  Open Outlook <a class="yt-timestamp" data-t="00:00:15">[00:00:15]</a>.
2.  Navigate to the top menu <a class="yt-timestamp" data-t="00:00:24">[00:00:24]</a>.
3.  Select "Options" <a class="yt-timestamp" data-t="00:00:31">[00:00:31]</a>.
4.  In the options menu, go to the "Mail" or "Email" section <a class="yt-timestamp" data-t="00:00:35">[00:00:35]</a>.
5.  Find and select "Signatures" (or "Firmas" in Spanish) <a class="yt-timestamp" data-t="00:00:40">[00:00:40]</a>. This option allows you to create or modify signatures <a class="yt-timestamp" data-t="00:00:43">[00:00:43]</a>.

## Creating and Customizing Signatures

Outlook enables users to [[creating_a_new_signature_with_images_and_text_in_outlook | create a new signature]] with various elements <a class="yt-timestamp" data-t="00:00:49">[00:00:49]</a>:
1.  Click "New" in the Signatures window to create a new signature <a class="yt-timestamp" data-t="00:01:00">[00:01:00]</a>.
2.  Provide a name for your new signature <a class="yt-timestamp" data-t="00:01:02">[00:01:02]</a>.
3.  Click "OK" to confirm the name <a class="yt-timestamp" data-t="00:01:09">[00:01:09]</a>.
4.  In the editing area, you can type your desired signature text <a class="yt-timestamp" data-t="00:01:12">[00:01:12]</a>.
5.  You can import images and apply various text formats to [[exploring_customization_options_for_outlook_email_signatures | customize your Outlook email signatures]] <a class="yt-timestamp" data-t="00:01:14">[00:01:14]</a>. For example, a signature can include a name, company logo, email address, and phone number <a class="yt-timestamp" data-t="00:01:28">[00:01:28]</a>.
6.  Once you have designed your signature, click "Save" <a class="yt-timestamp" data-t="00:01:36">[00:01:36]</a>.

## Configuring Signature Usage

After creating signatures, you can [[configuring_signatures_for_different_email_accounts_in_outlook | configure them for different email accounts]] and message types <a class="yt-timestamp" data-t="00:01:58">[00:01:58]</a>:
*   **Email Account Selection:** Choose which email account the signature should be associated with, especially if you have multiple accounts <a class="yt-timestamp" data-t="00:01:58">[00:01:58]</a>.
*   **New Messages:** Select which signature to use for new emails <a class="yt-timestamp" data-t="00:02:02">[00:02:02]</a>.
*   **Replies and Forwards:** Configure a separate signature (or no signature) to be used for [[using_signatures_for_replies_and_forwarded_emails_in_outlook | replies and forwarded emails]] <a class="yt-timestamp" data-t="00:02:05">[00:02:05]</a>.
*   After making your selections, click "OK" to apply the changes <a class="yt-timestamp" data-t="00:02:32">[00:02:32]</a>.

## Verifying Your Signature

To confirm your signature is set up correctly:
*   Compose a new email, and your default signature should automatically appear <a class="yt-timestamp" data-t="00:02:38">[00:02:38]</a>.
*   If you reply or forward an email, the configured signature for those actions will be displayed <a class="yt-timestamp" data-t="00:02:50">[00:02:50]</a>.
