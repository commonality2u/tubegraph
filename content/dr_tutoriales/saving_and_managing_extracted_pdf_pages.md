When you need to [[extracting_specific_pdf_pages | separate specific pages]] from a PDF document, there are two primary methodologies you can use: one involves leveraging your web browser, and the other utilizes an online PDF tool. Both methods allow you to [[saving_and_downloading_edited_pdfs | save and download the edited PDFs]] <a class="yt-timestamp" data-t="00:00:05">[00:00:05]</a>.

## Method 1: [[separating_pdf_pages_using_google_chrome | Separating PDF Pages Using Google Chrome]]

This method is simple and does not require external programs <a class="yt-timestamp" data-t="00:01:43">[00:01:43]</a>.

### Steps:
1.  **Open the PDF with Google Chrome**: Instead of opening the PDF with Adobe Acrobat, choose "open with" Google Chrome <a class="yt-timestamp" data-t="00:00:28">[00:00:28]</a>. The PDF will open with its different pages visible in a Chrome tab <a class="yt-timestamp" data-t="00:00:36">[00:00:36]</a>.
2.  **Access the Print Function**: Click the printer button located at the top of the browser <a class="yt-timestamp" data-t="00:00:49">[00:00:49]</a>.
3.  **Select Custom Pages**: In the print dialogue, change the "Pages" setting from "all" to "Custom" <a class="yt-timestamp" data-t="00:00:52">[00:00:52]</a>.
4.  **Specify Desired Pages**: Enter the specific pages you wish to extract. For example, to extract pages 3 and 4, you would type "3,4" <a class="yt-timestamp" data-t="00:01:03">[00:01:03]</a>.
5.  **Print/Save**: Click "print" <a class="yt-timestamp" data-t="00:01:13">[00:01:13]</a>. This will prompt you to choose a location to store the new PDF <a class="yt-timestamp" data-t="00:01:15">[00:01:15]</a>.
6.  **Rename and Save**: Give the file a descriptive name, such as "cropped PDF," and click "save" <a class="yt-timestamp" data-t="00:01:17">[00:01:17]</a>.
7.  **Verification**: Open the newly saved PDF to confirm it contains only the selected pages <a class="yt-timestamp" data-t="00:01:34">[00:01:34]</a>.

## Method 2: [[using_pdf_tools_to_split_pdf_documents | Using PDF Tools to Split PDF Documents]] with iLovePDF

This method utilizes an online service to split and manage PDF pages.

### Steps:
1.  **Access iLovePDF**: Open Google Chrome and search for "iLovePDF" <a class="yt-timestamp" data-t="00:01:48">[00:01:48]</a>.
2.  **Select "Split PDF"**: On the iLovePDF website, choose the "Split PDF" option <a class="yt-timestamp" data-t="00:01:57">[00:01:57]</a>.
3.  **Upload the PDF**: Select the PDF document you wish to split <a class="yt-timestamp" data-t="00:02:01">[00:02:01]</a>. For example, if you have a 28-page document, upload it <a class="yt-timestamp" data-t="00:02:08">[00:02:08]</a>.
4.  **Extract Pages**: Click "Extract Pages" or "Split PDF" <a class="yt-timestamp" data-t="00:02:11">[00:02:11]</a>. You can choose to extract all pages individually <a class="yt-timestamp" data-t="00:02:22">[00:02:22]</a>.
5.  **Download Extracted Pages**: Click "download" <a class="yt-timestamp" data-t="00:02:27">[00:02:27]</a>. The extracted pages will typically download as a compressed file (e.g., WinRAR) <a class="yt-timestamp" data-t="00:02:30">[00:02:30]</a>.
6.  **Access Individual Pages**: Open the compressed file. You will see all the individual pages of the original PDF <a class="yt-timestamp" data-t="00:02:34">[00:02:34]</a>. Each page will be a separate PDF file within the archive <a class="yt-timestamp" data-t="00:02:37">[00:02:37]</a>. You can open any individual page by clicking on it <a class="yt-timestamp" data-t="00:02:44">[00:02:44]</a>.
