When creating a new YouTube channel, there are certain features that are not enabled by default, such as the ability to upload videos longer than 15 minutes <a class="yt-timestamp" data-t="00:00:15">[00:00:15]</a>. To [[growing_your_youtube_channel | grow your YouTube channel]] and utilize its full capabilities, you will need to activate these features <a class="yt-timestamp" data-t="00:00:27">[00:00:27]</a>.

## Features Requiring Verification

For a newly [[Creating and managing YouTube channel | created YouTube channel]], some advanced features require phone verification. These include:
*   Uploading videos longer than 15 minutes <a class="yt-timestamp" data-t="00:01:17">[00:01:17]</a>
*   Using custom thumbnails <a class="yt-timestamp" data-t="00:01:18">[00:01:18]</a>
*   Initiating live broadcasts <a class="yt-timestamp" data-t="00:01:19">[00:01:19]</a>

Basic functions like uploading videos, [[Creating YouTube playlists | creating playlists]], and adding video collaborations are typically enabled by default <a class="yt-timestamp" data-t="00:01:07">[00:01:07]</a>.

## Steps to Activate Features

To activate these advanced features, you need to perform a phone verification within [[Using YouTube Studio for video uploads | YouTube Studio]] <a class="yt-timestamp" data-t="00:01:21">[00:01:21]</a>.

1.  **Access YouTube Studio**: From your YouTube channel, directly access [[Using YouTube Studio for video uploads | YouTube Studio]] <a class="yt-timestamp" data-t="00:00:47">[00:00:47]</a>.
2.  **Navigate to Settings**: Go to the "Configuration" (Settings) section <a class="yt-timestamp" data-t="00:00:58">[00:00:58]</a>.
3.  **Select Channel Settings**: Within "Configuration," select the "Channel" sub-tab <a class="yt-timestamp" data-t="00:01:01">[00:01:01]</a>.
4.  **Access Feature Eligibility**: Go to the "Access to Functions" (Feature Eligibility) part <a class="yt-timestamp" data-t="00:01:05">[00:01:05]</a>.
5.  **Initiate Phone Verification**: Here, you will find "Functions that require verification by phone" <a class="yt-timestamp" data-t="00:01:14">[00:01:14]</a>. It will be marked as "available" but require verification <a class="yt-timestamp" data-t="00:01:21">[00:01:21]</a>. Select the option to verify the phone number <a class="yt-timestamp" data-t="00:01:30">[00:01:30]</a>.
6.  **Enter Phone Number**: You will need to select your country and choose how you prefer to receive the verification code (text message or call) <a class="yt-timestamp" data-t="00:01:51">[00:01:51]</a>. Then, enter your phone number <a class="yt-timestamp" data-t="00:01:57">[00:01:57]</a>.
7.  **Submit for Verification**: Click "Send" <a class="yt-timestamp" data-t="00:02:01">[00:02:01]</a>. YouTube will send a verification code to the provided number <a class="yt-timestamp" data-t="00:01:39">[00:01:39]</a>.
8.  **Enter Verification Code**: Once you receive the code, enter it into the provided field <a class="yt-timestamp" data-t="00:02:07">[00:02:07]</a> and confirm.

Once verified, your account will have these features enabled <a class="yt-timestamp" data-t="00:02:11">[00:02:11]</a>, allowing you to upload videos longer than 15 minutes <a class="yt-timestamp" data-t="00:02:13">[00:02:13]</a>.
