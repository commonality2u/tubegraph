This guide teaches you [[how_to_activate_dark_mode_on_facebook | how to set]] the dark mode within the Facebook application on your mobile phone <a class="yt-timestamp" data-t="00:00:07">[00:00:07]</a>. This tutorial is applicable for both iPhone and Android devices <a class="yt-timestamp" data-t="00:00:48">[00:00:48]</a>.

## Prerequisites

Before proceeding, ensure you have the [[facebook_application_setup_on_mobile_phones | Facebook application downloaded]] <a class="yt-timestamp" data-t="00:00:52">[00:00:52]</a>. It is crucial that your app is [[updating_facebook_app_on_iphone_and_android | updated]] to the latest version <a class="yt-timestamp" data-t="00:00:57">[00:00:57]</a>.

*   To update or download:
    1.  Go to the App Store (or Google Play Store) <a class="yt-timestamp" data-t="00:01:00">[00:01:00]</a>.
    2.  Search for "Facebook" <a class="yt-timestamp" data-t="00:01:06">[00:01:06]</a>.
    3.  Select Facebook <a class="yt-timestamp" data-t="00:01:09">[00:01:09]</a>.
    4.  Tap "Update" if available; otherwise, if already fully [[updating_facebook_app_on_iphone_and_android | updated]], tap "Open" <a class="yt-timestamp" data-t="00:01:10">[00:01:10]</a>.

## Accessing Dark Mode Settings

Once the Facebook app is open:

1.  Navigate to the three lines on the bottom right of the menu <a class="yt-timestamp" data-t="00:01:16">[00:01:16]</a>.
2.  In the "Configuration and privacy" section, you will find the "Dark Mode" option, identifiable by a moon icon <a class="yt-timestamp" data-t="00:01:22">[00:01:22]</a>.
3.  Click on it to reveal three options <a class="yt-timestamp" data-t="00:01:28">[00:01:28]</a>.

## Dark Mode Options

Within the dark mode settings, you have three preferences:

*   **System** <a class="yt-timestamp" data-t="00:01:31">[00:01:31]</a>: This option, selected by default, means Facebook will automatically adjust its theme to match your mobile device's [[using_system_settings_for_facebook_dark_mode | system settings]] <a class="yt-timestamp" data-t="00:01:36">[00:01:36]</a>.
    *   If your phone is in dark mode, Facebook will also be in dark mode <a class="yt-timestamp" data-t="00:01:43">[00:01:43]</a>.
    *   If your phone is in light (white) mode, Facebook will not be in dark mode <a class="yt-timestamp" data-t="00:01:47">[00:01:47]</a>.

*   **Deactivated (Off)** <a class="yt-timestamp" data-t="00:01:53">[00:01:53]</a>: Choosing this option will display Facebook in white (light) mode, regardless of your phone's system settings <a class="yt-timestamp" data-t="00:01:56">[00:01:56]</a>. This is useful if you have your entire mobile device in dark mode but prefer Facebook to be in white <a class="yt-timestamp" data-t="00:01:56">[00:01:56]</a>.

*   **Activated (On)** <a class="yt-timestamp" data-t="00:02:04">[00:02:04]</a>: This option forces Facebook into dark mode, even if your mobile device's system settings are set to light mode <a class="yt-timestamp" data-t="00:02:04">[00:02:04]</a>.

### Important Note on Color

It's important to note that Facebook's dark mode is not a pure black; rather, it features a shade of gray that is slightly lighter than true black <a class="yt-timestamp" data-t="00:02:20">[00:02:20]</a>.
