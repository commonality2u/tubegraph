## Application Requirements

To be eligible to apply for a Payoneer card, a user must have received payments totaling at least one hundred dollars ($100) within the last six months <a class="yt-timestamp" data-t="00:07:26">[00:07:26]</a>. This serves as a minimum requirement for the application <a class="yt-timestamp" data-t="00:07:33">[00:07:33]</a>.
