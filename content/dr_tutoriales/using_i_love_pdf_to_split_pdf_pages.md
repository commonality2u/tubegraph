[[using_i_love_pdf_tool | I Love PDF]] is an online tool that allows users to [[how_to_merge_pdf_files | work with PDFs]] and perform various operations without the need for external software installation <a class="yt-timestamp" data-t="00:01:50">[00:01:50]</a>. This method provides a way to split PDF pages using a web browser.

## How to Split PDF Pages using I Love PDF

To split a PDF document using [[using_i_love_pdf_tool | I Love PDF]], follow these steps:

1.  **Access the tool**: Open a web browser (e.g., Google Chrome) and search for "I love PDF" <a class="yt-timestamp" data-t="00:01:48">[00:01:48]</a>.
2.  **Select "Split PDF"**: On the [[using_i_love_pdf_tool | I Love PDF]] website, choose the "Split PDF" option <a class="yt-timestamp" data-t="00:01:57">[00:01:57]</a>. This may appear in Spanish as "Dividir PDF" <a class="yt-timestamp" data-t="00:02:01">[00:02:01]</a>.
3.  **Upload the PDF**: Click to select the PDF file you wish to separate from your computer <a class="yt-timestamp" data-t="00:02:01">[00:02:01]</a>. Once selected, click "Open" <a class="yt-timestamp" data-t="00:02:08">[00:02:08]</a>.
4.  **Extract pages**: After uploading, click on the "Extract Pages" button <a class="yt-timestamp" data-t="00:02:11">[00:02:11]</a>.
5.  **Choose pages for extraction**: The tool will display all pages of the PDF. You can select specific pages you wish to extract <a class="yt-timestamp" data-t="00:02:17">[00:02:17]</a>. In the demonstration, all pages were chosen <a class="yt-timestamp" data-t="00:02:22">[00:02:22]</a>.
6.  **Initiate splitting**: Click "Split PDF" <a class="yt-timestamp" data-t="00:02:22">[00:02:22]</a>. The tool will then divide the document into separate pages <a class="yt-timestamp" data-t="00:02:25">[00:02:25]</a>.
7.  **Download results**: Click "Download" to save the separated PDFs <a class="yt-timestamp" data-t="00:02:27">[00:02:27]</a>. The output will be a compressed file, such as a WinRAR archive, containing all the individual pages <a class="yt-timestamp" data-t="00:02:30">[00:02:30]</a>.
8.  **Access individual pages**: Once downloaded and extracted, you can open each individual page as a separate PDF file <a class="yt-timestamp" data-t="00:02:34">[00:02:34]</a>.
