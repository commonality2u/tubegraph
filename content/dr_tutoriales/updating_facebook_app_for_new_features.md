To access new features, such as [[activating_dark_mode_on_facebook | dark mode]], it is crucial to have the Facebook application updated to its latest version <a class="yt-timestamp" data-t="00:00:57">[00:00:57]</a>. This tutorial is applicable for both [[setting_facebook_to_dark_mode_for_iphone | iPhone]] and [[setting_facebook_to_dark_mode_for_android | Android]] devices <a class="yt-timestamp" data-t="00:00:46">[00:00:46]</a>.

## How to Update

To ensure your Facebook application is updated:
1.  Navigate to your device's App Store <a class="yt-timestamp" data-t="00:01:00">[01:00:00]</a>.
2.  Search for "Facebook" <a class="yt-timestamp" data-t="01:06:00">[01:06:00]</a>.
3.  Select the Facebook application from the search results <a class="yt-timestamp" data-t="01:09:00">[01:09:00]</a>.
4.  If an update is available, click "Update" <a class="yt-timestamp" data-t="01:10:00">[01:10:00]</a>.
5.  If the application is already at its maximum updated version, click "Open" <a class="yt-timestamp" data-t="01:13:00">[01:13:00]</a>.

> [!NOTE]
> If you do not have the Facebook application downloaded, you must first download it from the App Store <a class="yt-timestamp" data-t="00:00:55">[00:00:55]</a>.
