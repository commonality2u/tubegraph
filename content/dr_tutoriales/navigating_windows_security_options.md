This guide details how to access and manage Windows security settings, focusing on the activation and deactivation of Windows Defender. These steps can be particularly useful when downloading updates or installing products that Windows Defender might otherwise block <a class="yt-timestamp" data-t="00:00:10">[00:00:10]</a>.

## Accessing Windows Security

To begin navigating Windows security options:
1.  Go to the Windows taskbar, located at the bottom left of your computer screen <a class="yt-timestamp" data-t="00:00:39">[00:00:39]</a>.
2.  Type "defend" into the search bar <a class="yt-timestamp" data-t="00:00:43">[00:00:43]</a>.
3.  From the results, select the "Windows Security" option found under "Antivirus protection against threats" <a class="yt-timestamp" data-t="00:00:51">[00:00:51]</a>.

Upon accessing [[understanding_windows_defender_settings | Windows Security]], you can navigate to the "Home" section to view all available options for defending against external threats <a class="yt-timestamp" data-t="00:01:00">[00:01:00]</a>. It is also possible to access [[understanding_windows_defender_settings | Windows Defender]] via the left-hand menu within the Windows Security interface <a class="yt-timestamp" data-t="00:01:31">[00:01:31]</a>.

### Managing Antivirus and Firewall Protection

The primary focus for activating or deactivating [[understanding_windows_defender_settings | Windows Defender]] is the "Antivirus protection against threats" option, which is typically the first one listed <a class="yt-timestamp" data-t="00:01:08">[00:01:08]</a>.

Other security systems, such as [[managing_antivirus_protection_and_firewall_settings | firewall protection]], can also be managed from this section by selecting option number three on the right-hand side <a class="yt-timestamp" data-t="00:01:16">[00:01:16]</a>.

## [[how_to_activate_windows_defender_step_by_step | Activating and Deactivating Windows Defender]]

### [[how_to_disable_windows_defender_in_windows_10 | Deactivating Windows Defender]]

If you wish to [[how_to_disable_windows_defender_in_windows_10 | disable Windows Defender]]:
1.  Within the "Antivirus protection against threats" section, go to "Manage settings" <a class="yt-timestamp" data-t="00:01:42">[00:01:42]</a>.
2.  Toggle the blue bar from "on" to "deactivated" <a class="yt-timestamp" data-t="00:01:53">[00:01:53]</a>.
3.  A confirmation prompt will appear; select "Yes" to allow changes to your device <a class="yt-timestamp" data-t="00:01:57">[00:01:57]</a>.

This process involves deactivating several components:
*   Real-time protection <a class="yt-timestamp" data-t="00:02:09">[00:02:09]</a>
*   Cloud-based protection <a class="yt-timestamp" data-t="00:02:26">[00:02:26]</a>
*   Automatic sample submission <a class="yt-timestamp" data-t="00:02:27">[00:02:27]</a>
*   Tamper protection <a class="yt-timestamp" data-t="00:02:30">[00:02:30]</a>

Once deactivated, the antivirus protection will display a red cross, indicating that it is completely disabled <a class="yt-timestamp" data-t="00:02:47">[00:02:47]</a>.

### [[how_to_activate_windows_defender_step_by_step | Activating Windows Defender]]

To [[how_to_activate_windows_defender_step_by_step | activate Windows Defender]], the process is the exact reverse of deactivation <a class="yt-timestamp" data-t="00:02:01">[00:02:01]</a>.
