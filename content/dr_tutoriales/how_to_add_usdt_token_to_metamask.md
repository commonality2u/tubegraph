Adding the USDT (Tether) token to your [[introduction_to_metamask_platform | MetaMask]] account allows you to make the token available for use, convert cryptocurrencies into USDT, and store it within your wallet <a class="yt-timestamp" data-t="00:00:07">[00:00:07]</a>. This process is identical whether you are using a computer or a cell phone <a class="yt-timestamp" data-t="00:00:42">[00:00:42]</a> <a class="yt-timestamp" data-t="00:00:45">[00:00:45]</a>.

Before proceeding, ensure you have already logged into your [[using_metamask_for_cryptocurrency_transactions | MetaMask]] wallet <a class="yt-timestamp" data-t="00:00:50">[00:00:50]</a>.

## Finding the USDT Contract Address

To add the USDT token, you first need its contract address:

1.  Open a new tab in your browser <a class="yt-timestamp" data-t="00:01:05">[00:01:05]</a>.
2.  Navigate to the web page that lists all cryptocurrencies and their existing contracts <a class="yt-timestamp" data-t="00:01:15">[00:01:15]</a> <a class="yt-timestamp" data-t="00:01:19">[00:01:19]</a>. (The link for this platform is typically found in the first comment of the video <a class="yt-timestamp" data-t="00:01:08">[00:01:08]</a> <a class="yt-timestamp" data-t="00:01:12">[00:01:12]</a>.)
3.  In the search bar on this platform, type "USDT" <a class="yt-timestamp" data-t="00:01:22">[00:01:22]</a>.
4.  Locate the USDT contract address, which will typically appear with a little fox icon indicating compatibility with [[finding_and_using_contract_addresses_on_metamask | MetaMask]] <a class="yt-timestamp" data-t="00:01:29">[00:01:29]</a>.

## Methods for Adding the USDT Token

There are two primary ways to add the USDT token to your [[introduction_to_metamask_platform | MetaMask]] wallet:

### Method 1: Manual Import via Contract Address

If USDT does not automatically appear or you prefer to add it manually:

1.  Copy the USDT contract address from the webpage <a class="yt-timestamp" data-t="00:01:54">[00:01:54]</a> <a class="yt-timestamp" data-t="00:01:57">[00:01:57]</a>.
2.  Go to your [[introduction_to_metamask_platform | MetaMask]] platform <a class="yt-timestamp" data-t="00:01:37">[00:01:37]</a> <a class="yt-timestamp" data-t="00:01:39">[00:01:39]</a>.
3.  Look for an option to "Import Token" or "Add Token" (sometimes listed as "contract address token") <a class="yt-timestamp" data-t="00:01:51">[00:01:51]</a>.
4.  Paste the copied contract address into the designated field in [[introduction_to_metamask_platform | MetaMask]] <a class="yt-timestamp" data-t="00:01:59">[00:01:59]</a>.
5.  [[introduction_to_metamask_platform | MetaMask]] should automatically populate the token symbol (USDT) and other details <a class="yt-timestamp" data-t="00:02:03">[00:02:03]</a>.
6.  Confirm to add the token <a class="yt-timestamp" data-t="00:02:08">[00:02:08]</a>.

### Method 2: One-Click Addition via Fox Icon

This is a simpler method if the fox icon is available on the contract listing page:

1.  Click the little fox icon next to the USDT contract address on the webpage <a class="yt-timestamp" data-t="00:02:12">[00:02:12]</a>.
2.  A [[introduction_to_metamask_platform | MetaMask]] notification tab will open, indicating it is loading the token <a class="yt-timestamp" data-t="00:02:15">[00:02:15]</a>.
3.  Once loaded, you can add the token directly to your [[introduction_to_metamask_platform | MetaMask]] platform <a class="yt-timestamp" data-t="00:02:19">[00:02:19]</a> <a class="yt-timestamp" data-t="00:02:22">[00:02:22]</a>.
4.  Refresh your [[introduction_to_metamask_platform | MetaMask]] page, and the USDT token should now appear in your wallet <a class="yt-timestamp" data-t="00:02:24">[00:02:24]</a> <a class="yt-timestamp" data-t="00:02:28">[00:02:28]</a>.

Once the USDT token is listed, you can use it to convert cryptocurrencies and withdraw them if desired <a class="yt-timestamp" data-t="00:02:32">[00:02:32]</a>. This method makes importing the USDT token into [[introduction_to_metamask_platform | MetaMask]] straightforward <a class="yt-timestamp" data-t="00:02:35">[00:02:35]</a>.
