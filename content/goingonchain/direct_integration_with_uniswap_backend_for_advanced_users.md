## Purpose

By using the backend directly, users can bypass the 0.15% swap fee that [[Uniswap swap fees | Uniswap]] implemented for transactions made through their user interface (UI) <a class="yt-timestamp" data-t="00:00:09">[00:00:09]</a>.

1.  Navigate to EtherScan <a class="yt-timestamp" data-t="00:02:01">[00:02:01]</a>.
2.  Access the [[Uniswap swap fees | Uniswap]] V2 router <a class="yt-timestamp" data-t="00:02:01">[00:02:01]</a>.
3.  Go to the "Contract" tab and then "Write Contract" <a class="yt-timestamp" data-t="00:02:03">[00:02:03]</a>.
4.  Connect your Web3 wallet <a class="yt-timestamp" data-t="00:02:05">[00:02:05]</a>.
5.  Locate and call the desired function, for example, "sort if for exact tokens" <a class="yt-timestamp" data-t="00:02:07">[00:02:07]</a>.
