Unlike traditional wallets, users do not need to remember a seed phrase <a class="yt-timestamp" data-t="00:02:30">[00:02:30]</a>. Login is managed via email and phone <a class="yt-timestamp" data-t="00:02:34">[00:02:34]</a>. ### Guardian Feature

If a user's phone is lost, a Guardian can remotely lock the wallet for them <a class="yt-timestamp" data-t="00:02:46">[00:02:46]</a>.
