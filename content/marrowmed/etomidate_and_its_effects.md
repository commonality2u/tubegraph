Etomidate is a drug that has been a repeated topic in examinations, notably appearing in NEET PG 2017 and 2018 <a class="yt-timestamp" data-t="00:17:53">[00:17:53]</a>.

## Key Properties and Effects

Etomidate is known for its ability to cause adrenal suppression <a class="yt-timestamp" data-t="00:17:50">[00:17:50]</a>. This specific effect has been the subject of multiple questions <a class="yt-timestamp" data-t="00:19:55">[00:19:55]</a>.

A mnemonic can be used to remember the features of etomidate <a class="yt-timestamp" data-t="00:18:09">[00:18:09]</a>:

> [!INFO] Etomidate Mnemonic
> - **E** for `emetic` <a class="yt-timestamp" data-t="00:18:25">[00:18:25]</a>.
> - **E** for `epileptogenic` <a class="yt-timestamp" data-t="00:18:31">[00:18:31]</a>, meaning it might cause epileptic activity.
> - **O** for `oily` <a class="yt-timestamp" data-t="00:18:36">[00:18:36]</a>. It contains propylene glycol <a class="yt-timestamp" data-t="00:18:39">[00:18:39]</a>. As an oily preparation, it can be painful upon intravenous administration <a class="yt-timestamp" data-t="00:18:49">[00:18:49]</a>.
>   - Another oily preparation used in anesthesia is Propofol, which contains soybean oil and is also painful <a class="yt-timestamp" data-t="00:19:04">[00:19:04]</a>.
> - **T** for `tortures the hand` <a class="yt-timestamp" data-t="00:18:53">[00:18:53]</a>, indicating it causes pain at the injection site <a class="yt-timestamp" data-t="00:18:59">[00:18:59]</a>.
> - **M** for `myoclonus` <a class="yt-timestamp" data-t="00:19:25">[00:19:25]</a>, characterized by sudden involuntary or jerky movements of the limb <a class="yt-timestamp" data-t="00:19:29">[00:19:29]</a>.
> - **I** for `inhibits adrenocortical synthesis` <a class="yt-timestamp" data-t="00:19:43">[00:19:43]</a>, which is considered a negative property of the drug <a class="yt-timestamp" data-t="00:19:50">[00:19:50]</a>.
> - **DATE** (from etomidate) signifies that it is the `most cardio stable` agent <a class="yt-timestamp" data-t="00:20:23">[00:20:23]</a>, making it the agent of choice for cardiac surgery <a class="yt-timestamp" data-t="00:20:46">[00:20:46]</a> and for procedures like DC cardioversion <a class="yt-timestamp" data-t="00:20:50">[00:20:50]</a>.
