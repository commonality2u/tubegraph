## Modes of AI Assistance

AI primarily assists in two main modes:

1.  **Co-pilot Mode**
    *   In this mode, a human actively decides to use AI for specific tasks in real-time, such as asking ChatGPT for help with a problem <a class="yt-timestamp" data-t="01:57:01">[01:57:01]</a>. The human remains the primary agent driving the process <a class="yt-timestamp" data-t="02:24:00">[02:24:00]</a>.
    *   This mode is about helping humans be better and is a classic example of symbiosis or intelligence augmentation <a class="yt-timestamp" data-t="03:03:01">[03:03:01]</a>.

2.  **Delegation Mode**
    *   This involves truly offloading a task to AI, with the goal of receiving consistent output that requires minimal human review <a class="yt-timestamp" data-t="02:40:00">[02:40:00]</a>.
    *   Delegation mode can save significant time and money on tedious tasks or enable scaling of things that were previously unscalable <a class="yt-timestamp" data-t="03:10:00">[03:10:00]</a>.
    *   Between co-pilot and full delegation, there's a developing space for "ad hoc delegation," where larger subtasks can be delegated to AI on the fly <a class="yt-timestamp" data-t="03:55:00">[03:55:00]</a>. This gap is expected to close as AI agents become more capable <a class="yt-timestamp" data-t="04:28:00">[04:28:00]</a>.

*   **Generating Code**: AI can write the necessary code for specific functionalities, even for complex tasks, allowing developers to focus on reading and refining the code rather than writing it from scratch <a class="yt-timestamp" data-t="36:21:00">[36:21:00]</a> <a class="yt-timestamp" data-t="10:15:00">[10:15:00]</a>. *   **Troubleshooting and Refinement**: When issues arise (e.g., code appearing in the wrong place), AI can analyze the problem and provide specific instructions with code modifications <a class="yt-timestamp" data-t="35:20:00">[35:20:00]</a>. ## AI Prompting Best Practices

Effective prompting is crucial for maximizing AI's utility:

*   **Avoid Immediate Answers**: Prompting in a way that forces an immediate answer prevents the model from engaging in "Chain of Thought" reasoning <a class="yt-timestamp" data-t="02:34:00">[02:34:00]</a>.
*   **Analysis Before Answer (AAA)**: Always prompt for "Analysis before Answer" for better results <a class="yt-timestamp" data-t="02:56:00">[02:56:00]</a> <a class="yt-timestamp" data-t="02:56:00">[02:56:00]</a>. If an AI jumps directly to an answer, it indicates performance is being left on the table <a class="yt-timestamp" data-t="03:27:00">[03:27:00]</a>.
*   **Custom Instructions**: [[personal_productivity_hacks_using_ai | Custom instructions]] in tools like ChatGPT allow users to provide context about themselves (e.g., profession, personal relationships, goals, areas for self-improvement) <a class="yt-timestamp" data-t="04:03:00">[04:03:00]</a>. This context helps the AI provide more relevant and personalized assistance, understanding the user's needs without constant re-explanation <a class="yt-timestamp" data-t="04:15:00">[04:15:00]</a>.
*   **Iterative "Dance"**: Engage in a mutual "dance" with the AI, where the user provides specific project details and the AI offers explanations and solutions <a class="yt-timestamp" data-t="03:56:00">[03:56:00]</a>. This iterative process allows both the human and the AI to build understanding and accomplish tasks more effectively <a class="yt-timestamp" data-t="04:04:00">[04:04:00]</a>.
