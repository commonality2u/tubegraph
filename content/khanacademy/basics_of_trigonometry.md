## What is Trigonometry?
[[definition_of_trigonometry | Trigonometry]] is fundamentally the study of the ratios of sides of triangles <a class="yt-timestamp" data-t="00:00:10">[00:00:10]</a>. The word "trig" literally means "triangle," and "metry" literally means "measure" <a class="yt-timestamp" data-t="00:00:14">[00:00:14]</a>.

## The Right Triangle
Trigonometry often begins with the study of [[right_triangles_and_trigonometry | right triangles]] <a class="yt-timestamp" data-t="00:00:26">[00:00:26]</a>. A right triangle has one angle that is 90 degrees <a class="yt-timestamp" data-t="00:00:33">[00:00:33]</a>.

In a right triangle, the side opposite the right angle is called the **hypotenuse** <a class="yt-timestamp" data-t="00:01:16">[00:01:16]</a>. This is always the longest side of a right triangle <a class="yt-timestamp" data-t="00:01:20">[00:01:20]</a>.

**Example Right Triangle (3-4-5 Triangle):**
Consider a right triangle with sides of length 3, 4, and 5 <a class="yt-timestamp" data-t="00:00:57">[00:00:57]</a>:
*   One side (height) is 3 <a class="yt-timestamp" data-t="00:00:59">[00:00:59]</a>.
*   The base is 4 <a class="yt-timestamp" data-t="00:01:02">[00:01:02]</a>.
*   The hypotenuse is 5 <a class="yt-timestamp" data-t="00:01:07">[00:01:07]</a>.

This triangle satisfies the Pythagorean theorem, where 3² + 4² = 5² <a class="yt-timestamp" data-t="00:01:32">[00:01:32]</a>.

## [[core_trigonometric_functions_sine_cosine_and_tangent | Core Trigonometric Functions]]
The core functions in trigonometry are:
*   [[sine_cosine_and_tangent_functions | Sine]] (sin) <a class="yt-timestamp" data-t="00:02:01">[00:02:01]</a>
*   [[sine_cosine_and_tangent_functions | Cosine]] (cos) <a class="yt-timestamp" data-t="00:02:04">[00:02:04]</a>
*   [[sine_cosine_and_tangent_functions | Tangent]] (tan) <a class="yt-timestamp" data-t="00:02:07">[00:02:07]</a>

These functions specify the ratios of certain sides for any given angle within a right triangle <a class="yt-timestamp" data-t="00:02:15">[00:02:15]</a>.

## [[trigonometric_ratios_and_soh_cah_toa_mnemonic | SOH CAH TOA Mnemonic]]
A mnemonic device called **SOH CAH TOA** helps remember the definitions of these functions <a class="yt-timestamp" data-t="00:02:24">[00:02:24]</a>:

> [!definition] SOH CAH TOA
> *   **SOH**: **S**ine = **O**pposite / **H**ypotenuse <a class="yt-timestamp" data-t="00:02:47">[00:02:47]</a>
> *   **CAH**: **C**osine = **A**djacent / **H**ypotenuse <a class="yt-timestamp" data-t="00:03:03">[00:03:03]</a>
> *   **TOA**: **T**angent = **O**pposite / **A**djacent <a class="yt-timestamp" data-t="00:03:17">[00:03:17]</a>

### Defining Opposite and Adjacent Sides
The terms "opposite" and "adjacent" depend on the specific angle being considered in the triangle <a class="yt-timestamp" data-t="00:02:23">[00:02:23]</a>:
*   The **opposite** side is the side across from the angle <a class="yt-timestamp" data-t="00:04:00">[00:04:00]</a>.
*   The **adjacent** side is one of the sides that forms the angle, but is not the hypotenuse <a class="yt-timestamp" data-t="00:05:09">[00:05:09]</a>.

## [[trigonometric_ratios_and_angle_identification | Applying Trigonometric Ratios]]

### Example 1: Angle Theta (θ)
Consider the 3-4-5 right triangle. Let's define an angle, theta (θ), between the side of length 4 and the hypotenuse of length 5 <a class="yt-timestamp" data-t="00:03:30">[00:03:30]</a>.

For angle θ:
*   **Opposite side**: 3 <a class="yt-timestamp" data-t="00:04:12">[00:04:12]</a>
*   **Adjacent side**: 4 <a class="yt-timestamp" data-t="00:05:18">[00:05:18]</a>
*   **Hypotenuse**: 5 <a class="yt-timestamp" data-t="00:04:22">[00:04:22]</a>

Using SOH CAH TOA:
*   **Sine of θ (sin θ)** = Opposite / Hypotenuse = 3 / 5 <a class="yt-timestamp" data-t="00:04:27">[00:04:27]</a>
    *   The ratio of the opposite side to the hypotenuse for a given angle will always be the same, regardless of the triangle's size <a class="yt-timestamp" data-t="00:04:38">[00:04:38]</a>.
*   **Cosine of θ (cos θ)** = Adjacent / Hypotenuse = 4 / 5 <a class="yt-timestamp" data-t="00:05:42">[00:05:42]</a>
*   **Tangent of θ (tan θ)** = Opposite / Adjacent = 3 / 4 <a class="yt-timestamp" data-t="00:06:09">[00:06:09]</a>

### Example 2: Angle X
Now, consider the other acute angle in the same 3-4-5 right triangle. Let's call this angle X <a class="yt-timestamp" data-t="00:07:06">[00:07:06]</a>.

For angle X:
*   **Opposite side**: 4 <a class="yt-timestamp" data-t="00:07:23">[00:07:23]</a> (Note: This was the adjacent side for angle θ <a class="yt-timestamp" data-t="00:07:32">[00:07:32]</a>)
*   **Adjacent side**: 3 <a class="yt-timestamp" data-t="00:08:07">[00:08:07]</a> (Note: This was the opposite side for angle θ <a class="yt-timestamp" data-t="00:05:03">[00:05:03]</a>)
*   **Hypotenuse**: 5 <a class="yt-timestamp" data-t="00:07:41">[00:07:41]</a>

Using SOH CAH TOA:
*   **Sine of X (sin X)** = Opposite / Hypotenuse = 4 / 5 <a class="yt-timestamp" data-t="00:07:46">[00:07:46]</a>
*   **Cosine of X (cos X)** = Adjacent / Hypotenuse = 3 / 5 <a class="yt-timestamp" data-t="00:08:14">[00:08:14]</a>
*   **Tangent of X (tan X)** = Opposite / Adjacent = 4 / 3 <a class="yt-timestamp" data-t="00:08:34">[00:08:34]</a>

This [[introduction_to_basic_trigonometric_functions | introduction to basic trigonometric functions]] using SOH CAH TOA is highly effective for angles between 0 and 90 degrees <a class="yt-timestamp" data-t="00:08:57">[00:08:57]</a>.
