A parabola is the distinctive U-shaped curve that represents a quadratic function <a class="yt-timestamp" data-t="00:01:38">[00:01:38]</a>. For example, the function `f(x) = 3x² + 6x - 2` is a quadratic function, and its graph is a parabola <a class="yt-timestamp" data-t="00:00:02">[00:00:02]</a>, <a class="yt-timestamp" data-t="00:01:35">[00:01:35]</a>.

## Domain of a Quadratic Function

The domain of a function refers to the set of all valid input values (x-values) for which the function is defined <a class="yt-timestamp" data-t="00:00:08">[00:00:08]</a>, <a class="yt-timestamp" data-t="00:00:12">[00:00:12]</a>. For a quadratic function like `f(x) = 3x² + 6x - 2`, any real number can be an input <a class="yt-timestamp" data-t="00:00:16">[00:00:16]</a>. You can square it, multiply it by 3, add 6 times that number, and subtract 2, resulting in a valid output <a class="yt-timestamp" data-t="00:00:18">[00:00:18]</a>, <a class="yt-timestamp" data-t="00:01:05">[00:01:05]</a>.

> [!INFO] Real Numbers
> Most traditional numbers, including integers, fractions, and decimals, are part of the set of real numbers <a class="yt-timestamp" data-t="00:00:57">[00:00:57]</a>, <a class="yt-timestamp" data-t="00:01:00">[00:01:00]</a>. The primary exceptions are imaginary and complex numbers <a class="yt-timestamp" data-t="00:00:48">[00:00:48]</a>, <a class="yt-timestamp" data-t="00:00:53">[00:00:53]</a>.
>
The domain for `f(x) = 3x² + 6x - 2` is **all real numbers** <a class="yt-timestamp" data-t="00:00:30">[00:00:30]</a>, <a class="yt-timestamp" data-t="00:00:35">[00:00:35]</a>.

## Range of a Quadratic Function

The range of a function is the set of all possible output values (y-values) that the function can produce <a class="yt-timestamp" data-t="00:01:16">[00:01:16]</a>, <a class="yt-timestamp" data-t="00:01:21">[00:01:21]</a>, <a class="yt-timestamp" data-t="00:01:26">[00:01:26]</a>. For a parabola, the range is determined by its [[vertex_of_a_parabola | vertex]] <a class="yt-timestamp" data-t="00:01:51">[00:01:51]</a>, <a class="yt-timestamp" data-t="00:01:55">[00:01:55]</a>.

### The [[vertex_of_a_parabola | Vertex]] of a Parabola

The [[vertex_of_a_parabola | vertex]] is the turning point of the parabola.
*   For an upward-opening parabola (like `f(x) = 3x² + 6x - 2` because the coefficient of x² is positive), the [[vertex_of_a_parabola | vertex]] represents the minimum y-value the function will take on <a class="yt-timestamp" data-t="00:01:51">[00:01:51]</a>, <a class="yt-timestamp" data-t="00:07:03">[00:07:03]</a>, <a class="yt-timestamp" data-t="00:07:05">[00:07:05]</a>.
*   For a downward-opening parabola, the [[vertex_of_a_parabola | vertex]] represents the maximum y-value <a class="yt-timestamp" data-t="00:01:55">[00:01:55]</a>.

The x-coordinate of the [[vertex_of_a_parabola | vertex]] for a quadratic function in the form `ax² + bx + c` can be found using the formula: `x = -b / (2a)` <a class="yt-timestamp" data-t="00:02:16">[00:02:16]</a>, <a class="yt-timestamp" data-t="00:03:14">[00:03:14]</a>.

For `f(x) = 3x² + 6x - 2`:
*   a = 3
*   b = 6
*   c = -2

So, the x-coordinate of the [[vertex_of_a_parabola | vertex]] is:
`x = -6 / (2 * 3)` <a class="yt-timestamp" data-t="00:03:19">[00:03:19]</a>, <a class="yt-timestamp" data-t="00:03:22">[00:03:22]</a>
`x = -6 / 6`
`x = -1` <a class="yt-timestamp" data-t="00:03:26">[00:03:26]</a>

To find the y-coordinate of the [[vertex_of_a_parabola | vertex]], substitute this x-value back into the function:
`f(-1) = 3(-1)² + 6(-1) - 2` <a class="yt-timestamp" data-t="00:02:53">[00:02:53]</a>
`f(-1) = 3(1) - 6 - 2` <a class="yt-timestamp" data-t="00:02:56">[00:02:56]</a>, <a class="yt-timestamp" data-t="00:03:02">[00:03:02]</a>
`f(-1) = 3 - 6 - 2`
`f(-1) = -3 - 2`
`f(-1) = -5` <a class="yt-timestamp" data-t="00:03:07">[00:03:07]</a>, <a class="yt-timestamp" data-t="00:03:10">[00:03:10]</a>

Thus, the [[vertex_of_a_parabola | vertex]] of this parabola is at the point `(-1, -5)` <a class="yt-timestamp" data-t="00:03:10">[00:03:10]</a>, <a class="yt-timestamp" data-t="00:05:56">[00:05:56]</a>.

### Graphing and Symmetry

Graphing a parabola helps visualize its range. We can plot points by choosing various x-values and calculating their corresponding f(x) (y) values <a class="yt-timestamp" data-t="00:02:10">[00:02:10]</a>, <a class="yt-timestamp" data-t="00:02:22">[00:02:22]</a>:

*   **x = -2**: `f(-2) = 3(-2)² + 6(-2) - 2 = 3(4) - 12 - 2 = 12 - 12 - 2 = -2`. Point: `(-2, -2)` <a class="yt-timestamp" data-t="00:02:30">[00:02:30]</a>, <a class="yt-timestamp" data-t="00:02:33">[00:02:33]</a>, <a class="yt-timestamp" data-t="00:02:44">[00:02:44]</a>.
*   **x = -1**: `f(-1) = -5` (the [[vertex_of_a_parabola | vertex]]) <a class="yt-timestamp" data-t="00:02:49">[00:02:49]</a>, <a class="yt-timestamp" data-t="00:03:10">[00:03:10]</a>. Point: `(-1, -5)` <a class="yt-timestamp" data-t="00:05:56">[00:05:56]</a>.
*   **x = 0**: `f(0) = 3(0)² + 6(0) - 2 = -2`. Point: `(0, -2)` <a class="yt-timestamp" data-t="00:03:34">[00:03:34]</a>, <a class="yt-timestamp" data-t="00:03:36">[00:03:36]</a>, <a class="yt-timestamp" data-t="00:06:03">[00:06:03]</a>.
*   **x = 1**: `f(1) = 3(1)² + 6(1) - 2 = 3 + 6 - 2 = 9 - 2 = 7`. Point: `(1, 7)` <a class="yt-timestamp" data-t="00:04:01">[00:04:01]</a>, <a class="yt-timestamp" data-t="00:04:04">[00:04:04]</a>, <a class="yt-timestamp" data-t="00:04:14">[00:04:14]</a>, <a class="yt-timestamp" data-t="00:06:14">[00:06:14]</a>.

The graph of `f(x) = 3x² + 6x - 2` is an upward-opening parabola <a class="yt-timestamp" data-t="00:01:43">[00:01:43]</a>, <a class="yt-timestamp" data-t="00:06:24">[00:06:24]</a>. It is symmetrical around the vertical line passing through its [[vertex_of_a_parabola | vertex]] (x = -1) <a class="yt-timestamp" data-t="00:04:46">[00:04:46]</a>, <a class="yt-timestamp" data-t="00:06:42">[00:06:42]</a>. As x moves away from -1 in either direction (larger or smaller), the y-values of the parabola increase indefinitely <a class="yt-timestamp" data-t="00:02:25">[00:02:25]</a>, <a class="yt-timestamp" data-t="00:07:22">[00:07:22]</a>, <a class="yt-timestamp" data-t="00:07:37">[00:07:37]</a>. This means the lowest point the parabola reaches is the y-coordinate of its [[vertex_of_a_parabola | vertex]], which is -5 <a class="yt-timestamp" data-t="00:07:16">[00:07:16]</a>, <a class="yt-timestamp" data-t="00:07:19">[00:07:19]</a>. The function will never produce an output less than -5 <a class="yt-timestamp" data-t="00:07:30">[00:07:30]</a>, <a class="yt-timestamp" data-t="00:07:32">[00:07:32]</a>.

## Conclusion

For the function `f(x) = 3x² + 6x - 2`:
*   The **domain** is all real numbers <a class="yt-timestamp" data-t="00:07:42">[00:07:42]</a>, <a class="yt-timestamp" data-t="00:07:46">[00:07:46]</a>.
*   The **range** is all real numbers greater than or equal to -5 <a class="yt-timestamp" data-t="00:07:46">[00:07:46]</a>, <a class="yt-timestamp" data-t="00:07:52">[00:07:52]</a>, <a class="yt-timestamp" data-t="00:07:58">[00:07:58]</a>. This is because the lowest y-value it reaches is its [[vertex_of_a_parabola | vertex]] at `(-1, -5)` <a class="yt-timestamp" data-t="00:03:10">[00:03:10]</a>.
