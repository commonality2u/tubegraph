## Subtracting a Positive Number

When [[using_subtraction_in_equations | subtracting]] a positive number from another positive number, if you subtract a larger number from a smaller number, the result will be a [[introduction_to_negative_numbers | negative number]] <a class="yt-timestamp" data-t="00:00:08">[00:00:08]</a>.

### Example: `2 - 3`

To solve `2 - 3` using a number line <a class="yt-timestamp" data-t="00:00:21">[00:00:21]</a>:
1.  Start at 2 on the number line <a class="yt-timestamp" data-t="00:00:34">[00:00:34]</a>.
2.  Subtracting 3 means moving 3 units to the left on the number line <a class="yt-timestamp" data-t="00:00:41">[00:00:41]</a>.
3.  Moving 1, 2, then 3 units to the left from 2 lands on -1 <a class="yt-timestamp" data-t="00:00:44">[00:00:44]</a>.
    *   Therefore, `2 - 3 = -1` <a class="yt-timestamp" data-t="00:00:51">[00:00:51]</a>.

### Example: `negative 2 - 3`

To solve `negative 2 - 3` using a number line <a class="yt-timestamp" data-t="00:01:06">[00:01:06]</a>:
1.  Start at -2 on the number line <a class="yt-timestamp" data-t="00:01:24">[00:01:24]</a>.
2.  Subtracting 3 means moving 3 units to the left from -2 <a class="yt-timestamp" data-t="00:01:28">[00:01:28]</a>, <a class="yt-timestamp" data-t="00:01:30">[00:01:30]</a>.
3.  Moving 1, 2, then 3 units to the left from -2 lands on -5 <a class="yt-timestamp" data-t="00:01:33">[00:01:33]</a>.
    *   Therefore, `negative 2 - 3 = -5` <a class="yt-timestamp" data-t="00:01:39">[00:01:39]</a>.

Regardless of the starting point, [[using_subtraction_in_equations | subtracting]] a positive number always means moving to the left on the number line <a class="yt-timestamp" data-t="00:01:40">[00:01:40]</a>, <a class="yt-timestamp" data-t="00:01:42">[00:01:42]</a>.

## Adding a Positive Number

### Example: `negative 2 + 3`

To solve `negative 2 + 3` using a number line <a class="yt-timestamp" data-t="00:02:10">[00:02:10]</a>:
1.  Start at -2 on the number line, which is 2 units to the left of zero <a class="yt-timestamp" data-t="00:02:23">[00:02:23]</a>, <a class="yt-timestamp" data-t="00:02:24">[00:02:24]</a>.
2.  [[Adding negative numbers | Adding]] 3 means moving 3 units to the right from -2 <a class="yt-timestamp" data-t="00:02:27">[00:02:27]</a>, <a class="yt-timestamp" data-t="00:02:30">[00:02:30]</a>.
3.  Moving 1, 2, then 3 units to the right from -2 lands on 1 <a class="yt-timestamp" data-t="00:02:31">[00:02:31]</a>.
    *   Therefore, `negative 2 + 3 = 1` <a class="yt-timestamp" data-t="00:02:40">[00:02:40]</a>.

## Subtracting a Negative Number

When you [[using_subtraction_in_equations | subtract]] a negative number, it is the same as [[adding_negative_numbers | adding]] a positive number <a class="yt-timestamp" data-t="00:02:59">[00:02:59]</a>, <a class="yt-timestamp" data-t="00:03:01">[00:03:01]</a>. This concept is further explained in other videos on [[understanding_subtracting_a_negative_is_the_same_as_adding_a_positive | understanding subtracting a negative is the same as adding a positive]] <a class="yt-timestamp" data-t="00:02:54">[00:02:54]</a>.

### Example: `2 - negative 3`

*   `2 - negative 3` is equivalent to `2 + positive 3` <a class="yt-timestamp" data-t="00:03:04">[00:03:04]</a>, <a class="yt-timestamp" data-t="00:03:09">[00:03:09]</a>.
*   Therefore, `2 - negative 3 = 5` <a class="yt-timestamp" data-t="00:03:18">[00:03:18]</a>.

### Example: `negative 2 - negative 3`

Even with multiple negatives, remember that [[using_subtraction_in_equations | subtracting]] a negative results in [[adding_negative_numbers | adding]] a positive <a class="yt-timestamp" data-t="00:03:34">[00:03:34]</a>, <a class="yt-timestamp" data-t="00:03:38">[00:03:38]</a>, <a class="yt-timestamp" data-t="00:03:41">[00:03:41]</a>.

1.  `negative 2 - negative 3` is the same as `negative 2 + 3` <a class="yt-timestamp" data-t="00:03:45">[00:03:45]</a>, <a class="yt-timestamp" data-t="00:03:52">[00:03:52]</a>.
2.  As shown previously, `negative 2 + 3` starts at -2 (2 units left of 0) and moves 3 units to the right <a class="yt-timestamp" data-t="00:03:55">[00:03:55]</a>, <a class="yt-timestamp" data-t="00:03:59">[00:03:59]</a>.
3.  Moving 1, 2, then 3 units to the right from -2 lands on 1 <a class="yt-timestamp" data-t="00:04:01">[00:04:01]</a>.
    *   Therefore, `negative 2 - negative 3 = 1` <a class="yt-timestamp" data-t="00:04:01">[00:04:01]</a>.
