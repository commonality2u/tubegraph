## Subtracting a Positive Number

When subtracting a positive number, movement on the number line is to the left <a class="yt-timestamp" data-t="00:00:41">[00:00:41]</a>.

### Example: 2 minus 3
Consider the expression `2 - 3` <a class="yt-timestamp" data-t="00:00:03">[00:00:03]</a>. This is an example of [[understanding_subtracting_a_larger_number_from_a_smaller_number | subtracting a larger number from a smaller number]], which will result in a [[introduction_to_negative_numbers | negative number]] <a class="yt-timestamp" data-t="00:00:13">[00:00:13]</a>.

Using a [[visualizing_negative_numbers_on_a_number_line | number line]] <a class="yt-timestamp" data-t="00:00:21">[00:00:21]</a>:
1.  Start at 2 on the number line <a class="yt-timestamp" data-t="00:00:34">[00:00:34]</a>.
2.  Subtracting 3 means moving 3 units to the left <a class="yt-timestamp" data-t="00:00:41">[00:00:41]</a>.
3.  Moving 1, 2, 3 units to the left from 2 lands on -1 <a class="yt-timestamp" data-t="00:00:44">[00:00:44]</a>.
    `2 - 3 = -1` <a class="yt-timestamp" data-t="00:00:51">[00:00:51]</a>

### Example: Negative 2 minus 3
Consider the expression `negative 2 - 3` <a class="yt-timestamp" data-t="00:00:54">[00:00:54]</a>.

Using a [[visualizing_negative_numbers_on_a_number_line | number line]] <a class="yt-timestamp" data-t="00:01:06">[00:01:06]</a>:
1.  Start at -2 on the number line <a class="yt-timestamp" data-t="00:01:24">[00:01:24]</a>.
2.  Subtracting 3 means moving 3 units to the left of -2 <a class="yt-timestamp" data-t="00:01:28">[00:01:28]</a>.
3.  Moving 1, 2, 3 units to the left from -2 lands on -5 <a class="yt-timestamp" data-t="00:01:33">[00:01:33]</a>.
    `negative 2 - 3 = -5` <a class="yt-timestamp" data-t="00:01:39">[00:01:39]</a>

## Adding a Positive Number

When [[adding_negative_numbers | adding a positive number]], movement on the number line is to the right <a class="yt-timestamp" data-t="00:02:30">[00:02:30]</a>.

### Example: Negative 2 plus 3
Consider the expression `negative 2 + 3` <a class="yt-timestamp" data-t="00:01:57">[00:01:57]</a>.

## Subtracting a Negative Number

The [[concept_of_subtracting_a_negative_number | concept of subtracting a negative number]] is equivalent to [[adding_negative_numbers | adding a positive number]] <a class="yt-timestamp" data-t="00:02:59">[00:02:59]</a>.

> Subtracting a negative number is the same thing as adding the positive number <a class="yt-timestamp" data-t="00:02:59">[00:02:59]</a>.

### Example: 2 minus negative 3
Consider the expression `2 - negative 3` <a class="yt-timestamp" data-t="00:02:47">[00:02:47]</a>.
Applying the rule, this expression is equivalent to `2 + positive 3` <a class="yt-timestamp" data-t="00:03:04">[00:03:04]</a>.
`2 - negative 3 = 2 + 3 = 5` <a class="yt-timestamp" data-t="00:03:18">[00:03:18]</a>

### Example: Negative 2 minus negative 3
Consider the expression `negative 2 - negative 3` <a class="yt-timestamp" data-t="00:03:23">[00:03:23]</a>.
Despite the presence of multiple negatives, remember that subtracting a negative yields a positive <a class="yt-timestamp" data-t="00:03:36">[00:03:36]</a>.
Therefore, `negative 2 - negative 3` is the same as `negative 2 + 3` <a class="yt-timestamp" data-t="00:03:45">[00:03:45]</a>.

As demonstrated previously with [[adding_negative_numbers | negative 2 + 3]]:
1.  Start at -2 (2 units to the left of 0) <a class="yt-timestamp" data-t="00:03:55">[00:03:55]</a>.
2.  Add 3 by moving 3 units to the right <a class="yt-timestamp" data-t="00:03:59">[00:03:59]</a>.
3.  Moving 1, 2, 3 units to the right from -2 lands on 1 <a class="yt-timestamp" data-t="00:04:01">[00:04:01]</a>.
    `negative 2 - negative 3 = 1`
