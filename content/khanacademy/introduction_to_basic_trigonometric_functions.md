## Core Trigonometric Ratios

To remember the [[core_trigonometric_functions_sine_cosine_and_tangent | core trigonometric functions]] (sine, cosine, and tangent), a useful [[mnemonic_device_for_remembering_trigonometric_functions | mnemonic device]] is "SOH CAH TOA" <a class="yt-timestamp" data-t="00:02:09">[00:02:09]</a>:
*   **SOH**: **S**ine = **O**pposite / **H**ypotenuse <a class="yt-timestamp" data-t="00:04:25">[00:04:25]</a>
*   **CAH**: **C**osine = **A**djacent / **H**ypotenuse <a class="yt-timestamp" data-t="00:02:43">[00:02:43]</a>
*   **TOA**: **T**angent = **O**pposite / **A**djacent <a class="yt-timestamp" data-t="00:05:27">[00:05:27]</a>

## Example 1: General Right Triangle

Consider a [[right_triangles_and_trigonometry | right triangle]] with two sides measuring 7 and 4 <a class="yt-timestamp" data-t="00:00:31">[00:00:31]</a>.

### Calculating the Hypotenuse

Using the Pythagorean theorem, if 'h' is the hypotenuse, then:
h² = 7² + 4² <a class="yt-timestamp" data-t="00:00:45">[00:00:45]</a>
h² = 49 + 16 <a class="yt-timestamp" data-t="00:01:04">[00:01:04]</a>
h² = 65 <a class="yt-timestamp" data-t="00:01:25">[00:01:25]</a>
h = √65 <a class="yt-timestamp" data-t="00:01:39">[00:01:39]</a>
This value cannot be simplified further as 65 = 13 × 5, and both are prime <a class="yt-timestamp" data-t="00:01:44">[00:01:44]</a>.

### Finding [[trigonometric_ratios_and_angle_identification | Trigonometric Ratios]] for Angle Theta (θ)

Let's find the [[core_trigonometric_functions_sine_cosine_and_tangent | trigonometric functions]] for the angle theta (θ) at the top vertex <a class="yt-timestamp" data-t="00:02:02">[00:02:02]</a>.

#### Cosine of Theta (cos θ)
Using "CAH" (Cosine = Adjacent / Hypotenuse):
*   **Adjacent side** to θ: 4 <a class="yt-timestamp" data-t="00:03:04">[00:03:04]</a>
*   **Hypotenuse**: √65 <a class="yt-timestamp" data-t="00:03:17">[00:03:17]</a>
cos θ = 4 / √65 <a class="yt-timestamp" data-t="00:03:21">[00:03:21]</a>

To rationalize the denominator, multiply the numerator and denominator by √65:
cos θ = (4 × √65) / (√65 × √65) <a class="yt-timestamp" data-t="00:03:39">[00:03:39]</a>
cos θ = 4√65 / 65 <a class="yt-timestamp" data-t="00:03:54">[00:03:54]</a>

#### Sine of Theta (sin θ)
Using "SOH" (Sine = Opposite / Hypotenuse):
*   **Opposite side** to θ: 7 <a class="yt-timestamp" data-t="00:04:34">[00:04:34]</a>
*   **Hypotenuse**: √65 <a class="yt-timestamp" data-t="00:04:47">[00:04:47]</a>
sin θ = 7 / √65 <a class="yt-timestamp" data-t="00:04:51">[00:04:51]</a>

To rationalize the denominator:
sin θ = (7 × √65) / (√65 × √65) <a class="yt-timestamp" data-t="00:04:55">[00:04:55]</a>
sin θ = 7√65 / 65 <a class="yt-timestamp" data-t="00:05:04">[00:05:04]</a>

#### Tangent of Theta (tan θ)
Using "TOA" (Tangent = Opposite / Adjacent):
*   **Opposite side** to θ: 7 <a class="yt-timestamp" data-t="00:05:38">[00:05:38]</a>
*   **Adjacent side** to θ: 4 <a class="yt-timestamp" data-t="00:05:46">[00:05:46]</a>
tan θ = 7 / 4 <a class="yt-timestamp" data-t="00:05:51">[00:05:51]</a>

## Example 2: 30-60-90 Triangle

Consider a [[right_triangles_and_trigonometry | right triangle]] with sides 2, 2√3, and hypotenuse 4 <a class="yt-timestamp" data-t="00:06:17">[00:06:17]</a>.

### Verifying with Pythagorean Theorem
(2√3)² + 2² = (4 × 3) + 4 = 12 + 4 = 16 <a class="yt-timestamp" data-t="00:06:42">[00:06:42]</a>
This equals 4², confirming it satisfies the theorem <a class="yt-timestamp" data-t="00:06:53">[00:06:53]</a>.

This specific triangle is a [[trigonometric_functions_in_30_60_90_triangles | 30-60-90 triangle]] <a class="yt-timestamp" data-t="00:07:07">[00:07:07]</a>, where:
*   The side opposite the 30-degree angle is half the hypotenuse <a class="yt-timestamp" data-t="00:07:27">[00:07:27]</a>.
*   The side opposite the 60-degree angle is √3 times the side opposite the 30-degree angle <a class="yt-timestamp" data-t="00:07:31">[00:07:31]</a>.

### [[trigonometric_ratios_and_angle_identification | Trigonometric Ratios]] for 30 Degrees

Let's find the [[core_trigonometric_functions_sine_cosine_and_tangent | trigonometric ratios]] for the 30-degree angle <a class="yt-timestamp" data-t="00:07:51">[00:07:51]</a>.

#### Sine of 30 Degrees (sin 30°)
Using "SOH" (Sine = Opposite / Hypotenuse):
*   **Opposite side** to 30°: 2 <a class="yt-timestamp" data-t="00:08:22">[00:08:22]</a>
*   **Hypotenuse**: 4 <a class="yt-timestamp" data-t="00:08:30">[00:08:30]</a>
sin 30° = 2 / 4 = 1/2 <a class="yt-timestamp" data-t="00:08:32">[00:08:32]</a>

#### Cosine of 30 Degrees (cos 30°)
Using "CAH" (Cosine = Adjacent / Hypotenuse):
*   **Adjacent side** to 30°: 2√3 <a class="yt-timestamp" data-t="00:09:05">[00:09:05]</a>
*   **Hypotenuse**: 4 <a class="yt-timestamp" data-t="00:09:12">[00:09:12]</a>
cos 30° = 2√3 / 4 = √3 / 2 <a class="yt-timestamp" data-t="00:09:16">[00:09:16]</a>

#### Tangent of 30 Degrees (tan 30°)
Using "TOA" (Tangent = Opposite / Adjacent):
*   **Opposite side** to 30°: 2 <a class="yt-timestamp" data-t="00:09:38">[00:09:38]</a>
*   **Adjacent side** to 30°: 2√3 <a class="yt-timestamp" data-t="00:09:46">[00:09:46]</a>
tan 30° = 2 / (2√3) = 1 / √3 <a class="yt-timestamp" data-t="00:09:54">[00:09:54]</a>

To rationalize the denominator:
tan 30° = (1 × √3) / (√3 × √3) = √3 / 3 <a class="yt-timestamp" data-t="00:09:56">[00:09:56]</a>

### [[trigonometric_ratios_and_angle_identification | Trigonometric Ratios]] for 60 Degrees

Now, let's find the [[core_trigonometric_functions_sine_cosine_and_tangent | trigonometric ratios]] for the 60-degree angle in the same triangle <a class="yt-timestamp" data-t="00:10:17">[00:10:17]</a>.

#### Sine of 60 Degrees (sin 60°)
Using "SOH" (Sine = Opposite / Hypotenuse):
*   **Opposite side** to 60°: 2√3 <a class="yt-timestamp" data-t="00:10:36">[00:10:36]</a>
*   **Hypotenuse**: 4 <a class="yt-timestamp" data-t="00:10:50">[00:10:50]</a>
sin 60° = 2√3 / 4 = √3 / 2 <a class="yt-timestamp" data-t="00:10:54">[00:10:54]</a>

#### Cosine of 60 Degrees (cos 60°)
Using "CAH" (Cosine = Adjacent / Hypotenuse):
*   **Adjacent side** to 60°: 2 <a class="yt-timestamp" data-t="00:11:13">[00:11:13]</a>
*   **Hypotenuse**: 4 <a class="yt-timestamp" data-t="00:11:13">[00:11:13]</a>
cos 60° = 2 / 4 = 1/2 <a class="yt-timestamp" data-t="00:11:17">[00:11:17]</a>

#### Tangent of 60 Degrees (tan 60°)
Using "TOA" (Tangent = Opposite / Adjacent):
*   **Opposite side** to 60°: 2√3 <a class="yt-timestamp" data-t="00:11:34">[00:11:34]</a>
*   **Adjacent side** to 60°: 2 <a class="yt-timestamp" data-t="00:11:39">[00:11:39]</a>
tan 60° = 2√3 / 2 = √3 <a class="yt-timestamp" data-t="00:11:48">[00:11:48]</a>

### Relationships Noted
*   sin 30° (1/2) is equal to cos 60° (1/2) <a class="yt-timestamp" data-t="00:11:54">[00:11:54]</a>.
*   cos 30° (√3/2) is equal to sin 60° (√3/2) <a class="yt-timestamp" data-t="00:11:57">[00:11:57]</a>.
