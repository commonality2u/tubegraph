## Understanding Right Triangles

Trigonometry initially focuses on [[understanding_right_triangles | right triangles]] <a class="yt-timestamp" data-t="00:00:26">[00:00:26]</a>. A [[understanding_right_triangles | right triangle]] is defined by having one angle that measures 90 degrees <a class="yt-timestamp" data-t="00:00:33">[00:00:33]</a>.

### Key Terms
*   **Right Angle**: The 90-degree angle, typically marked with a square symbol <a class="yt-timestamp" data-t="00:00:37">[00:00:37]</a>.
*   **Hypotenuse**: The side opposite the [[understanding_right_triangles | right angle]] <a class="yt-timestamp" data-t="00:01:16">[00:01:16]</a>. It is always the longest side of a [[understanding_right_triangles | right triangle]] <a class="yt-timestamp" data-t="00:01:20">[00:01:20]</a>.
    *   *Example*: In a triangle with sides 3, 4, and 5, the hypotenuse is 5 <a class="yt-timestamp" data-t="00:01:07">[00:01:07]</a>. This configuration adheres to the [[applying_the_pythagorean_theorem | Pythagorean theorem]] ($3^2 + 4^2 = 5^2$) <a class="yt-timestamp" data-t="00:01:32">[00:01:32]</a>.
*   **Opposite Side**: The side directly across from the angle of interest <a class="yt-timestamp" data-t="00:04:00">[00:04:00]</a>.
*   **Adjacent Side**: The side next to the angle of interest that is *not* the hypotenuse <a class="yt-timestamp" data-t="00:05:09">[00:05:09]</a>.

## Basic Trigonometric Functions

The core [[trigonometric_functions_and_their_definitions | trigonometric functions]] are [[basic_trigonometric_functions_sine_cosine_and_tangent | sine]], [[basic_trigonometric_functions_sine_cosine_and_tangent | cosine]], and [[basic_trigonometric_functions_sine_cosine_and_tangent | tangent]] <a class="yt-timestamp" data-t="00:02:01">[00:02:01]</a>. These functions define the ratios of specific sides for any given angle within a [[understanding_right_triangles | right triangle]] <a class="yt-timestamp" data-t="00:02:15">[00:02:15]</a>.

### SOH CAH TOA Mnemonic
This mnemonic helps remember the definitions of the basic [[trigonometric_functions_and_their_definitions | trigonometric functions]] <a class="yt-timestamp" data-t="00:02:24">[00:02:24]</a>:
*   **SOH**: **S**ine = **O**pposite / **H**ypotenuse <a class="yt-timestamp" data-t="00:02:44">[00:02:44]</a>
*   **CAH**: **C**osine = **A**djacent / **H**ypotenuse <a class="yt-timestamp" data-t="00:03:03">[00:03:03]</a>
*   **TOA**: **T**angent = **O**pposite / **A**djacent <a class="yt-timestamp" data-t="00:03:17">[00:03:17]</a>

## [[application_of_trigonometry_in_right_triangles | Applying Trigonometry]] to Angles

### Example 1: Angle Theta (θ)

Consider a [[understanding_right_triangles | right triangle]] with sides 3, 4, and hypotenuse 5 <a class="yt-timestamp" data-t="00:00:57">[00:00:57]</a>. Let's find the [[calculating_trigonometric_functions_sine_cosine_tangent | trig ratios]] for angle theta (θ), located between the sides of length 4 and 5 <a class="yt-timestamp" data-t="00:03:30">[00:03:30]</a>.

*   **Sine of θ**:
    *   Opposite side: 3 <a class="yt-timestamp" data-t="00:04:12">[00:04:12]</a>
    *   Hypotenuse: 5 <a class="yt-timestamp" data-t="00:04:22">[00:04:22]</a>
    *   Sine(θ) = Opposite / Hypotenuse = 3 / 5 <a class="yt-timestamp" data-t="00:04:27">[00:04:27]</a>
*   **Cosine of θ**:
    *   Adjacent side: 4 <a class="yt-timestamp" data-t="00:05:37">[00:05:37]</a>
    *   Hypotenuse: 5 <a class="yt-timestamp" data-t="00:05:42">[00:05:42]</a>
    *   Cosine(θ) = Adjacent / Hypotenuse = 4 / 5 <a class="yt-timestamp" data-t="00:05:42">[00:05:42]</a>
*   **Tangent of θ**:
    *   Opposite side: 3 <a class="yt-timestamp" data-t="00:05:59">[00:05:59]</a>
    *   Adjacent side: 4 <a class="yt-timestamp" data-t="00:06:06">[00:06:06]</a>
    *   Tangent(θ) = Opposite / Adjacent = 3 / 4 <a class="yt-timestamp" data-t="00:06:09">[00:06:09]</a>

### Example 2: Angle X

Using the same [[understanding_right_triangles | right triangle]] (sides 3, 4, hypotenuse 5) <a class="yt-timestamp" data-t="00:06:34">[00:06:34]</a>, let's now consider angle X, located at the top vertex <a class="yt-timestamp" data-t="00:07:06">[00:07:06]</a>. Note how the opposite and adjacent sides change relative to this new angle.

*   **Sine of X**:
    *   Opposite side to X: 4 <a class="yt-timestamp" data-t="00:07:23">[00:07:23]</a>
    *   Hypotenuse: 5 <a class="yt-timestamp" data-t="00:07:44">[00:07:44]</a>
    *   Sine(X) = Opposite / Hypotenuse = 4 / 5 <a class="yt-timestamp" data-t="00:07:46">[00:07:46]</a>
*   **Cosine of X**:
    *   Adjacent side to X: 3 <a class="yt-timestamp" data-t="00:08:07">[00:08:07]</a>
    *   Hypotenuse: 5 <a class="yt-timestamp" data-t="00:08:14">[00:08:14]</a>
    *   Cosine(X) = Adjacent / Hypotenuse = 3 / 5 <a class="yt-timestamp" data-t="00:08:14">[00:08:14]</a>
*   **Tangent of X**:
    *   Opposite side to X: 4 <a class="yt-timestamp" data-t="00:08:34">[00:08:34]</a>
    *   Adjacent side to X: 3 <a class="yt-timestamp" data-t="00:08:38">[00:08:38]</a>
    *   Tangent(X) = Opposite / Adjacent = 4 / 3 <a class="yt-timestamp" data-t="00:08:38">[00:08:38]</a>

The definitions using "soh cah toa" are particularly useful for angles between 0 and 90 degrees <a class="yt-timestamp" data-t="00:08:57">[00:08:57]</a>. For angles larger than 90 degrees, a more generalized definition of [[trigonometric_functions_and_their_definitions | trigonometric functions]] is introduced <a class="yt-timestamp" data-t="00:09:06">[00:09:06]</a>.
