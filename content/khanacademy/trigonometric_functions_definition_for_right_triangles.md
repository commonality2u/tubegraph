This article provides further examples to solidify understanding of [[trigonometric_functions_sine_cosine_and_tangent|trigonometric functions]]. The definitions discussed here apply specifically to [[properties_of_right_triangles|right triangles]] <a class="yt-timestamp" data-t="00:00:15">[00:00:15]</a>. For angles not directly part of [[properties_of_right_triangles|right triangles]], it is often necessary to construct one <a class="yt-timestamp" data-t="00:00:23">[00:00:23]</a>.

## The SOH CAH TOA Mnemonic

A helpful mnemonic for remembering the definitions of [[trigonometric_functions_sine_cosine_and_tangent|sine]], [[trigonometric_functions_sine_cosine_and_tangent|cosine]], and [[trigonometric_functions_sine_cosine_and_tangent|tangent]] is "SOH CAH TOA" <a class="yt-timestamp" data-t="00:02:09">[00:02:09]</a>:

*   **SOH**: **S**ine = **O**pposite / **H**ypotenuse <a class="yt-timestamp" data-t="00:04:19">[00:04:19]</a>
*   **CAH**: **C**osine = **A**djacent / **H**ypotenuse <a class="yt-timestamp" data-t="00:02:43">[00:02:43]</a>
*   **TOA**: **T**angent = **O**pposite / **A**djacent <a class="yt-timestamp" data-t="00:05:27">[00:05:27]</a>

The "opposite" side is the side across from the angle, while the "adjacent" side is the side next to the angle that is not the hypotenuse <a class="yt-timestamp" data-t="00:03:04">[00:03:04]</a>. The "hypotenuse" is always the longest side, opposite the right angle <a class="yt-timestamp" data-t="00:02:55">[00:02:55]</a>.

## Example 1: Calculating [[trigonometric_functions_sine_cosine_and_tangent|Trig Functions]] for an Angle Theta

Consider a [[properties_of_right_triangles|right triangle]] with side lengths 7 and 4 <a class="yt-timestamp" data-t="00:00:31">[00:00:31]</a>.

### Calculating the Hypotenuse

First, find the length of the hypotenuse, `h`, using the [[Pythagorean theorem in right triangles|Pythagorean theorem]]: `h² = a² + b²` <a class="yt-timestamp" data-t="00:00:52">[00:00:52]</a>.
*   `h² = 7² + 4²` <a class="yt-timestamp" data-t="00:00:45">[00:00:45]</a>
*   `h² = 49 + 16` <a class="yt-timestamp" data-t="00:01:04">[00:01:04]</a>
*   `h² = 65` <a class="yt-timestamp" data-t="00:01:18">[00:01:18]</a>
*   `h = √65` <a class="yt-timestamp" data-t="00:01:33">[00:01:33]</a> (This cannot be simplified further <a class="yt-timestamp" data-t="00:01:42">[00:01:42]</a>).

### Calculating [[trigonometric_functions_sine_cosine_and_tangent|Cosine]] of Theta

Let's find the [[trigonometric_functions_sine_cosine_and_tangent|cosine]] of the angle theta located at the top of the triangle <a class="yt-timestamp" data-t="00:02:02">[00:02:02]</a>.
*   Using CAH: Cosine = Adjacent / Hypotenuse <a class="yt-timestamp" data-t="00:02:43">[00:02:43]</a>.
*   For theta, the adjacent side is 4 <a class="yt-timestamp" data-t="00:03:07">[00:03:07]</a> and the hypotenuse is √65 <a class="yt-timestamp" data-t="00:03:17">[00:03:17]</a>.
*   `cos(theta) = 4 / √65` <a class="yt-timestamp" data-t="00:03:21">[00:03:21]</a>.
*   To rationalize the denominator, multiply the numerator and denominator by √65 <a class="yt-timestamp" data-t="00:03:39">[00:03:39]</a>:
    *   `cos(theta) = (4 * √65) / (√65 * √65) = 4√65 / 65` <a class="yt-timestamp" data-t="00:03:54">[00:03:54]</a>.

### Calculating [[trigonometric_functions_sine_cosine_and_tangent|Sine]] of Theta

Next, find the [[trigonometric_functions_sine_cosine_and_tangent|sine]] of theta:
*   Using SOH: Sine = Opposite / Hypotenuse <a class="yt-timestamp" data-t="00:04:19">[00:04:19]</a>.
*   For theta, the opposite side is 7 <a class="yt-timestamp" data-t="00:04:34">[00:04:34]</a> and the hypotenuse is √65 <a class="yt-timestamp" data-t="00:04:47">[00:04:47]</a>.
*   `sin(theta) = 7 / √65` <a class="yt-timestamp" data-t="00:04:51">[00:04:51]</a>.
*   To rationalize the denominator:
    *   `sin(theta) = (7 * √65) / (√65 * √65) = 7√65 / 65` <a class="yt-timestamp" data-t="00:04:55">[00:04:55]</a>.

### Calculating [[trigonometric_functions_sine_cosine_and_tangent|Tangent]] of Theta

Finally, find the [[trigonometric_functions_sine_cosine_and_tangent|tangent]] of theta:
*   Using TOA: Tangent = Opposite / Adjacent <a class="yt-timestamp" data-t="00:05:27">[00:05:27]</a>.
*   For theta, the opposite side is 7 <a class="yt-timestamp" data-t="00:05:38">[00:05:38]</a> and the adjacent side is 4 <a class="yt-timestamp" data-t="00:05:46">[00:05:46]</a>.
*   `tan(theta) = 7 / 4` <a class="yt-timestamp" data-t="00:05:51">[00:05:51]</a>.

## Example 2: Calculating [[trigonometric_functions_sine_cosine_and_tangent|Trig Functions]] for a [[understanding_and_applying_trigonometric_ratios_in_306090_triangles|30-60-90 Triangle]]

Consider a [[properties_of_right_triangles|right triangle]] with a hypotenuse of length 4, and sides of length 2 and 2√3 <a class="yt-timestamp" data-t="00:06:17">[00:06:17]</a>.

### Verifying Side Lengths

Verify these lengths using the [[Pythagorean theorem in right triangles|Pythagorean theorem]]:
*   `(2√3)² + 2²` <a class="yt-timestamp" data-t="00:06:36">[00:06:36]</a>
*   `= (4 * 3) + 4` <a class="yt-timestamp" data-t="00:06:42">[00:06:42]</a>
*   `= 12 + 4 = 16` <a class="yt-timestamp" data-t="00:06:49">[00:06:49]</a>
*   Since `16 = 4²`, the sides satisfy the [[Pythagorean theorem in right triangles|Pythagorean theorem]] <a class="yt-timestamp" data-t="00:06:53">[00:06:53]</a>.
*   This specific configuration identifies it as a [[understanding_and_applying_trigonometric_ratios_in_306090_triangles|30-60-90 triangle]] <a class="yt-timestamp" data-t="00:07:07">[00:07:07]</a>. The side opposite the 30-degree angle is half the hypotenuse, and the side opposite the 60-degree angle is √3 times the side opposite the 30-degree angle <a class="yt-timestamp" data-t="00:07:27">[00:07:27]</a>.

### Calculating [[trigonometric_functions_sine_cosine_and_tangent|Trig Functions]] for 30 Degrees

For the 30-degree angle <a class="yt-timestamp" data-t="00:07:15">[00:07:15]</a>:
*   **[[trigonometric_functions_sine_cosine_and_tangent|Sine]] of 30 degrees**: Opposite (2) / Hypotenuse (4) = 2/4 = 1/2 <a class="yt-timestamp" data-t="00:08:26">[00:08:26]</a>.
*   **[[trigonometric_functions_sine_cosine_and_tangent|Cosine]] of 30 degrees**: Adjacent (2√3) / Hypotenuse (4) = 2√3 / 4 = √3 / 2 <a class="yt-timestamp" data-t="00:09:05">[00:09:05]</a>.
*   **[[trigonometric_functions_sine_cosine_and_tangent|Tangent]] of 30 degrees**: Opposite (2) / Adjacent (2√3) = 2 / (2√3) = 1/√3 <a class="yt-timestamp" data-t="00:09:38">[00:09:38]</a>.
    *   Rationalizing the denominator: `(1 * √3) / (√3 * √3) = √3 / 3` <a class="yt-timestamp" data-t="00:09:56">[00:09:56]</a>.

### Calculating [[trigonometric_functions_sine_cosine_and_tangent|Trig Functions]] for 60 Degrees

For the 60-degree angle <a class="yt-timestamp" data-t="00:07:23">[00:07:23]</a>:
*   **[[trigonometric_functions_sine_cosine_and_tangent|Sine]] of 60 degrees**: Opposite (2√3) / Hypotenuse (4) = 2√3 / 4 = √3 / 2 <a class="yt-timestamp" data-t="00:10:36">[00:10:36]</a>.
*   **[[trigonometric_functions_sine_cosine_and_tangent|Cosine]] of 60 degrees**: Adjacent (2) / Hypotenuse (4) = 2 / 4 = 1/2 <a class="yt-timestamp" data-t="00:11:10">[00:11:10]</a>.
*   **[[trigonometric_functions_sine_cosine_and_tangent|Tangent]] of 60 degrees**: Opposite (2√3) / Adjacent (2) = 2√3 / 2 = √3 <a class="yt-timestamp" data-t="00:11:34">[00:11:34]</a>.

Notice the relationships: `sin(30°) = cos(60°) = 1/2` and `cos(30°) = sin(60°) = √3 / 2` <a class="yt-timestamp" data-t="00:11:54">[00:11:54]</a>. The tangent values are also inverses of each other <a class="yt-timestamp" data-t="00:12:01">[00:12:01]</a>.
