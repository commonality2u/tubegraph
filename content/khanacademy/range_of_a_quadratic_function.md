The range of a function defines the set of all possible output values (often represented as 'y' or 'f(x)') that the function can produce <a class="yt-timestamp" data-t="01:16:00">[01:16:00]</a>. For [[quadratic_polynomials | quadratic functions]], which are typically in the form `f(x) = ax^2 + bx + c`, their [[graphing_a_quadratic_function | graph]] is a parabola <a class="yt-timestamp" data-t="01:38:00">[01:38:00]</a>. The shape of a parabola significantly influences its range.

## Understanding the Domain

Before determining the range, it's useful to consider the [[domain_of_a_quadratic_function | domain]]. For any [[quadratic_polynomials | quadratic function]], the domain (the set of valid input 'x' values) is all real numbers <a class="yt-timestamp" data-t="00:30:00">[00:30:00]</a>. This means you can substitute any real number for 'x' into the function, and it will always produce a real number output <a class="yt-timestamp" data-t="01:05:00">[01:05:00]</a>.

## Role of the Vertex in Determining Range

The range of a [[quadratic_polynomials | quadratic function]] is constrained by its **vertex** <a class="yt-timestamp" data-t="01:51:00">[01:51:00]</a>.
*   If the parabola opens upwards (when the 'a' coefficient in `ax^2 + bx + c` is positive), the vertex represents the minimum output value the function can achieve <a class="yt-timestamp" data-t="01:51:00">[01:51:00]</a>, <a class="yt-timestamp" data-t="07:01:00">[07:01:00]</a>.
*   If the parabola opens downwards (when 'a' is negative), the vertex represents the maximum output value <a class="yt-timestamp" data-t="01:55:00">[01:55:00]</a>.

All other output values will extend infinitely in the opposite direction from the vertex.

### Calculating the Vertex

The x-coordinate of the vertex for a [[understanding_components_of_a_quadratic_equation | quadratic equation]] in the form `ax^2 + bx + c` can be found using the formula: `x = -b / (2a)` <a class="yt-timestamp" data-t="02:16:00">[02:16:00]</a>. This formula is derived from the [[introduction_to_the_quadratic_formula | quadratic formula]] which itself comes from [[methods_for_solving_quadratic_equations | completing the square]] <a class="yt-timestamp" data-t="02:18:00">[02:18:00]</a>.

Once the x-coordinate is found, substitute it back into the function to find the corresponding y-coordinate, which is the minimum or maximum value of the range.

### Example: `f(x) = 3x^2 + 6x - 2`

Let's determine the range for the function `f(x) = 3x^2 + 6x - 2` <a class="yt-timestamp" data-t="00:02:00">[00:02:00]</a>.

1.  **Identify coefficients**: In this function, `a = 3`, `b = 6`, and `c = -2` <a class="yt-timestamp" data-t="03:20:00">[03:20:00]</a>. Since `a` (3) is positive, the parabola opens upwards, meaning the vertex will be the minimum point <a class="yt-timestamp" data-t="01:43:00">[01:43:00]</a>, <a class="yt-timestamp" data-t="07:01:00">[07:01:00]</a>.

2.  **Calculate the x-coordinate of the vertex**:
    `x = -b / (2a)` <a class="yt-timestamp" data-t="02:16:00">[02:16:00]</a>
    `x = -6 / (2 * 3)` <a class="yt-timestamp" data-t="03:22:00">[03:22:00]</a>
    `x = -6 / 6` <a class="yt-timestamp" data-t="03:26:00">[03:26:00]</a>
    `x = -1` <a class="yt-timestamp" data-t="03:26:00">[03:26:00]</a>

3.  **Calculate the y-coordinate of the vertex**: Substitute `x = -1` back into the function `f(x)` <a class="yt-timestamp" data-t="02:49:00">[02:49:00]</a>:
    `f(-1) = 3(-1)^2 + 6(-1) - 2` <a class="yt-timestamp" data-t="02:53:00">[02:53:00]</a>
    `f(-1) = 3(1) - 6 - 2` <a class="yt-timestamp" data-t="02:59:00">[02:59:00]</a>
    `f(-1) = 3 - 6 - 2` <a class="yt-timestamp" data-t="03:07:00">[03:07:00]</a>
    `f(-1) = -3 - 2` <a class="yt-timestamp" data-t="03:07:00">[03:07:00]</a>
    `f(-1) = -5` <a class="yt-timestamp" data-t="03:10:00">[03:10:00]</a>

    Thus, the vertex is at `(-1, -5)` <a class="yt-timestamp" data-t="03:10:00">[03:10:00]</a>, <a class="yt-timestamp" data-t="05:56:00">[05:56:00]</a>.

4.  **Determine the range**: Since the parabola opens upwards and its minimum point (vertex) is at `y = -5`, the function can take on any value greater than or equal to -5 <a class="yt-timestamp" data-t="07:16:00">[07:16:00]</a>.

### Visualizing the Range

Plotting a few points helps visualize the parabolic shape and confirm the vertex:
*   `f(-2) = 3(-2)^2 + 6(-2) - 2 = 12 - 12 - 2 = -2` <a class="yt-timestamp" data-t="02:33:00">[02:33:00]</a>
*   `f(-1) = -5` (vertex) <a class="yt-timestamp" data-t="03:10:00">[03:10:00]</a>
*   `f(0) = 3(0)^2 + 6(0) - 2 = -2` <a class="yt-timestamp" data-t="03:36:00">[03:36:00]</a>
*   `f(1) = 3(1)^2 + 6(1) - 2 = 3 + 6 - 2 = 7` <a class="yt-timestamp" data-t="04:04:00">[04:04:00]</a>

The points `(-2, -2)` and `(0, -2)` show symmetry around the vertex's x-value of -1 <a class="yt-timestamp" data-t="03:44:00">[03:44:00]</a>, <a class="yt-timestamp" data-t="06:42:00">[06:42:00]</a>. The graph shows that `f(x)` never goes below -5 <a class="yt-timestamp" data-t="07:32:00">[07:32:00]</a>.

## Conclusion

For the function `f(x) = 3x^2 + 6x - 2`:
*   The [[domain_of_a_quadratic_function | domain]] is all real numbers <a class="yt-timestamp" data-t="07:42:00">[07:42:00]</a>.
*   The range is all real numbers greater than or equal to -5 <a class="yt-timestamp" data-t="07:46:00">[07:46:00]</a>. This can be written as `y ≥ -5` or in interval notation as `[-5, ∞)`.
