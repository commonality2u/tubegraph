The range of a function refers to the set of all possible output values (often denoted as `y` or `f(x)`) that the function can produce <a class="yt-timestamp" data-t="00:01:16">[00:01:16]</a>. When thinking about a function's [[graphing_quadratic_functions | graph]], the range represents all the possible `y` values <a class="yt-timestamp" data-t="00:01:26">[00:01:26]</a>.

## Domain of a Quadratic Function

Before discussing the range, it's helpful to understand the [[domain_of_a_quadratic_function | domain]]. The [[domain_of_a_quadratic_function | domain]] is the set of all valid input values (or `x` values) for which the function is defined <a class="yt-timestamp" data-t="00:00:08">[00:00:08]</a>.

For a function like `f(x) = 3x^2 + 6x - 2`, any [[real_and_imaginary_numbers_in_quadratics | real number]] can be an input <a class="yt-timestamp" data-t="00:00:16">[00:00:16]</a>. You can square it, multiply it, add or subtract from it without any mathematical restrictions <a class="yt-timestamp" data-t="00:01:05">[00:01:05]</a>.
Most traditional numbers are considered [[real_and_imaginary_numbers_in_quadratics | real numbers]], as opposed to [[real_and_imaginary_numbers_in_quadratics | imaginary numbers]] or [[real_and_imaginary_numbers_in_quadratics | complex numbers]] <a class="yt-timestamp" data-t="00:00:48">[00:00:48]</a>.
Therefore, the [[domain_of_a_quadratic_function | domain]] for this quadratic function is all [[real_and_imaginary_numbers_in_quadratics | real numbers]] <a class="yt-timestamp" data-t="00:00:35">[00:00:35]</a>.

## Determining the Range of a Quadratic Function

A [[graphing_quadratic_functions | quadratic function]] like `f(x) = 3x^2 + 6x - 2` has a parabolic shape when [[graphing_quadratic_functions | graphed]] <a class="yt-timestamp" data-t="00:01:38">[00:01:38]</a>. This parabolic shape is crucial for determining its range.

The range of a parabola is determined by its [[vertex_of_a_quadratic_function | vertex]] and whether it opens upwards or downwards <a class="yt-timestamp" data-t="00:01:51">[00:01:51]</a>.
*   If the parabola opens upwards (like `f(x) = 3x^2 + 6x - 2`, where the coefficient of `x^2` is positive), the [[vertex_of_a_quadratic_function | vertex]] represents the minimum point of the function <a class="yt-timestamp" data-t="00:01:51">[00:01:51]</a>, <a class="yt-timestamp" data-t="00:07:03">[00:07:03]</a>. The function will take on all `y` values greater than or equal to the `y`-coordinate of the [[vertex_of_a_quadratic_function | vertex]].
*   If the parabola opens downwards, the [[vertex_of_a_quadratic_function | vertex]] represents the maximum point <a class="yt-timestamp" data-t="00:01:55">[00:01:55]</a>. The function will take on all `y` values less than or equal to the `y`-coordinate of the [[vertex_of_a_quadratic_function | vertex]].

### Finding the Vertex

The x-coordinate of the [[vertex_of_a_quadratic_function | vertex]] for a [[graphing_quadratic_functions | quadratic function]] in the standard form `ax^2 + bx + c` can be found using the formula `x = -b / (2a)` <a class="yt-timestamp" data-t="00:02:16">[00:02:16]</a>, <a class="yt-timestamp" data-t="00:03:14">[00:03:14]</a>. This formula is derived from the [[quadratic_formula_introduction | quadratic formula]] by completing the square <a class="yt-timestamp" data-t="00:02:18">[00:02:18]</a>.

For `f(x) = 3x^2 + 6x - 2`:
*   `a = 3`
*   `b = 6`
*   `c = -2`

The x-coordinate of the [[vertex_of_a_quadratic_function | vertex]] is `x = -6 / (2 * 3) = -6 / 6 = -1` <a class="yt-timestamp" data-t="00:03:22">[00:03:22]</a>, <a class="yt-timestamp" data-t="00:03:26">[00:03:26]</a>.

To find the y-coordinate of the [[vertex_of_a_quadratic_function | vertex]], substitute this x-value back into the function:
`f(-1) = 3(-1)^2 + 6(-1) - 2` <a class="yt-timestamp" data-t="00:02:53">[00:02:53]</a>
`f(-1) = 3(1) - 6 - 2` <a class="yt-timestamp" data-t="00:02:59">[00:02:59]</a>, <a class="yt-timestamp" data-t="00:03:02">[00:03:02]</a>
`f(-1) = 3 - 6 - 2` <a class="yt-timestamp" data-t="00:03:07">[00:03:07]</a>
`f(-1) = -3 - 2` <a class="yt-timestamp" data-t="00:03:07">[00:03:07]</a>
`f(-1) = -5` <a class="yt-timestamp" data-t="00:03:10">[00:03:10]</a>

So, the [[vertex_of_a_quadratic_function | vertex]] of the function `f(x) = 3x^2 + 6x - 2` is at `(-1, -5)` <a class="yt-timestamp" data-t="00:03:10">[00:03:10]</a>.

### Graphing to Visualize the Range

To further understand the range, we can [[graphing_quadratic_functions | graph]] the function by plotting several points:

| x      | f(x) (Calculations)                          | f(x)   |
| :----- | :------------------------------------------- | :----- |
| -2     | `3(-2)^2 + 6(-2) - 2 = 3(4) - 12 - 2 = 12 - 12 - 2` | -2     |
| -1     | `3(-1)^2 + 6(-1) - 2 = 3(1) - 6 - 2 = 3 - 6 - 2`    | -5     |
| 0      | `3(0)^2 + 6(0) - 2`                          | -2     |
| 1      | `3(1)^2 + 6(1) - 2 = 3(1) + 6 - 2 = 3 + 6 - 2`    | 7      |

The calculated points are: `(-2, -2)` <a class="yt-timestamp" data-t="00:02:48">[00:02:48]</a>, `(-1, -5)` <a class="yt-timestamp" data-t="00:03:10">[00:03:10]</a>, `(0, -2)` <a class="yt-timestamp" data-t="00:03:39">[00:03:39]</a>, and `(1, 7)` <a class="yt-timestamp" data-t="00:04:15">[00:04:15]</a>.
The parabola opens upwards because the coefficient of `x^2` (which is 3) is positive <a class="yt-timestamp" data-t="00:01:43">[00:01:43]</a>. The [[vertex_of_a_quadratic_function | vertex]] `(-1, -5)` is the lowest point on the [[graphing_quadratic_functions | graph]] <a class="yt-timestamp" data-t="00:07:05">[00:07:05]</a>.

As `x` moves away from the [[vertex_of_a_quadratic_function | vertex]] (either to the left or right), the `f(x)` values increase indefinitely <a class="yt-timestamp" data-t="00:07:22">[00:07:22]</a>, <a class="yt-timestamp" data-t="00:07:37">[00:07:37]</a>. The lowest `f(x)` value achieved is -5, at the [[vertex_of_a_quadratic_function | vertex]] <a class="yt-timestamp" data-t="00:07:19">[00:07:19]</a>. The function will never produce an output less than -5 <a class="yt-timestamp" data-t="00:07:32">[00:07:32]</a>.

### Conclusion

Given that the parabola opens upwards and its minimum point (the [[vertex_of_a_quadratic_function | vertex]]) is at `(-1, -5)`, the range of `f(x) = 3x^2 + 6x - 2` is all [[real_and_imaginary_numbers_in_quadratics | real numbers]] greater than or equal to -5 <a class="yt-timestamp" data-t="00:07:52">[00:07:52]</a>, <a class="yt-timestamp" data-t="00:08:00">[00:08:00]</a>.
This can be expressed as `f(x) ≥ -5` or `[-5, ∞)`.
