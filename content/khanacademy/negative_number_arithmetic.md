## [[Understanding negative numbers on a number line | Understanding Subtraction on a Number Line]]

When subtracting a positive number, you move to the left on the number line.

### Example 1: Subtracting a larger positive from a smaller positive

Consider the expression `2 - 3` <a class="yt-timestamp" data-t="00:00:03">[00:00:03]</a>.
Starting at `2` on the number line, subtracting `3` means moving `3` units to the left <a class="yt-timestamp" data-t="00:00:41">[00:00:41]</a>.
<a class="yt-timestamp" data-t="00:00:23">[00:00:23]</a>
- Start at `2` <a class="yt-timestamp" data-t="00:00:34">[00:00:34]</a>.
- Move `3` units to the left (1, 2, 3) <a class="yt-timestamp" data-t="00:00:44">[00:00:44]</a>.
- This results in `-1` <a class="yt-timestamp" data-t="00:00:48">[00:00:48]</a>.
Therefore, `2 - 3 = -1` <a class="yt-timestamp" data-t="00:00:51">[00:00:51]</a>.

### Example 2: Subtracting a positive from a negative

Consider the expression `-2 - 3` <a class="yt-timestamp" data-t="00:00:54">[00:00:54]</a>.
Starting at `-2` on the number line, subtracting `3` means moving `3` units to the left <a class="yt-timestamp" data-t="00:01:30">[00:01:30]</a>.
<a class="yt-timestamp" data-t="00:01:06">[00:01:06]</a>
- Start at `-2` <a class="yt-timestamp" data-t="00:01:24">[00:01:24]</a>.
- Move `3` units to the left (1, 2, 3) <a class="yt-timestamp" data-t="00:01:33">[00:01:33]</a>.
- This results in `-5` <a class="yt-timestamp" data-t="00:01:36">[00:01:36]</a>.
Therefore, `-2 - 3 = -5` <a class="yt-timestamp" data-t="00:01:39">[00:01:39]</a>.
In both scenarios, subtracting `3` means moving `3` units to the left on the number line <a class="yt-timestamp" data-t="00:01:40">[00:01:40]</a>.

## [[Understanding negative numbers on a number line | Understanding Addition on a Number Line]]

When [[adding_negative_numbers | adding a positive number]], you move to the right on the number line.

### Example 3: Adding a positive to a negative

Consider the expression `-2 + 3` <a class="yt-timestamp" data-t="00:01:57">[00:01:57]</a>.
Starting at `-2` on the number line, adding `3` means moving `3` units to the right <a class="yt-timestamp" data-t="00:02:30">[00:02:30]</a>.
<a class="yt-timestamp" data-t="00:02:10">[00:02:10]</a>
- Start at `-2` <a class="yt-timestamp" data-t="00:02:23">[00:02:23]</a>.
- Move `3` units to the right (1, 2, 3) <a class="yt-timestamp" data-t="00:02:31">[00:02:31]</a>.
- This results in `1` <a class="yt-timestamp" data-t="00:02:36">[00:02:36]</a>.
Therefore, `-2 + 3 = 1` <a class="yt-timestamp" data-t="00:02:36">[00:02:36]</a>.

## [[Understanding subtraction of a negative as addition | Subtracting a Negative Number]]

Subtracting a negative number is equivalent to [[understanding_subtraction_of_a_negative_as_addition | adding the positive counterpart]] <a class="yt-timestamp" data-t="00:02:59">[00:02:59]</a>.

### Example 4: Subtracting a negative from a positive

Consider the expression `2 - (-3)` <a class="yt-timestamp" data-t="00:02:47">[00:02:47]</a>.
Subtracting a negative number is the same as adding a positive number <a class="yt-timestamp" data-t="00:03:01">[00:03:01]</a>.
So, `2 - (-3)` is equivalent to `2 + 3` <a class="yt-timestamp" data-t="00:03:04">[00:03:04]</a>.
Therefore, `2 - (-3) = 5` <a class="yt-timestamp" data-t="00:03:18">[00:03:18]</a>.

### Example 5: Subtracting a negative from a negative

Consider the expression `-2 - (-3)` <a class="yt-timestamp" data-t="00:03:23">[00:03:23]</a>.
Again, subtracting a negative is the same as adding a positive <a class="yt-timestamp" data-t="00:03:38">[00:03:38]</a>.
So, `-2 - (-3)` is equivalent to `-2 + 3` <a class="yt-timestamp" data-t="00:03:45">[00:03:45]</a>.
As shown in Example 3, `-2 + 3 = 1` <a class="yt-timestamp" data-t="00:03:52">[00:03:52]</a>.
Therefore, `-2 - (-3) = 1` <a class="yt-timestamp" data-t="00:04:01">[00:04:01]</a>.
