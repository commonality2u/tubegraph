## Understanding Ratios

[[Understanding ratios | Ratios]] are mathematical expressions used to compare quantities <a class="yt-timestamp" data-t="00:00:04"></a>. They indicate how much of one quantity there is in relation to another <a class="yt-timestamp" data-t="00:00:36"></a>.

For example, if there are 10 horses and 5 dogs, the ratio of horses to dogs can be expressed in several ways <a class="yt-timestamp" data-t="00:00:15"></a> <a class="yt-timestamp" data-t="00:00:26"></a>:
*   Using a colon: 10:5 <a class="yt-timestamp" data-t="00:00:45"></a>
*   As a fraction: 10/5 <a class="yt-timestamp" data-t="00:00:51"></a> <a class="yt-timestamp" data-t="00:00:53"></a>
*   Using the word "to": 10 to 5 <a class="yt-timestamp" data-t="00:00:57"></a> <a class="yt-timestamp" data-t="00:00:59"></a>

These different forms all convey the same comparison <a class="yt-timestamp" data-t="00:01:02"></a>. The quantity mentioned first in the ratio (or written on top in a fraction) corresponds to the first number in the expression <a class="yt-timestamp" data-t="00:01:05"></a> <a class="yt-timestamp" data-t="00:01:07"></a>.

## Simplifying Ratios

While a ratio like 10:5 accurately compares quantities, it can often be [[Simplifying expressions | simplified]] to its reduced form to make the comparison clearer <a class="yt-timestamp" data-t="00:01:12"></a> <a class="yt-timestamp" data-t="00:01:14"></a> <a class="yt-timestamp" data-t="00:01:16"></a> <a class="yt-timestamp" data-t="00:01:18"></a> <a class="yt-timestamp" data-t="00:01:21"></a> <a class="yt-timestamp" data-t="00:01:22"></a>. [[Simplifying ratios | Simplifying ratios]] is similar to [[Simplifying fractions | simplifying fractions]] <a class="yt-timestamp" data-t="00:02:18"></a>.

To [[Simplifying fractions after conversion | simplify a ratio]], find the largest number that divides evenly into both (or all) quantities in the ratio <a class="yt-timestamp" data-t="00:02:04"></a> <a class="yt-timestamp" data-t="00:02:06"></a>. This is known as the greatest common divisor (GCD) <a class="yt-timestamp" data-t="00:02:08"></a>. Divide each part of the ratio by this GCD <a class="yt-timestamp" data-t="00:02:15"></a>.

For the ratio of 10 horses to 5 dogs:
1.  Identify the quantities: 10 and 5 <a class="yt-timestamp" data-t="00:00:40"></a>.
2.  Find the greatest common divisor of 10 and 5, which is 5 <a class="yt-timestamp" data-t="00:02:12"></a>.
3.  Divide both numbers by 5:
    *   10 ÷ 5 = 2 <a class="yt-timestamp" data-t="00:02:15"></a>
    *   5 ÷ 5 = 1 <a class="yt-timestamp" data-t="00:02:15"></a>
4.  The simplified ratio is 2:1 <a class="yt-timestamp" data-t="00:01:38"></a> <a class="yt-timestamp" data-t="00:01:55"></a> <a class="yt-timestamp" data-t="00:02:01"></a> <a class="yt-timestamp" data-t="00:02:20"></a> <a class="yt-timestamp" data-t="00:02:22"></a>. This means for every 2 horses, there is 1 dog <a class="yt-timestamp" data-t="00:01:43"></a> <a class="yt-timestamp" data-t="00:02:37"></a> <a class="yt-timestamp" data-t="00:02:39"></a>.

### Importance of Order

The order in which quantities are compared is crucial <a class="yt-timestamp" data-t="00:03:09"></a> <a class="yt-timestamp" data-t="00:03:34"></a> <a class="yt-timestamp" data-t="00:06:37"></a>.
If the question asked for the ratio of **dogs to horses**:
1.  The initial ratio would be 5:10 <a class="yt-timestamp" data-t="00:03:15"></a>.
2.  Dividing both by their GCD (5), the simplified ratio becomes 1:2 <a class="yt-timestamp" data-t="00:03:20"></a> <a class="yt-timestamp" data-t="00:03:25"></a>. This means for every 1 dog, there are 2 horses <a class="yt-timestamp" data-t="00:03:23"></a>.

The general convention is to express ratios in their simplest, or reduced, form <a class="yt-timestamp" data-t="00:03:45"></a> <a class="yt-timestamp" data-t="00:03:48"></a> <a class="yt-timestamp" data-t="00:03:51"></a>.

### Ratios with Multiple Quantities

The same principle applies to ratios involving more than two quantities.
For example, with 20 apples, 40 oranges, and 60 strawberries <a class="yt-timestamp" data-t="00:03:57"></a> <a class="yt-timestamp" data-t="00:04:05"></a> <a class="yt-timestamp" data-t="00:04:12"></a>:
1.  The initial ratio of apples to oranges to strawberries is 20:40:60 <a class="yt-timestamp" data-t="00:04:25"></a> <a class="yt-timestamp" data-t="00:04:28"></a> <a class="yt-timestamp" data-t="00:04:29"></a> <a class="yt-timestamp" data-t="00:04:48"></a> <a class="yt-timestamp" data-t="00:05:03"></a> <a class="yt-timestamp" data-t="00:05:04"></a> <a class="yt-timestamp" data-t="00:05:06"></a>.
2.  Find the largest number that divides into all three quantities. In this case, the GCD of 20, 40, and 60 is 20 <a class="yt-timestamp" data-t="00:05:15"></a> <a class="yt-timestamp" data-t="00:05:18"></a> <a class="yt-timestamp" data-t="00:05:19"></a> <a class="yt-timestamp" data-t="00:05:22"></a> <a class="yt-timestamp" data-t="00:05:25"></a> <a class="yt-timestamp" data-t="00:05:27"></a>.
3.  Divide each number by 20:
    *   20 ÷ 20 = 1 <a class="yt-timestamp" data-t="00:05:28"></a> <a class="yt-timestamp" data-t="00:05:34"></a>
    *   40 ÷ 20 = 2 <a class="yt-timestamp" data-t="00:05:39"></a>
    *   60 ÷ 20 = 3 <a class="yt-timestamp" data-t="00:05:45"></a>
4.  The simplified ratio is 1:2:3 <a class="yt-timestamp" data-t="00:05:47"></a> <a class="yt-timestamp" data-t="00:05:50"></a> <a class="yt-timestamp" data-t="00:05:54"></a>. This means for every 1 apple, there are 2 oranges and 3 strawberries <a class="yt-timestamp" data-t="00:05:34"></a> <a class="yt-timestamp" data-t="00:05:39"></a> <a class="yt-timestamp" data-t="00:05:45"></a>.

If the order is changed, for example, to strawberries to oranges to apples, the numbers in the ratio must also change to match the new order <a class="yt-timestamp" data-t="00:06:06"></a> <a class="yt-timestamp" data-t="00:06:08"></a> <a class="yt-timestamp" data-t="00:06:13"></a> <a class="yt-timestamp" data-t="00:06:18"></a> <a class="yt-timestamp" data-t="00:06:20"></a>: 3:2:1 <a class="yt-timestamp" data-t="00:06:22"></a> <a class="yt-timestamp" data-t="00:06:29"></a> <a class="yt-timestamp" data-t="00:06:31"></a> <a class="yt-timestamp" data-t="00:06:34"></a>.

## Finding Quantities from Given Ratios

Ratios can also be used to determine the exact quantity of items when a total is known <a class="yt-timestamp" data-t="00:06:42"></a> <a class="yt-timestamp" data-t="00:06:46"></a> <a class="yt-timestamp" data-t="00:06:48"></a> <a class="yt-timestamp" data-t="00:06:50"></a> <a class="yt-timestamp" data-t="00:06:54"></a> <a class="yt-timestamp" data-t="00:06:56"></a>.

### Example: Boys and Girls in a Classroom

If the ratio of boys to girls in a classroom is 2:3, and there are 40 students in total <a class="yt-timestamp" data-t="00:07:07"></a> <a class="yt-timestamp" data-t="00:07:12"></a> <a class="yt-timestamp" data-t="00:07:16"></a> <a class="yt-timestamp" data-t="00:07:23"></a> <a class="yt-timestamp" data-t="00:07:23"></a> <a class="yt-timestamp" data-t="00:07:23"></a> <a class="yt-timestamp" data-t="00:07:23"></a> <a class="yt-timestamp" data-t="00:07:37"></a>:

1.  **Determine the "group size"**: Add the parts of the ratio together: 2 boys + 3 girls = 5 students per "group" <a class="yt-timestamp" data-t="00:08:27"></a> <a class="yt-timestamp" data-t="00:08:39"></a> <a class="yt-timestamp" data-t="00:08:50"></a> <a class="yt-timestamp" data-t="00:09:32"></a> <a class="yt-timestamp" data-t="00:10:45"></a> <a class="yt-timestamp" data-t="00:10:48"></a>.
2.  **Calculate the number of groups**: Divide the total number of students by the group size: 40 students ÷ 5 students/group = 8 groups <a class="yt-timestamp" data-t="00:08:53"></a> <a class="yt-timestamp" data-t="00:08:56"></a> <a class="yt-timestamp" data-t="00:09:01"></a> <a class="yt-timestamp" data-t="00:09:04"></a> <a class="yt-timestamp" data-t="00:09:07"></a> <a class="yt-timestamp" data-t="00:09:11"></a> <a class="yt-timestamp" data-t="00:09:13"></a> <a class="yt-timestamp" data-t="00:09:16"></a> <a class="yt-timestamp" data-t="00:09:28"></a> <a class="yt-timestamp" data-t="00:09:35"></a> <a class="yt-timestamp" data-t="00:09:38"></a> <a class="yt-timestamp" data-t="00:09:43"></a> <a class="yt-timestamp" data-t="00:09:50"></a> <a class="yt-timestamp" data-t="00:10:52"></a> <a class="yt-timestamp" data-t="00:10:53"></a> <a class="yt-timestamp" data-t="00:10:57"></a>.
3.  **Find the quantity for each part of the ratio**: Multiply the number of groups by the respective ratio part <a class="yt-timestamp" data-t="00:09:53"></a> <a class="yt-timestamp" data-t="00:10:07"></a>:
    *   Girls: 8 groups × 3 girls/group = 24 girls <a class="yt-timestamp" data-t="00:10:08"></a> <a class="yt-timestamp" data-t="00:10:11"></a> <a class="yt-timestamp" data-t="00:10:17"></a> <a class="yt-timestamp" data-t="00:10:59"></a> <a class="yt-timestamp" data-t="00:11:03"></a>.
    *   Boys: 8 groups × 2 boys/group = 16 boys <a class="yt-timestamp" data-t="00:10:25"></a> <a class="yt-timestamp" data-t="00:10:27"></a> <a class="yt-timestamp" data-t="00:10:30"></a> <a class="yt-timestamp" data-t="00:10:31"></a>.
    *   Alternatively, if you know the total and one quantity, subtract to find the other: 40 students - 24 girls = 16 boys <a class="yt-timestamp" data-t="00:10:32"></a> <a class="yt-timestamp" data-t="00:10:34"></a> <a class="yt-timestamp" data-t="00:10:36"></a> <a class="yt-timestamp" data-t="00:10:39"></a>.

This method can be applied to ratios with any number of quantities <a class="yt-timestamp" data-t="00:11:08"></a> <a class="yt-timestamp" data-t="00:11:16"></a> <a class="yt-timestamp" data-t="00:11:21"></a>. For example, if the ratio of sheep to chickens to pigs is 2:5:10, and there are 51 animals in total <a class="yt-timestamp" data-t="00:11:40"></a> <a class="yt-timestamp" data-t="00:11:45"></a> <a class="yt-timestamp" data-t="00:11:46"></a> <a class="yt-timestamp" data-t="00:11:49"></a> <a class="yt-timestamp" data-t="00:11:51"></a> <a class="yt-timestamp" data-t="00:12:02"></a> <a class="yt-timestamp" data-t="00:12:06"></a> <a class="yt-timestamp" data-t="00:12:10"></a> <a class="yt-timestamp" data-t="00:12:21"></a>:

1.  Sum the ratio parts: 2 + 5 + 10 = 17 animals per group <a class="yt-timestamp" data-t="00:12:34"></a> <a class="yt-timestamp" data-t="00:12:38"></a> <a class="yt-timestamp" data-t="00:12:48"></a> <a class="yt-timestamp" data-t="00:12:51"></a> <a class="yt-timestamp" data-t="00:12:52"></a> <a class="yt-timestamp" data-t="00:12:55"></a>.
2.  Calculate number of groups: 51 total animals ÷ 17 animals/group = 3 groups <a class="yt-timestamp" data-t="00:13:12"></a> <a class="yt-timestamp" data-t="00:13:16"></a> <a class="yt-timestamp" data-t="00:13:18"></a> <a class="yt-timestamp" data-t="00:13:22"></a> <a class="yt-timestamp" data-t="00:13:24"></a> <a class="yt-timestamp" data-t="00:13:35"></a>.
3.  Find number of chickens: 3 groups × 5 chickens/group = 15 chickens <a class="yt-timestamp" data-t="00:13:37"></a> <a class="yt-timestamp" data-t="00:13:40"></a> <a class="yt-timestamp" data-t="00:13:42"></a> <a class="yt-timestamp" data-t="00:13:43"></a> <a class="yt-timestamp" data-t="00:13:44"></a> <a class="yt-timestamp" data-t="00:13:45"></a> <a class="yt-timestamp" data-t="00:13:49"></a>.

This process, part of [[Finding quantities from given ratios | finding quantities from given ratios]], can be applied to find the number of sheep or pigs as well <a class="yt-timestamp" data-t="00:14:07"></a> <a class="yt-timestamp" data-t="00:14:10"></a>.
