The [[number_line_for_addition | number line]] is a visual tool that can be used to understand [[how_to_add_and_subtract_negative_numbers | addition and subtraction]] operations, especially when dealing with negative numbers <a class="yt-timestamp" data-t="00:00:21">[00:00:21]</a>.

## Basic Principles on the Number Line

*   **Subtraction**: When subtracting a number, you move to the left on the [[number_line_for_addition | number line]] <a class="yt-timestamp" data-t="00:00:41">[00:00:41]</a>.
*   **Addition**: When adding a number, you move to the right on the [[number_line_for_addition | number line]] <a class="yt-timestamp" data-t="00:02:30">[00:02:30]</a>.

## Examples of Subtraction

### Subtracting a Larger Number from a Smaller Number (e.g., 2 - 3)

When subtracting a larger positive number from a smaller positive number, the result will be a negative number <a class="yt-timestamp" data-t="00:00:13">[00:00:13]</a>.

To visualize `2 - 3` on a [[number_line_for_addition | number line]]:
1.  Start at 2 <a class="yt-timestamp" data-t="00:00:34">[00:00:34]</a>.
2.  Subtract 3 by moving 3 units to the left <a class="yt-timestamp" data-t="00:00:41">[00:00:41]</a>.
3.  Moving 1 unit left from 2 reaches 1, 2 units reaches 0, and 3 units reaches [[visualizing_negative_numbers_on_a_number_line | negative 1]] <a class="yt-timestamp" data-t="00:00:44">[00:00:44]</a>.
4.  Therefore, `2 - 3 = -1` <a class="yt-timestamp" data-t="00:00:51">[00:00:51]</a>.

### Subtracting a Positive Number from a Negative Number (e.g., -2 - 3)

To visualize `negative 2 minus 3` on a [[number_line_for_addition | number line]]:
1.  Start at [[visualizing_negative_numbers_on_a_number_line | negative 2]] <a class="yt-timestamp" data-t="00:01:24">[00:01:24]</a>.
2.  Subtract 3 by moving 3 units to the left from [[visualizing_negative_numbers_on_a_number_line | negative 2]] <a class="yt-timestamp" data-t="00:01:30">[00:01:30]</a>.
3.  Moving 1 unit left from [[visualizing_negative_numbers_on_a_number_line | negative 2]] reaches [[visualizing_negative_numbers_on_a_number_line | negative 3]], 2 units reaches [[visualizing_negative_numbers_on_a_number_line | negative 4]], and 3 units reaches [[visualizing_negative_numbers_on_a_number_line | negative 5]] <a class="yt-timestamp" data-t="00:01:33">[00:01:33]</a>.
4.  Therefore, `negative 2 minus 3 = negative 5` <a class="yt-timestamp" data-t="00:01:39">[00:01:39]</a>.

## Examples of Addition

### Adding a Positive Number to a Negative Number (e.g., -2 + 3)

To visualize `negative 2 plus 3` on a [[number_line_for_addition | number line]]:
1.  Start at [[visualizing_negative_numbers_on_a_number_line | negative 2]] <a class="yt-timestamp" data-t="00:02:27">[00:02:27]</a>.
2.  Add 3 by moving 3 units to the right <a class="yt-timestamp" data-t="00:02:30">[00:02:30]</a>.
3.  Moving 1 unit right from [[visualizing_negative_numbers_on_a_number_line | negative 2]] reaches [[visualizing_negative_numbers_on_a_number_line | negative 1]], 2 units reaches 0, and 3 units reaches positive 1 <a class="yt-timestamp" data-t="00:02:31">[00:02:31]</a>.
4.  Therefore, `negative 2 plus 3 = 1` <a class="yt-timestamp" data-t="00:02:37">[00:02:37]</a>.

## Subtracting Negative Numbers

Subtracting a negative number is equivalent to [[adding_negative_numbers | adding]] a positive number <a class="yt-timestamp" data-t="00:03:01">[00:03:01]</a>. This means `X - (-Y)` is the same as `X + Y` <a class="yt-timestamp" data-t="00:03:04">[00:03:04]</a>.

### Subtracting a Negative Number from a Positive Number (e.g., 2 - (-3))

1.  The expression `2 minus negative 3` is equivalent to `2 plus positive 3` <a class="yt-timestamp" data-t="00:03:04">[00:03:04]</a>.
2.  `2 + 3 = 5` <a class="yt-timestamp" data-t="00:03:18">[00:03:18]</a>.

### Subtracting a Negative Number from a Negative Number (e.g., -2 - (-3))

1.  The expression `negative 2 minus negative 3` simplifies to `negative 2 plus 3` <a class="yt-timestamp" data-t="00:03:45">[00:03:45]</a>.
2.  As shown previously, `negative 2 plus 3` is equal to 1 <a class="yt-timestamp" data-t="00:03:52">[00:03:52]</a>.
    *   Start at [[visualizing_negative_numbers_on_a_number_line | negative 2]] <a class="yt-timestamp" data-t="00:03:55">[00:03:55]</a>.
    *   Add 3 by moving 3 units to the right <a class="yt-timestamp" data-t="00:03:59">[00:03:59]</a>.
    *   This results in positive 1 <a class="yt-timestamp" data-t="00:04:01">[00:04:01]</a>.
