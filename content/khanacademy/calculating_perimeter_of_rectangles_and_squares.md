Perimeter is defined as the total distance around the outside of a shape <a class="yt-timestamp" data-t="00:00:16">[00:00:16]</a>. It can be thought of as the length of a fence required to enclose an area, or the length of tape needed to measure around a figure <a class="yt-timestamp" data-t="00:00:19">[00:00:19]</a>, <a class="yt-timestamp" data-t="00:00:20">[00:00:20]</a>, <a class="yt-timestamp" data-t="00:00:22">[00:00:22]</a>, <a class="yt-timestamp" data-t="00:00:26">[00:00:26]</a>. ## Calculating the Perimeter of Rectangles

A rectangle is a four-sided figure with four right angles <a class="yt-timestamp" data-t="00:00:33">[00:00:33]</a>, <a class="yt-timestamp" data-t="00:00:39">[00:00:39]</a>, <a class="yt-timestamp" data-t="00:00:41">[00:00:41]</a>. Its opposite sides are equal in length <a class="yt-timestamp" data-t="00:00:45">[00:00:45]</a>, <a class="yt-timestamp" data-t="00:00:47">[00:00:47]</a>.

To find the perimeter of a rectangle, sum the lengths of all its sides <a class="yt-timestamp" data-t="00:01:25">[00:01:25]</a>, <a class="yt-timestamp" data-t="00:01:28">[00:01:28]</a>.

### Example: Rectangle ABCD

Consider a rectangle ABCD <a class="yt-timestamp" data-t="00:00:28">[00:00:28]</a>.
*   Side AB is 7 units long <a class="yt-timestamp" data-t="00:01:03">[00:01:03]</a>.
*   Side BC is 5 units long <a class="yt-timestamp" data-t="00:01:07">[00:01:07]</a>.

Given the properties of a rectangle:
*   If AB = 7, then the opposite side DC also equals 7 <a class="yt-timestamp" data-t="00:01:49">[00:01:49]</a>, <a class="yt-timestamp" data-t="00:01:51">[00:01:51]</a>, <a class="yt-timestamp" data-t="00:01:54">[00:01:54]</a>.
*   If BC = 5, then the opposite side DA (or AD) also equals 5 <a class="yt-timestamp" data-t="00:01:55">[00:01:55]</a>, <a class="yt-timestamp" data-t="00:01:58">[00:01:58]</a>, <a class="yt-timestamp" data-t="00:02:02">[00:02:02]</a>.

Therefore, the perimeter of rectangle ABCD is:
Perimeter = AB + BC + DC + DA <a class="yt-timestamp" data-t="00:01:25">[00:01:25]</a>
Perimeter = 7 + 5 + 7 + 5 <a class="yt-timestamp" data-t="00:01:38">[00:01:38]</a>, <a class="yt-timestamp" data-t="00:01:40">[00:01:40]</a>, <a class="yt-timestamp" data-t="00:01:44">[00:01:44]</a>, <a class="yt-timestamp" data-t="00:01:45">[00:01:45]</a>, <a class="yt-timestamp" data-t="00:01:46">[00:01:46]</a>, <a class="yt-timestamp" data-t="00:02:03">[00:02:03]</a>
Perimeter = 12 + 12 <a class="yt-timestamp" data-t="00:02:03">[00:02:03]</a>, <a class="yt-timestamp" data-t="00:02:07">[00:02:07]</a>
Perimeter = 24 <a class="yt-timestamp" data-t="00:02:07">[00:02:07]</a>

## Calculating the Perimeter of Squares

A square is a special type of rectangle <a class="yt-timestamp" data-t="00:02:19">[00:02:19]</a>. It has four sides and four right angles, but all of its sides are equal in length <a class="yt-timestamp" data-t="00:02:22">[00:02:22]</a>, <a class="yt-timestamp" data-t="00:02:29">[00:02:29]</a>.

To find the perimeter of a square, you can either sum all four equal sides or multiply the length of one side by four. If 'x' represents the length of one side of a square, its perimeter is x + x + x + x = 4x <a class="yt-timestamp" data-t="00:03:01">[00:03:01]</a>, <a class="yt-timestamp" data-t="00:03:03">[00:03:03]</a>, <a class="yt-timestamp" data-t="00:03:09">[00:03:09]</a>, <a class="yt-timestamp" data-t="00:03:11">[00:03:11]</a>, <a class="yt-timestamp" data-t="00:03:12">[00:03:12]</a>, <a class="yt-timestamp" data-t="00:03:15">[00:03:15]</a>, <a class="yt-timestamp" data-t="00:03:17">[00:03:17]</a>, <a class="yt-timestamp" data-t="00:03:20">[00:03:20]</a>, <a class="yt-timestamp" data-t="00:03:25">[00:03:25]</a>.

### Example: Square ABCD with Known Perimeter

Suppose a square ABCD has a perimeter of 36 units <a class="yt-timestamp" data-t="00:02:46">[00:02:46]</a>, <a class="yt-timestamp" data-t="00:02:49">[00:02:49]</a>. To find the length of each side:
*   Let 'x' be the length of each side <a class="yt-timestamp" data-t="00:03:01">[00:03:01]</a>.
*   The perimeter is 4x <a class="yt-timestamp" data-t="00:03:20">[00:03:20]</a>, <a class="yt-timestamp" data-t="00:03:25">[00:03:25]</a>.
*   Set up the equation: 4x = 36 <a class="yt-timestamp" data-t="00:03:25">[00:03:25]</a>, <a class="yt-timestamp" data-t="00:03:28">[00:03:28]</a>.
*   Divide both sides by 4 to solve for x <a class="yt-timestamp" data-t="00:03:34">[00:03:34]</a>, <a class="yt-timestamp" data-t="00:03:37">[00:03:37]</a>.
*   x = 9 <a class="yt-timestamp" data-t="00:03:37">[00:03:37]</a>.

Thus, each side of the square is 9 units long <a class="yt-timestamp" data-t="00:03:42">[00:03:42]</a>, <a class="yt-timestamp" data-t="00:03:45">[00:03:45]</a>, <a class="yt-timestamp" data-t="00:03:47">[00:03:47]</a>, <a class="yt-timestamp" data-t="00:03:50">[00:03:50]</a>.
