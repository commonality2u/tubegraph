A [[definition_of_a_function | function]] is an abstract concept that takes an input, processes it, and produces a single, specific output based on that input <a class="yt-timestamp" data-t="00:00:06">[00:00:06]</a>.

## Function Notation and Evaluation
The most common notation for a [[definition_of_a_function | function]] is `f(x)`, where `f` is the name of the [[definition_of_a_function | function]] and `x` is the variable representing the input <a class="yt-timestamp" data-t="00:00:22">[00:00:22]</a>. When [[expressions_and_their_evaluation | evaluating]] a [[definition_of_a_function | function]], the input value replaces the variable `x` within the [[definition_of_a_function | function]]'s definition <a class="yt-timestamp" data-t="00:01:14">[00:01:14]</a>.

### Example 1: Piecewise Function
Consider a [[definition_of_a_function | function]] `f(x)` defined as:
*   `f(x) = x^2`, if x is even <a class="yt-timestamp" data-t="00:00:41">[00:00:41]</a>
*   `f(x) = x + 5`, if x is odd <a class="yt-timestamp" data-t="00:00:48">[00:00:48]</a>

To evaluate `f(2)`:
1.  Since 2 is an even number, use the `x^2` rule <a class="yt-timestamp" data-t="00:01:27">[00:01:27]</a>.
2.  Substitute 2 for `x`: `f(2) = 2^2 = 4` <a class="yt-timestamp" data-t="00:01:31">[00:01:31]</a>.

To evaluate `f(3)`:
1.  Since 3 is an odd number, use the `x + 5` rule <a class="yt-timestamp" data-t="00:01:54">[00:01:54]</a>.
2.  Substitute 3 for `x`: `f(3) = 3 + 5 = 8` <a class="yt-timestamp" data-t="00:01:57">[00:01:57]</a>.

### Example 2: Abstract Function
[[definition_of_a_function | Functions]] can operate on various types of inputs and perform complex transformations. For instance, a [[definition_of_a_function | function]] `h(a)` could be defined as "the next largest number that starts with the same letter as variable `a` (assuming English numbers)" <a class="yt-timestamp" data-t="00:02:30">[00:02:30]</a>.

To evaluate `h(2)`:
1.  "2" starts with 'T' <a class="yt-timestamp" data-t="00:03:09">[00:03:09]</a>.
2.  The next largest number starting with 'T' is "3" <a class="yt-timestamp" data-t="00:03:12">[00:03:12]</a>.
3.  Therefore, `h(2) = 3` <a class="yt-timestamp" data-t="00:03:14">[00:03:14]</a>.

To evaluate `h(8)`:
1.  "8" starts with 'E' <a class="yt-timestamp" data-t="00:03:27">[00:03:27]</a>.
2.  The next largest number starting with 'E' is "11" <a class="yt-timestamp" data-t="00:03:31">[00:03:31]</a>.
3.  Therefore, `h(8) = 11` <a class="yt-timestamp" data-t="00:03:37">[00:03:37]</a>.

### Example 3: Traditional Equations as Functions
Many familiar equations can be viewed as [[definition_of_a_function | functions]]. For example, `y = x + 1` can be expressed as `f(x) = x + 1` <a class="yt-timestamp" data-t="00:03:55">[00:03:55]</a>.

To evaluate `f(0)`:
1.  Substitute 0 for `x`: `f(0) = 0 + 1 = 1` <a class="yt-timestamp" data-t="00:04:23">[00:04:23]</a>.

To evaluate `f(2)`:
1.  Substitute 2 for `x`: `f(2) = 2 + 1 = 3` <a class="yt-timestamp" data-t="00:04:44">[00:04:44]</a>.

## What is Not a Function?
A key characteristic of a [[definition_of_a_function | function]] is that for every given input, there can be only one possible output <a class="yt-timestamp" data-t="00:05:36">[00:05:36]</a>. A [[function_vs_nonfunction_relationships | relationship]] that produces multiple outputs for a single input is not considered a [[definition_of_a_function | function]] <a class="yt-timestamp" data-t="00:07:47">[00:07:47]</a>.

### Example: A Circle
Consider the equation of a circle centered at the origin with radius 2: `x^2 + y^2 = 4` <a class="yt-timestamp" data-t="00:06:23">[00:06:23]</a>. This [[function_vs_nonfunction_relationships | relationship]] is not a [[definition_of_a_function | function]] because a single `x` input can correspond to two `y` outputs <a class="yt-timestamp" data-t="00:06:45">[00:06:45]</a>.

For example, if `x = 1`:
1.  Substitute `x = 1` into the equation: `1^2 + y^2 = 4` <a class="yt-timestamp" data-t="00:07:02">[00:07:02]</a>.
2.  Simplify: `1 + y^2 = 4` <a class="yt-timestamp" data-t="00:07:07">[00:07:07]</a>.
3.  Solve for `y^2`: `y^2 = 3` <a class="yt-timestamp" data-t="00:07:11">[00:07:11]</a>.
4.  Solve for `y`: `y = ±√3` <a class="yt-timestamp" data-t="00:07:15">[00:07:15]</a>.

Here, the input `x = 1` produces two outputs: positive square root of 3 and negative square root of 3 <a class="yt-timestamp" data-t="00:07:20">[00:07:20]</a>. Therefore, this [[function_vs_nonfunction_relationships | relationship]] is not a [[definition_of_a_function | function]] <a class="yt-timestamp" data-t="00:07:40">[00:07:40]</a>.
