## Subtracting a Positive Number from a Smaller Positive Number

When a larger number is subtracted from a smaller positive number, the result will be a [[introduction_to_negative_numbers | negative number]] <a class="yt-timestamp" data-t="00:00:13">[00:00:13]</a>.

*   **Example: `2 - 3`** <a class="yt-timestamp" data-t="00:00:03">[00:00:03]</a>
    *   To solve this, a [[visualizing_negative_numbers_on_a_number_line | number line]] can be used <a class="yt-timestamp" data-t="00:00:21">[00:00:21]</a>.
    *   Start at `2` on the number line <a class="yt-timestamp" data-t="00:00:34">[00:00:34]</a>.
    *   Subtracting `3` means moving `3` units to the left on the number line <a class="yt-timestamp" data-t="00:00:41">[00:00:41]</a>.
    *   Moving `1`, `2`, `3` units to the left from `2` lands on `-1` <a class="yt-timestamp" data-t="00:00:44">[00:00:44]</a>.
    *   Result: `2 - 3 = -1` <a class="yt-timestamp" data-t="00:00:51">[00:00:51]</a>

## Subtracting a Positive Number from a Negative Number

When subtracting a positive number from a negative number, the result becomes even more negative.

*   **Example: `-2 - 3`** <a class="yt-timestamp" data-t="00:00:54">[00:00:54]</a>
    *   Using a [[visualizing_negative_numbers_on_a_number_line | number line]] <a class="yt-timestamp" data-t="00:01:06">[00:01:06]</a>:
    *   Start at `-2` <a class="yt-timestamp" data-t="00:01:24">[00:01:24]</a>.
    *   Subtracting `3` means moving `3` units to the left from `-2` <a class="yt-timestamp" data-t="00:01:30">[00:01:30]</a>.
    *   Moving `1`, `2`, `3` units to the left from `-2` lands on `-5` <a class="yt-timestamp" data-t="00:01:33">[00:01:33]</a>.
    *   Result: `-2 - 3 = -5` <a class="yt-timestamp" data-t="00:01:39">[00:01:39]</a>
    *   In both `2 - 3` and `-2 - 3`, subtracting `3` meant moving `3` units to the left on the number line <a class="yt-timestamp" data-t="00:01:40">[00:01:40]</a>.

## Adding a Positive Number to a Negative Number

When adding a positive number to a negative number, the operation moves the value towards zero or into positive territory.

*   **Example: `-2 + 3`** <a class="yt-timestamp" data-t="00:01:57">[00:01:57]</a>
    *   Consider this example on your own before looking at the solution <a class="yt-timestamp" data-t="00:02:05">[00:02:05]</a>.
    *   Using a [[visualizing_negative_numbers_on_a_number_line | number line]] <a class="yt-timestamp" data-t="00:02:10">[00:02:10]</a>:
    *   Start at `-2` (which is `2` units to the left of zero) <a class="yt-timestamp" data-t="00:02:23">[00:02:23]</a>.
    *   Adding `3` means moving `3` units to the right <a class="yt-timestamp" data-t="00:02:27">[00:02:27]</a>.
    *   Moving `1`, `2`, `3` units to the right from `-2` lands on `1` <a class="yt-timestamp" data-t="00:02:30">[00:02:30]</a>.
    *   Result: `-2 + 3 = 1` <a class="yt-timestamp" data-t="00:02:31">[00:02:31]</a>

## Subtracting a Negative Number

[[Subtracting negative numbers | Subtracting a negative number]] is equivalent to [[adding negative numbers | adding a positive number]] <a class="yt-timestamp" data-t="00:02:59">[00:02:59]</a>.

*   **Example 1: `2 - (-3)`** <a class="yt-timestamp" data-t="00:02:40">[00:02:40]</a>
    *   This is the same as [[adding negative numbers | adding]] the positive value <a class="yt-timestamp" data-t="00:03:01">[00:03:01]</a>.
    *   Therefore, `2 - (-3)` is equivalent to `2 + 3` <a class="yt-timestamp" data-t="00:03:04">[00:03:04]</a>.
    *   Result: `2 - (-3) = 5` <a class="yt-timestamp" data-t="00:03:18">[00:03:18]</a>

*   **Example 2: `-2 - (-3)`** <a class="yt-timestamp" data-t="00:03:23">[00:03:23]</a>
    *   Even with multiple negatives, remember that [[subtracting negative numbers | subtracting a negative]] results in a positive <a class="yt-timestamp" data-t="00:03:38">[00:03:38]</a>.
    *   This expression is equivalent to `-2 + 3` <a class="yt-timestamp" data-t="00:03:45">[00:03:45]</a>.
    *   As seen in a previous example, starting at `-2` and adding `3` means moving `3` units to the right on the number line <a class="yt-timestamp" data-t="00:03:55">[00:03:55]</a>.
    *   Result: `-2 - (-3) = 1` <a class="yt-timestamp" data-t="00:04:01">[00:04:01]</a>
