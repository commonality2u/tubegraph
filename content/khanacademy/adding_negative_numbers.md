## Key Concepts

### [[understanding_subtracting_a_negative_is_the_same_as_adding_a_positive|Subtracting a Negative Number]]
A fundamental rule in [[arithmetic_examples_with_negative_numbers|arithmetic with negative numbers]] is that [[understanding_subtracting_a_negative_is_the_same_as_adding_a_positive|subtracting a negative number]] is equivalent to [[understanding_subtracting_a_negative_is_the_same_as_adding_a_positive|adding the corresponding positive number]] <a class="yt-timestamp" data-t="00:02:59">[00:02:59]</a>. This means that an expression like `X - (-Y)` is the same as `X + Y` <a class="yt-timestamp" data-t="00:03:01">[00:03:01]</a>.

## Examples

The following examples demonstrate [[arithmetic_examples_with_negative_numbers|arithmetic examples with negative numbers]], illustrating how to [[handling_equations_with_negative_numbers|handle equations with negative numbers]] and visualize them on a [[visualizing_negative_numbers_on_a_number_line|number line]].

### Example 1: Subtracting a Positive Number from a Smaller Positive Number
Let's consider the expression `2 - 3` <a class="yt-timestamp" data-t="00:00:03">[00:00:03]</a>.
When subtracting a larger number from a smaller number, the result will be a [[introduction_to_negative_numbers|negative number]] <a class="yt-timestamp" data-t="00:00:13">[00:00:13]</a>.

Using a [[visualizing_negative_numbers_on_a_number_line|number line]]:
1. Start at 2 <a class="yt-timestamp" data-t="00:00:34">[00:00:34]</a>.
2. Subtracting 3 means moving 3 units to the left on the number line <a class="yt-timestamp" data-t="00:00:41">[00:00:41]</a>.
3. Moving 1, 2, 3 units to the left from 2 lands on -1 <a class="yt-timestamp" data-t="00:00:44">[00:00:44]</a>.

Therefore, `2 - 3 = -1` <a class="yt-timestamp" data-t="00:00:51">[00:00:51]</a>.

### Example 2: Subtracting a Positive Number from a Negative Number
Consider the expression `-2 - 3` <a class="yt-timestamp" data-t="00:00:54">[00:00:54]</a>.

Using a [[visualizing_negative_numbers_on_a_number_line|number line]]:
1. Start at -2 <a class="yt-timestamp" data-t="00:01:24">[00:01:24]</a>.
2. Subtracting 3 means moving 3 units to the left from -2 <a class="yt-timestamp" data-t="00:01:28">[00:01:28]</a>.
3. Moving 1, 2, 3 units to the left from -2 lands on -5 <a class="yt-timestamp" data-t="00:01:33">[00:01:33]</a>.

Therefore, `-2 - 3 = -5` <a class="yt-timestamp" data-t="00:01:39">[00:01:39]</a>.

### Example 3: Adding a Positive Number to a Negative Number
Consider the expression `-2 + 3` <a class="yt-timestamp" data-t="00:01:57">[00:01:57]</a>.

Using a [[visualizing_negative_numbers_on_a_number_line|number line]]:
1. Start at -2 <a class="yt-timestamp" data-t="00:02:23">[00:02:23]</a>.
2. Adding 3 means moving 3 units to the right on the number line <a class="yt-timestamp" data-t="00:02:29">[00:02:29]</a>.
3. Moving 1, 2, 3 units to the right from -2 lands on 1 <a class="yt-timestamp" data-t="00:02:31">[00:02:31]</a>.

Therefore, `-2 + 3 = 1` <a class="yt-timestamp" data-t="00:02:31">[00:02:31]</a>.

### Example 4: Subtracting a Negative Number
Consider the expression `2 - (-3)` <a class="yt-timestamp" data-t="00:02:47">[00:02:47]</a>.
As established, [[understanding_subtracting_a_negative_is_the_same_as_adding_a_positive|subtracting a negative is the same as adding a positive]] <a class="yt-timestamp" data-t="00:03:01">[00:03:01]</a>.
So, `2 - (-3)` is equivalent to `2 + 3` <a class="yt-timestamp" data-t="00:03:04">[00:03:04]</a>.

Therefore, `2 - (-3) = 5` <a class="yt-timestamp" data-t="00:03:18">[00:03:18]</a>.

### Example 5: Subtracting a Negative Number from a Negative Number
Consider the expression `-2 - (-3)` <a class="yt-timestamp" data-t="00:03:23">[00:03:23]</a>.
Despite multiple negatives, remember that [[understanding_subtracting_a_negative_is_the_same_as_adding_a_positive|subtracting a negative]] results in a positive <a class="yt-timestamp" data-t="00:03:38">[00:03:38]</a>.
So, `-2 - (-3)` is equivalent to `-2 + 3` <a class="yt-timestamp" data-t="00:03:45">[00:03:45]</a>.

This expression has been previously solved in Example 3 <a class="yt-timestamp" data-t="00:03:52">[00:03:52]</a>:
1. Start at -2 <a class="yt-timestamp" data-t="00:03:55">[00:03:55]</a>.
2. Add 3 by moving 3 units to the right <a class="yt-timestamp" data-t="00:03:59">[00:03:59]</a>.
3. This lands on 1 <a class="yt-timestamp" data-t="00:04:01">[00:04:01]</a>.

Therefore, `-2 - (-3) = 1` <a class="yt-timestamp" data-t="00:04:01">[00:04:01]</a>.
