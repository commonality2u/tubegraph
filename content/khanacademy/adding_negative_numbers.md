## Subtracting Positive Numbers

When [[subtracting negative numbers|subtracting a positive number]] from another positive number, if the number being subtracted is larger, the result will be a negative number <a class="yt-timestamp" data-t="00:00:13">[00:00:13]</a>.

### Example: 2 - 3

Consider the expression 2 - 3 <a class="yt-timestamp" data-t="00:00:03">[00:00:03]</a>.
On a [[understanding_negative_numbers_on_a_number_line|number line]] <a class="yt-timestamp" data-t="00:00:23">[00:00:23]</a>:
1.  Start at 2 <a class="yt-timestamp" data-t="00:00:34">[00:00:34]</a>.
2.  Subtracting 3 means moving 3 units to the left on the [[understanding_negative_numbers_on_a_number_line|number line]] <a class="yt-timestamp" data-t="00:00:38">[00:00:38]</a>.
3.  Moving 3 units left (1, 2, 3) from 2 leads to -1 <a class="yt-timestamp" data-t="00:00:44">[00:00:44]</a>.
    > 2 - 3 = -1 <a class="yt-timestamp" data-t="00:00:51">[00:00:51]</a>

### Example: -2 - 3

Consider the expression -2 - 3 <a class="yt-timestamp" data-t="00:00:54">[00:00:54]</a>.
On a [[understanding_negative_numbers_on_a_number_line|number line]] <a class="yt-timestamp" data-t="00:01:06">[00:01:06]</a>:
1.  Start at -2 <a class="yt-timestamp" data-t="00:01:24">[00:01:24]</a>.
2.  Subtracting 3 means moving 3 units to the left from -2 <a class="yt-timestamp" data-t="00:01:28">[00:01:28]</a>.
3.  Moving 3 units left (1, 2, 3) from -2 leads to -5 <a class="yt-timestamp" data-t="00:01:33">[00:01:33]</a>.
    > -2 - 3 = -5 <a class="yt-timestamp" data-t="00:01:39">[00:01:39]</a>
In both cases of [[subtracting negative numbers|subtracting 3]], the operation involved moving 3 units to the left on the [[understanding_negative_numbers_on_a_number_line|number line]] <a class="yt-timestamp" data-t="00:01:40">[00:01:40]</a>.

## Adding Positive Numbers to Negative Numbers

### Example: -2 + 3

Consider the expression -2 + 3 <a class="yt-timestamp" data-t="00:01:57">[00:01:57]</a>.
On a [[understanding_negative_numbers_on_a_number_line|number line]] <a class="yt-timestamp" data-t="00:02:10">[00:02:10]</a>:
1.  Start at -2 (2 units to the left of zero) <a class="yt-timestamp" data-t="00:02:23">[00:02:23]</a>.
2.  Adding 3 means moving 3 units to the right <a class="yt-timestamp" data-t="00:02:27">[00:02:27]</a>.
3.  Moving 3 units right (1, 2, 3) from -2 leads to positive 1 <a class="yt-timestamp" data-t="00:02:30">[00:02:30]</a>.
    > -2 + 3 = 1 <a class="yt-timestamp" data-t="00:02:40">[00:02:40]</a>

## Subtracting a Negative Number (Equivalent to Addition)

A key rule in [[negative_number_arithmetic|negative number arithmetic]] is that [[understanding_subtraction_of_a_negative_as_addition|subtracting a negative number]] is equivalent to [[adding_and_subtracting_negative_numbers|adding the positive version]] of that number <a class="yt-timestamp" data-t="00:02:59">[00:02:59]</a>.

### Example: 2 - (-3)

Consider the expression 2 - (-3) <a class="yt-timestamp" data-t="00:02:47">[00:02:47]</a>.
This is the same as adding the positive:
> 2 - (-3) = 2 + 3 <a class="yt-timestamp" data-t="00:03:04">[00:03:04]</a>
> 2 + 3 = 5 <a class="yt-timestamp" data-t="00:03:18">[00:03:18]</a>

### Example: -2 - (-3)

Consider the expression -2 - (-3) <a class="yt-timestamp" data-t="00:03:23">[00:03:23]</a>.
Despite the appearance of multiple negatives <a class="yt-timestamp" data-t="00:03:34">[00:03:34]</a>, remember that [[understanding_subtraction_of_a_negative_as_addition|subtracting a negative]] results in a positive <a class="yt-timestamp" data-t="00:03:38">[00:03:38]</a>.
So, this expression is equivalent to:
> -2 - (-3) = -2 + 3 <a class="yt-timestamp" data-t="00:03:45">[00:03:45]</a>
As seen in a previous example, -2 + 3 means starting at -2 (2 units left of 0) and moving 3 units to the right <a class="yt-timestamp" data-t="00:03:55">[00:03:55]</a>.
Moving 3 units right (1, 2, 3) from -2 leads to positive 1 <a class="yt-timestamp" data-t="00:03:59">[00:03:59]</a>.
> -2 + 3 = 1 <a class="yt-timestamp" data-t="00:04:01">[00:04:01]</a>
