This article explores how to calculate [[basic_trigonometric_functions_sine_cosine_and_tangent | trigonometric functions]] like sine, cosine, and tangent using [[examples_of_calculating_sine_cosine_and_tangent | examples]] of [[application_of_trigonometry_in_right_triangles | right triangles]] <a class="yt-timestamp" data-t="00:00:07">[00:00:07]</a>. The definitions discussed here are specifically for [[application_of_trigonometry_in_right_triangles | right triangles]] <a class="yt-timestamp" data-t="00:00:15">[00:00:15]</a>.

## The SOH CAH TOA Mnemonic

A useful mnemonic for remembering the definitions of sine, cosine, and tangent is [[sohcahtoa_mnemonic_for_trigonometric_functions | SOH CAH TOA]] <a class="yt-timestamp" data-t="00:02:09">[00:02:09]</a>:
*   **SOH**: **S**ine = **O**pposite / **H**ypotenuse <a class="yt-timestamp" data-t="00:04:22">[00:04:22]</a>
*   **CAH**: **C**osine = **A**djacent / **H**ypotenuse <a class="yt-timestamp" data-t="00:02:43">[00:02:43]</a>
*   **TOA**: **T**angent = **O**pposite / **A**djacent <a class="yt-timestamp" data-t="00:05:27">[00:05:27]</a>

## Example 1: General Right Triangle

Consider a [[application_of_trigonometry_in_right_triangles | right triangle]] with sides of length 7 and 4 <a class="yt-timestamp" data-t="00:00:31">[00:00:31]</a>.

### Calculating the Hypotenuse

First, find the [[calculating_the_hypotenuse_and_other_sides | hypotenuse]] (h) using the Pythagorean theorem (<a class="yt-timestamp" data-t="00:00:52">[00:00:52]</a>):
*   h² = 7² + 4² <a class="yt-timestamp" data-t="00:00:45">[00:00:45]</a>
*   h² = 49 + 16 <a class="yt-timestamp" data-t="00:01:04">[00:01:04]</a>
*   h² = 65 <a class="yt-timestamp" data-t="00:01:25">[00:01:25]</a>
*   h = √65 <a class="yt-timestamp" data-t="00:01:39">[00:01:39]</a> (This cannot be simplified further <a class="yt-timestamp" data-t="00:01:50">[00:01:50]</a>.)

### Calculating Trig Functions for Angle Theta (θ)

Let's find the [[basic_trigonometric_functions_sine_cosine_and_tangent | trig functions]] for an angle θ in this triangle <a class="yt-timestamp" data-t="00:01:55">[00:01:55]</a>.

#### Cosine of Theta (cos θ)
Using CAH (Adjacent / Hypotenuse) <a class="yt-timestamp" data-t="00:02:43">[00:02:43]</a>:
*   The side adjacent to θ is 4 <a class="yt-timestamp" data-t="00:03:04">[00:03:04]</a>.
*   The [[calculating_the_hypotenuse_and_other_sides | hypotenuse]] is √65 <a class="yt-timestamp" data-t="00:03:17">[00:03:17]</a>.
*   cos θ = 4 / √65 <a class="yt-timestamp" data-t="00:03:21">[00:03:21]</a>
*   To rationalize the denominator, multiply by √65 / √65 <a class="yt-timestamp" data-t="00:03:39">[00:03:39]</a>:
    cos θ = (4 * √65) / (√65 * √65) = 4√65 / 65 <a class="yt-timestamp" data-t="00:03:54">[00:03:54]</a>

#### Sine of Theta (sin θ)
Using SOH (Opposite / Hypotenuse) <a class="yt-timestamp" data-t="00:04:22">[00:04:22]</a>:
*   The side opposite θ is 7 <a class="yt-timestamp" data-t="00:04:34">[00:04:34]</a>.
*   The [[calculating_the_hypotenuse_and_other_sides | hypotenuse]] is √65 <a class="yt-timestamp" data-t="00:04:47">[00:04:47]</a>.
*   sin θ = 7 / √65 <a class="yt-timestamp" data-t="00:04:51">[00:04:51]</a>
*   Rationalizing the denominator:
    sin θ = (7 * √65) / (√65 * √65) = 7√65 / 65 <a class="yt-timestamp" data-t="00:04:55">[00:04:55]</a>

#### Tangent of Theta (tan θ)
Using TOA (Opposite / Adjacent) <a class="yt-timestamp" data-t="00:05:27">[00:05:27]</a>:
*   The side opposite θ is 7 <a class="yt-timestamp" data-t="00:05:38">[00:05:38]</a>.
*   The side adjacent to θ is 4 <a class="yt-timestamp" data-t="00:05:46">[00:05:46]</a>.
*   tan θ = 7 / 4 <a class="yt-timestamp" data-t="00:05:51">[00:05:51]</a>

## Example 2: 30-60-90 Right Triangle

Consider a [[application_of_trigonometry_in_right_triangles | right triangle]] with a [[calculating_the_hypotenuse_and_other_sides | hypotenuse]] of 4, one side of 2, and another side of 2√3 <a class="yt-timestamp" data-t="00:06:17">[00:06:17]</a>. This is a special 30-60-90 triangle <a class="yt-timestamp" data-t="00:07:07">[00:07:07]</a>, where:
*   The angle opposite the side of length 2 is 30 degrees <a class="yt-timestamp" data-t="00:07:15">[00:07:15]</a>.
*   The angle opposite the side of length 2√3 is 60 degrees <a class="yt-timestamp" data-t="00:07:23">[00:07:23]</a>.

### Calculating Trig Functions for 30 Degrees

#### Sine of 30 Degrees (sin 30°)
Using SOH (Opposite / Hypotenuse) <a class="yt-timestamp" data-t="00:08:20">[00:08:20]</a>:
*   The side opposite 30° is 2 <a class="yt-timestamp" data-t="00:08:26">[00:08:26]</a>.
*   The [[calculating_the_hypotenuse_and_other_sides | hypotenuse]] is 4 <a class="yt-timestamp" data-t="00:08:30">[00:08:30]</a>.
*   sin 30° = 2 / 4 = 1/2 <a class="yt-timestamp" data-t="00:08:32">[00:08:32]</a>

#### Cosine of 30 Degrees (cos 30°)
Using CAH (Adjacent / Hypotenuse) <a class="yt-timestamp" data-t="00:08:52">[00:08:52]</a>:
*   The side adjacent to 30° is 2√3 <a class="yt-timestamp" data-t="00:08:59">[00:08:59]</a>.
*   The [[calculating_the_hypotenuse_and_other_sides | hypotenuse]] is 4 <a class="yt-timestamp" data-t="00:09:12">[00:09:12]</a>.
*   cos 30° = 2√3 / 4 = √3 / 2 <a class="yt-timestamp" data-t="00:09:13">[00:09:13]</a>

#### Tangent of 30 Degrees (tan 30°)
Using TOA (Opposite / Adjacent) <a class="yt-timestamp" data-t="00:09:31">[00:09:31]</a>:
*   The side opposite 30° is 2 <a class="yt-timestamp" data-t="00:09:38">[00:09:38]</a>.
*   The side adjacent to 30° is 2√3 <a class="yt-timestamp" data-t="00:09:42">[00:09:42]</a>.
*   tan 30° = 2 / (2√3) = 1 / √3 <a class="yt-timestamp" data-t="00:09:54">[00:09:54]</a>
*   Rationalizing the denominator:
    tan 30° = (1 * √3) / (√3 * √3) = √3 / 3 <a class="yt-timestamp" data-t="00:09:56">[00:09:56]</a>

### Calculating Trig Functions for 60 Degrees

#### Sine of 60 Degrees (sin 60°)
Using SOH (Opposite / Hypotenuse) <a class="yt-timestamp" data-t="00:10:30">[00:10:30]</a>:
*   The side opposite 60° is 2√3 <a class="yt-timestamp" data-t="00:10:36">[00:10:36]</a>.
*   The [[calculating_the_hypotenuse_and_other_sides | hypotenuse]] is 4 <a class="yt-timestamp" data-t="00:10:50">[00:10:50]</a>.
*   sin 60° = 2√3 / 4 = √3 / 2 <a class="yt-timestamp" data-t="00:10:54">[00:10:54]</a>

#### Cosine of 60 Degrees (cos 60°)
Using CAH (Adjacent / Hypotenuse) <a class="yt-timestamp" data-t="00:11:05">[00:11:05]</a>:
*   The side adjacent to 60° is 2 <a class="yt-timestamp" data-t="00:11:10">[00:11:10]</a>.
*   The [[calculating_the_hypotenuse_and_other_sides | hypotenuse]] is 4 <a class="yt-timestamp" data-t="00:11:13">[00:11:13]</a>.
*   cos 60° = 2 / 4 = 1/2 <a class="yt-timestamp" data-t="00:11:17">[00:11:17]</a>

#### Tangent of 60 Degrees (tan 60°)
Using TOA (Opposite / Adjacent) <a class="yt-timestamp" data-t="00:11:27">[00:11:27]</a>:
*   The side opposite 60° is 2√3 <a class="yt-timestamp" data-t="00:11:32">[00:11:32]</a>.
*   The side adjacent to 60° is 2 <a class="yt-timestamp" data-t="00:11:38">[00:11:38]</a>.
*   tan 60° = 2√3 / 2 = √3 <a class="yt-timestamp" data-t="00:11:44">[00:11:44]</a>

### Observations
Notice the relationship between the [[basic_trigonometric_functions_sine_cosine_and_tangent | sine]] and [[basic_trigonometric_functions_sine_cosine_and_tangent | cosine]] of these [[applying_trigonometry_to_different_angles_in_a_triangle | different angles]]:
*   sin 30° = cos 60° (both 1/2) <a class="yt-timestamp" data-t="00:11:54">[00:11:54]</a>
*   cos 30° = sin 60° (both √3 / 2) <a class="yt-timestamp" data-t="00:11:57">[00:11:57]</a>
*   The [[basic_trigonometric_functions_sine_cosine_and_tangent | tangents]] of 30° and 60° are reciprocals of each other (√3 / 3 and √3) <a class="yt-timestamp" data-t="00:12:01">[00:12:01]</a>.
