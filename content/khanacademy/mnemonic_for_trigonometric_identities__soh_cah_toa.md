The [[soh_cah_toa_mnemonic | SOH CAH TOA mnemonic]] is a useful tool for remembering how to calculate the basic [[trigonometric_functions_sine_cosine_and_tangent | trigonometric functions]] (sine, cosine, and tangent) for angles within [[trigonometric_functions_and_right_triangles | right triangles]] <a class="yt-timestamp" data-t="00:00:15">[00:00:15]</a>. It is a common aid in [[introduction_to_basic_trigonometry | trigonometry]] <a class="yt-timestamp" data-t="00:02:23">[00:02:23]</a>.

## How to Use SOH CAH TOA

To apply the [[soh_cah_toa_mnemonic | SOH CAH TOA mnemonic]], one should always write it down before calculating [[finding_sine_cosine_and_tangent | trigonometric ratios]] <a class="yt-timestamp" data-t="00:02:09">[00:02:09]</a>:

*   **SOH**: Sine is Opposite over Hypotenuse <a class="yt-timestamp" data-t="00:04:25">[00:04:25]</a>.
*   **CAH**: Cosine is Adjacent over Hypotenuse <a class="yt-timestamp" data-t="00:02:43">[00:02:43]</a>.
*   **TOA**: Tangent is Opposite over Adjacent <a class="yt-timestamp" data-t="00:05:27">[00:05:27]</a>.

The "opposite" side refers to the side directly across from the angle in question, while the "adjacent" side is the side next to the angle that is not the hypotenuse <a class="yt-timestamp" data-t="00:03:00">[00:03:00]</a>, <a class="yt-timestamp" data-t="00:03:10">[00:03:10]</a>. The hypotenuse is always the longest side, opposite the right angle <a class="yt-timestamp" data-t="00:02:55">[00:02:55]</a>.

## Examples of Applying SOH CAH TOA

### Example 1: Triangle with Sides 7, 4, and √65

Consider a [[trigonometric_functions_and_right_triangles | right triangle]] where one leg is 7 units and the other leg is 4 units <a class="yt-timestamp" data-t="00:00:31">[00:00:31]</a>.
Using the Pythagorean theorem, the hypotenuse (h) is found:
h² = 7² + 4² <a class="yt-timestamp" data-t="00:00:45">[00:00:45]</a>
h² = 49 + 16 <a class="yt-timestamp" data-t="00:01:04">[00:01:04]</a>
h² = 65 <a class="yt-timestamp" data-t="00:01:18">[00:01:18]</a>
h = √65 <a class="yt-timestamp" data-t="00:01:39">[00:01:39]</a>

Let's find the [[using_soh_cah_toa_to_determine_trigonometric_ratios | trigonometric ratios]] for the angle (theta) opposite the side of length 7 <a class="yt-timestamp" data-t="00:02:02">[00:02:02]</a>.

*   **Cosine of theta (cos θ)**:
    Using CAH (Adjacent / Hypotenuse) <a class="yt-timestamp" data-t="00:02:43">[00:02:43]</a>:
    The side adjacent to theta is 4 <a class="yt-timestamp" data-t="00:03:07">[00:03:07]</a>.
    cos θ = 4 / √65 <a class="yt-timestamp" data-t="00:03:17">[00:03:17]</a>
    To rationalize the denominator, multiply by √65/√65 <a class="yt-timestamp" data-t="00:03:39">[00:03:39]</a>:
    cos θ = (4 * √65) / 65 <a class="yt-timestamp" data-t="00:03:54">[00:03:54]</a>

*   **Sine of theta (sin θ)**:
    Using SOH (Opposite / Hypotenuse) <a class="yt-timestamp" data-t="00:04:25">[00:04:25]</a>:
    The side opposite to theta is 7 <a class="yt-timestamp" data-t="00:04:34">[00:04:34]</a>.
    sin θ = 7 / √65 <a class="yt-timestamp" data-t="00:04:47">[00:04:47]</a>
    Rationalized: sin θ = (7 * √65) / 65 <a class="yt-timestamp" data-t="00:04:59">[00:04:59]</a>

*   **Tangent of theta (tan θ)**:
    Using TOA (Opposite / Adjacent) <a class="yt-timestamp" data-t="00:05:27">[00:05:27]</a>:
    The side opposite to theta is 7 <a class="yt-timestamp" data-t="00:05:38">[00:05:38]</a>.
    The side adjacent to theta is 4 <a class="yt-timestamp" data-t="00:05:46">[00:05:46]</a>.
    tan θ = 7 / 4 <a class="yt-timestamp" data-t="00:05:51">[00:05:51]</a>

### Example 2: [[understanding_and_applying_trigonometric_ratios_in_306090_triangles | 30-60-90 Triangle]]

Consider a [[trigonometric_functions_and_right_triangles | right triangle]] with sides 2, 2√3, and a hypotenuse of 4 <a class="yt-timestamp" data-t="00:06:17">[00:06:17]</a>. This is a [[understanding_and_applying_trigonometric_ratios_in_306090_triangles | 30-60-90 triangle]] <a class="yt-timestamp" data-t="00:07:07">[00:07:07]</a>, where the angle opposite the side of length 2 is 30 degrees, and the angle opposite the side of length 2√3 is 60 degrees <a class="yt-timestamp" data-t="00:07:15">[00:07:15]</a>, <a class="yt-timestamp" data-t="00:07:23">[00:07:23]</a>.

#### For the 30-degree angle:

*   **Sine of 30 degrees (sin 30°)**:
    Using SOH (Opposite / Hypotenuse) <a class="yt-timestamp" data-t="00:08:20">[00:08:20]</a>:
    Opposite side is 2 <a class="yt-timestamp" data-t="00:08:26">[00:08:26]</a>. Hypotenuse is 4 <a class="yt-timestamp" data-t="00:08:30">[00:08:30]</a>.
    sin 30° = 2 / 4 = 1/2 <a class="yt-timestamp" data-t="00:08:32">[00:08:32]</a>

*   **Cosine of 30 degrees (cos 30°)**:
    Using CAH (Adjacent / Hypotenuse) <a class="yt-timestamp" data-t="00:08:52">[00:08:52]</a>:
    Adjacent side is 2√3 <a class="yt-timestamp" data-t="00:08:59">[00:08:59]</a>. Hypotenuse is 4 <a class="yt-timestamp" data-t="00:09:09">[00:09:09]</a>.
    cos 30° = (2√3) / 4 = √3 / 2 <a class="yt-timestamp" data-t="00:09:13">[00:09:13]</a>

*   **Tangent of 30 degrees (tan 30°)**:
    Using TOA (Opposite / Adjacent) <a class="yt-timestamp" data-t="00:09:31">[00:09:31]</a>:
    Opposite side is 2 <a class="yt-timestamp" data-t="00:09:38">[00:09:38]</a>. Adjacent side is 2√3 <a class="yt-timestamp" data-t="00:09:42">[00:09:42]</a>.
    tan 30° = 2 / (2√3) = 1 / √3 <a class="yt-timestamp" data-t="00:09:54">[00:09:54]</a>
    Rationalized: tan 30° = (1 * √3) / (√3 * √3) = √3 / 3 <a class="yt-timestamp" data-t="00:09:56">[00:09:56]</a>

#### For the 60-degree angle:

*   **Sine of 60 degrees (sin 60°)**:
    Using SOH (Opposite / Hypotenuse) <a class="yt-timestamp" data-t="00:10:30">[00:10:30]</a>:
    Opposite side is 2√3 <a class="yt-timestamp" data-t="00:10:36">[00:10:36]</a>. Hypotenuse is 4 <a class="yt-timestamp" data-t="00:10:50">[00:10:50]</a>.
    sin 60° = (2√3) / 4 = √3 / 2 <a class="yt-timestamp" data-t="00:10:54">[00:10:54]</a>

*   **Cosine of 60 degrees (cos 60°)**:
    Using CAH (Adjacent / Hypotenuse) <a class="yt-timestamp" data-t="00:11:05">[00:11:05]</a>:
    Adjacent side is 2 <a class="yt-timestamp" data-t="00:11:10">[00:11:10]</a>. Hypotenuse is 4 <a class="yt-timestamp" data-t="00:11:13">[00:11:13]</a>.
    cos 60° = 2 / 4 = 1/2 <a class="yt-timestamp" data-t="00:11:17">[00:11:17]</a>

*   **Tangent of 60 degrees (tan 60°)**:
    Using TOA (Opposite / Adjacent) <a class="yt-timestamp" data-t="00:11:27">[00:11:27]</a>:
    Opposite side is 2√3 <a class="yt-timestamp" data-t="00:11:32">[00:11:32]</a>. Adjacent side is 2 <a class="yt-timestamp" data-t="00:11:39">[00:11:39]</a>.
    tan 60° = (2√3) / 2 = √3 <a class="yt-timestamp" data-t="00:11:48">[00:11:48]</a>

> [!NOTE] Relationship between 30° and 60° angles
> Observe that the sine of 30 degrees (1/2) is the same as the cosine of 60 degrees (1/2) <a class="yt-timestamp" data-t="00:11:54">[00:11:54]</a>.
> Similarly, the cosine of 30 degrees (√3/2) is the same as the sine of 60 degrees (√3/2) <a class="yt-timestamp" data-t="00:11:57">[00:11:57]</a>.
