This article explores how to apply [[core_trigonometric_functions_sine_cosine_and_tangent | trigonometric functions]] to specific angles within [[right_triangles_and_trigonometry | right triangles]], with a particular focus on 30-60-90 triangles. ## [[Basics of trigonometry | Foundations of Trigonometry]] in Right Triangles

When working with [[core_trigonometric_functions_sine_cosine_and_tangent | trigonometric functions]], a helpful [[mnemonic_device_for_remembering_trigonometric_functions | mnemonic device]] is "SOH CAH TOA" <a class="yt-timestamp" data-t="00:02:09">[00:02:09]</a>, <a class="yt-timestamp" data-t="00:02:23">[00:02:23]</a>:

*   **SOH**: **S**ine = **O**pposite / **H**ypotenuse <a class="yt-timestamp" data-t="00:04:22">[00:04:22]</a>
*   **CAH**: **C**osine = **A**djacent / **H**ypotenuse <a class="yt-timestamp" data-t="00:02:43">[00:02:43]</a>
*   **TOA**: **T**angent = **O**pposite / **A**djacent <a class="yt-timestamp" data-t="00:05:27">[00:05:27]</a>

The hypotenuse of a [[right_triangles_and_trigonometry | right triangle]] can be found using the Pythagorean theorem, which states that the square of the hypotenuse (h) is equal to the sum of the squares of the other two sides (a and b): h² = a² + b² <a class="yt-timestamp" data-t="00:00:45">[00:00:45]</a>, <a class="yt-timestamp" data-t="00:00:55">[00:00:55]</a>.

### Example 1: General Right Triangle

Consider a [[right_triangles_and_trigonometry | right triangle]] with sides of length 7 and 4 <a class="yt-timestamp" data-t="00:00:31">[00:00:31]</a>, <a class="yt-timestamp" data-t="00:00:37">[00:00:37]</a>.

1.  **Find the Hypotenuse**:
    *   h² = 7² + 4² <a class="yt-timestamp" data-t="00:00:45">[00:00:45]</a>
    *   h² = 49 + 16 <a class="yt-timestamp" data-t="00:01:04">[00:01:04]</a>
    *   h² = 65 <a class="yt-timestamp" data-t="00:01:18">[00:01:18]</a>
    *   h = √65 <a class="yt-timestamp" data-t="00:01:33">[00:01:33]</a> (This cannot be simplified further <a class="yt-timestamp" data-t="00:01:42">[00:01:42]</a>).

2.  **Calculate [[core_trigonometric_functions_sine_cosine_and_tangent | Trigonometric Functions]] for Angle Theta (θ)**:
    Let θ be the angle opposite the side of length 7.

    *   **Cosine (θ)**:
        *   Adjacent side: 4 <a class="yt-timestamp" data-t="00:03:04">[00:03:04]</a>
        *   Hypotenuse: √65 <a class="yt-timestamp" data-t="00:03:17">[00:03:17]</a>
        *   cos(θ) = Adjacent / Hypotenuse = 4 / √65 <a class="yt-timestamp" data-t="00:02:43">[00:02:43]</a>, <a class="yt-timestamp" data-t="00:03:21">[00:03:21]</a>
        *   Rationalized: (4√65) / 65 <a class="yt-timestamp" data-t="00:03:54">[00:03:54]</a>, <a class="yt-timestamp" data-t="00:03:57">[00:03:57]</a>

    *   **Sine (θ)**:
        *   Opposite side: 7 <a class="yt-timestamp" data-t="00:04:34">[00:04:34]</a>
        *   Hypotenuse: √65 <a class="yt-timestamp" data-t="00:04:47">[00:04:47]</a>
        *   sin(θ) = Opposite / Hypotenuse = 7 / √65 <a class="yt-timestamp" data-t="00:04:22">[00:04:22]</a>, <a class="yt-timestamp" data-t="00:04:51">[00:04:51]</a>
        *   Rationalized: (7√65) / 65 <a class="yt-timestamp" data-t="00:05:04">[00:05:04]</a>

    *   **Tangent (θ)**:
        *   Opposite side: 7 <a class="yt-timestamp" data-t="00:05:38">[00:05:38]</a>
        *   Adjacent side: 4 <a class="yt-timestamp" data-t="00:05:46">[00:05:46]</a>
        *   tan(θ) = Opposite / Adjacent = 7 / 4 <a class="yt-timestamp" data-t="00:05:27">[00:05:27]</a>, <a class="yt-timestamp" data-t="00:05:51">[00:05:51]</a>

### Example 2: 30-60-90 Triangle

Consider a [[right_triangles_and_trigonometry | right triangle]] with a hypotenuse of length 4, and sides of length 2 and 2√3 <a class="yt-timestamp" data-t="00:06:17">[00:06:17]</a>, <a class="yt-timestamp" data-t="00:06:21">[00:06:21]</a>, <a class="yt-timestamp" data-t="00:06:26">[00:06:26]</a>.

1.  **Verify using Pythagorean Theorem**:
    *   (2√3)² + 2² <a class="yt-timestamp" data-t="00:06:36">[00:06:36]</a>
    *   = (4 × 3) + 4 <a class="yt-timestamp" data-t="00:06:42">[00:06:42]</a>
    *   = 12 + 4 = 16 <a class="yt-timestamp" data-t="00:06:49">[00:06:49]</a>
    *   This is equal to 4², confirming the sides form a [[right_triangles_and_trigonometry | right triangle]] <a class="yt-timestamp" data-t="00:06:53">[00:06:53]</a>.
    *   This specific triangle is a 30-60-90 triangle <a class="yt-timestamp" data-t="00:07:01">[00:07:01]</a>, with the side opposite the 30-degree angle being half the hypotenuse <a class="yt-timestamp" data-t="00:07:27">[00:07:27]</a>.

2.  **Calculate [[core_trigonometric_functions_sine_cosine_and_tangent | Trigonometric Functions]] for 30 Degrees**:
    The 30-degree angle is opposite the side of length 2 <a class="yt-timestamp" data-t="00:07:15">[00:07:15]</a>.

    *   **Sine (30°)**:
        *   Opposite: 2 <a class="yt-timestamp" data-t="00:08:26">[00:08:26]</a>
        *   Hypotenuse: 4 <a class="yt-timestamp" data-t="00:08:30">[00:08:30]</a>
        *   sin(30°) = Opposite / Hypotenuse = 2 / 4 = 1/2 <a class="yt-timestamp" data-t="00:08:22">[00:08:22]</a>, <a class="yt-timestamp" data-t="00:08:32">[00:08:32]</a>.
        *   The sine of 30 degrees is always 1/2 <a class="yt-timestamp" data-t="00:08:35">[00:08:35]</a>.

    *   **Cosine (30°)**:
        *   Adjacent: 2√3 <a class="yt-timestamp" data-t="00:09:05">[00:09:05]</a>
        *   Hypotenuse: 4 <a class="yt-timestamp" data-t="00:09:09">[00:09:09]</a>
        *   cos(30°) = Adjacent / Hypotenuse = 2√3 / 4 = √3 / 2 <a class="yt-timestamp" data-t="00:08:52">[00:08:52]</a>, <a class="yt-timestamp" data-t="00:09:13">[00:09:13]</a>.

    *   **Tangent (30°)**:
        *   Opposite: 2 <a class="yt-timestamp" data-t="00:09:38">[00:09:38]</a>
        *   Adjacent: 2√3 <a class="yt-timestamp" data-t="00:09:42">[00:09:42]</a>
        *   tan(30°) = Opposite / Adjacent = 2 / (2√3) = 1 / √3 <a class="yt-timestamp" data-t="00:09:31">[00:09:31]</a>, <a class="yt-timestamp" data-t="00:09:54">[00:09:54]</a>.
        *   Rationalized: √3 / 3 <a class="yt-timestamp" data-t="00:10:00">[00:10:00]</a>, <a class="yt-timestamp" data-t="00:10:05">[00:10:05]</a>.

3.  **Calculate [[core_trigonometric_functions_sine_cosine_and_tangent | Trigonometric Functions]] for 60 Degrees**:
    The 60-degree angle is opposite the side of length 2√3 <a class="yt-timestamp" data-t="00:07:23">[00:07:23]</a>.

    *   **Sine (60°)**:
        *   Opposite: 2√3 <a class="yt-timestamp" data-t="00:10:36">[00:10:36]</a>
        *   Hypotenuse: 4 <a class="yt-timestamp" data-t="00:10:50">[00:10:50]</a>
        *   sin(60°) = Opposite / Hypotenuse = 2√3 / 4 = √3 / 2 <a class="yt-timestamp" data-t="00:10:30">[00:10:30]</a>, <a class="yt-timestamp" data-t="00:10:54">[00:10:54]</a>.

    *   **Cosine (60°)**:
        *   Adjacent: 2 <a class="yt-timestamp" data-t="00:11:13">[00:11:13]</a>
        *   Hypotenuse: 4 <a class="yt-timestamp" data-t="00:11:13">[00:11:13]</a>
        *   cos(60°) = Adjacent / Hypotenuse = 2 / 4 = 1/2 <a class="yt-timestamp" data-t="00:11:05">[00:11:05]</a>, <a class="yt-timestamp" data-t="00:11:17">[00:11:17]</a>.

    *   **Tangent (60°)**:
        *   Opposite: 2√3 <a class="yt-timestamp" data-t="00:11:34">[00:11:34]</a>
        *   Adjacent: 2 <a class="yt-timestamp" data-t="00:11:39">[00:11:39]</a>
        *   tan(60°) = Opposite / Adjacent = 2√3 / 2 = √3 <a class="yt-timestamp" data-t="00:11:31">[00:11:31]</a>, <a class="yt-timestamp" data-t="00:11:48">[00:11:48]</a>.

### Relationships between 30° and 60° Functions

Notice the following relationships:
*   sin(30°) = cos(60°) = 1/2 <a class="yt-timestamp" data-t="00:11:54">[00:11:54]</a>
*   cos(30°) = sin(60°) = √3 / 2 <a class="yt-timestamp" data-t="00:11:57">[00:11:57]</a>
*   tan(30°) = 1/√3 and tan(60°) = √3, which are reciprocals of each other <a class="yt-timestamp" data-t="00:12:01">[00:12:01]</a>.

These relationships are inherent to the properties of 30-60-90 triangles and the complementary nature of sine and cosine functions for angles that sum to 90 degrees.
