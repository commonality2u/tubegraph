## Row Asset Management's Alus Program

Row Asset Management's Alus program began in November 2007 <a class="yt-timestamp" data-t="06:45:00">[06:45:00]</a>. It is a medium-term trend following program <a class="yt-timestamp" data-t="06:58:00">[06:58:00]</a>. The firm aimed to develop a program that "thinks like discretionary traders" to capture various types of market behavior into a single program <a class="yt-timestamp" data-t="07:07:00">[07:07:00]</a>, <a class="yt-timestamp" data-t="07:10:00">[07:10:00]</a>, <a class="yt-timestamp" data-t="07:13:00">[07:13:00]</a>.

### Design Philosophy
