## General Approach to Diagnosis

When approaching the [[diagnosis_and_identification_of_bacterial_infections | diagnosis of bacterial infections]], it is essential to first determine if the bacteria are Gram-positive or Gram-negative, and if they are cocci or bacilli, to narrow down the possibilities <a class="yt-timestamp" data-t="00:09:19">[00:09:19]</a>. This initial classification helps in subsequent [[microbiological_techniques_and_tests | microbiological examination techniques]] <a class="yt-timestamp" data-t="00:09:30">[00:09:30]</a>.

## [[characteristics_of_specific_bacterial_cultures | Characteristics of Specific Bacterial Cultures]] and Diagnostic Tests

### _Corynebacterium diphtheriae_ (Diphtheria)

*   **Sample Collection**: For diphtheria, a throat swab should be collected from *under* the pseudomembrane covering the tonsils and pharynx <a class="yt-timestamp" data-t="00:10:59">[00:10:59]</a>. Taking a sample directly from the membrane itself will only yield dead cells <a class="yt-timestamp" data-t="00:10:56">[00:10:56]</a>.
*   **Microscopy and Staining**:
    *   **Gram Stain**: _Corynebacterium diphtheriae_ appear as Gram-positive bacilli, staining purple and appearing long <a class="yt-timestamp" data-t="00:25:52">[00:25:52]</a>, <a class="yt-timestamp" data-t="00:25:56">[00:25:56]</a>.
    *   **Arrangement**: They characteristically show a "Chinese letter" or "cuneiform" arrangement <a class="yt-timestamp" data-t="00:26:03">[00:26:03]</a>, <a class="yt-timestamp" data-t="00:26:26">[00:26:26]</a>.
    *   **Granules**: The bacteria possess bulbous ends containing large, dark-staining inclusions called Valtan granules (also known as Babes-Ernst granules, bipolar granules, or metachromatic granules) <a class="yt-timestamp" data-t="00:26:44">[00:26:44]</a>, <a class="yt-timestamp" data-t="00:26:52">[00:26:52]</a>, <a class="yt-timestamp" data-t="00:27:07">[00:27:07]</a>, <a class="yt-timestamp" data-t="00:27:10">[00:27:10]</a>. These granules contain stored energy in the form of polymetaphosphate <a class="yt-timestamp" data-t="00:30:39">[00:30:39]</a>, <a class="yt-timestamp" data-t="00:30:41">[00:30:41]</a>.
    *   **Albert Stain**: Albert stain is used to visualize these granules <a class="yt-timestamp" data-t="00:27:17">[00:27:17]</a>. The stain uses Toluidine blue, Iodine, and Methylene blue (TIM mnemonic) <a class="yt-timestamp" data-t="00:27:31">[00:27:31]</a>, <a class="yt-timestamp" data-t="00:27:41">[00:27:41]</a>, <a class="yt-timestamp" data-t="00:27:45">[00:27:45]</a>, <a class="yt-timestamp" data-t="00:27:50">[00:27:50]</a>. With Albert stain, the bacterial body appears green, while the Valtan granules stain purple <a class="yt-timestamp" data-t="00:28:09">[00:28:09]</a>, <a class="yt-timestamp" data-t="00:28:14">[00:28:14]</a>, <a class="yt-timestamp" data-t="00:28:16">[00:28:16]</a>, <a class="yt-timestamp" data-t="00:30:20">[00:30:20]</a>, <a class="yt-timestamp" data-t="00:30:22">[00:30:22]</a>.
*   **Culture Media**:
    *   **Enriched Culture Media**: Loeffler's serum slope is an enriched medium <a class="yt-timestamp" data-t="00:31:09">[00:31:09]</a>, <a class="yt-timestamp" data-t="00:31:17">[00:31:17]</a>. It allows for rapid growth, with colonies visible in 6-8 hours <a class="yt-timestamp" data-t="00:31:25">[00:31:25]</a>. This makes it useful for early diagnosis <a class="yt-timestamp" data-t="00:32:18">[00:32:18]</a>.
    *   **Selective Culture Media**:
        *   Potassium Tellurite Agar (PTA) is a selective medium <a class="yt-timestamp" data-t="00:31:38">[00:31:38]</a>, <a class="yt-timestamp" data-t="00:31:41">[00:31:41]</a>. Tellurite gives colonies a black color <a class="yt-timestamp" data-t="00:31:50">[00:31:50]</a>, <a class="yt-timestamp" data-t="00:31:53">[00:31:53]</a>, <a class="yt-timestamp" data-t="00:31:57">[00:31:57]</a>. Growth on PTA takes longer, around 48 hours <a class="yt-timestamp" data-t="00:32:02">[00:32:02]</a>. It is considered the best culture medium for diagnosis <a class="yt-timestamp" data-t="00:32:27">[00:32:27]</a>.
        *   Tinsdale agar is another selective medium for _Corynebacterium diphtheriae_ <a class="yt-timestamp" data-t="00:32:38">[00:32:38]</a>, <a class="yt-timestamp" data-t="00:32:40">[00:32:40]</a>, <a class="yt-timestamp" data-t="01:26:54">[01:26:54]</a>.
*   **Toxin Detection**: The Elek gel precipitation test is used to determine the toxigenicity of the bacteria (i.e., whether it produces toxins) <a class="yt-timestamp" data-t="00:21:31">[00:21:31]</a>, <a class="yt-timestamp" data-t="00:21:56">[00:21:56]</a>, <a class="yt-timestamp" data-t="00:22:02">[00:22:02]</a>. In this test, a strip impregnated with antitoxin is placed on a culture medium. If the bacterium produces toxin, a reaction occurs with the antitoxin, forming visible precipitant lines <a class="yt-timestamp" data-t="00:22:18">[00:22:18]</a>, <a class="yt-timestamp" data-t="00:22:21">[00:22:21]</a>, <a class="yt-timestamp" data-t="00:22:23">[00:22:23]</a>, <a class="yt-timestamp" data-t="00:22:42">[00:22:42]</a>, <a class="yt-timestamp" data-t="00:22:44">[00:22:44]</a>, <a class="yt-timestamp" data-t="00:22:47">[00:22:47]</a>, <a class="yt-timestamp" data-t="00:22:50">[00:22:50]</a>.
*   **Susceptibility Testing**: The Schick Test is an intradermal test used to determine a patient's susceptibility to diphtheria <a class="yt-timestamp" data-t="00:23:52">[00:23:52]</a>, <a class="yt-timestamp" data-t="00:24:00">[00:24:00]</a>, <a class="yt-timestamp" data-t="00:24:04">[00:24:04]</a>. A positive Schick test (redness at the injection site) indicates the patient is susceptible to the infection <a class="yt-timestamp" data-t="00:25:07">[00:25:07]</a>, <a class="yt-timestamp" data-t="00:25:12">[00:25:12]</a>. A negative test indicates immunity <a class="yt-timestamp" data-t="00:25:18">[00:25:18]</a>, <a class="yt-timestamp" data-t="00:25:21">[00:25:21]</a>. The Schick test involves a type 3 hypersensitivity reaction <a class="yt-timestamp" data-t="00:24:23">[00:24:23]</a>, <a class="yt-timestamp" data-t="00:24:26">[00:24:26]</a>.

### _Nocardia_ and _Actinomyces_

Both _Nocardia_ and _Actinomyces_ are Gram-positive branching rods <a class="yt-timestamp" data-t="00:43:06">[00:43:06]</a>, <a class="yt-timestamp" data-t="00:43:15">[00:43:15]</a>, <a class="yt-timestamp" data-t="00:51:58">[00:51:58]</a>.

*   **_Nocardia_**:
    *   **Growth Conditions**: _Nocardia_ is a strict aerobe (requires oxygen) <a class="yt-timestamp" data-t="00:44:52">[00:44:52]</a>, <a class="yt-timestamp" data-t="00:45:00">[00:45:00]</a>.
    *   **Acid-Fastness**: _Nocardia_ is acid-fast <a class="yt-timestamp" data-t="00:45:05">[00:45:05]</a>, <a class="yt-timestamp" data-t="00:45:07">[00:45:07]</a>, and stains reddish <a class="yt-timestamp" data-t="00:49:43">[00:49:43]</a>.
    *   **Culture**: Paraffin Bait technique is a culture method for _Nocardia_ <a class="yt-timestamp" data-t="00:47:00">[00:47:00]</a>. Paraffin provides CO2, which aids its growth <a class="yt-timestamp" data-t="00:47:40">[00:47:40]</a>, <a class="yt-timestamp" data-t="00:47:45">[00:47:45]</a>, <a class="yt-timestamp" data-t="00:47:57">[00:47:57]</a>.
*   **_Actinomyces_**:
    *   **Growth Conditions**: _Actinomyces_ is a strict anaerobe <a class="yt-timestamp" data-t="00:45:03">[00:45:03]</a>.
    *   **Acid-Fastness**: _Actinomyces_ are generally not acid-fast, with the exception of _Actinomyces_ israelii <a class="yt-timestamp" data-t="00:45:10">[00:45:10]</a>, <a class="yt-timestamp" data-t="00:45:17">[00:45:17]</a>, <a class="yt-timestamp" data-t="00:49:55">[00:49:55]</a>.
    *   **Culture Morphology**: On solid agar, _Actinomyces_ forms "molar tooth colonies" <a class="yt-timestamp" data-t="00:48:29">[00:48:29]</a>, <a class="yt-timestamp" data-t="00:48:32">[00:48:32]</a>, <a class="yt-timestamp" data-t="00:48:37">[00:48:37]</a>.
    *   **Macroscopic Features**: In clinical samples, _Actinomyces_ can appear as "sulfur-colored granules" <a class="yt-timestamp" data-t="00:41:59">[00:41:59]</a>, "cotton ball appearance," "woolly ball appearance," or "sunray appearance" <a class="yt-timestamp" data-t="00:52:27">[00:52:27]</a>, <a class="yt-timestamp" data-t="00:52:30">[00:52:30]</a>, <a class="yt-timestamp" data-t="00:52:32">[00:52:32]</a>, <a class="yt-timestamp" data-t="00:52:35">[00:52:35]</a>. These are also known as Gupta bodies <a class="yt-timestamp" data-t="00:52:43">[00:52:43]</a>, <a class="yt-timestamp" data-t="00:52:46">[00:52:46]</a>.
    *   **Splendore-Hoeppli phenomenon**: This refers to the pinkish, radiating, club-like structures formed around the central bacterial colony, indicative of an antigen-antibody complex <a class="yt-timestamp" data-t="00:53:34">[00:53:34]</a>, <a class="yt-timestamp" data-t="00:53:49">[00:53:49]</a>, <a class="yt-timestamp" data-t="00:53:51">[00:53:51]</a>, <a class="yt-timestamp" data-t="00:53:54">[00:53:54]</a>, <a class="yt-timestamp" data-t="00:53:55">[00:53:55]</a>, <a class="yt-timestamp" data-t="00:53:58">[00:53:58]</a>, <a class="yt-timestamp" data-t="00:54:10">[00:54:10]</a>. It is notably seen in actinomycosis and sporotrichosis (_Sporothrix schenckii_) <a class="yt-timestamp" data-t="00:54:26">[00:54:26]</a>, <a class="yt-timestamp" data-t="00:54:34">[00:54:34]</a>, <a class="yt-timestamp" data-t="00:54:37">[00:54:37]</a>.

### _Bacillus anthracis_ (Anthrax)

*   **Microscopy**:
    *   **Gram Stain**: _Bacillus anthracis_ are Gram-positive bacilli <a class="yt-timestamp" data-t="01:03:07">[01:03:07]</a>.
    *   **Arrangement**: They often appear in long chains, resembling "bamboo sticks" or "box cars" <a class="yt-timestamp" data-t="01:02:44">[01:02:44]</a>, <a class="yt-timestamp" data-t="01:02:46">[01:02:46]</a>, <a class="yt-timestamp" data-t="01:02:53">[01:02:53]</a>.
    *   **Capsule Staining**: _Bacillus anthracis_ has a polypeptide capsule (different from the polysaccharide capsules of most other bacteria) <a class="yt-timestamp" data-t="00:57:16">[00:57:16]</a>, <a class="yt-timestamp" data-t="00:57:19">[00:57:19]</a>, <a class="yt-timestamp" data-t="00:57:22">[00:57:22]</a>, <a class="yt-timestamp" data-t="00:57:26">[00:57:26]</a>. This capsule can be visualized using Polychrome Methylene Blue (McFadyean reaction), which stains the capsular material purple around the bacteria <a class="yt-timestamp" data-t="01:11:51">[01:11:51]</a>, <a class="yt-timestamp" data-t="01:12:11">[01:12:11]</a>, <a class="yt-timestamp" data-t="01:12:14">[01:12:14]</a>, <a class="yt-timestamp" data-t="01:12:20">[01:12:20]</a>, <a class="yt-timestamp" data-t="01:12:23">[01:12:23]</a>, <a class="yt-timestamp" data-t="01:12:26">[01:12:26]</a>, <a class="yt-timestamp" data-t="01:12:30">[01:12:30]</a>, <a class="yt-timestamp" data-t="01:12:43">[01:12:43]</a>, <a class="yt-timestamp" data-t="01:12:45">[01:12:45]</a>.

*   **Culture Media**:

    *   **Penicillin Agar**: On penicillin agar, it produces a "string of pearls" appearance <a class="yt-timestamp" data-t="00:55:01">[00:55:01]</a>, <a class="yt-timestamp" data-t="01:02:21">[01:02:21]</a>, <a class="yt-timestamp" data-t="01:02:24">[01:02:24]</a>, <a class="yt-timestamp" data-t="01:05:01">[01:05:01]</a>, <a class="yt-timestamp" data-t="01:05:05">[01:05:05]</a>, <a class="yt-timestamp" data-t="01:05:09">[01:05:09]</a>, <a class="yt-timestamp" data-t="01:05:12">[01:05:12]</a>, <a class="yt-timestamp" data-t="01:05:15">[01:05:15]</a>, <a class="yt-timestamp" data-t="01:05:17">[01:05:17]</a>.

### _Mycobacterium tuberculosis_ (Tuberculosis)

*   **Culture Media**:
    *   **Lowenstein-Jensen (LJ) Medium**: This is a classic medium for _Mycobacterium tuberculosis_ <a class="yt-timestamp" data-t="01:21:26">[01:21:26]</a>, <a class="yt-timestamp" data-t="01:22:25">[01:22:25]</a>.     *   **Colony Appearance**: _Mycobacterium tuberculosis_ forms "rough, buff, tough" colonies on LJ media <a class="yt-timestamp" data-t="01:21:18">[01:21:18]</a>, <a class="yt-timestamp" data-t="01:21:20">[01:21:20]</a>, <a class="yt-timestamp" data-t="01:23:53">[01:23:53]</a>, <a class="yt-timestamp" data-t="01:23:58">[01:23:58]</a>, <a class="yt-timestamp" data-t="01:24:03">[01:24:03]</a>.

        *   "Tough" means the colonies are difficult to remove <a class="yt-timestamp" data-t="01:24:05">[01:24:05]</a>, <a class="yt-timestamp" data-t="01:24:08">[01:24:08]</a>.
