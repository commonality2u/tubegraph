The "Legendary Collection 3: Yugi's World Game Board Edition" is a Yu-Gi-Oh! set that includes the [[promotional_yugioh_cards | Seal of Orichalcos]] card, other [[finding_ultra_rare_yugioh_cards | ultra rare cards]], and tokens <a class="yt-timestamp" data-t="00:00:08">[00:00:08]</a>. This game board edition also features dank mega packs containing reprints from Generation 1 Yu-Gi-Oh! <a class="yt-timestamp" data-t="00:00:21">[00:00:21]</a>. These mega packs include classic spell/trap cards and normal monsters such as Dark Magician and Blue-Eyes White Dragon <a class="yt-timestamp" data-t="00:00:28">[00:00:28]</a>. The set comes with a promo pack and five mega packs <a class="yt-timestamp" data-t="00:01:27">[00:01:27]</a>.

## Promo Pack Contents

The promo pack within the Legendary Collection 3 includes:
*   **The Seal of Orichalcos** <a class="yt-timestamp" data-t="00:01:22">[00:01:22]</a>, <a class="yt-timestamp" data-t="00:02:28">[00:02:28]</a>
    *   This is presented as one of the main attractions of the set <a class="yt-timestamp" data-t="00:01:15">[00:01:15]</a>.
*   **Three Ultra Rares** <a class="yt-timestamp" data-t="00:02:07">[00:02:07]</a>, identified as:
    *   Dark Necrofear <a class="yt-timestamp" data-t="00:02:30">[00:02:30]</a>
    *   Guardian Grarl <a class="yt-timestamp" data-t="00:02:35">[00:02:35]</a>
    *   Five-Headed Dragon <a class="yt-timestamp" data-t="00:02:46">[00:02:46]</a>
*   **Tokens** <a class="yt-timestamp" data-t="00:02:12">[00:02:12]</a>, three included with good artwork <a class="yt-timestamp" data-t="00:02:51">[00:02:51]</a>.
