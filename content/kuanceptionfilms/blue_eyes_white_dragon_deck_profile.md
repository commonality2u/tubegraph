## Main Deck

### Monsters

The monster lineup focuses on establishing powerful Dragon-type monsters and facilitating summons and searches for the main card, Blue-Eyes White Dragon <a class="yt-timestamp" data-t="00:00:30">[00:00:30]</a>.

*   **Blue-Eyes White Dragon** (x3) <a class="yt-timestamp" data-t="00:00:24">[00:00:24]</a>
    *   The main monster of the deck <a class="yt-timestamp" data-t="00:00:30">[00:00:30]</a>.
*   **Maiden with Eyes of Blue** (x3) <a class="yt-timestamp" data-t="00:00:41">[00:00:41]</a>
    *   A one-star tuner <a class="yt-timestamp" data-t="00:00:46">[00:00:46]</a>.
    *   Its effect allows for a Blue-Eyes White Dragon to be summoned to the field when it is targeted by an effect or attacked <a class="yt-timestamp" data-t="00:00:52">[00:00:52]</a>.
    *   Considered the main MVP of the deck <a class="yt-timestamp" data-t="00:00:56">[00:00:56]</a>.
*   **White Stone of Legend** (x3) <a class="yt-timestamp" data-t="00:01:02">[00:01:02]</a>
    *   A great tuner <a class="yt-timestamp" data-t="00:01:05">[00:01:05]</a>.
    *   Adds a Blue-Eyes White Dragon to your hand when sent to the graveyard, which is useful for discarding abilities or milling <a class="yt-timestamp" data-t="00:01:09">[00:01:09]</a>.
*   **Rider of the Stormwinds** (x1) <a class="yt-timestamp" data-t="00:01:19">[00:01:19]</a>
    *   Provides an extra tuner with a useful effect <a class="yt-timestamp" data-t="00:01:22">[00:01:22]</a>.
*   **Kaiba Man** (x3) <a class="yt-timestamp" data-t="00:01:28">[00:01:28]</a>
    *   Features a modified artwork <a class="yt-timestamp" data-t="00:01:31">[00:01:31]</a>.
*   **Parsec, the Interstellar Dragon** (x1) <a class="yt-timestamp" data-t="00:01:33">[00:01:33]</a>
    *   Can be normal summoned without tributing if you control a Level 8 monster, enabling easy Xyz Level 8 summons <a class="yt-timestamp" data-t="00:01:38">[00:01:38]</a>.
*   **Dark Storm Dragon** (x1) <a class="yt-timestamp" data-t="00:01:52">[00:01:52]</a>
    *   An alternative Level 8 monster with a good effect <a class="yt-timestamp" data-t="00:01:59">[00:01:59]</a>.
    *   It is a Gemini card, treated as a normal monster <a class="yt-timestamp" data-t="00:02:03">[00:02:03]</a>.
*   **Red-Eyes Darkness Metal Dragon** (x1) <a class="yt-timestamp" data-t="00:02:10">[00:02:10]</a>
    *   Has an ability similar to Azure-Eyes Silver Dragon's <a class="yt-timestamp" data-t="00:02:16">[00:02:16]</a>.
    *   Easy to summon by removing one Dragon-type monster from play <a class="yt-timestamp" data-t="00:02:19">[00:02:19]</a>.
*   **Shining Angel** (x2) <a class="yt-timestamp" data-t="00:02:26">[00:02:26]</a>
    *   Used to easily summon Maiden with Eyes of Blue if she isn't drawn in the first turn <a class="yt-timestamp" data-t="00:02:31">[00:02:31]</a>.
*   **Honest** (x1) <a class="yt-timestamp" data-t="00:02:47">[00:02:47]</a>
    *   Great for any Light-type deck, as the majority of cards in this [[key_monsters_in_blue_eyes_white_dragon_deck | Blue Eyes White Dragon Deck]] are Light attribute <a class="yt-timestamp" data-t="00:02:52">[00:02:52]</a>.
    *   Features a modified artwork <a class="yt-timestamp" data-t="00:02:47">[00:02:47]</a>.
*   **Degard** (x1) <a class="yt-timestamp" data-t="00:03:02">[00:03:02]</a>
    *   Can tribute one monster to change another monster's level to eight, which is great for Level 8 Xyz summoning <a class="yt-timestamp" data-t="00:03:05">[00:03:05]</a>.
*   **Gagaga Magician** (x1) <a class="yt-timestamp" data-t="00:03:13">[00:03:13]</a>
    *   Can declare a level from 1 to 8 once per turn, becoming that level, which is excellent for Xyz summoning <a class="yt-timestamp" data-t="00:03:16">[00:03:16]</a>.
