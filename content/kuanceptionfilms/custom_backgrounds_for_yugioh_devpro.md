Yu-Gi-Oh DevPro allows players to personalize their game experience by adding custom backgrounds and wallpapers <a class="yt-timestamp" data-t="00:00:15">[00:00:15]</a>. This guide outlines the steps to add these customizations. For general customization methods, see [[how_to_add_game_customizations_in_yugioh_devpro | How to Add Game Customizations in YuGiOh Devpro]].

## Navigating to Customization Settings

To begin customizing your DevPro experience:
1.  From the game list, go to the "Customize" tab <a class="yt-timestamp" data-t="00:00:23">[00:00:23]</a>.
2.  Click on "Content" <a class="yt-timestamp" data-t="00:00:27">[00:00:27]</a>.
3.  This section also allows for adding [[custom_sleeves_for_yugioh_devpro | custom sleeves]] <a class="yt-timestamp" data-t="00:00:34">[00:00:34]</a>.

## Adding Custom Wallpapers/Backgrounds

### General Backgrounds
1.  From the "Content" menu, select "Backgrounds" <a class="yt-timestamp" data-t="01:46:00">[01:46:00]</a>.
2.  Click "Add" <a class="yt-timestamp" data-t="01:53:00">[01:53:00]</a>.
3.  Navigate to your desired image file <a class="yt-timestamp" data-t="01:56:00">[01:56:00]</a>.
4.  If prompted to resize the image to the correct size, select "Yes" <a class="yt-timestamp" data-t="02:05:00">[02:05:00]</a>, <a class="yt-timestamp" data-t="02:09:00">[02:09:00]</a>.
5.  Right-click on the newly added image and select "Install" <a class="yt-timestamp" data-t="02:12:00">[02:12:00]</a>. A "success" prompt will appear <a class="yt-timestamp" data-t="02:16:00">[02:16:00]</a>.
6.  You can then "Preview" your new background <a class="yt-timestamp" data-t="02:19:00">[02:19:00]</a>.

### In-Game Backgrounds (Game Backgrounds)
The background shown in the customization menu is separate from the actual in-game background during a duel <a class="yt-timestamp" data-t="02:24:00">[02:24:00]</a>. To change the background seen during a duel:
1.  From the "Content" menu, select "Game backgrounds" <a class="yt-timestamp" data-t="02:36:00">[02:36:00]</a>.
2.  Click "Add" <a class="yt-timestamp" data-t="02:38:00">[02:38:00]</a>.
3.  Select your desired image file <a class="yt-timestamp" data-t="02:48:00">[02:48:00]</a>.
4.  If prompted to resize, confirm "Yes" <a class="yt-timestamp" data-t="02:51:00">[02:51:00]</a>.
5.  Right-click on the added image and select "Install" <a class="yt-timestamp" data-t="02:57:00">[02:57:00]</a>. You will see a "success" notification <a class="yt-timestamp" data-t="03:00:00">[03:00:00]</a>.
6.  Use the "Preview" option to confirm the installation <a class="yt-timestamp" data-t="03:02:00">[03:02:00]</a>.

> "As you can see it's correctly installed." <a class="yt-timestamp" data-t="03:09:00">[03:09:00]</a>

You can change your in-game wallpaper as many times as you like by repeating the installation process for a different image <a class="yt-timestamp" data-t="03:13:00">[03:13:00]</a>.

## Verifying Installation
To confirm that both your custom sleeves and wallpaper are correctly applied, start a duel <a class="yt-timestamp" data-t="03:36:00">[03:36:00]</a>. Both elements should appear as selected <a class="yt-timestamp" data-t="03:46:00">[03:46:00]</a>.
