This guide details how to add and preview custom content, such as card sleeves and backgrounds, within the Yu-Gi-Oh! Devpro application <a class="yt-timestamp" data-t="00:00:09">[00:00:09]</a>.

## Accessing Customization Settings

To begin customizing your [[gameplay experience | gameplay experience]] in Devpro:
1.  Navigate to the "Game List" within the Devpro application <a class="yt-timestamp" data-t="00:00:23">[00:00:23]</a>.
2.  Select the "Customize" tab <a class="yt-timestamp" data-t="00:00:25">[00:00:25]</a>.
3.  Click on "Content" <a class="yt-timestamp" data-t="00:00:30">[00:00:30]</a>.

## [[Custom sleeves for YuGiOh Devpro | Custom Sleeves]]

Custom sleeves are player-specific card covers that appear during duels.

### Adding Custom Sleeves
1.  From the "Content" section, select "Player One Covers" to customize your sleeves <a class="yt-timestamp" data-t="00:00:34">[00:00:34]</a>, <a class="yt-timestamp" data-t="00:00:36">[00:00:36]</a>.
2.  Ensure your image is correctly sized to avoid distortion <a class="yt-timestamp" data-t="00:00:48">[00:00:48]</a>, <a class="yt-timestamp" data-t="00:00:51">[00:00:51]</a>.
    *   Devpro will prompt you to resize the image to the correct dimensions if needed <a class="yt-timestamp" data-t="00:01:15">[00:01:15]</a>, <a class="yt-timestamp" data-t="00:01:17">[00:01:17]</a>.
3.  Click "Add" <a class="yt-timestamp" data-t="00:00:56">[00:00:56]</a>.
4.  Browse to the location of your desired image file and click "Open" <a class="yt-timestamp" data-t="00:00:58">[00:00:58]</a>, <a class="yt-timestamp" data-t="00:01:13">[00:01:13]</a>.
5.  If prompted to resize the image, click "Yes" <a class="yt-timestamp" data-t="00:01:15">[00:01:15]</a>, <a class="yt-timestamp" data-t="00:01:17">[00:01:17]</a>.
6.  Right-click on the newly added sleeve in the list <a class="yt-timestamp" data-t="00:01:26">[00:01:26]</a>, <a class="yt-timestamp" data-t="00:01:28">[00:01:28]</a>.
7.  Select "Install" <a class="yt-timestamp" data-t="00:01:28">[00:01:28]</a>. A "Success" prompt will appear <a class="yt-timestamp" data-t="00:01:31">[00:01:31]</a>.

### Previewing Custom Sleeves
1.  Click on the newly installed sleeve <a class="yt-timestamp" data-t="00:01:24">[00:01:24]</a>.
2.  Click "Preview" <a class="yt-timestamp" data-t="00:01:34">[00:01:34]</a>.
3.  The game will load a preview where you can see your customized sleeves <a class="yt-timestamp" data-t="00:01:37">[00:01:37]</a>, <a class="yt-timestamp" data-t="00:01:41">[00:01:41]</a>.

## [[Custom backgrounds for YuGiOh Devpro | Custom Backgrounds]]

There are two types of backgrounds: general backgrounds and in-game duel backgrounds.

### General Custom Backgrounds
1.  From the "Content" section, select "Backgrounds" <a class="yt-timestamp" data-t="00:01:49">[00:01:49]</a>, <a class="yt-timestamp" data-t="00:01:52">[00:01:52]</a>.
2.  Click "Add" <a class="yt-timestamp" data-t="00:01:56">[00:01:56]</a>.
3.  Browse to your desired image file and click "Open" <a class="yt-timestamp" data-t="00:01:58">[00:01:58]</a>, <a class="yt-timestamp" data-t="00:02:00">[00:02:00]</a>.
4.  If prompted to resize the image, click "Yes" <a class="yt-timestamp" data-t="00:02:05">[00:02:05]</a>, <a class="yt-timestamp" data-t="00:02:07">[00:02:07]</a>, <a class="yt-timestamp" data-t="00:02:09">[00:02:09]</a>.
5.  Right-click on the added background and select "Install" <a class="yt-timestamp" data-t="00:02:12">[00:02:12]</a>, <a class="yt-timestamp" data-t="00:02:16">[00:02:16]</a>. A "Success" prompt will appear <a class="yt-timestamp" data-t="00:02:16">[00:02:16]</a>.

### Previewing General Backgrounds
1.  Click "Preview" <a class="yt-timestamp" data-t="00:02:19">[00:02:19]</a>.
2.  The game will load a preview showing your selected background <a class="yt-timestamp" data-t="00:02:19">[00:02:19]</a>, <a class="yt-timestamp" data-t="00:02:21">[00:02:21]</a>.

### In-Game Duel Backgrounds
General backgrounds do not automatically change the background seen during an actual duel <a class="yt-timestamp" data-t="00:02:24">[00:02:24]</a>, <a class="yt-timestamp" data-t="00:02:26">[00:02:26]</a>. To change the in-game background:
1.  From the "Customize" tab, select "Game Backgrounds" <a class="yt-timestamp" data-t="00:02:36">[00:02:36]</a>, <a class="yt-timestamp" data-t="00:02:38">[00:02:38]</a>.
2.  The process for adding, installing, and previewing is the same as for general backgrounds and sleeves <a class="yt-timestamp" data-t="00:02:41">[00:02:41]</a>, <a class="yt-timestamp" data-t="00:02:44">[00:02:44]</a>.
3.  After adding your desired image, right-click on it and select "Install" <a class="yt-timestamp" data-t="00:02:51">[00:02:51]</a>, <a class="yt-timestamp" data-t="00:02:57">[00:02:57]</a>, <a class="yt-timestamp" data-t="00:03:00">[00:03:00]</a>.
4.  Click "Preview" to see the in-game background <a class="yt-timestamp" data-t="00:03:02">[00:03:02]</a>. It may take a moment to buffer <a class="yt-timestamp" data-t="00:03:06">[00:03:06]</a>, <a class="yt-timestamp" data-t="00:03:09">[00:03:09]</a>.

## Verifying Custom Content in a Duel

To confirm that your custom sleeves and in-game backgrounds are correctly applied, start an actual duel <a class="yt-timestamp" data-t="00:03:36">[00:03:36]</a>. The customized wallpaper and sleeves should be visible during the duel <a class="yt-timestamp" data-t="00:03:46">[00:03:46]</a>, <a class="yt-timestamp" data-t="00:03:48">[00:03:48]</a>.
