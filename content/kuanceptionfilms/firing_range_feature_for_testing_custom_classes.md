## Accessing the Firing Range
Players can access the firing range when creating custom classes <a class="yt-timestamp" data-t="02:49:00">[02:49:00]</a>. It can be used both when not in a pre-game lobby and when in a pre-game lobby, serving as a way to pass time while waiting for a game to start <a class="yt-timestamp" data-t="03:19:00">[03:19:00]</a>.

## Benefits and Uses
The firing range provides several advantages for players optimizing their loadouts:
*   **Weapon Testing** Players can test any gun setup they wish to try <a class="yt-timestamp" data-t="03:30:00">[03:30:00]</a>, including various weapon types like the IMR assault rifle <a class="yt-timestamp" data-t="01:47:00">[01:47:00]</a> or sniper rifles like the Moors <a class="yt-timestamp" data-t="03:39:00">[03:39:00]</a>.
*   **Recoil Assessment** It allows players to see if their weapon has recoil <a class="yt-timestamp" data-t="03:03:00">[03:03:00]</a>.
*   **Optics and Sights** Players can evaluate if they like the optics or iron sights on their weapon <a class="yt-timestamp" data-t="03:06:00">[03:06:00]</a>.
*   **Fire Rate Check** The feature helps determine if the weapon's fire rate is suitable for the player <a class="yt-timestamp" data-t="03:13:00">[03:13:00]</a>.

The ability to instantly test setups makes the firing range a "very, very useful" tool for players <a class="yt-timestamp" data-t="03:54:00">[03:54:00]</a>.
