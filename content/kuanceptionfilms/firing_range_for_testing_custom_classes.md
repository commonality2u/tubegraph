*Call of Duty: Advanced Warfare* introduces a useful feature for players to test out their custom class setups: a dedicated firing range <a class="yt-timestamp" data-t="02:49:00">[02:49:00]</a>. This feature is integrated directly into the custom class creation menu <a class="yt-timestamp" data-t="02:49:00">[02:49:00]</a>.

## Accessing the Firing Range
When creating custom classes in *Call of Duty: Advanced Warfare*, players can press "start" to access the firing range <a class="yt-timestamp" data-t="02:49:00">[02:49:00]</a>.

## Utility of the Firing Range
The firing range is highly practical for optimizing weapon setups <a class="yt-timestamp" data-t="03:00:00">[03:00:00]</a>. It allows players to:
*   Assess weapon recoil <a class="yt-timestamp" data-t="03:03:00">[03:03:00]</a>.
*   Evaluate different optics <a class="yt-timestamp" data-t="03:06:00">[03:06:00]</a>.
*   Check iron sights <a class="yt-timestamp" data-t="03:09:00">[03:09:00]</a>.
*   Determine if the fire rate is satisfactory <a class="yt-timestamp" data-t="03:13:00">[03:13:00]</a>.

It supports testing any gun setup, including specific weapons like the Moors <a class="yt-timestamp" data-t="03:32:00">[03:32:00]</a>.

## Availability
A key advantage of the firing range is its accessibility:
*   It can be used both when not in a pre-game lobby <a class="yt-timestamp" data-t="03:21:00">[03:21:00]</a>.
*   It can also be utilized while waiting in a pre-game lobby, serving as a way to pass the time before a match starts <a class="yt-timestamp" data-t="03:25:00">[03:25:00]</a>.

This feature is considered very useful and a favorite new addition to the game <a class="yt-timestamp" data-t="03:52:00">[03:52:00]</a>.
