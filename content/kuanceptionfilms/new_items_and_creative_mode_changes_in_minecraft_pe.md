The [[minecraft_pocket_edition_080_update_features | 0.8.0 update]] for Minecraft Pocket Edition, available on iOS and Android, is described as the biggest update for Minecraft Pocket Edition so far <a class="yt-timestamp" data-t="00:00:27">[00:00:27]</a>. It includes significant changes to [[gameplay_mechanics_and_item_usage | gameplay mechanics]] and expands the range of available items, particularly within Creative Mode.

## New Items

The update introduces several new blocks and items, enhancing building and farming capabilities:

*   **Colored Wood** Different types of colored wood are now available <a class="yt-timestamp" data-t="00:01:27">[00:01:27]</a>.
*   **Hay Stacks** These provide an efficient way to store hay, allowing for more hay to be kept in a smaller space, such as a chest <a class="yt-timestamp" data-t="00:01:33">[00:01:33]</a>.
*   **Building Materials**
    *   Iron bars <a class="yt-timestamp" data-t="00:01:56">[00:01:56]</a>
    *   Stone fences <a class="yt-timestamp" data-t="00:02:03">[00:02:03]</a>
*   **Sponges** Sponges have been added, though their functionality is noted as not doing anything specific <a class="yt-timestamp" data-t="00:02:07">[00:02:07]</a>.
*   **[[new_plant_types_and_seeds_in_minecraft_pe | New Plant Types and Seeds]]** The update introduces a variety of new seeds and plants, expanding farming options <a class="yt-timestamp" data-t="00:02:19">[00:02:19]</a>:
    *   Pumpkin seeds <a class="yt-timestamp" data-t="00:02:26">[00:02:26]</a>
    *   Melon seeds <a class="yt-timestamp" data-t="00:02:30">[00:02:30]</a>
    *   Carrot seeds <a class="yt-timestamp" data-t="00:02:30">[00:02:30]</a>
    *   Potato seeds <a class="yt-timestamp" data-t="00:02:32">[00:02:32]</a>
    *   Beetroot seeds <a class="yt-timestamp" data-t="00:02:37">[00:02:37]</a>
*   **Dyes** More dyes and ink sacks are now available <a class="yt-timestamp" data-t="00:02:37">[00:02:37]</a>.

## Creative Mode Enhancements

Creative Mode sees significant improvements, giving players more tools and flexibility:

*   **Expanded Item Slots** Creative Mode now features more item slots to accommodate the increased number of available items <a class="yt-timestamp" data-t="00:00:54">[00:00:54]</a>.
*   **Bedrock Building** Players can now build with bedrock in Creative Mode <a class="yt-timestamp" data-t="00:02:10">[00:02:10]</a>.
*   **TNT Functionality** For the first time, TNT can be used in Creative Mode, allowing players to detonate structures <a class="yt-timestamp" data-t="00:02:46">[00:02:46]</a>.

The [[minecraft_pocket_edition_080_update_features | 0.8.0 update]] is free for users who already own the app <a class="yt-timestamp" data-t="00:03:14">[00:03:14]</a>.
