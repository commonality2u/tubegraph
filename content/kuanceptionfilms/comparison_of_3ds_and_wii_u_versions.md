The video discusses differences and similarities between the Nintendo 3DS and Wii U versions of *[[super_smash_bros_4 | Super Smash Bros. 4]]*.

## Release Dates
The Nintendo 3DS version of *[[super_smash_bros_4 | Super Smash Bros. 4]]* was released on October 3, 2014 <a class="yt-timestamp" data-t="00:00:14">[00:00:14]</a>. The Wii U version was scheduled to be released later, in December of the same year <a class="yt-timestamp" data-t="00:00:50">[00:00:50]</a>.

## Character Roster
Both versions of the game feature different characters, with some being exclusive to the 3DS version <a class="yt-timestamp" data-t="00:01:28">[00:01:28]</a>. The manual for the 3DS version shows characters such as Rosalina, Little Mac, Greninja, Palutena, Sonic, Mega Man, and Pac-Man <a class="yt-timestamp" data-t="00:01:54">[00:01:54]</a>. Additionally, new Mii character types like Mii Brawler, Mii Swordfighter, and Mii Gunner are included, all featuring new moves and combos <a class="yt-timestamp" data-t="00:02:19">[00:02:19]</a>.
