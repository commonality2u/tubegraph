## Core Capabilities of AI Studio

### Model Selection and Performance <a class="yt-timestamp" data-t="00:07:07">[00:07:07]</a>
AI Studio provides access to various Gemini models, including:
*   **Gemini 2.0 Flash:** A powerful and slightly more expensive model <a class="yt-timestamp" data-t="00:07:39">[00:07:39]</a>.
*   **Gemini Flashlight:** Less intelligent but offers higher rate limits and similar core functions <a class="yt-timestamp" data-t="00:07:43">[00:07:43]</a>.
*   **Gemini Pro:** The most intelligent experimental model available <a class="yt-timestamp" data-t="00:07:49">[00:07:49]</a>.
*   **Gemma:** Google's open-source version of the models <a class="yt-timestamp" data-t="00:07:21">[00:07:21]</a>.

### Long Context Processing <a class="yt-timestamp" data-t="00:02:52">[00:02:52]</a>
One of AI Studio's standout features is its ability to handle very long contexts <a class="yt-timestamp" data-t="00:03:42">[00:03:42]</a>.
*   **Demonstration:** A 30-minute video of a Natural History Museum tour was processed to extract a list of all museum exhibits <a class="yt-timestamp" data-t="00:03:54">[00:03:54]</a>.
*   The model successfully handled 531,000 tokens for the video <a class="yt-timestamp" data-t="00:04:48">[00:04:48]</a>.
*   This capability can be used to extract intelligence from various media, including long audio files like podcasts <a class="yt-timestamp" data-t="00:06:01">[00:06:01]</a>.
*   Such data extraction is nearly impossible without these models, typically requiring extensive manual work or complex processing steps <a class="yt-timestamp" data-t="00:04:32">[00:04:32]</a>, <a class="yt-timestamp" data-t="00:05:35">[00:05:35]</a>.
*   [[ai_startup_ideas_and_frameworks | Startup Builders]] can leverage this for creating online directories by extracting trapped data from various media <a class="yt-timestamp" data-t="00:05:11">[00:05:11]</a>.

### Reasoning Models <a class="yt-timestamp" data-t="00:07:59">[00:07:59]</a>
Reasoning models represent a new frontier in AI capabilities, allowing the model to "think" before generating a final output <a class="yt-timestamp" data-t="00:08:04">[00:08:04]</a>, <a class="yt-timestamp" data-t="00:08:32">[00:08:32]</a>.
*   **Demonstration:** A Python code snippet was prompted to be turned into a "fully fledged website plus Landing Page Plus [[using_ai_tools_for_saas_development | SAS app]]" called "AI Studio" <a class="yt-timestamp" data-t="00:09:05">[00:09:05]</a>, <a class="yt-timestamp" data-t="00:10:02">[00:10:02]</a>, <a class="yt-timestamp" data-t="00:10:07">[00:10:07]</a>.
*   The model displayed a "thoughts" category in the UI, outlining desired outcomes, code concepts, technology stack, and MVP functionality (e.g., user authentication, dashboard, image generation tools) before generating the output <a class="yt-timestamp" data-t="00:10:16">[00:10:16]</a>, <a class="yt-timestamp" data-t="00:11:12">[00:11:12]</a>.
*   This thought process, similar to human outlining, allows for more structured and detailed outputs <a class="yt-timestamp" data-t="00:11:30">[00:11:30]</a>, <a class="yt-timestamp" data-t="00:11:44">[00:11:44]</a>.
*   The entire process for this complex task took only 23 seconds <a class="yt-timestamp" data-t="00:12:15">[00:12:15]</a>.
*   Users can obtain an API key for the reasoning model to use it in [[features_and_control_levels_of_various_ai_coding_platforms | other AI coding platforms]] like Cursor <a class="yt-timestamp" data-t="00:12:50">[00:12:50]</a>, <a class="yt-timestamp" data-t="00:13:04">[00:13:04]</a>.

### Starter Apps and Multimodal Capabilities <a class="yt-timestamp" data-t="00:13:26">[00:13:26]</a>
AI Studio includes three simple starter apps to demonstrate various model capabilities <a class="yt-timestamp" data-t="00:13:36">[00:13:36]</a>.
*   **Spatial Understanding:** This capability allows the model to deeply understand and visually represent objects <a class="yt-timestamp" data-t="00:13:50">[00:13:50]</a>.
    *   **Demonstration:** A picture was processed to identify item positions and generate 2D bounding boxes, dynamically overlaid on the image <a class="yt-timestamp" data-t="00:14:02">[00:14:02]</a>, <a class="yt-timestamp" data-t="00:14:14">[00:14:14]</a>.
    *   This is useful for generic object detection, such as identifying furniture in a room for online sales or inventory management <a class="yt-timestamp" data-t="00:14:34">[00:14:34]</a>, <a class="yt-timestamp" data-t="00:15:52">[00:15:52]</a>.
    *   It can also be applied to real-time video feeds for monitoring (e.g., parking garage utilization) or satellite analysis <a class="yt-timestamp" data-t="00:16:05">[00:16:05]</a>, <a class="yt-timestamp" data-t="00:16:21">[00:16:21]</a>.
*   **Function Calling:** Demonstrates how AI can integrate with existing products.
    *   **Demonstration:** AI Studio and Gemini were hooked up to the Google Maps API to create a "geoguesser" experience. Prompting "take me somewhere in ancient history" resulted in the model showing an isolated island with historical context <a class="yt-timestamp" data-t="00:18:18">[00:18:18]</a>, <a class="yt-timestamp" data-t="00:18:41">[00:18:41]</a>.
    *   This highlights the potential for "combinatorial explosion" where AI bridges historically separate products to create new business opportunities <a class="yt-timestamp" data-t="00:19:04">[00:19:04]</a>, <a class="yt-timestamp" data-t="00:19:15">[00:19:15]</a>.
*   All starter app code is available on GitHub, allowing users to download, modify, and build upon them with a free API key <a class="yt-timestamp" data-t="00:19:38">[00:19:38]</a>, <a class="yt-timestamp" data-t="00:19:52">[00:19:52]</a>.

### Real-Time Streaming and AI Co-Presence <a class="yt-timestamp" data-t="00:20:05">[00:20:05]</a>
The multimodal live API powers a real-time streaming experience, enabling AI to see and understand the user's screen in real-time, offering contextual help <a class="yt-timestamp" data-t="00:20:11">[00:20:11]</a>, <a class="yt-timestamp" data-t="00:20:22">[00:20:22]</a>.
*   **Demonstration:** The model listened to spoken commands and observed a Python code editor on screen <a class="yt-timestamp" data-t="00:20:35">[00:20:35]</a>.
    *   It identified the open file (`f.py`) and its use of the Gemini API <a class="yt-timestamp" data-t="00:20:52">[00:20:52]</a>, <a class="yt-timestamp" data-t="00:20:58">[00:20:58]</a>.
    *   When the user ran the code and encountered errors (wrong directory, invalid API key), the AI provided real-time debugging suggestions, understanding both the audio and visual context <a class="yt-timestamp" data-t="00:21:13">[00:21:13]</a>, <a class="yt-timestamp" data-t="00:21:33">[00:21:33]</a>, <a class="yt-timestamp" data-t="00:21:59">[00:21:59]</a>.
*   This feature is considered an early look at "AI co-presence," where AI acts as a pair programmer or assistant, understanding the entire screen, including browsers, and providing help <a class="yt-timestamp" data-t="00:23:14">[00:23:14]</a>, <a class="yt-timestamp" data-t="00:23:29">[00:23:29]</a>, <a class="yt-timestamp" data-t="00:23:40">[00:23:40]</a>.
*   It democratizes access to expert assistance, helping users overcome steep learning curves in areas like coding or video editing <a class="yt-timestamp" data-t="00:24:03">[00:24:03]</a>, <a class="yt-timestamp" data-t="00:24:31">[00:24:31]</a>, <a class="yt-timestamp" data-t="00:27:33">[00:27:33]</a>.
*   The multimodal live API allows for native tool integrations, pseudo function calls, code execution within a virtual environment, and internet browsing (grounding) for comprehensive assistance <a class="yt-timestamp" data-t="00:25:26">[00:25:26]</a>, <a class="yt-timestamp" data-t="00:25:51">[00:25:51]</a>.
*   This demo is seen as a glimpse into the [[ai_coding_tools_comparison | future of work]], where an AI partner provides intelligence and value to ongoing tasks, significantly increasing productivity <a class="yt-timestamp" data-t="00:24:52">[00:24:52]</a>, <a class="yt-timestamp" data-t="00:25:17">[00:25:17]</a>.
