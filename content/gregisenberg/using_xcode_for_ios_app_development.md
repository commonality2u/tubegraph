1.  Start a new project in Xcode <a class="yt-timestamp" data-t="04:30:00">[04:30:00]</a>.
2.  Save it to a chosen location <a class="yt-timestamp" data-t="04:34:00">[04:34:00]</a>.
3.  Open Cursor <a class="yt-timestamp" data-t="04:36:00">[04:36:00]</a>.
4.  Hit "open via file" and select the same project file <a class="yt-timestamp" data-t="04:38:00">[04:38:00]</a>.
5.  The two tools should now be connected <a class="yt-timestamp" data-t="04:41:00">[04:41:00]</a>.
