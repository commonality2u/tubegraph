## Categories of AI Coding Tools
AI coding tools can be broadly categorized into three groups based on the user's technical skill and the tool's functionality <a class="yt-timestamp" data-t="00:01:39">[00:01:39]</a>:

1.  **No-code Tools**: These tools are designed for non-technical users who don't need to know how to program <a class="yt-timestamp" data-t="00:01:46">[00:01:46]</a>.
2.  **Middle-Ground Tools**: These platforms cater to both non-programmers and those who know how to write code <a class="yt-timestamp" data-t="00:01:51">[00:01:51]</a>.
3.  **Technical AI Tools**: Primarily meant for developers and programmers, offering high levels of control <a class="yt-timestamp" data-t="00:02:02">[00:02:02]</a>. Examples include Wind Surf and Cursor <a class="yt-timestamp" data-t="00:02:05">[00:02:05]</a>.

While tools are generally suited for specific user types, non-technical users can still use more complex tools like Cursor, though it may involve more debugging and learning <a class="yt-timestamp" data-t="00:02:17">[00:02:17]</a>.

## Key Considerations When Choosing a Tool
Two major factors to consider when selecting an AI coding tool are **control** and **technical proficiency** <a class="yt-timestamp" data-t="00:02:30">[00:02:30]</a>.

### Control vs. Technicality
*   **Low Control / Non-Technical**: Lovable offers minimal code editing control, making it ideal for those with no programming grasp <a class="yt-timestamp" data-t="00:02:48">[00:02:48]</a>.
*   **Moderate Control / Semi-Technical**: Bolt and v0 provide access to code for tweaking <a class="yt-timestamp" data-t="00:03:07">[00:03:07]</a>. [[repet | Repet]] offers even more control with file creation <a class="yt-timestamp" data-t="00:03:13">[00:03:13]</a>. Tempo Labs, while less popular, combines elements of Lovable, Figma, and a code editor, allowing editing of components like in Figma <a class="yt-timestamp" data-t="00:03:38">[00:03:38]</a>. Onlook is excellent for technical or semi-technical users who struggle with design, helping create good-looking landing pages and providing the code <a class="yt-timestamp" data-t="00:03:26">[00:03:26]</a>.
*   **Ultimate Control / Technical**: Cursor and Wind Surf offer full access to the code, allowing changes, edits, and package installations <a class="yt-timestamp" data-t="00:03:53">[00:03:53]</a>.

Most users, particularly Greg's audience, are likely in the semi-technical range, where tools like v0, [[repet | Repet]], Tempo Labs, Bolt, and Lovable are straightforward to use <a class="yt-timestamp" data-t="00:04:12">[00:04:12]</a>. Cursor and Wind Surf are primarily for technical folks, but can be used by non-technical individuals if they are prepared for debugging and learning <a class="yt-timestamp" data-t="00:04:58">[00:04:58]</a>.

### Production Readiness
A common concern is whether these AI coding tools are capable of creating production-ready products rather than just prototypes <a class="yt-timestamp" data-t="00:06:33">[00:06:33]</a>.

Currently, no tool is 100% production-ready without significant roadblocks or headaches <a class="yt-timestamp" data-t="00:07:10">[00:07:10]</a>. However, some are very close:
*   Tempo Labs <a class="yt-timestamp" data-t="00:07:15">[00:07:15]</a>
*   [[repet | Repet]] <a class="yt-timestamp" data-t="00:07:19">[00:07:19]</a>
*   Bolt and Lovable are progressing towards this <a class="yt-timestamp" data-t="00:07:23">[00:07:23]</a>.
*   Cursor and Wind Surf can create production applications if the user is willing to endure the challenges <a class="yt-timestamp" data-t="00:07:26">[00:07:26]</a>.

## Tool Classification by Functionality
Tools can also be classified by their core functionalities:

*   **CD (Continuous Deployment) + Code Generation + Deployment**: Lovable, Bolt, [[repet | Repet]], v0 <a class="yt-timestamp" data-t="00:11:37">[00:11:37]</a>.
*   **Code Generation + Code Editor**: [[repet | Repet]], Bolt, v0, Wind Surf, Cursor <a class="yt-timestamp" data-t="00:11:47">[00:11:47]</a>.
*   **Code Generation + Code Editor + Figma-like (no CD)**: Tempo Labs, which allows tweaking individual components but only has preview deployments <a class="yt-timestamp" data-t="00:11:58">[00:11:58]</a>.
*   **CD + Code Editor**: [[repet | Repet]], Bolt, v0 <a class="yt-timestamp" data-t="00:12:16">[00:12:16]</a>.

## Comparison of Specific Tool Strengths
While all tools are valuable, some excel in particular areas:

*   **Integrations**: Lovable is highly effective, with easy integrations for services like Stripe and Superbase <a class="yt-timestamp" data-t="00:13:21">[00:13:21]</a>. [[repet | Repet]] is a close second <a class="yt-timestamp" data-t="00:13:40">[00:13:40]</a>.
*   **Collaboration**: Tempo Labs and [[repet | Repet]] are strong for team collaboration, especially for mixed teams with technical and design roles <a class="yt-timestamp" data-t="00:13:58">[00:13:58]</a>. Tempo often has an edge <a class="yt-timestamp" data-t="00:14:17">[00:14:17]</a>.
*   **Deployment**: Lovable, [[repet | Repet]], and Bolt offer excellent deployment experiences with little significant difference <a class="yt-timestamp" data-t="00:14:20">[00:14:20]</a>. Tempo Labs currently offers only preview deployments but is working on production deployments <a class="yt-timestamp" data-t="00:14:31">[00:14:31]</a>.
*   **Control**: Tempo leads in providing fine-grained control, followed by [[repet | Repet]], then Bolt and Lovable. Bolt offers more direct code editing than Lovable, which requires GitHub connection for edits <a class="yt-timestamp" data-t="00:14:41">[00:14:41]</a>.

## Pricing and Agents
Most AI coding tools have similar base pricing, typically between $20-$40 for starter tiers <a class="yt-timestamp" data-t="00:15:23">[00:15:23]</a>. Crucially, almost all offer generous free tiers with daily or weekly caps, allowing users to test extensively before committing to a paid plan <a class="yt-timestamp" data-t="00:15:35">[00:15:35]</a>.

Regarding "agents" (e.g., [[repet | Repet]] Agent, Cursor Agent), these are beneficial when a user doesn't have a clear idea of what they want to build, as agents can generate more complete outputs <a class="yt-timestamp" data-t="00:16:57">[00:16:57]</a>. However, if a user has detailed specifications (e.g., colors, functionalities, animations for a to-do list app), the presence of an agent feature might not be as critical <a class="yt-timestamp" data-t="00:18:35">[00:18:35]</a>. All tools generally use similar underlying AI models like Claude or Deep Seek <a class="yt-timestamp" data-t="00:18:11">[00:18:11]</a>. [[comparison_between_ai_agents_and_traditional_automation_tools | AI agents]] can produce a more finished product, but the choice depends on how much detailed information the user has about their desired product <a class="yt-timestamp" data-t="00:18:27">[00:18:27]</a>.

## Recommendations for Different User Profiles
The choice of tool depends on the user's profile and desired experience <a class="yt-timestamp" data-t="00:19:30">[00:19:30]</a>:

*   **Non-technical / Idea Guy**: If you're a non-technical person with ideas looking to quickly prototype, Lovable and Bolt are great options <a class="yt-timestamp" data-t="00:19:46">[00:19:46]</a>.
*   **Technical / Likes to Code**: For technical individuals who enjoy coding, Wind Surf and Cursor provide the necessary control <a class="yt-timestamp" data-t="00:19:56">[00:19:56]</a>.
*   **Technical / Wants Control but Not Code**: If you're technical but prefer not to code directly while still wanting control, [[repet | Repet]], Tempo, and Bolt are suitable <a class="yt-timestamp" data-t="00:20:01">[00:20:01]</a>.
*   **Non-technical / Product-minded / Wants Control**: For non-technical individuals who are product-minded, use Figma, or appreciate intricate design components and full product cycle access, Tempo is highly recommended <a class="yt-timestamp" data-t="00:20:08">[00:20:08]</a>.

Ultimately, any tool can be used by anyone, even if it requires overcoming initial challenges <a class="yt-timestamp" data-t="00:20:26">[00:20:26]</a>. It is advisable to align with tools that target your specific profile, as they are continuously improving and securing funding <a class="yt-timestamp" data-t="00:20:39">[00:20:39]</a>. While no tool is perfect or can build full-grade production applications effortlessly right now, many are 70-80% there <a class="yt-timestamp" data-t="00:20:43">[00:20:43]</a>.
