*   **Marketing Shift**: Changed how brands connect with audiences <a class="yt-timestamp" data-t="00:04:54">[00:04:54]</a>.

Instead of a physical team handling roles like content writer, graphic designer, and social media manager, one individual can now perform the exact same job using AI tools like Chat GPT and Leonardo AI <a class="yt-timestamp" data-t="01:10:09">[01:10:09]</a>.

## Sequential Prompting: A New AI Technique
Omar introduces a third way to use AI tools, beyond conversational prompts and master prompts: **sequential prompting** <a class="yt-timestamp" data-t="01:0:19">[01:0:19]</a>.
*   **Concept**: Use Chat GPT to generate a prompt that is then used in a different AI tool to create another output. This output can then be fed into another AI tool, creating a sequence of transformations <a class="yt-timestamp" data-t="01:17:17">[01:17:17]</a>.
*   **Workflow Example**:
    1.  Chat GPT (generates prompt from an image).
    2.  Leonardo AI (generates image from prompt).
    3.  Cling AI (animates image into video).
    4.  Eleven Labs (adds voiceover to video).
*   **Implementation**: Start manually to iterate and refine prompts and outputs. Once satisfied, automate the process using tools like Zapier or Make, or connect APIs programmatically <a class="yt-timestamp" data-t="01:21:04">[01:21:04]</a>.

### 1. Personalized Storytelling and Greeting Cards
This model focuses on creating unique, personalized animated stories or images using AI.

#### Creating Consistent AI Avatars
To maintain character consistency across different images or story frames, a specific prompt structure is used with Chat GPT <a class="yt-timestamp" data-t="01:14:15">[01:14:15]</a>.
*   Upload a reference image (e.g., a child's photo) <a class="yt-timestamp" data-t="01:46:04">[01:46:04]</a>.
*   Provide basic information (name, age, gender) <a class="yt-timestamp" data-t="01:56:49">[01:56:49]</a>.
*   Chat GPT analyzes the image and generates a detailed prompt description for the character, including hairstyle, skin tone, accessories, and outfit <a class="yt-timestamp" data-t="01:57:00">[01:57:00]</a>.
*   This prompt is then used in Leonardo AI to generate consistent Disney/Pixar-style images <a class="yt-timestamp" data-t="02:07:09">[02:07:09]</a>.

#### Example: "Greg and His Magic Cup"
An example video demonstrates how a consistent character (Greg) can be used to tell a story through a series of AI-generated images that resemble a Disney Pixar universe <a class="yt-timestamp" data-t="01:15:49">[01:15:49]</a>.

#### Business Potential: AI-First "Moonpig"
*   **Concept**: Create an AI-first version of online custom greeting card services like Moonpig, which generates hundreds of millions in revenue annually <a class="yt-timestamp" data-t="02:57:56">[02:57:56]</a>.
*   **Offering**:
    *   **Dynamic Character & Scene Creation**: Users upload photos, and AI generates personalized characters in various scenes <a class="yt-timestamp" data-t="03:32:32">[03:32:32]</a>.
    *   **AI-Generated Stories**: Turn every card into a mini-story (e.g., "Kaden the brave explorer") <a class="yt-timestamp" data-t="03:34:37">[03:34:37]</a>.

    *   **Video Integration**: Add AI-generated voiceovers (e.g., via Eleven Labs) or short animated videos (via Cling AI) <a class="yt-timestamp" data-t="03:55:00">[03:55:00]</a>.

### 3. Personalized Stickers
This business model involves generating personalized stickers using AI and automating the printing and fulfillment process.
*   **Concept**: Offer personalized stickers, perhaps for teachers to give to students, featuring AI-generated images of the teacher or student with congratulatory messages <a class="yt-timestamp" data-t="02:51:00">[02:51:00]</a>. This acts as an incentive for kids to work harder <a class="yt-timestamp" data-t="02:54:00">[02:54:00]</a>.
*   **Workflow**:
    1.  **Image Generation**: Use Chat GPT to create a prompt for a "kiss cut sticker artwork" (a style that can be cut out) <a class="yt-timestamp" data-t="02:56:10">[02:56:10]</a>.
    2.  **Image Creation**: Use Leonardo AI to generate the image with text (e.g., "Congratulations") <a class="yt-timestamp" data-t="02:59:00">[02:59:00]</a>.
    3.  **Background Removal**: Use a tool like Remove BG (which has an API) to automatically remove the background, creating a transparent PNG sticker <a class="yt-timestamp" data-t="03:08:00">[03:08:00]</a>.
    4.  **Printing & Fulfillment**: Connect to a Drop Shipping tool or third-party printing API like Printful API to automatically fulfill orders and print physical stickers or other products (t-shirts, clothing) <a class="yt-timestamp" data-t="03:10:00">[03:10:00]</a>.
*   **Marketing**: A "guerrilla hacking" approach involves taking photos of school staff from their websites, generating personalized sticker sheets with their AI avatar, and mailing them to the school <a class="yt-timestamp" data-t="03:21:00">[03:21:00]</a>. This can create a viral effect and grab attention <a class="yt-timestamp" data-t="03:17:00">[03:17:00]</a>.
