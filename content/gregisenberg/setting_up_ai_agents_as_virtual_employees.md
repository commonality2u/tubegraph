## Agent Swarms: Powering Parallel Tasks

An "agent swarm" is the ability to send a list of tasks to an [[building_ai_agents_for_business_automation | AI agent]], which then reliably, quickly, and in parallel performs all these tasks <a class="yt-timestamp" data-t="00:10:31">[00:10:31]</a>. While individual [[building_ai_agents_for_business_automation | AI agents]] are powerful, they can lose coherence over long-term, extensive tasks <a class="yt-timestamp" data-t="00:10:44">[00:10:44]</a>. Swarms address this by duplicating the agent to handle each item on the list concurrently, similar to "Agent Smith" duplicating himself in *The Matrix* <a class="yt-timestamp" data-t="00:11:15">[00:11:15]</a>. This allows for efficient execution of large-scale operations like personalized lead outreach to hundreds of people <a class="yt-timestamp" data-t="00:10:51">[00:10:51]</a>.
