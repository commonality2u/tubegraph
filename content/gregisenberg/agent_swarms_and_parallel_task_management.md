## What is an Agent Swarm?
An agent swarm is the capability to send a list of tasks to an [[ai_agents_and_their_applications | AI agent]] for reliable, quick, and parallel execution <a class="yt-timestamp" data-t="00:10:31">[00:10:31]</a>. While [[ai_agents_and_their_applications | AI agents]] are powerful, they can lose coherence or fail over long-term horizons, especially when given extensive lists of tasks <a class="yt-timestamp" data-t="00:10:44">[00:10:44]</a>. An agent swarm addresses this by duplicating the agent, sending a copy to handle each individual item on the list concurrently <a class="yt-timestamp" data-t="00:11:15">[00:11:15]</a>.

## Benefits and Applications of Agent Swarms

### Enhanced Reliability and Speed
Agent swarms prevent unreliability that can occur when a single agent processes a large number of tasks sequentially <a class="yt-timestamp" data-t="00:11:09">[00:11:09]</a>. By distributing tasks in parallel, they ensure that all items are completed reliably and quickly <a class="yt-timestamp" data-t="00:10:38">[00:10:38]</a>. This approach makes it possible to [[building_ai_agents_with_longterm_tasks | build AI agents with long-term tasks]] <a class="yt-timestamp" data-t="00:10:44">[00:10:44]</a>.

### Examples of Agent Swarm Use Cases

#### 1. Personalized Lead Outreach
One significant application for [[ai_agents_and_their_applications | AI agents]] and swarms is personalized lead outreach <a class="yt-timestamp" data-t="00:10:56">[00:10:56]</a>. Instead of an agent taking a long time and potentially failing on the 200th lead, an agent swarm can duplicate itself to send a personalized message to each lead simultaneously <a class="yt-timestamp" data-t="00:11:02">[00:11:02]</a>.
*   **Recruitment:** A recruiter Lindy can search for people based on criteria, present a list, and then, upon selection, deploy a swarm to research each chosen candidate on platforms like Perplexity and send personalized outreach emails <a class="yt-timestamp" data-t="00:16:11">[00:16:11]</a>, <a class="yt-timestamp" data-t="00:17:05">[00:17:05]</a>. This process is cost-effective and efficient for recruiting <a class="yt-timestamp" data-t="00:20:29">[00:20:29]</a>.
*   **Sales Prospecting:** Swarms can be used for sales prospecting, including uploading CSVs of leads from other platforms. A "sales outreacher" Lindy can contact hundreds of founders with personalized emails, streamlining developer support or managing data for free games programs <a class="yt-timestamp" data-t="00:20:50">[00:20:50]</a>, <a class="yt-timestamp" data-t="00:21:26">[00:21:26]</a>.

#### 2. Meeting Preparation ("Swarm of Swarms")
A complex application involves a "swarm of swarms" for meeting preparation <a class="yt-timestamp" data-t="00:11:25">[00:11:25]</a>.
*   Every morning, the primary agent checks scheduled meetings for the day <a class="yt-timestamp" data-t="00:11:31">[00:11:31]</a>.
*   It then deploys an agent swarm for *each* meeting <a class="yt-timestamp" data-t="00:11:37">[00:11:37]</a>.
*   Inside each meeting swarm, another swarm is deployed for *every attendee* <a class="yt-timestamp" data-t="00:11:43">[00:11:43]</a>.
*   Each sub-agent then retrieves relevant information, such as past meeting notes, emails, LinkedIn profiles, and then compiles a daily digest <a class="yt-timestamp" data-t="00:11:48">[00:11:48]</a>, <a class="yt-timestamp" data-t="00:12:06">[00:12:06]</a>.

#### 3. Virtual Standups and Team Reporting (Elon Lindy)
The "Elon Lindy" uses both phone call and swarm capabilities <a class="yt-timestamp" data-t="00:28:56">[00:28:56]</a>. It can be given a list of team members, and weekly, it will call each teammate for a virtual standup to ask what they accomplished <a class="yt-timestamp" data-t="00:29:01">[00:29:01]</a>. Afterward, it sends a report summarizing the entire team's activities to the CEO, effectively replacing aspects of middle management <a class="yt-timestamp" data-t="00:29:36">[00:29:36]</a>.

#### 4. Competitive Analysis
A "competitive tracker" Lindy uses a swarm to monitor competitors <a class="yt-timestamp" data-t="00:30:16">[00:30:16]</a>.
*   Monthly, it retrieves a list of competitors from a spreadsheet <a class="yt-timestamp" data-t="00:30:37">[00:30:37]</a>.
*   It deploys a swarm for each competitor <a class="yt-timestamp" data-t="00:30:42">[00:30:42]</a>.
*   Each sub-agent looks up specific information such as employee count, traffic estimates, recent news, funding, and hiring <a class="yt-timestamp" data-t="00:30:47">[00:30:47]</a>.
*   Finally, it sends a report identifying competitors gaining traction and logs all data in a spreadsheet, tracking trends over time <a class="yt-timestamp" data-t="00:30:57">[00:30:57]</a>. This offers a more objective and less emotionally taxing way to monitor competition <a class="yt-timestamp" data-t="00:31:49">[00:31:49]</a>.

#### 5. Influencer Outreach
Agent swarms are used for influencer outreach, with some customers deploying Lindys to find influencers on platforms like TikTok, Instagram, and YouTube, obtain their contact information, and offer partnerships <a class="yt-timestamp" data-t="00:30:00">[00:30:00]</a>, <a class="yt-timestamp" data-t="00:25:34">[00:25:34]</a>. Cases have even been observed where one Lindy (from a brand) interacts with another Lindy (from an influencer managing their emails) <a class="yt-timestamp" data-t="00:24:54">[00:24:54]</a>.

#### 6. Virtual Focus Groups and User Feedback
While not explicitly stated as a swarm, the concept of using LLMs to simulate human behavior is closely related to the parallel execution capability of swarms <a class="yt-timestamp" data-t="00:26:48">[00:26:48]</a>.
*   LLMs are good at emulating human responses, making them useful for simulating focus groups <a class="yt-timestamp" data-t="00:27:05">[00:27:05]</a>.
*   Lindy.AI uses an agent to simulate different user personas, allowing them to gather feedback on features and products <a class="yt-timestamp" data-t="00:27:22">[00:27:22]</a>. This approach is estimated to be 80% as effective as a real focus group but costs cents instead of thousands of dollars <a class="yt-timestamp" data-t="00:27:38">[00:27:38]</a>.
*   The next step for "prompt-to-app" platforms may involve building an app MVP, sending it to thousands of virtual users via a swarm, getting feedback in minutes, and iterating rapidly <a class="yt-timestamp" data-t="00:28:28">[00:28:28]</a>.

## Human in the Loop
When deploying [[ai_agents_and_their_applications | AI agents]], especially those involving swarms that perform outreach or critical actions, it's advisable to "insert a human in the loop" if the agent could potentially cause embarrassment <a class="yt-timestamp" data-t="00:18:02">[00:18:02]</a>. This is analogous to training an intern; initially, supervision is required <a class="yt-timestamp" data-t="00:18:14">[00:18:14]</a>. Future features will allow agents to learn from human feedback and corrections <a class="yt-timestamp" data-t="00:18:27">[00:18:27]</a>.
