## Sequential Prompting: A Key Technique for AI Content Creation

A powerful technique for [[utilizing_ai_for_writing_and_content_creation | using AI for content creation]] is **sequential prompting** <a class="yt-timestamp" data-t="00:10:37">[00:10:37]</a>. This involves using ChatGPT to generate a prompt, which is then used in a different AI tool to generate a new output <a class="yt-timestamp" data-t="00:11:17">[00:11:17]</a>.

*   **Workflow**:
    1.  Provide ChatGPT with an image reference (e.g., of a child) and basic details <a class="yt-timestamp" data-t="00:17:40">[00:17:40]</a>.
    2.  Ask ChatGPT to generate a descriptive prompt for an AI avatar based on the image, ensuring consistency in character traits <a class="yt-timestamp" data-t="00:15:35">[00:15:35]</a>.
    3.  Take this generated prompt and input it into an image generation tool like Leonardo AI to create high-resolution, Disney-Pixar style images <a class="yt-timestamp" data-t="00:20:07">[00:20:07]</a>.
    4.  (Optional) Further sequence the process by taking the generated image to a tool like Cling AI to turn it into a video, or Eleven Labs for voiceovers <a class="yt-timestamp" data-t="00:12:10">[00:12:10]</a>.

This sequential process allows for a controlled output, creating consistent characters for storytelling <a class="yt-timestamp" data-t="00:19:50">[00:19:50]</a>. While manual execution helps refine prompts, the ultimate goal is to connect these tools programmatically via APIs, enabling automated workflows and the rapid generation of diverse content <a class="yt-timestamp" data-t="00:18:48">[00:18:48]</a>.

### 1. AI-First Personalized Greeting Cards (Moonpig Model)

Inspired by companies like Moonpig.com, which generates $400 million in revenue from custom greeting cards <a class="yt-timestamp" data-t="00:28:54">[00:28:54]</a>, an AI-first version could revolutionize the industry.

*   **Concept**: Create a platform where users can upload a photo of themselves or a loved one, and AI generates personalized, Disney-Pixar style characters and narratives for greeting cards <a class="yt-timestamp" data-t="00:23:54">[00:23:54]</a>.
*   **AI-Driven Features**:
    *   **Dynamic Character & Scene Creation**: Automatically generate avatars and scenes based on user input <a class="yt-timestamp" data-t="00:32:32">[00:32:32]</a>.
    *   **AI-Generated Stories**: Turn every card into a mini-story, personalizing greetings beyond static images <a class="yt-timestamp" data-t="00:32:34">[00:32:34]</a>.
    *   **AI-Enhanced Templates**: Offer templates that adapt based on user preferences like tone or voice <a class="yt-timestamp" data-t="00:32:48">[00:32:48]</a>.
    *   **Video Integration**: Allow users to add AI-generated voiceovers or short animated videos (GIFs) to cards <a class="yt-timestamp" data-t="00:29:46">[00:29:46]</a>.
*   **Revenue Streams**: Premium card sales with a markup for AI-driven personalization, monthly memberships for unlimited cards, and upselling matched gifts like flowers or chocolates <a class="yt-timestamp" data-t="00:33:04">[00:33:04]</a>.
*   **Workflow**:
    1.  User uploads an image to a custom interface (e.g., built on Webflow or Framer) <a class="yt-timestamp" data-t="00:36:40">[00:36:40]</a>.
    2.  The image is sent to ChatGPT's API with a specific prompt to generate a consistent character description <a class="yt-timestamp" data-t="00:18:48">[00:18:48]</a>.
    3.  This description is then sent to Leonardo AI's API to generate the custom image <a class="yt-timestamp" data-t="00:20:59">[00:20:59]</a>.
    4.  (Optional) The image can be fed into Cling AI's API for video animation and lip-syncing for custom messages <a class="yt-timestamp" data-t="00:50:51">[00:50:51]</a>.
    5.  The final asset is delivered to the user for digital sharing or integration with print-on-demand services.
