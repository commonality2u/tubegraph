## Key AI Tools for Content Creation

Several [[content_creation_tools_and_ai_technology | AI tools]] are highlighted for their utility in creating Disney-style content:
*   **Leonardo AI:** Used for [[using_ai_for_graphic_design_and_branding | generating high-quality images]] from prompts <a class="yt-timestamp" data-t="00:32:28">[00:32:28]</a>. It supports different aspect ratios and can generate images quickly <a class="yt-timestamp" data-t="00:20:43">[00:20:43]</a>.
*   **Cling AI:** Excellent for turning static images into videos <a class="yt-timestamp" data-t="00:43:40">[00:43:40]</a>. It also offers a lip-sync function to make characters speak words <a class="yt-timestamp" data-t="00:50:52">[00:50:52]</a>.
*   **ChatGPT / Claude AI:** Primarily used for generating prompts, descriptions, and even full business plans <a class="yt-timestamp" data-t="00:14:10">[00:14:10]</a>. It can analyze images and provide detailed descriptions for consistent character generation <a class="yt-timestamp" data-t="00:19:27">[00:19:27]</a>.
*   **11 Labs:** A tool for adding voiceovers to animated content <a class="yt-timestamp" data-t="00:12:12">[00:12:12]</a>.
*   **Remove.bg:** Utilized to quickly remove image backgrounds, creating clean PNGs ideal for stickers or other products <a class="yt-timestamp" data-t="01:00:03">[01:00:03]</a>.
*   **Printful:** A third-party printing API that enables the automation of physical product fulfillment, connecting digital creations to physical goods like stickers or clothing <a class="yt-timestamp" data-t="01:01:03">[01:01:03]</a>.

## Sequential Prompting: The Core Workflow

A key technique for [[using_ai_for_content_creation | using AI for content creation]] is "sequential prompting" <a class="yt-timestamp" data-t="00:10:40">[00:10:40]</a>. This involves using one AI tool (like ChatGPT) to generate a specific prompt, which is then fed into another AI tool (like Leonardo AI) to create an output <a class="yt-timestamp" data-t="00:11:47">[00:11:47]</a>. This process can be chained across multiple tools, similar to "Lego bricks," to build complex content <a class="yt-timestamp" data-t="00:12:07">[00:12:07]</a>.

It is recommended to first perform this process manually to refine prompts and workflows before automating it with tools like Zapier or by bringing on an engineer for more complex integrations <a class="yt-timestamp" data-t="00:19:05">[00:19:05]</a>.

### Creating AI Avatars

The process of creating a consistent AI avatar typically starts by providing ChatGPT with an image reference (e.g., a photo of a child) along with basic information like name, age, and gender <a class="yt-timestamp" data-t="00:17:53">[00:17:53]</a>. ChatGPT then analyzes the image and generates a detailed prompt that includes descriptions of hairstyle, skin tone, accessories, and outfit <a class="yt-timestamp" data-t="00:19:27">[00:19:27]</a>. This prompt ensures character consistency across different scenes or stories <a class="yt-timestamp" data-t="00:19:57">[00:19:57]</a>.

This generated prompt is then input into Leonardo AI, which uses it to create high-resolution, Disney/Pixar-style images <a class="yt-timestamp" data-t="00:20:07">[00:20:07]</a>. Leonardo AI can generate these images in a matter of seconds <a class="yt-timestamp" data-t="00:21:28">[00:21:28]</a>. The quality is so high it feels like being in a Pixar film <a class="yt-timestamp" data-t="00:17:06">[00:17:06]</a>.

### Animating Avatars with Cling AI

Once static images are created in Leonardo AI, they can be uploaded to Cling AI to turn them into short video clips <a class="yt-timestamp" data-t="00:43:40">[00:43:40]</a>. These 5-second videos currently take around four minutes to generate but are expected to become much faster <a class="yt-timestamp" data-t="00:51:06">[00:51:06]</a>. Cling AI also offers a lip-sync feature, allowing the generated characters to speak specific words <a class="yt-timestamp" data-t="00:50:52">[00:50:52]</a>.
