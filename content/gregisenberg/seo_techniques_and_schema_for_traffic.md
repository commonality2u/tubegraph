## Case Study 1: Guilty Chef (Recipe Directory)

Omar, the guest, developed "Guilty Chef," a recipe website that uses AI and technology to generate recipes <a class="yt-timestamp" data-t="00:05:00">[00:05:00]</a>.
*   **Concept**: Create a directory of recipes from various restaurants globally, as no comprehensive online source existed <a class="yt-timestamp" data-t="00:06:05">[00:06:05]</a>.
*   **Technology**: Webflow for design <a class="yt-timestamp" data-t="00:05:40">[00:05:40]</a>, ChatGPT and Zapier for automatic content generation <a class="yt-timestamp" data-t="00:05:53">[00:05:53]</a>. Each page, including images, descriptions, prep/cook times, ingredients, directions, and FAQs, is automatically generated by AI <a class="yt-timestamp" data-t="00:06:35">[00:06:35]</a>.
*   **[[Conversion Rate Optimization Techniques | Schema]] Integration**: The site uses recipe schema (found on schema.org) to tell Google exactly what content is on the page, including cook time, cooking method, nutritional value, and steps <a class="yt-timestamp" data-t="00:07:16">[00:07:16]</a>. AI can write the entire schema <a class="yt-timestamp" data-t="00:09:03">[00:09:03]</a>.
*   **Ranking Success**: Guilty Chef ranks number one on Google for specific restaurant menu items, even above established sites like Trip Advisor, within six months <a class="yt-timestamp" data-t="00:14:40">[00:14:40]</a>. It leverages rich snippets by correctly implementing schema for FAQs <a class="yt-timestamp" data-t="00:15:06">[00:15:06]</a>.
*   **Traffic and Effort**: The site expects to exceed 100,000 monthly visitors by December, having started from zero in March <a class="yt-timestamp" data-t="00:11:52">[00:11:52]</a>. Omar spent only about a week full-time and a couple of hours a month on it as a side project <a class="yt-timestamp" data-t="00:12:12">[00:12:12]</a>.
*   **AI Re-indexing**: Content generated by AI and ranked on Google is then re-indexed by AI engines like ChatGPT and Claude, driving traffic back to the source <a class="yt-timestamp" data-t="00:17:01">[00:17:01]</a>.
*   **Monetization**: Current monetization includes a paid recipe generator <a class="yt-timestamp" data-t="00:11:23">[00:11:23]</a>. Potential avenues include sponsorships, partnerships with food brands, and a future ad platform based on ingredients <a class="yt-timestamp" data-t="00:18:28">[00:18:28]</a>. The long-term vision is to offer software to restaurants or become a go-to personalized recipe tool <a class="yt-timestamp" data-t="00:20:53">[00:20:53]</a>. ## Case Study 2: Best Dubai (Travel Directory)

Omar applied the same principles to a travel directory called "Best Dubai," acquiring the domain for around $1,200 <a class="yt-timestamp" data-t="00:25:04">[00:25:04]</a>.
*   **Domain Value**: Bestdubai.com was considered a bargain given Dubai's popularity and search trends like "best hotels in Dubai" <a class="yt-timestamp" data-t="00:25:31">[00:25:31]</a>.
*   **AI-Generated Content**: All content, including images, is AI-generated <a class="yt-timestamp" data-t="00:26:26">[00:26:26]</a>. Claude was used to write the algorithm for generating scores for places based on reviews <a class="yt-timestamp" data-t="00:26:47">[00:26:47]</a>.
*   **Unique Data Acquisition**: A custom Chrome extension, built with Claude's help, scrapes up-to-date restaurant menus and pricing from delivery services (e.g., Deliveroo, Talabat) to provide unique content and capture "restaurant name + menu" searches <a class="yt-timestamp" data-t="00:30:57">[00:30:57]</a>.
*   **Rapid Ranking**: Within 30 days, Best Dubai ranked as the fourth organic link for "Parkers Dubai reviews," competing with giants like Trip Advisor and Zomato <a class="yt-timestamp" data-t="00:27:15">[00:27:15]</a>.
*   **Marketing & Branding**: The site has unique tags like "Dubai's Best 100" and plans to issue physical awards to restaurants, building brand weight in the city <a class="yt-timestamp" data-t="00:29:41">[00:29:41]</a>.

## How to Build an AI-Powered Directory (Tutorial)

This section outlines the mechanics for building a similar directory for any industry, using Montreal as an example <a class="yt-timestamp" data-t="00:33:07">[00:33:07]</a>.

### 1. Niche Selection and Research
*   Choose a niche and a specific location, e.g., Montreal <a class="yt-timestamp" data-t="00:35:17">[00:35:17]</a>.
*   Instead of relying solely on AI for content, find nuanced information from "real-world" sources like local bloggers or TikTok accounts for unique insights, which can then be indexed on your site <a class="yt-timestamp" data-t="00:36:10">[00:36:10]</a>.

### 2. Data Collection with Apify
*   Apify is a platform with numerous scrapers for various platforms, including Google Maps <a class="yt-timestamp" data-t="00:37:09">[00:37:09]</a>.
*   Use the "Google Maps Business Scraper" to extract detailed information about places <a class="yt-timestamp" data-t="00:37:41">[00:37:41]</a>.
*   Find a place's ID using the Google Developer Platform's Place ID Finder <a class="yt-timestamp" data-t="00:38:07">[00:38:07]</a>.
*   Input the Place ID into the Apify scraper to retrieve rich data, including address, images, and popular times (occupancy percentages) <a class="yt-timestamp" data-t="00:39:53">[00:39:53]</a>.

### 3. Content Generation and Refinement with Claude
*   Apify provides data in JSON format <a class="yt-timestamp" data-t="00:40:02">[00:40:02]</a>.
*   Feed this JSON data to Claude (or ChatGPT) to understand and repurpose the content <a class="yt-timestamp" data-t="00:40:19">[00:40:19]</a>. Claude can identify useful information like "busiest periods" <a class="yt-timestamp" data-t="00:40:40">[00:40:40]</a>.
*   **Prompt Engineering**:
    *   Give Claude guidelines, e.g., print output in Canadian French <a class="yt-timestamp" data-t="00:56:05">[00:56:05]</a>.
    *   Structure prompts with mini-prompts for specific data points. Use asterisks around titles (e.g., `*Place Name*:`) followed by instructions for Claude <a class="yt-timestamp" data-t="00:58:18">[00:58:18]</a>.
    *   Be creative with mini-prompts (e.g., "print the base URL and append `?ref=gregr`") <a class="yt-timestamp" data-t="00:59:12">[00:59:12]</a>.
    *   You can even ask Claude to write an algorithm for scoring specific categories (e.g., food, service) <a class="yt-timestamp" data-t="01:02:17">[01:02:17]</a>.

### 4. Automation with Zapier and Code
*   **Trigger**: Use Google Sheets as the trigger. Adding a Place ID to a column in Google Sheets will initiate the automation <a class="yt-timestamp" data-t="00:45:19">[00:45:19]</a>.
*   **Apify Integration**: Zapier has direct Apify integration. Configure Zapier to run the Google Business Scraper using the Place ID from the Google Sheet <a class="yt-timestamp" data-t="00:47:20">[00:47:20]</a>.
*   **JSON Parsing with Python (Zapier Code Step)**: Apify's raw output needs to be formatted for Claude. Use a "Run Python" code step in Zapier to extract the complete JSON string from Apify's output <a class="yt-timestamp" data-t="00:52:00">[00:52:00]</a>.
*   **Claude Integration**: Send the parsed JSON data to Claude via Zapier. Claude processes this data based on the detailed prompts and outputs structured information <a class="yt-timestamp" data-t="00:55:08">[00:55:08]</a>.
*   **Parsing Claude's Output with Python (Zapier Code Step)**: Claude's output is a single text block. Use another "Run Python" code step to parse this text, using the asterisk delimiters to break it down into individual strings for each piece of information (e.g., place name, URL, scores) <a class="yt-timestamp" data-t="01:07:55">[01:07:55]</a>. This allows for dynamic data mapping in Webflow <a class="yt-timestamp" data-t="01:08:36">[01:08:36]</a>.

### 5. Website Building with Webflow
*   **CMS Setup**: Create a Collection in Webflow's Content Management System (CMS), e.g., "Restaurants." Add fields for all the information you want to display (name, total score, URL, etc.) <a class="yt-timestamp" data-t="01:09:06">[01:09:06]</a>. Remember to publish the collection for Zapier to find it <a class="yt-timestamp" data-t="01:11:08">[01:11:08]</a>.
*   **Dynamic Pages**: Design a "Restaurant Template" page in Webflow. Each design element (e.g., title, score, link) can be dynamically connected to a specific field in your CMS <a class="yt-timestamp" data-t="01:09:58">[01:09:58]</a>.
*   **Zapier to Webflow**: Configure Zapier to create a "Live Item" in your Webflow CMS. Map the individual strings (e.g., place name, scores, URLs) from the parsed Claude output to the corresponding fields in your Webflow CMS <a class="yt-timestamp" data-t="01:11:23">[01:11:23]</a>.
*   **Slug Creation**: To optimize [[keyword_research_and_organic_growth_strategies | SEO]], have Claude generate the URL slug (e.g., `/parkers-dubai`) so it can be correctly referenced when generating schema <a class="yt-timestamp" data-t="01:12:17">[01:12:17]</a>.

This automated process allows for mass uploading of data, generating entire styled pages from a single input like a Place ID <a class="yt-timestamp" data-t="00:46:02">[00:46:02]</a>.
