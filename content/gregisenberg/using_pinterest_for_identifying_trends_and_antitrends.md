## Identifying Trends with Pinterest

Pinterest displays various trends, showing both weekly and monthly changes in popularity <a class="yt-timestamp" data-t="03:15:00">[03:15:00]</a>. Examples of trends observed on Pinterest include:
*   Fall outfits <a class="yt-timestamp" data-t="03:17:00">[03:17:00]</a>
*   Back to school <a class="yt-timestamp" data-t="03:20:00">[03:20:00]</a>
*   Home gardening <a class="yt-timestamp" data-t="03:25:00">[03:25:00]</a>
*   Main character mode <a class="yt-timestamp" data-t="03:27:00">[03:27:00]</a>
*   Sun Soo spaces <a class="yt-timestamp" data-t="03:31:00">[03:31:00]</a>

When trends show significant monthly changes, such as 10,000% or more, they indicate a large and growing interest <a class="yt-timestamp" data-t="03:34:00">[03:34:00]</a>, <a class="yt-timestamp" data-t="03:37:00">[03:37:00]</a>.

## Identifying Antitrends with Pinterest

One method for [[antitrends_in_startup_ideas | identifying antitrends]] is to observe what is currently trending on platforms like Pinterest and then consider the opposite <a class="yt-timestamp" data-t="03:42:00">[03:42:00]</a>, <a class="yt-timestamp" data-t="03:45:00">[03:45:00]</a>, <a class="yt-timestamp" data-t="03:48:49">[03:48:49]</a>. For instance, if "brat summer" is trending, its [[antitrends_in_startup_ideas | antitrend]] might be "demure fall" <a class="yt-timestamp" data-t="04:00:00">[04:00:00]</a>, <a class="yt-timestamp" data-t="04:01:00">[04:01:00]</a>, <a class="yt-timestamp" data-t="04:03:00">[04:03:00]</a>. There is always an [[antitrends_in_startup_ideas | antitrend]] to be found <a class="yt-timestamp" data-t="04:07:00">[04:07:00]</a>.

While Pinterest Trends only shows data for one year <a class="yt-timestamp" data-t="04:53:00">[04:53:00]</a>, <a class="yt-timestamp" data-t="04:55:00">[04:55:00]</a>, it highlights "related trends" <a class="yt-timestamp" data-t="04:58:00">[04:58:00]</a>.

For example, when exploring "slow living," related trends like "minimalist living rooms" or "cozy bedroom" appear, which can spark ideas for products or services <a class="yt-timestamp" data-t="05:01:00">[05:01:00]</a>, <a class="yt-timestamp" data-t="05:03:00">[05:03:00]</a>, <a class="yt-timestamp" data-t="05:05:00">[05:05:00]</a>, <a class="yt-timestamp" data-t="05:07:00">[05:07:00]</a>.
