## What are AI Agents?
Unlike traditional workflow automation platforms, AI agents on Lindy.AI operate as intelligent entities capable of understanding natural language prompts for each step in a sequence <a class="yt-timestamp" data-t="00:03:30">[00:03:30]</a>. This means a user can "speak to" a step in English, telling it what to do without needing separate AI steps for processing information <a class="yt-timestamp" data-t="00:03:42">[00:03:42]</a>. The agent maintains context across steps, allowing for more fluid and intelligent execution <a class="yt-timestamp" data-t="00:04:10">[00:04:10]</a>. If an agent makes a mistake, it can often recover from it <a class="yt-timestamp" data-t="00:04:29">[00:04:29]</a>.

## Agent Swarms
A significant advancement in AI agents is the concept of "agent swarms" <a class="yt-timestamp" data-t="00:10:20">[00:10:20]</a>. An agent swarm enables an AI agent to duplicate itself and handle a list of tasks reliably, quickly, and in parallel <a class="yt-timestamp" data-t="00:11:15">[00:11:15]</a>. This overcomes the challenge of single AI agents losing coherence or failing over long-term tasks involving many items <a class="yt-timestamp" data-t="00:10:44">[00:10:44]</a>. For example, instead of one agent sending personalized outreach to 500 people slowly and potentially unreliably, an agent swarm can deploy a copy of itself for each lead, ensuring efficiency and consistency <a class="yt-timestamp" data-t="00:10:51">[00:10:51]</a>.

*   **Start Small:** Begin with simpler, personal assistant use cases to understand how the platform works, then gradually add more complexity <a class="yt-timestamp" data-t="00:15:07">[00:15:07]</a>.
