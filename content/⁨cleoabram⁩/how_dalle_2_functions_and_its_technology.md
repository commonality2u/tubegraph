## Core Technologies: CLIP and unCLIP

DALLE 2 is built upon two primary technologies: CLIP and unCLIP <a class="yt-timestamp" data-t="00:06:46">[00:06:46]</a>.

### CLIP (Contrastive Language–Image Pre-training)
CLIP is a model trained on a vast dataset of images paired with their corresponding captions from the internet <a class="yt-timestamp" data-t="00:06:53">[00:06:53]</a>, <a class="yt-timestamp" data-t="00:09:02">[00:09:02]</a>. Its function is to learn the common ground between visual information (images) and linguistic information (text captions) <a class="yt-timestamp" data-t="00:07:04">[00:07:04]</a>.

Through this training, CLIP understands various concepts, including:
*   Aesthetics <a class="yt-timestamp" data-t="00:07:13">[00:07:13]</a>
*   Style <a class="yt-timestamp" data-t="00:07:13">[00:07:13]</a>
*   The general meaning of what might be written as a caption for an image <a class="yt-timestamp" data-t="00:07:13">[00:07:13]</a>

When a user provides a text prompt, DALLE 2 first utilizes CLIP to generate CLIP’s "conception" of what should be in the image <a class="yt-timestamp" data-t="00:07:25">[00:07:25]</a>. This means it understands the abstract qualities and contextual elements implied by the text.

### unCLIP
Following CLIP's conceptualization, unCLIP takes over <a class="yt-timestamp" data-t="00:07:27">[00:07:27]</a>. It uses CLIP's understanding, the original text prompt, and random noise to generate multiple visual options for what the final image might look like <a class="yt-timestamp" data-t="00:07:28">[00:07:28]</a>. This process is distinct from collaging, which involves combining existing image segments <a class="yt-timestamp" data-t="00:09:10">[00:09:10]</a>. Instead, DALLE 2 creates entirely new images based on its learned understanding of styles, colors, and concepts <a class="yt-timestamp" data-t="00:09:19">[00:09:19]</a>.

## User Interaction and Creative Potential

Users can provide highly specific prompts to guide the AI's image generation <a class="yt-timestamp" data-t="00:02:43">[00:02:43]</a>. This includes:
*   Adding specific colors (e.g., "a yellow astronaut riding a skateboard") <a class="yt-timestamp" data-t="00:02:43">[00:02:43]</a>
*   Modifying elements (e.g., "With the head of a duck") <a class="yt-timestamp" data-t="00:02:58">[00:02:58]</a>
*   Specifying artistic styles (e.g., "A banana riding a skateboard painted by Van Gogh") <a class="yt-timestamp" data-t="00:03:16">[00:03:16]</a>

Advanced users leverage community-created tools, such as Excel spreadsheets, to input detailed parameters like sub-ideas, exclusions, aspect ratios, camera types, lighting, and colors <a class="yt-timestamp" data-t="00:03:23">[00:03:23]</a>. This level of control allows for precise and sophisticated image generation.
