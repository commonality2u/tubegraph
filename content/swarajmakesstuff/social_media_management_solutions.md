## Introduction to Apertures

Apertures is presented as a social media management solution specifically designed for [[time_management_for_busy_professionals | busy professionals]] <a class="yt-timestamp" data-t="00:00:14">[00:00:14]</a>. It's described not just as a social media automation platform, but as a "secret weapon" for unlocking potential on social media <a class="yt-timestamp" data-t="00:00:21">[00:00:21]</a>. The first [[beta_launch_of_a_social_media_tool | beta version]] of Apertures has been launched, with a waitlist currently open <a class="yt-timestamp" data-t="00:00:30">[00:00:30]</a>.

Users can sign in or sign up using their email and password, or by connecting with Google <a class="yt-timestamp" data-t="00:00:47">[00:00:47]</a>. Upon signing up, users are welcomed and guided to connect their social accounts <a class="yt-timestamp" data-t="00:01:05">[00:01:05]</a>. Currently supported platforms include Twitter, LinkedIn, and Lens Protocol <a class="yt-timestamp" data-t="00:01:21">[00:01:21]</a>.

## Key Features

### Post Creation & Scheduling

The platform offers a dashboard where users can create new posts <a class="yt-timestamp" data-t="00:01:37">[00:01:37]</a>. When creating a post, users can select specific social media accounts, such as Twitter or LinkedIn <a class="yt-timestamp" data-t="00:01:55">[00:01:55]</a>.

Key functionalities for post creation include:
*   **Saving**: Posts can be saved as drafts <a class="yt-timestamp" data-t="00:01:57">[00:01:57]</a>. Drafts are viewable, editable, and deletable from a dedicated section <a class="yt-timestamp" data-t="00:02:44">[00:02:44]</a>.
*   **Publishing**: Posts can be published immediately to selected social media platforms <a class="yt-timestamp" data-t="00:01:57">[00:01:57]</a>. The platform shows confirmation of posts going to Twitter and LinkedIn <a class="yt-timestamp" data-t="00:05:21">[00:05:21]</a>.
*   **Scheduling**: Posts can be scheduled for future publication <a class="yt-timestamp" data-t="00:01:59">[00:01:59]</a>.
*   **Version Control**: Users can create different versions of a post for each selected social media platform <a class="yt-timestamp" data-t="00:02:01">[00:02:01]</a>. There is also a "sync to default" option to apply changes across all platforms <a class="yt-timestamp" data-t="00:02:31">[00:02:31]</a>.

### [[automated_social_media_posts_from_github_commits | GitHub Integration]] and [[the_role_of_ai_in_content_automation_for_social_media | AI Content Generation]]

A unique feature of Apertures is its integration with GitHub, allowing for [[automated_social_media_posts_from_github_commits | automated social media posts from GitHub commits]] <a class="yt-timestamp" data-t="00:03:03">[00:03:03]</a>.

*   Users connect their GitHub account and select a repository <a class="yt-timestamp" data-t="00:03:05">[00:03:05]</a>.
*   [[The role of AI in content automation for social media | AI]] can generate posts based on commit messages <a class="yt-timestamp" data-t="00:03:31">[00:03:31]</a>. Users can add context to help the [[the_role_of_ai_in_content_automation_for_social_media | AI]] better understand the project and generate more relevant posts <a class="yt-timestamp" data-t="00:03:29">[00:03:29]</a>.
*   Generated posts can be reviewed, saved, edited, and then published or scheduled like any other post <a class="yt-timestamp" data-t="00:04:12">[00:04:12]</a>.
*   A faster experience is promised on the web application compared to a local development environment <a class="yt-timestamp" data-t="00:04:24">[00:04:24]</a>.
