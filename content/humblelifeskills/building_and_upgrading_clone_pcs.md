## Specifications
The specifications for these clone PCs include:
*   **Processor Speed** Most are 3 gigahertz <a class="yt-timestamp" data-t="00:00:20">[00:00:20]</a>.
*   **Hard Drives** 120 gig hard drives <a class="yt-timestamp" data-t="00:00:24">[00:00:24]</a>.
*   **Motherboards** Asus motherboards, which the builder notes are "very reliable server boards" <a class="yt-timestamp" data-t="00:00:26">[00:00:26]</a>.
*   **Memory** Each PC has two gigabytes of memory <a class="yt-timestamp" data-t="00:00:30">[00:00:30]</a>.
*   **Graphics/Expansion Cards** Some units are equipped with Gigabyte cards <a class="yt-timestamp" data-t="00:00:33">[00:00:33]</a>.
