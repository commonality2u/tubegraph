The Rode NTG2 is highlighted as a specific type of [[compatibility_with_xlr_microphones | XLR microphone]] that can be used effectively with an iPhone through a custom cable <a class="yt-timestamp" data-t="00:00:53">[00:00:53]</a>. ## Key Features

*   **Powered Microphone**
    The Rode NTG2 is identified as a [[using_powered_microphones_with_iphones | powered microphone]] <a class="yt-timestamp" data-t="00:00:53">[00:00:53]</a>. This characteristic is crucial because not all [[compatibility_with_xlr_microphones | XLR microphones]] will work with the specific iPhone adapter; microphones must either be dynamic or [[using_powered_microphones_with_iphones | powered]] <a class="yt-timestamp" data-t="00:00:59">[00:00:59]</a>.

*   **Power Source**
    The microphone operates using a single 1.5 volt AA battery housed within the unit <a class="yt-timestamp" data-t="00:01:07">[00:01:07]</a>.

*   **Audio Quality**
    When paired with an iPhone, the Rode NTG2 is capable of producing broadcast-quality sound, making it suitable for mobile recording applications <a class="yt-timestamp" data-t="00:01:11">[00:01:11]</a>.

### Usage with iPhone

To use the Rode NTG2 with an iPhone, a custom XLR cable with a three-stripe connector for the iPhone is required <a class="yt-timestamp" data-t="00:00:17">[00:00:17]</a>. The connection sequence involves plugging the custom cable into the iPhone first, then attaching the XLR microphone to the cable <a class="yt-timestamp" data-t="00:00:28">[00:00:28]</a>. This setup allows for recording audio using applications such as Audio Boo or Fire the Road <a class="yt-timestamp" data-t="00:00:50">[00:00:50]</a>.
