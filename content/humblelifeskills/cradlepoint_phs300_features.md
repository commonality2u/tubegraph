The Cradlepoint PHS300 is a wireless hotspot router <a class="yt-timestamp" data-t="00:00:07">[00:00:07]</a>, designed to provide a portable broadband connection.

## Connectivity and Compatibility
The PHS300 features a USB connector <a class="yt-timestamp" data-t="00:00:12">[00:00:12]</a>, allowing it to connect to a [[usage_of_3g_dongle_in_cradlepoint | 3G dongle]] <a class="yt-timestamp" data-t="00:00:15">[00:00:15]</a>. It can also connect to an adapter for a PCMCIA card <a class="yt-timestamp" data-t="00:00:25">[00:00:25]</a>, supporting EVDO cards <a class="yt-timestamp" data-t="00:00:32">[00:00:32]</a> and HSDPA connectors <a class="yt-timestamp" data-t="00:00:43">[00:00:43]</a>. It's recommended to check the Cradlepoint website for an updated list of supported devices, as it is frequently updated <a class="yt-timestamp" data-t="00:00:34">[00:00:34]</a>.

## Technology and Power
The device incorporates "Wipipe technology" <a class="yt-timestamp" data-t="00:00:49">[00:00:49]</a>, functioning as a battery-operated version of a home broadband router <a class="yt-timestamp" data-t="00:00:54">[00:00:54]</a>.

### [[battery_and_power_options_for_cradlepoint | Battery and Power Options]]
The PHS300 has an internal battery that typically lasts 3 to 4 hours <a class="yt-timestamp" data-t="00:00:59">[00:00:59]</a>. For extended use, it can be connected to an external power unit, such as a [[tekkeon_3450i_lithium_polymer_battery_features | Tekkeon power unit]] <a class="yt-timestamp" data-t="00:01:04">[00:01:04]</a>, which can provide power for up to a day and a half <a class="yt-timestamp" data-t="00:01:08">[00:01:08]</a>. The DC input for the Tekkeon unit is 5 volts <a class="yt-timestamp" data-t="00:01:27">[00:01:27]</a>.

## Functionality
The PHS300 creates a personal wireless hotspot <a class="yt-timestamp" data-t="00:01:22">[00:01:22]</a>. Users can insert a dongle with an international unlimited SIM card to establish a wireless network <a class="yt-timestamp" data-t="00:01:14">[00:01:14]</a>. The hotspot can be password-protected <a class="yt-timestamp" data-t="00:01:23">[00:01:23]</a>.

### Web Interface and Features
The device includes a simple web interface for configuration <a class="yt-timestamp" data-t="00:01:32">[00:01:32]</a>. Users can set various parameters, including:
*   Wireless channel <a class="yt-timestamp" data-t="00:01:35">[00:01:35]</a>
*   Encryption type (e.g., 64-bit, 128-bit) <a class="yt-timestamp" data-t="00:01:37">[00:01:37]</a>

Firmware updates regularly introduce new features <a class="yt-timestamp" data-t="00:01:42">[00:01:42]</a>, such as GPS tracking, which allows the device to constantly broadcast its location <a class="yt-timestamp" data-t="00:01:46">[00:01:46]</a>.

## Project Involvement
Cradlepoint has sponsored the "Amo Box project" <a class="yt-timestamp" data-t="00:01:53">[00:01:53]</a>, providing ten PHS300 units for an upcoming project <a class="yt-timestamp" data-t="00:01:55">[00:01:55]</a>.
