The [[usb_turntable_overview_and_features | USB turntable]] described is designed for digital conversion of analogue records, and comes with specific software and operating system compatibilities <a class="yt-timestamp" data-t="01:19:54">[01:19:54]</a>.

## Operating System Compatibility

The [[usb_turntable_overview_and_features | USB turntable]] is compatible with both PC and Mac operating systems <a class="yt-timestamp" data-t="01:19:54">[01:19:54]</a>.

## Included Software

The [[usb_turntable_overview_and_features | USB turntable]] comes bundled with "Odyssey" <a class="yt-timestamp" data-t="01:33:64">[01:33:64]</a>. This program, likely Audacity given its common association with USB turntables and description, is a free wave program available for both Mac and PC <a class="yt-timestamp" data-t="01:33:64">[01:33:64]</a>.

## Software Features for Record Conversion

The accompanying software provides several key features for [[converting_78_rpm_records_to_digital_format | converting records to digital format]]:

*   **Speed Conversion** The software allows users to convert the speed of 78 RPM records by reducing it by a specified number of seconds <a class="yt-timestamp" data-t="01:51:00">[01:51:00]</a>.
*   **Audio Preview** Users can preview the audio coming through the record player directly through the software <a class="yt-timestamp" data-t="02:21:00">[02:21:00]</a>.
*   **Speed Adjustment** For issues like 33 RPM records sounding too slow, the software offers the ability to speed up the audio <a class="yt-timestamp" data-t="02:44:00">[02:44:00]</a>. It may require some trial and error to find the optimal setting for tweaking <a class="yt-timestamp" data-t="02:50:00">[02:50:00]</a>.
*   **External Source Recording** In addition to the turntable, the software can also record from other external audio sources connected via RCA phono and line inputs on the unit <a class="yt-timestamp" data-t="05:48:00">[05:48:00]</a>.
