## Lighting Setup

For improved video quality, using external lighting is highly recommended.
*   **Type of Lights**: Two [[Benefits of LED lighting for cameras | LED lights]] are used in the described setup <a class="yt-timestamp" data-t="00:01:21">[00:01:21]</a>.
*   **Positioning**: These lights are placed approximately half a meter away from the subject's face <a class="yt-timestamp" data-t="00:01:24">[00:01:24]</a>. They should be positioned at roughly eye height <a class="yt-timestamp" data-t="00:04:37">[00:04:37]</a>.
*   **Color Temperature**: The [[Benefits of LED lighting for cameras | LED lights]] are set to "daylight mode," which is 5600 Kelvin <a class="yt-timestamp" data-t="00:01:31">[00:01:31]</a>, <a class="yt-timestamp" data-t="00:01:33">[00:01:33]</a>. This specific Kelvin setting is crucial for coordinating with camera settings <a class="yt-timestamp" data-t="00:01:40">[00:01:40]</a>.

### Key Settings Adjustments

*   **Anti-Flicker**: Set to PAL 50Hz for Europe, or 60Hz for regions like the United States, based on local power and lighting frequencies <a class="yt-timestamp" data-t="00:01:48">[00:01:48]</a>, <a class="yt-timestamp" data-t="00:01:49">[00:01:49]</a>, <a class="yt-timestamp" data-t="00:01:50">[00:01:50]</a>, <a class="yt-timestamp" data-t="00:01:52">[00:01:52]</a>, <a class="yt-timestamp" data-t="00:01:54">[00:01:54]</a>.
*   **HDR**: While High Dynamic Range (HDR) can be enabled, its impact on the image might be minimal <a class="yt-timestamp" data-t="00:01:57">[00:01:57]</a>, <a class="yt-timestamp" data-t="00:01:59">[00:01:59]</a>, <a class="yt-timestamp" data-t="00:02:00">[00:02:00]</a>.
*   **Field of View**: A 78-degree field of view provides a closer shot, while 90 degrees offers a wider view <a class="yt-timestamp" data-t="00:02:04">[00:02:04]</a>, <a class="yt-timestamp" data-t="00:02:06">[00:02:06]</a>, <a class="yt-timestamp" data-t="00:02:08">[00:02:08]</a>, <a class="yt-timestamp" data-t="00:02:10">[00:02:10]</a>.
*   **Brightness**: Typically dialed down to 40 <a class="yt-timestamp" data-t="00:02:15">[00:02:15]</a>, <a class="yt-timestamp" data-t="00:02:17">[00:02:17]</a>. Keeping brightness too high can wash out the image <a class="yt-timestamp" data-t="00:04:04">[00:04:04]</a>, <a class="yt-timestamp" data-t="00:04:06">[00:04:06]</a>.
*   **Color Intensity**: Set to 38 <a class="yt-timestamp" data-t="00:02:18">[00:02:18]</a>.
*   **Contrast**: Set to 45 <a class="yt-timestamp" data-t="00:02:19">[00:02:19]</a>, <a class="yt-timestamp" data-t="00:02:21">[00:02:21]</a>.
*   [[Adjusting brightness contrast and white balance for webcams | **White Balance**]]: It is crucial to turn off [[Adjusting brightness contrast and white balance for webcams | auto white balance]] <a class="yt-timestamp" data-t="00:02:23">[00:02:23]</a>, <a class="yt-timestamp" data-t="00:02:24">[00:02:24]</a>. Instead, manually set the white balance to be as close to 5600 Kelvin as possible <a class="yt-timestamp" data-t="00:02:46">[00:02:46]</a>, <a class="yt-timestamp" data-t="00:02:48">[00:02:48]</a>. In the app, the closest achievable setting might be 5592 Kelvin <a class="yt-timestamp" data-t="00:02:53">[00:02:53]</a>, <a class="yt-timestamp" data-t="00:02:56">[00:02:56]</a>.
*   [[Manual focus and settings adjustment for webcams | **Autofocus**]]: Webcam tracking is not as advanced as DSLR cameras <a class="yt-timestamp" data-t="00:03:03">[00:03:03]</a>. It is recommended to use autofocus to get the initial focus, then turn it off before recording, especially if remaining relatively still <a class="yt-timestamp" data-t="00:03:16">[00:03:16]</a>, <a class="yt-timestamp" data-t="00:03:18">[00:03:18]</a>, <a class="yt-timestamp" data-t="00:03:21">[00:03:21]</a>, <a class="yt-timestamp" data-t="00:03:23">[00:03:23]</a>, <a class="yt-timestamp" data-t="00:03:25">[00:03:25]</a>.
