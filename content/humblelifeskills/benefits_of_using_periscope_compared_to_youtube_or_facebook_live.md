## Enhanced Desktop Capabilities

Periscope offers significant advantages when accessed from a desktop, providing a different way to stream than just from a mobile device <a class="yt-timestamp" data-t="00:00:32">[00:00:32]</a>.

Instead of being confined to a mobile smart device or phone, Periscope's desktop access allows for a richer production experience <a class="yt-timestamp" data-t="00:00:41">[00:00:41]</a>. When utilizing tools like [[setting_up_periscope_producer_with_wirecast | Periscope Producer]] with streaming software such as Wirecast or [[using_obs_software_for_periscope_producer | OBS Project]], users gain access to a wider array of features <a class="yt-timestamp" data-t="00:00:06">[00:00:06]</a>.

### Advanced Production Features

Through desktop access, Periscope enables various advanced production elements:
*   **Animation** <a class="yt-timestamp" data-t="00:00:37">[00:00:37]</a>
*   **Lower thirds** <a class="yt-timestamp" data-t="00:00:37">[00:00:37]</a>
*   **Playlists** <a class="yt-timestamp" data-t="00:00:39">[00:00:39]</a>
*   **Different inputs** <a class="yt-timestamp" data-t="00:00:39">[00:00:39]</a>

These capabilities allow for the creation of more dynamic and professional [[creating_regular_shows_with_periscope_from_the_desktop | regular shows]] that go beyond the limitations of mobile-only streaming <a class="yt-timestamp" data-t="00:00:30">[00:00:30]</a>.
