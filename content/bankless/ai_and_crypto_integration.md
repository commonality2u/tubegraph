## Emerging Trends

### 1. Defi AI (DeFi + AI)
Defi AI (or "DeFi") refers to the [[Integration of AI in cryptocurrency]] decentralized finance. This trend is categorized into three stages:
1.  **Abstract Away Complexity**: Creating "agent engines" that act as chat interfaces for crypto, similar to ChatGPT <a class="yt-timestamp" data-t="00:46:05">[00:46:05]</a>. This addresses the clunky and daunting user experience of DeFi by allowing natural language processing to execute complex on-chain actions (e.g., bridging coins between chains, buying tokens) <a class="yt-timestamp" data-t="00:46:49">[00:46:49]</a>.
    *   **Graffan**: An agent engine that offers a ChatGPT-like UI with a connected wallet. It features embedded "agent apps" for specific tasks, such as "flipper" agents that can execute trades (e.g., rumored to turn 1 SOL into 500 SOL) <a class="yt-timestamp" data-t="00:47:38">[00:47:38]</a>. It can also autonomously create and airdrop tokens to specific holders based on qualifications (e.g., NFT or token holdings) <a class="yt-timestamp" data-t="00:50:09">[00:50:09]</a>. This dramatically reduces the effort from multi-person teams to a single AI-assisted prompt <a class="yt-timestamp" data-t="00:50:50">[00:50:50]</a>.
    *   **The Hive Mind**: A project from a Solana hackathon that abstracts away the complexity of staking, allowing users to stake coins without manually moving assets or finding validators <a class="yt-timestamp" data-t="00:49:10">[00:49:10]</a>.
    *   **Wayfinder**: A platform that has been building since last year, offering an interface for complex DeFi actions. Uniquely, it allows any developer to design "paths" for agents (e.g., LP into a pool, manage LP, swap tokens) and earn a percentage of fees or profits generated when others use their paths <a class="yt-timestamp" data-t="00:54:30">[00:54:30]</a>. Wayfinder has also demonstrated an agent autonomously writing and deploying smart contracts (e.g., for token creation and staking) <a class="yt-timestamp" data-t="00:55:16">[00:55:16]</a>.
    *   AI-powered interfaces are expected to become deeply personalized, understanding user habits and preferences to predict and execute desired actions, much like Instagram or Twitter algorithms <a class="yt-timestamp" data-t="00:52:30">[00:52:30]</a>. This approach could also mitigate pervasive phishing attacks by abstracting transaction management and private keys from direct user interaction <a class="yt-timestamp" data-t="00:59:37">[00:59:37]</a>.

2.  **Enable Agents with DeFi Capabilities**: The next step involves plugging autonomous agents directly into these "agent engines" <a class="yt-timestamp" data-t="00:56:03">[00:56:03]</a>. Instead of a human typing prompts, an agent would command the interface, aligning its actions with its predefined mission (e.g., "find me an agent that could create me a token") <a class="yt-timestamp" data-t="00:56:32">[00:56:32]</a>. This concept aligns with "agent swarms" or "agentic entities" where an operator agent manages other specialized agents <a class="yt-timestamp" data-t="00:56:58">[00:56:58]</a>.

3.  **Enhancing Apps Natively (AI-Enabled Apps)**: Existing decentralized applications (dApps) can integrate AI to improve user experience, leverage agents for backend data pulling (replacing oracles), or autonomously manage treasury risk <a class="yt-timestamp" data-t="00:57:25">[00:57:25]</a>.

## Addressing FUD: "LLM Wrappers"
A common criticism (FUD) is that AI agents are merely "LLM wrappers" or "rebranded bots" <a class="yt-timestamp" data-t="01:19:02">[01:19:02]</a>. This view is considered naive because:
*   **Bots vs. Agents**: Bots are deterministic and hardcoded, following predefined paths <a class="yt-timestamp" data-t="01:19:57">[01:19:57]</a>. Agents, conversely, are probabilistic; they think, act, ingest data, and readapt, much like humans <a class="yt-timestamp" data-t="01:20:05">[01:20:05]</a>.
*   **Brain Analogy**: While an LLM serves as the "brain," just like all humans have brains, not all humans are the same <a class="yt-timestamp" data-t="01:20:54">[01:20:54]</a>. The challenge lies in building "agentic frameworks" that leverage LLMs to generate useful outputs and take meaningful actions, providing the "body" and "appendages" for the brain to interact with the world <a class="yt-timestamp" data-t="01:21:04">[01:21:04]</a>. The Eliza framework, for example, focuses on building the "body" around the LLM "brain" to give agents capabilities to connect to Twitter, Discord, Telegram, and the metaverse <a class="yt-timestamp" data-t="01:23:33">[01:23:33]</a>.

It is believed that many people cannot yet comprehend the profound impact AI will have, making current day-to-day work and life potentially "irrelevant" not in the sense of eliminating jobs, but by freeing up time for different, more enhanced activities <a class="yt-timestamp" data-t="01:23:36">[01:23:36]</a>.
