Sovereign is a platform designed for [[creating_and_editing_video_assets | creating and editing video assets]] and generating various video ad components <a class="yt-timestamp" data-t="00:00:06">[00:00:06]</a>. It supports [[utilizing_ai_tools_for_marketing_and_content_creation | utilizing AI tools for marketing and content creation]] throughout the video creation process.

## Key Features

### Bulk Create Hooks
This feature allows users to combine selected or uploaded visual hooks with multiple text hook variations <a class="yt-timestamp" data-t="00:00:09">[00:00:09]</a>. Users can upload their own visual hooks <a class="yt-timestamp" data-t="00:00:42">[00:00:42]</a> and add custom text hooks, choosing fonts, sizes, and even uploading unique fonts <a class="yt-timestamp" data-t="00:00:54">[00:00:54]</a>. The system can generate multiple new hooks for testing by pairing different visual and text elements <a class="yt-timestamp" data-t="00:01:31">[00:01:31]</a>. Users can also preview layouts for platforms like Instagram Stories or TikTok <a class="yt-timestamp" data-t="00:01:09">[00:01:09]</a>.

### Create Asset Feature
This tool functions as a video editor, enabling users to create individual video components such as hooks, body videos, or call-to-action (CTA) end cards <a class="yt-timestamp" data-t="00:02:00">[00:02:00]</a>. It allows users to select videos from an asset library <a class="yt-timestamp" data-t="00:02:14">[00:02:14]</a>, edit video clips, and combine multiple clips using a drag-and-drop interface <a class="yt-timestamp" data-t="00:02:40">[00:02:40]</a>. Text overlays can be added, with customizable colors, strokes, and a selection of Google Fonts or uploaded fonts <a class="yt-timestamp" data-t="00:03:10">[00:03:10]</a>. Once created, assets can be named and assigned to specific projects <a class="yt-timestamp" data-t="00:03:54">[00:03:54]</a>.

### [[ai_generated_video_content | AI Assets]] Feature
The platform integrates [[ai_generated_video_content | AI-generated video content]] capabilities <a class="yt-timestamp" data-t="00:04:15">[00:04:15]</a>:
*   **Image Generation**: Users can generate images from text prompts <a class="yt-timestamp" data-t="00:04:25">[00:04:25]</a>.
*   **Image to Video Conversion**: Generated images can be converted into short videos, such as 5-second clips <a class="yt-timestamp" data-t="00:05:00">[00:05:00]</a>. Users can also upload their own images for this process <a class="yt-timestamp" data-t="00:05:25">[00:05:25]</a>.
*   **AI Scripts**: The system can generate scripts based on the description of an image <a class="yt-timestamp" data-t="00:05:06">[00:05:06]</a>.
*   **[[ai_generated_visuals_in_video_advertising | AI Visual Hooks]] and B-roll**: This feature allows for the easy creation of [[ai_generated_visuals_in_video_advertising | AI visual hooks]] and [[ai_generated_video_content | AI b-roll]] footage, which can help in differentiating content from competitors <a class="yt-timestamp" data-t="00:05:51">[00:05:51]</a>.

### [[automatic_video_ad_creation | Automatic Video Ad Creation]]
The main page for creative remixing allows users to combine various assets to create new video ads <a class="yt-timestamp" data-t="00:06:17">[00:06:17]</a>. Users can select multiple hooks, a body video, call-to-action elements, and music tracks <a class="yt-timestamp" data-t="00:06:35">[00:06:35]</a>. The system can then merge these selected assets into numerous unique video iterations <a class="yt-timestamp" data-t="00:06:55">[00:06:55]</a>.

#### Customization and Export
*   **Export Ratios**: Videos can be exported in standard asset ratios including horizontal, square, vertical, and 9x16 vertical <a class="yt-timestamp" data-t="00:07:12">[00:07:12]</a>.
*   **Asset Adjustments**: Users can adjust the start and end times, as well as audio levels, for all assets within the project <a class="yt-timestamp" data-t="00:07:30">[00:07:30]</a>.
*   **Dynamic Naming Convention**: A crucial feature for performance marketers is the ability to customize the ad naming convention <a class="yt-timestamp" data-t="00:07:51">[00:07:51]</a>. This allows for dynamic pulling of hook, body, CTA, and audio names from file names, adding custom fields (e.g., creator, testing round), and reordering the naming sequence <a class="yt-timestamp" data-t="00:08:00">[00:08:00]</a>.
*   **Batch Generation and Download**: After configuration, the platform can generate numerous video ad combinations, complete with the specified naming conventions <a class="yt-timestamp" data-t="00:08:40">[00:08:40]</a>. These can be downloaded as a zip file via email, ready for upload to platforms like Facebook, TikTok, or Apple <a class="yt-timestamp" data-t="00:09:05">[00:09:05]</a>. This streamlines the [[video_ad_generation_process | video ad generation process]] and testing of [[using_aigenerated_content_in_video_ads | AI-generated content in video ads]] <a class="yt-timestamp" data-t="00:09:11">[00:09:11]</a>.
