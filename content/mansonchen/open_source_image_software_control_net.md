Control Net is an open-source software that functions as a text-to-image or image-to-image tool <a class="yt-timestamp" data-t="00:01:35">[00:01:35]</a><a class="yt-timestamp" data-t="00:01:43">[00:01:43]</a>. It is built on top of [[making_images_with_midjourney_dalle_3_or_stable_diffusion | Stable Diffusion]] <a class="yt-timestamp" data-t="00:01:47">[00:01:47]</a>.

With Control Net, users can generate images and videos <a class="yt-timestamp" data-t="00:01:57">[00:01:57]</a>. A key feature of Control Net is its ability to provide additional controls for elements like poses or compositions when creating text-to-image outputs <a class="yt-timestamp" data-t="00:01:59">[00:01:59]</a>.
