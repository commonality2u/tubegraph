Sovran is a platform designed to [[tools_for_automating_video_ad_creation | automate video ad creation]] and streamline the [[video_ad_creation_workflow | video ad creation workflow]] for performance marketers and creative strategists <a class="yt-timestamp" data-t="00:00:01">[00:00:01]</a>. It offers features for bulk content generation, asset creation, AI-powered media, and extensive creative remixing, enabling users to create and test numerous ad variations quickly <a class="yt-timestamp" data-t="00:06:05">[00:06:05]</a>.

## Key Features for [[Automatic video ad creation | Automatic Video Ad Creation]]

### Bulk Create Hooks
This feature allows users to combine multiple visual elements with various text variations to generate numerous unique hooks for video ads <a class="yt-timestamp" data-t="00:00:09">[00:00:09]</a>.
*   **Visual Hooks** Users can select from existing visual hooks or upload their own filmed content <a class="yt-timestamp" data-t="00:00:13">[00:00:13]</a>.
*   **Text Hook Variations** Multiple text hooks can be added and customized with different fonts, sizes, and positions <a class="yt-timestamp" data-t="00:00:48">[00:00:48]</a>. Users can also upload their own fonts <a class="yt-timestamp" data-t="00:01:02">[00:01:02]</a>.
*   **Combinatorial Creation** For example, combining three visual hooks with two text hooks generates six new hooks for testing <a class="yt-timestamp" data-t="00:01:31">[00:01:31]</a>.
*   **Export Options** Hooks can be previewed for formats like Instagram Stories or TikTok and exported with specific ratios such as highlight safe areas <a class="yt-timestamp" data-t="00:01:06">[00:01:06]</a>.

### Create Asset Editor
Sovran includes a video editor for creating individual components of an ad: hooks, body videos, or call-to-action (CTA) end cards <a class="yt-timestamp" data-t="00:02:00">[00:02:00]</a>.
*   **Video Editing** Users can edit existing video clips from the asset library <a class="yt-timestamp" data-t="00:02:14">[00:02:14]</a>.
*   **Clip Combination** Multiple video clips can be combined using a drag-and-drop interface <a class="yt-timestamp" data-t="00:02:48">[00:02:48]</a>.
*   **Text Overlays** Text can be added to videos with customizable fonts, colors, strokes, and positioning <a class="yt-timestamp" data-t="00:03:10">[00:03:10]</a>.
*   **Asset Naming** Assets can be named and categorized as a hook, body video, or call to action <a class="yt-timestamp" data-t="00:03:54">[00:03:54]</a>.

### AI Assets
Sovran integrates AI capabilities to generate visual content for ads <a class="yt-timestamp" data-t="00:04:15">[00:04:15]</a>.
*   **Image Generation** Users can generate images from text prompts <a class="yt-timestamp" data-t="00:04:25">[00:04:25]</a>.
*   **Image-to-Video Conversion** Generated or uploaded images can be converted into 5-second videos <a class="yt-timestamp" data-t="00:05:03">[00:05:03]</a>. This feature can be used to create [[using_aigenerated_content_in_video_ads | AI-generated visual hooks]] or b-roll footage <a class="yt-timestamp" data-t="00:05:51">[00:05:51]</a>.
*   **AI Scripts** The platform can generate scripts based on image descriptions <a class="yt-timestamp" data-t="00:05:06">[00:05:06]</a>.

### Creative Remixing for [[Automated ad creation and naming conventions with Sovereign | Automated Ad Creation]]
The platform's main page for "create video ads" allows for creative remixing, combining various assets to generate numerous ad variations <a class="yt-timestamp" data-t="00:06:17">[00:06:17]</a>. This is a core component of [[creating_video_ads_faster | creating video ads faster]] through a [[modular_approach_to_video_ad_creation | modular approach to video ad creation]].
*   **Asset Combination** Users can select multiple hooks, body videos, and call-to-action elements <a class="yt-timestamp" data-t="00:06:29">[00:06:29]</a>.
*   **Music Tracks** Different music tracks can be chosen to iterate on ad performance <a class="yt-timestamp" data-t="00:06:50">[00:06:50]</a>.
*   **Ad Variations** For example, selecting nine different text hooks, one body video, and three music tracks will generate 27 distinct ad iterations <a class="yt-timestamp" data-t="00:07:00">[00:07:00]</a>.
*   **Customization** Adjustments can be made to start and end times of assets and audio levels <a class="yt-timestamp" data-t="00:07:30">[00:07:30]</a>.
*   **Export Ratios** Ads can be exported in standard aspect ratios like horizontal, square, and vertical (9x16) <a class="yt-timestamp" data-t="00:07:15">[00:07:15]</a>.

### Customizable Naming Conventions
A critical feature for performance marketers is the ability to customize ad naming conventions <a class="yt-timestamp" data-t="00:07:51">[00:07:51]</a>.
*   **Dynamic Fields** Elements like hook name, body name, CTA name, and audio name are pulled dynamically from the file names <a class="yt-timestamp" data-t="00:08:00">[00:08:00]</a>.
*   **Customization** Users can add custom fields (e.g., Creator A, Testing Round Two), choose separators, and drag and drop the order of naming convention elements <a class="yt-timestamp" data-t="00:08:12">[00:08:12]</a>.

### Batch Generation and Download
After selecting assets and defining naming conventions, Sovran can generate all combinations <a class="yt-timestamp" data-t="00:08:52">[00:08:52]</a>.
*   **Fast Creation** The platform can generate many videos (e.g., 27) in approximately 5 minutes <a class="yt-timestamp" data-t="00:09:11">[00:09:11]</a>.
*   **Bulk Download** Users receive an email with a zip file containing all generated videos <a class="yt-timestamp" data-t="00:09:18">[00:09:18]</a>.
*   **Platform Integration** The downloaded videos can then be uploaded to platforms like [[automating_facebook_video_ads | Facebook]], TikTok, or Apple for testing <a class="yt-timestamp" data-t="00:09:24">[00:09:24]</a>.

Sovran aims to help users stand out from competitors by facilitating rapid creative testing and content generation <a class="yt-timestamp" data-t="00:06:00">[00:06:00]</a>.
