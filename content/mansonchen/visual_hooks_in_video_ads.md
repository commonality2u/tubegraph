## Script Frameworks for [[video_ads | Video Ads]]

Several script frameworks can be followed when creating [[video_ads | video ads]] <a class="yt-timestamp" data-t="00:00:08">[00:00:08]</a>. These include:
*   The Unique Mechanism script framework <a class="yt-timestamp" data-t="00:00:20">[00:00:20]</a>
*   The Pain/Solution script framework <a class="yt-timestamp" data-t="00:00:20">[00:00:20]</a>
*   The Education script framework <a class="yt-timestamp" data-t="00:00:23">[00:00:23]</a>

These frameworks share a similar structure and allow for the creation of many different [[video_ads | video ads]] <a class="yt-timestamp" data-t="00:00:26">[00:00:26]</a>. Using these frameworks, it's possible to A/B test various elements such as social proofs, calls to action, and benefits <a class="yt-timestamp" data-t="00:00:41">[00:00:41]</a>.
