## How to Generate a Single Purchase Order PDF

The process for [[generating_and_downloading_pdf_documents | generating and downloading PDF documents]] for individual purchase orders involves a few steps:
1.  **Login to PDF Output.com** Upon logging in, users will see an interface where they can select a template <a class="yt-timestamp" data-t="02:58:48">[02:58:48]</a>.
2.  **Select Template** From the dropdown menu on the left, choose the "purchase AO template" <a class="yt-timestamp" data-t="03:01:21">[03:01:21]</a>.
3.  **Choose Currency** Users can select their desired currency <a class="yt-timestamp" data-t="03:10:52">[03:10:52]</a>. If a specific currency is not available, it can be requested through the platform <a class="yt-timestamp" data-t="03:14:15">[03:14:15]</a>.
4.  **Single PDF Mode** Ensure the tool is set to "single PDF mode," which is designed for generating one document at a time <a class="yt-timestamp" data-t="03:22:56">[03:22:56]</a>.
5.  **Fill in Details** The template on screen will display fields that need to be populated with purchase order information, such as order date, expected delivery date, company details, item descriptions, quantities, and unit prices <a class="yt-timestamp" data-t="03:32:02">[03:32:02]</a>. This information can be copied from a database <a class="yt-timestamp" data-t="03:55:04">[03:55:04]</a>. The system automatically computes subtotals, shipping costs, and tax rates to provide a total amount <a class="yt-timestamp" data-t="04:02:18">[04:02:18]</a>.
6.  **Download PDF** Once all details are filled, click "download PDF" to generate the document <a class="yt-timestamp" data-t="04:14:14">[04:14:14]</a>. The generated PDF will be clean and precise, reflecting the chosen currency <a class="yt-timestamp" data-t="04:21:04">[04:21:04]</a>. The currency can be changed, and the document will update instantly <a class="yt-timestamp" data-t="04:27:54">[04:27:54]</a>.
