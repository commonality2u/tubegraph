[[Tracking personal finances in Notion | Notion]] can be effectively utilized as a monthly budget planner to analyze financial data <a class="yt-timestamp" data-t="00:00:07">[00:00:07]</a>. This system allows users to budget for each month and gain insights into their spending habits <a class="yt-timestamp" data-t="00:00:08">[00:00:08]</a>. The core of this system revolves around different views that provide a comprehensive [[creating_a_comprehensive_finance_dashboard_in_notion | finance dashboard]] <a class="yt-timestamp" data-t="00:00:11">[00:00:11]</a>.

## Key Views for Analysis

The monthly budget planner offers four primary views for [[managing_asset_financials_in_notion | managing asset financials]] and [[tracking_income_and_expenses_using_notion | tracking income and expenses]] <a class="yt-timestamp" data-t="00:00:11">[00:00:11]</a>:

### Expenses Overview
This view presents a breakdown of different expense categories, such as shopping, entertainment, food, and utilities <a class="yt-timestamp" data-t="00:00:22">[00:00:22]</a>. For each expense, it displays:
*   Total expense amount <a class="yt-timestamp" data-t="00:00:32">[00:00:32]</a>
*   Budgeted amount <a class="yt-timestamp" data-t="00:00:33">[00:00:33]</a>
*   Percentage of total budgeted amount incurred <a class="yt-timestamp" data-t="00:00:34">[00:00:34]</a>
*   A status indicating whether the budget has been exceeded or is under budget <a class="yt-timestamp" data-t="00:00:38">[00:00:38]</a>

### Monthly Overview
This view categorizes expenses by month, allowing users to see all expenses at a glance in a monthly format <a class="yt-timestamp" data-t="00:00:45">[00:00:45]</a>.

### Yearly Overview
Similar to the monthly overview, this view presents the exact same information but in a yearly format, providing a broader perspective <a class="yt-timestamp" data-t="00:00:57">[00:00:57]</a>.

### Summary Section
The summary section offers a complete expense overview in a yearly format, consolidating all expenses to give an overall performance assessment <a class="yt-timestamp" data-t="00:01:03">[00:01:03]</a>.

## Data Input: The Expense Details Database

To utilize this system for [[tracking_personal_finances using_notion | tracking personal finances using Notion]], all financial information is entered into the "expense details database" <a class="yt-timestamp" data-t="00:01:14">[00:01:14]</a>. This database automatically calculates and updates expenses based on the daily entries <a class="yt-timestamp" data-t="00:01:18">[00:01:18]</a>. Expenses are categorized by month and sorted in descending order (later months appear at the top) <a class="yt-timestamp" data-t="00:01:33">[00:01:33]</a>.

### Entering Transactions
To enter a transaction:
1.  Click "New" <a class="yt-timestamp" data-t="00:01:41">[00:01:41]</a>.
2.  Enter the date (e.g., February 1, 2024) <a class="yt-timestamp" data-t="00:01:47">[00:01:47]</a>. The month will automatically adjust based on the date <a class="yt-timestamp" data-t="00:01:55">[00:01:55]</a>.
3.  Input the expense detail (e.g., "Wi-Fi expense") <a class="yt-timestamp" data-t="00:02:05">[00:02:05]</a>.
4.  Select the period (e.g., "Feb 2024") <a class="yt-timestamp" data-t="00:02:13">[00:02:13]</a>. New periods can be added if not displayed <a class="yt-timestamp" data-t="00:02:20">[00:02:20]</a>.
5.  Select the year (e.g., "2024") <a class="yt-timestamp" data-t="00:02:25">[00:02:25]</a>.
6.  Choose the category of expense (e.g., "utilities") <a class="yt-timestamp" data-t="00:02:30">[00:02:30]</a>. Users can add custom categories <a class="yt-timestamp" data-t="00:02:35">[00:02:35]</a>.
7.  Add an expense code, used to segregate expenses by category and month (e.g., "utility Feb") <a class="yt-timestamp" data-t="00:02:44">[00:02:44]</a>.
8.  Enter the expense amount in the "expense amount" column <a class="yt-timestamp" data-t="00:03:00">[00:03:00]</a>.

### Setting Budgets
At the start of each month, users can [[how_to_set_financial_goals_in_notion | set financial goals]] by creating a budget for specific expenses <a class="yt-timestamp" data-t="00:03:22">[00:03:22]</a>.
1.  Select the date (e.g., February 1) <a class="yt-timestamp" data-t="00:03:35">[00:03:35]</a>.
2.  Enter the expense name for which the budget is being set (e.g., "food expense") <a class="yt-timestamp" data-t="00:03:39">[00:03:39]</a>.
3.  Select the period (e.g., "February") <a class="yt-timestamp" data-t="00:03:46">[00:03:46]</a> and year <a class="yt-timestamp" data-t="00:03:50">[00:03:50]</a>.
4.  Select the category (e.g., "food category") <a class="yt-timestamp" data-t="00:03:52">[00:03:52]</a>.
    *   For budget setup, the expense name and category name should be the same <a class="yt-timestamp" data-t="00:03:58">[00:03:58]</a>.
5.  Set the expense code (e.g., "food February") <a class="yt-timestamp" data-t="00:04:16">[00:04:16]</a>.
6.  Input the budget amount in the "budget amount" column <a class="yt-timestamp" data-t="00:04:18">[00:04:18]</a>.
Once entered, all figures in the overviews are automatically updated <a class="yt-timestamp" data-t="00:04:21">[00:04:21]</a>.

## Deep Dive into Financial Analysis

After booking expenses, the [[advanced_features_in_notion_for_financial_tracking | advanced features in Notion for financial tracking]] allow users to extract further insights by interacting with the database <a class="yt-timestamp" data-t="00:05:03">[00:05:03]</a>.

### Analyzing by Period
Clicking on a specific period (e.g., a month) opens a new window <a class="yt-timestamp" data-t="00:05:15">[00:05:15]</a>. This view displays:
*   All transactions booked for that period <a class="yt-timestamp" data-t="00:05:24">[00:05:24]</a>.
*   Details including whether the amount booked was a budget or an actual expense <a class="yt-timestamp" data-t="00:05:27">[00:05:27]</a>.

### Analyzing by Year
Similarly, clicking on a specific year shows a comprehensive view for the entire year <a class="yt-timestamp" data-t="00:05:39">[00:05:39]</a>, including:
*   Budgeted amount <a class="yt-timestamp" data-t="00:05:44">[00:05:44]</a>
*   Expenses amount <a class="yt-timestamp" data-t="00:05:45">[00:05:45]</a>
*   Expense as a percentage <a class="yt-timestamp" data-t="00:05:45">[00:05:45]</a>
*   Budget status <a class="yt-timestamp" data-t="00:05:47">[00:05:47]</a>
*   Transactions for the entire year, with amounts and indication of whether they were expenses or budgeted amounts <a class="yt-timestamp" data-t="00:05:50">[00:05:50]</a>.

### Analyzing by Category
By selecting a category, users can view how much expense has been booked for that specific category <a class="yt-timestamp" data-t="00:06:03">[00:06:03]</a>. This includes:
*   Expense amount <a class="yt-timestamp" data-t="00:06:08">[00:06:08]</a>
*   Budgeted amount <a class="yt-timestamp" data-t="00:06:09">[00:06:09]</a>
*   Expenses as a percentage <a class="yt-timestamp" data-t="00:06:11">[00:06:11]</a>
*   Status <a class="yt-timestamp" data-t="00:06:12">[00:06:12]</a>
*   All particular category-related expenses booked, with budgeted and expense amounts <a class="yt-timestamp" data-t="00:06:15">[00:06:15]</a>.

### Analyzing by Expense Code
The expense code allows users to see the amount booked for a particular category of expense and for a specific month, compared to the total budget <a class="yt-timestamp" data-t="00:06:31">[00:06:31]</a>. This helps in detailed [[using_databases_in_notion_for_finance_management | finance management using Notion databases]] <a class="yt-timestamp" data-t="00:06:21">[00:06:21]</a>.
