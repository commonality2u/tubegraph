## Generating Single PDF Documents

To [[generating_single_and_bulk_pdf_documents | generate a single PDF document]], users can log in to [[using_pdfoutputcom_for_bulk_pdf_generation | PDFOutput.com]] and select a [[using_templates_to_create_pdf_documents in_bulk | template]] from a dropdown menu <a class="yt-timestamp" data-t="03:22:50">[03:22:50]</a>. An example shown is the [[bulk_pdf_invoice_generation | invoice template]] <a class="yt-timestamp" data-t="03:25:70">[03:25:70]</a>. Users can also select their preferred currency from several available options <a class="yt-timestamp" data-t="03:30:17">[03:30:17]</a>.

Once a template and currency are selected, users fill in the necessary details related to the document <a class="yt-timestamp" data-t="03:36:78">[03:36:78]</a>. As details are filled, the PDF is populated <a class="yt-timestamp" data-t="03:41:00">[03:41:00]</a>. After inputting all information, clicking "download PDF" generates the document <a class="yt-timestamp" data-t="04:12:28">[04:12:28]</a>. The generated PDF will be precisely formatted with the entered details and chosen currency <a class="yt-timestamp" data-t="04:21:00">[04:21:00]</a>.

## [[utilizing_pdf_output_for_bulk_pdf_generation | Bulk PDF Generation]]

For [[utilizing_pdf_output_for_bulk_pdf_generation | generating PDF documents in bulk]], users can switch to the "bulk export mode" <a class="yt-timestamp" data-t="04:42:00">[04:42:00]</a>. In this mode, a database is available where users can fill in multiple entries for documents like invoices <a class="yt-timestamp" data-t="04:46:00">[04:46:00]</a>. This database includes fields such as invoice number, invoice date, and other details mirroring those found in the [[generating_single_and_bulk_pdf_documents | single PDF mode]] <a class="yt-timestamp" data-t="05:23:00">[05:23:00]</a>. Columns in the database can be expanded for better visibility <a class="yt-timestamp" data-t="05:29:00">[05:29:00]</a>.

Once all required information is entered into the database rows, users can click "download all PDF" to start [[utilizing_pdf_output_for_bulk_pdf_generation | generating PDFs one by one]] based on the provided data <a class="yt-timestamp" data-t="05:46:00">[05:46:00]</a>. The generated [[using_pdfoutput_for_bulk_document_generation | bulk PDFs]] maintain the same format as those created in [[generating_single_and_bulk_pdf_documents | single PDF mode]] and reflect the selected currency <a class="yt-timestamp" data-t="05:57:00">[05:57:00]</a>.
