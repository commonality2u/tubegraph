This guide demonstrates [[how_to_set_up_subscription_payment_reminders_on_whatsapp | how to receive subscription payment reminders]] directly on WhatsApp using a Notion database <a class="yt-timestamp" data-t="00:00:15">[00:00:15]</a>.

## Setup Process

Follow these steps to set up automated subscription payment reminders:

1.  **Create a New Template** <a class="yt-timestamp" data-t="00:00:23">[00:00:23]</a>
    *   Click on the drop-down menu in your database <a class="yt-timestamp" data-t="00:00:23">[00:00:23]</a>.
    *   Select the option to "create a new template" <a class="yt-timestamp" data-t="00:00:25">[00:00:25]</a>.
    *   Choose "setting up the subscription reminder alert" <a class="yt-timestamp" data-t="00:00:28">[00:00:28]</a>.

2.  **Specify Subscription Details** <a class="yt-timestamp" data-t="00:00:31">[00:00:31]</a>
    *   Enter the name of the subscription (e.g., "Spotify") <a class="yt-timestamp" data-t="00:00:34">[00:00:34]</a>.
    *   Specify the subscription amount (e.g., "$150") <a class="yt-timestamp" data-t="00:00:37">[00:00:37]</a>.
    *   Click "background" to add the new template <a class="yt-timestamp" data-t="00:00:41">[00:00:41]</a>. A new tab will be added <a class="yt-timestamp" data-t="00:00:45">[00:00:45]</a>.

3.  **Set Repetition Schedule** <a class="yt-timestamp" data-t="00:00:48">[00:00:48]</a>
    *   Click on the three dots associated with the newly created template <a class="yt-timestamp" data-t="00:00:49">[00:00:49]</a>.
    *   Click "repeat" and select your desired repetition mode <a class="yt-timestamp" data-t="00:00:51">[00:00:51]</a>. For a monthly subscription, choose "every month" <a class="yt-timestamp" data-t="00:00:56">[00:00:56]</a>.
    *   A new window will open to set the specific time and start date <a class="yt-timestamp" data-t="00:01:11">[00:01:11]</a>.
    *   Input the desired time (e.g., 11:56) <a class="yt-timestamp" data-t="00:01:18">[00:01:18]</a>.
    *   Confirm it repeats "every month" on a specific day (e.g., the 23rd) <a class="yt-timestamp" data-t="00:01:30">[00:01:30]</a>.

4.  **Save the Reminder** <a class="yt-timestamp" data-t="00:01:37">[00:01:37]</a>
    *   Click "Save" <a class="yt-timestamp" data-t="00:01:38">[00:01:38]</a>.
    *   A repetition arrow will appear, indicating the reminder is set to automatically repeat monthly (e.g., on the 23rd at 11:56) <a class="yt-timestamp" data-t="00:01:40">[00:01:40]</a>.

## Receiving Notifications

Once the reminder is set, a new line item will be added to your database at the scheduled time each month <a class="yt-timestamp" data-t="00:01:54">[00:01:54]</a>. Subsequently, you will automatically receive a notification alert on WhatsApp <a class="yt-timestamp" data-t="00:02:01">[00:02:01]</a>.

The WhatsApp notification will include:
*   A payment reminder for the specified amount (e.g., "$150") <a class="yt-timestamp" data-t="00:02:14">[00:02:14]</a>.
*   A "pay now" link to facilitate direct payment <a class="yt-timestamp" data-t="00:02:31">[00:02:31]</a>.

## Customization

The reminder alerts received on WhatsApp are fully [[customizing_reminder_alerts_in_whatsapp | customizable]] to your specific requirements <a class="yt-timestamp" data-t="00:03:00">[00:03:00]</a>. There is no fixed template <a class="yt-timestamp" data-t="00:03:07">[00:03:07]</a>. Instructions and a guide on setting up this automation are available upon downloading the template <a class="yt-timestamp" data-t="00:03:12">[00:03:12]</a>.
