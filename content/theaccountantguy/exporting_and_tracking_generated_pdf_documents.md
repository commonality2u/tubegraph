PDFoutput.com is a tool designed to [[generating_pdf_documents_for_businesses | generate PDF documents in bulk]] by utilizing Google Docs as a template and a Notion database for data [00:00:02].

## Accessing PDFoutput.com

To begin, navigate to PDFoutput.com and click "create PDF" [00:00:10]. You will be prompted to sign in through a Google account; select your account and click "continue" [00:00:15]. It is necessary to enable the checkbox to grant PDFoutput specific file access for functionality [00:00:22]. Once enabled, click "continue" to be redirected to the homepage [00:00:29].

## Generating PDF Documents in Bulk

The platform allows users to [[generating_pdf_documents_for_businesses | generate PDF documents]] by combining a Google Document template with data from a Notion database.

### Selecting a Google Document Template

Users can choose from pre-populated PDF templates [00:00:34] or select a raw document from their Google Drive [00:00:37]. To select a document, click the link next to the Google document option, which will display all documents listed on your Google Drive [00:00:42]. For example, an "invitation letter PDF output invitation letter" can be selected to send invitation letters, referencing fields like the user's first and last name [00:00:49].

### Setting up a Notion Database

1.  **Select Notion as the data source**: From the dropdown menu, select "Notion" [00:01:22].
2.  **Create a new Notion page and database**: Click the provided link to open the Notion workspace [00:01:28]. Create a new page, for instance, named "PDF output database" [00:01:39]. Inside this page, type `/database` and select "inline" to create a new database [00:01:47]. Name the database, for example, "database PDF output" [00:01:53].
3.  **Define database columns**: Rename the default columns to match your data needs, such as "first name" and "last name" [00:01:58].
4.  **Populate the database**: Add names like "Priya Sharma," "Miller Stark," and "John Quotes" as entries [00:02:11].
5.  **Copy Database URL**: Once the database view is set, click "share" and then "copy link" to obtain the database URL [00:02:42]. Paste this URL into the PDFoutput interface [00:02:47].
6.  **Create and connect API Key**:
    *   Click "add API key" in PDFoutput to create a new Notion API key [00:02:50].
    *   Name the key (e.g., "notion PDF output connection") and save it [00:02:57].
    *   Go back to the Notion API key section, click "show," and copy the generated key [00:03:12]. Paste this key into the PDFoutput interface [00:03:17].
    *   Ensure the Notion database is connected to this API key: In Notion, click the three dots, then "connections," and search for "PDF output connection" [00:03:26]. Confirm the connection [00:03:34].

### Mapping Properties and Defining Output

1.  **Load Properties**: In PDFoutput, click "load properties" to fetch all column properties from your Notion database [00:03:42].
2.  **Define Output File Name**: Select a property (e.g., "first name") from the dropdown to use as the output file name [00:03:51].
3.  **Choose Output Location**:
    *   **Google Drive**: Select "Google drive" and create a new folder (e.g., "invitation documents") in your Google Drive [00:04:01]. Then, click "add folder" in PDFoutput and select the newly created folder [00:04:26].
    *   **Downloads Folder**: Alternatively, choose the "downloads folder" option to save files directly to your desktop's download directory [00:04:35].
4.  **Map Template Fields**: Replace placeholders in your Google Doc template (e.g., "first name" and "last name") with the corresponding properties fetched from the Notion database [00:04:41]. Simply copy the property name from PDFoutput and paste it into the Google Doc [00:04:52].

### Initiating Export and Verification

Once all settings are configured, click [[exporting_and_downloading_generated_pdf_documents | export PDF]] [00:05:15]. The system will start [[generating_pdf_documents_for_businesses | generating the documents]] and display an increasing count of generated PDFs [00:05:17]. After completion, click "view in Google Drive" to open the output folder [00:05:48]. The generated PDFs will show the replaced data, such as "Jonty Roads," "Mirror Stark," and "Priya Sharma" [00:05:58]. The more information added to the Notion database, the more documents will be [[generating_pdf_documents_for_businesses | generated]] [00:06:21].
