## Generating AI Content in Notion Databases

[[generating_content_using_ai_prompts_for_notion_pages_and_databases | AI-generated content]] can be created within [[utilizing_notion_database_for_document_data | Notion databases]] using [[using_wizzygen_to_create_aigenerated_content_in_notion | Wizzygen]] <a class="yt-timestamp" data-t="00:00:21">[00:00:21]</a>. This process allows for content generation specific to each item or topic within the database's rows <a class="yt-timestamp" data-t="01:40:53">[01:40:53]</a>.

### Process for Database Content Generation

To generate AI content in a Notion database:
1.  **Connect Wizzygen**: Start by creating a connection within [[using_wizzygen_to_create_aigenerated_content_in_notion | Wizzygen]], naming it (e.g., "Notion database") <a class="yt-timestamp" data-t="01:21:20">[01:21:20]</a>.
2.  **Select Database Type**: Choose "Notion database" as the content destination <a class="yt-timestamp" data-t="01:25:00">[01:25:00]</a>.
3.  **Provide Database URL**: Copy the URL of the specific Notion database and paste it into Wizzygen <a class="yt-timestamp" data-t="01:27:00">[01:27:00]</a>.
4.  **Define AI Prompt**: Input a prompt to guide the AI's content generation. For example, to create a 100-word blog for each topic in the database, the prompt could be "write me a block of 100 words on the topic" <a class="yt-timestamp" data-t="01:48:19">[01:48:19]</a>. This ensures the AI generates content relevant to each row's specific data.

### Outcome

Once the process is initiated, [[using_wizzygen_to_create_aigenerated_content_in_notion | Wizzygen]] will populate the designated fields within the Notion database with the AI-generated content <a class="yt-timestamp" data-t="02:06:00">[02:06:00]</a>. The content will be generated for each row, based on the [[properties_in_notion_databases | properties]] or topics defined in that row <a class="yt-timestamp" data-t="02:17:00">[02:17:00]</a>. This allows for automated content creation across multiple entries in a structured format <a class="yt-timestamp" data-t="02:17:00">[02:17:00]</a>.

> [[ai_content_automation_in_notion_workspace | AI content automation]] in Notion using Wizzygen can streamline the creation of content for various topics stored in a database, such as blog posts for different subjects <a class="yt-timestamp" data-t="01:40:53">[01:40:53]</a>.
