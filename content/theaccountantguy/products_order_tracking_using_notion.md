A Notion template can be used as a simple product order tracker to help businesses keep track of all products ordered by customers <a class="yt-timestamp" data-t="00:00:03">[00:00:03]</a>. This template is entirely customizable to suit specific business needs <a class="yt-timestamp" data-t="00:01:16">[00:01:16]</a>.

## Database Structure

The template incorporates a database with key information for each order:

*   **Product Details**
    *   Product name <a class="yt-timestamp" data-t="00:00:12">[00:00:12]</a>
*   **Customer Information**
    *   Customer name <a class="yt-timestamp" data-t="00:00:15">[00:00:15]</a>
    *   Billing address <a class="yt-timestamp" data-t="00:00:17">[00:00:17]</a>
    *   Customer phone number <a class="yt-timestamp" data-t="00:00:18">[00:00:18]</a>
    *   Customer email <a class="yt-timestamp" data-t="00:00:20">[00:00:20]</a>
*   **Payment Information**
    *   Payment sources: How the customer intends to pay <a class="yt-timestamp" data-t="00:00:22">[00:00:22]</a>, <a class="yt-timestamp" data-t="00:00:24">[00:00:24]</a>
*   **Order Details**
    *   Gift item status: Indicates if the product is intended as a gift <a class="yt-timestamp" data-t="00:00:27">[00:00:27]</a>, <a class="yt-timestamp" data-t="00:00:29">[00:00:29]</a>
    *   Amount: The total value for the specific order placed <a class="yt-timestamp" data-t="00:00:31">[00:00:31]</a>, <a class="yt-timestamp" data-t="00:00:33">[00:00:33]</a>

The database can be expanded to include more items and details as needed <a class="yt-timestamp" data-t="00:01:32">[00:01:32]</a>, <a class="yt-timestamp" data-t="00:01:33">[00:01:33]</a>.

## Order Summary

The template includes an order summary at the top, providing an overview of order statistics <a class="yt-timestamp" data-t="00:00:34">[00:00:34]</a>.

Key summary metrics include:

*   Total order value of all placed orders <a class="yt-timestamp" data-t="00:00:36">[00:00:36]</a>, <a class="yt-timestamp" data-t="00:00:38">[00:00:38]</a>
*   Number of orders placed <a class="yt-timestamp" data-t="00:00:40">[00:00:40]</a>
*   Number of customers who have placed orders <a class="yt-timestamp" data-t="00:00:42">[00:00:42]</a>
*   Percentage of customers looking to place products as gift items <a class="yt-timestamp" data-t="00:00:44">[00:00:44]</a>, <a class="yt-timestamp" data-t="00:00:47">[00:00:47]</a>. This percentage, such as 67%, dynamically changes as values in the database are updated <a class="yt-timestamp" data-t="00:00:50">[00:00:50]</a>, <a class="yt-timestamp" data-t="00:01:00">[00:01:00]</a>.
*   Breakdown of payment sources, showing the number of customers and respective percentages for methods like debit card, PayPal, credit card, and other sources <a class="yt-timestamp" data-t="00:01:02">[00:01:02]</a>, <a class="yt-timestamp" data-t="00:01:13">[00:01:13]</a>.
