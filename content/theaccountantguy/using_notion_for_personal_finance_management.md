This article details a Notion-based system designed to [[tracking_personal_finances_in_notion | track personal finances]] <a class="yt-timestamp" data-t="00:00:10">[00:00:10]</a>. The tracker is divided into several key sections to provide a comprehensive financial overview <a class="yt-timestamp" data-t="00:00:40">[00:00:40]</a>.

## Overview of the Notion Tracker

The Notion personal finance tracker features three main sections <a class="yt-timestamp" data-t="00:00:16">[00:00:16]</a>:
1.  **Net Worth Target**: Allows users to specify a financial year net worth goal <a class="yt-timestamp" data-t="00:00:19">[00:00:19]</a>.
2.  **Account Section**: Where different accounts for income, expenses, assets, and liabilities are specified <a class="yt-timestamp" data-t="00:00:24">[00:00:24]</a>. This includes opening balances, payment frequency, and category for each <a class="yt-timestamp" data-t="00:00:31">[00:00:31]</a>.
3.  **Summary Section**: Provides an at-a-glance view of all finances <a class="yt-timestamp" data-t="00:00:40">[00:00:40]</a>, including:
    *   Finance view: Details income, expenses, assets, and liabilities, showing each account's contribution <a class="yt-timestamp" data-t="00:00:42">[00:00:42]</a>.
    *   Debt section: Tracks upcoming and past debt payments <a class="yt-timestamp" data-t="00:00:52">[00:00:52]</a>.
    *   [[budget_management_with_notion | Budget section]]: For setting estimated income and expenses <a class="yt-timestamp" data-t="00:01:08">[00:01:08]</a>.
    *   Savings section: To track personal savings <a class="yt-timestamp" data-t="00:01:16">[00:01:16]</a>.
    *   Bill section: Shows upcoming and paid bills <a class="yt-timestamp" data-t="00:01:23">[00:01:23]</a>.
    *   Subscription section: Tracks active subscriptions and payment dates <a class="yt-timestamp" data-t="00:01:32">[00:01:32]</a>.
    *   Net Worth section: Gives an overview of achieved net worth and progress towards the target <a class="yt-timestamp" data-t="00:01:45">[00:01:45]</a>.
4.  **Transaction Section**: Used for inputting all income, expense, asset, or liability transactions, which then affect all other sections <a class="yt-timestamp" data-t="00:01:51">[00:01:51]</a>.

## Getting Started: Setting Up Accounts

To begin [[using_notion_for_expense_management | tracking income and expenses]], navigate to the "Account" section under the "Get Started" view <a class="yt-timestamp" data-t="00:02:30">[00:02:30]</a>. Here, you can find and categorize your income (e.g., salary, side hustle) and expenses (e.g., groceries, utility bills, Netflix) <a class="yt-timestamp" data-t="00:02:40">[00:02:40]</a>.

### Adding and Modifying Accounts

*   **Edit Existing**: Click the "edit" button next to an account to modify it <a class="yt-timestamp" data-t="00:02:57">[00:02:57]</a>.
*   **Add New**: Click the "new" button to create a new account <a class="yt-timestamp" data-t="00:03:12">[00:03:12]</a>.
*   **Expense Classification**: Expenses can be classified (e.g., "bills," "subscriptions," "savings," "debt") <a class="yt-timestamp" data-t="00:03:31">[00:03:31]</a>.
*   **Payment Frequency**: Set a payment frequency for expenses (e.g., monthly, annually, quarterly, weekly) <a class="yt-timestamp" data-t="00:03:46">[00:03:46]</a>.
*   **Opening Balance**: For income and expense accounts, opening balance can be ignored <a class="yt-timestamp" data-t="00:04:16">[00:04:16]</a>.

## Recording Transactions

After setting up accounts, head to the "Transactions" view <a class="yt-timestamp" data-t="00:04:53">[00:04:53]</a>.
*   **Transaction Views**: There are five views: "All" (for all transaction types), and specific views for "Income," "Expenses," "Assets," and "Liabilities" <a class="yt-timestamp" data-t="00:05:15">[00:05:15]</a>.
*   **Adding a Transaction**:
    1.  Navigate to the relevant section (e.g., "Income") <a class="yt-timestamp" data-t="00:05:33">[00:05:33]</a>.
    2.  Click "new" <a class="yt-timestamp" data-t="00:05:38">[00:05:38]</a>.
    3.  Select the date <a class="yt-timestamp" data-t="00:05:44">[00:05:44]</a> (transactions are bifurcated by month and listed in descending order by date) <a class="yt-timestamp" data-t="00:05:52">[00:05:52]</a>.
    4.  Add a description (e.g., "Salary income") <a class="yt-timestamp" data-t="00:06:05">[00:06:05]</a>.
    5.  Select the account (e.g., "Salary") <a class="yt-timestamp" data-t="00:06:14">[00:06:14]</a>.
    6.  Choose payment mode (e.g., "Bank" or "Cash") <a class="yt-timestamp" data-t="00:06:28">[00:06:28]</a>.
    7.  Select transaction type: "Inflow" for money coming in (income, new loans) <a class="yt-timestamp" data-t="00:06:46">[00:06:46]</a>, "Outflow" for money going out (expenses, loan repayments) <a class="yt-timestamp" data-t="00:07:14">[00:07:14]</a>.
    8.  Specify the amount <a class="yt-timestamp" data-t="00:06:59">[00:06:59]</a>.

## Summary and Tracking

### [[using_notion_for_cash_flow_management | Tracking Income and Expenses]]

The "Finances" view under the "Summary" section displays total income and expenses, along with the percentage contribution of each account <a class="yt-timestamp" data-t="00:07:44">[00:07:44]</a>. This view updates dynamically as transaction figures are changed <a class="yt-timestamp" data-t="00:08:39">[00:08:39]</a>.

### Tracking Debt Payments

1.  **Set up Debt Accounts**: Navigate to the "Liabilities" part of the "Account" section to add or edit debt accounts <a class="yt-timestamp" data-t="00:09:10">[00:09:10]</a>. Classify them as "debt" and set a payment frequency <a class="yt-timestamp" data-t="00:09:41">[00:09:41]</a>.
2.  **Record Debt Transactions**: In the "Liabilities" section under "Transactions," enter details for debt payments (e.g., home loan repayment) <a class="yt-timestamp" data-t="00:10:31">[00:10:31]</a>.*   For repayments, the transaction type is "Outflow" <a class="yt-timestamp" data-t="00:11:30">[00:11:30]</a>.
*   For obtaining new loans, the transaction type is "Inflow" <a class="yt-timestamp" data-t="00:11:59">[00:11:59]</a>.
*   Opening balances for loans can be set in the "Account" section <a class="yt-timestamp" data-t="00:12:16">[00:12:16]</a>.
3.  **Monitor Debt Summary**: The "Debt" view under "Summary" shows "Upcoming Payments" (due in the future) and "Past Payments" (missed or older payments) <a class="yt-timestamp" data-t="00:13:17">[00:13:17]</a>. It shows the next payment due date, total debt, and debt paid so far <a class="yt-timestamp" data-t="00:01:01">[00:01:01]</a>. The view can be customized (e.g., list, gallery, board) <a class="yt-timestamp" data-t="00:15:30">[00:15:30]</a>.

### [[budget_management_with_notion | Budgeting]]

The "Budget" view under "Summary" outlines estimated income and expenses <a class="yt-timestamp" data-t="00:17:16">[00:17:16]</a>.
*   It reflects total incurred expenses and income from transactions <a class="yt-timestamp" data-t="00:17:22">[00:17:22]</a>.
*   Users can set and modify budget amounts for each expense or income category <a class="yt-timestamp" data-t="00:17:39">[00:17:39]</a>.
*   The budget view shows the percentage of actual spend/earn against the budgeted amount <a class="yt-timestamp" data-t="00:17:27">[00:17:27]</a>.

### Managing Savings Funds

1.  **Set up Savings Accounts**: In the "Account" section, under "Assets," create new accounts for savings funds (e.g., "Education Fund," "Car Fund") and classify them as "savings" <a class="yt-timestamp" data-t="00:19:11">[00:19:11]</a>.
2.  **Set Goal Amounts**: For each savings fund, a goal amount can be set <a class="yt-timestamp" data-t="00:20:16">[00:20:16]</a>.
3.  **Record Savings Transactions**: Navigate to the "Asset" section under "Transactions" to add contributions to savings funds <a class="yt-timestamp" data-t="00:23:05">[00:23:05]</a>. These are typically "Outflow" transactions from your bank account to the savings account <a class="yt-timestamp" data-t="00:23:42">[00:23:42]</a>.
4.  **Track Savings Progress**: The "Savings" view under "Summary" shows the goal amount, total saved amount, and the percentage achieved for each fund <a class="yt-timestamp" data-t="00:20:15">[00:20:15]</a>.

### Tracking Bills

1.  **Set up Bill Accounts**: In the "Account" section, under "Expenses," add accounts for bills (e.g., "Utility Bills," "Groceries," "Wi-Fi Bills") <a class="yt-timestamp" data-t="00:25:08">[00:25:08]</a>. Classify them as "bills" and set a monthly or weekly payment frequency <a class="yt-timestamp" data-t="00:25:16">[00:25:16]</a>.
2.  **Record Bill Transactions**: In the "Expense" section under "Transactions," enter bill payment details <a class="yt-timestamp" data-t="00:25:58">[00:25:58]</a>. These are "Outflow" transactions <a class="yt-timestamp" data-t="00:26:16">[00:26:16]</a>.
3.  **Monitor Bill Summary**: The "Bills" view under "Summary" shows "Upcoming Payments" and "Past Payments," similar to the debt section <a class="yt-timestamp" data-t="00:26:24">[00:26:24]</a>. It indicates the next expected payment date based on the last payment and frequency <a class="yt-timestamp" data-t="00:26:50">[00:26:50]</a>.

### Managing Subscriptions

1.  **Set up Subscription Accounts**: In the "Account" section, under "Expenses," add accounts for subscriptions (e.g., "Netflix," "Amazon") <a class="yt-timestamp" data-t="00:29:11">[00:29:11]</a>. Classify them as "subscriptions" and set payment frequency (e.g., annual, quarterly) <a class="yt-timestamp" data-t="00:29:21">[00:29:21]</a>.
2.  **Record Subscription Transactions**: In the "Expense" section under "Transactions," enter subscription payments <a class="yt-timestamp" data-t="00:30:06">[00:30:06]</a>. These are "Outflow" transactions <a class="yt-timestamp" data-t="00:30:47">[00:30:47]</a>.
3.  **Monitor Subscription Summary**: The "Subscription" view under "Summary" shows "Upcoming Payments" and "Past Payments," detailing the next due date based on payment frequency <a class="yt-timestamp" data-t="00:31:04">[00:31:04]</a>. It also displays the subscription status <a class="yt-timestamp" data-t="00:32:20">[00:32:20]</a>.

### [[using_notion_for_yearly_financial_overview | Calculating Net Worth]]

Net worth is the difference between assets and liabilities <a class="yt-timestamp" data-t="00:32:57">[00:32:57]</a>.
1.  **Set Net Worth Goal**: In the "Net Worth Goal" section, specify your target net worth for the financial year <a class="yt-timestamp" data-t="00:33:05">[00:33:05]</a>.
2.  **View Net Worth Summary**: The "Net Worth" section under "Summary" displays:
    *   Your Net Worth Goal <a class="yt-timestamp" data-t="00:33:32">[00:33:32]</a>.
    *   Total Net Worth achieved (Assets - Liabilities) <a class="yt-timestamp" data-t="00:33:36">[00:33:36]</a>.
    *   Difference (how much you are lagging behind) <a class="yt-timestamp" data-t="00:33:57">[00:33:57]</a>.
    *   Percentage of Net Worth Goal achieved <a class="yt-timestamp" data-t="00:34:09">[00:34:09]</a>.
    *   Total Growth (Closing Net Worth - Opening Net Worth) <a class="yt-timestamp" data-t="00:34:20">[00:34:20]</a>. Opening Net Worth is derived from the opening balances of assets and liabilities in the Account section <a class="yt-timestamp" data-t="00:34:28">[00:34:28]</a>.

### Tracking Investments

1.  **Set up Investment Accounts**: In the "Account" tab, under "Assets," add investment accounts (e.g., "Apple Investment," "Bitcoin Investment") <a class="yt-timestamp" data-t="00:38:18">[00:38:18]</a>.
    *   Specify an opening balance and opening units if applicable <a class="yt-timestamp" data-t="00:38:34">[00:38:34]</a>.
    *   Classify investments (e.g., "Stocks," "Crypto") <a class="yt-timestamp" data-t="00:38:45">[00:38:45]</a>.
    *   Crucially, set the "Asset Class" to "Investments" <a class="yt-timestamp" data-t="00:39:22">[00:39:22]</a>.
2.  **Record Investment Transactions**: In the "Asset" section under "Transactions," record purchase and sale of investments <a class="yt-timestamp" data-t="00:40:36">[00:40:36]</a>.
    *   **Purchases**: Always record at the *cost price* <a class="yt-timestamp" data-t="00:41:14">[00:41:14]</a>.
    *   **Sales**: Always record at the *cost value of the units being sold*, not the sale price <a class="yt-timestamp" data-t="00:42:12">[00:42:12]</a>. This enables the calculation of profit or loss on the transaction <a class="yt-timestamp" data-t="00:42:19">[00:42:19]</a>.
3.  **View Investment Performance**: The "Investment" tab in the summary provides details on total investment amounts, units available, and calculated profit or loss for each investment <a class="yt-timestamp" data-t="00:37:08">[00:37:08]</a>. It also shows the percentage contribution of each investment to the total <a class="yt-timestamp" data-t="00:37:46">[00:37:46]</a>.

## Additional Features and Tips

### Couples Tracking Finances

For couples, the Notion tracker can be adapted by splitting different account categories under each head (income, expenses, assets, liabilities) by individual names (e.g., "Salary by Him," "Side Hustle by Her") <a class="yt-timestamp" data-t="00:35:39">[00:35:39]</a>.

### Viewing Specific Transactions

To quickly view all transactions related to a particular account (e.g., "Salary" income):
1.  Go to the "Finances" view under "Summary" <a class="yt-timestamp" data-t="00:45:21">[00:45:21]</a>.
2.  Click on the specific income/expense account you wish to view (e.g., "Salary") <a class="yt-timestamp" data-t="00:45:50">[00:45:50]</a>.
3.  A side panel will open; scroll down and click "View Transactions" at the bottom <a class="yt-timestamp" data-t="00:46:22">[00:46:22]</a>. This will filter and display only transactions related to that specific account <a class="yt-timestamp" data-t="00:46:44">[00:46:44]</a>.

### Do I need to pay anything extra for Notion?

### Is the template available on multiple currencies?
Yes, Notion supports a wide range of currencies <a class="yt-timestamp" data-t="00:48:45">[00:48:45]</a>. To change the currency:
1.  Look for any property with a dollar sign ($) associated with it (e.g., in the "Transactions" or "Net Worth" views) <a class="yt-timestamp" data-t="00:49:01">[00:49:01]</a>.
2.  Click on the property, then "Edit Property," and go to "Number Format" <a class="yt-timestamp" data-t="00:49:08">[00:49:08]</a>.
3.  Select your desired currency (e.g., Euro) from the options <a class="yt-timestamp" data-t="00:49:12">[00:49:12]</a>. This must be done for all relevant views <a class="yt-timestamp" data-t="00:49:26">[00:49:26]</a>.
