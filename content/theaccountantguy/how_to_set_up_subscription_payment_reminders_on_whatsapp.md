## Step-by-Step Setup

### 1. Accessing the Database
The process begins within a database designed for managing reminders <a class="yt-timestamp" data-t="00:00:15">[00:00:15]</a>.

### 2. Creating a New Reminder Template
To create a new reminder:
*   Click on the drop-down menu within the database <a class="yt-timestamp" data-t="00:00:23">[00:00:23]</a>.
*   Select "create a new template" <a class="yt-timestamp" data-t="00:00:25">[00:00:25]</a>.
*   Choose the option for "setting up the subscription reminder alert" <a class="yt-timestamp" data-t="00:00:28">[00:00:28]</a>.

### 3. Specifying Subscription Details
Once the template is selected:
*   Enter the name of the subscription, for example, "Spotify" <a class="yt-timestamp" data-t="00:00:32">[00:00:32]</a>, <a class="yt-timestamp" data-t="00:00:34">[00:00:34]</a>.
*   Specify the subscription amount, such as "$150" <a class="yt-timestamp" data-t="00:00:37">[00:00:37]</a>.
*   Click on "background" to add the new template <a class="yt-timestamp" data-t="00:00:41">[00:00:41]</a>.
    *   A new tab for the subscription will appear <a class="yt-timestamp" data-t="00:00:43">[00:00:43]</a>, <a class="yt-timestamp" data-t="00:00:45">[00:00:45]</a>.

### 4. [[recurring_bill_payment_reminders | Setting Repetition for the Reminder]]
To set the reminder to recur:
*   Click on the three dots next to the newly added subscription <a class="yt-timestamp" data-t="00:00:48">[00:00:48]</a>, <a class="yt-timestamp" data-t="00:00:49">[00:00:49]</a>.
*   Click "repeat" <a class="yt-timestamp" data-t="00:00:50">[00:00:50]</a>.
*   Select the desired repetition mode, such as "every month" <a class="yt-timestamp" data-t="00:00:52">[00:00:52]</a>, <a class="yt-timestamp" data-t="00:00:56">[00:00:56]</a>. This ensures the subscription amount repeats each month <a class="yt-timestamp" data-t="00:00:59">[00:00:59]</a>.

### 5. Setting Time and Start Date
A new window will open to specify details:
*   Set the exact time for the reminder, e.g., 11:56 <a class="yt-timestamp" data-t="00:01:13">[00:01:13]</a>, <a class="yt-timestamp" data-t="00:01:20">[00:01:20]</a>.
*   Choose the specific day of the month for the reminder to repeat, e.g., the 23rd <a class="yt-timestamp" data-t="00:01:30">[00:01:30]</a>, <a class="yt-timestamp" data-t="00:01:32">[00:01:32]</a>.
*   Click "Save" <a class="yt-timestamp" data-t="00:01:38">[00:01:38]</a>.
    *   A repetition arrow will appear next to the subscription, indicating it will automatically repeat monthly on the specified date and time <a class="yt-timestamp" data-t="00:01:40">[00:01:40]</a>, <a class="yt-timestamp" data-t="00:01:44">[00:01:44]</a>, <a class="yt-timestamp" data-t="00:01:46">[00:01:46]</a>.

## How it Works
At the set time and date, a new line item will automatically be added to the database <a class="yt-timestamp" data-t="00:01:54">[00:01:54]</a>. Subsequently, a notification alert will be received on WhatsApp <a class="yt-timestamp" data-t="00:02:01">[00:02:01]</a>.

## Receiving WhatsApp Notifications
When the reminder triggers:
*   You will receive a payment reminder notification on WhatsApp for the specified amount (e.g., $150 for Spotify) <a class="yt-timestamp" data-t="00:02:11">[00:02:11]</a>, <a class="yt-timestamp" data-t="00:02:14">[00:02:14]</a>, <a class="yt-timestamp" data-t="00:02:22">[00:02:22]</a>.
*   The message includes a "Pay now" link, allowing direct payment through the provided link <a class="yt-timestamp" data-t="00:02:31">[00:02:31]</a>, <a class="yt-timestamp" data-t="00:02:34">[00:02:34]</a>.
