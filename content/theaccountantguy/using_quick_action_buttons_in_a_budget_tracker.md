A budget tracker template can include [[using_quick_action_buttons_for_managing_transactions | quick action buttons]] to streamline the process of managing your finances <a class="yt-timestamp" data-t="01:18:08">[01:18:08]</a>. These buttons are designed to help you quickly add transactions or new expense sources <a class="yt-timestamp" data-t="01:19:07">[01:19:07]</a>.

## Add Transaction Button

One [[using_quick_action_buttons_for_managing_transactions | quick action button]] is "Add transaction" <a class="yt-timestamp" data-t="01:21:00">[01:21:00]</a>. When clicked, this button automatically adds a new transaction entry with the current date <a class="yt-timestamp" data-t="01:22:04">[01:22:04]</a>. You can then fill in the specific details for that transaction <a class="yt-timestamp" data-t="01:25:07">[01:25:07]</a>. This feature aids in [[entering_and_tracking_transactions_in_notion_budget_planner | entering and tracking transactions]] efficiently.

## Add Expense Source Button

Another [[using_quick_action_buttons_for_managing_transactions | quick action button]] is "Add expense source" <a class="yt-timestamp" data-t="01:30:00">[01:30:00]</a>. Clicking this button automatically adds a new expense source, typically under the "Fixed expenses" category by default <a class="yt-timestamp" data-t="01:32:00">[01:32:00]</a>. After creation, you have the flexibility to move this new expense source to any other relevant category (e.g., variable expense, savings and investments) by clicking on the three dots icon next to it <a class="yt-timestamp" data-t="01:40:00">[01:40:00]</a>. This functionality supports the [[customizing_budget_tracking_templates | customization of budget tracking templates]].

Utilizing these [[using_quick_action_buttons_for_managing_transactions | quick action buttons]] helps users stay on top of their finances and manage budgeted payments effectively <a class="yt-timestamp" data-t="01:46:00">[01:46:00]</a>.
