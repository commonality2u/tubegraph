## Accessing and Using PDF Output.com

To use [[using_pdf_output_com_for_document_generation | PDF Output.com]], navigate to the website (PDFoutput.com) <a class="yt-timestamp" data-t="03:11:00">[03:11:00]</a> and log in <a class="yt-timestamp" data-t="03:13:00">[03:13:00]</a>.

### Template Selection

Once logged in, a drop-down menu on the top left displays various templates <a class="yt-timestamp" data-t="03:23:00">[03:23:00]</a>. To find the invoice template, search for "inv" <a class="yt-timestamp" data-t="03:28:00">[03:28:00]</a>. New templates are added regularly <a class="yt-timestamp" data-t="03:26:00">[03:26:00]</a>.

### Filling in Details

After selecting a template, fill in the necessary details one by one <a class="yt-timestamp" data-t="03:33:00">[03:33:00]</a>. For an invoice, this includes the invoice number, company name, address, and item specifics like description, quantity, and unit price <a class="yt-timestamp" data-t="03:46:00">[03:46:00]</a>.

### Currency Adjustment

The desired currency can be set, and changing it will update the entire invoice accordingly <a class="yt-timestamp" data-t="03:55:00">[03:55:00]</a>.

### [[using_pdf_output_to_create_invoices | Generating Single PDF Invoices]]

Once all details are entered, click the "Download PDF" button to generate and download the invoice <a class="yt-timestamp" data-t="04:16:00">[04:16:00]</a>. A "PDF downloaded successfully" message will appear upon completion <a class="yt-timestamp" data-t="04:20:00">[04:20:00]</a>.

## [[generating_pdf_documents_in_bulk | Generating PDF Documents in Bulk]]

Currently, direct Notion database support is not available, but details can be copied and pasted into the rows <a class="yt-timestamp" data-t="04:41:00">[04:41:00]</a>. Column widths can be expanded as needed <a class="yt-timestamp" data-t="04:50:00">[04:50:00]</a>. New rows can be added by clicking "add new row" <a class="yt-timestamp" data-t="05:04:00">[05:04:00]</a>.

### Initiating Bulk Download

After filling in all the required details for multiple invoices, click "Download all PDF" to start generating each PDF one by one <a class="yt-timestamp" data-t="04:54:00">[04:54:00]</a>, <a class="yt-timestamp" data-t="05:10:00">[05:10:00]</a>.

### Generating Individual PDFs from Bulk Mode
