## Notion Template Overview

This particular [[tracking_expenses_with_notion | Notion template]] is designed to manage restaurant business expenses <a class="yt-timestamp" data-t="00:00:05">[00:00:05]</a> <a class="yt-timestamp" data-t="00:01:11">[00:01:11]</a>. It organizes expenses into various categories, providing a clear overview <a class="yt-timestamp" data-t="00:00:13">[00:00:13]</a>.

### Key Features

*   **Expense Categories**: The template displays three main categories of expenses at the top <a class="yt-timestamp" data-t="00:00:13">[00:00:13]</a>.
*   **Budgeted vs. Actual Expenses**: For each category, you can see a budgeted expense and the actual expenses incurred <a class="yt-timestamp" data-t="00:00:17">[00:00:17]</a>. The actual expenses are dynamically pulled from a table located at the bottom of the template <a class="yt-timestamp" data-t="00:00:21">[00:00:21]</a>.
    *   **Defining Budgets**: To define a new budgeted expense, simply click on the existing value <a class="yt-timestamp" data-t="00:00:24">[00:00:24]</a>.
    *   **Percentage Indicator**: A percentage indicator shows the proportion of actual expenses incurred compared to the budgeted expense <a class="yt-timestamp" data-t="00:00:28">[00:00:28]</a>.
*   **Managing Categories**:
    *   **Modifying Categories**: To modify an existing category, click the pencil icon next to it, and the changes will automatically update <a class="yt-timestamp" data-t="00:00:35">[00:00:35]</a>. This helps in [[categorizing_and_managing_travel_expenses | categorizing and managing]] various types of expenses.
    *   **[[adding_and_managing_expenses | Adding New Categories]]**: A "new page" option allows you to add new expense categories <a class="yt-timestamp" data-t="00:00:40">[00:00:40]</a>.
    *   **Deleting Categories**: Categories can be deleted by clicking the three-dot icon and selecting "delete" <a class="yt-timestamp" data-t="00:00:43">[00:00:43]</a>.
*   **Dynamic Updates**: All expense values are updated dynamically as figures are inputted into the system <a class="yt-timestamp" data-t="00:00:51">[00:00:51]</a> <a class="yt-timestamp" data-t="00:00:56">[00:00:56]</a>.
*   **Summary Section**: The template includes a summary section that provides a comprehensive overview <a class="yt-timestamp" data-t="00:00:57">[00:00:57]</a>:
    *   Total budgeted expense <a class="yt-timestamp" data-t="00:00:59">[00:00:59]</a>
    *   Total actual expenses incurred <a class="yt-timestamp" data-t="00:01:00">[00:01:00]</a>
    *   The difference between budgeted and actual expenses <a class="yt-timestamp" data-t="00:01:04">[00:01:04]</a>
    *   The overall percentage of actual expenses incurred so far <a class="yt-timestamp" data-t="00:01:06">[00:01:06]</a>

This template offers a quick and easy way for [[budgeting_and_expense_tracking | budgeting and expense tracking]] in restaurant operations <a class="yt-timestamp" data-t="00:01:09">[00:01:09]</a>.
