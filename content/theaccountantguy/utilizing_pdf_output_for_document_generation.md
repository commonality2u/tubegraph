The [[PDF Output tool for document generation|PDF Output tool for document generation]] is a product designed to help businesses [[automating_pdf_document_generation|automate PDF document generation]], such as sales receipts <a class="yt-timestamp" data-t="02:53:00">[02:53:00]</a>. This tool, accessible via [[using_pdf_output_com_for_pdf_generation|PDFoutput.com]], allows users to select templates and fill in information to generate various PDF documents <a class="yt-timestamp" data-t="03:07:90">[03:07:90]</a>.

## How to Use PDF Output for Document Generation

To generate PDF documents, navigate to `PDFoutput.com` <a class="yt-timestamp" data-t="03:08:90">[03:08:90]</a>.

### Template Selection

Upon arriving at the website, locate the "template selection" dropdown at the top <a class="yt-timestamp" data-t="03:13:98">[03:13:98]</a>. This dropdown contains a list of available templates <a class="yt-timestamp" data-t="03:16:00">[03:16:00]</a>. Users can search for a specific template, such as "sales receipts," by clicking on it to open the corresponding template <a class="yt-timestamp" data-t="03:19:00">[03:19:00]</a>.

### Single PDF Document Generation

To generate a single PDF document:
1.  **Fill in Information:** Populate the template with the necessary details, including:
    *   Receipt number <a class="yt-timestamp" data-t="03:48:98">[03:48:98]</a>
    *   Receipt date <a class="yt-timestamp" data-t="03:50:00">[03:50:00]</a>
    *   Payment method (selectable from a dropdown) <a class="yt-timestamp" data-t="03:50:50">[03:50:50]</a>
    *   Business name and address <a class="yt-timestamp" data-t="03:53:50">[03:53:50]</a>
    *   Item description, quantity, and unit price (multiple items can be added) <a class="yt-timestamp" data-t="03:57:00">[03:57:00]</a>
    *   Tax rate <a class="yt-timestamp" data-t="04:26:00">[04:26:00]</a>
    *   Notes <a class="yt-timestamp" data-t="04:35:98">[04:35:98]</a>
2.  **Select Currency:** Choose the desired currency (e.g., US Dollars, Euro), which automatically updates the template <a class="yt-timestamp" data-t="04:40:50">[04:40:50]</a>.
3.  **Download PDF:** Click the "Download PDF" button to generate and download the document <a class="yt-timestamp" data-t="04:53:57">[04:53:57]</a>. The generated PDF will reflect the entered values <a class="yt-timestamp" data-t="05:00:00">[05:00:00]</a>.

### [[Bulk export of PDF documents using PDF Output|Bulk Export of PDF Documents]]

For [[bulk_export_of_pdf_documents_using_pdf_output|bulk export of PDF documents using PDF Output]]:
1.  **Select Bulk Export Mode:** Choose the "bulk export mode document" option <a class="yt-timestamp" data-t="05:08:98">[05:08:98]</a>.
2.  **Input Data in Database Format:** Enter all required sales receipt details into the provided database format <a class="yt-timestamp" data-t="05:14:98">[05:14:98]</a>.
    *   Users can resize column widths and row heights for better readability <a class="yt-timestamp" data-t="05:41:88">[05:41:88]</a>.
    *   New rows can be added using the "add new row" button <a class="yt-timestamp" data-t="05:58:00">[05:58:00]</a>.
    *   Individual rows or items can be deleted using the delete icon <a class="yt-timestamp" data-t="06:03:00">[06:03:00]</a>.
3.  **Download All PDFs:** Click "Download all PDF" to generate and download all documents in one go <a class="yt-timestamp" data-t="05:18:98">[05:18:98]</a>.
    *   Individual PDFs can also be downloaded by clicking on specific rows <a class="yt-timestamp" data-t="06:16:00">[06:16:00]</a>.
4.  **Currency Impact:** Changes to the currency setting will automatically apply to all generated PDFs during bulk export <a class="yt-timestamp" data-t="07:05:00">[07:05:00]</a>.
