Sales receipts can be generated as PDF documents for business purposes using a tool like PDF Output <a class="yt-timestamp" data-t="02:52:53">[02:52:53]</a>. This process involves accessing a specific template, filling in the required information, and then generating the PDF either individually or in bulk.

## Using PDF Output for Single Sales Receipt Generation

To generate a single sales receipt PDF:
1.  Navigate to [[pdf_output | PDFoutput.com]] <a class="yt-timestamp" data-t="03:08:08">[03:08:08]</a>.
2.  Click on the "Template Selection" dropdown at the top of the page <a class="yt-timestamp" data-t="03:13:58">[03:13:58]</a>.
3.  Search for and select the "sales receipts" template from the list <a class="yt-timestamp" data-t="03:20:54">[03:20:54]</a>. This will open the sales receipt template <a class="yt-timestamp" data-t="03:29:00">[03:29:00]</a>.
4.  Fill in the necessary information, including:
    *   Receipt number and date <a class="yt-timestamp" data-t="03:48:00">[03:48:00]</a>
    *   Payment method <a class="yt-timestamp" data-t="03:50:00">[03:50:00]</a>
    *   Business name and address <a class="yt-timestamp" data-t="03:53:00">[03:53:00]</a>
    *   Individual items with descriptions, quantities, and unit prices <a class="yt-timestamp" data-t="03:55:00">[03:55:00]</a>. For example, two items might be "item one" (10 units at $20 unit price) and "item two" (10 units at $67 unit price) <a class="yt-timestamp" data-t="04:06:00">[04:06:00]</a>.
    *   Tax rate (e.g., 5%) <a class="yt-timestamp" data-t="04:26:00">[04:26:00]</a>
    *   Any relevant notes <a class="yt-timestamp" data-t="04:35:00">[04:35:00]</a>
    *   Currency (defaulting to US dollars, but can be changed to Euro or other currencies) <a class="yt-timestamp" data-t="04:40:00">[04:40:00]</a>.
5.  Once all information is entered, click the "Download PDF" button <a class="yt-timestamp" data-t="04:53:00">[04:53:00]</a>.
6.  The system will generate and download the sales receipt PDF document <a class="yt-timestamp" data-t="04:54:00">[04:54:00]</a>.

## Bulk PDF Sales Receipt Generation

For generating multiple sales receipt PDFs simultaneously:
1.  On the PDF Output platform, select the "Bulk Export Mode Document" option <a class="yt-timestamp" data-t="05:10:00">[05:10:00]</a>.
2.  Input all sales receipt details into the provided database format <a class="yt-timestamp" data-t="05:15:00">[05:15:00]</a>.
    *   Columns and row heights can be resized for readability <a class="yt-timestamp" data-t="05:40:00">[05:40:00]</a>.
    *   New rows can be added using the "add new row" button <a class="yt-timestamp" data-t="05:58:00">[05:58:00]</a>.
    *   Existing rows or individual items within a row can be deleted using the delete icon <a class="yt-timestamp" data-t="06:03:00">[06:03:00]</a>.
3.  To download all PDFs at once, click "Download all PDF" <a class="yt-timestamp" data-t="06:11:00">[06:11:00]</a>. The system will then download all generated PDFs <a class="yt-timestamp" data-t="06:21:00">[06:21:00]</a>.
4.  Alternatively, specific PDFs can be downloaded by clicking on individual rows <a class="yt-timestamp" data-t="06:16:00">[06:16:00]</a>.
5.  The currency displayed in the generated PDFs will automatically update if changed in the input interface <a class="yt-timestamp" data-t="07:05:00">[07:05:00]</a>.
