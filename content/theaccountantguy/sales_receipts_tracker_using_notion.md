## Sales Receipts Database

The core of the tracker is a sales receipts database, which includes several key columns for detailed tracking:
*   **Receipt Number** <a class="yt-timestamp" data-t="00:00:16">[00:00:16]</a>
*   **Receipt Date** <a class="yt-timestamp" data-t="00:00:17">[00:00:17]</a>
*   **Payment Method**: Offers various ways to clear payment <a class="yt-timestamp" data-t="00:00:19">[00:00:19]</a>.
*   **Business Name** <a class="yt-timestamp" data-t="00:00:22">[00:00:22]</a>
*   **Address** <a class="yt-timestamp" data-t="00:00:24">[00:00:24]</a>
*   **Customer Details**: This is a relational property, linking to another database containing client information <a class="yt-timestamp" data-t="00:00:27">[00:00:27]</a>.
*   **Description**: Details what was sold <a class="yt-timestamp" data-t="00:00:38">[00:00:38]</a>.
*   **Quantity of Sales** <a class="yt-timestamp" data-t="00:00:41">[00:00:41]</a>
*   **Unit Price** <a class="yt-timestamp" data-t="00:00:42">[00:00:42]</a>
    *   Multiple descriptions, quantities, and unit prices can be added for each row/sale <a class="yt-timestamp" data-t="00:00:45">[00:00:45]</a>.
*   **Subtotal**: Sums the total amount of individual items sold <a class="yt-timestamp" data-t="00:01:00">[00:01:00]</a>.
*   **Tax Rate** <a class="yt-timestamp" data-t="00:01:10">[00:01:10]</a>
*   **Total Amount**: Automatically computed <a class="yt-timestamp" data-t="00:01:11">[00:01:11]</a>.
*   **Notes**: For any relevant payment-related notes <a class="yt-timestamp" data-t="00:01:14">[00:01:14]</a>.
*   **Additional Columns**: Some columns are linked to other databases and are not necessary for direct input <a class="yt-timestamp" data-t="00:01:18">[00:01:18]</a>.

If adding more columns for descriptions and quantities, ensure to update the formulas for subtotal and total amount <a class="yt-timestamp" data-t="00:01:41">[00:01:41]</a>. The "quantity sold" column also requires updates if additional quantities are added <a class="yt-timestamp" data-t="00:02:04">[00:02:04]</a>.

## Sales Receipt Summary

Located at the top of the template, the summary provides key performance indicators:
*   **Total Sales**: Drawn from the sales value in the database <a class="yt-timestamp" data-t="00:01:28">[00:01:28]</a>.
*   **Units Sold**: Reflects the total quantity of sales <a class="yt-timestamp" data-t="00:01:34">[00:01:34]</a>.
*   **Average Sales Value**: Calculated as total sales divided by units sold, offering crucial business insights <a class="yt-timestamp" data-t="00:02:10">[00:02:10]</a>.

## Generating Sales Receipts PDF Documents with PDFOutput.com

PDFOutput.com is a tool that can be used to generate sales receipts PDF documents for businesses <a class="yt-timestamp" data-t="00:02:52">[00:02:52]</a>.

### Steps to Generate a Single PDF:
1.  Navigate to PDFOutput.com <a class="yt-timestamp" data-t="00:03:08">[00:03:08]</a>.
2.  Click on the "template selection" dropdown at the top <a class="yt-timestamp" data-t="00:03:13">[00:03:13]</a>.
3.  Search for and select the "sales receipts" template <a class="yt-timestamp" data-t="00:03:19">[00:03:19]</a>.
4.  Fill in the required information: receipt number, receipt date, payment method, business name, address, customer details, item descriptions, quantities, unit prices, tax rate, and notes <a class="yt-timestamp" data-t="00:03:30">[00:03:30]</a>.
5.  Select the desired currency from the dropdown (e.g., US dollars, Euro) <a class="yt-timestamp" data-t="00:04:40">[00:04:40]</a>.
6.  Click "Download PDF" to generate the document <a class="yt-timestamp" data-t="00:04:53">[00:04:53]</a>.

### Steps for Bulk PDF Export:
1.  On PDFOutput.com, select the "bulk export mode document" option <a class="yt-timestamp" data-t="00:05:09">[00:05:09]</a>.
2.  Enter all sales receipt information into the provided database-like interface <a class="yt-timestamp" data-t="00:05:15">[00:05:15]</a>.
    *   Column widths and row heights can be resized for readability <a class="yt-timestamp" data-t="00:05:40">[00:05:40]</a>.
    *   New rows can be added, and individual rows or items can be deleted <a class="yt-timestamp" data-t="00:05:58">[00:05:58]</a>.
3.  To download all PDFs at once, click "Download All PDF" <a class="yt-timestamp" data-t="00:06:11">[00:06:11]</a>.
4.  Alternatively, click on individual rows to download specific PDFs <a class="yt-timestamp" data-t="00:06:16">[00:06:16]</a>.
5.  Changing the currency setting will automatically update the currency in the generated PDFs <a class="yt-timestamp" data-t="00:07:05">[00:07:05]</a>.
