## Key Features of the Budget Tracker

The Film Production Budget Tracker in Notion offers several functionalities:

*   **Expense Tracking Table**: A table at the bottom lists expenses, their categories, and the corresponding amounts <a class="yt-timestamp" data-t="00:00:16">[00:00:16]</a>.
*   **Customizable Categories**: Users can define their own expense categories by clicking a pencil icon next to the category field <a class="yt-timestamp" data-t="00:27:00">[00:27:00]</a>. Changes made here are updated bidirectionally <a class="yt-timestamp" data-t="00:30:00">[00:30:00]</a>.
*   **Budgeted Expense Values**: The budgeted expense value can be updated directly within the [[customizing_and_using_notion_templates | template]] <a class="yt-timestamp" data-t="00:32:00">[00:32:00]</a>.
*   **Automatic Updates**: All values in the [[customizing_and_using_notion_templates | template]] update automatically once new data is entered <a class="yt-timestamp" data-t="00:36:00">[00:36:00]</a>.
*   **Summary Section**: This section displays:
    *   Budgeted expense amount <a class="yt-timestamp" data-t="00:48:00">[00:48:00]</a>
    *   Actual expenses <a class="yt-timestamp" data-t="00:49:00">[00:49:00]</a>
    *   The difference between budgeted and actual expenses <a class="yt-timestamp" data-t="00:51:00">[00:51:00]</a>
    *   Percentage of expenses relative to the budgeted amount <a class="yt-timestamp" data-t="00:53:00">[00:53:00]</a>
    This provides a total expense summary <a class="yt-timestamp" data-t="00:56:00">[00:56:00]</a>.
*   **Category Management**: Users can add new categories by clicking "new page," edit existing ones using the pencil icon, or delete them via the three-dot menu <a class="yt-timestamp" data-t="01:04:00">[01:04:00]</a>.
