## How to Generate a Single PDF Invoice

The process for [[using_pdf_output_to_create_invoices | creating individual PDF invoices]] involves several steps:
1.  **Access PDF Output.com**: Navigate to PDF Output.com <a class="yt-timestamp" data-t="01:51:00">[01:51:00]</a>.
2.  **Select Template**: Upon logging in, an interface allows users to select a template from a dropdown menu <a class="yt-timestamp" data-t="01:58:00">[01:58:00]</a>. The "invoice template" is available for this purpose <a class="yt-timestamp" data-t="02:02:00">[02:02:00]</a>.
3.  **Choose Currency**: Users can select their preferred currency, with multiple options available (e.g., USD, Euro) <a class="yt-timestamp" data-t="02:05:00">[02:05:00]</a><a class="yt-timestamp" data-t="02:08:00">[02:08:00]</a>. The invoice will populate with the chosen currency <a class="yt-timestamp" data-t="03:06:00">[03:06:00]</a>.
4.  **Fill in Details**: Input all necessary invoice-related information <a class="yt-timestamp" data-t="02:14:00">[02:14:00]</a>. As details are entered, the PDF will populate accordingly <a class="yt-timestamp" data-t="02:19:00">[02:19:00]</a>.
5.  **Download PDF**: After filling in the details, click "Download PDF" to automatically [[generating_and_downloading_invoice_pdf_documents | generate and download the PDF document]] based on the entered information <a class="yt-timestamp" data-t="02:48:00">[02:48:00]</a><a class="yt-timestamp" data-t="02:52:00">[02:52:00]</a>.

The generated invoice will be ready in the same format as filled out <a class="yt-timestamp" data-t="02:59:00">[02:59:00]</a><a class="yt-timestamp" data-t="03:02:00">[03:02:00]</a>.

## Generating PDF Invoices in Bulk

For [[exporting_and_managing_bulk_pdf_sales_receipts | exporting multiple PDF documents]], a "bulk export mode" is available <a class="yt-timestamp" data-t="03:21:00">[03:21:00]</a>.
1.  **Switch to Bulk Export Mode**: Click on the "bulk export mode" option <a class="yt-timestamp" data-t="03:21:00">[03:21:00]</a>.
2.  **Fill Database**: A database interface will appear where users can fill in details for multiple invoices <a class="yt-timestamp" data-t="03:24:00">[03:24:00]</a>. This includes fields like invoice number, invoice date, and other details present in the single PDF mode <a class="yt-timestamp" data-t="04:02:00">[04:02:00]</a>. Columns can be expanded for better visibility <a class="yt-timestamp" data-t="04:06:00">[04:06:00]</a>.
3.  **Generate Bulk PDFs**: Once the database is filled, click "Download all PDF" to [[generating_pdf_invoices_in_bulk_with_notion | generate PDF documents]] one by one based on the entered information <a class="yt-timestamp" data-t="04:24:00">[04:24:00]</a><a class="yt-timestamp" data-t="04:26:00">[04:26:00]</a>.

The bulk-generated PDFs will appear in the same format as single PDFs, reflecting the chosen currency <a class="yt-timestamp" data-t="04:34:00">[04:34:00]</a><a class="yt-timestamp" data-t="04:38:00">[04:38:00]</a>. Users can also download a PDF for a particular row individually <a class="yt-timestamp" data-t="05:00:00">[05:00:00]</a>.
