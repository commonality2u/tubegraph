This Notion template is designed to help track and manage business expenses for restaurant operations <a class="yt-timestamp" data-t="00:00:02">[00:00:02]</a>. It provides a quick and easy way to monitor expenditures within a single [[notion_finance_templates_overview | Notion template]] <a class="yt-timestamp" data-t="00:01:09">[00:01:09]</a>.

## Template Overview

The template features three distinct categories of expenses at the top <a class="yt-timestamp" data-t="00:00:13">[00:00:13]</a>. For each category, it displays:
*   A budgeted expense <a class="yt-timestamp" data-t="00:00:17">[00:00:17]</a>.
*   Actual expenses, which are dynamically pulled from a table located at the bottom of the template <a class="yt-timestamp" data-t="00:00:19">[00:00:19]</a>.
*   A percentage indicator showing the proportion of actual expenses incurred relative to the budgeted expense <a class="yt-timestamp" data-t="00:00:28">[00:00:28]</a>.

## Expense Management Features

The template offers several functionalities for managing expense categories and inputting data:

*   **Setting Budgets**
    *   To define or change a budgeted expense, click on the existing value <a class="yt-timestamp" data-t="00:00:24">[00:00:24]</a>.
*   **Modifying Categories**
    *   Existing expense categories can be modified by clicking the pencil icon next to them; updates will automatically apply <a class="yt-timestamp" data-t="00:00:33">[00:00:33]</a>.
*   **Adding New Categories**
    *   New expense categories can be added by clicking "new page" <a class="yt-timestamp" data-t="00:00:40">[00:00:40]</a>.
*   **Deleting Categories**
    *   Categories can be deleted by clicking the three-dot icon and selecting delete <a class="yt-timestamp" data-t="00:00:43">[00:00:43]</a>.
*   **Dynamic Updates**
    *   All expense values and totals update dynamically as figures are inputted into the template <a class="yt-timestamp" data-t="00:00:49">[00:00:49]</a>.

## Summary Section

A dedicated summary section provides an overview of overall financial performance:
*   **Total Budgeted Expense** <a class="yt-timestamp" data-t="00:00:59">[00:00:59]</a>.
*   **Actual Expenses Incurred** (pulled from the values within the template) <a class="yt-timestamp" data-t="00:01:00">[00:01:00]</a>.
*   **Difference** between the total budgeted and actual expenses <a class="yt-timestamp" data-t="00:01:04">[00:01:04]</a>.
*   **Percentage of Actual Expenses Incurred** so far <a class="yt-timestamp" data-t="00:01:06">[00:01:06]</a>.

This template serves as a comprehensive tool for [[using_notion_for_budgeting | Notion budgeting]] and [[using_notion_for_expense_management | expense management]] specifically tailored for restaurant operations <a class="yt-timestamp" data-t="00:01:09">[00:01:09]</a>.
