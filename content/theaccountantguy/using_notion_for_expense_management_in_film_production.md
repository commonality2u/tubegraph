This template helps manage all related film production expenses <a class="yt-timestamp" data-t="00:00:06">[00:00:06]</a>.

## Template Overview
The [[overview_of_notion_expense_tracker | Notion expense tracker]] template features a table that lists expenses individually <a class="yt-timestamp" data-t="00:00:16">[00:00:16]</a>. Each entry includes the expense amount and its respective category <a class="yt-timestamp" data-t="00:00:20">[00:00:20]</a>.

## Key Features

### Expense Entry and Tracking
*   **Expense Details:** Users can enter the amount for each expense <a class="yt-timestamp" data-t="00:00:40">[00:00:40]</a>. All updates are reflected immediately in the system <a class="yt-timestamp" data-t="00:00:36">[00:00:36]</a>.
*   **Budgeted Expense Value:** The budgeted expense value can be updated directly within the template <a class="yt-timestamp" data-t="00:00:32">[00:00:32]</a>.

### Category Management
The template allows for comprehensive management of expense categories:
*   **Defining Categories:** Categories are displayed under the "category" section <a class="yt-timestamp" data-t="00:00:25">[00:00:25]</a>. Users can define their own categories by clicking a pencil icon, and these updates will reflect both ways in the system <a class="yt-timestamp" data-t="00:00:28">[00:00:28]</a>.
*   **Adding New Categories:** New categories can be added by clicking on the "new page" option <a class="yt-timestamp" data-t="00:01:06">[00:01:06]</a>.
*   **Editing Categories:** Existing categories can be edited via a pencil icon <a class="yt-timestamp" data-t="00:01:08">[00:01:08]</a>.
*   **Deleting Categories:** Categories can be deleted by clicking the three-dot menu and selecting "delete" <a class="yt-timestamp" data-t="00:01:10">[00:01:10]</a>.

### Summary Section
A dedicated summary section provides an [[using_notion_for_financial_tracking | overview of financial tracking]]:
*   **Budgeted Expense Amount:** Displays the planned expense amount <a class="yt-timestamp" data-t="00:00:48">[00:00:48]</a>.
*   **Actual Expenses:** Shows the real-time incurred expenses <a class="yt-timestamp" data-t="00:00:49">[00:00:49]</a>.
*   **Difference:** Calculates the variance between budgeted and actual expenses <a class="yt-timestamp" data-t="00:00:51">[00:00:51]</a>.
*   **Percentage of Expenses:** Indicates the percentage of actual expenses relative to the budgeted amount <a class="yt-timestamp" data-t="00:00:53">[00:00:53]</a>.
This section offers a comprehensive summary of total expenses and how they compare to the current budget <a class="yt-timestamp" data-t="00:00:56">[00:00:56]</a>.
