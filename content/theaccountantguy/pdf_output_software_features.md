[[using_pdfoutputcom_for_bulk_pdf_generation | PDFOutput.com]] is a software designed to generate professional PDF documents in bulk using Notion templates and databases <a class="yt-timestamp" data-t="00:00:07">[00:00:07]</a>.

## Core Functionality
The primary purpose of PDFOutput is to [[using_pdf_output_tool_for_efficient_document_creation | automate document exports]] by replacing elements in a Notion template with corresponding data from a Notion database <a class="yt-timestamp" data-t="00:00:22">[00:00:22]</a>. This enables [[utilizing_pdf_output_for_bulk_pdf_generation | bulk PDF generation]] for various use cases, such as [[generating_pdf_invoices_using_pdfoutput | generating PDF invoices]] <a class="yt-timestamp" data-t="00:00:07">[00:00:07]</a> or [[generating_pdf_documents_for_purchase_orders | purchase orders]] <a class="yt-timestamp" data-t="00:08:47">[00:08:47]</a>.

## Key Features

### Notion Integration
*   **Template and Database Connection**: Users connect their Notion database and Notion template to the PDFOutput portal <a class="yt-timestamp" data-t="00:01:28">[00:01:28]</a>.
*   **Dynamic Content Replacement**: Elements defined within curly braces in the Notion template are automatically replaced with data from corresponding items in the Notion database <a class="yt-timestamp" data-t="00:00:22">[00:00:22]</a>. For example, `{{client name}}` would be replaced by the 'Client Name' property from the database <a class="yt-timestamp" data-t="00:00:29">[00:00:29]</a>.
*   **Template Duplication**: Users can easily duplicate pre-added templates and databases from PDFOutput onto their local Notion workspace <a class="yt-timestamp" data-t="00:01:43">[00:01:43]</a>, <a class="yt-timestamp" data-t="00:02:22">[00:02:22]</a>.
*   **Workspace Selection**: The tool prompts users to select their Notion workspace when connecting templates and databases <a class="yt-timestamp" data-t="00:02:29">[00:02:29]</a>, <a class="yt-timestamp" data-t="00:03:48">[00:03:48]</a>.
*   **Support for Relational Properties**: If a database uses relational properties linked to other Notion databases, those additional databases also need to be allowed during the connection setup <a class="yt-timestamp" data-t="00:10:50">[00:10:50]</a>.

### Document Generation
*   **Bulk Generation**: The software generates a separate PDF document for every row in the connected Notion database <a class="yt-timestamp" data-t="00:05:57">[00:05:57]</a>.
*   **Automatic Mapping**: If the names of template elements (in curly braces) exactly match the property names in the Notion database, PDFOutput automatically maps them <a class="yt-timestamp" data-t="00:05:29">[00:05:29]</a>, <a class="yt-timestamp" data-t="00:07:41">[00:07:41]</a>.
*   **Manual Mapping**: If names differ, users can manually map database properties to template elements via a drop-down menu <a class="yt-timestamp" data-t="00:05:35">[00:05:35]</a>, <a class="yt-timestamp" data-t="00:07:47">[00:07:47]</a>.
*   **Preview Option**: After generation, users can preview a sample PDF document to verify the output <a class="yt-timestamp" data-t="00:06:16">[00:06:16]</a>.
*   **Download All**: All generated PDF documents can be downloaded in a single action <a class="yt-timestamp" data-t="00:06:18">[00:06:18]</a>, <a class="yt-timestamp" data-t="00:06:40">[00:06:40]</a>.

### User Interface and Experience
*   **Login System**: Users need to log in to access the platform <a class="yt-timestamp" data-t="00:01:10">[00:01:10]</a>.
*   **Connection Management**: The "Connections" sidebar option allows users to view and reload previously defined connections, eliminating the need to redefine templates and databases each time <a class="yt-timestamp" data-t="00:08:03">[00:08:03]</a>.
*   **Template Library**: A "Templates" section provides a list of pre-added templates for various use cases, complete with database and template links for easy duplication <a class="yt-timestamp" data-t="00:01:51">[00:01:51]</a>, <a class="yt-timestamp" data-t="00:08:46">[00:08:46]</a>.
*   **Search Functionality**: Users can search for specific templates within the library (e.g., "invoice") <a class="yt-timestamp" data-t="00:02:04">[00:02:04]</a>.
*   **Custom Templates**: While pre-built templates are available, users can also connect their own custom Notion templates and databases <a class="yt-timestamp" data-t="00:10:10">[00:10:10]</a>.

### Settings and Customization
*   **Page Format**: Users can define the specific page format (e.g., A4 size) for their generated PDFs in the settings window <a class="yt-timestamp" data-t="00:09:24">[00:09:24]</a>.
*   **Add More Databases/Templates**: The settings window also allows for adding more databases and templates after the initial setup <a class="yt-timestamp" data-t="00:09:33">[00:09:33]</a>.

### Subscription and Support
*   **Upgrade Section**: An "Upgrade" section allows users to manage their subscription <a class="yt-timestamp" data-t="00:08:57">[00:08:57]</a>.
*   **Watermark**: Free plans include a "Made with PDF Output" watermark on generated documents, which is removed upon upgrading to a paid plan <a class="yt-timestamp" data-t="00:09:01">[00:09:01]</a>.
*   **Feedback Option**: A feedback option is available for sending messages or queries to support <a class="yt-timestamp" data-t="00:09:58">[00:09:58]</a>.
*   **Help Section**: A "Help" section details how to connect custom PDF documents and databases <a class="yt-timestamp" data-t="00:10:15">[00:10:15]</a>.
*   **Collapsible Sidebar**: The sidebar can be collapsed for a cleaner view <a class="yt-timestamp" data-t="00:10:41">[00:10:41]</a>.
