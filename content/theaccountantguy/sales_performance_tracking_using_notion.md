## Sales Dashboard Components

The sales dashboard at the top of the template features three main components, all divided by month <a class="yt-timestamp" data-t="00:00:09">[00:00:09]</a> <a class="yt-timestamp" data-t="00:01:20">[00:01:20]</a>:

*   **Salesperson**
    This section tracks sales made by each salesperson for specific months <a class="yt-timestamp" data-t="00:00:13">[00:00:13]</a>. For each person, it displays:
    *   Sales achieved for the month <a class="yt-timestamp" data-t="00:00:24">[00:00:24]</a>
    *   Targeted sales <a class="yt-timestamp" data-t="00:00:27">[00:00:27]</a>
    *   Percentage of targeted sales achieved <a class="yt-timestamp" data-t="00:00:30">[00:00:30]</a>
    *   Target achievement status (e.g., "Target achieved" if sales exceed target, "Target not achieved" otherwise) <a class="yt-timestamp" data-t="00:00:32">[00:00:32]</a>

*   **Sales Channel**
    This component shows sales made through various sources, such as a website, Instagram, or YouTube <a class="yt-timestamp" data-t="00:00:49">[00:00:49]</a> <a class="yt-timestamp" data-t="00:00:56">[00:00:56]</a>. Users can add custom categories <a class="yt-timestamp" data-t="00:00:59">[00:00:59]</a>. It also displays achieved sales, targeted sales, percentage of sales, and status <a class="yt-timestamp" data-t="00:01:05">[00:01:05]</a>.

*   **Sales Location**
    This section tracks sales made to different geographical locations, showing the same characteristics as the other components: sales achieved, targeted sales, percentage, and status <a class="yt-timestamp" data-t="00:01:11">[00:01:11]</a> <a class="yt-timestamp" data-t="00:01:16">[00:01:16]</a>.

## Overview Sections

On the left panel, there are detailed overviews that summarize performance across different categories <a class="yt-timestamp" data-t="00:01:34">[00:01:34]</a>:

*   **Salesperson Overview**
    This shows the total sales, sale units, and average sales (sales per unit) made by each individual salesperson over time <a class="yt-timestamp" data-t="00:01:35">[00:01:35]</a>.

*   **Sales Channel Overview**
    Similar to the salesperson overview, this displays total sales, sale units, and average sales for each different sales channel <a class="yt-timestamp" data-t="00:01:54">[00:01:54]</a>.

*   **Sales Location Overview**
    This provides information on total sales, average sales, and sale units for each location where sales have been made <a class="yt-timestamp" data-t="00:02:02">[00:02:02]</a>.

These overviews help track and provide a comprehensive understanding of business performance <a class="yt-timestamp" data-t="00:02:10">[00:02:10]</a>.

## Sales Transactions Input

The template allows for detailed input of sales transactions, which populates the dashboard information <a class="yt-timestamp" data-t="00:02:19">[00:02:19]</a>. The transaction window includes fields for <a class="yt-timestamp" data-t="00:02:37">[00:02:37]</a>:

*   **Date** <a class="yt-timestamp" data-t="00:02:40">[00:02:40]</a>
*   **Account Details**: Information about the salesperson <a class="yt-timestamp" data-t="00:02:42">[00:02:42]</a>
*   **Sales Detail**: What is being sold and any additional information <a class="yt-timestamp" data-t="00:02:48">[00:02:48]</a>
*   **Month and Year** <a class="yt-timestamp" data-t="00:02:52">[00:02:52]</a>
*   **Salesperson Detail**: Assigned to the salesperson responsible <a class="yt-timestamp" data-t="00:02:55">[00:02:55]</a>
*   **Client Detail**: To whom the sale was made <a class="yt-timestamp" data-t="00:02:59">[00:02:59]</a>
*   **Sales Code**: A unique code for each sale, often combining salesperson, month, and year (e.g., "jcore-december23") <a class="yt-timestamp" data-t="00:03:03">[00:03:03]</a>
*   **Sales Target**: Set at the start of the month for each person <a class="yt-timestamp" data-t="00:03:21">[00:03:21]</a>
*   **Sales Amount**: Entered as sales are made <a class="yt-timestamp" data-t="00:03:32">[00:03:32]</a>
*   **Sales Unit**: Number of units sold <a class="yt-timestamp" data-t="00:03:36">[00:03:36]</a>

*   **Filtering Data**: Filters can be applied to view data for specific years or months, allowing for detailed analysis of sales performance over different periods <a class="yt-timestamp" data-t="00:08:15">[00:08:15]</a>.
