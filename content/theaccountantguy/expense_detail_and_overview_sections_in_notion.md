## Main Sections of the Tracker

The Notion credit card expenses tracker features several key sections for [[tracking_income_and_expenses_in_notion | managing finances]]:

*   **Credit Cards Overview** <a class="yt-timestamp" data-t="00:00:11">[00:00:11]</a>: Displays added credit cards, showing the total amount due, amount repaid, and remaining balance due for each <a class="yt-timestamp" data-t="00:00:23">[00:00:23]</a>. Examples include "Student Credit Card," "Cash Back Credit Card," "Travel Credit Card," and "Rewards Credit Card" <a class="yt-timestamp" data-t="00:00:18">[00:00:18]</a>.
*   **Monthly Overview** <a class="yt-timestamp" data-t="00:00:32">[00:00:32]</a>: Shows monthly amounts due, repaid, and balance due, which can be customized <a class="yt-timestamp" data-t="00:00:32">[00:00:32]</a>.
*   **Yearly Overview** <a class="yt-timestamp" data-t="00:00:41">[00:00:41]</a>: Provides annual totals for amount due, repaid, and balance due <a class="yt-timestamp" data-t="00:00:41">[00:00:41]</a>.
*   **Notification Section** <a class="yt-timestamp" data-t="00:01:16">[00:01:16]</a>: Indicates the next repayment date and the days remaining until that date for each credit card <a class="yt-timestamp" data-t="00:01:16">[00:01:16]</a>. Users can set their own repayment dates <a class="yt-timestamp" data-t="00:04:40">[00:04:40]</a>.

## Expense Detail Section

The "Expense Detail" section is where all individual expenses are recorded <a class="yt-timestamp" data-t="00:00:51">[00:00:51]</a>. This section is crucial for [[detailing_the_transaction_section_in_notion | recording detailed transactions]] and populating the overviews <a class="yt-timestamp" data-t="00:01:37">[00:01:37]</a>.

### Data Fields

For each expense entry, the following information can be entered:
*   **Date** <a class="yt-timestamp" data-t="00:01:42">[00:01:42]</a>, <a class="yt-timestamp" data-t="00:02:18">[00:02:18]</a>
*   **Expense Details** <a class="yt-timestamp" data-t="00:01:44">[00:01:44]</a>: A description of the expense <a class="yt-timestamp" data-t="00:02:19">[00:02:19]</a>.
*   **Period** <a class="yt-timestamp" data-t="00:01:46">[00:01:46]</a>: The specific month or period for the expense <a class="yt-timestamp" data-t="00:02:24">[00:02:24]</a>.
*   **Year** <a class="yt-timestamp" data-t="00:01:48">[00:01:48]</a>: The year the expense occurred <a class="yt-timestamp" data-t="00:02:28">[00:02:28]</a>.
*   **Credit Card** <a class="yt-timestamp" data-t="00:01:51">[00:01:51]</a>: The credit card used for the expense <a class="yt-timestamp" data-t="00:02:33">[00:02:33]</a>. New credit cards can be added directly by typing them in <a class="yt-timestamp" data-t="00:02:37">[00:02:37]</a>.
*   **Category** <a class="yt-timestamp" data-t="00:01:54">[00:01:54]</a>: An assigned category for the expense <a class="yt-timestamp" data-t="00:02:47">[00:02:47]</a>. Categories can be added or edited from this section or a dedicated category section <a class="yt-timestamp" data-t="00:05:22">[00:05:22]</a>.
*   **Expense Amount** <a class="yt-timestamp" data-t="00:01:57">[00:01:57]</a>: The monetary value of the expense <a class="yt-timestamp" data-t="00:02:50">[00:02:50]</a>.
*   **Repayment Section** <a class="yt-timestamp" data-t="00:02:01">[00:02:01]</a>: To record the amount repaid for a particular credit card <a class="yt-timestamp" data-t="00:03:00">[00:03:00]</a>.
*   **Opening Balance Due** <a class="yt-timestamp" data-t="00:02:07">[00:02:07]</a>: For any balances carried forward from previous months <a class="yt-timestamp" data-t="00:02:09">[00:02:09]</a>.
*   **Receipt Column** <a class="yt-timestamp" data-t="00:05:34">[00:05:34]</a>: Allows users to attach bills or receipts for expenses <a class="yt-timestamp" data-t="00:05:39">[00:05:39]</a>.

As data is entered in the Expense Detail section, all top-level overviews automatically update <a class="yt-timestamp" data-t="00:03:07">[00:03:07]</a>.

The Expense Detail section also provides different summarization views for [[different_views_for_expense_tracking_in_notion | reviewing transactions]]:
*   **Period View** <a class="yt-timestamp" data-t="00:05:44">[00:05:44]</a>: Clicking on a specific period (e.g., month) opens a new window showing transactions for that period, including dates, details, amounts, and tags (e.g., "repayment" in green, "expense") <a class="yt-timestamp" data-t="00:05:49">[00:05:49]</a>.
*   **Year View** <a class="yt-timestamp" data-t="00:06:14">[00:06:14]</a>: Provides similar detailed information for a particular year <a class="yt-timestamp" data-t="00:06:16">[00:06:16]</a>.
*   **Credit Card View** <a class="yt-timestamp" data-t="00:06:21">[00:06:21]</a>: Clicking on a credit card shows all related transactions, including a status and category for the credit card itself <a class="yt-timestamp" data-t="00:06:26">[00:06:26]</a>.
*   **Category View** <a class="yt-timestamp" data-t="00:06:48">[00:06:48]</a>: Selecting an expense category reveals all expenses attributed to it, along with relevant details <a class="yt-timestamp" data-t="00:06:53">[00:06:53]</a>.

## Expenses Overview Section

The "Expenses Overview" section provides a summary of all booked expenses, categorized and compared against a budget <a class="yt-timestamp" data-t="00:00:58">[00:00:58]</a>. This section helps in [[budgeting_and_tracking_in_notion | understanding spending habits]] and managing finances.

### Data Fields

This section displays:
*   **Expense Categories** <a class="yt-timestamp" data-t="00:01:01">[00:01:01]</a>: A list of all categories added in the Expense Detail section <a class="yt-timestamp" data-t="00:03:42">[00:03:42]</a>.
*   **Expense Amount** <a class="yt-timestamp" data-t="00:03:52">[00:03:52]</a>: The total amount incurred for each category to date <a class="yt-timestamp" data-t="00:03:52">[00:03:52]</a>.
*   **Budgeted Amount** <a class="yt-timestamp" data-t="00:03:55">[00:03:55]</a>: The pre-set budget for each expense category <a class="yt-timestamp" data-t="00:03:55">[00:03:55]</a>. Users can click to set or adjust their budgeted amounts <a class="yt-timestamp" data-t="00:03:59">[00:03:59]</a>.
*   **Budget Status** <a class="yt-timestamp" data-t="00:01:08">[00:01:08]</a>: Indicates whether the spending is "Under Budget" or "Over Budget" compared to the set amount <a class="yt-timestamp" data-t="00:01:11">[00:01:11]</a>, <a class="yt-timestamp" data-t="00:04:26">[00:04:26]</a>.

This [[presentation_of_expense_data_in_notion | overview and transaction management in Notion | presentation of expense data]] allows users to quickly assess their spending against their [[budgeting_and_tracking_in_notion | budget]] for each category, aiding in timely credit card debt payments <a class="yt-timestamp" data-t="00:05:05">[00:05:05]</a>. The template can be customized to personal needs <a class="yt-timestamp" data-t="00:07:25">[00:07:25]</a>.
