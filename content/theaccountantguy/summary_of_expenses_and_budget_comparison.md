## Expense Tracking

The core of the template is a table that lists individual expenses, each with its respective category and the amount <a class="yt-timestamp" data-t="00:00:16">[00:00:16]</a>. Users can input the expense amount <a class="yt-timestamp" data-t="00:00:40">[00:00:40]</a> and update the [[setting_and_tracking_budgeted_expenses | budgeted expense]] value directly within the table <a class="yt-timestamp" data-t="00:00:32">[00:00:32]</a>. All updates are reflected immediately <a class="yt-timestamp" data-t="00:00:36">[00:00:36]</a>.

### Customizing Categories
Categories can be customized to fit specific needs <a class="yt-timestamp" data-t="00:01:18">[00:01:18]</a>:
*   **Defining new categories**: Click a pencil icon to define a custom category, which updates bidirectionally <a class="yt-timestamp" data-t="00:00:28">[00:00:28]</a>.
*   **Adding categories**: A "new page" option allows for adding new categories <a class="yt-timestamp" data-t="00:01:06">[00:01:06]</a>.
*   **Editing categories**: Categories can be edited via the pencil icon <a class="yt-timestamp" data-t="00:01:09">[00:01:09]</a>.
*   **Deleting categories**: Categories can be deleted using the three-dot menu <a class="yt-timestamp" data-t="00:01:11">[00:01:11]</a>.
