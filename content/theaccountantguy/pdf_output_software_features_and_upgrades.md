PDFOutput.com is a software designed to help users [[using_pdf_output_to_generate_invoices | generate professional invoice PDF documents in bulk]] <a class="yt-timestamp" data-t="00:00:07">[00:00:07]</a>. It leverages Notion templates and databases to automate the creation of multiple PDF files <a class="yt-timestamp" data-t="00:00:09">[00:00:09]</a>.

## Core Functionality: Notion Integration
[[integration_with_pdf_output_tools | PDFOutput integrates with Notion]] to pull data from a database and populate a predefined template <a class="yt-timestamp" data-t="00:00:26">[00:00:26]</a>.

### Template and Database Structure
A professional invoice template within PDFOutput defines elements that will be replaced by data from a Notion database <a class="yt-timestamp" data-t="00:00:16">[00:00:16]</a>. These elements are enclosed in curly braces, such as `{client name}`, `{client company}`, and `{client address}` <a class="yt-timestamp" data-t="00:00:22">[00:00:22]</a>. The corresponding Notion database must contain properties with the exact same names for automatic mapping <a class="yt-timestamp" data-t="00:00:35">[00:00:35]</a>.

### Bulk Document Generation
Once the template and database are set up, PDFOutput can [[pdfoutput_for_bulk_document_generation | generate PDF documents in bulk]] for every row in the Notion database <a class="yt-timestamp" data-t="00:00:55">[00:00:55]</a>.

## Getting Started

1.  **Login to PDFOutput.com**: Users must log in to the PDFOutput.com portal <a class="yt-timestamp" data-t="00:01:06">[00:01:06]</a>.
2.  **Duplicate Notion Templates and Databases**: Before connecting, users need to copy the desired template and database from PDFOutput's pre-added templates onto their local Notion workspace <a class="yt-timestamp" data-t="00:01:43">[00:01:43]</a>. This is done by searching for a template (e.g., "invoice") <a class="yt-timestamp" data-t="00:02:04">[00:02:04]</a>, then clicking the "database link" and "template link" to open them <a class="yt-timestamp" data-t="00:02:00">[00:02:00]</a>. From there, click the "Duplicate" option in Notion to add them to your private workspace <a class="yt-timestamp" data-t="00:02:22">[00:02:22]</a>.
3.  **Connect Notion Workspace**: In PDFOutput, navigate to the connection details screen <a class="yt-timestamp" data-t="00:01:28">[00:01:28]</a>. Click "Add Database" or "Add Template" <a class="yt-timestamp" data-t="00:03:40">[00:03:40]</a>, select your Notion workspace (e.g., "the accountant guy") <a class="yt-timestamp" data-t="00:03:54">[00:03:54]</a>, and then select the duplicated template and database pages <a class="yt-timestamp" data-t="00:04:01">[00:04:01]</a>. Grant access to establish the connection <a class="yt-timestamp" data-t="00:04:07">[00:04:07]</a>.
4.  **Define Connection Name**: Give the connection a name, such as "invoice template" <a class="yt-timestamp" data-t="00:04:51">[00:04:51]</a>.

### Data Mapping
After connection, PDFOutput automatically loads and maps database properties to template elements <a class="yt-timestamp" data-t="00:05:07">[00:05:07]</a>. If the Notion database property names exactly match the template element names (e.g., `{client name}` in template and "Client name" property in database), mapping occurs automatically <a class="yt-timestamp" data-t="00:05:25">[00:05:25]</a>. If names differ, users can manually map elements by clicking on the unmapped item and searching for the correct property <a class="yt-timestamp" data-t="00:05:35">[00:05:36]</a>.

### Generating and Previewing PDFs
Once all elements are mapped, click "Create PDF" to start the [[using_pdf_output_for_document_export_and_preview | PDF generation process]] <a class="yt-timestamp" data-t="00:05:52">[00:05:52]</a>. After successful export, users can preview a sample PDF <a class="yt-timestamp" data-t="00:06:16">[00:06:16]</a> or download all generated documents <a class="yt-timestamp" data-t="00:06:18">[00:06:18]</a>. The generated PDFs reflect the data from each row of the Notion database <a class="yt-timestamp" data-t="00:06:23">[00:06:23]</a>.

## Customization and Advanced Features

### Template Customization
The Notion template is fully [[custom_pdf_document_solutions | customizable]] to fit specific requirements <a class="yt-timestamp" data-t="00:07:30">[00:07:30]</a>. Elements intended to be populated from the database must be enclosed in curly braces <a class="yt-timestamp" data-t="00:07:35">[00:07:35]</a>.

### Relational Properties
If a Notion database uses relational properties (meaning it's connected to another database), users must also allow access for that related database during the connection setup process in PDFOutput <a class="yt-timestamp" data-t="00:10:50">[00:10:50]</a>.
