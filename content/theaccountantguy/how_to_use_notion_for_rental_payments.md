The Notion House Rent Payments Tracker is a template designed to help users efficiently [[using_notion_for_expense_management | keep track of]] house rental payments <a class="yt-timestamp" data-t="00:00:08">[00:00:08]</a>. This template provides a comprehensive system for [[tracking_invoice_payments_in_notion | managing tenant information]], property details, payment history, and overall financial summaries <a class="yt-timestamp" data-t="00:00:06">[00:00:06]</a>.

## Summary Section

The top section of the template features a summary with four distinct views <a class="yt-timestamp" data-t="00:00:11">[00:00:11]</a>:

### Basic Information

This view allows for the addition of fundamental details, including tenants and the classification of properties owned <a class="yt-timestamp" data-t="00:00:15">[00:00:15]</a>. Properties can be classified (e.g., "apartment," "duplex"), and additional classifications can be added as needed <a class="yt-timestamp" data-t="00:00:23">[00:00:23]</a>.

To add a new tenant:
1.  Click the "ADD tenant" button on the left <a class="yt-timestamp" data-t="00:00:29">[00:00:29]</a>. This will create a new tenant entry <a class="yt-timestamp" data-t="00:00:32">[00:00:32]</a>.
2.  Click the three-dot icon next to the new tenant entry to move it under the appropriate property category <a class="yt-timestamp" data-t="00:00:37">[00:00:37]</a>.
3.  Click the pencil icon to add detailed tenant information <a class="yt-timestamp" data-t="00:00:41">[00:00:41]</a>.

### Property Section

This section displays different property classifications <a class="yt-timestamp" data-t="00:00:48">[00:00:48]</a>. Underneath each property, you can view:
*   The tenant's name <a class="yt-timestamp" data-t="00:00:51">[00:00:51]</a>
*   Total rent amount <a class="yt-timestamp" data-t="00:00:54">[00:00:54]</a>
*   Rent amount that has been paid <a class="yt-timestamp" data-t="00:00:56">[00:00:56]</a>
*   Balance rent amount that is due <a class="yt-timestamp" data-t="00:00:58">[00:00:58]</a>
*   Advanced rent amount that has been paid <a class="yt-timestamp" data-t="00:00:59">[00:00:59]</a>

### Rental History

The rental history view provides a record of all rent payments made by tenants based on recorded transactions <a class="yt-timestamp" data-t="00:01:07">[00:01:07]</a>.

### Summary

This final summary section shows key financial figures:
*   Total amount due <a class="yt-timestamp" data-t="00:01:14">[00:01:14]</a>
*   Amount that has been paid <a class="yt-timestamp" data-t="00:01:15">[00:01:15]</a>
*   Outstanding rent amount <a class="yt-timestamp" data-t="00:01:17">[00:01:17]</a>
*   Percentage of rent collected <a class="yt-timestamp" data-t="00:01:18">[00:01:18]</a>

## Transaction Section

The transaction section offers three distinct views in addition to a view showing all transactions <a class="yt-timestamp" data-t="00:01:22">[00:01:22]</a>:

### Rent Paid

This view displays all transactions for which rent has been successfully paid, along with the respective tenants and amounts <a class="yt-timestamp" data-t="00:01:28">[00:01:28]</a>.

### Rent Due

This section shows information for rent payments that are due but have not yet been paid <a class="yt-timestamp" data-t="00:01:34">[00:01:34]</a>. If a rent payment has been received, clicking the corresponding checkbox will move that transaction to the "Rent Paid" category <a class="yt-timestamp" data-t="00:01:39">[00:01:39]</a>.

### Quarterly Summary

The quarterly summary provides an overview of rental payments, collections, and the balance due for each month within a given quarter <a class="yt-timestamp" data-t="00:01:45">[00:01:45]</a>.

To quickly add a new rental payment, click the dedicated button provided in the template <a class="yt-timestamp" data-t="00:01:53">[00:01:53]</a>.
