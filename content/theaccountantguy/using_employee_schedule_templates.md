From: [[theaccountantguy]] <br/> 

The employee schedule template is a tool designed to [[setting_up_employee_schedules_in_notion | keep track of the schedule of all employees]] working in an organization <a class="yt-timestamp" data-t="00:00:05">[00:00:05]</a>. It offers different views to manage various aspects of [[tracking_employee_tasks_and_shifts | employee tasks and shifts]] and [[monitoring_employee_performance_and_hours | employee performance and hours]] <a class="yt-timestamp" data-t="00:00:12">[00:00:12]</a>.

## Template Views

The template provides four primary views:
*   Employee Details <a class="yt-timestamp" data-t="00:00:14">[00:00:14]</a>
*   Task Schedule <a class="yt-timestamp" data-t="00:00:18">[00:00:18]</a>
*   Monthly View <a class="yt-timestamp" data-t="00:00:24">[00:00:24]</a>
*   Employee Performance <a class="yt-timestamp" data-t="00:00:33">[00:00:33]</a>

### Employee Details

This section lists the details of all employees along with relevant information <a class="yt-timestamp" data-t="00:00:16">[00:00:16]</a>.
*   **Adding New Employees**: Click "new" to start adding an employee under relevant categories <a class="yt-timestamp" data-t="00:00:41">[00:00:41]</a>. A quick action button "add employee" on the top can also be used <a class="yt-timestamp" data-t="00:01:31">[00:01:31]</a>.
*   **Categories**: Examples include Operations, Marketing, and Finance <a class="yt-timestamp" data-t="00:00:49">[00:00:49]</a>. New categories can be added using the plus icon, and existing ones can be edited <a class="yt-timestamp" data-t="00:00:53">[00:00:53]</a>.
*   **Employee Information**: When adding a new employee, you can enter details such as:
    *   Name <a class="yt-timestamp" data-t="00:01:07">[00:01:07]</a>
    *   Employee ID <a class="yt-timestamp" data-t="00:01:07">[00:01:07]</a>
    *   Contact information <a class="yt-timestamp" data-t="00:01:07">[00:01:07]</a>
    *   Department <a class="yt-timestamp" data-t="00:01:17">[00:01:17]</a>
    *   Position <a class="yt-timestamp" data-t="00:01:17">[00:01:17]</a>
    *   Email ID <a class="yt-timestamp" data-t="00:01:20">[00:01:20]</a>
This view supports [[employee_record_management | employee record management]] <a class="yt-timestamp" data-t="00:01:12">[00:01:12]</a>.

### Task Schedule

This section helps you enter all the tasks for employees for different dates <a class="yt-timestamp" data-t="00:00:20">[00:00:20]</a>.
*   **Shifts**: Tasks are typically divided into different shifts, such as first, second, and third shifts <a class="yt-timestamp" data-t="00:01:50">[00:01:50]</a>.
*   **Task Information**: For each task, you can enter:
    *   Start time: Including the desired date and time <a class="yt-timestamp" data-t="00:01:59">[00:01:59]</a>.
    *   End time: Indicating the period the time belongs to <a class="yt-timestamp" data-t="00:02:04">[00:02:04]</a>.
    *   Task detail: Where you can add specific task-related information <a class="yt-timestamp" data-t="00:02:10">[00:02:10]</a>.
*   **Employee Assignment**: Select the employee name from those added in the Employee Details section <a class="yt-timestamp" data-t="00:02:15">[00:02:15]</a>. Employees can also be added directly from this section <a class="yt-timestamp" data-t="00:02:21">[00:02:21]</a>.
*   **Automatic Department Assignment**: The respective department for each employee will automatically appear based on the settings in Employee Details <a class="yt-timestamp" data-t="00:02:25">[00:02:25]</a>.
*   **Hours Computation**: The hours an employee has worked on a specific task are automatically computed based on the entered start and end times <a class="yt-timestamp" data-t="00:02:33">[00:02:33]</a>.
*   **Attendance**: Employees can mark their attendance as present, on leave, or a day off <a class="yt-timestamp" data-t="00:02:40">[00:02:40]</a>.
*   **Task Status**: Employees can update the task status as in progress, completed, or not started <a class="yt-timestamp" data-t="00:02:49">[00:02:49]</a>. For instances like a day off or leave, the task status can be marked as "not applicable" <a class="yt-timestamp" data-t="00:02:57">[00:02:57]</a>.

### Monthly View

The monthly view allows you to see performance information, with months divided into different quarters <a class="yt-timestamp" data-t="00:02:21">[00:02:21]</a>.
*   **Performance Overview**: It displays:
    *   Total hours worked for a specific month <a class="yt-timestamp" data-t="00:02:29">[00:02:29]</a>.
    *   Total hours on leave <a class="yt-timestamp" data-t="00:02:30">[00:02:30]</a>.
    *   Total hours (summation of working and leave hours) <a class="yt-timestamp" data-t="00:02:33">[00:02:33]</a>.
This provides a quick overview of overall monthly performance <a class="yt-timestamp" data-t="00:03:38">[00:03:38]</a>.

### Employee Performance

This section shows individual employee performance <a class="yt-timestamp" data-t="00:03:45">[00:03:45]</a>.
*   **Individual Performance Metrics**: For each employee, you can see:
    *   Hours worked <a class="yt-timestamp" data-t="00:03:57">[00:03:57]</a>.
    *   Hours on leave <a class="yt-timestamp" data-t="00:03:59">[00:03:59]</a>.
    *   Total hours in totality <a class="yt-timestamp" data-t="00:04:00">[00:04:00]</a>.
This view is useful for [[monitoring_employee_performance_and_hours | monitoring employee performance and hours]] for specific individuals <a class="yt-timestamp" data-t="00:03:50">[00:03:50]</a>.

## Customization

The employee schedule template is completely customizable to fit specific needs <a class="yt-timestamp" data-t="00:04:13">[00:04:13]</a>. For additional customized requirements or similar templates, assistance can be requested via email <a class="yt-timestamp" data-t="00:04:16">[00:04:16]</a>.
