This Notion template is designed to help you keep track of your house rental payments <a class="yt-timestamp" data-t="00:00:06">[00:00:06]</a>.

## Summary Section

The top section of the tracker provides a summary with four different views <a class="yt-timestamp" data-t="00:00:10">[00:00:10]</a>.

### Basic Information

This view allows for the addition of basic information, including:
*   **Tenants:** Tenants can be added and classified under different properties you own <a class="yt-timestamp" data-t="00:00:15">[00:00:15]</a>.
*   **Properties:** Property classifications like "apartment" and "duplex" are pre-set, but you can add others as needed <a class="yt-timestamp" data-t="00:00:20">[00:00:20]</a>.
    *   To add a new tenant, click the "ADD tenant" button on the left <a class="yt-timestamp" data-t="00:00:27">[00:00:27]</a>.
    *   A new tenant entry will be created automatically <a class="yt-timestamp" data-t="00:00:30">[00:00:30]</a>.
    *   You can then drag the new entry to its respective property category using the three-dot menu <a class="yt-timestamp" data-t="00:00:33">[00:00:33]</a>.
    *   Click the pencil icon to add detailed information for the tenant <a class="yt-timestamp" data-t="00:00:37">[00:00:37]</a>.

### Property Section

This view displays different property classifications <a class="yt-timestamp" data-t="00:00:46">[00:00:46]</a>. Under each property, you can see the tenant's name and key payment information <a class="yt-timestamp" data-t="00:00:49">[00:00:49]</a>:
*   Total rent amount <a class="yt-timestamp" data-t="00:00:54">[00:00:54]</a>
*   Rent amount that has been paid <a class="yt="yt-timestamp" data-t="00:00:55">[00:00:55]</a>
*   Balance rent amount that is due <a class="yt-timestamp" data-t="00:00:57">[00:00:57]</a>
*   Advanced rent that has been paid <a class="yt-timestamp" data-t="00:00:58">[00:00:58]</a>

### Rental History

This view provides a complete rental history for all tenants, based on the transactions recorded at the bottom of the tracker <a class="yt-timestamp" data-t="00:01:04">[00:01:04]</a>.

### Overall Summary

The final summary section shows a financial overview <a class="yt-timestamp" data-t="00:01:12">[00:01:12]</a>:
*   Total amount due <a class="yt-timestamp" data-t="00:01:14">[00:01:14]</a>
*   Amount that has been paid <a class="yt-timestamp" data-t="00:01:15">[00:01:15]</a>
*   Outstanding rent amount <a class="yt-timestamp" data-t="00:01:16">[00:01:16]</a>
*   Percentage of rent collected <a class="yt-timestamp" data-t="00:01:17">[00:01:17]</a>

## Transaction Section

The transaction section offers three different views, in addition to a view showing all transactions <a class="yt-timestamp" data-t="00:01:20">[00:01:20]</a>.

### Rent Paid

This view lists all transactions where rent has been successfully paid, showing respective tenants and amounts <a class="yt-timestamp" data-t="00:01:24">[00:01:24]</a>.

### Rent Due

This view displays information for rent amounts that are due and have not yet been paid <a class="yt-timestamp" data-t="00:01:31">[00:01:31]</a>.
*   Clicking the checkbox next to an entry indicates that the rent has been paid <a class="yt-timestamp" data-t="00:01:35">[00:01:35]</a>.
*   Once checked, the transaction will automatically move to the "Rent Paid" category <a class="yt-timestamp" data-t="00:01:38">[00:01:38]</a>.

### Quarterly Summary

This view provides a summary of rental payments, collections, and the balance due for every month within a respective quarter <a class="yt-timestamp" data-t="00:01:41">[00:01:41]</a>.

## Adding a New Payment

To quickly add a new rental payment, click the dedicated button within the transaction section <a class="yt-timestamp" data-t="00:01:50">[00:01:50]</a>. This will add the payment for you <a class="yt-timestamp" data-t="00:01:53">[00:01:53]</a>.
