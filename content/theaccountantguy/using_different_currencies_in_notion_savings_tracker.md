## How to Change Currency

To change the currency displayed in your Notion savings tracker:
1.  Click on any amount that currently displays a dollar currency symbol <a class="yt-timestamp" data-t="00:04:09">[00:04:09]</a>.
2.  Select "Edit property" <a class="yt-timestamp" data-t="00:04:15">[00:04:15]</a>.
3.  Within the property settings, find the "Number format" option <a class="yt-timestamp" data-t="00:04:16">[00:04:16]</a>.
4.  This section will show all the different currencies that Notion supports <a class="yt-timestamp" data-t="00:04:18">[00:04:18]</a>.
