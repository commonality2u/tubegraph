## Template Sections

The bookkeeping template in [[overview_and_transaction_management_in_notion | Notion]] is divided into three main sections <a class="yt-timestamp" data-t="00:00:26">[00:00:26]</a>:
*   **Start Here Section** <a class="yt-timestamp" data-t="00:00:27">[00:00:27]</a>: Located at the top, used for initial setup and business information <a class="yt-timestamp" data-t="00:00:40">[00:00:40]</a>.
*   **Summary Section** <a class="yt-timestamp" data-t="00:00:30">[00:00:30]</a>: Displays a summary of all entered [[adding_and_editing_transactions_in_notion | transactions]] <a class="yt-timestamp" data-t="00:00:32">[00:00:32]</a>.
*   **Transactions Window** <a class="yt-timestamp" data-t="00:00:35">[00:00:35]</a>: Where all transaction details related to the business are entered <a class="yt-timestamp" data-t="00:00:37">[00:00:37]</a>.

## Start Here Section: Setup and Configuration

Before entering transactions, the "Start Here" section is used to input relevant business information <a class="yt-timestamp" data-t="00:00:42">[00:00:42]</a>.

### Accounts

This section includes various account categories <a class="yt-timestamp" data-t="00:00:49">[00:00:49]</a>:
*   **Expenses** <a class="yt-timestamp" data-t="00:00:50">[00:00:50]</a>: E.g., purchase expense, utility bills, subscription, salaries and wages <a class="yt-timestamp" data-t="00:00:58">[00:00:58]</a>.
*   **Liabilities** <a class="yt-timestamp" data-t="00:00:52">[00:00:52]</a>: E.g., accounts payable, taxes payable, accrued expenses, credit card <a class="yt-timestamp" data-t="00:01:03">[00:01:03]</a>.
*   **Income** <a class="yt-timestamp" data-t="00:00:52">[00:00:52]</a>.
*   **Assets** <a class="yt-timestamp" data-t="00:00:54">[00:00:54]</a>.

#### Managing Accounts

*   **Editing Items**: Hover over an item and click the pencil icon to edit its details <a class="yt-timestamp" data-t="00:01:10">[00:01:10]</a>.
*   **Deleting Items**: Click the three-dot icon next to an item and select "delete" <a class="yt-timestamp" data-t="00:01:21">[00:01:21]</a>.
*   **Adding Items**: Go to the respective category and click "new page" to add a new account <a class="yt-timestamp" data-t="00:01:28">[00:01:28]</a>.

#### Account Properties

Each account item has several properties <a class="yt-timestamp" data-t="00:01:40">[00:01:40]</a>:
*   **Opening Balance**: For liabilities and assets, you can enter any carried-forward balances <a class="yt-timestamp" data-t="00:01:44">[00:01:44]</a>.
*   **Classification**: Allows categorizing items (e.g., bills, debt, savings) <a class="yt-timestamp" data-t="00:01:53">[00:01:53]</a>. For expenses like utility bills and subscriptions, classifications such as "bills" and "subscription" are pre-added <a class="yt-timestamp" data-t="00:02:01">[00:02:01]</a>.
*   **Payment Frequency**: For recurring payments (e.g., monthly, half-yearly, quarterly) <a class="yt-timestamp" data-t="00:02:12">[00:02:12]</a>.
*   **Opening Units**: For inventory or assets, this indicates starting units at the beginning of the business <a class="yt-timestamp" data-t="00:02:27">[00:02:27]</a>.
*   **Asset Class**: Can be classified as "inventory" or "investment" <a class="yt-timestamp" data-t="00:02:42">[00:02:42]</a>.

## Transactions Window: Entering Details

This section allows for the [[adding_and_editing_transactions_in_notion | adding and editing of transactions]] for income, expense, assets, and liabilities <a class="yt-timestamp" data-t="00:03:04">[00:03:04]</a>.

### Adding an Income Transaction (Example)

1.  **Add Income Button**: Click "add income" to automatically generate an income transaction with the current date <a class="yt-timestamp" data-t="00:03:10">[00:03:10]</a>.
2.  **Period**: Enter or select the relevant month and year (e.g., "October 2024") <a class="yt-timestamp" data-t="00:03:24">[00:03:24]</a>.
3.  **Description**: Add a description for the transaction <a class="yt-timestamp" data-t="00:03:42">[00:03:42]</a>.
4.  **Category**: Automatically mapped to "income" when "add income" is clicked <a class="yt-timestamp" data-t="00:03:45">[00:03:45]</a>.
5.  **Account**: Choose from the accounts set up in the "Start Here" section <a class="yt-timestamp" data-t="00:03:50">[00:03:50]</a>. New accounts added here will also reflect at the top <a class="yt-timestamp" data-t="00:04:00">[00:04:00]</a>.
6.  **Payment Mode**: Select how payment is received (e.g., bank account, cash account) <a class="yt-timestamp" data-t="00:04:21">[00:04:21]</a>. More accounts can be added as needed <a class="yt-timestamp" data-t="00:04:29">[00:04:29]</a>.
7.  **Transaction Type**: Designate as "inflow" (money coming in) or "outflow" (money going out) <a class="yt-timestamp" data-t="00:04:51">[00:04:51]</a>.
8.  **Amount**: Enter the numerical value of the transaction <a class="yt-timestamp" data-t="00:05:07">[00:05:07]</a>.
9.  **Document Section**: Attach relevant documents <a class="yt-timestamp" data-t="00:05:15">[00:05:15]</a>.
10. **Tax Rate and Tax Amount**: Enter a tax rate, and the tax amount will be automatically calculated <a class="yt-timestamp" data-t="00:05:21">[00:05:21]</a>. For non-taxable transactions, set the rate to 0% <a class="yt-timestamp" data-t="00:05:27">[00:05:27]</a>.
11. **Multiplier and Currency**: For multiple currencies, a multiplier can be added to convert values (e.g., USD to EUR) <a class="yt-timestamp" data-t="00:05:36">[00:05:36]</a>.

The same process applies for [[business_expense_tracking_with_notion | expense]], asset, and liability transactions <a class="yt-timestamp" data-t="00:06:01">[00:06:01]</a>.

### Investments and Inventory within Transactions

*   **Investments**: Tracks investments, units, price, total value, and investment value <a class="yt-timestamp" data-t="00:06:07">[00:06:07]</a>. It also helps track profit or loss when selling investments <a class="yt-timestamp" data-t="00:06:17">[00:06:17]</a>.
*   **Inventory**: Provides details on existing inventory, including profit or loss obtained from specific items <a class="yt-timestamp" data-t="00:06:26">[00:06:26]</a>.

## Summary Section: Financial Overview

All [[adding_and_editing_transactions_in_notion | transactions]] entered in the transactions window are automatically updated in the summary section <a class="yt-timestamp" data-t="00:06:48">[00:06:48]</a>.

### Finances

*   **Assets**: Shows accounts receivable, input taxes, other receivables, cash value, and their percentages of total asset value <a class="yt-timestamp" data-t="00:07:05">[00:07:05]</a>.
*   **Income Sources**, **Liabilities**, and **Expenses** are also reflected <a class="yt-timestamp" data-t="00:07:01">[00:07:01]</a>.

### Debt

Displays debt payments based on entries made on the liability side <a class="yt-timestamp" data-t="00:07:23">[00:07:23]</a>:
*   **Past Payments**: Records payments already made <a class="yt-timestamp" data-t="00:07:34">[00:07:34]</a>.
*   **Upcoming Payments**: Shows future payments due <a class="yt-timestamp" data-t="00:07:37">[00:07:37]</a>.

### Net Worth

*   **Total Net Worth**: Calculated from all assets, liabilities, and transactions <a class="yt-timestamp" data-t="00:09:08">[00:09:08]</a>.
*   **Difference**: The difference between the set goal and the total net worth <a class="yt-timestamp" data-t="00:09:15">[00:09:15]</a>.
*   **Total Growth**: Shows growth in net worth from the start of the financial year <a class="yt-timestamp" data-t="00:09:19">[00:09:19]</a>.
*   **Percentage of Net Worth Goal Achieved**: Indicates progress towards the net worth goal <a class="yt-timestamp" data-t="00:09:26">[00:09:26]</a>.

## Workflow for Managing Finances

To effectively [[overview_and_transaction_management_in_notion | manage transactions]] and finances in [[overview_and_transaction_management_in_notion | Notion]] <a class="yt-timestamp" data-t="00:09:53">[00:09:53]</a>:
1.  Begin by setting up different accounts under various heads in the "Start Here" section <a class="yt-timestamp" data-t="00:09:58">[00:09:58]</a>.
2.  Add opening balances for relevant assets and liabilities <a class="yt-timestamp" data-t="00:10:04">[00:10:04]</a>.
3.  Proceed to enter details for every business transaction in the transactions window <a class="yt-timestamp" data-t="00:10:10">[00:10:10]</a>.
This process automatically updates the summary section, providing an immediate overview of business finances <a class="yt-timestamp" data-t="00:10:15">[00:10:15]</a>.
