## Operational Management Section

This section of the Asset Planner focuses on core asset information and ongoing activities <a class="yt-timestamp" data-t="00:00:08">[00:00:08]</a>.

### Asset Details
This sub-section allows for the detailed categorization and tracking of individual assets <a class="yt-timestamp" data-t="00:00:10">[00:00:10]</a>. Key information captured includes:
*   Asset ID <a class="yt-timestamp" data-t="00:00:18">[00:00:18]</a>
*   Asset Name <a class="yt-timestamp" data-t="00:00:18">[00:00:18]</a>
*   Asset Type <a class="yt-timestamp" data-t="00:00:19">[00:00:19]</a>
And many other categorized details <a class="yt-timestamp" data-t="00:00:20">[00:00:20]</a>.

### Maintenance Details
This area is dedicated to [[maintenance_tracking_for_assets | maintenance tracking for assets]] <a class="yt-timestamp" data-t="00:00:12">[00:00:12]</a>. It records all necessary information pertaining to asset upkeep, such as:
*   Maintenance ID <a class="yt-timestamp" data-t="00:00:27">[00:00:27]</a>
*   Asset ID <a class="yt-timestamp" data-t="00:00:27">[00:00:27]</a>
*   All other maintenance-related details <a class="yt-timestamp" data-t="00:00:30">[00:00:30]</a>

### Financial Details
The financial aspect of assets is managed here, supporting the [[financial_management_of_assets | financial management of assets]] <a class="yt-timestamp" data-t="00:00:14">[00:00:14]</a>. It tracks critical financial data:
*   Current Value <a class="yt-timestamp" data-t="00:00:33">[00:00:33]</a>

### Performance
This section provides insights into [[asset_performance_analysis | asset performance analysis]] as assets are created and utilized <a class="yt-timestamp" data-t="00:00:37">[00:00:37]</a>.
