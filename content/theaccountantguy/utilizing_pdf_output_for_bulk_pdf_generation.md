This article details how to use PDF Output to generate sales receipts and other PDF documents, focusing on both single and bulk generation methods.

## Accessing PDF Output <a class="yt-timestamp" data-t="03:08:00">[03:08:00]</a>

To begin, navigate to [[using_pdfoutputcom_for_bulk_pdf_generation | PDF output.com]] <a class="yt-timestamp" data-t="03:08:00">[03:08:00]</a>. Upon arrival, you'll find a template selection dropdown at the top of the page <a class="yt-timestamp" data-t="03:13:00">[03:13:00]</a>. Here, you can search for and select the specific template you need, such as the "Sales Receipts" template <a class="yt-timestamp" data-t="03:21:00">[03:21:00]</a>.

## [[generating_single_and_bulk_pdf_documents | Generating Single PDF Documents]] <a class="yt-timestamp" data-t="02:57:00">[02:57:00]</a>

Once the sales receipt template is open, you can fill in the required information <a class="yt-timestamp" data-t="03:30:00">[03:30:00]</a>. This includes details like:
*   Receipt number <a class="yt-timestamp" data-t="03:48:00">[03:48:00]</a>
*   Receipt date <a class="yt-timestamp" data-t="03:50:00">[03:50:00]</a>
*   Payment method (selectable from a dropdown) <a class="yt-timestamp" data-t="03:50:00">[03:50:00]</a>
*   Business name and address <a class="yt-timestamp" data-t="03:51:00">[03:51:00]</a>
*   Item descriptions, quantities, and unit prices <a class="yt-timestamp" data-t="03:55:00">[03:55:00]</a>
*   Tax rate <a class="yt-timestamp" data-t="04:26:00">[04:26:00]</a>
*   Notes <a class="yt-timestamp" data-t="04:36:00">[04:36:00]</a>

You can also select the currency for the document, such as US Dollars or Euro <a class="yt-timestamp" data-t="04:40:00">[04:40:00]</a>. After entering all details, click "Download PDF" to generate the document <a class="yt-timestamp" data-t="04:54:00">[04:54:00]</a>.

## [[using_pdfoutput_for_bulk_document_generation | Utilizing PDF Output for Bulk Document Generation]] <a class="yt-timestamp" data-t="05:07:00">[05:07:00]</a>

[[automating_document_exports_using_pdf_output | PDF Output also supports exporting PDF documents in bulk]] <a class="yt-timestamp" data-t="05:07:00">[05:07:00]</a>.

### Bulk Export Mode <a class="yt-timestamp" data-t="05:10:00">[05:10:00]</a>

To use this feature:
1.  Select the "bulk export mode document" option <a class="yt-timestamp" data-t="05:10:00">[05:10:00]</a>.
2.  Add all necessary information in a database-like format within the tool <a class="yt-timestamp" data-t="05:15:00">[05:15:00]</a>. This allows for entering multiple sales receipts simultaneously <a class="yt-timestamp" data-t="05:31:00">[05:31:00]</a>.
    *   You can resize column widths and row heights for better readability <a class="yt-timestamp" data-t="05:42:00">[05:42:00]</a>.
    *   New rows can be added using the "add new row" button <a class="yt-timestamp" data-t="05:58:00">[05:58:00]</a>.
    *   Individual rows or items can be deleted <a class="yt-timestamp" data-t="06:03:00">[06:03:00]</a>.
3.  Once all data is entered, you have two options for downloading <a class="yt-timestamp" data-t="06:11:00">[06:11:00]</a>:
    *   Click "Download all PDF" to get all documents in one go <a class="yt-timestamp" data-t="06:12:00">[06:12:00]</a>.
    *   Click on individual rows to download specific PDFs <a class="yt-timestamp" data-t="06:16:00">[06:16:00]</a>.

### Currency Customization in Bulk <a class="yt-timestamp" data-t="07:05:00">[07:05:00]</a>

Similar to single PDF generation, the currency can be changed for all bulk-generated documents <a class="yt-timestamp" data-t="07:05:00">[07:05:00]</a>. Changing the currency and clicking "Download PDF" again will re-generate all documents with the updated currency <a class="yt-timestamp" data-t="07:13:00">[07:13:00]</a>.
