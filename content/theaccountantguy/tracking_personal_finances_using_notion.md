This guide explains how to track personal finances using a dedicated Notion setup, covering various aspects from income and expenses to debt, savings, and investments. The system is designed to provide a comprehensive overview of financial health at a glance <a class="yt-timestamp" data-t="00:00:10">[00:00:10]</a>.

## System Sections

The Notion tracker is divided into several key sections to organize financial data:
*   **Net Worth Target:** Allows users to specify a net worth goal for the financial year <a class="yt-timestamp" data-t="00:00:19">[00:00:19]</a>.
*   **Account Section:** Where different accounts for income, expenses, assets, and liabilities are specified, including opening balances, payment frequencies, and categories <a class="yt-timestamp" data-t="00:00:23">[00:00:23]</a>.
*   **Summary Section:** Provides a quick overview of all finances <a class="yt-timestamp" data-t="00:00:38">[00:00:38]</a>.
*   **Debt Section:** Displays upcoming and past debt payments, including next due dates, total debt, and amount paid <a class="yt-timestamp" data-t="00:00:52">[00:00:52]</a>.
*   **Budget Section:** Enables setting budgets for estimated income and expenses <a class="yt-timestamp" data-t="00:01:08">[00:01:08]</a>.
*   **Savings Section:** For tracking personal savings and associated goals <a class="yt-timestamp" data-t="00:01:15">[00:01:15]</a>.
*   **Bill Section:** Shows upcoming and past bill payments <a class="yt-timestamp" data-t="00:01:23">[00:01:23]</a>.
*   **Subscription Section:** Tracks subscription payments with similar principles to bills and debt <a class="yt-timestamp" data-t="00:01:32">[00:01:32]</a>.
*   **Net Worth Section:** Offers an overview of current net worth and progress towards the target <a class="yt-timestamp" data-t="00:01:45">[00:01:45]</a>.
*   **Transaction Section:** The primary area for recording all income, expenses, asset, and liability transactions that affect other sections <a class="yt-timestamp" data-t="00:01:51">[00:01:51]</a>.

## Detailed Usage Guide

### [[tracking_income_and_expenses_using_notion | Tracking Income and Expenses]]

1.  **Set Up Accounts:**
    *   Navigate to the "Account Section" under "Get Started" <a class="yt-timestamp" data-t="02:36:00">[02:36:00]</a>.
    *   Find existing income (e.g., Salary, Side Hustle) and expenses (e.g., Groceries, Utility Bills, Netflix) <a class="yt-timestamp" data-t="02:40:00">[02:40:00]</a>.
    *   Click "Edit" to modify existing accounts or "New" to add new ones <a class="yt-timestamp" data-t="02:55:00">[02:55:00]</a>.
    *   When adding expenses, classify them (e.g., "Bills", "Subscriptions") and set a payment frequency (e.g., "Monthly", "Annually") <a class="yt-timestamp" data-t="03:31:00">[03:31:00]</a>.
    *   Income and expense accounts typically do not have an opening balance <a class="yt-timestamp" data-t="04:16:00">[04:16:00]</a>.

2.  **Enter Transactions:**
    *   Go to the "Transactions" view <a class="yt-timestamp" data-t="04:54:00">[04:54:00]</a>.
    *   Select the "Income" or "Expense" category view <a class="yt-timestamp" data-t="05:30:00">[05:30:00]</a>.
    *   Click "New" to add a transaction <a class="yt-timestamp" data-t="05:38:00">[05:38:00]</a>.
    *   Enter the date, description, relevant account, payment mode (bank/cash), and amount <a class="yt-timestamp" data-t="05:43:00">[05:43:00]</a>.
    *   For income, set the transaction type to "Inflow"; for expenses, set it to "Outflow" <a class="yt-timestamp" data-t="06:46:00">[06:46:00]</a>.

3.  **View Summary:**
    *   Navigate to the "Summary" section, then "Finances" view <a class="yt-timestamp" data-t="07:47:00">[07:47:00]</a>.
    *   This view displays total income and expenses, along with the percentage contribution of each account <a class="yt-timestamp" data-t="07:56:00">[07:56:00]</a>.
    *   Changes in transaction figures automatically update the summary <a class="yt-timestamp" data-t="08:39:00">[08:39:00]</a>.

### [[using_notion_for_debt_tracking | Tracking Debt]]

1.  **Set Up Debt Accounts:**
    *   Go to the "Account Section" and navigate to the "Liabilities" part <a class="yt-timestamp" data-t="09:11:00">[09:11:00]</a>.
    *   Add new debt accounts (e.g., Home Loan, Car Loan, Credit Card) by clicking "New" <a class="yt-timestamp" data-t="09:27:00">[09:27:00]</a>.
    *   Specify the name, opening balance (if any), classify as "Debt", and set a payment frequency (e.g., "Monthly") <a class="yt-timestamp" data-t="09:31:00">[09:31:00]</a>.

2.  **Enter Debt Transactions:**
    *   Go to the "Transactions" section and select "Liability" <a class="yt-timestamp" data-t="10:35:00">[10:35:00]</a>.
    *   Click "New" to add a payment <a class="yt-timestamp" data-t="10:53:00">[10:53:00]</a>.
    *   Enter date, account, payment mode, and amount <a class="yt-timestamp" data-t="10:55:00">[10:55:00]</a>.
    *   For repayments, set the transaction type to "Outflow" <a class="yt-timestamp" data-t="11:30:00">[11:30:00]</a>.
    *   If obtaining a new loan, set the transaction type to "Inflow" <a class="yt-timestamp" data-t="11:59:00">[11:59:00]</a>.

3.  **Track Debt Summary:**
    *   Under the "Summary" section, go to "Debt" view <a class="yt-timestamp" data-t="13:21:00">[13:21:00]</a>.
    *   It offers "Upcoming Payments" and "Past Payments" views <a class="yt-timestamp" data-t="13:27:00">[13:27:00]</a>.
    *   "Upcoming Payments" shows future due dates, while "Past Payments" highlights missed or overdue payments <a class="yt-timestamp" data-t="13:31:00">[13:31:00]</a>.
    *   The view also displays total loan amount, repayment percentage, and remaining amount <a class="yt-timestamp" data-t="15:06:00">[15:06:00]</a>.
    *   Users can change the layout (e.g., list, gallery) and visible properties <a class="yt-timestamp" data-t="15:30:00">[15:30:00]</a>.

### [[creating_a_budget_tracker_with_notion | Creating a Budget Tracker with Notion]]

1.  **Access Budget Section:**
    *   After entering income and expense transactions, go to the "Summary" view and click on "Budget" <a class="yt-timestamp" data-t="17:12:00">[17:12:00]</a>.
2.  **Set Estimated Amounts:**
    *   This view shows incurred expenses and income, along with the percentage of the total budgeted amount <a class="yt-timestamp" data-t="17:22:00">[17:22:00]</a>.
    *   Click on the budget amount field for any expense or income category and enter an estimated value <a class="yt-timestamp" data-t="17:39:00">[17:39:00]</a>. The system automatically updates percentages and reflects overages <a class="yt-timestamp" data-t="18:00:00">[18:00:00]</a>.

### [[creating_and_tracking_budgets_and_savings_in_notion | Tracking Savings]]

1.  **Set Up Savings Funds:**
    *   Navigate to the "Account Section" under "Get Started" and go to the "Asset" part <a class="yt-timestamp" data-t="19:07:00">[19:07:00]</a>.
    *   Create new savings funds (e.g., Education Fund, Car Fund) <a class="yt-timestamp" data-t="19:18:00">[19:18:00]</a>.
    *   Classify these accounts under the "Savings" category <a class="yt-timestamp" data-t="19:23:00">[19:23:00]</a>.
    *   Specify a "Goal Amount" for each saving fund in the account's edit panel <a class="yt-timestamp" data-t="22:38:00">[22:38:00]</a>.

2.  **Add Savings Transactions:**
    *   Go to the "Transactions" section and select "Asset" <a class="yt-timestamp" data-t="23:09:00">[23:09:00]</a>.
    *   Create a new transaction for savings <a class="yt-timestamp" data-t="23:16:00">[23:16:00]</a>.
    *   Enter details like description, savings account, payment mode, and amount <a class="yt-timestamp" data-t="23:23:00">[23:23:00]</a>.
    *   The transaction type will typically be "Outflow" as money moves from a bank account to the savings fund <a class="yt-timestamp" data-t="23:42:00">[23:42:00]</a>.

3.  **View Savings Overview:**
    *   The "Savings" section in the summary provides an updated overview of saved amounts, goal amounts, and percentage progress <a class="yt-timestamp" data-t="20:11:00">[20:11:00]</a>.

### [[tracking_expenses_with_notion | Tracking Bills]]

1.  **Set Up Bill Accounts:**
    *   Go to the "Account Section" under "Get Started" and locate "Expenses" <a class="yt-timestamp" data-t="25:10:00">[25:10:00]</a>.
    *   Add or modify expenses, classifying them as "Bills" and setting their payment frequency (e.g., "Monthly") <a class="yt-timestamp" data-t="25:16:00">[25:16:00]</a>.

2.  **Enter Bill Transactions:**
    *   Navigate to the "Expenses" part under "Transactions" <a class="yt-timestamp" data-t="26:01:00">[26:01:00]</a>.
    *   Enter the date, description, bill account, payment mode, and amount <a class="yt-timestamp" data-t="26:08:00">[26:26:08]</a>.
    *   The transaction type is "Outflow" <a class="yt-timestamp" data-t="26:16:00">[26:16:00]</a>.

3.  **View Bill Overview:**
    *   In the "Summary" section, select the "Bills" view <a class="yt-timestamp" data-t="26:24:00">[26:24:00]</a>.
    *   It displays "Upcoming Payments" and "Past Payments" based on the last payment date and frequency <a class="yt-timestamp" data-t="26:28:00">[26:28:00]</a>.

### Tracking Subscriptions

1.  **Set Up Subscription Accounts:**
    *   Similar to bills, go to the "Account Section" -> "Expenses" <a class="yt-timestamp" data-t="29:13:00">[29:13:00]</a>.
    *   Add or modify expenses, classifying them as "Subscription" and setting their payment frequency (e.g., "Annual", "Quarterly") <a class="yt-timestamp" data-t="29:18:00">[29:18:00]</a>.

2.  **Enter Subscription Transactions:**
    *   Navigate to the "Expenses" part under "Transactions" <a class="yt-timestamp" data-t="30:09:00">[30:09:00]</a>.
    *   Enter the date, description, subscription account, payment mode, and amount <a class="yt-timestamp" data-t="30:17:00">[30:17:00]</a>.
    *   The transaction type is "Outflow" <a class="yt-timestamp" data-t="30:47:00">[30:47:00]</a>.

3.  **View Subscription Overview:**
    *   In the "Summary" section, select the "Subscription" view <a class="yt-timestamp" data-t="31:02:00">[31:02:00]</a>.
    *   It shows "Upcoming Payments" and "Past Payments" for subscriptions, along with their status (e.g., active) <a class="yt-timestamp" data-t="31:06:00">[31:06:00]</a>.

### [[using_notion_to_track_net_worth | Tracking Net Worth]]

1.  **Set Net Worth Goal:**
    *   Go to the "Net Worth Goal" section and specify the target net worth for the financial year <a class="yt-timestamp" data-t="33:05:00">[33:05:00]</a>.

2.  **View Net Worth:**
    *   Navigate to the "Net Worth" section under "Summary" <a class="yt-timestamp" data-t="33:29:00">[33:29:00]</a>.
    *   It displays the "Net Worth Goal" and the "Total Net Worth" (Assets - Liabilities) <a class="yt-timestamp" data-t="33:36:00">[33:36:00]</a>.
    *   The "Difference" shows how much is remaining to reach the goal <a class="yt-timestamp" data-t="33:57:00">[00:33:57]</a>.
    *   "Total Growth" reflects the change from opening net worth (based on opening balances of assets and liabilities) to the closing net worth <a class="yt-timestamp" data-t="34:20:00">[34:20:00]</a>.

### Tracking Investments

1.  **Set Up Investment Accounts:**
    *   Go to the "Accounts" tab and create new investment accounts (e.g., Apple Investment, Bitcoin) <a class="yt-timestamp" data-t="38:19:00">[38:19:00]</a>.
    *   Specify an opening balance and opening units if applicable <a class="yt-timestamp" data-t="39:05:00">[39:05:00]</a>.
    *   Classify them (e.g., "Stocks", "Crypto") and, crucially, select "Investments" as the Asset Class <a class="yt-timestamp" data-t="39:22:00">[39:22:00]</a>.

2.  **Enter Investment Transactions:**
    *   Under the "Assets" view in "Transactions", record purchase and sale transactions <a class="yt-timestamp" data-t="40:37:00">[40:37:00]</a>.
    *   **Important:**
        *   Purchases should be recorded at the **cost price** in the amount field <a class="yt-timestamp" data-t="41:14:00">[41:14:00]</a>.
        *   Sales should also be recorded at the **cost value** of the units being sold, not the sale price <a class="yt-timestamp" data-t="42:01:00">[42:01:00]</a>. This ensures proper calculation of profit or loss <a class="yt-timestamp" data-t="42:25:00">[42:25:00]</a>.

3.  **View Investment Summary:**
    *   The "Investment" tab shows total investment amounts, units available, and calculated profit or loss for each investment <a class="yt-timestamp" data-t="37:00:00">[37:00:00]</a>.

## Additional Tips

*   **Couples:** For couples, accounts can be split by individual (e.g., "Salary by Him", "Side Hustle by Her") to track finances separately within the same system <a class="yt-timestamp" data-t="35:47:00">[35:47:00]</a>.
*   **Specific Account Transactions:** To view transactions for a particular account (e.g., only "Salary" transactions), click on the account name in the summary or finances view. A "View Transactions" option will appear at the bottom, populating all related transactions <a class="yt-timestamp" data-t="45:15:00">[45:15:00]</a>.
*   **Multi-Device & Free to Use:** The Notion template is available on all devices (iOS, Android, Windows, Mac OS) <a class="yt-timestamp" data-t="47:44:00">[47:44:00]</a>. Notion itself is a free tool; only a free email ID is required to sign up <a class="yt-timestamp" data-t="48:14:00">[48:14:00]</a>.
*   **Multiple Currencies:** Notion supports various currencies. To change the currency, go to any amount field, click "Edit property", then "Number format", and select the desired currency (e.g., Euro) <a class="yt-timestamp" data-t="48:43:00">[48:43:00]</a>. This must be done for all relevant views <a class="yt-timestamp" data-t="49:26:00">[49:26:00]</a>.
