## Tracker Sections Overview

The credit card expenses tracker features several key sections for comprehensive financial management:

### Credit Cards Overview
This section displays all added credit cards, showing the total amount due, the amount repaid so far, and the remaining balance due for each card <a class="yt-timestamp" data-t="00:00:23">[00:00:23]</a>. Example credit cards include Student Credit Card, Cash Back Credit Card, Travel Credit Card, and Rewards Credit Card <a class="yt-timestamp" data-t="00:00:18">[00:00:18]</a>. Users can edit existing credit card information or add new ones <a class="yt-timestamp" data-t="00:05:16">[00:05:16]</a>.

### Monthly Overview
A monthly breakdown shows the amount due, amount repaid, and the balance amount due for each month <a class="yt-timestamp" data-t="00:00:32">[00:00:32]</a>. This view can be customized as per individual requirements <a class="yt-timestamp" data-t="00:00:37">[00:00:37]</a>.

### Yearly Overview
Similar to the monthly overview, this section presents the total amount due, amount repaid, and balance amount due on a yearly basis <a class="yt-timestamp" data-t="00:00:41">[00:00:41]</a>.

### Expense Detail Section
This is where all individual expenses are recorded in detail <a class="yt-timestamp" data-t="00:00:51">[00:00:51]</a>. It's the primary area for booking new expenses <a class="yt-timestamp" data-t="00:01:37">[00:01:37]</a>.

### Expenses Overview
This section provides a summary of all booked expenses categorized by type <a class="yt-timestamp" data-t="00:01:00">[00:01:00]</a>. It shows the total expenses added, the budgeted amount, the comparison between expenses and budget, and whether the budget is "under budget" or "over budget" <a class="yt-timestamp" data-t="00:01:05">[00:01:05]</a>.

### Notification Section
This feature shows when the next repayment is due and how many days are remaining until that date for each added credit card <a class="yt-timestamp" data-t="00:01:16">[00:01:16]</a>. This helps users pay credit card debts on time <a class="yt-timestamp" data-t="00:03:26">[00:03:26]</a>.

## How to Use the Tracker

### Booking Expenses
To begin [[creating_a_notion_expense_tracker | tracking income and expenses using Notion]], users must book expenses under the "Expense Details" section <a class="yt-timestamp" data-t="00:01:37">[00:01:37]</a>:
*   **Date:** Enter the date of the expense <a class="yt-timestamp" data-t="00:01:42">[00:01:42]</a>.
*   **Expense Details:** Describe the expense <a class="yt-timestamp" data-t="00:01:44">[00:01:44]</a>.
*   **Period:** Select the relevant period (e.g., month) <a class="yt-timestamp" data-t="00:01:46">[00:01:46]</a>.
*   **Year:** Enter the year of the expense <a class="yt-timestamp" data-t="00:01:49">[00:01:49]</a>.
*   **Credit Card:** Link the expense to a specific credit card. New credit cards can be created as needed <a class="yt-timestamp" data-t="00:01:51">[00:01:51]</a>.
*   **Category:** Assign a category to each expense <a class="yt-timestamp" data-t="00:01:54">[00:01:54]</a>. New categories can be added <a class="yt-timestamp" data-t="00:05:24">[00:05:24]</a>.
*   **Expense Amount:** Input the monetary value of the expense <a class="yt-timestamp" data-t="00:01:57">[00:01:57]</a>.
*   **Opening Balance:** For any balances carried forward from previous months, input them as an opening balance <a class="yt-timestamp" data-t="00:02:08">[00:02:08]</a>.
*   **Repayment Section:** When repaying an amount for a credit card, enter it here <a class="yt-timestamp" data-t="00:02:00">[00:02:00]</a>.

As details are entered, all information in the overview sections updates automatically <a class="yt-timestamp" data-t="00:03:07">[00:03:07]</a>.

### Setting Budgets and Notifications
*   **Setting Budgets:** In the "Expenses Overview" section, users can set a budgeted amount for each expense category <a class="yt-timestamp" data-t="00:03:56">[00:03:56]</a>. The tracker will then compare incurred expenses to the budget and indicate whether the spending is "under budget" or "over budget" <a class="yt-timestamp" data-t="00:04:19">[00:04:19]</a>.
*   **Setting Repayment Dates:** In the "Notification" section, users can set their next repayment dates <a class="yt-timestamp" data-t="00:04:35">[00:04:35]</a>. The tracker will then display how many days are pending until the next repayment date <a class="yt-timestamp" data-t="00:04:49">[00:04:49]</a>. This feature is crucial for [[using_notion_for_debt_tracking | tracking income and expenses in Notion]] and paying debts on time <a class="yt-timestamp" data-t="00:05:05">[00:05:05]</a>.

## Additional Features and Views

The tracker provides several ways to review and summarize [[expense_tracking_in_notion | expense tracking in Notion]]:

*   **Receipt Column:** In the "Expense Details" section, users can attach bills or receipts for expenses in the receipt column <a class="yt-timestamp" data-t="00:05:34">[00:05:34]</a>.
*   **Summarization Views:**
    *   **Period View:** Clicking on a particular period opens a new window showing transaction details like date, period, transaction details, amount, and a tag (green for repayment, regular for expense) <a class="yt-timestamp" data-t="00:05:44">[00:05:44]</a>.
    *   **Year View:** Similar information is available for a specific year <a class="yt-timestamp" data-t="00:06:16">[00:06:16]</a>.
    *   **Credit Card View:** Clicking on a specific credit card displays all transactions related to it, including status and assigned categories <a class="yt-timestamp" data-t="00:06:21">[00:06:21]</a>.
    *   **Category View:** Selecting an expense category provides details of all expenses incurred under that category <a class="yt-timestamp" data-t="00:06:48">[00:06:48]</a>.
