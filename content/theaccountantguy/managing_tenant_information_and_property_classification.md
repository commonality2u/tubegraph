The [[notion_template_for_tracking_house_rent_payments | House Rent Payments Tracker template]] can be utilized to keep track of house rental payments and manage tenant and property information efficiently <a class="yt-timestamp" data-t="00:00:04">[00:00:04]</a>.

## Template Overview

The template includes a summary section with four different views <a class="yt-timestamp" data-t="00:00:10">[00:00:10]</a>. This section allows for the addition of basic information, including tenants, classified under the properties owned <a class="yt-timestamp" data-t="00:00:13">[00:00:13]</a>.

## Property Classification

The template supports classifying properties. Examples provided include "apartment" and "duplex" <a class="yt-timestamp" data-t="00:00:20">[00:00:20]</a>. Users can add any other property classifications as needed <a class="yt-timestamp" data-t="00:00:23">[00:00:23]</a>.

## Tenant Management

Tenants can be added and managed within the template <a class="yt-timestamp" data-t="00:00:25">[00:00:25]</a>:

*   **Adding a Tenant**: To add a new tenant, click the "ADD tenant" button on the left <a class="yt-timestamp" data-t="00:00:28">[00:00:28]</a>. This action automatically creates a new tenant entry <a class="yt-timestamp" data-t="00:00:30">[00:00:30]</a>.
*   **Assigning to Categories**: The new tenant entry can be dragged and placed under the respective property category by clicking on its three-dot icon <a class="yt-timestamp" data-t="00:00:33">[00:00:33]</a>.
*   **Adding Tenant Details**: To add specific details for a tenant, click the pencil icon associated with their entry <a class="yt-timestamp" data-t="00:00:37">[00:00:37]</a>.

## Property Section Details

The "property section" view within the template displays different classifications of properties <a class="yt-timestamp" data-t="00:00:46">[00:00:46]</a>. Underneath each property, the name of the tenant is visible along with key financial information <a class="yt-timestamp" data-t="00:00:49">[00:00:49]</a>:

*   Total rent amount <a class="yt-timestamp" data-t="00:00:54">[00:00:54]</a>
*   Rent amount that has been paid <a class="yt-timestamp" data-t="00:00:55">[00:00:55]</a>
*   Balance rent amount due <a class="yt-timestamp" data-t="00:00:57">[00:00:57]</a>
*   Advanced rent paid <a class="yt-timestamp" data-t="00:00:58">[00:00:58]</a>

## Related Features

*   **[[tracking_rental_history_and_transaction_summaries | Rental History]]**: The "rental history" section shows the rental history of all tenants based on recorded transactions <a class="yt-timestamp" data-t="00:01:04">[00:01:04]</a>.
*   **Summary Section**: A general summary section provides totals for amount due, paid amount, outstanding rent, and the percentage of rent collected <a class="yt-timestamp" data-t="00:01:12">[00:01:12]</a>.
*   **Transaction Section**: This section offers views for rent paid, rent due, and a quarterly summary, enabling [[viewing_and_updating_rental_payment_details | viewing and updating rental payment details]] <a class="yt-timestamp" data-t="00:01:20">[00:01:20]</a>.
