A debt payment tracker can be utilized to pay off all your debts efficiently <a class="yt-timestamp" data-t="00:00:07">[00:00:07]</a>. This tool is designed to help you organize and [[payment_tracking_and_debt_progression | track your debt payments effectively]] <a class="yt-timestamp" data-t="00:00:07">[00:00:07]</a>.

## Debt Overview Section

The [[debt_overview_and_classification | debt overview]] section provides a comprehensive breakdown of your debts, categorized for clarity <a class="yt-timestamp" data-t="00:00:11">[00:00:11]</a>. Debts are typically divided into categories such as annual and monthly payments <a class="yt-timestamp" data-t="00:00:17">[00:00:17]</a>.

Key columns in this section include:
*   **Debt Details**: Specifics of the debt <a class="yt-timestamp" data-t="00:00:22">[00:00:22]</a>.
*   **Loan Amount**: The total amount of the loan <a class="yt-timestamp" data-t="00:00:24">[00:00:24]</a>.
*   **EMI Amount**: The amount to be paid for each installment <a class="yt-timestamp" data-t="00:00:26">[00:00:26]</a>.
*   **Interest Rate**: The specified interest rate <a class="yt-timestamp" data-t="00:00:29">[00:00:29]</a>.
*   **Initial Repayment**: The first installment payment date <a class="yt-timestamp" data-t="00:00:33">[00:00:33]</a>.
*   **Debt Free By**: A calculated field showing when you'll be debt-free if payments continue at the specified amount and interest rate <a class="yt-timestamp" data-t="00:00:38">[00:00:38]</a>.
*   **Number of EMIs**: The total number of installments required to pay off the debt <a class="yt-timestamp" data-t="00:00:49">[00:00:49]</a>.

### Managing Debt Categories

You can add new debt categories (e.g., weekly, quarterly) by clicking an "add a group" button <a class="yt-timestamp" data-t="00:00:54">[00:00:54]</a>. Pre-added payment schedules like monthly, annual, quarterly, and weekly are also available for reference <a class="yt-timestamp" data-t="00:01:13">[00:01:13]</a>. Debts can be moved between different repayment schedules, which automatically updates the repayment schedule and other relevant calculations <a class="yt-timestamp" data-t="00:01:27">[00:01:27]</a>.

### Adding and Editing Debts

*   To add a new debt, click the "added details" button, which initially places it in a "no payment frequency" column <a class="yt-timestamp" data-t="00:02:43">[00:02:43]</a>. You can then move it to the appropriate frequency and input details <a class="yt-timestamp" data-t="00:02:51">[00:02:51]</a>.
*   To edit existing loan details, click on the debt field and type in changes, which automatically update across all linked columns and descriptions <a class="yt-timestamp" data-t="00:02:59">[00:02:59]</a>.

## Debt Payoff Progress

This section reflects your overall progress in paying off debts <a class="yt-timestamp" data-t="00:01:48">[00:01:48]</a>. It integrates data from your initial [[debt_overview_and_classification | debt categories]] and individual payment transactions <a class="yt-timestamp" data-t="00:01:51">[00:01:51]</a>.

Key metrics displayed include:
*   **Total Loans**: All loans added to the tracker <a class="yt-timestamp" data-t="00:01:53">[00:01:53]</a>.
*   **Total Amount Repaid**: The sum of all EMI amounts paid <a class="yt-timestamp" data-t="00:01:57">[00:01:57]</a>. This updates automatically as you mark installments as paid <a class="yt-timestamp" data-t="00:02:05">[00:02:05]</a>.
*   **Net Payoff in Percentage**: The percentage of the total debt paid off so far <a class="yt-timestamp" data-t="00:02:12">[00:02:12]</a>.
*   **Debt Free By**: A quick reference to the estimated debt-free date <a class="yt-timestamp" data-t="00:02:17">[00:02:17]</a>.
*   **Interest Amount**: The total interest paid as part of your repayments <a class="yt-timestamp" data-t="00:02:23">[00:02:23]</a>.
*   **Principal Amount**: The total principal amount repaid <a class="yt-timestamp" data-t="00:02:30">[00:02:30]</a>. Every debt repayment is split into interest and principal components <a class="yt-timestamp" data-t="00:02:36">[00:02:36]</a>.

## Transaction Section

The transaction section allows you to [[tracking_debt_payments_effectively | track debt payments]] you've made <a class="yt-timestamp" data-t="00:03:13">[00:03:13]</a>.

### Paid Installments

This area lists all paid debts, bifurcated by month, with the latest month appearing at the top <a class="yt-timestamp" data-t="00:03:19">[00:03:19]</a>. A summation of all EMIs paid for a particular month is also displayed <a class="yt-timestamp" data-t="00:03:27">[00:03:27]</a>.

Each payment entry includes:
*   **Paid Status**: A checkbox to mark if the amount has been paid <a class="yt-timestamp" data-t="00:03:42">[00:03:42]</a>. Ticking or unticking this instantly updates the paid or unpaid sections <a class="yt-timestamp" data-t="00:03:45">[00:03:45]</a>.
*   **Payment Date**: The date the payment was made <a class="yt-timestamp" data-t="00:03:57">[00:03:57]</a>.
*   **Loan Details**: A dropdown to select the specific loan associated with the payment <a class="yt-timestamp" data-t="00:04:01">[00:04:01]</a>.
*   **EMI Amount of Payment**: The specific amount paid for that installment <a class="yt-timestamp" data-t="00:04:08">[00:04:08]</a>.
*   **Next Repayment Schedule**: Automatically calculated based on the loan's set payment frequency (e.g., monthly payments result in a next payment date exactly one month later) <a class="yt-timestamp" data-t="00:04:12">[00:04:12]</a>. For annual loans, the next repayment will be on the same date the following year <a class="yt-timestamp" data-t="00:04:30">[00:04:30]</a>.

### Adding New Debt Payments

You can add a new debt payment to the schedule by clicking a "pay debt amount" button, which automatically populates with the current date <a class="yt-timestamp" data-t="00:04:51">[00:04:51]</a>. You can then input all the necessary details and descriptions <a class="yt-timestamp" data-t="00:05:02">[00:05:02]</a>. All sections, including the debt payoff progress, update automatically as details are added or changed <a class="yt-timestamp" data-t="00:05:16">[00:05:16]</a>.
