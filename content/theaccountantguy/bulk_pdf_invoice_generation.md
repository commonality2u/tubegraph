[[generating_pdf_invoices_using_pdfoutput | PDF Output]] offers a solution for [[bulk_pdf_document_creation | generating PDF documents]] in bulk, including invoices <a class="yt-timestamp" data-t="03:07:00">[03:07:00]</a>. This feature is accessible through the "bulk PDF mode" on the PDF Output website <a class="yt-timestamp" data-t="03:34:36">[03:34:36]</a>.

## How to [[utilizing_pdf_output_for_bulk_pdf_generation | Generate Bulk PDF Invoices]]

To [[generating_single_and_bulk_pdf_documents | generate multiple invoice PDFs]] simultaneously:
1.  Navigate to the [[generating_pdf_invoices_using_pdfoutput | PDF Output]] website <a class="yt-timestamp" data-t="03:11:00">[03:11:00]</a>.
2.  Once logged in, locate the "bulk PDF mode" option <a class="yt-timestamp" data-t="03:34:36">[03:34:36]</a>.
3.  Fill in the invoice details row by row, copying and pasting information as needed <a class="yt-timestamp" data-t="04:46:00">[04:46:00]</a>.
    *   New rows can be added by clicking "add new row" to accommodate additional invoices <a class="yt-timestamp" data-t="05:04:00">[05:04:00]</a>.
    *   Column widths can be expanded for better visibility <a class="yt-timestamp" data-t="04:52:00">[04:52:00]</a>.
4.  After entering all details, click "download all PDF" to begin generating all the PDF documents one by one <a class="yt-timestamp" data-t="04:56:00">[04:56:00]</a>.

## Features

*   **Individual PDF Generation from Bulk Mode**
    From the bulk mode interface, it's also possible to [[generating_single_and_bulk_pdf_documents | generate a PDF for a single, specific row]] by clicking "download PDF" on that particular row <a class="yt-timestamp" data-t="05:13:00">[05:13:00]</a>.
