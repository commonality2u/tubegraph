## Single PDF Document Generation

Upon logging into PDFoutput.com, users are presented with an interface where they can select a specific template from a dropdown menu, such as an invoice template <a class="yt-timestamp" data-t="01:58:00">[01:58:00]</a>. After selecting a template, users can choose their preferred currency from several available options <a class="yt-timestamp" data-t="02:08:00">[02:08:00]</a>. Once the necessary details for the invoice are filled in, the PDF document is automatically populated with this information <a class="yt-timestamp" data-t="02:19:00">[02:19:00]</a>.

To [[generating_and_downloading_pdf_documents | generate and download the PDF]], users simply click the "download PDF" button <a class="yt-timestamp" data-t="02:50:00">[02:50:00]</a>. The system quickly produces a ready-to-use invoice in PDF format <a class="yt-timestamp" data-t="02:59:00">[02:59:00]</a>.

## Bulk PDF Document Generation

For scenarios requiring the export of multiple PDF documents, PDFoutput.com offers a "bulk export mode" <a class="yt-timestamp" data-t="03:17:00">[03:17:00]</a>. In this mode, users can input details into a database interface <a class="yt-timestamp" data-t="03:24:00">[03:24:00]</a>. This database includes fields mirroring those found in the single PDF mode, such as invoice number and date <a class="yt-timestamp" data-t="04:02:00">[04:02:00]</a>.

Once the database is populated, users can click "download all PDF" to [[generating_and_exporting_pdf_documents_for_business | generate PDF documents]] in bulk <a class="yt-timestamp" data-t="04:17:00">[04:17:00]</a>. The system processes the information row by row, generating individual PDF files <a class="yt-timestamp" data-t="04:24:00">[04:24:00]</a>. For example, three PDF documents can be generated based on the entered data <a class="yt-timestamp" data-t="04:31:00">[04:31:00]</a>. These bulk-generated PDFs maintain the same format as those created in single PDF mode <a class="yt-timestamp" data-t="04:34:00">[04:34:00]</a>.

Users also have the flexibility to export a PDF for a particular row individually, or to utilize the "download all PDF" option for a complete bulk export <a class="yt-timestamp" data-t="05:00:00">[05:00:00]</a>.
