## Utilizing PDFOutput for Bulk Export

The process begins by navigating to PDFOutput.com <a class="yt-timestamp" data-t="03:08:04">[03:08:04]</a>.

### Selecting the Template and Mode

1.  **Template Selection**: On the website, locate the "template selection" dropdown menu <a class="yt-timestamp" data-t="03:13:39">[03:13:39]</a>. From the list of available templates, search for and select "sales receipts" <a class="yt-timestamp" data-t="03:24:41">[03:24:41]</a>.
2.  **Bulk Export Mode**: After selecting the sales receipt template, choose the "bulk export mode document" option <a class="yt-timestamp" data-t="05:09:47">[05:09:47]</a>.

### Entering Sales Data

In the bulk export mode, all sales receipt information needs to be added in a database format <a class="yt-timestamp" data-t="05:15:37">[05:15:37]</a>. This database includes fields such as:
*   Receipt number <a class="yt-timestamp" data-t="05:34:25">[05:34:25]</a>
*   Receipt date <a class="yt-timestamp" data-t="05:34:25">[05:34:25]</a>
*   Payment method <a class="yt-timestamp" data-t="03:49:57">[03:49:57]</a>
*   Business name and address <a class="yt-timestamp" data-t="03:53:36">[03:53:36]</a>
*   Item descriptions <a class="yt-timestamp" data-t="03:56:56">[03:56:56]</a>
*   Quantities <a class="yt-timestamp" data-t="04:11:51">[04:11:51]</a>
*   Unit prices <a class="yt-timestamp" data-t="04:13:26">[04:13:26]</a>
*   Tax rate <a class="yt-timestamp" data-t="04:26:47">[04:26:47]</a>
*   Notes <a class="yt-timestamp" data-t="04:35:10">[04:35:10]</a>

The column widths and row heights of the database can be freely resized for better readability and data entry <a class="yt-timestamp" data-t="05:40:48">[05:40:48]</a>. New rows can be added using the "add new row" option, and existing rows or items can be deleted using the respective icons <a class="yt-timestamp" data-t="05:58:36">[05:58:36]</a>.

### [[exporting_and_downloading_bulkgenerated_pdfs | Downloading Bulk-Generated PDFs]]

Once all the information is entered into the database:
*   Clicking "download all PDF" will [[generating_pdf_documents_in_bulk | generate all PDFs in one go]] <a class="yt-timestamp" data-t="06:11:47">[06:11:47]</a>.
*   Individual rows can also be selected to download a specific PDF <a class="yt-timestamp" data-t="06:16:32">[06:16:32]</a>.

The system will start [[exporting_and_downloading_generated_pdf_documents | downloading the PDFs]] as a bundled file <a class="yt-timestamp" data-t="06:20:53">[06:20:53]</a>. The generated PDFs reflect the exact values entered in the database <a class="yt-timestamp" data-t="06:31:17">[06:31:17]</a>.

### Currency Options

The currency for the receipts can be selected from a dropdown menu, with options like US dollars or Euros. Changing the currency automatically updates it in the generated PDFs <a class="yt-timestamp" data-t="04:40:34">[04:40:34]</a>.
