The [[overview_of_notion_expense_tracker | Notion Expense Tracker]] is a template designed to help users manage all their expenses in one place <a class="yt-timestamp" data-t="00:00:03">[00:00:03]</a>. It is highly customizable to suit various needs <a class="yt-timestamp" data-t="00:02:06">[00:02:06]</a>. This template can be used for tracking both personal and business expenses <a class="yt-timestamp" data-t="00:01:21">[00:01:21]</a>.

## Key Features

### Categories Section
At the top of the tracker, there is a categories section <a class="yt-timestamp" data-t="00:00:17">[00:00:17]</a>. Users can define and customize categories based on their specific requirements <a class="yt-timestamp" data-t="00:00:22">[00:00:22]</a>.

*   **Renaming Categories**: Hovering over a category reveals a pencil icon, which allows renaming the category <a class="yt-timestamp" data-t="00:00:25">[00:00:25]</a>.
*   **Editing/Deleting Categories**: Clicking the three dots next to a category allows for editing or deleting it <a class="yt-timestamp" data-t="00:00:41">[00:00:41]</a>.
*   **Adding New Categories**: A new category can be added by clicking on the "new page" option <a class="yt-timestamp" data-t="00:00:48">[00:00:48]</a>.
*   **Budgeted vs. Actual Expenses**: For each category, there is a budgeted expense amount and an actual expense amount <a class="yt-timestamp" data-t="00:00:52">[00:00:52]</a>. The budgeted expense amount can be set by the user <a class="yt-timestamp" data-t="00:01:42">[00:01:42]</a>.

### Expense Listing
Below the categories, individual expenses are listed <a class="yt-timestamp" data-t="00:00:56">[00:00:56]</a>. As amounts are entered into the expense list, the corresponding categories at the top are automatically updated <a class="yt-timestamp" data-t="00:01:03">[00:01:03]</a>.

### Summary Section
The summary section provides an overview of financial tracking <a class="yt-timestamp" data-t="00:01:27">[00:01:27]</a>. It includes:
*   The total budgeted expense amount <a class="yt-timestamp" data-t="00:01:28">[00:01:28]</a>.
*   The total actual expenses amount, computed from the expense table <a class="yt-timestamp" data-t="00:01:30">[00:01:30]</a>.
*   The difference between budgeted and actual expenses <a class="yt-timestamp" data-t="00:01:36">[00:01:36]</a>.
*   The percentage of the budget that has been spent <a class="yt-timestamp" data-t="00:01:37">[00:01:37]</a>.

Changes made to budgeted figures in the category section are reflected in the summary section <a class="yt-timestamp" data-t="00:01:50">[00:01:50]</a>.
