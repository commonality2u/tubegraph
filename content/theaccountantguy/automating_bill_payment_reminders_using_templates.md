This guide demonstrates how to set up and automate bill payment reminders, specifically showing an example for [[setting_up_whatsapp_bill_payment_reminders | WhatsApp]] notifications <a class="yt-timestamp" data-t="00:00:06">[00:00:06]</a>. The system utilizes [[creating_and_customizing_reminder_templates in databases | templates]] to manage various bills, such as Wi-Fi or mobile bills <a class="yt-timestamp" data-t="00:00:10">[00:00:10]</a>.

## [[Setting up WhatsApp bill payment reminders | Setting Up WhatsApp Bill Payment Reminders]]

The process begins by using a pre-configured template interface. An example shows a Wi-Fi bill due on September 20th for $3,000 <a class="yt-timestamp" data-t="00:00:10">[00:00:10]</a>.

### [[Creating and customizing reminder templates in databases | Creating a New Bill Reminder Template]]

To create a new bill payment reminder:
1.  Click on the provided drop-down menu <a class="yt-timestamp" data-t="00:00:18">[00:00:18]</a>.
2.  Select "new template" <a class="yt-timestamp" data-t="00:00:21">[00:00:21]</a>.
3.  Enter the bill's name, for instance, "Mobile bill" <a class="yt-timestamp" data-t="00:00:23">[00:00:23]</a>.
4.  Specify the bill amount (e.g., $1,500) <a class="yt-timestamp" data-t="00:00:28">[00:00:28]</a>.
5.  Click "back" to save the basic information <a class="yt-timestamp" data-t="00:00:33">[00:00:33]</a>.

### [[Scheduling repeating notifications for debt payments | Setting Recurring Notifications]]

To make a bill payment reminder repeat on a [[scheduling_repeating_notifications_for_debt_payments | recurring basis]]:
1.  Click on the drop-down menu again <a class="yt-timestamp" data-t="00:00:35">[00:00:35]</a>.
2.  Click on the three dots associated with the specific bill <a class="yt-timestamp" data-t="00:00:37">[00:00:37]</a>.
3.  Choose to set the bill to repeat on a [[scheduling_repeating_notifications_for_debt_payments | recurring basis]] <a class="yt-timestamp" data-t="00:00:39">[00:00:39]</a>.
4.  Select the desired frequency, such as "every month" <a class="yt-timestamp" data-t="00:00:43">[00:00:43]</a>.
5.  Specify the exact time and day for the notification (e.g., 11:55 on the 20th of every month) <a class="yt-timestamp" data-t="00:00:44">[00:00:44]</a>.
6.  Click "Save" <a class="yt-timestamp" data-t="00:00:56">[00:00:56]</a>.

Once saved, the system will automatically send a notification to your [[setting_up_whatsapp_bill_payment_reminders | WhatsApp]] at the specified time and day each month <a class="yt-timestamp" data-t="00:00:58">[00:00:58]</a>.

## [[Automating debt payment reminders on WhatsApp | Automation with Pabbly Connect]]

The [[automating_debt_payment_reminders on WhatsApp | automation]] of these reminders is facilitated by an external tool called Pabbly Connect <a class="yt-timestamp" data-t="00:01:06">[00:01:06]</a>. A detailed explanation and guide for setting up the [[automating_debt_payment_reminders on WhatsApp | automation]] flow in Pabbly Connect is included when the template is downloaded <a class="yt-timestamp" data-t="00:01:13">[00:01:13]</a>.

Once this [[automating_debt_payment_reminders on WhatsApp | automation]] is configured, [[setting_up_whatsapp_bill_payment_reminders | WhatsApp]] notifications for recurring bills will be sent automatically <a class="yt-timestamp" data-t="00:01:17">[00:01:17]</a>.

> A new bill, like the Mobile bill, is automatically added to the system as per the set schedule (e.g., 11:55 on September 20th) <a class="yt-timestamp" data-t="00:01:29">[00:01:29]</a>.

## Notification Delivery

The system is designed to send notifications promptly. For instance, if a notification time is set for 11:55, the message is typically received on [[setting_up_whatsapp_bill_payment_reminders | WhatsApp]] within a minute, e.g., at 11:56 <a class="yt-timestamp" data-t="00:01:54">[00:01:54]</a>.
