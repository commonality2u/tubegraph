A simple template for [[using_notion_templates_for_inventory_management | home inventory tracking]] using Notion helps to keep track of all household inventory in one centralized place <a class="yt-timestamp" data-t="00:00:06">[00:00:06]</a>.

## Key Features and Sections

The template is divided into several sections, providing a comprehensive overview and detailed tracking capabilities.

### Purchase Overview Dashboard

On the left side of the template, users can view a summary of purchases <a class="yt-timestamp" data-t="00:00:09">[00:00:09]</a>:
*   Total estimated purchases <a class="yt-timestamp" data-t="00:00:09">[00:00:09]</a>
*   Total purchases that have been made <a class="yt-timestamp" data-t="00:00:11">[00:00:11]</a>
*   The difference between estimated and actual purchases <a class="yt-timestamp" data-t="00:00:12">[00:00:12]</a>
*   The total percentage of purchases made in comparison to estimated purchases <a class="yt-timestamp" data-t="00:00:14">[00:00:14]</a>

### Inventory Management Sections

The right side of the template features three main sections for detailed inventory management <a class="yt-timestamp" data-t="00:00:18">[00:00:18]</a>:

#### Inventory Categories
This section allows users to set up different categories for their inventory <a class="yt-timestamp" data-t="00:00:20">[00:00:20]</a>. Examples provided include "Dairy & Eggs," "Medicines," and "Fruits & Veggies" <a class="yt-timestamp" data-t="00:00:23">[00:00:23]</a>.
*   Under each category, subcategories are listed <a class="yt-timestamp" data-t="00:00:28">[00:00:28]</a>.
*   An estimated budgeted purchase amount is specified for each subcategory <a class="yt-timestamp" data-t="00:00:30">[00:00:30]</a>.

#### Inventory Overview
This section provides key metrics for all subcategory items <a class="yt-timestamp" data-t="00:00:43">[00:00:43]</a>:
*   Total purchases made <a class="yt-timestamp" data-t="00:00:35">[00:00:35]</a>
*   Consumption units <a class="yt-timestamp" data-t="00:00:37">[00:00:37]</a>
*   Availability units <a class="yt-timestamp" data-t="00:00:39">[00:00:39]</a>
*   Percentage of total units consumed <a class="yt-timestamp" data-t="00:00:40">[00:00:40]</a>

#### Transaction Section
Users can specify individual inventory transactions in this section <a class="yt-timestamp" data-t="00:00:48">[00:00:48]</a>.
*   Entries for purchases, such as "cheese purchase," can be made <a class="yt-timestamp" data-t="00:00:53">[00:00:53]</a>.
*   When adding a purchase, users can select the subcategory <a class="yt-timestamp" data-t="00:00:57">[00:00:57]</a>, specify the inventory items purchased <a class="yt-timestamp" data-t="00:00:59">[00:00:59]</a>, and enter the amount <a class="yt-timestamp" data-t="00:01:00">[00:01:00]</a>.
*   Consumption can also be tracked by adding entries for "items used" <a class="yt-timestamp" data-t="00:01:04">[00:01:04]</a>.
*   All transaction entries automatically update the other sections of the template <a class="yt-timestamp" data-t="00:01:14">[00:01:14]</a>, allowing users to track available inventory and consumption needs <a class="yt-timestamp" data-t="00:01:19">[00:01:19]</a>.
