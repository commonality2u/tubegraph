The Notion Expense Tracker is a customizable tool designed to manage all expenses in one place <a class="yt-timestamp" data-t="00:00:07">[00:00:07]</a>. It features a category section at the top, initially showcasing three different categories <a class="yt-timestamp" data-t="00:00:17">[00:00:17]</a>. ## Managing Expense Categories

The [[setting_up_expense_categories_in_notion | expense categories]] within the [[creating_a_notion_expense_tracker | Notion expense tracker]] are highly flexible, allowing users to [[customizing_notion_for_personal_budget_management | customize]] them for personal or business tracking <a class="yt-timestamp" data-t="00:01:21">[00:01:21]</a>.

### Renaming Categories

To [[modifying_expense_categories_in_notion | rename]] a category:
1.  Hover over the desired category until a pencil icon appears <a class="yt-timestamp" data-t="00:00:25">[00:00:25]</a>.
2.  Click the pencil icon <a class="yt-timestamp" data-t="00:00:27">[00:00:27]</a>.
3.  Type the new name for the category <a class="yt-timestamp" data-t="00:00:31">[00:00:31]</a>. For example, a category can be renamed to "my category" <a class="yt-timestamp" data-t="00:00:33">[00:00:33]</a>.
4.  Press enter to save the new name <a class="yt-timestamp" data-t="00:00:37">[00:00:37]</a>.

### Editing and Deleting Categories

To [[adding_and_deleting_categories_in_notion | edit]] or [[adding_and_deleting_categories_in_notion | delete]] an existing category:
1.  Click on the three dots next to the category <a class="yt-timestamp" data-t="00:00:42">[00:00:42]</a>.
2.  Select "delete" to remove the category <a class="yt-timestamp" data-t="00:00:44">[00:00:44]</a>.

### Adding New Categories

To [[adding_and_deleting_categories_in_notion | add]] a new category:
*   Simply click on the "new page" option within the category section <a class="yt-timestamp" data-t="00:00:48">[00:00:48]</a>.

### Setting Budgeted Amounts

Each category includes fields for a "budgeted expense amount" and an "actual expense amount" <a class="yt-timestamp" data-t="00:00:52">[00:00:52]</a>.
*   Users can set their own budgeted expense amount for each category <a class="yt-timestamp" data-t="00:01:42">[00:01:42]</a>.
*   The total budget for all categories is reflected in the summary section <a class="yt-timestamp" data-t="00:01:52">[00:01:52]</a>, along with actual expenses, the difference, and the percentage of the budget spent <a class="yt-timestamp" data-t="00:01:27">[00:01:27]</a>.
*   The actual expenses are computed from the detailed expense table below the categories <a class="yt-timestamp" data-t="00:01:33">[00:01:33]</a>.

The template is fully [[customizing_notion_for_personal_budget_management | customizable]] to fit various needs and requirements <a class="yt-timestamp" data-t="00:02:05">[00:02:05]</a>.
