A Notion expense tracker allows for significant customization, enabling users to manage and track various financial outflows efficiently <a class="yt-timestamp" data-t="00:00:03">[00:00:03]</a>. This template is designed to [[using_Notion_for_expense_tracking | track all expenses]] in one place <a class="yt-timestamp" data-t="00:00:07">[00:00:07]</a>.

### [[creating_an_expense_categories_system | Expense Categories]]

The tracker features a dedicated category section, typically found at the top, which can initially contain a few predefined categories <a class="yt-timestamp" data-t="00:00:17">[00:00:17]</a>. Each category can be customized to suit individual needs <a class="yt-timestamp" data-t="00:00:22">[00:00:22]</a>.

### [[Modifying and adding new expense categories in Notion | Customizing Categories]]

#### Renaming Categories
To [[Modifying and adding new expense categories in Notion | rename an existing category]]:
1.  Hover over the category name <a class="yt-timestamp" data-t="00:00:25">[00:00:25]</a>.
2.  Click on the pencil icon that appears <a class="yt-timestamp" data-t="00:00:26">[00:00:26]</a>.
3.  Type the new name for the category <a class="yt-timestamp" data-t="00:00:29">[00:00:29]</a>.
4.  Press Enter to save the changes <a class="yt-timestamp" data-t="00:00:37">[00:00:37]</a>.

#### [[Adding or deleting categories in a Notion expense tracker | Deleting Categories]]
To [[Adding or deleting categories in a Notion expense tracker | delete a category]]:
1.  Click on the three dots associated with the category <a class="yt-timestamp" data-t="00:00:42">[00:00:42]</a>.
2.  Select "Delete" from the options <a class="yt-timestamp" data-t="00:00:44">[00:00:44]</a>.

#### [[Modifying and adding new expense categories in Notion | Adding New Categories]]
To [[Modifying and adding new expense categories in Notion | add a new category]]:
1.  Click on the "New page" button <a class="yt-timestamp" data-t="00:00:48">[00:00:48]</a>. This will create a new category entry.

### Managing Expenses with Categories
For each category, the tracker displays both a budgeted expense amount and an actual expense amount <a class="yt-timestamp" data-t="00:00:52">[00:00:52]</a>. As expenses are entered into the tracker, the corresponding category totals are automatically updated <a class="yt-timestamp" data-t="00:01:03">[00:01:03]</a>.

### Summary Section
The summary section provides an overview of financial performance:
*   **Budgeted Expense Amount**: The total amount allocated for expenses <a class="yt-timestamp" data-t="00:01:28">[00:01:28]</a>.
*   **Actual Expenses Amount**: The sum of all recorded expenses <a class="yt-timestamp" data-t="00:01:31">[00:01:31]</a>.
*   **Difference**: The variance between budgeted and actual expenses <a class="yt-timestamp" data-t="00:01:37">[00:01:37]</a>.
*   **Percentage of Budget Spent**: The proportion of the budget that has been utilized <a class="yt-timestamp" data-t="00:01:37">[00:01:37]</a>.

Users can adjust the budgeted expense amount for each category by clicking on the budget figure <a class="yt-timestamp" data-t="00:01:44">[00:01:44]</a>. Any changes made to the budget figures will reflect in the total budget shown in the summary section <a class="yt-timestamp" data-t="00:01:54">[00:01:54]</a>.

Its [[customizing_notion_dashboard_for_personal_finances | customizable nature]] allows it to be adapted to diverse needs <a class="yt-timestamp" data-t="00:02:07">[00:02:07]</a>.
