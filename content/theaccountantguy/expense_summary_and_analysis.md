### Managing Expense Categories

### Budgeted vs. Actual Expenses

For every expense category, the template displays:
*   **Budgeted Expense**: This value can be defined or modified by clicking on it <a class="yt-timestamp" data-t="00:00:24">[00:00:24]</a>.
*   **Actual Expenses**: This figure is automatically pulled from the detailed table at the bottom <a class="yt-timestamp" data-t="00:00:21">[00:00:21]</a>.
*   **Percentage Indicator**: A percentage indicator shows the proportion of [[understanding_the_expenses_overview | actual expenses]] <a class="yt-timestamp" data-t="00:00:30">[00:00:30]</a> incurred relative to the budgeted amount <a class="yt-timestamp" data-t="00:00:28">[00:00:28]</a>.

All [[understanding_the_expenses_overview | expenses values]] <a class="yt-timestamp" data-t="00:00:51">[00:00:51]</a> in the template update dynamically as figures are entered into the table <a class="yt-timestamp" data-t="00:00:52">[00:00:52]</a>, providing real-time insights <a class="yt-timestamp" data-t="00:00:56">[00:00:56]</a>.

## Expense Summary and Analysis

The template includes a dedicated [[summarizing_and_analyzing_construction_expenses | summary]] <a class="yt-timestamp" data-t="00:00:57">[00:00:57]</a> section that offers a comprehensive overview of the restaurant's financial performance:
*   **Total Budgeted Expense**: The sum of all defined budgeted expenses <a class="yt-timestamp" data-t="00:00:59">[00:00:59]</a>.
*   **Actual Expenses Incurred**: The total of all [[understanding_the_expenses_overview | actual expenses]] <a class="yt-timestamp" data-t="00:01:00">[00:01:00]</a> pulled from the entry table <a class="yt-timestamp" data-t="00:01:03">[00:01:03]</a>.
*   **Difference**: The variance between the total budgeted and [[understanding_the_expenses_overview | actual expenses]] <a class="yt-timestamp" data-t="00:01:04">[00:01:04]</a>.
*   **Percentage of Actual Expenses Incurred**: The overall percentage of actual expenses relative to the total budget <a class="yt-timestamp" data-t="00:01:06">[00:01:06]</a>.
