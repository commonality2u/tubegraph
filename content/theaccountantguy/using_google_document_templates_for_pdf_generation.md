[[Using PDF Output tool for efficient document creation | PDF Output]] is a tool designed to [[using_templates_to_create_pdf_documents_in_bulk | generate PDF documents in bulk]] <a class="yt-timestamp" data-t="00:00:08">[00:00:08]</a>. It achieves this by using a Google Document as a template and a Notion database to hold the data elements that will replace placeholders in the Google Document <a class="yt-timestamp" data-t="00:00:11">[00:00:11]</a>. This process enables [[generating_pdf_documents_in_bulk_using_google_docs_and_notion | generating PDF documents in bulk using Google Docs and Notion]] <a class="yt-timestamp" data-t="00:00:08">[00:00:08]</a>, combining [[using_templates_and_databases_to_create_professional_pdfs | templates and databases to create professional PDFs]] <a class="yt-timestamp" data-t="00:00:11">[00:00:11]</a>.

## Getting Started with PDF Output

To begin, users need to log in to PDF Output.com <a class="yt-timestamp" data-t="00:00:19">[00:00:19]</a>. The main interface allows users to add a Google Document, add a Notion database, and then export PDFs <a class="yt-timestamp" data-t="00:00:27">[00:00:27]</a>.

## Creating or Selecting a Google Document Template

The first step in [[customizing_templates_for_pdf_generation | customizing templates for PDF generation]] is to add a Google Document <a class="yt-timestamp" data-t="00:00:37">[00:00:37]</a>. There are two primary methods:
*   **Create a blank document from scratch** <a class="yt-timestamp" data-t="00:00:40">[00:00:40]</a>. This option creates a new Google Document that can be renamed and edited to serve as the template <a class="yt-timestamp" data-t="00:01:09">[00:01:09]</a>.
*   **Choose an existing document** from your Google Drive <a class="yt-timestamp" data-t="00:00:44">[00:00:44]</a>. Existing documents can be searched for by name <a class="yt-timestamp" data-t="00:03:47">[00:03:47]</a>.

Once the document is open, users can add content and define placeholders for dynamic data. Placeholders are typically indicated by specific text that will be replaced by data from the Notion database <a class="yt-timestamp" data-t="00:01:37">[00:01:37]</a>.

### Utilizing Predefined Templates

PDF Output also offers [[utilizing_predefined_pdf_templates_for_efficient_workflow | predefined PDF templates]] <a class="yt-timestamp" data-t="00:07:22">[00:07:22]</a>, such as an invoice template <a class="yt-timestamp" data-t="00:07:35">[00:07:35]</a>. To use these:
1.  Click on a predefined template <a class="yt-timestamp" data-t="00:07:34">[00:07:34]</a>.
2.  Make a copy of the template in your Google Workspace by selecting `File` > `Make a copy` <a class="yt-timestamp" data-t="00:07:46">[00:07:46]</a>.
3.  Alternatively, copy all content (`Ctrl+A`, then `Ctrl+C`) and paste it into a blank document created within PDF Output <a class="yt-timestamp" data-t="00:07:53">[00:07:53]</a>.

## Integrating with Notion Database

After setting up the Google Document template, the next step is [[integrating_google_docs_templates_with_notion_database | integrating Google Docs templates with a Notion database]] <a class="yt-timestamp" data-t="00:02:26">[00:02:26]</a>. This involves:
1.  Clicking "Add Notion Database" and connecting with Notion credentials <a class="yt-timestamp" data-t="00:02:26">[00:02:26]</a>.
2.  Selecting the specific Notion pages (databases) to connect <a class="yt-timestamp" data-t="00:02:43">[00:02:43]</a>. A new database can be created directly in Notion if needed <a class="yt-timestamp" data-t="00:02:53">[00:02:53]</a>.
3.  Once connected, the Notion database's properties (columns) will be displayed <a class="yt-timestamp" data-t="00:03:22">[00:03:22]</a>. If new columns are added to the Notion database, click "Refresh properties" to fetch them <a class="yt-timestamp" data-t="00:04:46">[00:04:46]</a>.

## Mapping Data to Template

With both the Google Document template and Notion database connected, the data mapping can occur:
1.  In the PDF Output interface, select a property (column) from the Notion database to copy its value <a class="yt-timestamp" data-t="00:05:02">[00:05:02]</a>.
2.  In the Google Document template preview, select the placeholder text that needs to be replaced <a class="yt-timestamp" data-t="00:05:04">[00:05:04]</a>.
3.  Paste the copied property value into the placeholder <a class="yt-timestamp" data-t="00:05:06">[00:05:06]</a>. This will replace the selected part of the template with the Notion database field <a class="yt-timestamp" data-t="00:05:28">[00:05:28]</a>.

This mapping ensures that for each row in the Notion database, a unique PDF document will be generated <a class="yt-timestamp" data-t="00:05:38">[00:05:38]</a>.

## Generating PDFs

After the template is set up and the database properties are mapped, click "Export PDF" <a class="yt-timestamp" data-t="00:05:43">[00:05:43]</a>. This initiates Google authentication if not already done <a class="yt-timestamp" data-t="00:05:47">[00:05:47]</a>. The tool will then read each row from the Notion database and generate a corresponding PDF document <a class="yt-timestamp" data-t="00:06:02">[00:06:02]</a>, which will be downloaded <a class="yt-timestamp" data-t="00:06:09">[00:06:09]</a>.

## Additional Features

*   **Search Properties**: Users can search for specific properties within the Notion database <a class="yt-timestamp" data-t="00:08:12">[00:08:12]</a>.
*   **Sorting**: Properties can be sorted for easier management <a class="yt-timestamp" data-t="00:08:20">[00:08:20]</a>.
*   **Live Editing**: Changes made in the PDF Output interface are reflected in the Google Document template <a class="yt-timestamp" data-t="00:06:51">[00:06:51]</a>.
*   **Connected Databases**: If your Notion database has connections to other databases, ensure all relevant databases are approved during the connection process to allow data from linked tables to be used <a class="yt-timestamp" data-t="00:07:07">[00:07:07]</a>.
