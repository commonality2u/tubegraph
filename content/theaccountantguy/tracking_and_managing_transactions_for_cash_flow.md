This template helps users to [[managing_and_tracking_income_and_expenses | track and manage their cash flow]] for business operations <a class="yt-timestamp" data-t="00:00:06">[00:00:06]</a>. It provides a comprehensive overview of financial movements, allowing for efficient management of income, expenses, assets, liabilities, investments, and inventory <a class="yt-timestamp" data-t="00:00:08">[00:00:08]</a>.

## Cash Flow Overview

The "Cash Flow Overview" section on the left side of the template provides a summary of your business's cash position <a class="yt-timestamp" data-t="00:00:10">[00:00:10]</a>. It displays:
*   **Opening balance of cash** <a class="yt-timestamp" data-t="00:00:13">[00:00:13]</a>
*   **Total inflow of cash** from all transactions <a class="yt-timestamp" data-t="00:00:14">[00:00:14]</a>
*   **Cash outflow** from all transactions <a class="yt-timestamp" data-t="00:00:19">[00:00:19]</a>
*   **Closing balance**, which updates automatically as transactions are entered <a class="yt-timestamp" data-t="00:00:21">[00:00:21]</a>

## Account Section

The "Account Section" allows you to set up and manage different financial accounts under various heads <a class="yt-timestamp" data-t="00:00:34">[00:00:34]</a>. These categories include:

*   **Assets**: Default assets like cash, bank, receivables, and input taxes are provided, with the option to add or edit existing ones <a class="yt-timestamp" data-t="00:00:40">[00:00:40]</a>.
*   **Liabilities** <a class="yt-timestamp" data-t="00:00:36">[00:00:36]</a>
*   **Income** <a class="yt-timestamp" data-t="00:00:36">[00:00:36]</a>
*   **Expenses** <a class="yt-timestamp" data-t="00:02:40">[00:02:40]</a>

For each account, you can configure the following details:
*   **Name**: Edit by clicking the pencil icon <a class="yt-timestamp" data-t="00:00:51">[00:00:51]</a>.
*   **Opening Balance**: The starting balance of the asset when you begin using the template <a class="yt-timestamp" data-t="00:01:05">[00:01:05]</a>.
*   **Classifications**: Categorize items, such as classifying a credit card as "debt" <a class="yt-timestamp" data-t="00:01:12">[00:01:12]</a>. New classifications can be added or existing ones edited <a class="yt-timestamp" data-t="00:01:30">[00:01:30]</a>.
*   **Payment Frequency**: Specifies how frequently payments are received or made for a particular item, useful for income and expenses <a class="yt-timestamp" data-t="00:01:39">[00:01:39]</a>.
*   **Opening Units**: Relevant for investments or inventory, indicating the starting number of units <a class="yt-timestamp" data-t="00:02:04">[00:02:04]</a>.
*   **Asset Class**: Used for inventory (e.g., "inventory asset class") or investments (e.g., "investment asset class") <a class="yt-timestamp" data-t="00:02:17">[00:02:17]</a>.

To add a new item, click "New" under the respective category <a class="yt-timestamp" data-t="00:02:49">[00:02:49]</a>.

## Quarterly Overview

The "Quarterly Overview" section provides a detailed look at cash flow movement for each month, divided into quarters <a class="yt-timestamp" data-t="00:03:09">[00:03:09]</a>. It shows:
*   **Inflow**: Cash coming into the business <a class="yt-timestamp" data-t="00:03:23">[00:03:23]</a>.
*   **Outflow**: Cash leaving the business <a class="yt-timestamp" data-t="00:03:27">[00:03:27]</a>.
*   **Net Balance**: The difference between inflow and outflow <a class="yt-timestamp" data-t="00:03:30">[00:03:30]</a>.

These figures automatically update as you add transactions <a class="yt-timestamp" data-t="00:03:49">[00:03:49]</a>.

## Business Activity

The "Business Activity" section categorizes cash flow based on the type of business operation:
*   **Operating Activities**: Relate to the core operations of the business, such as salary payments or revenue receipts <a class="yt-timestamp" data-t="00:03:57">[00:03:57]</a>.
*   **Investing Activities**: Cover investments made by the business, such as stock or mutual fund investments <a class="yt-timestamp" data-t="00:04:12">[00:04:12]</a>.
*   **Financing Activities**: Involve raising loans or debt to run the business <a class="yt-timestamp" data-t="00:04:33">[00:04:33]</a>.

Each activity type displays its specific inflow, outflow, and net balance, offering a quick overview of financial health <a class="yt-timestamp" data-t="00:04:45">[00:04:45]</a>.

## [[adding_transactions_to_track_income_and_expenses | Entering and managing transactions for small businesses]]

The template offers different views for [[entering_and_managing_transactions_for_small_businesses | entering and managing transactions for small businesses]]:
*   Income <a class="yt-timestamp" data-t="00:04:57">[00:04:57]</a>
*   Expense <a class="yt-timestamp" data-t="00:04:58">[00:04:58]</a>
*   Asset <a class="yt-timestamp" data-t="00:04:59">[00:04:59]</a>
*   Liability <a class="yt-timestamp" data-t="00:05:00">[00:05:00]</a>
*   Investments: Allows specifying units purchased and price <a class="yt-timestamp" data-t="00:05:07">[00:05:07]</a>.
*   Inventory: Allows specifying inventory units and price <a class="yt-timestamp" data-t="00:05:14">[00:05:14]</a>.

To add a transaction, for example, an income transaction:
1.  Click the "Add Income" button on the left <a class="yt-timestamp" data-t="00:05:35">[00:05:35]</a>. This automatically adds a new entry with the current date <a class="yt-timestamp" data-t="00:05:41">[00:05:41]</a>.
2.  **Date**: Can be changed using the calendar <a class="yt-timestamp" data-t="00:05:51">[00:05:51]</a>.
3.  **Period**: Specify the relevant month or period for the transaction <a class="yt-timestamp" data-t="00:06:06">[00:06:06]</a>.
4.  **Description**: Enter a description, e.g., "Sales Revenue" <a class="yt-timestamp" data-t="00:06:32">[00:06:32]</a>.
5.  **Account**: Select the relevant account from the list set up in the "Account Section" <a class="yt-timestamp" data-t="00:06:42">[00:06:42]</a>.
6.  **Payment Mode**: Indicate how the payment was received (e.g., cash, bank) <a class="yt-timestamp" data-t="00:06:54">[00:06:54]</a>.
7.  **Amount**: Enter the monetary value of the transaction <a class="yt-timestamp" data-t="00:07:09">[00:07:09]</a>.
8.  **Type of Cash Flow**: Select whether it's an "Operating," "Investing," or "Financing" activity <a class="yt-timestamp" data-t="00:07:17">[00:07:17]</a>.
9.  **Transaction Type**: Designates whether cash is flowing "Inflow" (e.g., income) or "Outflow" (e.g., expense) <a class="yt-timestamp" data-t="00:07:54">[00:07:54]</a>. For asset and liability transactions, you must assess if cash is coming in or going out <a class="yt-timestamp" data-t="00:08:21">[00:08:21]</a>.
10. **Documents**: Attach relevant documents like sales receipts <a class="yt-timestamp" data-t="00:08:39">[00:08:39]</a>.
11. **Tax Rate**: Enter a tax rate to automatically calculate the tax amount on the transaction <a class="yt-timestamp" data-t="00:08:48">[00:08:48]</a>.

All summary figures and overviews update automatically as transactions are added <a class="yt-timestamp" data-t="00:09:07">[00:09:07]</a>. If figures do not immediately update, clicking on the sum or balance may refresh them <a class="yt-timestamp" data-t="00:11:21">[00:11:21]</a>.
