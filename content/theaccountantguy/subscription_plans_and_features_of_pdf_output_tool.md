The [[PDF Outputcom Features and Subscription Plans | PDF Output]] tool allows users to generate bulk PDF documents using [[using_pdf_output_tool_with_notion | Notion pages and databases]] <a class="yt-timestamp" data-t="00:00:05">[00:00:05]</a>.

### Core Features
*   **Template-Based Generation**
    *   Utilizes a Notion page as a template, where specific items are placed inside curly braces (e.g., `{{Purchase Order Number}}`) <a class="yt-timestamp" data-t="00:00:26">[00:00:26]</a>.
    *   These bracketed items are replaced with data from a Notion database <a class="yt-timestamp" data-t="00:00:38">[00:00:38]</a>.
    *   Users can customize the entire template after duplication, ensuring that data fields are kept within curly braces and follow exact naming conventions for proper mapping <a class="yt-timestamp" data-t="05:17:09">[05:17:09]</a> <a class="yt-timestamp" data-t="05:22:15">[05:22:15]</a> <a class="yt-timestamp" data-t="05:30:57">[05:30:57]</a>.
*   **Data Mapping**
    *   The tool automatically maps database properties to template fields if the naming conventions are identical <a class="yt-timestamp" data-t="04:43:08">[04:43:08]</a> <a class="yt-timestamp" data-t="04:57:39">[04:57:39]</a>.
    *   Manual mapping is available if automatic mapping is not possible <a class="yt-timestamp" data-t="04:47:48">[04:47:48]</a>.
    *   Supports relational properties in databases, requiring explicit access to those related databases during connection setup <a class="yt-timestamp" data-t="09:00:03">[09:00:03]</a> <a class="yt-timestamp" data-t="09:13:00">[09:13:00]</a>.
*   **Document Management**
    *   Allows previewing of sample generated PDF files <a class="yt-timestamp" data-t="05:41:40">[05:41:40]</a>.
    *   Enables downloading all generated documents in bulk <a class="yt-timestamp" data-t="06:07:07">[06:07:07]</a>.
    *   Provides a "Download PDF again" option if the initial download is missed <a class="yt-timestamp" data-t="06:49:00">[06:49:00]</a>.
*   **Platform Navigation & Reuse**
    *   The "Connections" section displays previously created connections, allowing users to quickly load the database and template without re-entering them for future exports <a class="yt-timestamp" data-t="06:57:08">[06:57:08]</a>.
    *   Includes a search function for finding specific templates and databases <a class="yt-timestamp" data-t="04:07:46">[04:07:46]</a>.
    *   The interface can be minimized, making it suitable for mobile or smaller devices <a class="yt-timestamp" data-t="08:46:25">[08:46:25]</a>.

### Settings and Support
*   **Page Format**: By default, the page format for exported PDFs is set to A4 <a class="yt-timestamp" data-t="07:48:29">[07:48:29]</a>.
*   **Adding More Resources**: Users can add additional databases and templates through the settings once an initial connection is established <a class="yt-timestamp" data-t="07:55:04">[07:55:04]</a>.
*   **Feedback**: A dedicated feedback option is available for reporting difficulties or issues <a class="yt-timestamp" data-t="08:09:59">[08:09:59]</a>.
*   **Help Section**: Offers guidance, including a video demonstrating how to use a custom template from scratch <a class="yt-timestamp" data-t="08:23:44">[08:23:44]</a>.

### Subscription Plans
*   **Free Plan**: Documents exported under the free plan will include a "Made with PDF Output" watermark <a class="yt-timestamp" data-t="07:29:02">[07:29:02]</a> <a class="yt-timestamp" data-t="07:31:39">[07:31:39]</a>.
*   **Upgrade Option**: Users can upgrade from the free plan to remove the watermark and any limits on the number of PDF documents generated <a class="yt-timestamp" data-t="07:34:40">[07:34:40]</a> <a class="yt-timestamp" data-t="07:37:39">[07:37:39]</a> <a class="yt-timestamp" data-t="07:44:03">[07:44:03]</a>.
