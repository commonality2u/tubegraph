A house rent payments tracker template can be utilized to keep track of [[tracking_house_rental_payments | house rental payments]] <a class="yt-timestamp" data-t="00:00:06">[00:00:06]</a>. This template allows users to monitor the status of [[how to use notion for rental payments | rental payments]] and manage property and tenant information effectively <a class="yt-timestamp" data-t="00:01:48">[00:01:48]</a>.

## Summary Section

The top section of the tracker provides a comprehensive summary across four different views <a class="yt-timestamp" data-t="00:00:10">[00:00:10]</a>.

### Basic Information
Users can add basic information, including tenants, which can be classified under different properties they own, such as "apartment" and "duplex" <a class="yt-timestamp" data-t="00:00:17">[00:00:17]</a>. New tenants can be added by clicking the "ADD tenant" button and then detailing their information <a class="yt-timestamp" data-t="00:00:28">[00:00:28]</a>.

### Property Section
This view categorizes properties and displays key financial details for each tenant under their respective property <a class="yt-timestamp" data-t="00:00:46">[00:00:46]</a>:
*   Total rent amount <a class="yt-timestamp" data-t="00:00:54">[00:00:54]</a>
*   Rent amount that has been paid <a class="yt-timestamp" data-t="00:00:55">[00:00:55]</a>
*   Balance rent amount that is due <a class="yt-timestamp" data-t="00:00:57">[00:00:57]</a>
*   Advanced rent that has been paid <a class="yt-timestamp" data-t="00:00:58">[00:00:58]</a>

### Rental History
The [[overview_of_rental_payment_history | rental history]] section displays the payment history for all tenants, based on the [[tracking_payments_and_transactions | transactions]] recorded at the bottom of the template <a class="yt-timestamp" data-t="00:01:04">[00:01:04]</a>.

### Overall Summary
A final summary section provides an overview of financial status <a class="yt-timestamp" data-t="00:01:12">[00:01:12]</a>:
*   Total amount due <a class="yt-timestamp" data-t="00:01:14">[00:01:14]</a>
*   Amount that has been paid <a class="yt-timestamp" data-t="00:01:15">[00:01:15]</a>
*   Outstanding rent amount <a class="yt-timestamp" data-t="00:01:16">[00:01:16]</a>
*   Percentage of rent collected <a class="yt-timestamp" data-t="00:01:17">[00:01:17]</a>

## Transaction Section

The [[tracking_payments_and_transactions | transaction section]] offers three distinct views in addition to a general view of all [[tracking_payments_and_transactions | transactions]] <a class="yt-timestamp" data-t="00:01:20">[00:01:20]</a>.

### Rent Paid
This view lists all [[tracking_payments_and_transactions | transactions]] for which rent has been paid, including details of the respective tenants and amounts <a class="yt-timestamp" data-t="00:01:24">[00:01:24]</a>.

### Rent Due
This section shows information for rent that is due but has not yet been paid <a class="yt-timestamp" data-t="00:01:31">[00:01:31]</a>. Clicking a checkbox in this view marks the rent as paid, moving it to the "Rent Paid" category <a class="yt-timestamp" data-t="00:01:36">[00:01:36]</a>.

### Quarterly Summary
The quarterly summary view presents [[how to use notion for rental payments | rental payments]], collections, and the balance due for each month within a specific quarter <a class="yt-timestamp" data-t="00:01:41">[00:01:41]</a>.

## Adding Payments

To quickly add a new [[how to use notion for rental payments | rental payment]], users can click a designated button which will automatically create the entry <a class="yt-timestamp" data-t="00:01:51">[00:01:51]</a>.
