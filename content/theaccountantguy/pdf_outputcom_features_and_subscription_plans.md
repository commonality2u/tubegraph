PDF Output.com is a tool designed to [[generating_pdf_documents_for_business_needs | generate PDF documents]] in bulk, primarily by leveraging Notion templates and databases <a class="yt-timestamp" data-t="00:00:07">[00:00:07]</a> <a class="yt-timestamp" data-t="00:01:11">[00:01:11]</a>.

## Core Functionality: Bulk PDF Generation

The platform's primary feature is its ability to [[using_pdf_outputcom_for_bulk_pdf_creation | create bulk PDF documents]] from a Notion template and a Notion database <a class="yt-timestamp" data-t="00:00:09">[00:00:09]</a>. This process involves:
*   **Template Elements** <a class="yt-timestamp" data-t="00:00:20">[00:00:20]</a>: Templates include specific elements placed inside curly braces (e.g., `{receipt number}`, `{company website}`) that serve as placeholders <a class="yt-timestamp" data-t="00:00:22">[00:00:22]</a> <a class="yt-timestamp" data-t="00:01:51">[00:01:51]</a>.
*   **Database Integration** <a class="yt-timestamp" data-t="00:00:26">[00:00:26]</a>: A corresponding Notion database contains the actual information that will replace these placeholders <a class="yt-timestamp" data-t="00:00:26">[00:00:26]</a> <a class="yt-timestamp" data-t="00:01:57">[00:01:57]</a>. Each row in the database typically corresponds to a single PDF document <a class="yt-timestamp" data-t="00:00:45">[00:00:45]</a>.
*   **Mapping** <a class="yt-timestamp" data-t="00:06:08">[00:06:08]</a>: The tool automatically maps database properties (columns) to their corresponding template elements if the naming convention, including spaces and capitalization, is exact <a class="yt-timestamp" data-t="00:06:30">[00:06:30]</a> <a class="yt-timestamp" data-t="00:06:46">[00:06:46]</a>. If there's a mismatch, manual mapping can be done by searching for the correct element <a class="yt-timestamp" data-t="00:06:59">[00:06:59]</a>.

### Workflow for PDF Generation
1.  **Access PDF Output.com** <a class="yt-timestamp" data-t="00:01:24">[00:01:24]</a>: Navigate to PDF Output.com and log in <a class="yt-timestamp" data-t="00:01:29">[00:01:29]</a>.
2.  **Duplicate Template and Database** <a class="yt-timestamp" data-t="00:03:01">[00:03:01]</a>:
    *   Go to the "Template" section within PDF Output.com <a class="yt-timestamp" data-t="00:02:08">[00:02:08]</a>.
    *   Select the desired template (e.g., Payment Receipt template) <a class="yt-timestamp" data-t="00:02:10">[00:02:10]</a> <a class="yt-timestamp" data-t="00:02:21">[00:02:21]</a>.
    *   Click on the template link and database link to open them in Notion <a class="yt-timestamp" data-t="00:02:43">[00:02:43]</a>.
    *   Duplicate both the template and the database to your local Notion workspace <a class="yt-timestamp" data-t="00:03:02">[00:03:02]</a>.
3.  **[[setting_up_and_configuring_pdfoutput | Connect Notion Database and Template]]** <a class="yt-timestamp" data-t="00:01:47">[00:01:47]</a>:
    *   On PDF Output.com, click "Click here to add database" or "Add template" <a class="yt-timestamp" data-t="00:04:45">[00:04:45]</a>.
    *   Grant permission to your Notion workspace by selecting the workspace and relevant pages (template and database) <a class="yt-timestamp" data-t="00:04:52">[00:04:52]</a>.
    *   The platform will authenticate and connect to your Notion resources <a class="yt-timestamp" data-t="00:05:14">[00:05:14]</a>.
    *   Assign a connection name (e.g., "payment receipts") <a class="yt-timestamp" data-t="00:05:56">[00:05:56]</a>.
4.  **Review and Map Properties** <a class="yt-timestamp" data-t="00:06:02">[00:06:02]</a>:
    *   The platform displays all database properties and their corresponding template mappings <a class="yt-timestamp" data-t="00:06:09">[00:06:09]</a>.
    *   Verify that all elements are correctly mapped; manually adjust any mismatches <a class="yt-timestamp" data-t="00:06:59">[00:06:59]</a>.
    *   Filtering, searching, and sorting options are available for properties <a class="yt-timestamp" data-t="00:07:16">[00:07:16]</a>.
5.  **Generate and Export PDFs** <a class="yt-timestamp" data-t="00:07:27">[00:07:27]</a>:
    *   Click "Create PDF" to initiate the bulk generation <a class="yt-timestamp" data-t="00:07:27">[00:07:27]</a>.
    *   Once successful, you can preview a sample PDF <a class="yt-timestamp" data-t="00:07:46">[00:07:46]</a>.
    *   Click "Download all documents" to receive a zip file containing all generated PDFs <a class="yt-timestamp" data-t="00:08:17">[00:08:17]</a> <a class="yt-timestamp" data-t="00:08:20">[00:08:20]</a>.
