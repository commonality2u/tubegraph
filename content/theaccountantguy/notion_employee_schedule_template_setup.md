The [[notion_application | Notion application]] employee schedule template is designed to track employee schedules and tasks within an organization <a class="yt-timestamp" data-t="00:00:06">[00:00:06]</a>. It provides a comprehensive overview of employee details, assigned tasks, and performance metrics <a class="yt-timestamp" data-t="00:00:10">[00:00:10]</a>. This template is fully [[customizing_notion_templates_for_employee_scheduling | customizable]] to fit specific organizational needs <a class="yt-timestamp" data-t="00:13:00">[00:13:00]</a>.

## Template Views

The template is structured with four distinct views to manage employee scheduling effectively <a class="yt-timestamp" data-t="00:00:12">[00:00:12]</a>:

*   **Employee Details** <a class="yt-timestamp" data-t="00:00:14">[00:00:14]</a>
*   **Task Schedule** <a class="yt-timestamp" data-t="00:00:19">[00:00:19]</a>
*   **Monthly View** <a class="yt-timestamp" data-t="00:00:24">[00:00:24]</a>
*   **Employee Performance** <a class="yt-timestamp" data-t="00:00:32">[00:00:32]</a>

### Employee Details

This section serves as a central database for [[managing_employee_details_in_notion | managing employee details]] <a class="yt-timestamp" data-t="00:00:16">[00:00:16]</a>.

#### Adding New Employees

To add a new employee:
1.  Click the "New" button within the Employee Details section <a class="yt-timestamp" data-t="00:00:41">[00:00:41]</a>.
2.  Alternatively, use the "Add employee" quick action button located at the top of the template <a class="yt-timestamp" data-t="00:01:31">[00:01:31]</a>.
3.  Enter essential information such as name, employee ID, and contact details <a class="yt-timestamp" data-t="00:01:06">[00:01:06]</a>.
4.  Further details can be added, including department, position, and email ID <a class="yt-timestamp" data-t="00:01:17">[00:01:17]</a>.

#### Managing Categories

Employees can be organized into categories like operations, marketing, and finance <a class="yt-timestamp" data-t="00:00:49">[00:00:49]</a>.
*   New categories can be added by clicking the plus icon <a class="yt-timestamp" data-t="00:00:53">[00:00:53]</a>.
*   Existing categories can be edited by clicking on them <a class="yt-timestamp" data-t="00:00:57">[00:00:57]</a>.

### Task Schedule

The [[organizing_tasks_with_notion_task_schedule | Notion task schedule]] view is used to enter all task-related information for employees <a class="yt-timestamp" data-t="00:01:25">[00:01:25]</a>.

#### Shift Management

Tasks are organized by different shifts, such as first, second, and third shifts <a class="yt-timestamp" data-t="00:01:51">[00:01:51]</a>. For each shift, employees can enter:
*   **Start Time:** Including date and time <a class="yt-timestamp" data-t="00:01:59">[00:01:59]</a>.
*   **End Time:** Specifying the period <a class="yt-timestamp" data-t="00:02:04">[00:02:04]</a>.
*   **Task Detail:** Information about the specific task <a class="yt-timestamp" data-t="00:02:10">[00:02:10]</a>.
*   **Employee Name:** Select from previously added employees <a class="yt-timestamp" data-t="00:02:15">[00:02:15]</a>. An employee can also be added directly from this section <a class="yt-timestamp" data-t="00:02:21">[00:02:21]</a>.
*   **Department:** Automatically assigned based on the employee's details <a class="yt-timestamp" data-t="00:02:27">[00:02:27]</a>.
*   **Hours Worked:** Automatically computed based on the start and end times entered <a class="yt-timestamp" data-t="00:02:33">[00:02:33]</a>.
*   **Attendance:** Mark status as "Present," "On Leave," or "Day Off" <a class="yt-timestamp" data-t="00:02:40">[00:02:40]</a>.
*   **Task Status:** Indicate if a task is "In Progress," "Completed," "Not Started," or "Not Applicable" (e.g., for days off or leave) <a class="yt-timestamp" data-t="00:02:51">[00:02:51]</a>.

### Monthly View

This view provides an aggregated summary of employee activities per month <a class="yt-timestamp" data-t="00:03:18">[00:03:18]</a>.
*   Months are divided into different quarters <a class="yt-timestamp" data-t="00:03:25">[00:03:25]</a>.
*   It displays the total hours worked per month and the total hours employees have been on leave <a class="yt-timestamp" data-t="00:03:29">[00:03:29]</a>.
*   The total hours (summation of working and leave hours) are also displayed <a class="yt-timestamp" data-t="00:03:33">[00:03:33]</a>.
*   This view offers a quick overview of overall monthly performance <a class="yt-timestamp" data-t="00:03:39">[00:03:39]</a>.

### Employee Performance

For a detailed breakdown of individual employee performance, this section provides the same information as the Monthly View but segmented by each employee <a class="yt-timestamp" data-t="00:03:48">[00:03:48]</a>.
*   It shows the hours worked, hours on leave, and total hours for each employee <a class="yt-timestamp" data-t="00:03:56">[00:03:56]</a>.
*   This allows for individual performance evaluation <a class="yt-timestamp" data-t="00:03:45">[00:03:45]</a>.
