PDF output.com allows users to generate PDF documents, such as payment receipts, by [[using_templates_and_databases_to_create_professional_pdfs | using Notion databases with Notion templates]] <a class="yt-timestamp" data-t="00:00:02">[00:00:02]</a>. This process involves [[utilizing_predefined_pdf_templates_for_efficient_workflow | predefined templates]] that can be customized to specific requirements <a class="yt-timestamp" data-t="04:21:00">[04:21:00]</a>.

## Accessing and Duplicating Templates

To begin, log in to PDF output.com <a class="yt-timestamp" data-t="00:00:11">[00:00:11]</a>. From the interface, click on the "templates" option <a class="yt-timestamp" data-t="00:00:17">[00:00:17]</a>, which will display a list of available PDF output templates <a class="yt-timestamp" data-t="00:00:22">[00:00:22]</a>. Users can search or filter for specific templates, such as the "payment receipts PDF generator" <a class="yt-timestamp" data-t="00:00:32">[00:00:32]</a>.

After selecting a template, click the "download link" <a class="yt-timestamp" data-t="00:00:56">[00:00:56]</a>. This opens a page containing the template and its associated database <a class="yt-timestamp" data-t="01:03:00">[01:03:00]</a>. To use the template with PDF output, it must be duplicated to a Notion workspace <a class="yt-timestamp" data-t="02:49:50">[02:49:50]</a>. If the template is not yet published to the Notion Gallery, click "duplicate" and choose the Notion workspace <a class="yt-timestamp" data-t="02:35:00">[02:35:00]</a>. If it is published, click "start with this template" <a class="yt-timestamp" data-t="02:43:00">[02:43:00]</a>.

## Template Customization and Placeholders

The templates come with predefined elements for PDF output <a class="yt-timestamp" data-t="01:22:00">[01:22:00]</a>. Key to [[customizing_pdf_templates_in_notion | customizing these templates]] are placeholder text elements, which are enclosed in curly braces (e.g., `{receipt number}`, `{receipt date}`, `{customer name}`) <a class="yt-timestamp" data-t="01:28:00">[01:28:00]</a>. These placeholders will be replaced by corresponding information from the Notion database <a class="yt-timestamp" data-t="01:40:00">[01:40:00]</a>.

### Customization Rules

*   **Matching Names**: It is crucial that the exact names of the placeholder elements in the template (e.g., `{customer name}`) precisely match the corresponding column names in the Notion database <a class="yt-timestamp" data-t="02:06:00">[02:06:00]</a>. This ensures proper syncing and error-free PDF generation <a class="yt-timestamp" data-t="04:59:00">[04:59:00]</a>.
*   **Aesthetic Changes**: Users can customize the template's format by making elements bold, adjusting spacing, or making any other desired changes directly within the Notion template <a class="yt-timestamp" data-t="04:27:00">[04:27:00]</a>.
*   **Placeholder Integrity**: When customizing, ensure that any placeholder elements intended for replacement from the database remain within curly braces <a class="yt-timestamp" data-t="04:43:00">[04:43:00]</a>.

### Connecting to PDF Output

After duplicating the template to Notion, navigate to the PDF output settings <a class="yt-timestamp" data-t="03:24:00">[03:24:00]</a> and click "click here to add templates" <a class="yt-timestamp" data-t="03:32:00">[03:32:00]</a>. Select the Notion workspace where the template was duplicated <a class="yt-timestamp" data-t="03:44:00">[03:44:00]</a>. Grant access to allow PDF output to read the template and database elements <a class="yt-timestamp" data-t="04:02:00">[04:02:00]</a>.

Once connected, select the Notion database and its corresponding template from dropdown menus <a class="yt-timestamp" data-t="05:25:00">[05:25:00]</a>. The system will automatically map the Notion database properties (columns) to the template elements if the names match correctly <a class="yt-timestamp" data-t="05:56:00">[05:56:00]</a>. If any elements are mismatched, they can be manually searched for and mapped <a class="yt-timestamp" data-t="06:04:00">[06:04:00]</a>. The mapping view also provides filter, search, and sorting options <a class="yt-timestamp" data-t="06:19:00">[06:19:00]</a>.

## Generating and Managing PDFs

With everything set up, click "export" to [[utilizing_pdf_output_for_bulk_pdf_generation | generate the PDF documents]] <a class="yt-timestamp" data-t="06:39:00">[06:39:00]</a>. A "PDF status" column will be added to the Notion database, indicating which rows have been generated <a class="yt-timestamp" data-t="06:43:00">[06:43:00]</a>.

After successful export, users can preview a sample PDF <a class="yt-timestamp" data-t="07:07:00">[07:07:00]</a> or [[exporting_and_managing_generated_pdf_documents | download all generated files]] <a class="yt-timestamp" data-t="07:17:00">[07:17:00]</a>. Each generated file corresponds to a row in the Notion database, with all elements placed correctly according to the template <a class="yt-timestamp" data-t="07:36:00">[07:36:00]</a>.

### Additional Features

*   **Connections**: The "connections" section shows past configurations, allowing users to quickly load a database and template for re-generation without manual setup <a class="yt-timestamp" data-t="07:56:00">[07:56:00]</a>.
*   **Templates**: Users can select multiple templates if needed <a class="yt-timestamp" data-t="08:20:00">[08:20:00]</a>.
*   **Free Plan Watermark**: On the free plan, generated PDFs will include a PDF output watermark <a class="yt-timestamp" data-t="08:24:00">[08:24:00]</a>. Upgrading to a paid plan removes this watermark <a class="yt-timestamp" data-t="08:31:00">[08:31:00]</a>.
*   **Settings**: Options to change the page format for generated PDFs are available under settings <a class="yt-timestamp" data-t="08:41:00">[08:41:00]</a>.
*   **Support**: A feedback option allows users to send messages for assistance <a class="yt-timestamp" data-t="08:50:00">[08:50:00]</a>, and a demonstration video is available under the "help" section to guide setup <a class="yt-timestamp" data-t="09:00:00">[09:00:00]</a>.
