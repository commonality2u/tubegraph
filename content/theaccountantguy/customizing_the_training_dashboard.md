The employee training expenses tracker provides a training dashboard with various customizable elements to manage training overviews, employee records, and expenses <a class="yt-timestamp" data-t="00:00:37">[00:00:37]</a> <a class="yt-timestamp" data-t="00:06:07">[00:06:07]</a>.

## Dashboard Views

The training dashboard presents two primary views:
*   **Training Overview** <a class="yt-timestamp" data-t="00:00:08">[00:00:08]</a>
*   **Employee Records** <a class="yt-timestamp" data-t="00:00:09">[00:00:09]</a>

### Training Overview Customization
The Training Overview section provides an overview of various trainings. Information like training date, employee count, percentage of attendance, and training expenses can be edited and customized <a class="yt-timestamp" data-t="00:00:11">[00:00:11]</a>.

### Employee Records Customization
The employee records section lists employees by department <a class="yt-timestamp" data-t="00:00:47">[00:00:47]</a>.
*   **Editing Departments:** To edit a department, click on its name and start typing <a class="yt-timestamp" data-t="00:00:50">[00:00:50]</a>.
*   **Adding Departments:** A new department can be added by holding a slider to the right and clicking the plus icon <a class="yt-timestamp" data-t="00:01:01">[00:01:01]</a>.
*   **Moving Employees between Departments:** Employees can be dragged and dropped between departments to automatically shift their records <a class="yt-timestamp" data-t="00:01:15">[00:01:15]</a>. Alternatively, click the three-dot icon next to an employee and move them to a different department <a class="yt-timestamp" data-t="00:01:55">[00:01:55]</a>.
*   **Next Training Date:** The next training date for an employee can be specified by clicking on the designated field <a class="yt-timestamp" data-t="00:01:40">[00:01:40]</a>. This will automatically update the number of days remaining until the training <a class="yt-timestamp" data-t="00:01:48">[00:01:48]</a>.
*   **Employee Details:** Individual employee details can be edited by clicking the "edit" button and typing the new name, which updates the employee records <a class="yt-timestamp" data-t="00:02:58">[00:02:58]</a>.
*   **Adding Employees:** New employee records can be added using the "Add employee" quick action button, placing them under a "No Department" view initially <a class="yt-timestamp" data-t="00:02:47">[00:02:47]</a>. These can then be moved to any department <a class="yt-timestamp" data-t="00:02:53">[00:02:53]</a>. While possible to add new employees directly through the training database, using the quick action button is suggested <a class="yt-timestamp" data-t="00:03:39">[00:03:39]</a> <a class="yt-timestamp" data-t="00:03:45">[00:03:45]</a>.

## Expense Overview Customization

The expenses overview displays total expenses, number of trainings, total employees attending, and average expense per employee <a class="yt-timestamp" data-t="00:02:06">[00:02:06]</a>. The average expense automatically updates as transaction details are entered <a class="yt-timestamp" data-t="00:02:29">[00:02:29]</a>.

## Detailed Management Sections

Below the main dashboard, there are three detailed views for managing records:
*   Training Records <a class="yt-timestamp" data-t="00:03:36">[00:03:36]</a>
*   Employee Details <a class="yt-timestamp" data-t="00:02:42">[00:02:42]</a>
*   Training Expenses <a class="yt-timestamp" data-t="00:05:16">[00:05:16]</a>

### Training Records Customization
In the training record section, users can find all attended training sessions <a class="yt-timestamp" data-t="00:03:13">[00:03:13]</a>.
*   **Set Dates:** The date for each training session can be set <a class="yt-timestamp" data-t="00:03:21">[00:03:21]</a>.
*   **Add Details:** Users can [[editing_and_adding_training_details | feed in training details]] <a class="yt-timestamp" data-t="00:03:25">[00:03:25]</a>.
*   **Assign Employees:** Employees can be selected for specific trainings <a class="yt-timestamp" data-t="00:03:29">[00:03:29]</a>. Trainings themselves are editable, and users can [[editing_and_adding_training_details | edit training details]] directly from the training database <a class="yt-timestamp" data-t="00:03:51">[00:03:51]</a>.
*   **Track Attendance:** A checkbox indicates if employees have attended a training, automatically updating attendance percentages in the training records <a class="yt-timestamp" data-t="00:04:08">[00:04:08]</a>.
*   **Add Records:** New training records can be added using the "Add training" quick action button <a class="yt-timestamp" data-t="00:04:59">[00:04:59]</a>.
*   **Tag Multiple Employees:** Users can tag multiple employees to a training record, and the attendance data will update automatically <a class="yt-timestamp" data-t="00:04:50">[00:04:50]</a>.

### Training Expenses Customization
This section allows users to view and edit expenses incurred for each training <a class="yt-timestamp" data-t="00:05:16">[00:05:16]</a>.
*   **Edit Expenses:** Click the pencil icon next to a training to edit its expense information <a class="yt-timestamp" data-t="00:05:24">[00:05:24]</a>.
*   **Change View:** The expense view can be changed to a table view by clicking the three-dot icon, allowing for quick editing <a class="yt-timestamp" data-t="00:05:53">[00:05:53]</a>.

## Currency Customization
All currencies displayed in the tracker can be changed.
*   **Database Currency Settings:** To change the currency, go to the currency settings within a particular database, click on "edit property," and select the desired currency from the number format options <a class="yt-timestamp" data-t="00:06:11">[00:06:11]</a>.
*   **Dashboard Currency:** For the dashboard on the top, click on the displayed currency information. The dollar sign within the formula can be changed to suit specific requirements <a class="yt-timestamp" data-t="00:06:32">[00:06:32]</a>.
