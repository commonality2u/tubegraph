A Film Production Budget Tracker is a tool designed to [[budgeting_and_tracking_expenses_against_set_budgets | keep track of all expenses related to Film Production]] <a class="yt-timestamp" data-t="00:00:06">[00:00:06]</a>. This template helps users to [[managing_budget_and_actual_expenses | manage both budgeted and actual expenses]] <a class="yt-timestamp" data-t="00:00:49">[00:00:49]</a>, providing a comprehensive overview of financial outlay for a film project.

## Template Overview
The tracker features a table that systematically lists individual expenses <a class="yt-timestamp" data-t="00:00:16">[00:00:16]</a>. For each expense, the respective category and the amount are recorded <a class="yt-timestamp" data-t="00:00:20">[00:00:20]</a>.

### Key Features
*   **Expense Entry**
    *   Users can input individual expense amounts into the tracker <a class="yt-timestamp" data-t="00:00:40">[00:00:40]</a>.
*   **Category Management**
    *   Users can define their own expense categories by clicking a pencil icon <a class="yt-timestamp" data-t="00:00:28">[00:00:28]</a>.
    *   New categories can be added via a "new page" option <a class="yt-timestamp" data-t="00:01:06">[00:01:06]</a>.
    *   Existing categories can be edited using a pencil icon <a class="yt-timestamp" data-t="00:01:08">[00:01:08]</a>.
    *   Categories can be deleted via a three-dot menu <a class="yt-timestamp" data-t="00:01:10">[00:01:10]</a>.
*   **Budgeted Expense Values**
    *   The template allows users to [[setting_and_tracking_budgeted_expenses | update budgeted expense values]] directly within the system <a class="yt-timestamp" data-t="00:00:32">[00:00:32]</a>. Any updates made are reflected instantly <a class="yt-timestamp" data-t="00:00:36">[00:00:36]</a>.

## Summary Section
The summary section provides a concise financial overview, enabling [[monitoring_budget_versus_actual_expenses | monitoring budget versus actual expenses]] <a class="yt-timestamp" data-t="00:00:49">[00:00:49]</a>. It includes:
*   The [[setting_budget_and_goal_amounts_for_different_components | budgeted expense amount]] <a class="yt-timestamp" data-t="00:00:48">[00:00:48]</a>.
*   The actual expenses incurred <a class="yt-timestamp" data-t="00:00:49">[00:00:49]</a>.
*   The difference between the budgeted and actual expenses <a class="yt-timestamp" data-t="00:00:51">[00:00:51]</a>.
*   The percentage of expenses relative to the budgeted amount <a class="yt-timestamp" data-t="00:00:53">[00:00:53]</a>.
This section offers a [[summary_of_expenses_and_budget_comparison | total expense summary]] <a class="yt-timestamp" data-t="00:00:54">[00:00:54]</a>, illustrating how current expenditures compare to the budget <a class="yt-timestamp" data-t="00:00:59">[00:00:59]</a>.
