This article explains how to generate PDF payment receipt documents by integrating a Notion database with a Notion template using PDF output.com <a class="yt-timestamp" data-t="00:00:05">[00:00:05]</a>. ## Getting Started with PDF output.com

To begin, log in to PDF output.com <a class="yt-timestamp" data-t="00:00:11">[00:00:11]</a>. From the interface, click on "Templates" <a class="yt-timestamp" data-t="00:00:17">[00:00:17]</a>. This will open a new page listing all available templates for PDF output <a class="yt-timestamp" data-t="00:00:22">[00:00:22]</a>.

You can use the search option to find specific templates, for example, by typing "payment receipts PDF generator" <a class="yt-timestamp" data-t="00:00:32">[00:00:32]</a>, <a class="yt-timestamp" data-t="00:00:38">[00:00:38]</a>. Filter and sorting options are also available <a class="yt-timestamp" data-t="00:00:49">[00:00:49]</a>.

Once you've found the desired template, such as the "payment receipts PDF generator," click on the "download" link next to it <a class="yt-timestamp" data-t="00:00:52">[00:00:52]</a>, <a class="yt-timestamp" data-t="00:00:56">[00:00:56]</a>. This will open a new page displaying the associated Notion database and template <a class="yt-timestamp" data-t="00:01:03">[00:01:03]</a>.

## Understanding Notion Database and Template Structure

The template contains predefined elements for generating PDF output <a class="yt-timestamp" data-t="00:01:22">[00:01:22]</a>. Elements enclosed in curly braces, like `{receipt number}`, `{receipt date}`, `{customer name}`, and `{amount paid}`, are placeholder text elements <a class="yt-timestamp" data-t="00:01:28">[00:01:28]</a>. These placeholders will be replaced with corresponding information from the Notion database <a class="yt-timestamp" data-t="00:01:40">[00:01:40]</a>, <a class="yt-timestamp" data-t="00:01:44">[00:01:44]</a>.

For successful replacement, the placeholder text within the curly braces in the template must *exactly match* the corresponding column names (properties) in your Notion database <a class="yt-timestamp" data-t="00:02:02">[00:02:02]</a>, <a class="yt-timestamp" data-t="00:02:06">[00:02:06]</a>, <a class="yt-timestamp" data-t="00:05:07">[00:05:07]</a>. This setup facilitates [[using_notion_databases_for_income_details | using Notion databases for income details]] and [[using_databases_to_manage_invoice_details_in_notion | managing invoice details]].

### Template Customization

The entire template is customizable to your specific requirements <a class="yt-timestamp" data-t="00:04:21">[00:04:21]</a>. You can format elements (e.g., make them bold) and adjust spacing directly within the Notion template <a class="yt-timestamp" data-t="00:04:27">[00:04:27]</a>, <a class="yt-timestamp" data-t="00:04:30">[00:04:30]</a>. Remember to keep placeholder elements within curly braces for proper syncing <a class="yt-timestamp" data-t="00:04:43">[00:04:43]</a>, <a class="yt-timestamp" data-t="00:04:47">[00:04:47]</a>. This is key for [[using_notion_for_invoice_template_creation | invoice template creation]] and [[using_notion_templates_for_invoice_pdfs | creating invoice PDFs]].

## Duplicating the Template to Your Notion Workspace

If the template isn't yet published to the Notion Gallery, you'll see a "duplicate" option <a class="yt-timestamp" data-t="00:02:24">[00:02:24]</a>, <a class="yt-timestamp" data-t="00:02:30">[00:02:30]</a>. Click "Duplicate" and choose your desired Notion workspace where you want to add the template <a class="yt-timestamp" data-t="00:02:35">[00:02:35]</a>, <a class="yt-timestamp" data-t="00:02:40">[00:02:40]</a>. If it's published, click "Start with this template" <a class="yt-timestamp" data-t="00:02:43">[00:02:43]</a>. The template needs to be in your Notion workspace to be used with PDF output <a class="yt-timestamp" data-t="00:02:50">[00:02:50]</a>.

## Connecting Notion to PDF output.com

After duplicating the template, navigate back to PDF output.com and click on "Settings" <a class="yt-timestamp" data-t="00:03:22">[00:03:22]</a>, <a class="yt-timestamp" data-t="00:03:26">[00:03:26]</a>. Then, click "Click here to add templates" <a class="yt-timestamp" data-t="00:03:32">[00:03:32]</a>.

You will be prompted to choose a Notion workspace; ensure you select the same workspace where you duplicated the template <a class="yt-timestamp" data-t="00:03:40">[00:03:40]</a>, <a class="yt-timestamp" data-t="00:03:44">[00:03:44]</a>. Select the specific template page (e.g., "payment receipts PDF generator file") <a class="yt-timestamp" data-t="00:03:56">[00:03:56]</a>. Click "Allow access" to enable PDF output to read your Notion template and database elements <a class="yt-timestamp" data-t="00:04:02">[00:04:02]</a>. Authentication success will lead you back to PDF output.com <a class="yt-timestamp" data-t="00:04:08">[00:04:08]</a>. This is part of [[setting_up_a_notion_database_and_template_for_pdf_generation | setting up a Notion database and template for PDF generation]].

## Mapping Database Properties to Template Placeholders

On PDF output.com, refresh the screen if needed to load elements <a class="yt-timestamp" data-t="00:05:20">[00:05:20]</a>.

1.  From the drop-down menu, select your Notion database (e.g., "payment receipts database") <a class="yt-timestamp" data-t="00:05:25">[00:05:25]</a>, <a class="yt-timestamp" data-t="00:05:26">[00:05:26]</a>.
2.  Then, select the corresponding Notion template (e.g., "payment receipt template") <a class="yt-timestamp" data-t="00:05:30">[00:05:30]</a>, <a class="yt-timestamp" data-t="00:05:32">[00:05:32]</a>.
3.  Give the connection a name (e.g., "payment receipt") and click "Next" <a class="yt-timestamp" data-t="00:05:35">[00:05:35]</a>, <a class="yt-timestamp" data-t="00:05:37">[00:05:37]</a>.

The system will load and automatically map the Notion database properties (columns) to the template elements if the naming conventions match <a class="yt-timestamp" data-t="00:05:40">[00:05:40]</a>, <a class="yt-timestamp" data-t="00:05:47">[00:05:47]</a>. If any elements are mismatched, you can manually search and map them <a class="yt-timestamp" data-t="00:06:04">[00:06:04]</a>, <a class="yt-timestamp" data-t="00:06:06">[00:06:06]</a>. Filter, search, and sorting options are available in this view <a class="yt-timestamp" data-t="00:06:19">[00:06:19]</a>. This mapping process is crucial for [[using_notion_templates_and_databases_for_pdf_generation | using Notion templates and databases for PDF generation]].

## Generating PDFs

Once everything is correctly set up, click "Export" <a class="yt-timestamp" data-t="00:06:39">[00:06:39]</a>, <a class="yt-timestamp" data-t="00:06:41">[00:06:41]</a>.

Upon export, a "PDF status" column will be added to your Notion database <a class="yt-timestamp" data-t="00:06:43">[00:06:43]</a>, <a class="yt-timestamp" data-t="00:06:45">[00:06:45]</a>. As PDFs are generated for each row, the corresponding box in this column will be ticked <a class="yt-timestamp" data-t="00:06:50">[00:06:50]</a>, <a class="yt-timestamp" data-t="00:06:57">[00:06:57]</a>. A confirmation of "export successful" will appear <a class="yt-timestamp" data-t="00:07:04">[00:07:04]</a>.

You can then preview a sample of the generated PDF <a class="yt-timestamp" data-t="00:07:07">[00:07:07]</a> or click "Download all" to get all generated files <a class="yt-timestamp" data-t="00:07:17">[00:07:17]</a>. Each generated file corresponds to a row in the Notion database, with all elements placed correctly <a class="yt-timestamp" data-t="00:07:36">[00:07:36]</a>. This entire process demonstrates [[using_a_notion_database_and_templates_to_create_pdf_invoices | using a Notion database and templates to create PDF invoices]].
