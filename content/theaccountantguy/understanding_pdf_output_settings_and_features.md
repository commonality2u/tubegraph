[[Introduction to PDFOutput tool | PDFOutput]] is a tool designed to generate PDF documents from Notion databases using predefined templates <a class="yt-timestamp" data-t="00:00:04"></a>. This article outlines the key settings and features available for [[using_pdfoutput_features_and_settings | customizing]] and exporting PDF documents.

## Getting Started with PDFOutput

To begin [[using_pdf_output_interface_for_exporting_documents | exporting documents]], users first need to log into PDFOutput.com <a class="yt-timestamp" data-t="00:00:11"></a>.

### Accessing Templates

Upon logging in, users can access a variety of available templates by clicking on the "templates" option in the interface <a class="yt-timestamp" data-t="00:00:17"></a>. These templates are pre-designed for various purposes, such as generating payment receipts <a class="yt-timestamp" data-t="00:00:32"></a>.

### Searching, Filtering, and Sorting Templates

The template section offers several options to efficiently find the desired template:
*   **Search Option:** Users can search for specific templates by typing a name <a class="yt-timestamp" data-t="00:00:36"></a>.
*   **Filter Option:** Available for narrowing down template choices <a class="yt-timestamp" data-t="00:00:49"></a>.
*   **Sorting Option:** Allows users to sort templates as needed <a class="yt-timestamp" data-t="00:00:49"></a>.

### Duplicating Templates to Notion

Before a template can be used with [[Introduction to PDFOutput tool | PDFOutput]], it must be duplicated to a Notion workspace <a class="yt-timestamp" data-t="00:02:50"></a>. If a template is not yet published to the Notion Gallery, users will find a "duplicate" option <a class="yt-timestamp" data-t="00:02:35"></a>. If it's published, the option will be "start with this template" <a class="yt-timestamp" data-t="00:02:31"></a>. Users must select the Notion workspace where they want the template to be duplicated <a class="yt-timestamp" data-t="00:02:38"></a>.

## Template Structure and Customization

PDFOutput templates, like the payment receipt generator, are designed to pull information from a Notion database <a class="yt-timestamp" data-t="00:01:06"></a>.

### Placeholder Elements

Within the template, specific elements enclosed in curly braces (e.g., `{receipt number}`, `{customer name}`) serve as placeholder text <a class="yt-timestamp" data-t="00:01:38"></a>. These placeholders are dynamically replaced by corresponding information from the Notion database <a class="yt-timestamp" data-t="00:01:40"></a>.

### Customizing Templates

Templates are highly [[customizing_document_settings_and_layout_in_pdf_generation | customizable]] to fit specific requirements <a class="yt-timestamp" data-t="00:04:20"></a>. Users can:
*   Make elements bold <a class="yt-timestamp" data-t="00:04:30"></a>.
*   Adjust spacing <a class="yt-timestamp" data-t="00:04:35"></a>.

### Matching Naming Conventions

For proper synchronization and error-free PDF generation, it is crucial that the exact naming convention of placeholder elements in the template matches the corresponding column names in the Notion database <a class="yt-timestamp" data-t="00:02:06"></a>, <a class="yt-timestamp" data-t="00:04:43"></a>.

## Configuring PDF Output for Generation

After duplicating the template to Notion, the next step is to configure it within [[Introduction to PDFOutput tool | PDFOutput]].

### Adding Templates to PDF Output

Navigate to the settings in PDFOutput and click "click here to add templates" to import the duplicated Notion template <a class="yt-timestamp" data-t="00:03:21"></a>.

### Selecting Notion Workspace

When prompted, select the specific Notion workspace where the template was duplicated <a class="yt-timestamp" data-t="00:03:40"></a>.

### Mapping Database Properties to Template Elements

[[Customizing and exporting PDF documents with PDF output tool | PDFOutput]] automatically attempts to map Notion database properties (columns) to the template's placeholder elements based on matching names <a class="yt-timestamp" data-t="00:05:40"></a>. If any elements are mismatched, users can manually search and select the correct corresponding element <a class="yt-timestamp" data-t="00:06:04"></a>. This mapping interface also includes filter, search, and sorting options <a class="yt-timestamp" data-t="00:06:19"></a>.

### Exporting and PDF Status Column

Upon initiating the export process, [[Introduction to PDFOutput tool | PDFOutput]] automatically adds a "PDF status" column to the Notion database <a class="yt-timestamp" data-t="00:06:43"></a>. This column denotes which rows have had their PDFs generated <a class="yt-timestamp" data-t="00:06:50"></a>.

## Generated PDF Management

Once the export is successful, users can manage the generated PDF files.

### Previewing and Downloading

Users can preview a sample of the generated PDF <a class="yt-timestamp" data-t="00:07:07"></a>. Additionally, there is an option to download all generated files, which correspond to individual rows in the Notion database <a class="yt-timestamp" data-t="00:07:17"></a>, <a class="yt-timestamp" data-t="00:07:36"></a>.

## Advanced Features and Settings

[[Troubleshooting and Customization Options in PDF Output | PDFOutput]] offers additional features and settings to enhance the user experience.

### Connections

The "connections" section allows users to quickly reload past database and template connections, eliminating the need to manually add them each time for subsequent generations <a class="yt-timestamp" data-t="00:07:56"></a>.

### Subscription Plans and Watermarks

Users on the free plan will find a [[understanding_pdfoutput_subscription_plans_and_features | watermark]] of [[Introduction to PDFOutput tool | PDFOutput]] on their generated PDFs <a class="yt-timestamp" data-t="00:08:24"></a>. Upgrading to a paid subscription plan removes this watermark, providing a clean output <a class="yt-timestamp" data-t="00:08:31"></a>.

### Page Format

Within the settings, users can change the page format of the generated PDFs <a class="yt-timestamp" data-t="00:08:42"></a>.

### Feedback and Help

*   **Feedback Option:** Users can send messages directly to the support team for assistance <a class="yt-timestamp" data-t="00:08:50"></a>.
*   **Help Section:** Contains a demonstration video that guides users through the setup process for generating PDF documents <a class="yt-timestamp" data-t="00:08:59"></a>.
