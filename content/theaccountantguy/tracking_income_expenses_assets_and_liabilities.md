This Notion template is designed to help small businesses keep [[tracking_income_and_expenses | track of all their bookkeeping needs]] <a class="yt-timestamp" data-t="00:00:20">[00:00:20]</a>. It is divided into three main sections:
1.  **Start Here**: For initial setup and business information <a class="yt-timestamp" data-t="00:00:27">[00:00:27]</a>.
2.  **Summary**: Provides an overview of all entered transactions <a class="yt-timestamp" data-t="00:00:30">[00:00:30]</a>.
3.  **Transactions Window**: Where all business transaction details are entered <a class="yt-timestamp" data-t="00:00:35">[00:00:35]</a>.

## Start Here Section

This section allows users to input all necessary business information to begin using the template <a class="yt-timestamp" data-t="00:00:40">[00:00:40]</a>.

### Account Setup
The account section is categorized into four main heads:
*   Expenses <a class="yt-timestamp" data-t="00:00:50">[00:00:50]</a>
*   Liabilities <a class="yt-timestamp" data-t="00:00:51">[00:00:51]</a>
*   Income <a class="yt-timestamp" data-t="00:00:52">[00:00:52]</a>
*   Assets <a class="yt-timestamp" data-t="00:00:53">[00:00:53]</a>

Under each head, different accounts can be set up, such as "purchase expense," "utility bills," "subscription," and "salaries and wages" for expenses, or "accounts payable," "taxes payable," and "accrued expenses" for liabilities <a class="yt-timestamp" data-t="00:01:04">[00:01:04]</a>.

#### Managing Accounts
*   **Editing**: Hover over an item, click the pencil icon, edit details, and click anywhere outside to save <a class="yt-timestamp" data-t="00:01:10">[00:01:10]</a>.
*   **Deleting**: Click the three-dot menu next to an item and select "delete" <a class="yt-timestamp" data-t="00:01:21">[00:01:21]</a>.
*   **Adding**: Go to the respective category and click "new page" to add a new account <a class="yt-timestamp" data-t="00:01:28">[00:01:28]</a>.

#### Account Properties
For every added account, several properties can be defined:
*   **Opening Balance**: For liabilities and [[assets_and_liabilities_database_setup | assets and liabilities]] that carry forward balances <a class="yt-timestamp" data-t="00:01:46">[00:01:46]</a>.
*   **Classification**: Classify items as bills, debt, savings, or other relevant categories <a class="yt-timestamp" data-t="00:01:53">[00:01:53]</a>. For expenses, classifications like "bills" for utility bills or "subscription" for subscriptions are used <a class="yt-timestamp" data-t="00:02:01">[00:02:01]</a>.
*   **Payment Frequency**: Specify how often a payment occurs (e.g., monthly, half-yearly, quarterly) <a class="yt-timestamp" data-t="00:02:12">[00:02:12]</a>.
*   **Opening Units**: Starting units for inventory or [[asset_financials_and_depreciation | assets]] when beginning the business <a class="yt-timestamp" data-t="00:02:27">[00:02:27]</a>.
*   **Asset Class**: Classify [[asset_financials_and_depreciation | assets]] as either "inventory" or "investment" <a class="yt-timestamp" data-t="00:02:42">[00:02:42]</a>.

### Net Worth Goal
Users can set a net worth goal at the top of the template <a class="yt-timestamp" data-t="00:09:00">[00:09:00]</a>.

## Transactions Window

This section is where all business transactions are recorded. It has dedicated buttons for adding different types of transactions:
*   Add Income <a class="yt-timestamp" data-t="00:03:10">[00:03:10]</a>
*   Add Expense <a class="yt-timestamp" data-t="00:06:02">[00:06:02]</a>
*   Add Asset <a class="yt-timestamp" data-t="00:06:04">[00:06:04]</a>
*   Add Liability <a class="yt-timestamp" data-t="00:06:05">[00:06:05]</a>

When adding an income transaction, it automatically sets the current date <a class="yt-timestamp" data-t="00:03:16">[00:03:16]</a>.

### Transaction Fields
When [[using_the_transactions_tab_for_income_and_expenses | using the transactions tab for income and expenses]], the following details can be entered for each transaction:
*   **Date**: Automatically set to the current date when adding a new transaction <a class="yt-timestamp" data-t="00:03:16">[00:03:16]</a>.
*   **Period**: Users can choose or add the relevant month and year (e.g., "October 2024") <a class="yt-timestamp" data-t="00:03:24">[00:03:24]</a>.
*   **Description**: A detailed description of the transaction <a class="yt-timestamp" data-t="00:03:42">[00:03:42]</a>.
*   **Category**: Automatically mapped (e.g., "income" if "Add Income" was clicked) <a class="yt-timestamp" data-t="00:03:45">[00:03:45]</a>.
*   **Account**: Link to the accounts set up in the "Start Here" section <a class="yt-timestamp" data-t="00:03:50">[00:03:50]</a>. Adding an account here will also reflect it in the "Start Here" section <a class="yt-timestamp" data-t="00:04:05">[00:04:05]</a>.
*   **Payment Mode**: How payment was received or made (e.g., bank account, cash) <a class="yt-timestamp" data-t="00:04:21">[00:04:21]</a>. More accounts can be added as needed <a class="yt-timestamp" data-t="00:04:28">[00:04:28]</a>.
*   **Transaction Type**: Specifies if money is coming into (inflow) or going out of (outflow) the account <a class="yt-timestamp" data-t="00:04:51">[00:04:51]</a>.
*   **Amount**: The monetary value of the transaction <a class="yt-timestamp" data-t="00:05:07">[00:05:07]</a>.
*   **Document**: Option to attach any relevant documents <a class="yt-timestamp" data-t="00:05:15">[00:05:15]</a>.
*   **Tax Rate and Tax Amount**: Automatically calculates tax amount based on the entered rate. Can be set to 0% for non-taxable transactions <a class="yt-timestamp" data-t="00:05:21">[00:05:21]</a>.
*   **Multiplier and Currency**: For transactions involving multiple currencies, a multiplier can be added to convert to the base currency (e.g., USD to Euro) <a class="yt-timestamp" data-t="00:05:36">[00:05:36]</a>.

### Investment Tracking
The template also includes sections for tracking investments, showing units, price, total value, and investment value <a class="yt-timestamp" data-t="00:06:09">[00:06:09]</a>. It allows users to track profit or loss when selling investments <a class="yt-timestamp" data-t="00:06:17">[00:06:17]</a>.

### Inventory Management
A dedicated section for inventory lists all current inventory, along with profit or loss obtained from specific inventory items <a class="yt-timestamp" data-t="00:06:26">[00:06:26]</a>.

## Utilizing the Template for Bookkeeping

To effectively [[tracking_transactions_and_expenses | track transactions and expenses]] and other financials using this small business bookkeeping template <a class="yt-timestamp" data-t="00:00:17">[00:00:17]</a>, the process involves:
1.  Setting up different accounts under the respective heads in the "Start Here" section <a class="yt-timestamp" data-t="00:10:00">[00:10:00]</a>.
2.  Adding opening balances for [[assets_and_liabilities_database_setup | assets and liabilities]], if applicable <a class="yt-timestamp" data-t="00:10:07">[00:10:07]</a>.
3.  Entering details for every business transaction in the "Transactions Window" <a class="yt-timestamp" data-t="00:10:12">[00:10:12]</a>.

These entries automatically populate the summary sections, providing an [[tracking_income_and_expenses_in_notion | entire glance of the business finances]] <a class="yt-timestamp" data-t="00:10:16">[00:10:16]</a>.
