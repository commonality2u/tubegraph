## Renaming Categories

To rename an existing category in the Notion expense tracker:
1.  Hover over the specific category <a class="yt-timestamp" data-t="00:00:25">[00:00:25]</a>.
2.  A pencil icon will appear <a class="yt-timestamp" data-t="00:00:26">[00:00:26]</a>.
3.  Click the pencil icon <a class="yt-timestamp" data-t="00:00:27">[00:00:27]</a>.
4.  Rename the category as desired <a class="yt-timestamp" data-t="00:00:31">[00:00:31]</a>. For example, a category can be renamed to "my category" <a class="yt-timestamp" data-t="00:00:33">[00:00:33]</a>.
5.  Press Enter to save the new category name <a class="yt-timestamp" data-t="00:00:37">[00:00:37]</a>.

## [[Adding and editing expense categories|Adding New Categories]]

To [[modifying_and_adding_new_expense_categories_in_notion|add a new category]] to the expense tracker:
*   Click the "new page" button <a class="yt-timestamp" data-t="00:00:48">[00:00:48]</a>.
*   A new category will be added <a class="yt-timestamp" data-t="00:00:50">[00:00:50]</a>.

## Deleting Categories

To [[adding_and_editing_expense_categories|delete an existing category]]:
1.  Click on the three dots next to the category name <a class="yt-timestamp" data-t="00:00:42">[00:00:42]</a>.
2.  Select "delete" <a class="yt-timestamp" data-t="00:00:44">[00:00:44]</a>.

## Category Functionality

Each category in the tracker displays both a budgeted expense amount and an actual expense amount <a class="yt-timestamp" data-t="00:00:53">[00:00:53]</a>. When expenses are entered, all categories are automatically updated with the respective figures <a class="yt-timestamp" data-t="00:01:05">[00:01:05]</a>, <a class="yt-timestamp" data-t="00:01:11">[00:01:11]</a>. Users can set their own budgeted expense amounts by clicking on the budget amount within each category <a class="yt-timestamp" data-t="00:01:44">[00:01:44]</a>.
