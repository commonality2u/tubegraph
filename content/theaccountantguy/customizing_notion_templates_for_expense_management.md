## Template Overview

The expense tracker template features a category section at the top, which displays three default categories <a class="yt-timestamp" data-t="00:00:17">[00:00:17]</a>. Each category can be defined according to specific requirements <a class="yt-timestamp" data-t="00:00:22">[00:00:22]</a>. For every category, you can see a budgeted expense amount and an actual expense amount <a class="yt-timestamp" data-t="00:00:52">[00:00:52]</a>.

At the bottom of the tracker, there's a detailed list where individual expenses are entered <a class="yt-timestamp" data-t="00:00:56">[00:00:56]</a>. As you input amounts into this list, the corresponding categories at the top automatically update with the respective actual expenses <a class="yt-timestamp" data-t="00:01:03">[00:01:03]</a> <a class="yt-timestamp" data-t="00:01:09">[00:01:09]</a>.

## Customizing Categories

[[Customizing Notion templates | Customizing the Notion template]] for expense management allows for precise tracking tailored to your financial habits.

### Renaming Categories

To rename a category:
1. Hover over the desired category until a pencil icon appears <a class="yt-timestamp" data-t="00:00:25">[00:00:25]</a>.
2. Click the pencil icon <a class="yt-timestamp" data-t="00:00:27">[00:00:27]</a>.
3. Type the new name for the category <a class="yt-timestamp" data-t="00:00:29">[00:00:29]</a>.
4. Press Enter to save the changes <a class="yt-timestamp" data-t="00:00:35">[00:00:35]</a>.

### Editing and Deleting Categories

If you need to edit or delete an existing category:
1. Click on the three dots next to the category <a class="yt-timestamp" data-t="00:00:41">[00:00:41]</a>.
2. Select "Delete" to remove it <a class="yt-timestamp" data-t="00:00:44">[00:00:44]</a>.

### Adding New Categories

To add a new category:
1. Simply click on the "New Page" button <a class="yt-timestamp" data-t="00:00:48">[00:00:48]</a>. This will allow you to create a new category page.

## Managing Budgets

The template includes a comprehensive summary section that provides an overview of your financial status <a class="yt-timestamp" data-t="00:01:27">[00:01:27]</a>.

### Setting Budgeted Amounts

You can set your own budgeted expense amount for each category:
1. Go to the category section <a class="yt-timestamp" data-t="00:01:41">[00:01:41]</a>.
2. Click on the "Budgeted Expense Amount" for a specific category <a class="yt-timestamp" data-t="00:01:42">[00:01:42]</a>.
3. Enter your desired budget figure <a class="yt-timestamp" data-t="00:01:46">[00:01:46]</a>. The total budget displayed will update accordingly <a class="yt-timestamp" data-t="00:01:50">[00:01:50]</a>.

### Summary Section Details

The summary section provides:
*   The total budgeted expense amount <a class="yt-timestamp" data-t="00:01:28">[00:01:28]</a>.
*   The actual expenses amount, which is computed from the detailed expense table <a class="yt-timestamp" data-t="00:01:30">[00:01:30]</a> <a class="yt-timestamp" data-t="00:01:33">[00:01:33]</a>.
*   The difference between the budgeted and actual expenses <a class="yt-timestamp" data-t="00:01:35">[00:01:35]</a>.
*   The percentage of the budget that has been spent so far <a class="yt-timestamp" data-t="00:01:37">[00:01:37]</a>.

The budgeted amount displayed in the summary will reflect the figures you set in the category section <a class="yt-timestamp" data-t="00:01:54">[00:01:54]</a>.
