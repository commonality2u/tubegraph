This template functions as a [[payments_tracker | payments tracker]] designed to help users keep track of their house rental payments <a class="yt-timestamp" data-t="00:00:02">[00:00:02]</a>.

## Summary Section Views

The template features a summary section with four distinct views <a class="yt-timestamp" data-t="00:00:10">[00:00:10]</a>.

### Basic Information & Tenant Management

This view allows for the addition of basic information, including tenants, who can be categorized under different property classifications <a class="yt-timestamp" data-t="00:00:13">[00:00:13]</a>. Property classifications, such as "apartment" and "duplex," can be customized by the user <a class="yt-timestamp" data-t="00:00:20">[00:00:20]</a>.

To add a new tenant:
1.  Click the "ADD tenant" button on the left <a class="yt-timestamp" data-t="00:00:28">[00:00:28]</a>.
2.  Click the three-dot icon to position the new tenant under the appropriate category <a class="yt-timestamp" data-t="00:00:33">[00:00:33]</a>.
3.  Click the pencil icon to input the tenant's details <a class="yt-timestamp" data-t="00:00:37">[00:00:37]</a>.

### Property Section

This section displays different property classifications <a class="yt-timestamp" data-t="00:00:46">[00:00:46]</a>. Underneath each property, users can view:
*   The tenant's name <a class="yt-timestamp" data-t="00:00:51">[00:00:51]</a>
*   Total rent amount <a class="yt-timestamp" data-t="00:00:54">[00:00:54]</a>
*   Rent amount that has been paid <a class="yt-timestamp" data-t="00:00:54">[00:00:54]</a>
*   Balance rent amount that is due <a class="yt-timestamp" data-t="00:00:56">[00:00:56]</a>
*   Advanced rent that has been paid <a class="yt-timestamp" data-t="00:00:58">[00:00:58]</a>

### Rental History

The rental history view shows the past payment records for all tenants, based on the transactions logged at the bottom of the template <a class="yt-timestamp" data-t="00:01:04">[00:01:04]</a>.

### Overall Summary

This final summary section provides an overview of:
*   Total amount due <a class="yt-timestamp" data-t="00:01:14">[00:01:14]</a>
*   Amount that has been paid <a class="yt-timestamp" data-t="00:01:15">[00:01:15]</a>
*   Outstanding rent amount <a class="yt-timestamp" data-t="00:01:16">[00:01:16]</a>
*   Percentage of rent collected <a class="yt-timestamp" data-t="00:01:17">[00:01:17]</a>

## Transaction Section Views

The transaction section offers three different views, in addition to a view showing all transactions <a class="yt-timestamp" data-t="00:01:20">[00:01:20]</a>.

### Rent Paid

This view displays all transactions for which rent has been paid, along with the respective tenants and amounts <a class="yt="yt-timestamp" data-t="00:01:23">[00:01:23]</a>.

### Rent Due

This section shows information for rent that is due and has not yet been paid <a class="yt-timestamp" data-t="00:01:31">[00:01:31]</a>. Checking a box for an entry in this view indicates that the rent has been paid, moving it to the "Rent Paid" category <a class="yt-timestamp" data-t="00:01:36">[00:01:36]</a>.

### Quarterly Summary

The quarterly summary provides an overview of rental payments and collections, as well as the balance due for each month within a given quarter <a class="yt-timestamp" data-t="00:01:41">[00:01:41]</a>.

## Adding a Payment

To quickly add a rental payment, click the designated button within the template <a class="yt-timestamp" data-t="00:01:50">[00:01:50]</a>.
