From: [[theaccountantguy]] <br/> 

Anubhav Biswal, known as "the accountant guy," explains how to utilize a Notion template for [[creating_a_savings_funds_tracker_in_notion | tracking emergency funds]] <a class="yt-timestamp" data-t="00:00:07">[00:00:07]</a>. The template is designed to help users [[managing_finances_with_notion | manage their finances]] and ensure they meet their emergency savings goals <a class="yt-timestamp" data-t="00:03:39">[00:03:39]</a>.

## Template Overview
The Notion template for emergency funds includes three main sections:
*   **Fund Setup** <a class="yt-timestamp" data-t="00:00:13">[00:00:13]</a>
*   **Savings Overview** <a class="yt-timestamp" data-t="00:00:15">[00:00:15]</a>
*   **Summary Section** <a class="yt-timestamp" data-t="00:00:19">[00:00:19]</a>

## Fund Setup
The "Fund Setup" section allows users to define and organize their emergency funds <a class="yt-timestamp" data-t="00:00:26">[00:00:26]</a>.
*   **Categories**: Default categories include Financial Safety, Asset Protection, and Health and Wellness <a class="yt-timestamp" data-t="00:00:30">[00:00:30]</a>. Users can edit these categories or add new ones <a class="yt-timestamp" data-t="00:00:35">[00:00:35]</a>.
*   **Adding a New Fund**: Click the "Add Fund" button to create a new entry, which initially appears under a "No classification" tab <a class="yt-timestamp" data-t="00:00:51">[00:00:51]</a>. Users can then move it to the appropriate category <a class="yt-timestamp" data-t="00:01:02">[00:01:02]</a>.
*   **Fund Details**: For each fund, users can specify:
    *   **Fund Detail**: A description of the fund <a class="yt-timestamp" data-t="00:01:14">[00:01:14]</a>.
    *   **Starting Balance**: The initial amount in the fund <a class="yt-timestamp" data-t="00:01:17">[00:01:17]</a>.
    *   **Goal Amount**: The target amount of savings desired for this emergency fund <a class="yt-timestamp" data-t="00:01:22">[00:01:22]</a>.
    *   **Start Date**: When contributions to this fund began <a class="yt-timestamp" data-t="00:01:32">[00:01:32]</a>.
    *   **Target End Date**: The date by which the total emergency fund is aimed to be available <a class="yt-timestamp" data-t="00:01:48">[00:01:48]</a>.
*   The system automatically calculates the savings made to date based on these inputs <a class="yt-timestamp" data-t="00:01:42">[00:01:42]</a>.

## Contributions
The "Contributions" section is where users record money put into their emergency savings funds <a class="yt-timestamp" data-t="00:02:17">[00:02:17]</a>.
*   **Monthly View**: Contributions are organized by month, with the latest month appearing at the top <a class="yt-timestamp" data-t="00:02:37">[00:02:37]</a>.
*   **Contribution Details**: For each entry, users provide:
    *   **Date**: The date of the contribution <a class="yt-timestamp" data-t="00:02:45">[00:02:45]</a>.
    *   **Description**: A brief note about the contribution <a class="yt-timestamp" data-t="00:02:46">[00:02:46]</a>.
    *   **Period**: The specific month and year of the contribution (predefined for 2023, but new months can be added) <a class="yt-timestamp" data-t="00:02:48">[00:02:48]</a>.
    *   **Fund**: Select the specific emergency fund to which the contribution is made <a class="yt-timestamp" data-t="00:03:13">[00:03:13]</a>.
    *   **Amount**: The amount of savings or contribution for that month <a class="yt-timestamp" data-t="00:03:24">[00:03:24]</a>.
*   **Monthly Requirement**: A "Requirement" column automatically calculates the monthly contribution needed to meet the fund's goal by the targeted end date <a class="yt-timestamp" data-t="00:03:35">[00:03:35]</a>.
*   **Progress Tracking**: If the entered amount meets or exceeds the required amount, a tick mark appears, indicating the monthly target has been met <a class="yt-timestamp" data-t="00:04:00">[00:04:00]</a>.
*   **Quick Action Button**: A "Quick Action" button automatically adds a new line item with the current month and date, simplifying data entry <a class="yt-timestamp" data-t="00:04:31">[00:04:31]</a>.

## Quarterly Overview
The "Quarterly Overview" provides a glance at contributions across all months within each quarter <a class="yt-timestamp" data-t="00:05:09">[00:05:09]</a>.
*   It shows the total savings made and the total required savings for each month <a class="yt-timestamp" data-t="00:05:30">[00:05:30]</a>.
*   A percentage indicates how much of the savings requirement has been met (e.g., 140% if exceeded, 71% if below target) <a class="yt-timestamp" data-t="00:06:04">[00:06:04]</a>.
*   Colors are used to provide a quick visual snapshot of financial progress <a class="yt-timestamp" data-t="00:06:29">[00:06:29]</a>.
*   The total savings for each quarter are also displayed <a class="yt-timestamp" data-t="00:06:41">[00:06:41]</a>.

## Savings Overview
The "Savings Overview" section offers a detailed breakdown for each emergency fund <a class="yt-timestamp" data-t="00:06:58">[00:06:58]</a>.
*   **Total Savings**: Shows the accumulated savings for each fund, combining the starting balance and all contributions <a class="yt-timestamp" data-t="00:07:09">[00:07:09]</a>.
*   **Targeted Savings**: The goal amount set for each fund, pulled directly from the "Fund Setup" <a class="yt-timestamp" data-t="00:07:35">[00:07:35]</a>.
*   **Achievement Percentage**: Indicates the percentage of the total savings requirement achieved <a class="yt-timestamp" data-t="00:07:47">[00:07:47]</a>.
*   **Targeted Savings Date**: The end goal date for the fund <a class="yt-timestamp" data-t="00:07:50">[00:07:50]</a>.
*   **Months Left**: Calculates the remaining months until the targeted end date from the current date <a class="yt-timestamp" data-t="00:08:06">[00:08:06]</a>. This value updates automatically over time <a class="yt-timestamp" data-t="00:09:03">[00:09:03]</a>.
*   **Contribution Required Per Month**: Shows the calculated monthly contribution needed to reach the target by the end date <a class="yt-timestamp" data-t="00:08:31">[00:08:31]</a>.

## Summary Section
The final "Summary Section" provides a high-level aggregate view of all emergency funds <a class="yt-timestamp" data-t="00:09:17">[00:09:17]</a>.
*   **Total Savings Made So Far**: The sum of the opening balances of all funds and all recorded contributions <a class="yt-timestamp" data-t="00:09:20">[00:09:20]</a>.
*   **Savings Required**: The total of all targeted savings amounts across all funds <a class="yt-timestamp" data-t="00:09:33">[00:09:33]</a>.
*   **Difference**: The total savings made minus the total savings required <a class="yt-timestamp" data-t="00:09:42">[00:09:42]</a>.
*   **Percentage**: The total savings made as a percentage of the total targeted savings amount <a class="yt-timestamp" data-t="00:09:51">[00:09:51]</a>.
