[[utilizing_pdf_output_for_document_generation | PDF output.com]] is a solution designed to [[automating_pdf_document_generation | generate PDF documents]] from templates, particularly useful for purposes like purchase orders <a class="yt-timestamp" data-t="02:32:00">[02:32:00]</a>. It offers features for both single and bulk document creation <a class="yt-timestamp" data-t="02:29:00">[02:29:00]</a>.

## Accessing PDF Output.com
To use the tool, users must first log in to [[using_pdf_output_com_for_pdf_generation | PDF output.com]] <a class="yt-timestamp" data-t="02:56:00">[02:56:00]</a>. Upon logging in, an interface appears where users can select a template from a dropdown menu <a class="yt-timestamp" data-t="02:58:00">[02:58:00]</a>.

## Generating Single PDF Documents
[[setting_up_pdfoutput_for_document_generation | Generating PDF documents]] one by one is done in "single PDF mode" <a class="yt-timestamp" data-t="03:23:00">[03:23:00]</a>.

### Process
1.  **Select Template**: Choose the desired template, such as the "purchase order template" <a class="yt-timestamp" data-t="03:08:00">[03:08:00]</a>.
2.  **Select Currency**: Users can select their desired currency <a class="yt-timestamp" data-t="03:11:00">[03:11:00]</a>. If a currency is not available, it can be requested through a dedicated window <a class="yt-timestamp" data-t="03:14:00">[03:14:00]</a>. Templates can also be requested if needed <a class="yt-timestamp" data-t="03:19:00">[03:19:00]</a>.
3.  **Fill in Details**: Input all necessary information into the template fields <a class="yt-timestamp" data-t="03:36:00">[03:36:00]</a>. This data can be copied from an existing database, such as a purchase order tracker <a class="yt-timestamp" data-t="03:55:00">[03:55:00]</a>.
4.  **Download PDF**: Once all details are filled, clicking "download PDF" generates the document <a class="yt-timestamp" data-t="04:14:00">[04:14:00]</a>. The generated PDF is clean and precise, reflecting the chosen currency <a class="yt-timestamp" data-t="04:21:00">[04:21:00]</a>. Currency can be changed instantly, updating the document before re-downloading <a class="yt-timestamp" data-t="04:28:00">[04:28:00]</a>.

## Generating Bulk PDF Documents
For [[automating_pdf_document_generation | exporting multiple documents]] at once, users can switch to "bulk export mode" <a class="yt-timestamp" data-t="04:55:00">[04:55:00]</a>.

### Process
1.  **Fill Multiple Rows**: Input all details for multiple documents by adding new rows <a class="yt-timestamp" data-t="05:00:00">[05:00:00]</a>.
2.  **Download Options**:
    *   Download individual documents by clicking specific options <a class="yt-timestamp" data-t="05:08:00">[05:08:00]</a>.
    *   Download all documents at once by clicking "download all PDF" <a class="yt-timestamp" data-t="05:11:00">[05:11:00]</a>.

## Future Integrations and Upgrades
The [[integration_and_upgrades_in_pdf_output_tool | PDF Output tool]] plans to add Notion integration, allowing users to select a Notion database directly and export PDFs by reading information from it <a class="yt-timestamp" data-t="05:23:00">[05:23:00]</a>.
The platform will also feature more templates in the future <a class="yt-timestamp" data-t="05:54:00">[05:54:00]</a>.
