This article demonstrates how to utilize a [[tracking_subscriptions_and_bill_payments_using_notion | Notion subscription expenses tracker]] to manage all your subscription expenses <a class="yt-timestamp" data-t="00:00:07">[00:00:07]</a>.

## Subscription Overview

The subscription overview section provides a quick glance at all subscriptions <a class="yt-timestamp" data-t="00:00:13">[00:00:13]</a>.

### Categories View
Subscriptions are divided into "active" and "cancel" subscriptions <a class="yt-timestamp" data-t="00:00:20">[00:00:20]</a>. Under each category, you will find:
*   The name of the subscription <a class="yt-timestamp" data-t="00:00:25">[00:00:25]</a>.
*   The total expenses incurred for the subscription <a class="yt-timestamp" data-t="00:00:26">[00:00:26]</a>.
*   The billing period for the subscription <a class="yt-timestamp" data-t="00:00:28">[00:00:28]</a>.
    *   The billing period can be changed by clicking on the option and selecting from available choices <a class="yt-timestamp" data-t="00:00:31">[00:00:31]</a>.

## Expenses Section

The [[tracking_expenses_and_income_in_notion | expenses section]] is divided into different sections based on subscription types, such as "infotainment," "education," and "music" <a class="yt-timestamp" data-t="00:00:43">[00:00:43]</a>.

### Managing Groups
*   **Add New Group:** You can add a new group by clicking the plus icon <a class="yt-timestamp" data-t="00:00:53">[00:00:53]</a>.
*   **Edit Existing Group:** Existing groups can be edited by clicking on them and changing the name <a class="yt-timestamp" data-t="00:00:58">[00:00:58]</a>.

### Classification Details
Under each subscription classification, you will find:
*   The name of the subscription <a class="yt-timestamp" data-t="00:01:05">[00:01:05]</a>.
*   The total expenses <a class="yt-timestamp" data-t="00:01:06">[00:01:06]</a>.
*   The percentage of each subscription towards the total subscription expenses incurred <a class="yt-timestamp" data-t="00:01:08">[00:01:08]</a>.

### Quick Action Button
A "quick action" button allows you to "add subscription" <a class="yt-timestamp" data-t="00:01:14">[00:01:14]</a>. Clicking this automatically adds a subscription under a "noted subscription" category <a class="yt-timestamp" data-t="00:01:18">[00:01:18]</a>. You can then move it to any classification of your choice using the three dots <a class="yt-timestamp" data-t="00:01:23">[00:01:23]</a>, and edit its details using the pencil icon <a class="yt-timestamp" data-t="00:01:29">[00:01:29]</a>.

## Transaction Section

The [[tracking_transactions_and_expenses_in_notion | transaction section]] is where you input all details to record subscription expenses <a class="yt-timestamp" data-t="00:01:37">[00:01:37]</a>.

### Views
*   **Active Subscriptions View:** Shows all currently active subscriptions <a class="yt-timestamp" data-t="00:01:44">[00:01:44]</a>.
*   **Cancelled Subscriptions View:** Shows subscriptions that have been cancelled <a class="yt-timestamp" data-t="00:01:52">[00:01:52]</a>.

Subscriptions are bifurcated by month, with the latest month appearing at the top, and the total sum of expenses incurred for that month displayed <a class="yt-timestamp" data-t="00:01:55">[00:01:55]</a>.

### Adding a Transaction
To add a transaction:
1.  **Status:** Click the checkbox to mark the subscription as active <a class="yt-timestamp" data-t="00:02:05">[00:02:05]</a>.
2.  **Subscription Account:** Select the subscription account from a list populated from subscriptions added previously <a class="yt-timestamp" data-t="00:02:13">[00:02:13]</a>.
3.  **Billing Period:** This is pulled automatically from the value set at the top <a class="yt-timestamp" data-t="00:02:21">[00:02:21]</a>.
4.  **Payment Date:** Choose the payment date for the subscription <a class="yt-timestamp" data-t="00:02:27">[00:02:27]</a>.
5.  **Amount:** Enter the amount of the subscription expense <a class="yt-timestamp" data-t="00:02:32">[00:02:32]</a>.
6.  **Renewal Date:** The renewal date is automatically calculated based on the billing period set <a class="yt-timestamp" data-t="00:02:35">[00:02:35]</a>. For example, a monthly Netflix subscription paid on September 30 will automatically show an October 30 renewal date <a class="yt-timestamp" data-t="00:02:40">[00:02:40]</a>.

### Cancelling a Subscription
To cancel a subscription, simply click its checkbox <a class="yt-timestamp" data-t="00:03:02">[00:03:02]</a>. This will automatically set its status to "cancel" and reflect it in the cancelled view <a class="yt-timestamp" data-t="00:03:04">[00:03:04]</a>.

### Quick Add Transaction
You can quickly add a subscription transaction by clicking the "space subscription" button <a class="yt-timestamp" data-t="00:03:16">[00:03:16]</a>. This adds a new entry where you can input details <a class="yt-timestamp" data-t="00:03:21">[00:03:21]</a>.
