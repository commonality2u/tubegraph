## Notion Sales Receipts Tracker Template

The template provides a structured way to record and summarize sales data, essential for [[sales_performance_tracking_using_notion | sales performance tracking using Notion]] <a class="yt-timestamp" data-t="00:00:06">[00:00:06]</a>.

### Database Columns

The primary database includes several key columns:
*   **Receipt Number** <a class="yt-timestamp" data-t="00:00:17">[00:00:17]</a>
*   **Receipt Date** <a class="yt-timestamp" data-t="00:00:17">[00:00:17]</a>
*   **Payment Method** – Allows selection from several options <a class="yt-timestamp" data-t="00:00:17">[00:00:17]</a>.
*   **Business Name** <a class="yt-timestamp" data-t="00:00:22">[00:00:22]</a>
*   **Address** <a class="yt-timestamp" data-t="00:00:24">[00:00:24]</a>
*   **Customer Details** – This is a relational property linked to another database containing client information <a class="yt-timestamp" data-t="00:00:27">[00:00:27]</a>.
*   **Description** of items sold <a class="yt-timestamp" data-t="00:00:38">[00:00:38]</a>
*   **Quantity** of sales <a class="yt-timestamp" data-t="00:00:41">[00:00:41]</a>
*   **Unit Price** <a class="yt-timestamp" data-t="00:00:42">[00:00:42]</a>
    *   Multiple descriptions, quantities, and unit prices can be added for a single sale <a class="yt-timestamp" data-t="00:00:45">[00:00:45]</a>.
*   **Subtotal** – Automatically sums the total amount of individual items sold <a class="yt-timestamp" data-t="00:01:00">[00:01:00]</a>.
*   **Tax Rate** <a class="yt-timestamp" data-t="00:01:11">[00:01:11]</a>
*   **Total Amount** – Computed automatically <a class="yt-timestamp" data-t="00:01:11">[00:01:11]</a>.
*   **Notes** – For any relevant information related to the payment <a class="yt-timestamp" data-t="00:01:14">[00:01:14]</a>.
*   **Additional Columns** – Linked to other databases to enhance understanding <a class="yt-timestamp" data-t="00:01:18">[00:01:18]</a>.

### Summary Sections

The template includes summary sections for a quick overview of sales performance:
*   **Sales Receipt Summary**:
    *   **Total Sales** – Drawn directly from the sales value <a class="yt-timestamp" data-t="00:01:28">[00:01:28]</a>.
    *   **Units Sold** – Represents the total quantity of sales <a class="yt="yt-timestamp" data-t="00:01:34">[00:01:34]</a>.
    *   **Average Sales Value** – Calculated as total sales divided by units sold <a class="yt-timestamp" data-t="00:02:10">[00:02:10]</a>. This provides key performance indicators for [[using_notion_for_business_tracking | business tracking using Notion]] <a class="yt-timestamp" data-t="00:02:16">[00:02:16]</a>.
    *   *Note*: If more columns for descriptions and quantities are added, the formulas for subtotal amount and total amount must be updated accordingly <a class="yt-timestamp" data-t="00:01:41">[00:01:41]</a>.

*   **Client Summary**:
    *   Shows total sales for each client <a class="yt-timestamp" data-t="00:02:21">[00:02:21]</a>.
    *   Displays the total quantity of sales made to each client <a class="yt-timestamp" data-t="00:02:33">[00:02:33]</a>.
    *   Calculates the average sales price for every client <a class="yt-timestamp" data-t="00:02:36">[00:02:36]</a>.

## Generating Sales Receipts PDF Documents

Beyond Notion, an external product like PDFOutput.com can be used to generate PDF sales receipts for business <a class="yt-timestamp" data-t="00:02:53">[00:02:53]</a>. This tool facilitates [[integrating_notion_database_with_receipt_templates | integrating a Notion database with receipt templates]] for PDF generation <a class="yt-timestamp" data-t="00:05:38">[00:05:38]</a>.

### Using PDF Output.com

1.  **Accessing Templates**: Navigate to PDFOutput.com and use the "template selection" dropdown to find and open the "sales receipts" template <a class="yt-timestamp" data-t="00:03:07">[00:03:07]</a>.
2.  **Filling Information**: Input details such as receipt number, date, payment method, business name, address, item descriptions, quantities, unit prices, tax rate, and notes <a class="yt-timestamp" data-t="00:03:30">[00:03:30]</a>.
3.  **Currency Selection**: The currency can be changed (e.g., from US Dollars to Euro), and it will automatically update in the generated PDF <a class="yt-timestamp" data-t="00:04:40">[00:04:40]</a>.
4.  **Downloading PDF**: After entering all details, click "Download PDF" to generate the sales receipt document <a class="yt-timestamp" data-t="00:04:53">[00:04:53]</a>.

### Bulk Export Mode

For multiple receipts, PDF Output.com offers a bulk export mode <a class="yt-timestamp" data-t="00:05:09">[00:05:09]</a>.

1.  **Database Input**: All sales receipt information can be added in a database format within the tool <a class="yt-timestamp" data-t="00:05:15">[00:05:15]</a>.
2.  **Data Management**:
    *   Column widths and row heights can be resized for readability <a class="yt-timestamp" data-t="00:05:42">[00:05:42]</a>.
    *   New rows can be added, and existing rows or individual items can be deleted <a class="yt-timestamp" data-t="00:05:58">[00:05:58]</a>.
3.  **Bulk Download**: Click "Download All PDF" to generate and download all sales receipts as individual PDF files in one go <a class="yt-timestamp" data-t="00:05:18">[00:05:18]</a>. Alternatively, individual PDFs can be downloaded by clicking on specific rows <a class="yt-timestamp" data-t="00:06:16">[00:06:16]</a>.
4.  **Currency Impact**: Changing the currency setting in bulk export mode will automatically apply the new currency to all generated PDFs <a class="yt-timestamp" data-t="00:07:05">[00:07:05]</a>.
