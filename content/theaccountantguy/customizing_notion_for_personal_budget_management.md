## Overview of the Notion Monthly Budget Planner

The monthly budget planner features four main views to help users track their finances <a class="yt-timestamp" data-t="00:00:11">[00:00:11]</a>:

*   **Expenses Overview** <a class="yt-timestamp" data-t="00:00:13">[00:00:13]</a>: This view provides a summary for different expense categories (e.g., shopping, entertainment, food, utilities) <a class="yt-timestamp" data-t="00:00:25">[00:00:25]</a>. For each category, it displays the total expense amount, budgeted amount, percentage of total budgeted amount incurred, and a status indicating whether the budget has been exceeded or is under budget <a class="yt-timestamp" data-t="00:00:32">[00:00:32]</a>.
*   **Monthly Overview** <a class="yt-timestamp" data-t="00:00:15">[00:00:15]</a>: This view organizes all expenses in a monthly format, allowing users to see all expenses for a specific month at a glance <a class="yt-timestamp" data-t="00:00:47">[00:00:47]</a>.
*   **Yearly Overview** <a class="yt-timestamp" data-t="00:00:15">[00:00:15]</a>: Similar to the monthly overview, this view presents financial information in a yearly format <a class="yt-timestamp" data-t="00:00:57">[00:00:57]</a>.
*   **Summary of All Expenses** <a class="yt-timestamp" data-t="00:00:19">[00:00:19]</a>: This section offers a complete expense overview in a yearly format, consolidating all expenses to show overall financial performance <a class="yt-timestamp" data-t="00:01:03">[00:01:03]</a>.

The core of this system relies on the "expense details database," where all financial information is entered for calculations <a class="yt-timestamp" data-t="00:01:14">[00:01:14]</a>.

## [[Setting up a personal finance tracker in Notion | Setting up a Personal Finance Tracker in Notion]]

To [[Budgeting and tracking in Notion | begin budgeting and tracking in Notion]], users must first input transactions into the "expense details" database <a class="yt-timestamp" data-t="00:01:28">[00:01:28]</a>. Expenses are categorized by month and sorted in descending order, with the most recent months appearing at the top <a class="yt-timestamp" data-t="00:01:33">[00:01:33]</a>.

### Entering Transactions

To enter a new transaction:
1.  Click on "new" <a class="yt-timestamp" data-t="00:01:41">[00:01:41]</a>.
2.  Enter the date (e.g., February 1, 2024) <a class="yt-timestamp" data-t="00:01:47">[00:01:47]</a>.
3.  Enter the expense detail (e.g., Wi-Fi expense) <a class="yt-timestamp" data-t="00:02:05">[00:02:05]</a>.
4.  Select the period (e.g., Feb 2024) <a class="yt-timestamp" data-t="00:02:13">[00:02:13]</a>. Users can add custom periods if needed <a class="yt-timestamp" data-t="00:02:20">[00:02:20]</a>.
5.  Select the year (e.g., 2024) <a class="yt-timestamp" data-t="00:02:25">[00:02:25]</a>.
6.  Choose the expense category (e.g., utilities) <a class="yt-timestamp" data-t="00:02:30">[00:02:30]</a>. Users can add their own categories <a class="yt-timestamp" data-t="00:02:35">[00:02:35]</a>.
7.  Input an expense code, which helps segregate expenses by category and month (e.g., "utility Feb") <a class="yt-timestamp" data-t="00:02:44">[00:02:44]</a>.
8.  Enter the expense amount in the "expense amount" column <a class="yt-timestamp" data-t="00:02:59">[00:02:59]</a>.

### [[Setting Budgets and Recording Transactions in Notion | Setting Budgets]]

At the beginning of each month, users can [[Using Notion for budgeting and savings | set budgets]] for specific expenses <a class="yt-timestamp" data-t="00:03:07">[00:03:07]</a>:
1.  Select the date (e.g., February 1) <a class="yt-timestamp" data-t="00:03:32">[00:03:32]</a>.
2.  Enter the expense name for which the budget is being set (e.g., "food expense") <a class="yt-timestamp" data-t="00:03:39">[00:03:39]</a>.
3.  Select the period (e.g., February) <a class="yt-timestamp" data-t="00:03:46">[00:03:46]</a>.
4.  Select the year <a class="yt-timestamp" data-t="00:03:50">[00:03:50]</a>.
5.  Choose the category (e.g., "food") <a class="yt-timestamp" data-t="00:03:52">[00:03:52]</a>. Note that for budgeting, the expense name and category name should typically be the same <a class="yt-timestamp" data-t="00:03:58">[00:03:58]</a>.
6.  Enter the budget amount in the "budget amount" column <a class="yt-timestamp" data-t="00:04:18">[00:04:18]</a>.

Once the budget is set, all figures in the overviews (monthly, yearly, expenses) are automatically updated <a class="yt-timestamp" data-t="00:04:21">[00:04:21]</a>. As more expense information is added, the system automatically updates, providing a clear picture of monthly progress <a class="yt-timestamp" data-t="00:04:37">[00:04:37]</a>.

## Analyzing Financial Data

The system provides various ways to analyze financial data <a class="yt-timestamp" data-t="00:04:55">[00:04:55]</a>:
*   **Period View**: Clicking on a specific period (e.g., Feb 2024) opens a new window displaying all transactions for that period, including amounts and whether they were budgeted or actual expenses <a class="yt-timestamp" data-t="00:05:12">[00:05:12]</a>.
*   **Year View**: Similarly, clicking on a year shows a comprehensive view of budgeted amounts, expense amounts, expense as a percentage of budget, and budget status for the entire year, along with all transactions <a class="yt-timestamp" data-t="00:05:39">[00:05:39]</a>.
*   **Category View**: Users can see how much has been spent for a particular category, including budgeted amounts, actual expenses, percentage, and status, along with associated transactions <a class="yt-timestamp" data-t="00:06:03">[00:06:03]</a>.
*   **Expense Code View**: This view allows users to see the amount booked for a specific expense category within a particular month, compared to the total budget for that item <a class="yt-timestamp" data-t="00:06:31">[00:06:31]</a>.

These views enable comprehensive [[Managing accounts for personal finances in Notion | financial tracking]] and analysis, helping users understand their financial situation <a class="yt-timestamp" data-t="00:06:21">[00:06:21]</a>.
