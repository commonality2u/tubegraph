The Home Inventory Tracker is a simple Notion template designed to help users [[tracking_inventory_status_and_transactions | keep track of all their home inventory]] in one place <a class="yt-timestamp" data-t="00:00:02">[00:00:02]</a>, <a class="yt-timestamp" data-t="00:00:06">[00:00:06]</a>. It facilitates monitoring both purchases and consumption of household items <a class="yt-timestamp" data-t="00:01:19">[00:01:19]</a>.

## Key Sections and Features

### Financial Overview
On the left side of the dashboard, users can see key financial metrics related to their home inventory <a class="yt-timestamp" data-t="00:00:07">[00:00:07]</a>:
*   Total estimated purchases <a class="yt-timestamp" data-t="00:00:09">[00:00:09]</a>
*   Total purchases made <a class="yt-timestamp" data-t="00:00:11">[00:00:11]</a>
*   The difference between estimated and actual purchases <a class="yt-timestamp" data-t="00:00:12">[00:00:12]</a>
*   The total percentage of purchases made compared to the estimated budget <a class="yt-timestamp" data-t="00:00:14">[00:00:14]</a>

### Inventory Categories
The right side features an "Inventory Categories" section where users can define different categories of inventory <a class="yt-timestamp" data-t="00:00:18">[00:00:18]</a>, <a class="yt-timestamp" data-t="00:00:20">[00:00:20]</a>. Examples provided include "Darian eggs," "medicines," and "fruits and veggies" <a class="yt-timestamp" data-t="00:00:23">[00:00:23]</a>, <a class="yt-timestamp" data-t="00:00:26">[00:00:26]</a>. Under each category, subcategories are listed along with their estimated budgeted purchase amounts <a class="yt-timestamp" data-t="00:00:28">[00:00:28]</a>, <a class="yt-timestamp" data-t="00:00:30">[00:00:30]</a>.

### Inventory Overview
This section provides a summary of [[tracking_inventory_status_and_transactions | inventory status]] for each subcategory <a class="yt-timestamp" data-t="00:00:34">[00:00:34]</a>, <a class="yt-timestamp" data-t="00:00:43">[00:00:43]</a>. It tracks:
*   Total purchases made <a class="yt-timestamp" data-t="00:00:35">[00:00:35]</a>
*   Consumption units <a class="yt-timestamp" data-t="00:00:37">[00:00:37]</a>
*   Availability units <a class="yt-timestamp" data-t="00:00:39">[00:00:39]</a>
*   The percentage of total units that have been consumed <a class="yt-timestamp" data-t="00:00:39">[00:00:39]</a>

### Transaction Section
The "Transaction" section allows users to record specific [[tracking_inventory_status_and_transactions | inventory transactions]] <a class="yt-timestamp" data-t="00:00:48">[00:00:48]</a>, <a class="yt-timestamp" data-t="00:00:50">[00:00:50]</a>. For each entry, users can specify:
*   The inventory item (e.g., "cheese purchase") <a class="yt-timestamp" data-t="00:00:53">[00:00:53]</a>
*   The amount purchased <a class="yt-timestamp" data-t="00:01:00">[00:01:00]</a>, <a class="yt-timestamp" data-t="00:01:02">[00:01:02]</a>
*   When the item is consumed <a class="yt-timestamp" data-t="00:01:04">[00:01:04]</a>
*   The number of items used, recorded in an "items used" column <a class="yt-timestamp" data-t="00:01:09">[00:01:09]</a>, <a class="yt-timestamp" data-t="00:01:10">[00:01:10]</a>

All transaction entries automatically update the other sections of the template <a class="yt-timestamp" data-t="00:01:14">[00:01:14]</a>. This allows users to continuously monitor how much [[tracking_inventory_status_and_transactions | inventory]] is available and what consumption is required for each item at home <a class="yt-timestamp" data-t="00:01:19">[00:01:19]</a>, <a class="yt-timestamp" data-t="00:01:21">[00:01:21]</a>.
