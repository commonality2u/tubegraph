This article details how to configure settings and manage subscription options within PDFOutput, a tool that generates PDF documents in bulk using a Notion database and template <a class="yt-timestamp" data-t="00:00:05">[00:00:05]</a>.

## Accessing PDFOutput.com

To begin, navigate to [[introduction_to_pdfoutput_tool | PDFoutput.com]] <a class="yt-timestamp" data-t="00:01:04">[00:01:04]</a>. Once logged in, you will see an interface that provides access to various features, including templates, settings, and connections <a class="yt-timestamp" data-t="00:01:07">[00:01:07]</a>.

## [[Configuring PDF settings and properties | Configuring PDF Settings]]

Within the settings tab, you can customize various PDF properties <a class="yt-timestamp" data-t="00:10:36">[00:10:36]</a>.

### Page Format

*   PDFOutput offers different page format options <a class="yt-timestamp" data-t="00:10:39">[00:10:39]</a>.
*   You can select a specific page format from a list of elements <a class="yt-timestamp" data-t="00:10:41">[00:10:41]</a>.
*   A search option is available to find particular page formats <a class="yt-timestamp" data-t="00:10:44">[00:10:44]</a>.
*   After defining the page format, click "save" to apply the changes for PDF generation <a class="yt-timestamp" data-t="00:10:49">[00:10:49]</a>.

## [[Understanding PDFOutput subscription plans | Understanding Subscription Options]]

PDFOutput offers various subscription plans that affect the functionality and output of your generated PDFs <a class="yt-timestamp" data-t="00:10:05">[00:10:05]</a>.

### Current Plan and Usage

*   Your current active plan, renewal date, and the number of files downloaded are visible <a class="yt-timestamp" data-t="00:10:07">[00:10:07]</a>.
*   For example, an account might be on an "Enterprise plan" <a class="yt-timestamp" data-t="00:10:07">[00:10:07]</a>.

### [[Options for upgrading PDF output features and settings | Upgrading Your Subscription]]

*   To upgrade your plan, click on the "upgrade subscription" option <a class="yt-timestamp" data-t="00:10:14">[00:10:14]</a>.
*   After upgrading, click "activate subscription" to apply the new plan <a class="yt-timestamp" data-t="00:10:17">[00:10:17]</a>.

### Watermark Information

*   New users or those not on a paid plan will see a watermark on their [[introduction_to_pdfoutput_tool | PDFOutput]] documents <a class="yt-timestamp" data-t="00:10:27">[00:10:27]</a>.
*   The watermark is not visible on PDFs generated under any paid subscription plan <a class="yt-timestamp" data-t="00:10:31">[00:10:31]</a>.

## Managing Connections for Re-Exporting

Once an export process has been successfully completed, the connection (database, template, and mapping) is saved <a class="yt-timestamp" data-t="00:09:22">[00:09:22]</a>.

*   To access previously saved connections, navigate to the "connections" window <a class="yt-timestamp" data-t="00:09:25">[00:09:25]</a>.
*   Select your particular connection from the list <a class="yt-timestamp" data-t="00:09:36">[00:09:36]</a>.
*   Search, filtration, and sorting options are available to help find specific connections <a class="yt-timestamp" data-t="00:09:38">[00:09:38]</a>.
*   Once selected, the elements from the database and template are automatically loaded <a class="yt-timestamp" data-t="00:09:45">[00:09:45]</a>.
*   Before regenerating a PDF, ensure the "PDF status" column in your Notion database for the desired rows is unchecked; if it's checked, the row will not be generated <a class="yt-timestamp" data-t="00:09:51">[00:10:01]</a>.
