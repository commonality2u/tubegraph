## How to Use PDFOutput.com

Upon logging into PDFOutput.com, users are presented with an interface where they can select a particular template and currency <a class="yt-timestamp" data-t="00:03:18">[00:03:18]</a>.

### Single PDF Mode

For individual document creation, the platform supports a "single PDF mode" <a class="yt-timestamp" data-t="00:04:37">[00:04:37]</a>.

- **Template Selection**: Users can select from available templates, such as the invoice template <a class="yt-timestamp" data-t="00:03:25">[00:03:25]</a>.
- **Currency Selection**: Various currency options are available, allowing users to choose their preferred currency, like USD or Euro <a class="yt-timestamp" data-t="00:03:30">[00:03:30]</a>. The invoice will populate with the selected currency <a class="yt-timestamp" data-t="00:04:29">[00:04:29]</a>.
- **Detail Entry**: Users fill in all relevant details for the document (e.g., invoice information) <a class="yt-timestamp" data-t="00:03:36">[00:03:36]</a>.
- **PDF Generation**: After filling in the details, clicking "download PDF" automatically [[generating_pdf_invoices_using_pdf_output_service | generates a PDF document]] based on the provided information <a class="yt-timestamp" data-t="00:04:12">[00:04:12]</a>.

### Bulk Export Mode

For generating multiple PDF documents simultaneously, PDFOutput.com features a "bulk export mode" <a class="yt-timestamp" data-t="00:04:42">[00:04:42]</a>, enabling [[bulk_pdf_document_generation | bulk PDF document generation]] <a class="yt-timestamp" data-t="00:04:58">[00:04:58]</a> and [[bulk_export_of_pdf_documents | bulk export of PDF documents]] <a class="yt-timestamp" data-t="00:04:40">[00:04:40]</a>.

- **Database Entry**: Users can fill in a database with the necessary details for multiple documents <a class="yt-timestamp" data-t="00:04:46">[00:04:46]</a>. This database includes fields like invoice number, invoice date, and other relevant information, similar to the single PDF mode <a class="yt-timestamp" data-t="00:05:23">[00:05:23]</a>. Columns can be expanded for better visibility <a class="yt-timestamp" data-t="00:05:28">[00:05:28]</a>.
- **Bulk Generation**: Once the database is filled, clicking "download all PDF" will generate PDFs one by one according to the entered information <a class="yt-timestamp" data-t="00:05:45">[00:05:45]</a>.
- **Individual Row Export**: It is also possible to export a PDF document for a specific row by clicking "download PDF" for that individual row <a class="yt-timestamp" data-t="00:06:22">[00:06:22]</a>.

The generated PDF documents maintain the same format as those created in single PDF mode <a class="yt-timestamp" data-t="00:05:57">[00:05:57]</a>.
