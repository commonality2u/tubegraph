This article details how to utilize a [[debt_payment_tracking_and_reporting | debt payment tracker]] to manage and pay off [[managing_debt_and_liabilities | debts]] effectively <a class="yt-timestamp" data-t="00:00:07">[00:00:07]</a>.

## Debt Overview Section

The debt overview section categorizes [[managing_debt_and_liabilities | debt]] into different types, such as annual and monthly payments <a class="yt-timestamp" data-t="00:00:13">[00:00:13]</a>. It provides key details for each loan:

*   **Debt details** <a class="yt-timestamp" data-t="00:00:22">[00:00:22]</a>
*   **Loan amount** <a class="yt-timestamp" data-t="00:00:23">[00:00:23]</a>
*   **EMI amount:** The amount to be paid for each installment <a class="yt-timestamp" data-t="00:00:24">[00:00:24]</a>.
*   **Interest rate:** User-specifiable <a class="yt-timestamp" data-t="00:00:29">[00:00:29]</a>.
*   **Initial repayment:** The first payment date for the first installment <a class="yt-timestamp" data-t="00:00:32">[00:00:32]</a>.
*   **Debt free by:** A calculation showing the projected date of being debt-free if repayments continue at the specified amount and interest rate <a class="yt-timestamp" data-t="00:00:38">[00:00:38]</a>.
*   **Number of EMIs:** The total number of installments required to pay off the [[managing_debt_and_liabilities | debt]] <a class="yt-timestamp" data-t="00:00:47">[00:00:47]</a>.

### Managing Debt Categories

Users can add new payment categories (e.g., weekly) by clicking the "add a group" button and specifying the new installment period <a class="yt-timestamp" data-t="00:00:54">[00:00:54]</a>. Existing debt can be moved to different repayment schedules (e.g., quarterly or weekly) by dragging and dropping the specific line item <a class="yt-timestamp" data-t="00:01:27">[00:01:27]</a>. This action automatically updates the repayment schedule and other relevant calculations <a class="yt-timestamp" data-t="00:01:40">[00:01:40]</a>.

## [[calculating_debt_payoff_progress | Debt Payoff Progress]] Tracking

This section provides a summary of all loans and their repayment status <a class="yt-timestamp" data-t="00:01:51">[00:01:51]</a>:

*   **Total amount repaid:** Calculated from the installments entered in the transaction section <a class="yt-timestamp" data-t="00:01:57">[00:01:57]</a>. Updates automatically when any EMI is paid <a class="yt-timestamp" data-t="00:02:07">[00:02:07]</a>.
*   **Net payoff in percentage:** Shows the proportion of total [[managing_debt_and_liabilities | debt]] paid off <a class="yt-timestamp" data-t="00:02:11">[00:02:11]</a>.
*   **Debt free by:** A quick reference to the projected debt-free date <a class="yt-timestamp" data-t="00:02:15">[00:02:15]</a>.
*   **Interest amount:** The total interest paid so far <a class="yt-timestamp" data-t="00:02:22">[00:02:22]</a>.
*   **Principal amount:** The total principal amount repaid <a class="yt-timestamp" data-t="00:02:30">[00:02:30]</a>.
    *   Every debt repayment is split into interest and principal components <a class="yt-timestamp" data-t="00:02:36">[00:02:36]</a>.

## Managing Debt Details

*   **Adding new debt:** Click "added details" to quickly add a new debt entry to the "no payment frequency" column. Once added, select its frequency and input specific details <a class="yt-timestamp" data-t="00:02:43">[00:02:43]</a>.
*   **Editing existing loan details:** Simply click on the debt field and type in changes; these updates automatically reflect across all relevant columns and descriptions <a class="yt-timestamp" data-t="00:02:59">[00:02:59]</a>.

## Transaction Section Overview

This section is where users record and view all [[managing_debt_payment_transactions | debt payments]] made <a class="yt-timestamp" data-t="00:03:12">[00:03:12]</a>.

*   **Paid installment section:** Reflects all paid [[managing_debt_and_liabilities | debts]], bifurcated by month, with the latest month appearing on top <a class="yt-timestamp" data-t="00:03:19">[00:03:19]</a>. A summation of all EMIs paid for that month is displayed <a class="yt-timestamp" data-t="00:03:27">[00:03:27]</a>.
*   **Payment status:** A checkbox indicates if an amount has been paid. Ticking or unticking this box automatically updates the paid or unpaid column sections <a class="yt-timestamp" data-t="00:03:41">[00:03:41]</a>.
*   **Payment details:** Users can input the payment date <a class="yt-timestamp" data-t="00:03:57">[00:03:57]</a>, select the specific loan from a dropdown of previously added loans <a class="yt-timestamp" data-t="00:04:01">[00:04:01]</a>, and enter the EMI amount <a class="yt-timestamp" data-t="00:04:08">[00:04:08]</a>.
*   **Next repayment schedule:** Automatically calculated based on the loan's set payment frequency <a class="yt-timestamp" data-t="00:04:11">[00:04:11]</a>. For example, a monthly student loan payment on September 30 will show October 30 as the next payment date <a class="yt-timestamp" data-t="00:04:20">[00:04:20]</a>, while an annual home loan payment on September 27, 2023, will show September 27, 2024, as the next repayment date <a class="yt-timestamp" data-t="00:04:29">[00:04:29]</a>.
*   **Adding new debt payments:** The "pay debt amount" button automatically adds a new debt payment entry to the schedule for the current date <a class="yt-timestamp" data-t="00:04:51">[00:04:51]</a>.

All details added in the transaction section, including changes to payment amounts, automatically update the [[calculating_debt_payoff_progress | debt payoff progress]] section <a class="yt-timestamp" data-t="00:05:16">[00:05:16]</a>.
