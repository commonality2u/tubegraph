## Getting Started with PDF output.com

To begin, log in to PDF output.com <a class="yt-timestamp" data-t="00:00:11">[00:00:11]</a>. From the interface, navigate to the "Templates" section <a class="yt-timestamp" data-t="00:00:17">[00:00:17]</a>. This page lists all available templates for PDF output <a class="yt-timestamp" data-t="00:00:22">[00:00:22]</a>.

### Finding and Duplicating a Template

You can search for a specific template, such as "payment receipts PDF generator," using the search option <a class="yt-timestamp" data-t="00:00:32">[00:00:32]</a>, <a class="yt-timestamp" data-t="00:00:36">[00:00:36]</a>. There are also filter and sorting options available <a class="yt-timestamp" data-t="00:00:49">[00:00:49]</a>.

Once you've found the desired template, click on its download link <a class="yt-timestamp" data-t="00:00:56">[00:00:56]</a>. This will open a new page featuring the template's database and template <a class="yt-timestamp" data-t="00:01:03">[00:01:03]</a>.

To use the template, you must duplicate it to a Notion workspace <a class="yt-timestamp" data-t="00:02:50">[00:02:50]</a>. If the template is published to the Notion Gallery, you can click "Start with this template" <a class="yt-timestamp" data-t="00:02:30">[00:02:30]</a>, <a class="yt-timestamp" data-t="00:02:43">[00:02:43]</a>. Otherwise, use the "Duplicate" option to choose your desired Notion workspace <a class="yt-timestamp" data-t="00:02:35">[00:02:35]</a>, <a class="yt-timestamp" data-t="00:02:38">[00:02:38]</a>.

## Understanding Notion Templates and Databases for PDF Generation

A Notion template for PDF generation, such as a payment receipt template, has pre-defined elements <a class="yt-timestamp" data-t="00:01:22">[00:01:22]</a>. Critical to its functionality are *placeholder text elements*, enclosed in curly braces (e.g., `{receipt number}`, `{customer name}`) <a class="yt-timestamp" data-t="00:01:28">[00:01:28]</a>.

These placeholders are designed to be replaced by corresponding elements from your Notion database <a class="yt-timestamp" data-t="00:01:38">[00:01:38]</a>, <a class="yt-timestamp" data-t="00:01:42">[00:01:42]</a>. For example, a `{customer name}` placeholder in the template will draw data from the "customer name" column in your database <a class="yt-timestamp" data-t="00:01:47">[00:01:47]</a>.

### Customizing Notion Templates

The entire template is customizable to your requirements <a class="yt-timestamp" data-t="00:04:21">[00:04:21]</a>, allowing you to adjust formatting like bolding or spacing <a class="yt-timestamp" data-t="00:04:27">[00:04:27]</a>, <a class="yt-timestamp" data-t="00:04:30">[00:04:30]</a>.

When customizing, ensure that:
*   Any placeholder elements you wish to replace from your database remain enclosed in curly braces <a class="yt-timestamp" data-t="00:04:41">[00:04:41]</a>.
*   The exact naming convention for these placeholder elements in the template matches the corresponding column names in your Notion database <a class="yt-timestamp" data-t="00:02:02">[00:02:02]</a>, <a class="yt-timestamp" data-t="00:04:54">[00:04:54]</a>, <a class="yt-timestamp" data-t="00:05:07">[00:05:07]</a>. This ensures proper syncing and error-free PDF generation <a class="yt-timestamp" data-t="00:04:50">[00:04:50]</a>, <a class="yt-timestamp" data-t="00:05:01">[00:05:01]</a>.

## Connecting and Generating PDFs

After duplicating the template to your Notion workspace, return to PDF output.com and go to the "Settings" section <a class="yt-timestamp" data-t="00:03:23">[00:03:23]</a>.

1.  **Add Templates**: Click "click here to add templates" <a class="yt-timestamp" data-t="00:03:32">[00:03:32]</a>.
2.  **Select Workspace**: Choose the Notion workspace where you duplicated the template <a class="yt-timestamp" data-t="00:03:42">[00:03:42]</a>, <a class="yt-timestamp" data-t="00:03:45">[00:03:45]</a>.
3.  **Allow Access**: Grant access to PDF output.com to read your template and database elements <a class="yt-timestamp" data-t="00:04:02">[00:04:02]</a>.

Once connected and authenticated <a class="yt-timestamp" data-t="00:04:08">[00:04:08]</a>, select your Notion database and the corresponding template from the dropdown menus <a class="yt-timestamp" data-t="00:05:24">[00:05:24]</a>, <a class="yt-timestamp" data-t="00:05:30">[00:05:30]</a>. Give your export a name (e.g., "payment receipt") <a class="yt-timestamp" data-t="00:05:35">[00:05:35]</a>.

### Mapping Database Properties to Template Elements

PDF output.com will automatically map Notion database properties (columns) to the corresponding template elements if their names match correctly <a class="yt-timestamp" data-t="00:05:43">[00:05:43]</a>, <a class="yt-timestamp" data-t="00:05:56">[00:05:56]</a>. If any elements are mismatched, you can manually map them using search, filter, or sort options within the interface <a class="yt-timestamp" data-t="00:06:04">[00:06:04]</a>, <a class="yt-timestamp" data-t="00:06:19">[00:06:19]</a>.

### Exporting and Downloading PDFs

After mapping, click "Export" <a class="yt-timestamp" data-t="00:06:39">[00:06:39]</a>. PDF output.com will automatically add a "PDF status" column to your Notion database to track which rows have been generated <a class="yt-timestamp" data-t="00:06:43">[00:06:43]</a>, <a class="yt-timestamp" data-t="00:06:50">[00:06:50]</a>.

Upon successful export, you can preview a sample PDF <a class="yt-timestamp" data-t="00:07:04">[00:07:04]</a>, <a class="yt-timestamp" data-t="00:07:07">[00:07:07]</a>, and download all generated files <a class="yt-timestamp" data-t="00:07:17">[00:07:17]</a>. Each generated file corresponds to a row in your Notion database <a class="yt-timestamp" data-t="00:07:36">[00:07:36]</a>.
