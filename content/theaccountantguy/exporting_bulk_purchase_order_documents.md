## Process for Bulk Export

To [[exporting_and_customizing_invoices_in_bulk | export multiple documents]] in one go:
1.  Navigate to [[pdfoutput_for_bulk_document_generation | PDFOutput.com]] <a class="yt-timestamp" data-t="02:56:00">[02:56:00]</a>.
2.  Select the "bulk export mode" <a class="yt-timestamp" data-t="04:57:00">[04:57:00]</a>.
3.  Choose the desired template from the dropdown, such as the Purchase Order template <a class="yt-timestamp" data-t="03:01:00">[03:01:00]</a>, <a class="yt-timestamp" data-t="03:08:00">[03:08:00]</a>.
4.  Fill in all the necessary details for each purchase order, mirroring the information from your [[setting_up_a_purchase_order_tracking_database | purchase order database]] <a class="yt-timestamp" data-t="05:00:00">[05:00:00]</a>, <a class="yt-timestamp" data-t="05:02:00">[05:02:00]</a>. You can add multiple rows for different purchase orders by clicking "add new row" <a class="yt-timestamp" data-t="05:15:00">[05:15:00]</a>, <a class="yt-timestamp" data-t="05:16:00">[05:16:00]</a>.
5.  Once all details are filled, you have two options for downloading:
    *   **Download individual documents:** Click the download button next to each row <a class="yt-timestamp" data-t="05:08:00">[05:08:00]</a>.
    *   **Download all PDF documents:** Click "download all PDF" to get all documents in one go <a class="yt-timestamp" data-t="05:11:00">[05:11:00]</a>, <a class="yt-timestamp" data-t="05:20:00">[05:20:00]</a>.

> [!info] Currency Selection
> Users can select the desired currency for the PDF documents. If a specific currency is not available, it can be requested through the platform <a class="yt-timestamp" data-t="03:12:00">[03:12:00]</a>, <a class="yt-timestamp" data-t="03:14:00">[03:14:00]</a>, <a class="yt-timestamp" data-t="04:28:00">[04:28:00]</a>.
