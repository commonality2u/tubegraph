[[PDFOutput software setup for document generation | PDF Output]] is a tool designed to generate PDF documents from a Notion database and template <a class="yt-timestamp" data-t="00:00:07">[00:00:07]</a>. This article covers various customization options and troubleshooting tips for effectively using the tool.

## [[Customizing and exporting PDF documents with PDF output tool | Document Customization]]

### Notion Template and Database Customization
Users can customize both the Notion template and the database used for PDF generation <a class="yt-timestamp" data-t="04:25:00">[04:25:00]</a>.
*   **Template Values**: Ensure that all values intended for replacement are placed within curly braces in the template <a class="yt-timestamp" data-t="04:34:00">[04:34:00]</a>.
*   **Name Matching**: The name of the database column must exactly match the placeholder text within the curly braces in the template for automatic linking <a class="yt-timestamp" data-t="04:44:00">[04:44:00]</a>. Avoid adding extra commas or spaces in names <a class="yt-timestamp" data-t="04:50:00">[04:50:00]</a>.

### Page Format Settings
Before connecting templates, users can select the desired page format for the PDF output in the settings section <a class="yt-timestamp" data-t="03:17:00">[03:17:00]</a>.

## Troubleshooting [[Using PDF Output interface for exporting documents | Document Generation]]

### Identifying Unmapped Properties
If database properties are not automatically matched with template elements, users can use the "filter unmapped properties" option to quickly identify them <a class="yt-timestamp" data-t="06:10:00">[06:10:00]</a>. These can then be manually mapped by selecting the corresponding name <a class="yt-timestamp" data-t="06:21:00">[06:21:00]</a>.

### Regenerating PDFs
When regenerating a PDF document, ensure that the "PDF status" checkbox in the Notion database is unticked <a class="yt-timestamp" data-t="07:12:00">[07:12:00]</a>. This column is automatically added by [[PDFOutput software setup for document generation | PDF Output]] to indicate generation status <a class="yt-timestamp" data-t="07:07:00">[07:07:00]</a>.

## Managing Connections and Templates

### Connections Tab
The "connections" tab stores previously generated PDF document configurations <a class="yt-timestamp" data-t="08:33:00">[08:33:00]</a>. This allows users to directly re-export documents without re-entering database and page information <a class="yt-timestamp" data-t="08:41:00">[08:41:00]</a>.

### Templates Tab
The "templates" tab lists predefined templates for generating PDF documents, such as invoices, payment receipts, and purchase orders <a class="yt-timestamp" data-t="00:32:00">[00:32:00]</a>.
*   **Search**: A search icon is available to quickly find required templates <a class="yt-timestamp" data-t="00:46:00">[00:46:00]</a>.
*   **Sorting and Filtering**: Sorting and filtering options are also available <a class="yt-timestamp" data-t="00:54:00">[00:54:00]</a>.
*   **Adding Templates**: Templates can be added to your Notion workspace by clicking the download link next to the desired template and then duplicating the page <a class="yt-timestamp" data-t="01:00:00">[01:00:00]</a>.
