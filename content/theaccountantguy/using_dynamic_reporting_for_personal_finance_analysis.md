The [[setting_up_a_personal_finance_tracker_in_google_sheets | personal finance tracker]] developed in a [[setting_up_a_personal_finance_tracker_in_google_sheets | Google Sheet]] includes a dedicated "report section" that offers dynamic reporting for comprehensive financial analysis <a class="yt-timestamp" data-t="00:00:48">[00:00:48]</a>. This feature is considered the core essence of the entire tracker <a class="yt-timestamp" data-t="00:09:18">[00:09:18]</a>.

## Overview of the Dynamic Reporting Section
The reporting section provides a dynamic presentation where users can easily switch between different financial views and specific time periods <a class="yt-timestamp" data-t="00:00:50">[00:00:50]</a>. This allows for quick insights into various financial aspects and how values change over time <a class="yt-timestamp" data-t="00:00:59">[00:00:59]</a>.

### Available Reports
Users can select from a dropdown menu to view reports on different financial categories <a class="yt-timestamp" data-t="00:09:25">[00:09:25]</a>:
*   **Expenses** <a class="yt-timestamp" data-t="00:09:27">[00:09:27]</a>
*   **Income** <a class="yt-timestamp" data-t="00:09:27">[00:09:27]</a>
*   **[[dashboard_overview_for_assessing_financial_goals | Sinking fund]] details** <a class="yt-timestamp" data-t="00:09:28">[00:09:28]</a>
*   **[[dashboard_overview_for_assessing_financial_goals | Debt payment]] details** <a class="yt-timestamp" data-t="00:09:29">[00:09:29]</a>

While an [[dashboard_overview_for_assessing_financial_goals | investment dashboard]] provides an updated view of investments, it is not primarily generated within this specific reporting section, unlike the other categories <a class="yt-timestamp" data-t="00:09:30">[00:09:30]</a>.

## How to Use Dynamic Reporting
To generate a report, users simply perform two main actions:
1.  **Select the desired financial category** from the dropdown menu (e.g., [[tracking_income_and_expenses | expense]], [[tracking_income_and_expenses | income]], [[dashboard_overview_for_assessing_financial_goals | sinking fund]], [[dashboard_overview_for_assessing_financial_goals | debt payment]]) <a class="yt-timestamp" data-t="00:09:25">[00:09:25]</a>.
2.  **Choose a specific month** by double-clicking and selecting any date within that month <a class="yt-timestamp" data-t="00:09:46">[00:09:46]</a>.

The report will automatically update to reflect the financial data for the selected category and month <a class="yt-timestamp" data-t="00:09:49">[00:09:49]</a>.

### Example: Income Report
For instance, to view the total [[tracking_income_and_expenses | income]] for January, select "Income" from the dropdown and then choose January. The report will display the total income for that month, such as $4150, which was entered as input in the [[tracking_income_and_expenses | transactions]] area <a class="yt-timestamp" data-t="00:09:52">[00:09:52]</a>. A quick pie chart visualizes the total division of the income <a class="yt-timestamp" data-t="00:10:03">[00:10:03]</a>.

### Dynamic Updates
The reports are highly dynamic:
*   Changing the month will automatically update the displayed values <a class="yt-timestamp" data-t="00:10:09">[00:10:09]</a>.
*   Switching the category in the dropdown (e.g., from income to [[tracking_income_and_expenses | expense]]) will reflect a different analysis for the selected month <a class="yt-timestamp" data-t="00:10:13">[00:10:13]</a>.

> "This is how dynamic it is and it quickly gives you a representation of what is happening" <a class="yt-timestamp" data-t="00:10:21">[00:10:21]</a>.
