The [[using_notion_formula_chatbot | Notion Formula Chatbot]] offers several options for customizing its interface, allowing users to personalize their experience with the tool <a class="yt-timestamp" data-t="00:07:56">[00:07:56]</a>.

## Changing the Color Theme <a class="yt-timestamp" data-t="00:07:27">[00:07:27]</a>

Users can change the chatbot's color theme to suit their preference:
1.  Click on the three dots located on the right side of the window <a class="yt-timestamp" data-t="00:07:30">[00:07:30]</a>.
2.  Select "Settings" <a class="yt-timestamp" data-t="00:07:32">[00:07:32]</a>.
3.  Under "App theme colors and font," click on the dropdown menu <a class="yt-timestamp" data-t="00:07:37">[00:07:37]</a>.
4.  Choose one of the following options <a class="yt-timestamp" data-t="00:07:40">[00:07:40]</a>:
    *   **Light**: Changes the background color to light <a class="yt-timestamp" data-t="00:07:41">[00:07:41]</a>.
    *   **Dark**: Returns the background to the default dark setting <a class="yt-timestamp" data-t="00:07:49">[00:07:49]</a>.
    *   **System settings**: Allows editing the active theme and changing colors to suit specific requirements <a class="yt-timestamp" data-t="00:07:54">[00:07:54]</a>.

## Changing the Font Family <a class="yt-timestamp" data-t="00:07:59">[00:07:59]</a>

Within the same "App theme colors and font" settings, users can also change the font family of the chatbot interface <a class="yt-timestamp" data-t="00:07:59">[00:07:59]</a>.

## Customizing the User Icon <a class="yt-timestamp" data-t="00:06:13">[00:06:13]</a>

Users can upload an image to personalize their display icon within the chat interface:
*   The image option is located on the left side of the window <a class="yt-timestamp" data-t="00:03:52">[00:03:52]</a>.
*   Uploading an image will update the user icon that appears next to the questions being generated <a class="yt-timestamp" data-t="00:06:13">[00:06:13]</a>.
*   This feature is optional and can be ignored <a class="yt-timestamp" data-t="00:06:25">[00:06:25]</a>.
*   To revert to the original user image, click the cross icon <a class="yt-timestamp" data-t="00:06:58">[00:06:58]</a>.

## Navigating the Interface

The chatbot interface is organized into distinct panels and sections to facilitate user interaction:

### Main Sections <a class="yt-timestamp" data-t="00:00:37">[00:00:37]</a>
The interface features three primary sections:
*   **Info Section**: Provides details related to the chatbot and includes an embedded explanatory video <a class="yt-timestamp" data-t="00:08:36">[00:08:36]</a>. Users can find contact information (notionformyuse@gmail.com) for feedback or issues here <a class="yt-timestamp" data-t="00:08:43">[00:08:43]</a>.
*   **Chat Section**: This is where users [[interacting_with_the_chatbot | interact with the chatbot]] by typing messages and receiving responses <a class="yt-timestamp" data-t="00:00:42">[00:00:42]</a>.
*   **Give Section**: Contains a donate button for users who wish to contribute to the product's improvement <a class="yt-timestamp" data-t="00:09:06">[00:09:06]</a>.

### Left Panel Visibility <a class="yt-timestamp" data-t="00:08:06">[00:08:06]</a>
If the left panel (where the API key and GPT model selections are located) is not visible, it can be opened by clicking the arrow icon on the left side of the window <a class="yt-timestamp" data-t="00:08:11">[00:08:11]</a>. This is particularly useful for smartphone or other device users <a class="yt-timestamp" data-t="00:08:21">[00:08:21]</a>.

### Chat History Management <a class="yt-timestamp" data-t="00:08:53">[00:08:53]</a>
*   **Clear Chat History**: A button is available to clear the entire chat history, refreshing the conversation from the beginning <a class="yt-timestamp" data-t="00:08:53">[00:08:53]</a>.
*   **Conversations Counter**: Displays the total number of conversations a user has had with the chatbot <a class="yt-timestamp" data-t="00:09:03">[00:09:03]</a>.

### Refresh Option <a class="yt-timestamp" data-t="00:09:16">[00:09:16]</a>
If any error or issue arises, users can simply click the refresh button to reset the chatbot's functionality <a class="yt-timestamp" data-t="00:09:16">[00:09:16]</a>.
