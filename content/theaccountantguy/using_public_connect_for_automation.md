[[integrating_p_connect_for_workflow_automation | Public Connect]] is utilized to set up and manage automated processes, particularly for bill payment reminders delivered via WhatsApp <a class="yt-timestamp" data-t="00:00:05">[00:00:05]</a>. ## Setting Up Bill Payment Reminders

To create a new bill payment reminder using this [[integrating_p_connect_for_workflow_automation | automation]] system, one needs to:
1.  Click on a dropdown menu and select "new template" <a class="yt-timestamp" data-t="00:00:18">[00:00:18]</a>.
2.  Specify the bill name, such as "Mobile bill" <a class="yt-timestamp" data-t="00:00:23">[00:00:23]</a>.
3.  Enter the bill amount, for example, $1500 <a class="yt-timestamp" data-t="00:00:28">[00:00:28]</a>.
4.  Set the bill to repeat on a recurring basis, such as "every month" <a class="yt-timestamp" data-t="00:00:37">[00:00:37]</a>.
5.  [[configuring_time_and_settings_for_automation | Specify the time]] for receiving notifications, for instance, 11:55 AM on the 20th of every month <a class="yt-timestamp" data-t="00:00:44">[00:00:44]</a>.

## Automation Process

Once the [[integrating_p_connect_for_workflow_automation | automation]] is set up, notifications will be automatically sent to WhatsApp <a class="yt-timestamp" data-t="00:01:17">[00:01:17]</a>. When a template is set on a recurring basis, notifications are automatically sent every month <a class="yt-timestamp" data-t="00:01:23">[00:01:23]</a>. For example, a new bill item will be added automatically, triggering a WhatsApp notification at the [[configuring_time_and_settings_for_automation | scheduled time]] <a class="yt-timestamp" data-t="00:01:29">[00:01:29]</a>.

### Notification Timing
The system demonstrates timely delivery of notifications. For a reminder set for 11:55 AM, the message was received at 11:56 AM, indicating it takes under a minute to send the notification <a class="yt-timestamp" data-t="00:01:54">[00:01:54]</a>.
