A dedicated template is available for tracking house rental payments using [[Notion]] <a class="yt-timestamp" data-t="00:00:02">[00:00:02]</a>. This template helps users keep a detailed record of all house rental transactions <a class="yt-timestamp" data-t="00:00:06">[00:00:06]</a>.

## Template Overview

The template is structured with a main summary section at the top, offering four different views <a class="yt-timestamp" data-t="00:00:10">[00:00:10]</a>:

### Basic Information View

This view allows for the addition of fundamental data <a class="yt-timestamp" data-t="00:00:13">[00:00:13]</a>. Tenants can be categorized under various property classifications owned by the user <a class="yt-timestamp" data-t="00:00:15">[00:00:15]</a>. Examples of classifications include "apartment" and "duplex," but users can add custom property types <a class="yt-timestamp" data-t="00:00:20">[00:00:20]</a>.

*   **Adding Tenants**: To add a tenant, click the "ADD tenant" button on the left <a class="yt-timestamp" data-t="00:00:27">[00:00:27]</a>. This automatically creates a new tenant entry <a class="yt-timestamp" data-t="00:00:30">[00:00:30]</a>. Details for the tenant can be added by clicking the pencil icon <a class="yt-timestamp" data-t="00:00:37">[00:00:37]</a>. The entry can then be moved to its respective property category <a class="yt-timestamp" data-t="00:00:33">[00:00:33]</a>.

### Property Section View

This section displays different property classifications <a class="yt-timestamp" data-t="00:00:46">[00:00:46]</a>. Under each property, users can see <a class="yt-timestamp" data-t="00:00:49">[00:00:49]</a>:
*   The tenant's name <a class="yt-timestamp" data-t="00:00:51">[00:00:51]</a>
*   Total rent amount <a class="yt-timestamp" data-t="00:00:54">[00:00:54]</a>
*   Rent amount that has been paid <a class="yt-timestamp" data-t="00:00:54">[00:00:54]</a>
*   Balance rent amount that is due <a class="yt-timestamp" data-t="00:00:56">[00:00:56]</a>
*   Advanced rent that has been paid so far <a class="yt-timestamp" data-t="00:00:58">[00:00:58]</a>

### Rental History View

This view provides a comprehensive history of all rental payments made by tenants, based on the transactions recorded in the system <a class="yt-timestamp" data-t="00:01:04">[00:01:04]</a>.

### Overall Summary View

The final summary section offers key financial metrics <a class="yt-timestamp" data-t="00:01:12">[00:01:12]</a>:
*   Total amount due <a class="yt-timestamp" data-t="00:01:14">[00:01:14]</a>
*   Amount that has been paid <a class="yt-timestamp" data-t="00:01:14">[00:01:14]</a>
*   Outstanding rent amount <a class="yt-timestamp" data-t="00:01:15">[00:01:15]</a>
*   Percentage of rent collected <a class="yt-timestamp" data-t="00:01:17">[00:01:17]</a>

## Transaction Section

Below the main summary, the transaction section features three distinct views, in addition to a view showing all transactions <a class="yt-timestamp" data-t="00:01:20">[00:01:20]</a>:

### Rent Paid

This view displays all transactions for which rent has been paid, along with the respective tenants and amounts <a class="yt-timestamp" data-t="00:01:24">[00:01:24]</a>.

### Rent Due

This section shows information for rent that is due and has not yet been paid <a class="yt-timestamp" data-t="00:01:31">[00:01:31]</a>. Checking the box next to an entry moves it to the "Rent Paid" category, indicating the rent has been settled <a class="yt-timestamp" data-t="00:01:35">[00:01:35]</a>.

### Quarterly Summary

This view presents rental payments, collections, and balance due for each month within a specific quarter <a class="yt-timestamp" data-t="00:01:41">[00:01:41]</a>.

## Adding Payments

Users can quickly add new rental payments by clicking the designated button in the template <a class="yt-timestamp" data-t="00:01:50">[00:01:50]</a>.
