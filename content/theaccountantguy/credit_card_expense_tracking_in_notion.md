From: [[theaccountantguy]] <br/> 

Sanat Biswal, also known as The Accountant Guy, introduces a Notion-based credit card expenses tracker designed to monitor and manage credit card payments effectively <a class="yt-timestamp" data-t="00:00:00"></a>. This tracker offers various overviews and detailed sections to ensure timely debt repayment <a class="yt-timestamp" data-t="05:05:00"></a>.

## Key Overview Sections

The tracker provides several high-level overviews for quick financial insights:

*   **Credit Cards Overview**
    *   Displays all added credit cards <a class="yt-timestamp" data-t="00:10:00"></a>. In a demonstration, four credit cards were added: Student Credit Card, Cash Back Credit Card, Travel Credit Card, and Rewards Credit Card <a class="yt-timestamp" data-t="00:15:00"></a>.
    *   For each card, it shows the total amount due, the amount repaid so far, and the remaining balance due <a class="yt-timestamp" data-t="00:23:00"></a>.
*   **Monthly Overview**
    *   Shows the amount due, amount repaid, and balance amount due for each month <a class="yt-timestamp" data-t="00:32:00"></a>.
    *   This section can be customized as per individual requirements <a class="yt-timestamp" data-t="00:37:00"></a>.
*   **Yearly Overview**
    *   Presents the total amount due, amount repaid, and balance amount due across different years <a class="yt-timestamp" data-t="00:41:00"></a>.
*   **Expenses Overview**
    *   Provides a summary of all booked expenses categorized by type <a class="yt-timestamp" data-t="00:59:00"></a>.
    *   It displays the total expenses incurred, the budgeted amount set for each category, the expense amount compared to the budget, and whether the budget is "under budget" or "over budget" <a class="yt-timestamp" data-t="01:05:00"></a>.
    *   Users can [[setting_up_expense_categories_in_notion | set budget amounts]] for expense categories by clicking on the budget amount field <a class="yt-timestamp" data-t="03:56:00"></a>. The tracker automatically updates expense details relative to the budgeted amount <a class="yt-timestamp" data-t="04:04:00"></a>. Expenses within the budget will show "under budget," while those exceeding it will show "over budget" <a class="yt-timestamp" data-t="04:26:00"></a>.
*   **Notification Section**
    *   Indicates when the next repayment is due and how many days remain until the repayment date for each credit card <a class="yt-timestamp" data-t="01:16:00"></a>.
    *   Users can set their own repayment dates for each credit card <a class="yt-timestamp" data-t="04:35:00"></a>.

## Booking and Managing Expenses

To utilize the tracker, expenses must be recorded in the dedicated section:

*   **Expense Details Section**
    *   This is the primary area for entering all expense transactions <a class="yt-timestamp" data-t="01:37:00"></a>.
    *   Columns include:
        *   **Date:** The date of the expense <a class="yt-timestamp" data-t="01:42:00"></a>.
        *   **Expense Details:** A description of the expense <a class="yt-timestamp" data-t="01:45:00"></a>. This can include an "opening balance" for amounts carried forward from previous months <a class="yt-timestamp" data-t="02:04:00"></a>.
        *   **Period:** The specific period for which the expense is booked <a class="yt-timestamp" data-t="01:46:00"></a>.
        *   **Year:** The year of the expense <a class="yt-timestamp" data-t="01:49:00"></a>.
        *   **Credit Card:** The credit card used for the expense <a class="yt-timestamp" data-t="01:51:00"></a>. Users can select from existing cards or add new ones by typing <a class="yt-timestamp" data-t="02:33:00"></a>.
        *   **Category:** The assigned category for each expense <a class="yt-timestamp" data-t="01:54:00"></a>. Users can [[setting_up_expense_categories_in_notion | assign or create new categories]] <a class="yt-timestamp" data-t="02:45:00"></a>.
        *   **Expense Amount:** The amount of the expense <a class="yt-timestamp" data-t="01:57:00"></a>.
        *   **Repayment Section:** For entering the amount repaid for a particular credit card <a class="yt-timestamp" data-t="02:00:00"></a>. Credit card repayments should be entered with "credit card repayment" in the details and the amount in the repayment column <a class="yt-timestamp" data-t="02:58:00"></a>.
    *   All entries in this section automatically update the overview sections at the top <a class="yt-timestamp" data-t="03:07:00"></a>.
    *   A **Receipt** column allows users to add bills or receipts for expenses <a class="yt-timestamp" data-t="05:34:00"></a>.

## Customization and Detailed Views

The tracker offers various ways to customize and deep-dive into financial data:

*   **Editing and Adding Items:**
    *   Credit card information can be edited using a pencil icon or new credit cards can be added via a "new" button <a class="yt-timestamp" data-t="05:16:00"></a>.
    *   Categories can also be added directly by entering their details <a class="yt-timestamp" data-t="05:24:00"></a>.
    *   Existing credit card categories can be edited or new categories added <a class="yt-timestamp" data-t="06:44:00"></a>.
*   **Summarization Views:**
    *   Clicking on a specific **period** opens a new window showing transaction details, including dates, period, transaction details, amounts, and a tag (green for repayment, regular for expense) <a class="yt-timestamp" data-t="05:44:00"></a>.
    *   Similarly, clicking on a **year** provides the same detailed information for that specific year <a class="yt-timestamp" data-t="06:16:00"></a>.
    *   Selecting a **credit card** displays all transactions related to that particular card, including its status and category <a class="yt-timestamp" data-t="06:26:00"></a>.
    *   Clicking on a **category** allows users to view all expenses associated with that category <a class="yt-timestamp" data-t="06:48:00"></a>.

This [[notion_expense_tracker_setup | Notion expense tracker setup]] is customizable to personal needs <a class="yt-timestamp" data-t="07:25:00"></a>.
