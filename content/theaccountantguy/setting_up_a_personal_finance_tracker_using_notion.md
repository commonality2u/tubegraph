This article details how to set up and utilize a [[setting_up_a_personal_finance_tracker_in_notion | personal finance tracker]] within Notion, designed to manage various aspects of your financial life <a class="yt-timestamp" data-t="00:00:10">[00:00:10]</a>. The system helps track income, expenses, assets, liabilities, and provides a comprehensive overview of your financial health <a class="yt-timestamp" data-t="00:00:31">[00:00:31]</a>.

## Tracker Overview

The Notion personal finance tracker is structured into several key sections:
*   **Net Worth Target** - Allows you to specify your financial year net worth goal <a class="yt-timestamp" data-t="00:00:19">[00:00:19]</a>.
*   **Account Section** - Where you define different accounts for income, expenses, assets, and liabilities, including opening balances, payment frequency, and category <a class="yt-timestamp" data-t="00:00:23">[00:00:23]</a>.
*   **Summary Section** - Provides a single-glance view of your entire finances <a class="yt-timestamp" data-t="00:00:38">[00:00:38]</a>. This includes:
    *   Finance View <a class="yt-timestamp" data-t="00:00:42">[00:00:42]</a>
    *   Debt Section <a class="yt-timestamp" data-t="00:00:52">[00:00:52]</a>
    *   Budget Section <a class="yt-timestamp" data-t="00:01:08">[00:01:08]</a>
    *   Savings Section <a class="yt-timestamp" data-t="00:01:15">[00:01:15]</a>
    *   Bill Section <a class="yt-timestamp" data-t="00:01:23">[00:01:23]</a>
    *   Subscription Section <a class="yt-timestamp" data-t="00:01:32">[00:01:32]</a>
    *   Net Worth Section <a class="yt-timestamp" data-t="00:01:45">[00:01:45]</a>
*   **Transaction Section** - Where all income, expense, asset, or liability transactions are recorded, impacting all other sections <a class="yt-timestamp" data-t="00:01:51">[00:01:51]</a>.

## Setting Up Accounts

Before tracking, you must set up your accounts in the "Account section" under the "Get Started" view <a class="yt-timestamp" data-t="00:02:36">[00:02:36]</a>.

### Income & Expenses
You can find and add various income (e.g., salary, side hustle) and expense accounts (e.g., groceries, utility bills, Netflix) <a class="yt-timestamp" data-t="00:02:40">[00:02:40]</a>.
To add a new income or expense:
1.  Click the "New" button <a class="yt-timestamp" data-t="00:03:14">[00:03:14]</a>.
2.  Type the account name <a class="yt-timestamp" data-t="00:04:10">[00:04:10]</a>.
3.  Ignore the "opening balance" for income and expenses <a class="yt-timestamp" data-t="00:04:16">[00:04:16]</a>.
4.  Classify the expense (e.g., "bills," "subscriptions") <a class="yt-timestamp" data-t="00:03:31">[00:03:31]</a>. Four classifications are typically available: bills, savings, subscription, and debt <a class="yt-timestamp" data-t="00:04:30">[00:04:30]</a>.
5.  Set a payment frequency (e.g., monthly, annually) <a class="yt-timestamp" data-t="00:03:46">[00:03:46]</a>.
Existing accounts can be modified by clicking the edit button <a class="yt-timestamp" data-t="00:02:57">[00:02:57]</a>.

### Debt
To track debt:
1.  Navigate to the "Account section" and then the "Liabilities" part <a class="yt-timestamp" data-t="00:09:11">[00:09:11]</a>.
2.  Click "New" to add a new debt <a class="yt-timestamp" data-t="00:09:27">[00:09:27]</a>.
3.  Specify an "opening balance" for the debt <a class="yt-timestamp" data-t="00:09:31">[00:09:31]</a>.
4.  Classify it as "debt" <a class="yt-timestamp" data-t="00:09:41">[00:09:41]</a>.
5.  Set a payment frequency (e.g., monthly) <a class="yt-timestamp" data-t="00:09:46">[00:09:46]</a>.
Existing debts can be edited via the pencil icon <a class="yt-timestamp" data-t="00:10:08">[00:10:08]</a>.

### Savings Funds
To set up savings funds:
1.  Go to the "Account section" and then the "Asset" part, as savings form part of your assets <a class="yt-timestamp" data-t="00:19:12">[00:19:12]</a>.
2.  Create a new account (e.g., "Education Fund," "Car Fund") <a class="yt-timestamp" data-t="00:19:18">[00:19:18]</a>.
3.  Classify it under the "savings" category <a class="yt-timestamp" data-t="00:19:24">[00:19:24]</a>.
4.  Specify a "goal amount" by clicking on the edit panel for the account <a class="yt-timestamp" data-t="00:20:33">[00:20:33]</a>.
Icons can be added to savings funds for visual distinction <a class="yt-timestamp" data-t="00:24:25">[00:24:25]</a>.

### Bills
To set up bills:
1.  Go to the "Account section" and then "Expenses" <a class="yt-timestamp" data-t="00:25:09">[00:25:09]</a>.
2.  Add a new expense account (e.g., "Wi-Fi bills") <a class="yt-timestamp" data-t="00:25:25">[00:25:25]</a>.
3.  Classify it as "bills" <a class="yt-timestamp" data-t="00:25:34">[00:25:34]</a>.
4.  Set a monthly payment frequency <a class="yt-timestamp" data-t="00:25:36">[00:25:36]</a>.

### Subscriptions
To track subscriptions:
1.  Navigate to the "Account section" and "Expenses" <a class="yt-timestamp" data-t="00:29:13">[00:29:13]</a>.
2.  Specify expenses that are subscriptions (e.g., "Netflix," "Amazon") <a class="yt-timestamp" data-t="00:29:19">[00:29:19]</a>.
3.  Classify them as "subscription" <a class="yt-timestamp" data-t="00:29:24">[00:29:24]</a>.
4.  Set the payment frequency (e.g., annually, quarterly) <a class="yt-timestamp" data-t="00:29:29">[00:29:29]</a>.

### Investments (Stocks, Cryptocurrencies)
To track investments:
1.  Navigate to the "Accounts" tab <a class="yt-timestamp" data-t="00:38:18">[00:38:18]</a>.
2.  Add a new investment account (e.g., "Apple investment," "Bitcoin") <a class="yt-timestamp" data-t="00:38:26">[00:38:26]</a>.
3.  Specify an "opening balance" (if applicable) and "opening units" for the stock or crypto held at the start of the financial year <a class="yt-timestamp" data-t="00:38:35">[00:38:35]</a>.
4.  Classify it (e.g., "stocks," "crypto") <a class="yt-timestamp" data-t="00:38:46">[00:38:46]</a>.
5.  Crucially, select "Investments" as the "Asset Class" <a class="yt-timestamp" data-t="00:39:22">[00:39:22]</a>. This enables the calculation of total investment value in the investment tab <a class="yt-timestamp" data-t="00:39:35">[00:39:35]</a>.

## Recording Transactions

All financial movements are recorded in the "Transaction section" <a class="yt-timestamp" data-t="00:04:54">[00:04:54]</a>. This section has five views: "all" (for all transaction types) and specific views for "income," "expenses," "assets," and "liabilities" <a class="yt-timestamp" data-t="00:05:15">[00:05:15]</a>.

### Income & Expenses Transactions
1.  Navigate to the "income" or "expense" section under "Transactions" <a class="yt-timestamp" data-t="00:05:34">[00:05:34]</a>.
2.  Click "New" <a class="yt-timestamp" data-t="00:05:38">[00:05:38]</a>.
3.  Select the date <a class="yt-timestamp" data-t="00:05:45">[00:05:45]</a>. Transactions are bifurcated by month and listed in descending order <a class="yt-timestamp" data-t="00:05:54">[00:05:54]</a>.
4.  Add a description <a class="yt-timestamp" data-t="00:06:05">[00:06:05]</a>.
5.  Select the relevant account (e.g., "salary," "groceries") <a class="yt-timestamp" data-t="00:06:14">[00:06:14]</a>.
6.  Choose the payment mode (e.g., "bank," "cash") <a class="yt-timestamp" data-t="00:06:28">[00:06:28]</a>.
7.  Set the transaction type: "inflow" for income (money coming in) <a class="yt-timestamp" data-t="00:06:52">[00:06:52]</a> or "outflow" for expenses (money going out) <a class="yt-timestamp" data-t="00:07:15">[00:07:15]</a>.
8.  Specify the amount <a class="yt-timestamp" data-t="00:06:59">[00:06:59]</a>.

### Debt Transactions
1.  Go to the "liability" section under "Transactions" <a class="yt-timestamp" data-t="00:10:35">[00:10:35]</a>.
2.  Click "New" <a class="yt-timestamp" data-t="00:10:53">[00:10:53]</a>.
3.  Select the date <a class="yt-timestamp" data-t="00:10:55">[00:10:55]</a>.
4.  Select the relevant debt account (e.g., "home loan") <a class="yt-timestamp" data-t="00:11:13">[00:11:13]</a>.
5.  Choose the payment mode (e.g., "bank") <a class="yt-timestamp" data-t="00:11:24">[00:11:24]</a>.
6.  Set the transaction type: "outflow" for loan repayments <a class="yt-timestamp" data-t="00:11:33">[00:11:33]</a> or "inflow" if you are obtaining a new loan <a class="yt-timestamp" data-t="00:12:03">[00:12:03]</a>.
7.  Put the amount of the payment <a class="yt-timestamp" data-t="00:11:43">[00:11:43]</a>.

### Savings Transactions
1.  Navigate to the "asset" section under "Transactions" <a class="yt-timestamp" data-t="00:23:07">[00:23:07]</a>.
2.  Add a new transaction <a class="yt-timestamp" data-t="00:23:16">[00:23:16]</a>.
3.  Add a description <a class="yt-timestamp" data-t="00:23:23">[00:23:23]</a>.
4.  Select the specific savings fund account (e.g., "car fund") <a class="yt-timestamp" data-t="00:23:28">[00:23:28]</a>.
5.  Choose the payment mode (e.g., "bank account") <a class="yt-timestamp" data-t="00:23:36">[00:23:36]</a>.
6.  Set the transaction type to "outflow" as money is flowing *out* of your main account *to* the savings account <a class="yt-timestamp" data-t="00:23:48">[00:23:48]</a>.
7.  Specify the amount <a class="yt-timestamp" data-t="00:24:01">[00:24:01]</a>.

### Bill Payments Transactions
1.  Go to the "expense" part under "Transactions" <a class="yt-timestamp" data-t="00:26:01">[00:26:01]</a>.
2.  Enter the date, description, necessary bill account, payment mode, and amount <a class="yt-timestamp" data-t="00:26:08">[00:26:08]</a>.
3.  The transaction type will be "outflow" <a class="yt-timestamp" data-t="00:26:16">[00:26:16]</a>.

### Subscription Payments Transactions
1.  Under "expenses" in the "Transaction" section, add a new transaction <a class="yt-timestamp" data-t="00:30:09">[00:30:09]</a>.
2.  Select the date, add a description, choose the subscription account (e.g., "Amazon"), payment mode, and amount <a class="yt-timestamp" data-t="00:30:17">[00:30:17]</a>.
3.  The transaction type will be "outflow" <a class="yt-timestamp" data-t="00:30:47">[00:30:47]</a>.

### Investment Transactions
1.  Under "assets" in the "Transactions" section, create transactions for purchases and sales <a class="yt-timestamp" data-t="00:40:48">[00:40:48]</a>.
2.  **Purchase**: Record the amount at the *cost price* of the investment <a class="yt-timestamp" data-t="00:41:14">[00:41:14]</a>.
3.  **Sale**: Record the amount at the *cost value* of the units being sold, not the sale price <a class="yt-timestamp" data-t="00:42:01">[00:42:01]</a>. This allows the system to compute profit or loss automatically <a class="yt-timestamp" data-t="00:42:18">[00:42:18]</a>. You can edit the "units" and "price" based on your market transactions <a class="yt-timestamp" data-t="00:43:06">[00:43:06]</a>.

## Tracking & Summary Views

Once transactions are entered, the "Summary" section provides various overviews.

### Finances Overview
Under the "finances" view in the "summary" section, you can see:
*   Total income and a breakdown of each income source by amount and percentage contribution <a class="yt-timestamp" data-t="00:07:56">[00:07:56]</a>.
*   Total expenses and a breakdown of each expense category by amount and percentage <a class="yt-timestamp" data-t="00:08:11">[00:08:11]</a>.
Any changes to figures automatically update the totals and percentages <a class="yt-timestamp" data-t="00:08:39">[00:08:39]</a>.

### Debt Tracking
The [[using_notion_for_debt_tracking | debt section]] under "Summary" has two views: "upcoming payments" and "past payments" <a class="yt-timestamp" data-t="00:13:27">[00:13:27]</a>.
*   **Upcoming payments**: Debts due in the future relative to the current date <a class="yt-timestamp" data-t="00:13:31">[00:13:31]</a>.
*   **Past payments**: Debts whose last payment date was in the past and no payment has been made in the intermediary period <a class="yt-timestamp" data-t="00:14:18">[00:14:18]</a>. This helps identify missed payments <a class="yt-timestamp" data-t="00:14:14">[00:14:14]</a>.
Each debt card shows the total loan amount, total repayment percentage, and the amount yet to be paid <a class="yt-timestamp" data-t="00:15:06">[00:15:06]</a>. The layout can be changed (e.g., list, gallery) <a class="yt-timestamp" data-t="00:15:30">[00:15:30]</a>.

### Budgeting
The [[monthly_budget_planner_setup_using_notion | budget section]] under "Summary" shows incurred expenses and income against budgeted amounts <a class="yt-timestamp" data-t="00:17:16">[00:17:16]</a>.
*   You can set estimated income and expense budgets <a class="yt-timestamp" data-t="00:01:10">[00:01:10]</a>.
*   The system reflects the percentage of incurred expenses against the total budgeted amount <a class="yt-timestamp" data-t="00:17:28">[00:17:28]</a>.
*   Budget amounts can be changed directly <a class="yt-timestamp" data-t="00:17:39">[00:17:39]</a>. If the budgeted amount property is hidden, click on "31 more properties" to make it visible and editable <a class="yt-timestamp" data-t="00:18:31">[00:18:31]</a>.

### Savings Tracking
The [[setting_up_a_savings_funds_tracker_in_notion | savings section]] under "Summary" provides an updated overview of saved amounts against goal amounts <a class="yt-timestamp" data-t="00:20:08">[00:20:08]</a>.
*   It displays the goal amount, the total amount saved so far, and the percentage of savings achieved <a class="yt-timestamp" data-t="00:20:16">[00:20:16]</a>.

### Bills Tracking
The "bills" section under "Summary" also has "upcoming payment" and "past payment" statuses, similar to debt tracking <a class="yt-timestamp" data-t="00:26:28">[00:26:28]</a>.
*   **Upcoming payments**: Bills due after the current date <a class="yt-timestamp" data-t="00:26:40">[00:26:40]</a>.
*   **Past payments**: Bills whose payment date has passed without a transaction being recorded <a class="yt-timestamp" data-t="00:27:19">[00:27:19]</a>.

### Subscriptions Tracking
The "subscription" view under "Summary" has "upcoming payments" and "past payments" sections <a class="yt-timestamp" data-t="00:31:04">[00:31:04]</a>.
*   **Upcoming payments**: Subscriptions with a due date beyond the current date <a class="yt-timestamp" data-t="00:31:12">[00:31:12]</a>.
*   **Past payments**: Subscriptions where the last payment date has passed and no new payment has been recorded for the expected frequency <a class="yt-timestamp" data-t="00:32:01">[00:32:01]</a>.
*   It also shows the subscription status <a class="yt-timestamp" data-t="00:32:20">[00:32:20]</a>. If not visible, click the pencil icon for the account to find the "subscription status" property <a class="yt-timestamp" data-t="00:32:27">[00:32:27]</a>.

### Net Worth Tracking
The [[building_a_net_worth_tracker_using_notion | net worth section]] provides an overview of your financial position <a class="yt-timestamp" data-t="00:01:45">[00:01:45]</a>.
1.  **Set Net Worth Goal**: In the "Net Worth Goal" section, specify your target net worth for the financial year <a class="yt-timestamp" data-t="00:33:05">[00:33:05]</a>.
2.  **Total Net Worth**: This is calculated as the difference between total assets and total liabilities after all transactions are entered <a class="yt-timestamp" data-t="00:33:36">[00:33:36]</a>.
3.  **Difference**: Shows how much you are lagging behind your net worth goal <a class="yt-timestamp" data-t="00:33:57">[00:33:57]</a>.
4.  **Percentage**: Reflects the percentage of your net worth goal achieved <a class="yt-timestamp" data-t="00:34:11">[00:34:11]</a>.
5.  **Total Growth**: Represents the "closing net worth" minus the "opening net worth" <a class="yt-timestamp" data-t="00:34:20">[00:34:20]</a>. The opening net worth is derived from the opening balances of assets and liabilities defined in the account section <a class="yt-timestamp" data-t="00:34:28">[00:34:28]</a>.

### Investments Tracking
The "investment" tab provides a summary of all investments <a class="yt-timestamp" data-t="00:37:08">[00:37:08]</a>.
*   It shows the total investment amount and the number of units available for each investment <a class="yt-timestamp" data-t="00:37:29">[00:37:29]</a>.
*   It reflects the profit or loss from transactions <a class="yt-timestamp" data-t="00:37:38">[00:37:38]</a>.
*   Displays the investment percentage, showing each investment's contribution to the total <a class="yt-timestamp" data-t="00:37:46">[00:37:46]</a>.
*   Shows the total value of available units <a class="yt-timestamp" data-t="00:38:11">[00:38:11]</a>.

## Additional Features & Tips

### Tracking for Couples
This dashboard is applicable for couples to track finances <a class="yt-timestamp" data-t="00:36:31">[00:36:31]</a>. In the "Account section," you can split different categories of accounts (income, expenses, assets, liabilities) by individual <a class="yt-timestamp" data-t="00:35:49">[00:35:49]</a>. For example, "salary by him" and "side hustle by her" <a class="yt-timestamp" data-t="00:36:00">[00:36:00]</a>.

### Viewing Specific Account Transactions
To quickly view all transactions related to a particular account (e.g., "salary"):
1.  In the "finances" section of the "Summary" view, click on any income or account you want to track <a class="yt-timestamp" data-t="00:45:50">[00:45:50]</a>.
2.  A window will open on the right side. Scroll down to find "View transactions" at the bottom <a class="yt-timestamp" data-t="00:46:01">[00:46:01]</a>.
3.  Clicking this will automatically populate all transactions related to that specific account <a class="yt-timestamp" data-t="00:46:37">[00:46:37]</a>.

## Frequently Asked Questions

### Is the template available on all devices?
Yes, the template is available on all devices, including smartphones, tablets, and laptops, supporting iOS, Android, Windows, and Mac OS <a class="yt-timestamp" data-t="00:47:44">[00:47:44]</a>.

### Do I need to pay anything extra for Notion?
Notion is a free tool for basic use; you only need to sign up with an email ID <a class="yt-timestamp" data-t="00:48:14">[00:48:14]</a>. There are no extra charges involved beyond the template's existing price <a class="yt-timestamp" data-t="00:48:35">[00:48:35]</a>. Notion also offers paid plans with additional features <a class="yt-timestamp" data-t="00:48:26">[00:48:26]</a>.

### Is it available on multiple currencies?
Notion supports many currencies <a class="yt-timestamp" data-t="00:48:45">[00:48:45]</a>. To change the currency:
1.  Go to any view with a dollar sign associated with amounts (e.g., "transactions" -> "expenses" view) <a class="yt-timestamp" data-t="00:49:01">[00:49:01]</a>.
2.  Click on the amount property, then "Edit property" <a class="yt-timestamp" data-t="00:49:06">[00:49:06]</a>.
3.  Change the "Number format" from US Dollars to your desired currency (e.g., Euro) <a class="yt-timestamp" data-t="00:49:10">[00:49:10]</a>. This needs to be done for all views where currency is displayed <a class="yt-timestamp" data-t="00:49:26">[00:49:26]</a>.
