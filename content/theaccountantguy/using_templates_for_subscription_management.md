## Setting Up Subscription Payment Reminders

The process involves using a specific template within a database to configure alerts for your subscriptions <a class="yt-timestamp" data-t="00:00:15">[00:00:15]</a>.

### Step-by-Step Setup

1.  **Create a New Template Entry**:
    *   Click on the drop-down menu within the database <a class="yt-timestamp" data-t="00:00:23">[00:00:23]</a>.
    *   Select the option to create a "new template" <a class="yt-timestamp" data-t="00:00:25">[00:00:25]</a>.
    *   Choose the "subscription reminder alert" option <a class="yt-timestamp" data-t="00:00:28">[00:00:28]</a>.
2.  **Define Subscription Details**:
    *   Enter the name of the subscription (e.g., "Spotify") <a class="yt-timestamp" data-t="00:00:32">[00:00:32]</a>.
    *   Specify the subscription amount (e.g., $150) <a class="yt-timestamp" data-t="00:00:37">[00:00:37]</a>.
    *   Click "background" to add this new entry <a class="yt-timestamp" data-t="00:00:41">[00:00:41]</a>. A new entry will appear in the database <a class="yt-timestamp" data-t="00:00:45">[00:00:45]</a>.
3.  **Set Repetition for Reminders**:
    *   Click on the three dots next to the newly added subscription entry <a class="yt-timestamp" data-t="00:00:48">[00:00:48]</a>.
    *   Select "repeat" <a class="yt-timestamp" data-t="00:00:50">[00:00:50]</a>.
    *   Choose your desired repetition mode, such as "every month" <a class="yt-timestamp" data-t="00:00:55">[00:00:55]</a>. This ensures the subscription amount repeats monthly <a class="yt-timestamp" data-t="00:01:00">[00:01:00]</a>.
4.  **Configure Time and Date**:
    *   A new window will open allowing you to specify the exact time and start date for the reminders <a class="yt-timestamp" data-t="00:01:11">[00:01:11]</a>.
    *   Set the desired time (e.g., 11:56) <a class="yt-timestamp" data-t="00:01:18">[00:01:18]</a>.
    *   Confirm the repetition frequency (e.g., "every month on the 23rd") <a class="yt-timestamp" data-t="00:01:28">[00:01:28]</a>.
    *   Click "Save" <a class="yt-timestamp" data-t="00:01:38">[00:01:38]</a>. A repetition arrow will indicate that the alert is set to repeat <a class="yt-timestamp" data-t="00:01:40">[00:01:40]</a>.

## Receiving WhatsApp Notifications

Once the reminder is set, the system will automatically add a new line item in the database at the specified time on the recurring date <a class="yt-timestamp" data-t="00:01:52">[00:01:52]</a>. Following this, a notification alert will be automatically sent to your WhatsApp <a class="yt-timestamp" data-t="00:02:01">[00:02:01]</a>.

These notifications include details like the subscription name (e.g., Spotify) and amount (e.g., $150) <a class="yt-timestamp" data-t="00:02:27">[00:02:27]</a>. Importantly, the message also contains a "pay now" link, allowing direct payment of the subscription amount <a class="yt-timestamp" data-t="00:02:31">[00:02:31]</a>.
