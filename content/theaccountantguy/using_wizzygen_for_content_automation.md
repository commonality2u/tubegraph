## Introduction to Wizzygen
[[introduction_to_wizzygen_platform_for_ai_generated_content | Wizzygen]] is a platform designed to create [[ai_content_generation | AI-generated content]] directly onto a Notion workspace <a class="yt-timestamp" data-t="00:00:00">[00:00:00]</a>.

## Content Creation Options
Wizzygen offers two primary options for [[ai_content_generation | content creation]] within Notion:
*   **Notion Page** A simple, blank page <a class="yt-timestamp" data-t="00:00:13">[00:00:13]</a>.
*   **Notion Database** Structured in rows and columns <a class="yt-timestamp" data-t="00:00:17">[00:00:17]</a>.

Both [[creating_and_managing_notion_pages_and_databases_with_wizzygen | Notion pages and databases]] can be used to host [[ai_content_generation | AI-generated content]] <a class="yt-timestamp" data-t="00:00:21">[00:00:21]</a>.

## Generating Content on a Notion Page
To generate content on a Notion page:
1.  Name the connection (e.g., "notion page") <a class="yt-timestamp" data-t="00:00:33">[00:00:33]</a>.
2.  Select "Notion Page" as the content type <a class="yt-timestamp" data-t="00:00:38">[00:00:38]</a>.
3.  Copy and paste the Notion page URL into the designated section <a class="yt-timestamp" data-t="00:00:43">[00:00:43]</a>.
4.  Click "next" <a class="yt-timestamp" data-t="00:00:46">[00:00:46]</a>.
5.  Input an [[using_wizzygen_to_generate_content_with_ai_prompts | AI prompt]] (e.g., "write me a block of 100 words on notion") <a class="yt-timestamp" data-t="00:00:49">[00:00:49]</a>.

Upon execution, [[ai_content_generation | AI content]] will be generated and populate the Notion page <a class="yt-timestamp" data-t="00:01:04">[00:01:04]</a>.

## Generating Content on a Notion Database
To generate content within a Notion database:
1.  Name the connection (e.g., "notion database") <a class="yt-timestamp" data-t="00:01:21">[00:01:21]</a>.
2.  Select "Notion Database" <a class="yt-timestamp" data-t="00:01:25">[00:01:25]</a>.
3.  Copy the URL of the Notion database and paste it into the URL section <a class="yt-timestamp" data-t="00:01:28">[00:01:28]</a>.
4.  Click "next" <a class="yt-timestamp" data-t="00:01:36">[00:01:36]</a>.
5.  Provide an [[using_wizzygen_to_generate_content_with_ai_prompts | AI prompt]] that can leverage the database's structure (e.g., "write me a block of 100 words on the topic") <a class="yt-timestamp" data-t="00:01:48">[00:01:48]</a>. This allows the AI to generate content for each specific topic or row within the database <a class="yt-timestamp" data-t="00:01:40">[00:01:40]</a>.

After the process begins, the [[ai_content_generation | content will start populating]] each relevant row in the database <a class="yt-timestamp" data-t="00:01:59">[00:01:59]</a>. This demonstrates how Wizzygen can capture every row to generate tailored content <a class="yt-timestamp" data-t="00:02:17">[00:02:17]</a>.
