Wizzygen is a platform designed to help create [[ai_content_automation_in_notion_workspace | AI-generated content]] directly within a [[using_notion_pages_for_ai_content | Notion]] workspace <a class="yt-timestamp" data-t="00:00:02">[00:00:02]</a>.

## Methods for Generating Content

Content can be generated using two primary options within Notion:
1.  [[using_notion_pages_for_ai_content | Notion Page]] <a class="yt-timestamp" data-t="00:00:11">[00:00:11]</a>
2.  [[utilizing_notion_databases_for_ai_content | Notion Database]] <a class="yt-timestamp" data-t="00:00:11">[00:00:11]</a>

A [[using_notion_pages_for_ai_content | Notion page]] is a simple blank page <a class="yt-timestamp" data-t="00:00:13">[00:00:13]</a>, while a [[utilizing_notion_databases_for_ai_content | database]] is structured in rows and columns <a class="yt-timestamp" data-t="00:00:17">[00:00:17]</a>.

## Generating Content on a Notion Page

To create [[generating_content_using_ai_prompts_for_notion_pages_and_databases | AI-generated content]] on a [[using_notion_pages_for_ai_content | Notion page]] using Wizzygen:
1.  Provide a name for the connection (e.g., "notion page") <a class="yt-timestamp" data-t="00:00:32">[00:00:32]</a>.
2.  Select "Notion page" as the content type <a class="yt-timestamp" data-t="00:00:36">[00:00:36]</a>.
3.  Copy and paste the URL of the desired [[using_notion_pages_for_ai_content | Notion page]] into Wizzygen <a class="yt-timestamp" data-t="00:00:41">[00:00:41]</a>.
4.  Input the desired [[generating_content_using_ai_prompts_for_notion_pages_and_databases | AI prompt]], such as "write me a block of 100 words on notion" <a class="yt-timestamp" data-t="00:00:49">[00:00:49]</a>.
5.  Once the prompt is entered, the content will be automatically generated by AI and populated directly onto the [[using_notion_pages_for_ai_content | Notion page]] <a class="yt-timestamp" data-t="00:01:04">[00:01:04]</a>.

## Generating Content on a Notion Database

To create [[generating_content_using_ai_prompts_for_notion_pages_and_databases | AI-generated content]] within a [[utilizing_notion_databases_for_ai_content | Notion database]]:
1.  Name the connection (e.g., "notion database") <a class="yt-timestamp" data-t="00:01:21">[00:01:21]</a>.
2.  Select "Notion database" as the content type <a class="yt-timestamp" data-t="00:01:25">[00:01:25]</a>.
3.  Copy the URL of the specific [[utilizing_notion_databases_for_ai_content | Notion database]] and paste it into Wizzygen <a class="yt-timestamp" data-t="00:01:27">[00:01:27]</a>.
4.  If the database contains multiple topics (e.g., in different rows), an [[generating_content_using_ai_prompts_for_notion_pages_and_databases | AI prompt]] can be used to generate content for each topic <a class="yt-timestamp" data-t="00:01:38">[00:01:38]</a>. For example, using the prompt "write me a block of 100 words on the topic" will instruct the AI to generate content based on the topic found in each respective row <a class="yt-timestamp" data-t="00:01:48">[00:01:48]</a>.
5.  The AI will then populate the content into the corresponding rows of the database <a class="yt-timestamp" data-t="00:02:01">[00:02:01]</a>.

This process allows users to [[using_wizzygen_to_create_aigenerated_content_in_notion | utilize Wizzygen]] to generate [[ai_content_automation_in_notion_workspace | AI content]] for both [[using_notion_pages_for_ai_content | Notion pages]] and [[utilizing_notion_databases_for_ai_content | Notion databases]] efficiently <a class="yt-timestamp" data-t="00:02:21">[00:02:21]</a>.
