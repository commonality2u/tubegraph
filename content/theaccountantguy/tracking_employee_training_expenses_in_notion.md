This Notion-based tracker is designed for managing and monitoring employee training, attendance, and associated expenses. It provides a comprehensive dashboard with various views to keep track of training programs and employee participation <a class="yt-timestamp" data-t="00:00:06">[00:00:06]</a>.

## Dashboard Overview

The main dashboard presents two primary views:
*   **Training Overview** <a class="yt-timestamp" data-t="00:00:08">[00:00:08]</a>
*   **Employee Records** <a class="yt-timestamp" data-t="00:00:09">[00:00:09]</a>

Additionally, an **Expenses Overview** section provides a summary of all training-related costs <a class="yt-timestamp" data-t="00:02:06">[00:02:06]</a>. All information within the tracker is editable and customizable <a class="yt-timestamp" data-t="00:00:37">[00:00:37]</a>.

### Training Overview

This section offers a summary of all scheduled trainings <a class="yt-timestamp" data-t="00:00:11">[00:00:11]</a>. For each individual training, the following details are displayed:
*   **Training Date**: When employees were scheduled to attend <a class="yt-timestamp" data-t="00:00:21">[00:00:21]</a>.
*   **Employees Count**: The number of employees expected to attend <a class="yt-timestamp" data-t="00:00:24">[00:00:24]</a>.
*   **Percentage of Attendance**: Indicates how many employees actually attended <a class="yt-timestamp" data-t="00:00:29">[00:00:29]</a>.
*   **Training Expense**: The cost associated with each training <a class="yt-timestamp" data-t="00:00:34">[00:00:34]</a>.

### Employee Records

This view lists employees organized by their respective departments <a class="yt-timestamp" data-t="00:00:47">[00:00:47]</a>.
Key features include:
*   **Department Management**:
    *   Departments can be edited by clicking on them <a class="yt-timestamp" data-t="00:00:50">[00:00:50]</a>.
    *   New departments can be added using a "plus" button accessible via a slider <a class="yt-timestamp" data-t="00:01:01">[00:01:01]</a>.
    *   Employees can be dragged and dropped between departments to reassign them <a class="yt-timestamp" data-t="00:01:13">[00:01:13]</a>.
    *   Alternatively, clicking the three-dot menu next to an employee allows moving them to a different department <a class="yt-timestamp" data-t="00:01:55">[00:01:55]</a>.
*   **Employee Training Summary**: Each employee's entry shows:
    *   The total number of trainings they were supposed to attend versus how many they actually attended (e.g., "Steve has attended two out of three trainings") <a class="yt-timestamp" data-t="00:01:24">[00:01:24]</a>.
    *   A percentage of their attendance <a class="yt-timestamp" data-t="00:01:36">[00:01:36]</a>.
    *   **Next Training Date**: Allows specifying an upcoming training date, which then calculates the number of days remaining until that date <a class="yt-timestamp" data-t="00:01:38">[00:01:38]</a>.

### Expenses Overview

This section provides a high-level summary of all training expenditures <a class="yt-timestamp" data-t="00:02:06">[00:02:06]</a>:
*   **Total Expense**: The cumulative cost incurred for all trainings <a class="yt-timestamp" data-t="00:02:08">[00:02:08]</a>.
*   **Number of Trainings**: The total count of all conducted trainings <a class="yt-timestamp" data-t="00:02:11">[00:02:11]</a>.
*   **Employees to Attend**: The total number of employees registered across all trainings <a class="yt-timestamp" data-t="00:02:19">[00:02:19]</a>.
*   **Average Expense**: The average expense incurred per employee <a class="yt-timestamp" data-t="00:02:25">[00:02:25]</a>. This metric automatically updates as transaction details are entered <a class="yt-timestamp" data-t="00:02:29">[00:02:29]</a>.

## Detailed Data Views

Below the main dashboard, there are three detailed views for managing data:
*   **Training Records** <a class="yt-timestamp" data-t="00:02:36">[00:02:36]</a>
*   **Employee Details** <a class="yt-timestamp" data-t="00:02:37">[00:02:37]</a>
*   **Training Expenses** <a class="yt-timestamp" data-t="00:02:39">[00:02:39]</a>

### Employee Details

This section lists employees by department, mirroring the Employee Records view above <a class="yt-timestamp" data-t="00:02:43">[00:02:43]</a>.
*   **Adding Employees**: New employees can be added using the "Add Employee" button <a class="yt-timestamp" data-t="00:02:47">[00:02:47]</a>. Newly added employees initially appear under a "No Department" view and can then be moved to any department <a class="yt-timestamp" data-t="00:02:50">[00:02:50]</a>.
*   **Editing Employee Details**: Employee information can be edited by clicking the "Edit" button next to their name <a class="yt-timestamp" data-t="00:03:00">[00:03:00]</a>.

### Training Records

This view contains details for all past and upcoming training sessions <a class="yt-timestamp" data-t="00:03:15">[00:03:15]</a>.
*   **Training Information**: Users can set the training date, feed in specific training details, and select which employees are assigned to the training <a class="yt-timestamp" data-t="00:03:21">[00:03:21]</a>.
*   **Attendance Tracking**: A checkbox is used to indicate if employees have attended a training <a class="yt-timestamp" data-t="00:04:08">[00:04:08]</a>. The attendance percentage in the "Training Overview" automatically updates based on these selections <a class="yt-timestamp" data-t="00:04:30">[00:04:30]</a>.
*   **Adding Trainings**: New training records should be added using the "Add Training" quick action button for efficient data entry <a class="yt-timestamp" data-t="00:04:59">[00:04:59]</a>.

### Training Expenses

This section allows for managing and editing the financial details of each training <a class="yt-timestamp" data-t="00:05:16">[00:05:16]</a>.
*   **Editing Expenses**: Training expenses can be edited by clicking a pencil icon next to each training entry <a class="yt-timestamp" data-t="00:05:23">[00:05:23]</a>. Updates here will automatically reflect in the dashboard's "Expenses Overview" <a class="yt-timestamp" data-t="00:05:29">[00:05:29]</a>.
*   **View Customization**: The view can be changed (e.g., to a table view) to quickly read and edit information <a class="yt-timestamp" data-t="00:05:53">[00:05:53]</a>.

## Customization and Support

The tracker is highly customizable to suit specific requirements <a class="yt-timestamp" data-t="00:06:07">[00:06:07]</a>.

*   **Currency Settings**: Currencies can be changed for any financial field. This involves editing the property of the number format in the database or modifying the dollar sign within formulas on the dashboard <a class="yt-timestamp" data-t="00:06:11">[00:06:11]</a>.
*   **General Customization**: The entire dashboard is customizable to fit unique needs <a class="yt-timestamp" data-t="00:06:41">[00:06:41]</a>.
