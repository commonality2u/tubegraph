## Template Structure and Features

This specific [[customizing_templates_in_notion | template]] organizes expenses into three main categories at the top <a class="yt-timestamp" data-t="00:00:13">[00:00:13]</a>. For each category, it displays a budgeted expense and the actual expenses incurred, with the latter being automatically pulled from a detailed table located at the bottom of the [[creating_and_using_templates_in_notion | template]] <a class="yt-timestamp" data-t="00:00:17">[00:00:17]</a>.

Key features include:
*   **Budget Indicator** A percentage indicator shows the proportion of actual expenses incurred compared to the budgeted expense <a class="yt-timestamp" data-t="00:00:28">[00:00:28]</a>.
*   **Dynamic Updates** All expense values at the top of the [[customizing_templates_in_notion | template]] are dynamically updated as figures are input into the table <a class="yt-timestamp" data-t="00:00:49">[00:00:49]</a>.
*   **Summary Section** A dedicated summary section provides an overview, displaying the total budgeted expense, the total actual expenses incurred, the difference between the two, and the overall percentage of actual expenses incurred <a class="yt-timestamp" data-t="00:00:57">[00:00:57]</a>.

## [[customizing_templates_in_notion | Customization]] Options

The [[customizing_templates_in_notion | Notion template]] offers several options for [[customizing_templates_in_notion | customization]]:
*   **Defining Budgeted Expenses** To set or change a budgeted expense, simply click on the value, allowing you to define a new budget as required <a class="yt-timestamp" data-t="00:00:24">[00:00:24]</a>.
*   **Modifying Categories** Existing expense categories can be modified by clicking on the pencil icon associated with them <a class="yt-timestamp" data-t="00:00:33">[00:00:33]</a>.
*   **Adding New Categories** A new expense category can be added by clicking on the "new page" option <a class="yt-timestamp" data-t="00:00:40">[00:00:40]</a>.
*   **Deleting Categories** To remove an existing category, click on the three-dot icon next to it and select the "delete" option <a class="yt-timestamp" data-t="00:00:43">[00:00:43]</a>.
