This Notion template serves as a [[using_notion_as_an_expense_tracker|business expense tracker]], designed to help keep all business expenses organized under one general ledger <a class="yt-timestamp" data-t="00:00:03">[00:00:03]</a>. It provides a comprehensive system for [[budgeting_and_tracking_expenses_in_Notion|budgeting and tracking expenses in Notion]], offering clear insights into spending habits <a class="yt-timestamp" data-t="00:01:25">[00:01:25]</a>.

## Notion Expense Tracker Setup

The [[notion_expense_tracker_setup|Notion expense tracker setup]] is intuitive, allowing for easy customization and data entry.

### Expense Categories

The tracker begins with predefined heads of business expenses <a class="yt-timestamp" data-t="00:00:13">[00:00:13]</a>. Users can:
*   **Define and Modify Categories**: Easily modify or change existing expense categories by clicking on them <a class="yt-timestamp" data-t="00:00:18">[00:00:18]</a>.
*   **Edit Subcategories**: Subcategories can be edited by clicking a pencil icon <a class="yt-timestamp" data-t="00:00:23">[00:00:23]</a>.
*   **Add New Categories**: A "new" button allows for the addition of entirely new expense categories <a class="yt-timestamp" data-t="00:00:29">[00:00:29]</a>.
*   **Set Budgeted Amounts**: For each category, a budgeted expense amount can be set, which automatically reflects in the overall tracker <a class="yt-timestamp" data-t="00:00:34">[00:00:34]</a>. Changes to budgeted amounts, such as increasing an advertising budget from $100 to $5,000, will update automatically in the expenses overview <a class="yt-timestamp" data-t="00:00:50">[00:00:50]</a>.

### Expenses Overview

The "expenses overview" section provides a summary of financial performance <a class="yt-timestamp" data-t="00:00:41">[00:00:41]</a>:
*   **Total Expenses Incurred**: Displays the total amount of expenses accumulated <a class="yt-timestamp" data-t="00:00:43">[00:00:43]</a>.
*   **Budgeted Expenses**: Shows the budget that was initially set for expenses <a class="yt-timestamp" data-t="00:00:45">[00:00:45]</a>.
*   **Percentage Reflection**: Indicates whether current spending is within or has exceeded the set budget <a class="yt-timestamp" data-t="00:00:47">[00:00:47]</a>.

### Transaction Entry

The "transaction side" is where individual expenses are entered <a class="yt-timestamp" data-t="00:01:05">[00:01:05]</a>:
*   **Date Selection**: Users can choose the date of the expense <a class="yt-timestamp" data-t="00:01:09">[00:01:09]</a>.
*   **Description**: A field is available for detailing the expense <a class="yt-timestamp" data-t="00:01:14">[00:01:14]</a>.
*   **Account Selection**: Users can select from predefined or added accounts <a class="yt-timestamp" data-t="00:01:16">[00:01:16]</a>.
*   **Amount Entry**: The monetary value of the expense is recorded here <a class="yt-timestamp" data-t="00:01:21">[00:01:21]</a>.

## Summary and Benefits

The left-hand summary panel provides a quick snapshot of key financial metrics <a class="yt-timestamp" data-t="00:01:25">[00:01:25]</a>:
