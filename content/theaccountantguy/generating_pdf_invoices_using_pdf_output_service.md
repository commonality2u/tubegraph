This article details how to [[generating_pdf_documents_for_business_purposes|generate PDF documents]], specifically invoices, using a dedicated PDF output service. It covers both single and bulk PDF generation, utilizing a pre-built invoice template.

## Overview of PDF Output Service

The PDF Output service, accessible at `PDFoutput.com`, is a platform designed to [[generating_pdf_documents_for_business_purposes|generate PDF documents]] from various templates <a class="yt-timestamp" data-t="03:08:00">[03:08:00]</a>. This service can be used for tasks like [[using_pdf_output_software_for_invoice_creation|invoice creation]] <a class="yt-timestamp" data-t="03:04:00">[03:04:00]</a>.

### Accessing the Service

To use the service, you need to log in to `PDFoutput.com` <a class="yt-timestamp" data-t="03:13:00">[03:13:00]</a>.

## [[using_templates_to_generate_pdf_invoices|Generating a Single PDF Invoice]]

The service allows for the creation of individual PDF invoices based on a template.

### Steps to Generate an Invoice

1.  **Select Template**: After logging in, use the drop-down menu on the top left to find and select the "invoice" template, often by searching for "inv" <a class="yt-timestamp" data-t="03:23:00">[03:23:00]</a>.
2.  **Fill in Details**: Populate the template with the necessary invoice details, such as:
    *   Invoice number <a class="yt-timestamp" data-t="03:48:00">[03:48:00]</a>
    *   Company name and address <a class="yt-timestamp" data-t="03:49:00">[03:49:00]</a>
    *   Item descriptions, quantities, and unit prices <a class="yt-timestamp" data-t="04:05:00">[04:05:00]</a>
    *   Notes <a class="yt-timestamp" data-t="04:29:00">[04:29:00]</a>
3.  **Currency Setting**: The desired currency can be set, and the entire invoice will update accordingly <a class="yt-timestamp" data-t="03:55:00">[03:55:00]</a>.
4.  **Download PDF**: Once all details are entered, click the "Download PDF" button to [[exporting_completed_pdf_invoices|generate and download the invoice]] <a class="yt-timestamp" data-t="04:16:00">[04:16:00]</a>. A "PDF downloaded successfully" message will appear upon completion <a class="yt-timestamp" data-t="04:20:00">[04:20:00]</a>.

### Invoice Structure

The generated invoice will display defined quantities, unit prices, computed total values, and any added notes <a class="yt-timestamp" data-t="04:26:00">[04:26:00]</a>.

## [[using_pdf4put_for_bulk_invoice_generation|Bulk PDF Invoice Generation]]

The service also offers a "bulk PDF mode" for [[using_pdf4put_for_bulk_invoice_generation|generating multiple invoices at once]] <a class="yt-timestamp" data-t="04:34:00">[04:34:00]</a>.

### Using Bulk Mode

1.  **Input Data**: In bulk mode, you can add multiple rows of invoice information <a class="yt-timestamp" data-t="04:47:00">[04:47:00]</a>. While support for [[generating_pdf_invoices_from_notion_database|Notion database]] integration is planned, currently, data can be copied and pasted into the rows <a class="yt-timestamp" data-t="04:41:00">[04:41:00]</a>.
2.  **Adjust Column Width**: Column widths can be expanded for better visibility <a class="yt-timestamp" data-t="04:52:00">[04:52:00]</a>.
3.  **Generate All PDFs**: Click "Download all PDF" to sequentially [[exporting_completed_pdf_invoices|generate and download]] all the invoices <a class="yt-timestamp" data-t="04:54:00">[04:54:00]</a>.
4.  **Generate Single PDF from Row**: Alternatively, you can click "Download PDF" on a specific row to generate only that invoice's PDF <a class="yt-timestamp" data-t="05:13:00">[05:13:00]</a>.

> [!NOTE]
> The service is continuously being updated, and support for [[generating_pdf_invoices_from_notion_database|Notion database]] integration for bulk generation is planned <a class="yt-timestamp" data-t="04:41:00">[04:41:00]</a>.
