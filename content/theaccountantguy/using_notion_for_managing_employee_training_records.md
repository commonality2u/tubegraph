From: [[theaccountantguy]] <br/> 

The "Employee Training Expenses Tracker" is a Notion-based system designed to manage and track employee training and associated expenses <a class="yt-timestamp" data-t="00:00:02"></a>. It offers a comprehensive [[creating_a_training_dashboard_in_notion | training dashboard]] with two primary views: Training Overview and Employee Records <a class="yt-timestamp" data-t="00:00:06"></a>.

## Training Dashboard Overview

The main dashboard provides quick insights into training activities and employee engagement <a class="yt-timestamp" data-t="00:00:06"></a>.

### Training Overview

This section offers an overview of all scheduled trainings for employees <a class="yt-timestamp" data-t="00:00:11"></a>. For each individual training, the following information is displayed:
*   **Training Date**: When employees were scheduled to attend <a class="yt-timestamp" data-t="00:00:21"></a>.
*   **Employees Count**: The number of employees expected to attend <a class="yt-timestamp" data-t="00:00:24"></a>.
*   **Percentage of Attendance**: Indicates how many employees actually attended <a class="yt-timestamp" data-t="00:00:29"></a>.
*   **Training Expense**: The cost associated with each training <a class="yt-timestamp" data-t="00:00:34"></a>.
All this information is editable and customizable <a class="yt-timestamp" data-t="00:00:37"></a>.

### Employee Records

This section lists employees categorized by their respective departments <a class="yt-timestamp" data-t="00:00:45"></a>.
*   **Department Management**: Departments can be edited <a class="yt-timestamp" data-t="00:00:50"></a> or new departments can be added <a class="yt-timestamp" data-t="00:01:01"></a>.
*   **Employee Training Summary**: Each employee's record shows the total number of trainings they have attended <a class="yt-timestamp" data-t="00:01:21"></a>, providing a quick overview of how many trainings they had and how many they attended <a class="yt-timestamp" data-t="00:01:32"></a>.
*   **Attendance Percentage**: A percentage of attendance is displayed for each employee <a class="yt-timestamp" data-t="00:01:36"></a>.
*   **Next Training Date**: Users can specify the next training date for an employee <a class="yt-timestamp" data-t="00:01:38"></a>, and the system automatically calculates the remaining days until that date <a class="yt-timestamp" data-t="00:01:48"></a>.
*   **Employee Movement**: Employees can be moved between departments by dragging their record <a class="yt-timestamp" data-t="00:01:13"></a> or by using the three-dot menu to shift them to a different department <a class="yt-timestamp" data-t="00:01:54"></a>.

### Expenses Overview

On the left side of the dashboard, an expenses overview provides key financial metrics <a class="yt-timestamp" data-t="00:02:05"></a>:
*   **Total Expense**: The cumulative expense incurred for all trainings <a class="yt-timestamp" data-t="00:02:07"></a>.
*   **Number of Trainings**: The total count of trainings conducted <a class="yt-timestamp" data-t="00:02:11"></a>.
*   **Total Employees**: The number of employees associated with the records <a class="yt-timestamp" data-t="00:02:19"></a>.
*   **Average Expense**: The average expense per employee <a class="yt-timestamp" data-t="00:02:25"></a>.
These figures update automatically as transaction details are entered <a class="yt-timestamp" data-t="00:02:29"></a>.

## Detailed Views and Management

Below the main dashboard, three detailed views allow for granular management of records <a class="yt-timestamp" data-t="00:02:36"></a>: Training Records, [[employee_details_management_in_notion | Employee Details]], and Training Expenses <a class="yt-timestamp" data-t="00:02:37"></a>.

### Employee Details

This section manages individual [[employee_details_management_in_notion | employee details]] and their departmental assignments <a class="yt-timestamp" data-t="00:02:42"></a>.
*   **Adding Employees**: New employees can be added using the "Add Employee" button, initially appearing under a "No Department" view <a class="yt-timestamp" data-t="00:02:47"></a>. They can then be moved to the appropriate department <a class="yt-timestamp" data-t="00:02:53"></a>.
*   **Editing Employees**: Employee names and details can be edited by clicking the edit button and typing <a class="yt-timestamp" data-t="00:02:58"></a>.

### Training Records

This view lists all training sessions attended by employees <a class="yt-timestamp" data-t="00:03:14"></a>.
*   **Setting Training Dates**: The date for each training can be set <a class="yt-timestamp" data-t="00:03:20"></a>.
*   **Training Details**: Specific details about the training can be entered <a class="yt-timestamp" data-t="00:03:25"></a>.
*   **Assigning Employees**: Employees can be selected and assigned to specific trainings <a class="yt-timestamp" data-t="00:03:29"></a>. Employee records can be edited or new ones added directly from this database <a class="yt-timestamp" data-t="00:03:34"></a>.
*   **Tracking Attendance**: A checkbox indicates if assigned employees have attended the training <a class="yt-timestamp" data-t="00:04:06"></a>. This directly impacts the attendance percentage shown in the Training Overview <a class="yt-timestamp" data-t="00:04:30"></a>.
*   **Adding Training Records**: New training records should be added using the "Add Training" quick action button <a class="yt-timestamp" data-t="00:04:59"></a>.

### Training Expenses

Once training records are added, this section displays all trainings and their incurred expenses <a class="yt-timestamp" data-t="00:05:16"></a>.
*   **Editing Expenses**: Expenses can be edited by clicking a pencil icon next to the training <a class="yt-timestamp" data-t="00:05:23"></a>.
*   **View Options**: The view can be switched to a table view for quick editing <a class="yt-timestamp" data-t="00:05:53"></a>.

## Customization

The dashboard is highly customizable to meet specific requirements <a class="yt-timestamp" data-t="00:06:07"></a>.
*   **Currency Settings**: All currencies can be changed. To do this, switch to table view, click on the expense property, edit the property, and change the number format to any desired currency <a class="yt-timestamp" data-t="00:06:11"></a>. Currency symbols in the dashboard's top section can also be edited directly in the formula <a class="yt-timestamp" data-t="00:06:32"></a>.
