This Notion template is designed to help users effectively keep track of house rental payments <a class="yt-timestamp" data-t="00:00:06">[00:00:06]</a>. It provides a comprehensive overview of financial transactions related to rental properties and tenants.

## Summary Section

The template's top section features a summary with four distinct views <a class="yt-timestamp" data-t="00:00:09">[00:00:09]</a>.

### Basic Information

This view allows for the addition of fundamental details, including tenants and the classification of owned properties <a class="yt-timestamp" data-t="00:00:13">[00:00:13]</a>. Property classifications, such as "apartment" and "duplex," are pre-defined, but users can add custom classifications as needed <a class="yt-timestamp" data-t="00:00:20">[00:00:20]</a>.

To add a new tenant:
1.  Click the "Add tenant" button on the left <a class="yt-timestamp" data-t="00:00:27">[00:00:27]</a>.
2.  A new tenant entry will be automatically created <a class="yt-timestamp" data-t="00:00:30">[00:00:30]</a>.
3.  Click the three-dot icon to move the new entry under its respective property category <a class="yt-timestamp" data-t="00:00:33">[00:00:33]</a>.
4.  Click the pencil icon to add the tenant's detailed information <a class="yt-timestamp" data-t="00:00:37">[00:00:37]</a>.

### Property Section

This view displays the different property classifications, with each property listing its tenants <a class="yt-timestamp" data-t="00:00:46">[00:00:46]</a>. For each tenant, the following financial information is visible <a class="yt-timestamp" data-t="00:00:52">[00:00:52]</a>:
*   Total rent amount <a class="yt-timestamp" data-t="00:00:52">[00:00:52]</a>
*   Rent amount that has been paid <a class="yt-timestamp" data-t="00:00:54">[00:00:54]</a>
*   Balance rent amount that is due <a class="yt-timestamp" data-t="00:00:56">[00:00:56]</a>
*   Advanced rent that has been paid so far <a class="yt-timestamp" data-t="00:00:58">[00:00:58]</a>

### Rental History

This section provides a historical record of all rental payments made by tenants, based on the transactions logged at the bottom of the template <a class="yt-timestamp" data-t="00:01:04">[00:01:04]</a>.

### Overall Summary

The final view in the summary section presents an aggregated financial overview, including <a class="yt-timestamp" data-t="00:01:12">[00:01:12]</a>:
*   Total amount due <a class="yt-timestamp" data-t="00:01:14">[00:01:14]</a>
*   Amount that has been paid <a class="yt-timestamp" data-t="00:01:15">[00:01:15]</a>
*   Outstanding rent amount <a class="yt-timestamp" data-t="00:01:17">[00:01:17]</a>
*   Percentage of rent collected <a class="yt-timestamp" data-t="00:01:17">[00:01:17]</a>

## Transaction Section

The transaction section offers three distinct views in addition to a view showing all transactions <a class="yt-timestamp" data-t="00:01:20">[00:01:20]</a>.

### Rent Paid

This view lists all transactions where rent has been successfully paid, showing the respective tenants and amounts <a class="yt-timestamp" data-t="00:01:23">[00:01:23]</a>.

### Rent Due

This section displays information for rent that is currently due and has not yet been paid <a class="yt-timestamp" data-t="00:01:31">[00:01:31]</a>. Clicking on a checkbox next to an entry will mark the rent as paid, moving it to the "Rent Paid" category <a class="yt-timestamp" data-t="00:01:36">[00:01:36]</a>.

### Quarterly Summary

The quarterly summary provides a monthly breakdown of rental payments, collections, and the balance due for each respective quarter <a class="yt-timestamp" data-t="00:01:41">[00:01:41]</a>.

To quickly add a new rental payment, click the dedicated button provided <a class="yt-timestamp" data-t="00:01:50">[00:01:50]</a>.
