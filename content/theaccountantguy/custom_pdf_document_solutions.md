## Template and Database Structure

A payment receipt template demonstrates the core concept <a class="yt-timestamp" data-t="00:00:05">[00:00:05]</a>. This template contains elements enclosed in curly braces, such as `{receipt number}` or `{company website}` <a class="yt-timestamp" data-t="00:00:20">[00:00:20]</a>. These elements are placeholders that will be replaced by data from a corresponding Notion database <a class="yt-timestamp" data-t="00:00:24">[00:00:24]</a>.

The database must have columns with names that exactly match the elements in curly braces within the template, including capitalization and spaces <a class="yt-timestamp" data-t="00:06:46">[00:06:46]</a>. Each row in the database represents a unique document to be generated <a class="yt-timestamp" data-t="00:00:45">[00:00:45]</a>. For example, a database for payment receipts would include columns like "receipt number," "receipt date," "receipt time," and "customer name" <a class="yt-timestamp" data-t="00:00:49">[00:00:49]</a>.

## Bulk Document Generation Process

The general workflow for [[bulk_pdf_document_generation_process | bulk PDF document generation]] involves the following steps:

1.  **Access PDF output.com** Log in to PDF output.com <a class="yt-timestamp" data-t="01:29">[01:29]</a>.
2.  **Duplicate Templates and Databases** From the "Templates" section on PDF output.com, find and duplicate the desired template (e.g., "Payment Receipts") and its corresponding database into your local Notion workspace <a class="yt-timestamp" data-t="02:08">[02:08]</a> <a class="yt-timestamp" data-t="03:01">[03:01]</a>.
3.  **Connect Notion to PDF output**
    *   In PDF output.com, click "Add database" or "Add template" <a class="yt-timestamp" data-t="04:45">[04:45]</a>.
    *   Select your Notion workspace (e.g., "accountant guy workspace") <a class="yt-timestamp" data-t="04:54">[04:54]</a> <a class="yt-timestamp" data-t="03:39">[03:39]</a>.
    *   Select both the duplicated template and database pages from your Notion workspace and allow access <a class="yt-timestamp" data-t="05:05">[05:05]</a> <a class="yt-timestamp" data-t="05:09">[05:09]</a>.
    *   Once authenticated, the selected database and template will be connected <a class="yt-timestamp" data-t="05:33">[05:33]</a>.
4.  **Map Database Properties to Template Elements**
    *   Give your connection a name (e.g., "payment receipts") <a class="yt-timestamp" data-t="05:56">[05:56]</a>.
    *   The system will automatically map database properties (columns) to template elements if naming conventions match <a class="yt-timestamp" data-t="06:28">[06:28]</a> <a class="yt-timestamp" data-t="06:55">[06:55]</a>.
    *   If there are mismatches, you can manually search for and select the correct database property for each template element <a class="yt-timestamp" data-t="06:59">[06:59]</a> <a class="yt-timestamp" data-t="07:04">[07:04]</a>.
    *   Filter and sorting options are available to manage properties <a class="yt-timestamp" data-t="07:16">[07:16]</a>.
5.  **Generate PDFs** Click "Create PDF" to generate the documents <a class="yt-timestamp" data-t="07:27">[07:27]</a>. The system reads each row from the database and populates the template, creating a separate PDF for each <a class="yt-timestamp" data-t="07:34">[07:34]</a>.
6.  **Preview and Download** You can preview a sample document to check the output <a class="yt-timestamp" data-t="07:46">[07:46]</a> <a class="yt-timestamp" data-t="07:49">[07:49]</a>. Once satisfied, click "Download all documents" to receive a zip file containing all the generated PDFs <a class="yt-timestamp" data-t="08:17">[08:17]</a>.
