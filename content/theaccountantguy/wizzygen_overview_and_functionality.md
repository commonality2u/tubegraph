Wizzygen is a platform designed to facilitate the creation of [[using_wizzygen_to_create_aigenerated_content_in_notion | AI-generated content]] directly within a [[notion_overview | Notion]] workspace <a class="yt-timestamp" data-t="00:00:04">[00:00:04]</a>.

## Content Generation Options

Wizzygen offers two primary options for generating content within [[notion_overview | Notion]]:
*   **[[notion_overview | Notion]] Database** <a class="yt-timestamp" data-t="00:00:11">[00:00:11]</a>: A database in [[notion_overview | Notion]] is structured in rows and columns <a class="yt-timestamp" data-t="00:00:17">[00:00:17]</a>.
*   **[[notion_overview | Notion]] Page** <a class="yt-timestamp" data-t="00:00:11">[00:00:11]</a>: A [[notion_overview | Notion]] page is a simple, blank page created within the application <a class="yt-timestamp" data-t="00:00:13">[00:00:13]</a>.

Wizzygen can create [[using_wizzygen_to_create_aigenerated_content_in_notion | AI-generated content]] for both [[notion_overview | Notion]] pages and databases <a class="yt-timestamp" data-t="00:00:22">[00:00:22]</a>.

## Functionality Overview

### Generating Content on a [[notion_overview | Notion]] Page

To generate content on a [[notion_overview | Notion]] page, users follow these steps:
1.  **Name the connection** <a class="yt-timestamp" data-t="00:00:35">[00:00:35]</a>.
2.  **Select "Notion page"** as the connection type <a class="yt-timestamp" data-t="00:00:38">[00:00:38]</a>.
3.  **Provide the [[notion_overview | Notion]] page URL** <a class="yt-timestamp" data-t="00:00:41">[00:00:41]</a>.
4.  **Input the content generation prompt** (e.g., "write me a block of 100 words on notion") <a class="yt-timestamp" data-t="00:00:49">[00:00:49]</a>.

Once submitted, Wizzygen processes the request and generates the specified content directly onto the designated [[notion_overview | Notion]] page <a class="yt-timestamp" data-t="00:01:04">[00:01:04]</a>.

### Generating Content on a [[notion_overview | Notion]] Database

For [[notion_overview | Notion]] databases, Wizzygen can generate content for multiple entries or topics simultaneously:
1.  **Name the connection** (e.g., "Notion database") <a class="yt-timestamp" data-t="00:01:21">[00:01:21]</a>.
2.  **Select "Notion database"** as the connection type <a class="yt-timestamp" data-t="00:01:25">[00:01:25]</a>.
3.  **Provide the [[notion_overview | Notion]] database URL** <a class="yt-timestamp" data-t="00:01:27">[00:01:27]</a>.
4.  **Input a general content generation prompt** that references the database's topics (e.g., "write me a block of 100 words on the topic") <a class="yt-timestamp" data-t="00:01:48">[00:01:48]</a>. This allows the AI to generate unique content for each topic or row in the database <a class="yt-timestamp" data-t="00:01:40">[00:01:40]</a>.

The content is then populated into the respective columns of the [[notion_overview | Notion]] database, with each row receiving relevant [[using_wizzygen_to_create_aigenerated_content_in_notion | AI-generated content]] <a class="yt-timestamp" data-t="00:02:08">[00:02:08]</a>.
