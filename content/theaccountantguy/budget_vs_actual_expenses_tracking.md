[[Budgeted vs actual expenses tracking | Tracking budgeted versus actual expenses]] is essential for understanding and managing business finances. A presented Notion template offers a straightforward method for [[budgeting_and_tracking_business_expenses | tracking business expenses]], particularly for restaurant operations <a class="yt-timestamp" data-t="00:00:03">[00:00:03]</a>.

## Template Overview

The template categorizes expenses, displaying both budgeted and actual figures at the top <a class="yt-timestamp" data-t="00:00:13">[00:00:13]</a>, <a class="yt-timestamp" data-t="00:00:17">[00:00:17]</a>. The actual expenses are dynamically pulled from a table located at the bottom of the template <a class="yt-timestamp" data-t="00:00:21">[00:00:21]</a>.

### Key Features

*   **Budget Definition**
    Users can define new budget expenses for any category by simply clicking on the relevant value <a class="yt-timestamp" data-t="00:00:24">[00:00:24]</a>.
*   **Performance Indicator**
    A percentage indicator shows the amount of actual expenses incurred in [[comparison_of_actual_vs_budgeted_expenses | comparison to the budgeted expense]] <a class="yt-timestamp" data-t="00:00:28">[00:00:28]</a>.
*   **Category Management**
    *   **Modification**: Categories can be modified by clicking a pencil icon <a class="yt-timestamp" data-t="00:00:35">[00:00:35]</a>.
    *   **Addition**: New categories can be added via a "new page" icon <a class="yt-timestamp" data-t="00:00:40">[00:00:40]</a>.
    *   **Deletion**: Categories can be deleted by clicking a three-dot icon <a class="yt-timestamp" data-t="00:00:43">[00:00:43]</a>.
*   **Dynamic Updates**
    All expense values update dynamically as figures are entered into the template <a class="yt-timestamp" data-t="00:00:51">[00:00:51]</a>.
*   **Summary Section**
    The template includes a summary that displays:
    *   Total budgeted expense <a class="yt-timestamp" data-t="00:00:59">[00:00:59]</a>.
    *   Total actual expenses incurred <a class="yt-timestamp" data-t="00:01:00">[00:01:00]</a>.
    *   The difference between budgeted and actual expenses <a class="yt-timestamp" data-t="00:01:04">[00:01:04]</a>.
    *   The percentage of actual expenses incurred so far <a class="yt-timestamp" data-t="00:01:06">[00:01:06]</a>.
