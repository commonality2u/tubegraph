## Essential MCP Server Types

To optimize the performance of AI coders, it is recommended to utilize these three categories of [[building_and_using_mcp_servers | MCP servers]]:

1.  **Up-to-date Documentation Server** <a class="yt-timestamp" data-t="00:00:10">[00:00:10]</a>
    *   Provides current documentation directly to the AI coding assistant <a class="yt-timestamp" data-t="00:00:12">[00:00:12]</a>.
    *   An example is the open-source Crawl for AI RAG [[setting_up_and_using_mcp_servers | MCP server]] <a class="yt-timestamp" data-t="00:00:18">[00:00:18]</a>.
2.  **[[web_search_with_mcp_servers | Web Search]] Server** <a class="yt-timestamp" data-t="00:00:20">[00:00:20]</a>
    *   Enables the AI to perform [[web_search_with_mcp_servers | web searches]] for additional resources <a class="yt-timestamp" data-t="00:00:58">[00:00:58]</a>.
    *   Brave is an example of a tool used for this purpose <a class="yt-timestamp" data-t="00:00:22">[00:00:22]</a>.
3.  **Database Management Server** <a class="yt-timestamp" data-t="00:00:27">[00:00:27]</a>
    *   Allows for [[managing_databases_with_mcp_servers | database management using natural language]] <a class="yt-timestamp" data-t="00:00:31">[00:00:31]</a>.
    *   Neon, a serverless PostgreSQL platform, is used for this function <a class="yt-timestamp" data-t="00:00:35">[00:00:35]</a>.
    *   Capabilities include creating projects, SQL tables, and managing records via natural language <a class="yt-timestamp" data-t="00:00:40">[00:00:40]</a>.

The AI can then leverage the established knowledge base to provide answers efficiently <a class="yt-timestamp" data-t="00:01:19">[00:01:19]</a>.
