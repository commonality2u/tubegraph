## Core Capabilities

### No-Code AI Automation
[[vectorshift_platform_for_building_ai_agents | Vector Shift]] allows users to build AI agents without needing to code <a class="yt-timestamp" data-t="00:00:41">[00:00:41]</a>. It functions similarly to other AI automation tools but offers unique features that make it suitable for a variety of use cases <a class="yt-timestamp" data-t="00:00:46">[00:00:46]</a>.

### [[using_vector_shift_for_voice_bots | Voice AI Agent Capabilities]]
One of the most significant features of [[vectorshift_platform_for_building_ai_agents | Vector Shift]] is its ease of implementing voice AI agents, which is often difficult with other platforms <a class="yt-timestamp" data-t="00:00:16">[00:00:16]</a>. This platform enables users to chat with agents as if they were a person <a class="yt-timestamp" data-t="00:00:10">[00:00:10]</a>.

Key steps for [[using_vector_shift_for_voice_bots | building a voice agent]] include:
*   **Audio Input**: The platform supports audio as an input type for workflows <a class="yt-timestamp" data-t="00:03:12">[00:03:12]</a>.
*   **Speech-to-Text Node**: Converts input audio into text, often utilizing models like OpenAI Whisper <a class="yt-timestamp" data-t="00:03:36">[00:03:36]</a>.
*   **Text-to-Speech Node**: Converts the AI agent's response back into audio, with options for OpenAI and 11 Labs <a class="yt-timestamp" data-t="00:08:00">[00:08:00]</a>.
*   **Audio Output**: The final output of the workflow can be set to audio <a class="yt-timestamp" data-t="00:08:31">[00:08:31]</a>.

### Knowledge Base Integration
[[vectorshift_platform_for_building_ai_agents | Vector Shift]] facilitates the creation and integration of knowledge bases, such as those connected to Google Drive <a class="yt-timestamp" data-t="00:00:39">[00:00:39]</a>. Users can easily ingest folders from Google Drive into a knowledge base <a class="yt-timestamp" data-t="00:02:20">[00:02:20]</a> and preview documents and their chunks within the knowledge base <a class="yt-timestamp" data-t="00:02:31">[00:02:31]</a>.

When [[creating_a_rag_ai_agent_with_vectorshift | building a RAG AI agent with VectorShift]], the knowledge base can be queried based on the user's input, with relevant document chunks being sent as context to the Large Language Model (LLM) <a class="yt-timestamp" data-t="00:07:22">[00:07:22]</a>.

### Workflow Builder and Node System
The platform features a visual workflow builder where users connect various nodes:
*   **Input Node**: Configurable for text or audio input <a class="yt-timestamp" data-t="00:03:03">[00:03:03]</a>.
*   **Speech to Text Node**: Converts audio to text <a class="yt-timestamp" data-t="00:03:39">[00:03:39]</a>.
*   **Large Language Model (LLM) Node**: Allows integration with models like GPT-4o Mini and custom API keys <a class="yt-timestamp" data-t="00:03:56">[00:03:56]</a>. It supports defining system messages for LLM behavior and user prompts with dynamic parameters using squiggly brackets `{{prompt}}` <a class="yt-timestamp" data-t="00:04:13">[00:04:13]</a>.
*   **Knowledge Base Node**: Retrieves context for the LLM <a class="yt-timestamp" data-t="00:07:20">[00:07:20]</a>.
*   **Text to Speech Node**: Converts LLM responses to audio <a class="yt-timestamp" data-t="00:08:00">[00:08:00]</a>.
*   **Output Node**: Configurable for text or audio output <a class="yt-timestamp" data-t="00:08:28">[00:08:28]</a>.
*   **Logic Tab**: Includes nodes for various operations like summarization <a class="yt-timestamp" data-t="00:07:07">[00:07:07]</a>.

### Broad Integrations and Flexibility
[[vectorshift_platform_for_building_ai_agents | Vector Shift]] offers an "Integrations" tab with numerous services, similar to Zapier or Make.com <a class="yt-timestamp" data-t="00:06:01">[00:06:01]</a>. It allows calling into external workflows from platforms like n8n, or custom agents built with Python, using an API node for hooking into any API endpoint <a class="yt-timestamp" data-t="00:06:27">[00:06:27]</a>. This provides significant flexibility for building various [[ai_capabilities_versus_tools | AI capabilities versus tools]] <a class="yt-timestamp" data-t="00:06:45">[00:06:45]</a>.

These voice bots can be embedded as a chat widget or an iframe on a website <a class="yt-timestamp" data-t="00:09:13">[00:09:13]</a>. [[vectorshift_platform_for_building_ai_agents | Vector Shift]] also provides customizable UI options for the embedded bot and offers embed code for easy integration <a class="yt-timestamp" data-t="00:10:06">[00:10:06]</a>, <a class="yt-timestamp" data-t="00:10:46">[00:10:46]</a>.
