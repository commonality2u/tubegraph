As a developer, a key lesson is to work smarter, not harder, by utilizing [[developer_productivity_tools | tools that increase productivity]] and simplify workflows <a class="yt-timestamp" data-t="00:00:08">[00:00:08]</a>. This article highlights seven specific tools that can be valuable for developers involved in various stages of the product creation process <a class="yt-timestamp" data-t="00:00:38">[00:00:38]</a>. These tools are personally used and recommended, and notably, they are all free and built with web technologies <a class="yt-timestamp" data-t="00:06:03">[00:06:03]</a>.

## Bundle Phobia

Bundle Phobia is a web application designed to help determine the impact an NPM package will have on an application's total bundle size <a class="yt-timestamp" data-t="00:00:52">[00:00:52]</a>.

### Key Features:
*   **Bundle Size Analysis** It allows developers to analyze the addition of new dependencies to a project <a class="yt-timestamp" data-t="00:01:07">[00:01:07]</a>.
*   **Detailed Information** By entering a package name, it provides information such as bundle size, download time, and whether the package is tree-shakable <a class="yt-timestamp" data-t="00:01:10">[00:01:10]</a>.
*   **Version Comparison** It offers this information for every version of a package <a class="yt-timestamp" data-t="00:01:17">[00:01:17]</a>.
*   **Dependency Ranking** Users can upload their entire `package.json` file to see all their dependencies ranked by size <a class="yt-timestamp" data-t="00:01:19">[00:01:19]</a>.

Keeping bundle size small is crucial for optimizing initial page load for modern [[comparison_of_web_development_frameworks | front-end frameworks]] <a class="yt-timestamp" data-t="00:00:58">[00:00:58]</a>.

## Cloud Craft

Cloud Craft is a web application that assists in designing and budgeting cloud infrastructure <a class="yt-timestamp" data-t="00:01:28">[00:01:28]</a>. Currently, it is exclusively available for AWS, but its utility extends to users of other cloud providers like GCP due to similar pricing models <a class="yt-timestamp" data-t="00:01:33">[00:01:33]</a>.

### Key Features:
*   **Intuitive Interface** Offers a drag-and-drop interface for connecting various cloud infrastructure services <a class="yt-timestamp" data-t="00:01:38">[00:01:38]</a>.
*   **Planning and Budgeting** Excellent for planning infrastructure and providing estimated breakdowns of total costs <a class="yt-timestamp" data-t="00:01:44">[00:01:44]</a>.

## Figma

Figma is a widely recognized design tool used for drawing graphics and creating mock-ups for applications <a class="yt-timestamp" data-t="00:02:09">[00:02:09]</a>. It offers both desktop and web applications, with the web app providing comparable performance due to its use of WebAssembly <a class="yt-timestamp" data-t="00:02:15">[00:02:15]</a>.

### Key Features:
*   **Cross-Platform Accessibility** Being web-based, projects can be opened on any machine, with all files immediately accessible <a class="yt-timestamp" data-t="00:02:30">[00:02:30]</a>.
*   **Streamlined Design** Similar to Sketch, Figma provides essential tools without unnecessary clutter <a class="yt-timestamp" data-t="00:02:40">[00:02:40]</a>.
*   **App Developer Superpowers** Allows for mock-ups of different events on elements and the export of styles as CSS, Android, or iOS styles <a class="yt-timestamp" data-t="00:02:46">[00:02:46]</a>.
*   **Free to Use** It is completely free for graphic design needs <a class="yt-timestamp" data-t="00:02:28">[00:02:28]</a>.

Figma is a highly productive tool, even surpassing the efficiency of traditional desktop apps like Adobe Illustrator for some users <a class="yt-timestamp" data-t="00:02:24">[00:02:24]</a>.

## Font Flipper

Font Flipper is a web application built with Angular and Firebase, designed solely to assist in choosing fonts for designs <a class="yt-timestamp" data-t="00:03:05">[00:03:05]</a>. It addresses the tedious process of selecting from hundreds or thousands of free fonts available on Google Fonts <a class="yt-timestamp" data-t="00:03:13">[00:03:13]</a>.

### Key Features:
*   **Graphic Integration** Users can import their own graphics to experiment with different fonts <a class="yt-timestamp" data-t="00:03:20">[00:03:20]</a>.
*   **Simple UI** Features a focused and minimalist user interface to avoid distractions <a class="yt-timestamp" data-t="00:03:25">[00:03:25]</a>.
*   **Tinder-style Selection** Utilizes a Tinder-style swipe UI to select fonts, swiping left for dislikes and right for likes <a class="yt-timestamp" data-t="00:03:30">[00:03:30]</a>.
*   **Direct Download** Once fonts are selected, they can be directly downloaded from Google Fonts <a class="yt-timestamp" data-t="00:03:37">[00:03:37]</a>.

## VizBug

VizBug is a Chrome extension released in 2019 that enables visual interaction with websites <a class="yt-timestamp" data-t="00:04:12">[00:04:12]</a>. It allows developers to make live changes in the browser without writing CSS <a class="yt-timestamp" data-t="00:04:05">[00:04:05]</a>.

### Key Features:
*   **Visual Interaction** Provides a toolbar and visual indicators (pink lines) hovering over DOM elements <a class="yt-timestamp" data-t="00:03:51">[00:03:51]</a>.
*   **Live Editing** Allows for measuring elements, moving items, and changing styles directly in the browser <a class="yt-timestamp" data-t="00:04:01">[00:04:01]</a>.
*   **CSS Export** Users can copy and paste the generated CSS from the inspector once they achieve a desired state <a class="yt-timestamp" data-t="00:04:11">[00:04:11]</a>.
*   **Element Search** Ability to search for specific types of elements, such as all links on a page, and highlight them <a class="yt-timestamp" data-t="00:04:16">[00:04:16]</a>.

This tool is a powerful [[front_end_development_aids | front end development aid]] that can save significant time once its learning curve is overcome <a class="yt-timestamp" data-t="00:04:25">[00:04:25]</a>.

## Insomnia

Insomnia is a backend [[api_debugging_tools | API debugging tool]] that helps organize the workflow for building and debugging APIs, including both REST and GraphQL <a class="yt-timestamp" data-t="00:04:30">[00:04:30]</a>.

### Key Features:
*   **Workflow Organization** Helps keep the entire API workflow organized <a class="yt-timestamp" data-t="00:04:44">[00:04:44]</a>.
*   **Reusable Values** Allows saving reusable values to avoid rewriting tedious HTTP requests during debugging <a class="yt-timestamp" data-t="00:04:46">[00:04:46]</a>.
*   **User-Friendly UI** Features a well-designed and laid-out user interface <a class="yt-timestamp" data-t="00:04:55">[00:04:55]</a>.
*   **GraphQL Introspection** Automatically performs introspection for GraphQL APIs, providing Intellisense for queries <a class="yt-timestamp" data-t="00:05:02">[00:05:02]</a>.

The combination of GraphQL's power and Insomnia's features provides an excellent overall [[software_engineering_and_development_tools | developer experience]] <a class="yt-timestamp" data-t="00:05:09">[00:05:09]</a>.

## Flare

Flare is a design tool specifically for creating 2D vector animations <a class="yt-timestamp" data-t="00:05:17">[00:05:17]</a>. Similar to Figma, it allows for drawing graphics, but also includes keyframes to control motion over time <a class="yt-timestamp" data-t="00:05:23">[00:05:23]</a>.

### Key Features:
*   **Open Design Movement** Part of the open design movement, allowing users to view and learn from other designers' work directly on the site <a class="yt-timestamp" data-t="00:05:31">[00:05:31]</a>.
*   **Flutter Export** Graphics can be exported to work directly in Flutter, simplifying the implementation of high-performance animations in native mobile apps <a class="yt-timestamp" data-t="00:05:42">[00:05:42]</a>.

Flare is considered an excellent option for building animations in native applications <a class="yt-timestamp" data-t="00:05:52">[00:05:52]</a>.
