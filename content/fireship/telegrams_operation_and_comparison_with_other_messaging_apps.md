## [[telegrams_encryption_and_privacy_features | Telegram's Encryption]] and Data Handling

While a common perception is that everything on Telegram is encrypted, most communications on the platform are not end-to-end encrypted <a class="yt-timestamp" data-t="00:01:57">[00:01:57]</a>. Messages are encrypted but typically on the server, not directly on the user's device <a class="yt-timestamp" data-t="00:02:00">[00:02:00]</a>.

### End-to-End Encryption

End-to-end encryption is the most secure method for private messaging <a class="yt-timestamp" data-t="00:02:04">[00:02:04]</a>. In this method, encryption and decryption occur solely on the user's devices <a class="yt-timestamp" data-t="00:02:07">[00:02:07]</a>. Even if messages are stored on a centralized server, only the person with the decryption key can read them, not the hosting company or authorities <a class="yt-timestamp" data-t="00:02:14">[00:02:14]</a>. Accessing such messages typically requires physical access to the user's device <a class="yt-timestamp" data-t="00:02:20">[00:02:20]</a>.

Apps like Signal utilize end-to-end encryption with their own open-source protocol <a class="yt-timestamp" data-t="00:02:26">[00:02:26]</a>. Signal itself was recently banned in Russia <a class="yt-timestamp" data-t="00:02:29">[00:02:29]</a>. Telegram does offer end-to-end encryption for private chats <a class="yt-timestamp" data-t="00:02:32">[00:02:32]</a>, but other communications are public and encrypted on the server <a class="yt-timestamp" data-t="00:02:35">[00:02:35]</a>. Telegram's protocol has faced criticism from security researchers <a class="yt-timestamp" data-t="00:02:37">[00:02:37]</a>.

### Distributed Infrastructure

A key reason authorities find Telegram challenging is its intentionally distributed infrastructure across multiple countries <a class="yt-timestamp" data-t="00:02:42">[00:02:42]</a>. This setup means that if an agent seeks user data, they must coordinate with several different legal systems <a class="yt-timestamp" data-t="00:02:46">[00:02:46]</a>. Even if a hard drive is seized in one country, the data would be encrypted with a key stored in another country <a class="yt-timestamp" data-t="00:02:51">[00:02:51]</a>.

Authorities, particularly in the EU, have expressed a strong aversion to encryption, with figures like Ilva Johansson pushing to make end-to-end encryption illegal <a class="yt-timestamp" data-t="00:03:04">[00:03:04]</a>. This push has been unsuccessful so far <a class="yt-timestamp" data-t="00:03:11">[00:03:11]</a>. Encryption makes it nearly impossible to enforce laws, such as those criminalizing hate speech in parts of Europe and the UK, if a major platform like Telegram does not cooperate <a class="yt-timestamp" data-t="00:03:13">[00:03:13]</a>.
