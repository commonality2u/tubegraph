## Tech Twitter

Twitter hosts a subculture known as "Tech Twitter," where many individuals have successfully landed jobs by contributing value to the community <a class="yt-timestamp" data-t="03:34:00">03:34:00</a>.

To grow your Twitter presence:
*   **Follow Influencers** Follow influential people you aspire to work for <a class="yt-timestamp" data-t="03:44:00">03:44:00</a>.
*   **Transparent Profile** Create a profile that clearly states your journey to land your first tech job <a class="yt-timestamp" data-t="03:47:00">03:47:00</a>.
*   **Engage Positively** Respond to other people's tweets with encouragement and positivity <a class="yt-timestamp" data-t="03:52:00">03:52:00</a>.
*   **Algorithm Hacks**
    *   Post threads about new things you learn <a class="yt-timestamp" data-t="04:02:00">04:02:00</a>.
    *   Share code tips using tools like Carbon to create snippets with emojis <a class="yt-timestamp" data-t="04:05:00">04:05:00</a>.
    *   Create lists of resources for developers <a class="yt-timestamp" data-t="04:09:00">04:09:00</a>.
    *   Tweet open-ended questions and create your own memes <a class="yt-timestamp" data-t="04:16:00">04:16:00</a>. Memes can be a significant driver of growth <a class="yt-timestamp" data-t="04:20:00">04:20:00</a>.
*   **Consistency** Tweet daily, often multiple times per day, to maintain consistency <a class="yt-timestamp" data-t="04:22:00">04:22:00</a>.
*   **Caution** Avoid negative or controversial tweets, as they can lead to public criticism and hinder your professional journey <a class="yt-timestamp" data-t="04:36:00">04:36:00</a>.

## LinkedIn

LinkedIn is a platform rich with recruiters who can assist in job placement within the tech industry <a class="yt-timestamp" data-t="04:53:00">04:53:00</a>.

To leverage LinkedIn:
*   **Connect with Role Models** Find and message people you admire in the industry, asking to connect <a class="yt-timestamp" data-t="05:02:00">05:02:00</a>. Most people are willing to connect <a class="yt-timestamp" data-t="05:08:00">05:08:00</a>.
*   **Join Groups** Participate in groups related to technologies you are interested in <a class="yt-timestamp" data-t="05:14:00">05:14:00</a>.
*   **Post Content** Reuse content from your Twitter threads and post it on LinkedIn to build your network <a class="yt-timestamp" data-t="05:18:00">05:18:00</a>.

## GitHub

For self-taught programmers, having an active GitHub profile is extremely important <a class="yt-timestamp" data-t="05:29:00">05:29:00</a>.
*   **Consistent Commits** Ideally, strive for daily commits to your own projects and to open-source projects <a class="yt-timestamp" data-t="05:34:00">05:34:00</a>. This consistency shows employers your capability to write code regularly <a class="yt-timestamp" data-t="05:50:00">05:50:00</a>.
*   **Open Source Contributions**
    *   Start by looking for typos in documentation of open-source projects (e.g., Fireship.io) and submit pull requests <a class="yt-timestamp" data-t="05:55:00">05:55:00</a>.
    *   Open issues when you find problems in open-source libraries <a class="yt-timestamp" data-t="06:08:00">06:08:00</a>.
    *   Look for issues tagged with "good first issue" if you want to contribute code <a class="yt-timestamp" data-t="06:13:00">06:13:00</a>.
*   **Benefits** While it's unpaid work, it significantly strengthens your GitHub profile and provides valuable development experience <a class="yt-timestamp" data-t="06:17:00">06:17:00</a>.
