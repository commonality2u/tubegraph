When you first launch the [[basic_linux_commands_and_shell_usage | shell]], it automatically runs a startup script <a class="yt-timestamp" data-t="01:00:00">[01:00:00]</a>. This script is defined in either the `bash_profile` or `bashrc` file located on your system <a class="yt-timestamp" data-t="01:02:00">[01:02:00]</a>, <a class="yt-timestamp" data-t="01:04:00">[01:04:00]</a>, <a class="yt-timestamp" data-t="01:07:00">[01:07:00]</a>.

## Customization Options
The startup script allows you to [[customizing_linux_through_configuration_files | customize the behavior and appearance]] of the shell every time you start a new session <a class="yt-timestamp" data-t="01:07:00">[01:07:00]</a>, <a class="yt-timestamp" data-t="01:09:00">[01:09:00]</a>, <a class="yt-timestamp" data-t="01:10:00">[01:10:00]</a>.
